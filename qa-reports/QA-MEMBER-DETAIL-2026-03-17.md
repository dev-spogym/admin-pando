#### BUG-07: D-Day 더블 대시 표시 [Medium]
- **현상**: 만료일이 과거인 경우 "D--183" 으로 표시 (대시가 2개)

## 10. 잘 된 점
- 상태 필터 5종 모두 정상 동작
- 이름/전화번호 검색 정상 (debounce 적용)
- 성별 필터 정상
- 페이지네이션 정상 (서버사이드)
- 서버사이드 정렬 구현
- 회원 상세 7개 탭 모두 정상 렌더링
- 회원 수정 시 기존 데이터 프리필 정상
- 일괄 상태 변경 + 메시지 발송 기능 구현
- 빈 상태(Empty State) 처리 대부분 양호
- 홀딩 해제 버튼이 홀딩 상태에서만 표시
- "위험 구역" 경고와 함께 삭제 버튼 하단 분리 배치 (안전한 UX)
