**액션**: 검색하기, 관심회원변경, 엑셀 다운로드

### 1.3 회원 상세 (/member/info/{id})

**11개 탭**:
| # | 탭명 | 주요 기능 |
|---|------|---------|
| 1 | 회원정보 | 기본정보, 방문이력/등록, 메인센터 변경, 일일사물함, 출입관리(NFC), 수업완료처리(지문), 회원권/옵션 내역, 상담내역 |
| 2 | 계약 정보 | 계약 이력, 계약 상세 |
| 3 | 신체정보 | 키, 몸무게, 혈압 등 신체 측정 |
| 4 | 체성분 정보 | InBody 등 체성분 데이터 |
| 5 | 종합평가 | 트레이너 종합 평가 |
| 6 | 설문 | 설문조사 응답 |
| 7 | 운동 프로그램 | 배정된 운동 프로그램 |
| 8 | 운동 이력 | 운동 수행 기록 |
| 9 | 온핏 분석 | AI 기반 회원 분석 |
| 10 | 상담 이력 | 상담 기록/메모 |
| 11 | 예약이력 | 수업/일정 예약 이력 |

**회원정보 탭 하단 섹션**:
- D+N일 만료 표시, 관심회원(별) 마킹
- 일일 사물함: 사용 이력 / 사물함 할당
- 출입 관리: 시설 이용(NFC), GX룸(NFC), 골프룸(NFC) + 카드 등록
- 수업 완료처리: 출석(지문) + 지문 등록
- 회원권,옵션 내역 (유효한 상품만 필터)
- 상담내역

### 1.4 방문 이력 (/member/visit-list)

**헤더**: 날짜 선택 (화살표 네비게이션), 상세검색 버튼, 리스트형 드롭다운, 실시간 입장 팝업 여부 체크박스

**테이블 컬럼**:
| 컬럼 | 설명 |
|------|------|
| No | 순번 |
| 회원명 | 이름 (회원번호) |
| 재실여부 | 입장/퇴장 상태 |
| 상태 | 정상 / 일중복입장 |
| 만료일 | 날짜 (D-day) |
| 입장시간 | HH:MM |
| 퇴장시간 | HH:MM |
| 방문회차 | 당일 N번째 |
| 회원권내역 | 보유 상품 목록 |
| 성별 | 남/여 |
| 락카번호 | 락카 번호 |

### 1.5 전자 계약 (/member/contract/elec-contract)

**5단계 위자드 플로**:
1. **회원정보입력**: 이름*, 성별*, 휴대전화*(중복확인), 생년월일*, 주소, 이메일, 유입경로 + 회원선택 버튼
2. **상품선택**: 계약할 상품 선택
3. **약관동의**: 이용약관 동의
4. **결제**: 결제 수단 선택 및 처리
5. **계약서 확인**: 최종 계약서 확인/출력
