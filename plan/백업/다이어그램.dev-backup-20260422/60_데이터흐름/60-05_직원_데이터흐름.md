## 1. 엔티티 개요

직원(`Staff`)의 근태 로그(`Attendance`)를 집계해 급여(`Payroll`)가 월별 자동 계산된다. S8 EmployeeStatus, S13 AttendanceStatus 참조.

## 2. ER 다이어그램

```mermaid
erDiagram
  Staff ||--o{ Attendance : 기록
  Staff ||--o{ Payroll : 급여
  Payroll ||--|{ PayrollItem : 라인
  Staff }o--|| Role : RBAC
  Staff }o--|| Branch : 소속
  Payroll }o--o| PayrollTemplate : 템플릿
```

## 3. 쓰기 경로 (근태 체크인)

```mermaid
flowchart LR
  KIOSK[키오스크] -->|E_DF05_01| API[POST /api/attendance/check-in]
  API --> DB1[(INSERT attendances type=IN)]
  DB1 --> Calc[지각 판정]
  Calc --> DB2[(UPDATE staff.last_check)]
  DB2 --> Evt[이벤트: staff.checked_in] --> Audit
```

## 4. 자동 급여 계산 (크론 A09)

```mermaid
flowchart LR
  Cron[[매월 1일 00:00]] -->|E_DF05_02| Job[급여 계산 Job]
  Job --> Agg[(SELECT SUM attendance 기간 WHERE staff)]
  Agg --> Calc[기본급+수당-공제 계산]
  Calc --> DB[(INSERT payrolls + payroll_items)]
  DB --> Notif[명세서 발송]
```

## 5. 주요 필드

| 필드 | 테이블 | 비고 |
|------|--------|------|
| staff.role | staff | 6종 RBAC |
| staff.status | staff | S8 |
| attendance.type | attendances | IN/OUT/BREAK |
| attendance.recorded_at | attendances | timestamp |
| payroll.period | payrolls | YYYY-MM |
| payroll_item.kind | payroll_items | BASE/BONUS/DEDUCTION |

## 6. 인덱스/제약

- `INDEX(attendance.staff_id, recorded_at)`
- `UNIQUE(payrolls.staff_id, period)`
