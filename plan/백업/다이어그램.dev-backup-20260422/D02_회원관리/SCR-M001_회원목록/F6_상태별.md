SCR-M001의 로딩/빈/에러/권한없음 등 UI 상태별 화면 분기를 명세한다. UI 상태 TC 원천.

## 4. 엣지 설명 테이블

| 엣지 ID | 출발 | 도착 | 조건 |
|---------|------|------|------|
| E_ENTER_AUTH_01 | 진입 | 권한 확인 | 항상 |
| E_AUTH_READONLY_01 | 권한 확인 | 권한없음 | readonly 역할 |
| E_AUTH_OK_01 | 권한 확인 | 로딩 | primary/owner/manager/fc/staff |
| E_LOAD_API_01 | 로딩 | API 응답 | GET /api/members 호출 |
| E_RESP_OK_DATA_01 | API 응답 | 정상 목록 | 200, 데이터 존재 |
| E_RESP_OK_EMPTY_01 | API 응답 | 빈 상태(검색) | 200, 검색 필터 결과 0건 |
| E_RESP_OK_EMPTY_02 | API 응답 | 빈 상태(전체) | 200, 등록 회원 없음 |
| E_RESP_ERR_01 | API 응답 | 에러 배너 | 4xx/5xx/네트워크 오류 |
| E_RETRY_01 | 에러 배너 | 로딩 | 다시 시도 클릭 |
| E_NORMAL_FILTER_01 | 정상 목록 | 로딩 | 필터/탭 변경 |
| E_NORMAL_BULK_01 | 정상 목록 | 부분 로딩 | BulkAction 실행 중 |
| E_BULK_DONE_01 | 부분 로딩 | 정상 목록 | 완료 후 refetch |
