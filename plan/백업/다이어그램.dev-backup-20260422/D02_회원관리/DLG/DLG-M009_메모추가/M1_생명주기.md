## 1. 목적

DLG-M009 메모 추가 인라인 폼의 트리거/저장/완료 생명주기를 명세한다.

## 2. 트리거/전제조건

- 상담메모 탭 > "메모 저장" 버튼 클릭 (인라인 폼, 모달 아님)

## 3. 다이어그램

```mermaid
flowchart TD
  TAB([상담메모 탭 활성화])

  TAB -->|E_SHOW_FORM_01| INLINE_FORM[인라인 메모 입력 폼 표시]
  INLINE_FORM --> FILL[카테고리 선택 + 내용 입력]

  FILL --> BTN_SAVE{메모 저장 버튼}

  BTN_SAVE -->|E_DISABLED_01 내용 비어있음| BTN_DISABLED[버튼 disabled]
  BTN_SAVE -->|E_SUBMIT_01 내용 입력됨| API_CALL[POST /api/members/:id/memos INSERT]

  API_CALL -->|E_SUCCESS_01 200| TOAST_OK((toast.success 메모가 저장되었습니다))
  TOAST_OK -->|E_RESET_01| RESET[폼 초기화 카테고리=상담 내용=빈값]
  RESET    -->|E_REFETCH_01| REFETCH[메모 목록 refetch]

  API_CALL -->|E_FAIL_01 오류| TOAST_ERR((toast.error 메모 저장 실패 error))

  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
  classDef error   fill:#FFEBEE,stroke:#C62828,color:#B71C1C
  classDef system  fill:#EDE7F6,stroke:#5E35B2

  class TOAST_OK,RESET,REFETCH success
  class BTN_DISABLED,TOAST_ERR error
  class INLINE_FORM,BTN_SAVE,API_CALL system
```

## 4. 엣지 설명

| 엣지 ID | 출발 | 도착 | 조건 |
|---------|------|------|------|
| E_SHOW_FORM_01 | 탭 활성 | 인라인 폼 | - |
| E_DISABLED_01 | 저장 버튼 | 비활성 | 내용 비어있음 |
| E_SUBMIT_01 | 저장 버튼 | API | 내용 입력됨 |
| E_SUCCESS_01 | API | toast.success | 200 |
| E_FAIL_01 | API | toast.error | 오류 |

## 5. TC 후보

| TC ID | 타입 | Given | When | Then |
|-------|------|-------|------|------|
| TC-DLG-M009-M1-01 | positive | 내용 입력 | 메모 저장 | toast.success + 폼 초기화 + 목록 갱신 |
| TC-DLG-M009-M1-02 | negative | 내용 비어있음 | 저장 클릭 | 버튼 disabled |
| TC-DLG-M009-M1-03 | exception | API 오류 | 메모 저장 | toast.error |
