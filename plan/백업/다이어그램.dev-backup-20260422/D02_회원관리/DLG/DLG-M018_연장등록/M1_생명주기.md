## 1. 목적

DLG-M018 연장 등록 다이얼로그의 열기/닫기/완료 생명주기를 명세한다.

## 2. 트리거/전제조건

- 상세내역 탭 > 연장 서브탭 > "연장하기" 버튼 클릭

## 3. 다이어그램

```mermaid
flowchart TD
  TRIGGER([상세내역 탭 연장하기 버튼 클릭])

  TRIGGER -->|E_OPEN_01| DLG_OPEN[DLG-M018 연장 등록 max-w-420px]
  DLG_OPEN --> INIT[상품명/연장일수=7/기존종료일/변경종료일 초기화]

  INIT --> USER_ACT{사용자 액션}

  USER_ACT -->|E_CANCEL_01 취소| CLOSE[모달 닫기]
  USER_ACT -->|E_SUBMIT_01 저장 필수 충족| API_CALL[POST /api/member-extensions INSERT]

  API_CALL -->|E_SUCCESS_01 200| TOAST_OK((toast.success 연장이 등록되었습니다))
  TOAST_OK -->|E_CLOSE_OK_01| POST_OK[모달 닫기 + refetch]

  API_CALL -->|E_FAIL_01 오류| TOAST_ERR((toast.error 연장 등록 실패 error))
  TOAST_ERR -->|E_STAY_01| DLG_OPEN

  classDef modal   fill:#FFF3E0,stroke:#E65100,color:#BF360C

```

| 엣지 ID | 출발 | 도착 | 조건 |
