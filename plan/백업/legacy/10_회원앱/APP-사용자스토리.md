## 2. 에픽별 사용자 스토리

### 2.1 에픽: 인증 (Authentication)

#### MUS-001: 회원 로그인
- **화면 ID**: MA-001
- **사용자 유형**: 회원
- **스토리**: 회원으로서 연락처와 비밀번호로 로그인하여 앱 서비스를 이용하고 싶다
- **우선순위**: Critical
- **사이즈**: M
- **CRM 연관**: FN-001
- **인수 조건**
  - [ ] 연락처(010-xxxx-xxxx) + 비밀번호 입력으로 로그인
  - [ ] 로그인 성공 시 홈 대시보드로 이동
  - [ ] 자동 로그인 옵션 (토큰 유지)
  - [ ] 5회 실패 시 잠금 안내 + SMS 인증 안내

#### MUS-002: 앱 연동 (최초 가입)
- **화면 ID**: MA-002
- **사용자 유형**: 신규 회원
- **스토리**: 센터에서 등록한 회원으로서 앱을 처음 설치하고 내 계정과 연동하고 싶다
- **우선순위**: Critical
- **사이즈**: L
- **CRM 연관**: FN-004
- **인수 조건**
  - [ ] 이름 + 등록된 연락처 입력
  - [ ] SMS 인증번호 확인
  - [ ] 비밀번호 설정 (앱 전용)
  - [ ] 연동 완료 시 홈으로 이동

#### MUS-003: 비밀번호 찾기
- **화면 ID**: MA-003
- **사용자 유형**: 회원
- **스토리**: 비밀번호를 잊은 회원으로서 SMS 인증으로 비밀번호를 재설정하고 싶다
- **우선순위**: High
- **사이즈**: S
- **CRM 연관**: FN-001
- **인수 조건**
  - [ ] 등록된 연락처로 SMS 인증번호 발송
  - [ ] 인증 성공 후 새 비밀번호 설정
  - [ ] 재설정 완료 후 로그인 화면으로 이동
