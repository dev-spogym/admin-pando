## 1. 개요

FitGenie CRM 회원앱은 피트니스 센터 회원이 직접 사용하는 모바일 앱입니다.
기존 CRM(관리자용 웹)의 데이터를 **회원 관점**으로 제공하며, 출석 체크인/PT 예약/이용권 조회 등 핵심 기능을 모바일에 최적화합니다.

### 1.1 목적
- 회원의 센터 이용 편의성 극대화
- 출석/예약을 앱으로 처리하여 현장 대기 시간 감소
- 이용권 만료/수업 알림으로 리텐션 향상
- 온라인 결제로 매출 채널 확장

### 1.2 대상 사용자
| 사용자 유형 | 설명 |
|---|---|
| **일반 회원** | 센터에 등록된 활성 회원 |
| **만료 회원** | 이용권 만료된 회원 (재등록 유도) |
| **신규 가입자** | 센터 방문 후 앱 연동하는 회원 |

### 1.3 CRM 연동
- 백엔드: 기존 Supabase DB 공유 (RLS로 회원 본인 데이터만 접근)
- 인증: Supabase Auth (회원 전용 role)
- API: 기존 CRM API 확장 + 회원 전용 엔드포인트 추가
