```
Next.js 15 App Router + TypeScript + Tailwind v4 기반 'use client' 컴포넌트를 작성하라.
화면: DLG-I001 수동출석등록 — 상태: 성공 처리

파일: src/components/attendance/ManualAttendanceModal.tsx + SCR-I001 연동

성공 처리 플로우:
1. API 200 OK 수신
2. toast.success(`${member.name}님 수동 출석이 등록되었습니다`)
3. onSuccess(newAttendance) 콜백 호출
   → SCR-I001에서: setAttendanceLogs(prev => [newAttendance, ...prev])
   → KPI 카드 갱신: setTodayCount(prev => prev + 1)
4. onClose() → 모달 닫힘 (open = false)

[SCR-I001 연동 — 부모 컴포넌트]
const handleManualSuccess = (newAttendance: Attendance) => {
  // 출석 로그 최상단에 새 행 추가
  setAttendanceLogs(prev => [newAttendance, ...prev])
  // KPI 갱신
  setKpi(prev => ({
    ...prev,
    todayTotal: prev.todayTotal + 1,
    manual: prev.manual + 1,
  }))
}

<ManualAttendanceModal
  open={isModalOpen}
  onClose={() => setIsModalOpen(false)}
  onSuccess={handleManualSuccess}
/>

[토스트 메시지]
toast.success(`${member.name}님 수동 출석이 등록되었습니다`, {
  description: `${format(attendanceTime, 'HH:mm')} · ${channel}`,
  duration: 4000,
})
```
