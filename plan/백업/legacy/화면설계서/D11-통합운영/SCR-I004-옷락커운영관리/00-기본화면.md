## 8. 비즈니스 룰
1. 배정 시 회원 정보 + RFID 밴드 스캔 필수 (DLG-I002)
2. 회수 시 밴드 RFID 스캔 → 락커 자동 해제 + IoT 자물쇠 열림
3. 고장 토글 → 해당 락커 선택 불가, 관리자만 해제
4. 일일 자동 회수: 자정(또는 지정 시간) 모든 `in-use` → `empty` (센터 정책)
5. IoT 락커잠금 연결 끊김 → `iotConnected=false` 배지, 기능 제한
6. 만실 감지: 가용 락커 < 5 → 상단 "만실 임박" 경고
7. 성별 구역: 남/여/공용 명시, 잘못된 성별 배정 시 경고
