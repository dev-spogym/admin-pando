## 인터랙션
| 액션 | 동작 |
|---|---|
| 토스트 [바로 결제] | `/pos?memberId={newId}` 이동 (5초 내) |
| 토스트 [닫기] | 토스트 제거 (네비 진행) |
| 자동 이동 | 200ms 딜레이 후 `/members/detail?id={newId}` |
| ESC | 토스트 닫기 |

## 룰 델타
1. **자동 리다이렉트**: `router.push('/members/detail?id=' + res.id)` — 사용자 대기 없음.
2. **"바로 결제" 액션**: 등록 후 이용권 구매 유도 (상태전이도 §1.2 INACTIVE → ACTIVE 시나리오).
   ```
   toast.success('신규 회원이 등록되었습니다.', {
     action: { label: '바로 결제', onClick: () => router.push(`/pos?memberId=${newId}`) },
     duration: 5000,
   });
   ```
3. **감사 로그**: `AUDIT.MEMBER_CREATE` (actorId, memberId, branchId, channel='web').
4. **cache invalidate**: `['members']`, `['memberStats']` (SCR-M001 복귀 시 최신 목록).
5. **Draft 제거**: `localStorage.removeItem('member:new:draft')`.
6. **analytics**: `member_new_success` (duration_ms, hasImage, memberType, referralSource).
7. **환영 메시지 (🆕 미구현)**: 자동 카카오톡/SMS 발송 토글.
8. **추천인 연결 (🆕)**: 추천인 선택 → `referralMileage` 자동 적립.

## 에지
- 성공 응답 직후 네트워크 단절 → navigate 실패 시 toast + 수동 이동 링크 제공.
- 라우팅 lazy load 실패 → 재시도 또는 `<a>` fallback.
- 5초 내 "바로 결제" 미클릭 → 토스트 자동 소멸 후 상세 페이지에 도달한 상태.
- 메모리 상에 이전 폼 상태 잔존 → `form.reset()` 호출하여 청소 (🆕 권장).

## 🧩 바이브코딩 (델타)
```
onSuccess: (res) => {
  // 1. 부수 효과
  queryClient.invalidateQueries({ queryKey: ['members'] });
  queryClient.invalidateQueries({ queryKey: ['memberStats'] });
  localStorage.removeItem('member:new:draft');
  analytics.track('member_new_success', {
    durationMs: performance.now() - submitStartRef.current,
    hasImage: !!data.profileImage,
    memberType: data.memberType,
    referralSource: data.referralSource,
  });
  // 2. 감사 로그 (서버에서 기록되는 경우 생략)

  // 3. 성공 토스트 with action
  toast.success('신규 회원이 등록되었습니다.', {
    action: {
      label: '바로 결제',
      onClick: () => router.push(`/pos?memberId=${res.id}`),
    },
    duration: 5000,
  });

  // 4. SR 공지
  announce('회원 등록이 완료되었습니다.');  // aria-live region

  // 5. 라우팅 (200ms 딜레이로 SR/토스트 확인 여유)
  setTimeout(() => router.push(`/members/detail?id=${res.id}`), 200);
}

// SR live region (AppLayout에 배치)
<div role="status" aria-live="assertive" className="sr-only" id="announcer" />
```
