# FitGenie CRM 관리자 화면설계서 (상태별 · 다이어그램 기반)

> **목적**: 바이브코딩(Vibe Coding) 친화적 상태별 화면설계서 — SCR(화면)·DLG(모달) 단위로 폴더 분리, 각 상태를 독립 파일로 작성, AI 코딩 프롬프트 + 디스크립션 포함
> **버전**: 2.0 (2026-04-21)
> **원천 소스**: `docs/다이어그램/` 의 Mermaid 다이어그램 체계 (SCR 9종 F1~F9, DLG 3종 M1~M3)
> **플랫폼**: Next.js 15 App Router + TypeScript + Tailwind CSS + Supabase (데스크톱)
> **자매 문서**: `client-pando/docs/화면설계서/` (회원앱 · 모바일)
