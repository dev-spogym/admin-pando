### 변경 요소
| 요소 | 변경 |
|---|---|
| KPI 4개 | `StatCardSkeleton` (animate-pulse) |
| 바 차트 | `ChartSkeleton` (5줄 회색 바, 너비 다양) |
| 테이블 | 5행 skeleton rows |
| 월 선택 | 활성 유지 (변경 가능) |
| 엑셀 버튼 | disabled |

## 역할별 차이
- primary/owner 동일 (manager 이하 접근 불가)

## 인터랙션
- 월 선택 변경 즉시 가능 → 재로딩 트리거
- 3초 초과 시 "지점 데이터를 집계 중입니다..." 문구
- 10초 초과 → 05-에러 전이

## 비즈니스 룰 델타
- 병렬 호출: `/branch-report` + `/branch-report/summary`
- windowFocus 시 stale 캐시 있으면 skeleton 대신 stale 데이터 + 작은 spinner

## 에지 케이스
- 월 변경 중 이전 요청 취소 (AbortController)
- JWT 만료: `/login?redirect=/branch-report`
