## 메타
| 항목 | 값 |

| 다이어그램 노드 | F6 `STATE_EMPTY` |

## 상태 진입 조건
- `isLoading=false` && `branches.length === 0` (신규 테넌트)
- OR 필터/검색 적용 후 `filtered.length === 0` (서브상태 `empty-filter`)

## 역할별 차이
| 역할 | 차이 |
|---|---|
| primary/super | CTA "첫 번째 지점을 등록하여 시작하세요." |
| owner | "담당 브랜드에 지점이 없습니다." + 본사 문의 링크 |

## 인터랙션
- 3A "+ 신규 지점 등록" → DLG-092-001
- 3B "검색 초기화" → `setKeyword('')` + refetch
- 3B "필터 초기화" → keyword + statusFilter 초기화

## 비즈니스 룰 델타
- 3A: 온보딩 유도 — SCR-096 온보딩 링크 안내
- 3B: 필터 상태 유지, 초기화 액션으로 02 복귀
- 3A audit_log `EMPTY_BRANCH_VIEW` 기록

## 에지 케이스
- 3A + owner maxBranches=0: "지점 생성 권한이 없습니다" + 구독 업그레이드 링크
- 3A → 등록 완료 → 02-정상 자동 전이

## 바이브코딩 프롬프트 (델타)
```
{branches.length === 0 && !isLoading ? (
  <EmptyState
    icon={<Building2 className="h-12 w-12 text-gray-400"/>}
    title="등록된 지점이 없습니다"
    description="첫 번째 지점을 등록하여 시작하세요."
    action={canAddBranch(user.role) && (
      <Button variant="primary" onClick={() => setAddOpen(true)}>
        <Plus className="h-4 w-4 mr-1"/> 신규 지점 등록
      </Button>
    )}
    hint={<Link href="/onboarding" className="text-sm text-blue-600 hover:underline">온보딩 가이드 보기 →</Link>}
  />
) : filtered.length === 0 ? (
  <EmptyState
    icon={<Search className="h-12 w-12 text-gray-400"/>}
    title="검색 결과에 해당하는 지점이 없습니다"
    description={`"${keyword}"${statusFilter ? ` (${STATUS_LABEL[statusFilter]})` : ''}에 해당하는 지점이 없어요.`}
    action={
      <div className="flex gap-2">
        <Button variant="secondary" onClick={() => setKeyword('')}>검색 초기화</Button>
        <Button variant="secondary" onClick={resetFilters}>필터 초기화</Button>
      </div>
    }
  />
) : (<DataTable {...} />)}
```
