## 🧩 바이브코딩 프롬프트

```
모달: DLG-081-004 역할 삭제 — 역할명 타이핑 확인

<Modal open={showDeleteRoleModal}>
  <ModalContent className="max-w-md">
    <ModalHeader>
      <div className="flex items-center gap-2">
        <Trash2 className="w-5 h-5 text-red-500" />
        <ModalTitle>역할 삭제</ModalTitle>
      </div>
    </ModalHeader>
    <ModalBody className="space-y-4">
      <p className="text-sm text-gray-600">
        이 역할을 삭제하시겠습니까? 삭제 확인을 위해 역할명을 입력해 주세요.
      </p>
      <div className="px-3 py-2 bg-gray-50 rounded-lg text-sm font-mono text-gray-800">
        {targetRole.name}
      </div>
      <Input
        value={confirmText}
        onChange={e => setConfirmText(e.target.value)}
        placeholder={`"${targetRole.name}" 입력`}
        className={cn(confirmText === targetRole.name && "border-red-300")}
      />
      {confirmText && confirmText !== targetRole.name && (
        <p className="text-xs text-red-400">역할명이 일치하지 않습니다.</p>
      )}
    </ModalBody>
    <ModalFooter className="flex justify-end gap-2">
      <Button variant="outline" onClick={() => setShowDeleteRoleModal(false)}>취소</Button>
      <Button
        variant="destructive"
        disabled={confirmText !== targetRole.name}
        onClick={handleDelete}
      >삭제</Button>
    </ModalFooter>
  </ModalContent>
</Modal>

handleDelete: async () => {
  const { error } = await supabase.from('custom_roles').delete().eq('id', targetRole.id)
  if (error?.code === '23503') {
    // M3_081004_03: 직원 배정됨 → DLG-087-001로 전환
    setShowDeleteRoleModal(false)
    setShowReassignModal(true)
    return
  }
  // M3_081004_02: 성공
  onRoleDeleted(targetRole.id)
  setShowDeleteRoleModal(false)
  toast.success('역할이 삭제되었습니다.')
}
```
