## 🧩 바이브코딩 프롬프트

```
모달: DLG-087-001 역할 삭제 + 직원 재배정

<Modal open={showReassignModal}>
  <ModalContent className="max-w-lg">
    <ModalHeader>
      <div className="flex items-center gap-2">
        <UserMinus className="w-5 h-5 text-red-500" />
        <ModalTitle>역할 삭제 — 직원 재배정 필요</ModalTitle>
      </div>
    </ModalHeader>
    <ModalBody className="space-y-4">
      <p className="text-sm text-gray-600">
        이 역할에 배정된 직원 <strong>{assignedStaff.length}명</strong>을
        다른 역할로 재배정한 후 삭제할 수 있습니다.
      </p>
      <!-- 배정 직원 목록 (M2_087001_01) -->
      <div className="max-h-32 overflow-y-auto space-y-1">
        {assignedStaff.map(s => (
          <div key={s.id} className="flex items-center gap-2 text-sm text-gray-700">
            <Avatar className="w-6 h-6" src={s.avatar_url} />
            {s.name} · {s.current_role}
          </div>
        ))}
      </div>
      <!-- 재배정 역할 Select (M2_087001_02~05) -->
      <div>
        <label className="text-sm font-medium">재배정 역할 <span className="text-red-500">*</span></label>
        <Select
          value={newRoleId}
          onValueChange={setNewRoleId}
          className="mt-1"
        >
          <option value="">역할 선택</option>
          {availableRoles
            .filter(r => r.id !== targetRole.id)  // M2_087001_04: 삭제 대상 제외
            .map(r => <option key={r.id} value={r.id}>{r.name}</option>)}
        </Select>
        {newRoleId === targetRole.id && (
          <p className="text-xs text-red-500 mt-1">삭제 대상 역할은 선택할 수 없습니다.</p>
        )}
      </div>
    </ModalBody>
    <ModalFooter className="flex justify-end gap-2">
      <Button variant="outline" onClick={() => setShowReassignModal(false)}>취소</Button>
      <Button
        variant="destructive"
        disabled={!newRoleId || newRoleId === targetRole.id}
        onClick={handleReassignAndDelete}
      >재배정 후 삭제</Button>
    </ModalFooter>
  </ModalContent>
</Modal>

handleReassignAndDelete: async () => {
  // M3_087001_01: PATCH /staff/bulk-role
  const { error: reassignError } = await supabase.rpc('bulk_reassign_role', {
    from_role_id: targetRole.id, to_role_id: newRoleId
  })
  if (reassignError) { toast.error('재배정 실패'); return }
  // M3_087001_02: DELETE /roles/{id}
  const { error: deleteError } = await supabase.from('custom_roles').delete().eq('id', targetRole.id)
  if (deleteError) { toast.error('역할 삭제 실패'); return }
  onRoleDeleted(targetRole.id)
  setShowReassignModal(false)
  toast.success('직원 재배정 후 역할이 삭제되었습니다.')
}
```
