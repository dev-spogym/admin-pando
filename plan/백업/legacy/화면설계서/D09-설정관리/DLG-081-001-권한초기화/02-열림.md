```
모달: DLG-081-001 권한 초기화 — 열림 (danger variant)

<Modal open={showResetModal}>
  <ModalContent className="max-w-md">
    <ModalHeader>
      <div className="flex items-center gap-2">
        <RotateCcw className="w-5 h-5 text-red-500" />
        <ModalTitle>권한 초기화</ModalTitle>
      </div>
    </ModalHeader>
    <ModalBody>
      <p className="text-sm text-gray-600">
        저장된 권한 설정으로 초기화합니다. 현재 변경사항이 모두 사라집니다.
      </p>
    </ModalBody>
    <ModalFooter className="flex justify-end gap-2">
      <Button variant="outline" onClick={() => setShowResetModal(false)}>취소</Button>
      <Button variant="destructive" onClick={handleReset}>초기화</Button>
    </ModalFooter>
  </ModalContent>
</Modal>

handleReset: () => {
  // M3_081001_01: savedPermissions로 복원
  setPermMatrix(deepClone(savedMatrix))
  setChangedCells(new Set())
  setIsDirty(false)
  setShowResetModal(false)
  // M3_081001_03: 인라인 메시지
  setInlineMsg('권한이 초기화되었습니다.')
  setTimeout(() => setInlineMsg(''), 3000)
}
```
