> 상속: [`00-기본화면.md`](./00-기본화면.md) — 이 문서는 **변경점(델타)**만 기술합니다.

## 상태 진입 조건
- 서버 `GET /settings/:branchId` 403 E403001
- 클라이언트 role이 `['superAdmin','primary','owner']`에 미포함
- super/primary가 무권한 branchId 조회 시도

## 상태 고유 인터랙션
1. **"홈으로 이동"** → `/` (역할별 대시보드)
2. **"로그아웃"** → `/logout` → `/login`
3. **관리자 문의** (mailto or 지원 페이지)
4. Tab 순서: 홈 → 로그아웃 → 문의

## 상태 고유 비즈니스 룰 델타
- 사이드바 항목 자체가 manager 이하에게 미노출 (공통.md §2.2)
- 직접 URL 입력 시에만 이 상태 도달
- `AUDIT.ACCESS_DENIED`(role, path, branchId) 자동 기록
- 404가 아닌 403 유지
- 3회 이상 접근 시도 시 보안 경고 이메일(owner/super)

## 에지 케이스
- **role 승격 직후**: JWT 미반영 → "세션 새로고침" 버튼 표시
- **초대 링크**: `?invite=...` → 수락 CTA 표시
- **뒤로가기**: 직전 페이지, fallback `/`
- **primary + 다른 브랜드 branchId**: 403
