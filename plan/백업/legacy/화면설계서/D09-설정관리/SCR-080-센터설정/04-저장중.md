## 상태 고유 인터랙션
- 모든 인터랙션 차단
- Esc 무시, 뒤로가기 beforeunload 차단
- 최대 15초 대기

## 상태 고유 비즈니스 룰 델타
- `pickDirty(values, dirtyFields)` → 변경 필드만 PATCH
- 타임아웃 15초 → abort → `05-에러` (E500001)
- Optimistic Update 사용 안 함
- 성공 시: form.reset(updatedData) + saveSuccess=true(3s) + queryCache 갱신 + AUDIT.SETTINGS_UPDATE
- 실패 시 폼 값 보존

## 에지 케이스
- **로고 업로드 병행**: 로고 POST 먼저 → PATCH
- **네트워크 불안정**: idempotent PATCH(updatedAt 기반 충돌 감지)
- **다른 사용자 동시 저장**: 409 E900007 → 재로드 옵션
- **세션 만료**: 401 → draft sessionStorage → /login

## 🧩 바이브코딩 프롬프트 (델타)

```
상속: 00-기본화면.md §14 프롬프트 전체
상태: settings-saving (저장 중)

이 상태에서만 적용할 변경점:
- isSaving = mutation.isPending = true
- 모든 필드 readOnly/disabled

저장 버튼:
<button disabled aria-busy="true"
        className={cn(primaryBtn, 'animate-pulse')}>
  <Loader2 className="size-4 animate-spin inline mr-1.5" />
  저장 중...
</button>

Fieldset 전체 disabled:
<fieldset disabled={isSaving} className={isSaving ? 'opacity-70 pointer-events-none' : ''}>
  {/* 탭 콘텐츠 */}
</fieldset>

mutation:
const onSave = form.handleSubmit(async (values) => {
  const diff = pickDirty(values, form.formState.dirtyFields);
  if (logoFile) { const { url } = await uploadLogo(branchId, logoFile); diff.logoUrl = url; }
  const ctrl = new AbortController();
  const timer = setTimeout(() => ctrl.abort(), 15_000);
  try {
    const updated = await patchSettings(branchId, diff, { signal: ctrl.signal });
    form.reset(updated);
    queryClient.setQueryData(['settings', branchId], updated);
    setSaveSuccess(true);
    setTimeout(() => setSaveSuccess(false), 3000);
    auditLog('SETTINGS_UPDATE', { branchId, changedKeys: Object.keys(diff) });
    if (pendingTab) { setActiveTab(pendingTab); setPendingTab(null); }
  } catch (err) {
    setErrorCode(mapError(err));
  } finally {
    clearTimeout(timer);
  }
});

저장 중 이탈 방지:
useBeforeUnload(isSaving, '저장 중입니다. 잠시만 기다려주세요.');

세션만료 draft 저장:
axios.interceptors.response.use(null, (err) => {
  if (err.response?.status === 401 && isSaving) {
    sessionStorage.setItem('settings_draft', JSON.stringify(form.getValues()));
  }
  return Promise.reject(err);
});
```
