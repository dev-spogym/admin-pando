## 상태 고유 인터랙션
1. **온라인 복귀** → invalidateQueries → 01-로딩 → 02-정상
2. **필드 수정 허용** → 로컬 dirty, 저장 차단
3. **저장 시도**(단축키) → 토스트 "오프라인입니다"
4. **수동 재연결** → /health ping → 성공 시 refetch
5. **탭 전환** → 허용(캐시)

## 상태 고유 비즈니스 룰 델타
- 캐시된 React Query 데이터(stale) 계속 표시
- dirty + offline → sessionStorage 백업
- 복귀 시 `updatedAt` 충돌 감지 → merge 프롬프트
- 로고 업로드/주소 검색 비활성
- beforeunload 유지(dirty 시)

## 에지 케이스
- **캡티브 포털**: onLine=true지만 API 실패 → 5초 내 3회 실패 시 오프라인 간주
- **Wi-Fi → LTE 전환**: 1초 debounce
- **오래된 캐시(30분+)**: "데이터가 오래되었습니다" 경고
- **dirty + 탭 닫기**: draft 보존 → 다음 진입 시 복원 프롬프트

```
상속: 00-기본화면.md §14 프롬프트 전체

{!isOnline && (

      <RotateCw className={cn('size-3', retrying && 'animate-spin')} />
      재연결
    </button>
  </div>
)}

온라인/오프라인 감지:
const [isOnline, setIsOnline] = useState(typeof navigator !== 'undefined' ? navigator.onLine : true);
useEffect(() => {
  const goOnline  = () => { setIsOnline(true); queryClient.invalidateQueries(['settings', branchId]); };
  const goOffline = () => setIsOnline(false);
  window.addEventListener('online',  goOnline);
  window.addEventListener('offline', goOffline);
  return () => {
    window.removeEventListener('online',  goOnline);
    window.removeEventListener('offline', goOffline);
  };
}, [branchId]);

저장 버튼:
<button disabled={!isOnline || !isDirty || isSaving}
        title={!isOnline ? '오프라인에서는 저장할 수 없습니다' : undefined}
        aria-describedby={!isOnline ? 'offline-hint' : undefined}
        className={cn(primaryBtn, 'disabled:bg-blue-400 disabled:cursor-not-allowed')}>
  저장하기
</button>
{!isOnline && <span id="offline-hint" className="sr-only">현재 오프라인이므로 저장이 비활성화되어 있습니다.</span>}

draft 저장:
useEffect(() => {
  if (!isOnline && isDirty) {
    sessionStorage.setItem(`settings_draft_${branchId}`, JSON.stringify(form.getValues()));
  }
}, [isOnline, isDirty, form, branchId]);

복귀 프롬프트:
useEffect(() => {
  if (isOnline) {
    const draft = sessionStorage.getItem(`settings_draft_${branchId}`);
    if (draft) setShowDraftRestorePrompt(true);
  }
}, [isOnline, branchId]);

캡티브 포털:
async function pingHealth() {
  try {
    const ctrl = new AbortController();
    const t = setTimeout(() => ctrl.abort(), 3000);
    await fetch('/api/health', { signal: ctrl.signal, cache: 'no-store' });
    clearTimeout(t);
    return true;
  } catch { return false; }
}
```
