## 진입 조건
- `devices.every(d => d.status === 'online')` AND `hubStatus === 'connected'`
- 모든 환경값이 임계값 이하

## 비주얼 델타
| 요소 | 상태 |
|---|---|
| 장비 카드 | 전부 🟢 "온라인" 배지 + "원격 열기" 활성 (gate only) |
| 환경 패널 | 기본 색상(회색/녹색) |

| 허브 배너 | 비표시 |

## 인터랙션
1. **원격 열기** → `POST /iot/devices/:id/test` → 토스트 "게이트가 열렸습니다(3초 후 자동 잠금)"
2. **정책 토글** → 즉시 PATCH (optimistic update)
3. **날짜 필터** → 해당 날짜 이력 재조회
4. **장비 추가** → `+ 장비 추가` → 별도 폼 다이얼로그

## 비즈니스 룰 델타
- 원격 열기 감사로그: `AUDIT.GATE_OPEN(userId, deviceId)`
- 원격 열기 쿨다운 5초 (스팸 방지)
- 정책 토글 실패 시 optimistic revert
- 이력 최신 50건 + "더 보기"로 페이지네이션

## 에지 케이스
- 원격 열기 네트워크 실패 → "게이트 응답 없음" + 재시도
- 정책 저장 실패 → 토글 revert + 토스트
- 이력 중 미등록 카드: "미등록" 배지

async function openGate(deviceId) {
  if (openCooldown.has(deviceId)) { toast.warn('5초 후 다시 시도하세요'); return; }
  openCooldown.add(deviceId);
  setTimeout(() => openCooldown.delete(deviceId), 5000);
  try {
    await fetch(`/api/iot/devices/${deviceId}/test`, { method:'POST' });
    toast.success('게이트가 열렸습니다 (3초 후 자동 잠금)');
    auditLog('GATE_OPEN', { deviceId });
  } catch {
    toast.error('게이트 응답 없음');
  }
}

정책 토글 (optimistic):
const policyMut = useMutation({
  mutationFn: (dto: Partial<Policy>) => patchPolicy(branchId, dto),
  onMutate: async (next) => {
    await queryClient.cancelQueries(['policies', branchId]);
    const prev = queryClient.getQueryData(['policies', branchId]);
    queryClient.setQueryData(['policies', branchId], { ...prev, ...next });
    return { prev };
  },
  onError: (err, next, ctx) => {
    queryClient.setQueryData(['policies', branchId], ctx.prev);
    toast.error('정책 저장 실패');
  },
});

실시간 로그:
useEffect(() => {

    .subscribe();
