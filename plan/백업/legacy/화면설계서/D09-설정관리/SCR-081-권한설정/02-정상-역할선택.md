## 인터랙션
1. **cell 클릭** → `04-편집-변경됨` 전환
2. **다른 역할 클릭** → 즉시 전환 (dirty 아님)
3. **"모두 허용"** → 현재 역할 전 cell true (primary 제외)
4. **"모두 거부"** → 전 cell false
5. **"복사"** → DLG-081-005 (커스텀 역할 대상)
6. **"삭제"** → DLG-081-004 (커스텀 역할 대상)
7. **"+ 역할 추가"** → DLG-081-003

## 비즈니스 룰 델타
- 선택 역할이 `primary`면 `03-최고관리자-선택`으로 자동 전환
- 시스템 역할(isSystem=true)은 삭제 버튼 미노출
- owner는 자신보다 낮은 역할만 복사 가능 (위계 검증)
- cell value는 `data[roleCode][menuId][action]` 우선, 없으면 `false`

## 에지 케이스
- **권한 위계 위반**: owner가 primary 복사 시도 → 서버 403
- **배정 직원 많음**: `userCount > 10`이면 "외 N명" 표시
- **cell 값 null/undefined**: false로 정규화
