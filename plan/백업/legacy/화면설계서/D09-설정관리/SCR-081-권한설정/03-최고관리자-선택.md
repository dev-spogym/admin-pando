## 인터랙션
1. **cell 클릭 시도** → 무시 (이벤트 차단)
2. **키보드 Space/Enter** → 무시
3. **다른 역할 클릭** → `02-정상-역할선택` 복귀
4. **"복사" 클릭** → DLG-081-005 (primary 권한으로 새 역할 생성 — 단 owner/super만)

## 비즈니스 룰 델타
- primary 행 `cell.value = true` 고정, state mutation 무시
- 안내 배너 항상 표시
- 감사로그 `AUDIT.PRIMARY_VIEW(userId, branchId)` 기록 (선택)
- "복사"는 super/primary 본인만 가능 (owner는 차단)

## 에지 케이스
- **키보드 탭 이동**: cell에 focus는 가능하나 토글 이벤트만 차단
- **JS 해킹으로 state 변경 시도**: 서버는 `roleCode='primary'` PATCH 거부 (E901002)
