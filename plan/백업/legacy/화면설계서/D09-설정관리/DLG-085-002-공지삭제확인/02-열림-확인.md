## 🧩 바이브코딩 프롬프트

```
모달: DLG-085-002 공지 삭제 확인

<Modal open={showDeleteNoticeModal}>
  <ModalContent className="max-w-md">
    <ModalHeader>
      <div className="flex items-center gap-2">
        <Trash2 className="w-5 h-5 text-red-500" />
        <ModalTitle>공지 삭제</ModalTitle>
      </div>
    </ModalHeader>
    <ModalBody>
      <p className="text-sm text-gray-600">
        이 공지를 삭제하시겠습니까? 삭제된 공지는 복구할 수 없습니다.
      </p>
      <div className="mt-3 px-3 py-2 bg-gray-50 rounded-lg text-sm text-gray-700 truncate">
        "{targetNotice?.title}"
      </div>
    </ModalBody>
    <ModalFooter className="flex justify-end gap-2">
      <Button variant="outline" onClick={() => setShowDeleteNoticeModal(false)}>취소</Button>
      <Button variant="destructive" onClick={handleDeleteNotice}>삭제</Button>
    </ModalFooter>
  </ModalContent>
</Modal>

handleDeleteNotice: async () => {
  const { error } = await supabase.from('notices').delete().eq('id', targetNotice.id)
  if (error) { toast.error('삭제 실패'); return }
  onNoticeDeleted(targetNotice.id)
  setShowDeleteNoticeModal(false)
  toast.success('공지가 삭제되었습니다.')
}
```
