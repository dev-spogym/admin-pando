## 상태 진입 조건
- SCR-052 `[+ 신규 등록]` 버튼 클릭
- `canCreateRfid(role)` 통과 (superAdmin/primary/owner/manager/front 허용)
- `useSessionStore.open === false`

## 상태 고유 비주얼 델타
- 제목: `"RFID 카드 등록"` (Wifi 아이콘)
- `cardNo` 입력 필드: 비어있음, 첫 포커스 자동 할당
- 스캔 버튼 노출(`mode==='create'` 전용), 기본 idle 스타일
- 수정 모드 전용 필드 **비노출**: `상태(status) Select` 렌더 안 함
- `lockerNo`: 비어있음 (optional placeholder `예: A-102`)
- 저장 버튼 상태: `!isValid` 면 `bg-blue-300 disabled`

## 상태 고유 인터랙션
1. **오픈 직후**: `cardNo` 필드 `autoFocus`
2. **스캔 버튼 클릭**: `scanning=true` → 800ms 펄싱 → `RF-XXXXXXXX` 자동 채움, 포커스 → 사용자 유형
3. **Enter 키**: `cardNo` 포커스 상태에서 Enter → 폼 submit (검증 트리거)
4. **유형 전환**: `member` ↔ `staff` RadioCard 클릭 시 반대편 ID 초기화
5. **AutoComplete 오픈**: Down arrow 또는 타이핑 시 회원/직원 후보 패널 표시
6. **ESC / 배경 / X**: 변경 없으면 즉시 닫기, `isDirty=true` 면 DLG-002 이탈 경고 선행

## 역할별 차이
- `front`: 사용자 유형 `직원` 라디오는 **disabled + helper "직원 카드는 관리자에게 요청"**
- `superAdmin/primary`: 헤더에 지점 라벨 `[{branchName}]` 부기

## 상태 고유 비즈니스 룰 델타
- `mode='create'` 고정
- `branchId` = `useAuthStore.currentBranchId` 서버 전송
- `cardNo` blur 시 사전 중복 체크: `GET /rfid?cardNo=...&branchId=...` — 충돌이면 인라인 에러 선표시(저장 전 방지)
- 초기값: `{ cardNo:'', userType:'member', memberId:null, staffId:null, lockerNo:null, status:'active' }`

## 에지 케이스
- **스캔 중복**: 스캔으로 자동 생성된 번호가 이미 존재 → blur 트리거 중복 체크로 인라인 에러 → 사용자 재스캔 유도
- **스캔 중 ESC**: 펄싱 상태에서 ESC → 스캔 취소(`scanning=false`) + 다이얼로그 닫기
- **외부 USB 리더기**: 물리 리더기가 `cardNo` 필드에 자동 입력할 수 있음 → `onKeyDown` 에서 `Enter` 수신 시 자동 blur → 중복 체크

```
오픈 시 초기화:
useEffect(() => {
  if (isOpen && mode === 'create') {
    reset({ cardNo:'', userType:'member', memberId:null, staffId:null, lockerNo:null, status:'active' });
    setTimeout(() => cardNoRef.current?.focus(), 0);
  }
}, [isOpen, mode]);

중복 사전 체크:
const debouncedCheck = useDebouncedCallback(async (cardNo) => {
  if (!/^RF-\d{8}$/.test(cardNo)) return;
  const { data } = await supabase.from('rfid_cards').select('id').eq('cardNo', cardNo).eq('branchId', currentBranchId).maybeSingle();
  if (data) setError('cardNo', { message: '이미 등록된 카드번호입니다' });
}, 400);

front 역할 분기:
const disableStaff = role === 'front';
<RadioCard value="staff" disabled={disableStaff} label="직원" helper={disableStaff ? '직원 카드는 관리자에게 요청' : undefined} />
```
