## 비주얼 델타

마스터 §2 와이어프레임 그대로. 실제 데이터 값 렌더.

| 변경 요소 | 기본(마스터) → 정상 |
|---|---|
| StatCard 값 | `{total}`, `{inUseCount}`, `{expiringCount}`, `{brokenCount}` 실제 숫자 |
| TabNav | 활성 `activeZone` 강조, 비활성 탭 hover 가능 |
| SearchBar | `value={searchQuery}` + `onChange` 작동 |
| LockerGrid | `filteredLockers` 실제 셀 렌더. `mapStatus`로 cell.{available/in_use/expiring/broken} 적용 |
| 각 셀 | 번호(13px bold) + 회원명(11px truncate) + D-Day(10px 색상별) |
| 호버 오버레이 | `opacity-0 group-hover:opacity-100` transition. 상태별 [기록][이동][회수 or 배정][고장/복구] |

## 상태 고유 인터랙션

- **셀 클릭(단일 모드)**: `setSelectedLockerId(id)` → `LockerDetailModal` 또는 `ActionSheet` 호출
- **셀 호버**: `HoverActions` 오버레이 표시 (데스크톱). 모바일은 탭 시 ActionSheet
- **셀 액션 클릭**: `openAction(action, locker)` → 해당 모달 (DLG-050-001~005)
- **탭 전환**: `setActiveZone(zone)` → queryKey 변경 → 새 데이터 fetch
- **검색 입력**: debounce 300ms → `filteredLockers` 재계산
- **새로고침**: `refetchAll()` + 60초 쿨다운
- **일괄 배정 버튼**: `bulkMode=true, bulkAssignOpen=true` → DLG-050-006
- **만료임박 일괄 해제**: `bulkMode=true` → expiring 셀 선택 → DLG-050-007
- **엑셀 다운로드**: 현재 필터 결과 → `exportToExcel()` + toast

## 비즈니스 룰 델타

- 매 60초 IoT 상태 폴링 → 오프라인 락커 배지 갱신
- 락커 만료일 경과 감지 시 자동으로 `status=EXPIRED` 전환 대신, UI는 `broken` 아닌 `in_use`로 유지하되 D-Day 음수 표시
- 일괄 선택 중 다른 탭 이동: 선택 초기화 + 확인 다이얼로그

## 에지 케이스

| 상황 | 처리 |
|---|---|
| 사용중 락커에 [배정] 클릭 시도 (없음) | HoverActions에서 조건부 숨김 |
| 빈 락커에 [회수] 클릭 시도 (없음) | HoverActions에서 조건부 숨김 |
| 검색 결과 0건 | 그리드 영역에 "검색 결과가 없습니다" + Search 아이콘 |
| 락커 500개 이상 (대용량) | Virtualized grid (react-window) |
| iotOnline=false 락커 | 우상단 `WifiOff` 배지, 액션은 가능하되 물리 확인 경고 toast |
| 선택한 회원이 이미 다른 락커 사용 | 배정 시도 시 409 `E409600` → 토스트 |

## 바이브코딩 프롬프트 (델타)

```
━━ 델타: 02-정상 상태 ━━
조건: isSuccess && lockers.length > 0
렌더:
- StatCardGrid: 실제 값
- LockerGrid:
  filteredLockers.map(l => (
    <LockerCell key={l.id} locker={l}
      selected={bulkMode && selectedBulkIds.has(l.id)}
      onClick={()=>handleCellClick(l)}
      overlay={<HoverActions locker={l} onAction={openAction}/>}/>
  ))
- 호버 오버레이: group-hover:opacity-100
- 셀 내부:
  <div className="relative group" style={cellStyle[l.status]}>
    <span className="text-[13px] font-bold">{l.number}</span>
    {l.memberName && <span className="text-[11px] truncate max-w-[60px]">{l.memberName}</span>}
    {dday !== null && <span className={ddayClass(dday)}>{ddayLabel}</span>}
    {!l.iotOnline && <WifiOff className="absolute top-1 right-1 w-3 h-3 text-state-error"/>}
    <div className="absolute inset-0 opacity-0 group-hover:opacity-100 ...">
      <HoverActions locker={l} onAction={openAction}/>
    </div>
  </div>
역할별:
- front: 수정 액션 버튼 aria-disabled, toast("읽기 전용 계정입니다")
- fc/trainer: 이 상태 도달 전에 /forbidden 리다이렉트
```
