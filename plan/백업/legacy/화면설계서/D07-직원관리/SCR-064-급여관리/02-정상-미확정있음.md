## 📝 디스크립션

### 사용 시점 (Trigger)
- `payrollList.length > 0 && payrollList.some(p => p.status === 'PENDING')`

### 필수 데이터
| 테이블 | 조건 |
|--------|------|
| `payroll` (가상) | `year_month = selectedMonth`, center_id |
| JOIN `staff` | name, department, position |

### 인터랙션 (User Actions)
1. MonthPicker 변경 → 해당 월 재fetch
2. "급여 명세서" → `/payroll/statements`
3. "급여 확정" → DLG-064-002 (미확정 인원 요약 표시)
4. 행 클릭 → DLG-064-001 (급여 상세/편집)
5. 상세/편집 버튼 클릭 → DLG-064-001

### 비즈니스 룰
- `hasPermission('payroll:confirm')`: primary / owner만 급여 확정 가능
  → manager는 "급여 확정" 버튼 hidden 또는 disabled
- CONFIRMED 급여는 DLG-064-001에서 수정 불가 (409 응답)
- 급여 확정 완료 후 자동 명세서 생성 트리거

### 에지 케이스
- 전월 미확정 급여 존재: 경고 배너 표시 ("이전 달 미확정 급여가 있습니다.")
- 직원 퇴사 후 해당 월 급여: 표시하되 "(퇴사)" 표기

### 접근성 (A11y)
- "급여 확정" 버튼 `aria-label="이번 달 미확정 급여 {N}명 일괄 확정"`
- DataTable 행 `role="row"` + `aria-selected`

### 연결 화면
- 이전: `01-로딩`
- 다음: DLG-064-001 / DLG-064-002 / `04-전원확정`
