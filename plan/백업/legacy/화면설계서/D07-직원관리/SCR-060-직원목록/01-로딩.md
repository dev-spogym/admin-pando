## 진입 조건
- `/staff` 마운트 직후 `isLoadingData === true`
- React Query: `useStaffList.isLoading === true`
- branch 전환 직후(primary/superAdmin) 재요청 중

## 비주얼 델타

```
┌──────────────────────────────────────────────────────────┐
│ PageHeader [+ 직원 등록]                                 │
├──────────────────────────────────────────────────────────┤
│ StatCardGrid                                              │
│ [■■■■■] [■■■■■] [■■■■■] [■■■■■]   ← 4개 Skeleton        │
├──────────────────────────────────────────────────────────┤
│ SearchFilter (활성, 입력 허용하되 필터 대기)             │
├──────────────────────────────────────────────────────────┤
│ 액션 버튼 바 (전부 disabled)                             │
├──────────────────────────────────────────────────────────┤
│ DataTable                                                 │
│   헤더: 실제 표시                                         │
│   rows: 5~10 skeleton rows (bg-gray-100 animate-pulse)    │
│   또는 emptyMessage="데이터를 불러오는 중..."             │
└──────────────────────────────────────────────────────────┘
```

| 변경 요소 | From → To |
|---|---|
| StatCard value | `text-3xl ...` → `<div className="h-8 w-16 bg-gray-200 rounded animate-pulse"/>` |
| DataTable body | 실제 rows → `emptyMessage="데이터를 불러오는 중..."` 또는 8 skeleton row |
| 액션 버튼 | 정상 → 전부 disabled + `aria-busy="true"` |
| 검색/필터 | 활성 (미리 타이핑 가능) |
| 페이지네이션 | 숨김 |

## 역할별 차이
- 모든 역할 동일(role-invariant). 완료 직후 권한별 뷰로 전환.
- primary/superAdmin: BranchSwitcher는 활성 상태로 먼저 렌더.

## 고유 인터랙션
- 취소 불가(자동 진행). 10s 경과 시 보조 텍스트 "계속 불러오는 중..." (선택).
- 키보드 포커스: 검색창에 자동 이동.

## 비즈니스 룰 델타
1. `isLoadingData === true` 동안 변경되는 로컬 state는 즉시 반영(쿼리 재발사 방지).
2. 10초 이상 로딩 시 `toast.info("잠시만 기다려주세요...")` (선택).
3. 스켈레톤 수는 이전 성공 응답 length를 cache에서 참조하거나 기본 8개.

## 에지 케이스
- **Tab 복귀**: `refetchOnWindowFocus` 시 기존 데이터 유지 + `isRefetching` 플래그.
- **오프라인 진입**: 로딩 1프레임 후 `06-오프라인` 전이.
- **HMR 무한 루프** (dev 전용): `refetchOnMount:'always'` 조합 시 주의.
