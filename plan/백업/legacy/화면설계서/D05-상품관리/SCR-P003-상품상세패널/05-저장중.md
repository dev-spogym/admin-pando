저장 핸들러:
  setSaving(true)
  try {
    if (mode === 'new' || mode === 'copy') {
      await supabase.from('products').insert(formData)
    } else {
      await supabase.from('products').update(formData).eq('id', product.id)
    }
    toast.success(mode === 'edit' ? '상품이 수정되었습니다.' : '상품이 등록되었습니다.')
    onClose()
    onRefresh()
  } catch {
    toast.error('저장에 실패했습니다.')
  } finally {
    setSaving(false)
  }

saving=true 동안:
- 저장 버튼: disabled, 텍스트 "저장 중..."
- 취소 버튼: disabled
- 폼 필드: pointer-events-none
