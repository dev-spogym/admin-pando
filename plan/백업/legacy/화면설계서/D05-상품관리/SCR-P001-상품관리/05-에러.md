## 상태 고유 인터랙션
- **[다시 시도]** → `refetch()` + `01-로딩` 전환.
- **[대시보드로 이동]** → `moveToPage('/')`.
- 자동 재시도: React Query `retry: 2, retryDelay: exponential(1s,4s)`.
- `online` 이벤트 감지 시 자동 `refetch()`.

## 비즈니스 룰 델타
- **errorCode 매핑**:
  - 500 → E500001, "서버 오류가 발생했습니다."
  - 503 → E503001, "서비스 점검 중입니다. 잠시 후 다시 시도해주세요." + 재시도 버튼 비활성
  - 401 → 전역 인터셉터가 `/login` 으로 이동(이 상태 진입 X)
  - 403 → `/forbidden` 전역 리다이렉트
  - 네트워크 단절 → `NETWORK` 코드, 오프라인 variant
  - 타임아웃 → `TIMEOUT`, 일반 에러 UI
- **감사 로그**: 실패 시 `AUDIT.PRODUCT_LIST_FETCH_ERR`(errorCode 포함).
- **부분 실패 분리**: products만 실패면 이 상태, sales/groups만 실패면 `02-정상`로 가되 인라인 경고.

## 에지 케이스
- 여러 번 재시도 실패(3회) → "다시 시도" 버튼 disabled + 5분 후 쿨다운 해제.
- 오프라인 + 캐시 있음 → "캐시된 데이터를 표시 중" 배너 + 본문은 이전 렌더 유지(soft error).
- RLS 위반(403) → 이 화면 대신 `/forbidden` 전역 라우팅.

## 바이브코딩 프롬프트 (델타)

```
상태: SCR-P001 / 05-에러

변경:
- 테이블 자리에 ErrorState 컴포넌트:
  <div role="alert" aria-live="assertive"
       className="bg-rose-50 border border-rose-200 rounded-xl p-8 text-center">
    {offline
      ? <WifiOff size={48} className="text-gray-500 mx-auto"/>
      : errorCode==='E503001'
        ? <Wrench size={48} className="text-amber-500 mx-auto"/>
        : <AlertCircle size={48} className="text-rose-400 mx-auto"/>}
    <h2 className="mt-4 text-base font-semibold text-gray-900">
      {offline ? '네트워크 연결을 확인해주세요.' :
       errorCode==='E503001' ? '서비스 점검 중입니다.' :
       '상품 목록을 불러오지 못했습니다.'}
    </h2>
    <p className="mt-2 text-xs text-gray-500">{errorCode}</p>
    <p className="mt-1 text-sm text-gray-600">
      {errorCode==='E503001'
        ? '점검 완료 후 자동으로 복구됩니다.'
        : '서버와의 연결에 문제가 발생했습니다. 잠시 후 다시 시도해주세요.'}
    </p>
    <div className="mt-4 flex justify-center gap-2">
      <Button variant="primary" onClick={refetch}
              disabled={retryCount>=3 || errorCode==='E503001'}
              icon={<RefreshCw size={14}/>}>다시 시도</Button>
      <Button variant="outline" onClick={()=>moveToPage('/')}>대시보드로 이동</Button>
    </div>
    {(role==='superAdmin'||role==='primary') && (
      <button className="mt-3 text-xs text-gray-500 underline"
              onClick={openErrorDetails}>상세 로그 보기</button>
    )}
  </div>

- PageHeader actions: 버튼 전부 disabled
- window.addEventListener('online', refetch) 등록
- 재시도 3회 실패 시 쿨다운 5분
```
