## 역할별 차이
- super/primary/owner: "첫 상품 등록하기" 링크 노출.
- owner(소속 브랜드 모두 empty) → "다른 지점에서 복사" 링크 추가 노출 가능 (DLG-P005-상품가져오기 트리거).
- manager는 진입 불가 (`04-빈상태-읽기전용` 으로 분기).
- 다른 역할 → 이 상태로 오지 않음.

## 상태 고유 인터랙션
- **[+ 첫 상품 등록하기]** 클릭 → `06-패널열림`(신규 모드).
- **[+ 상품 등록]** (PageHeader) 클릭 → 동일.
- **분류 관리 탭** 전환 가능 → 분류도 빈 상태면 별도 빈 안내.
- 검색 입력은 허용되지만 결과 없음 표시.

## 비즈니스 룰 델타
- 초기 구축 지점에서 자주 발생 → 온보딩 UX 단서.
- 첫 상품 등록 완료 시 전역 `products-new-onboarding-done` 플래그 세팅(옵션).
- 전지점 배포 버튼 표시는 유지하나 배포할 원본이 없으므로 클릭 시 "배포할 상품이 없습니다." 토스트.

## 에지 케이스
- 분류는 있는데 상품이 없는 지점 → 분류 탭 정상 렌더, 상품 탭만 빈.
- 상품 생성 직후 React Query refetch 지연 → optimistic insert로 `02-정상`으로 빠른 전환.
- 필터/검색이 켜진 채 진입(딥링크) → 이 상태와 구별되는 "결과 없음" 메시지 (그러나 실제 상품이 0이면 여전히 이 상태).
