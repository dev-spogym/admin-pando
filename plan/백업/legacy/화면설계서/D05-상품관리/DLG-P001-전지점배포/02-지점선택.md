## 📝 디스크립션

### 사용 시점 (Trigger)
- 지점 목록 로드 완료

### 인터랙션 (User Actions)
1. 지점 체크박스 선택/해제
2. 전체 선택/해제
3. "배포 확인" (선택 1개 이상) → `03-배포결과`
4. "취소"/ESC/X → 모달 닫기

### 비즈니스 룰
- 선택 0개: "배포 확인" disabled + 인라인 에러
- 전체 선택: indeterminate 중간 상태 지원

### 연결 화면
- 배포 확인: `03-배포결과`
