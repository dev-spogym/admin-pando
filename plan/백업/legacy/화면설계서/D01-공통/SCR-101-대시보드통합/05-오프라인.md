## 상태 진입 조건
- `navigator.onLine === false`
- API fetch → `TypeError: Failed to fetch` 반복
- `offline` 이벤트 수신

## 상태 고유 인터랙션
- 모든 network 기반 액션 차단 (사이드바 네비는 클라이언트 라우팅만 허용)
- 오프라인 상태에서 위젯 항목 클릭 → 캐시된 상세 페이지 시도 (성공 시 캐시 표시, 실패 시 토스트)
- 복구 시점에 자동 `refetchAll()` 실행
- 복구 시 토스트 "인터넷 연결이 복구되었습니다"

## 역할별 차이
- 없음. primary/super 는 지점 전환이 불가(캐시된 현재 지점만 사용 가능)

## 상태 고유 비즈니스 룰 델타
- **`online`/`offline` 이벤트 리스너** 를 Providers 에 등록
- **React Query 기본**: `staleTime: 60_000` + 네트워크 에러 시 캐시 fallback
- **오프라인 감지 디바운스**: 1초 이내 연속 on/off 는 무시
- **로그인 만료 인터셉트**: 오프라인 중에는 DLG-000 띄우지 않음(네트워크 문제가 우선)

## 에지 케이스
- **복구 직후 바로 401**: DLG-000 오픈(정상 플로우)
- **캐시 데이터 없음(최초 진입 + 오프라인)**: 오프라인 배너만 표시 + EmptyState 카드 "오프라인이라 데이터를 불러올 수 없습니다"
- **브라우저 오프라인 플래그 버그**: 주기적 ping(10초) 로 실제 네트워크 상태 검증(옵션)

## 🧩 바이브코딩 프롬프트 (델타)
```
상속: 00-기본화면.md §14 프롬프트 전체 적용
상태: dashboard-offline

online 훅:
export function useOnlineStatus() {
  const [online, setOnline] = useState(
    typeof window === 'undefined' ? true : navigator.onLine
  );
  useEffect(() => {
    const on = () => setOnline(true);
    const off = () => setOnline(false);
    window.addEventListener('online', on);
    window.addEventListener('offline', off);
    return () => { window.removeEventListener('online', on); window.removeEventListener('offline', off); };
  }, []);
  return online;
}

SystemStatusBar 오프라인 분기:
{!online && (
  <div role="status" aria-live="polite"
       className="flex items-center gap-2 rounded-md bg-slate-100 text-slate-700 border border-slate-200 p-3 text-sm sticky top-0 z-10">
    <WifiOff className="size-4" aria-hidden />
    <span>오프라인 상태입니다. 캐시된 데이터를 표시합니다.</span>
  </div>
)}

액션 비활성:
<RefreshButton disabled={!online || cooldown>0} title={online?undefined:'오프라인 상태입니다'} />
<BranchSwitcher disabled={!online} />

복구 토스트:
useEffect(() => {
  if (online && prevOnline === false) {
    toast.success('인터넷 연결이 복구되었습니다');
    refetchAll();
  }
}, [online]);

최초 진입 + 오프라인 + 캐시 없음:
{!online && !hasCache && (
  <EmptyState icon={WifiOff}
    title="오프라인 상태입니다"
    description="네트워크에 연결되면 대시보드를 불러옵니다." />
)}
```
