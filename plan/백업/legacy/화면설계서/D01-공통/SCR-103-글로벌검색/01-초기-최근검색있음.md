## 📝 디스크립션

### 사용 시점 (Trigger)
- 헤더 검색 아이콘 클릭 또는 Ctrl+K 단축키
- localStorage에 최근 검색 기록 1개 이상 존재

### 필수 데이터
| 항목 | 출처 | 비고 |
|------|------|------|
| 최근 검색어 | localStorage `search-history` | 최대 5개, 최신순 |

### 인터랙션 (User Actions)
1. 검색창 자동 포커스
2. 최근 검색어 클릭 → 해당 키워드 검색 실행
3. X 버튼 → 개별 검색어 삭제
4. "모두 지우기" → localStorage 초기화 → `02-초기-힌트`
5. ESC / 배경 클릭 → 오버레이 닫기

### 비즈니스 룰
- 검색어 저장: 검색 실행 시 localStorage 업데이트 (최대 5개, 중복 제거)
- 검색 범위: 회원, 수업, 직원, 거래 (역할별 필터)

### 에지 케이스
- localStorage 접근 불가 → 최근 검색 미표시 (힌트 상태로 폴백)

### 접근성 (A11y)
- 오버레이 `role="dialog"` `aria-modal="true"` `aria-label="글로벌 검색"`
- 검색 input `aria-label="검색어 입력"` autofocus
- 최근 검색 목록 `role="listbox"`

### 연결 화면
- 검색 실행: `03-검색중`
- 없음 상태: `02-초기-힌트`
