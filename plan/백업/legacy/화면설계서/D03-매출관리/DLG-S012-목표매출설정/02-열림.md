## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind v4 기반 'use client' 컴포넌트를 작성하라.
화면: DLG-S012 목표매출설정 — 상태: 열림 (신규 기획)

파일: src/app/sales-stats/page.tsx 내 SalesTargetModal

<Modal isOpen={isOpen} onClose={onClose} title="목표 매출 설정">
  <div className="p-6 space-y-4">
    {/* 연월 선택 */}
    <div className="flex gap-3">
      <div className="flex-1">
        <label className="block text-sm font-medium text-gray-700 mb-1">연도</label>
        <Select
          value={String(targetYear)}
          onChange={(v) => setTargetYear(Number(v))}
          options={[currentYear - 1, currentYear, currentYear + 1].map(y => ({
            label: `${y}년`, value: String(y)
          }))}
        />
      </div>
      <div className="flex-1">
        <label className="block text-sm font-medium text-gray-700 mb-1">월</label>
        <Select
          value={String(targetMonth)}
          onChange={(v) => setTargetMonth(Number(v))}
          options={Array.from({ length: 12 }, (_, i) => ({
            label: `${i + 1}월`, value: String(i + 1)
          }))}
        />
      </div>
    </div>

    {/* 목표 금액 */}
    <div>
      <label className="block text-sm font-medium text-gray-700 mb-1">
        목표 매출 (원) *
      </label>
      <input
        type="number"
        value={targetAmount}
        onChange={(e) => setTargetAmount(Number(e.target.value))}
        placeholder="0"
        min={0}
        step={100000}
        className="w-full border rounded-lg px-3 py-2 text-sm focus:outline-none focus:ring-2 focus:ring-blue-500"
      />
      <p className="text-xs text-gray-400 mt-1">
        {targetAmount > 0 ? formatKRW(targetAmount) : '금액을 입력하세요'}
      </p>
    </div>

    {/* 기존 목표 있으면 표시 */}
    {target && (
      <div className="bg-blue-50 rounded-lg p-3 text-sm text-blue-700">
        현재 목표: {formatKRW(target.amount)}
      </div>
    )}
  </div>

  <div className="p-4 border-t flex gap-3">
    <Button variant="outline" className="flex-1" onClick={onClose}>닫기</Button>
    <Button
      variant="primary"
      className="flex-1"
      onClick={handleSave}
      disabled={targetAmount <= 0}
    >
      저장
    </Button>
  </div>
</Modal>

const [targetYear, setTargetYear] = useState(new Date().getFullYear());
const [targetMonth, setTargetMonth] = useState(new Date().getMonth() + 1);
const [targetAmount, setTargetAmount] = useState(target?.amount ?? 0);
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- E_LOAD_OK_01: 기존 목표 조회 완료

### 필수 데이터
| 항목 | 설명 |
|------|------|
| targetYear | 목표 연도 |
| targetMonth | 목표 월 |
| targetAmount | 목표 금액 |
| target | 기존 목표 (없으면 null) |

### 인터랙션 (User Actions)
1. 연도 / 월 선택
2. 목표 금액 입력
3. [저장] → `03-처리중`
4. [닫기] / X → 닫힘

### 비즈니스 룰
- 금액 0 이하 시 [저장] 비활성화
- 기존 목표 있으면 upsert, 없으면 insert
- owner, manager만 접근 가능

### 연결 화면
- `03-처리중`
- 닫힘
