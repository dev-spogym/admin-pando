## 📝 디스크립션

### 사용 시점 (Trigger)
- E_NAV_PAY_01: DLG-S007 [납입 처리] 버튼 클릭

### 필수 데이터
| 항목 | 설명 |
|------|------|
| installment | 할부 상세 (paid_months, total_months, monthly_amount, remaining_amount) |
| payDate | 납입일 (기본값: 오늘) |
| payMethod | 납입 방법 (card/cash/transfer) |

### 인터랙션 (User Actions)
1. 납입일 변경 → 날짜 선택
2. 납입 방법 선택 → Select
3. [납입 확인] → `02-처리중` (E_CONFIRM_01)
4. [취소] / X → 닫힘

### 비즈니스 룰
- 납입 방법 미선택 시 [납입 확인] 비활성화
- 납입 회차: paid_months + 1
- 납입 후 잔여 = remaining_amount - monthly_amount

### 연결 화면
- `02-처리중` (확인 후)
- DLG-S007 (취소 시)
