## 📝 디스크립션

### 사용 시점 (Trigger)
- cartItems.length > 0 (상품 카드 1개 이상 클릭)

### 필수 데이터
| 항목 | 설명 |
|------|------|
| cartItems | 로컬 state (Product + quantity + cartId) |
| buyer | 선택된 구매자 (optional, 비회원 판매 시 null) |
| totalAmount | cartItems 합계 |

### 인터랙션 (User Actions)
1. 수량 +/−: cartItems 수량 변경
2. 아이템 삭제 (Trash2): 해당 항목 제거
3. 전체 삭제: cartItems=[] → `06-빈장바구니`
4. [구매자 검색] → DLG-S002 열림
5. [결제하기] → SCR-S003 결제처리 (장바구니 데이터 전달)

### 비즈니스 룰
- 비회원 판매 허용 (buyer=null 상태로 결제 가능)
- 동일 상품 중복 추가: quantity +1
- 최대 수량: 99개 (UI 제한)

### 에지 케이스
- 결제하기 클릭 시 cartItems=[] 되면: "결제할 상품이 없습니다" toast.warning

### 접근성 (A11y)
- 수량 버튼 `aria-label="수량 증가/감소"`
- 삭제 버튼 `aria-label="{상품명} 삭제"`

### 연결 화면
- DLG-S002 구매자검색
- SCR-S003 결제처리
