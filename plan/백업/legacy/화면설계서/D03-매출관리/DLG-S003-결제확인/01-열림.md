```
const onConfirm = async () => {
  setIsProcessing(true);
  // 중복 감지: 최근 5분 내 동일 buyer + 동일 amount
  const { data: dupCheck } = await supabase
    .from('sales')
    .select('id')
    .eq('buyer_id', buyer?.id)
    .eq('sale_price', totalAmount)
    .gte('purchase_date', fiveMinutesAgo)
    .limit(1);

  if (dupCheck && dupCheck.length > 0) {
    setShowDupWarning(true); // DLG-S004 열기
    setIsProcessing(false);
    return;
  }
  processPayment();
};
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- E_OPEN_01: SCR-S003 [결제 확인] 버튼 클릭 (isValid=true)

### 필수 데이터
| 항목 | 설명 |
|------|------|
| buyer | 선택된 구매자 (null=비회원) |
| paymentMethod | 선택된 결제수단 |
| totalAmount | 장바구니 합계 |

### 인터랙션 (User Actions)
1. [결제 확인] → 중복 감지 체크 → processPayment() 또는 DLG-S004
2. [취소] → 모달 닫힘 (SCR-S003 유지)
3. X / ESC → 닫힘

### 비즈니스 룰
- 중복 감지: 동일 buyer + 동일 금액 + 5분 이내
- 중복 감지 시: DLG-S004 중첩 표시
- 비회원: 중복 감지 불가 (buyer=null)
