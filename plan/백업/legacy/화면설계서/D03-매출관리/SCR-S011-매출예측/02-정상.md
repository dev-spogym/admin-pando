## 📝 디스크립션

### 사용 시점 (Trigger)
- isLoading=false AND pastSales.length > 0 (예측 계산 완료)

### 필수 데이터
| 항목 | 테이블 | 설명 |
|------|--------|------|
| 과거 매출 | `sales` | 최근 6개월 집계 |
| 목표 매출 | `sales_targets` (가상) | 이번달 목표 |

### 인터랙션 (User Actions)
1. [목표 매출 설정] → DLG-S012 목표매출설정 모달
2. 차트 바 hover → 툴팁 (금액)
3. 목표 설정 완료 → 달성률 재계산

### 비즈니스 룰
- 예측: 최근 3개월 단순 이동평균
- 목표 미설정: 달성률 바 숨김
- 예측 > 목표: "목표 달성 예상" 초록 표시

### 에지 케이스
- 과거 데이터 1개월 미만: "예측 데이터 부족" 안내
- 목표=0 설정: 달성률 계산 불가 → 0% 표시
