## 5. 상태 고유 인터랙션

| 이벤트 | 트리거 | 결과 |
|---|---|---|
| `E_SUBMIT_01` (진입) | [등록] 클릭 | schedule/monthly_amount 계산 → supabase INSERT 호출 |
| `E_SUCCESS` | `error === null` | `03-성공` 전환 |
| `E_ERROR` | `error !== null` / `NetworkError` | `04-에러` 전환 |
| `E_TIMEOUT` | 10s 초과 | AbortController abort → `04-에러`, 메시지 "응답 지연" |
| ESC/배경/X/닫기 | 사용자 입력 | **무시(preventDefault)** |

---

## 6. 상태 고유 비즈니스 룰 델타

1. **원자적 INSERT**: schedule 배열 포함 단일 INSERT 로 처리(트랜잭션 아님). 실패 시 롤백 필요 없음.
2. **낙관적 잠금 없음**: 신규 INSERT 이므로 etag/version 불필요.
3. **중복 제출 방지**: `isProcessing` 가드 + 버튼 disabled 이중화.
4. **타임아웃 처리**: `AbortController` 10초. 타임아웃 시 `E_ERROR` 분기.
5. **네트워크 단절**: `fetch` reject → 04-에러 (메시지 "네트워크 오류").
6. **감사로그**: 요청은 `AUDIT.INSTALLMENT_CREATE.attempt`, 응답은 성공/실패 분기에서 기록.

---

## 7. 에지 케이스

| 케이스 | 처리 |
|---|---|
| 동일 유저가 [등록] 연타 | 첫 클릭에서 `isProcessing=true` → 두 번째 이후 클릭 무시 |
| 네트워크 끊김 | `04-에러` (NETWORK) + 입력값 유지 |
| 세션 만료(401) | 마스터 §10: DLG-000 오픈, 이 다이얼로그 자동 정리 |
| 서버 500 에러 | `04-에러` + 토스트 "일시적 오류, 다시 시도해주세요" |
| RLS 위반(42501) | `04-에러` → 토스트 후 **닫힘**(재시도 무의미) |
| 응답은 왔으나 파싱 실패 | `04-에러` (E500001 취급) |

## 8. 바이브코딩 프롬프트 (델타)

```
상태: DLG-S009 — 02-처리중

// 제출 로직 (01-열림에서 [등록] 클릭 시 실행)
const onSubmit = async (e: FormEvent) => {
  e.preventDefault();
  if (!canSubmit || isProcessing) return;
  setIsProcessing(true);

  const monthly_amount = Math.ceil(form.total_amount / form.total_months);
  const schedule = Array.from({length: form.total_months}, (_, i) => {
    const d = new Date(form.start_date);
    d.setMonth(d.getMonth() + i);
    return {
      month: i + 1,
      due_date: d.toISOString().split('T')[0],
      amount: monthly_amount,
      paid: false,
    };
  });

  const ctrl = new AbortController();
  const timer = setTimeout(() => ctrl.abort(), 10_000);

  const { error } = await supabase.from('installments').insert({
    branch_id: branchId,
    member_id: selectedMember!.id,
    member_name: selectedMember!.name,
    product_name: form.product_name,
    total_amount: form.total_amount,
    monthly_amount,
    total_months: form.total_months,
    paid_months: 0,
    remaining_amount: form.total_amount,
    start_date: form.start_date,
    next_due_date: form.start_date,
    status: 'ACTIVE',
    schedule,
  }, { abortSignal: ctrl.signal });
  clearTimeout(timer);

  if (error) {
    toast.error(mapInstallmentError(error));
    setIsProcessing(false);   // 04-에러: 모달 유지
    if (error.code === '42501') onClose();   // 권한 오류는 닫기
    return;
  }
  toast.success('할부가 등록되었습니다.');
  onSuccess?.();
  onClose();   // 03-성공
};

// Footer 버튼 (처리 중)
<Button type="button" variant="outline" className="flex-1"
        disabled aria-disabled>닫기</Button>
<Button type="submit" variant="primary" className="flex-1"
        disabled aria-busy="true">
  <Loader2 className="h-4 w-4 animate-spin mr-2" />등록중...
</Button>

// ESC/배경 차단
useEffect(() => {
  if (!isProcessing) return;
  const onKey = (e: KeyboardEvent) => { if (e.key === 'Escape') e.preventDefault(); };
  window.addEventListener('keydown', onKey, {capture: true});
  return () => window.removeEventListener('keydown', onKey, {capture: true});
}, [isProcessing]);
```
