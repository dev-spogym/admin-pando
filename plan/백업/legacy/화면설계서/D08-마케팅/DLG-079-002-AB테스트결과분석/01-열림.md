## 📝 디스크립션

### 사용 시점 (Trigger)
- SCR-079 테스트 행 "결과 분석" 버튼 (E_DLG079002_OPEN)
- 테스트 상태가 `진행중` 또는 `완료`일 때만 접근 가능

### 필수 데이터
- GET /ab-tests/:id/result (결과 데이터)

### 인터랙션 (User Actions)
1. 안 A / 안 B 결과 비교 확인
2. 신뢰도 95% 이상 시 승자 채택 버튼 활성화
3. "안 A 채택" 또는 "안 B 채택" 클릭

### 비즈니스 룰
- 신뢰도 95% 미만: 채택 버튼 비활성화
- 채택 후 테스트 상태 → `완료`로 변경
- 채택된 안은 실제 운영에 적용
