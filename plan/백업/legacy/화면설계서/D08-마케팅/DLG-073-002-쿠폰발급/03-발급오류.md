## 🧩 바이브코딩 프롬프트

```
발급 오류 처리 (CouponIssueModal.tsx):

410 만료 (TC-073-M1-002-01):
- toast.error("만료된 쿠폰입니다. 유효기간을 확인해주세요.")
- 모달 닫힘 (재발급 불가) + refreshCoupons()

409 소진:
- toast.error("발급 한도를 초과했습니다.")
- 모달 닫힘 + refreshCoupons()

500 서버 오류:
- toast.error("발급에 실패했습니다.")
- 모달 유지 (재시도 가능)

catch (err: any) {
  if (err.status === 410) {
    toast.error("만료된 쿠폰입니다.")
    onClose()
  } else if (err.status === 409) {
    toast.error("발급 한도를 초과했습니다.")
    onClose()
  } else {
    toast.error("발급에 실패했습니다.")
  }
  setIsIssuing(false)
}
```

### 사용 시점 (Trigger)
- 410: 만료 쿠폰 발급 시도 (TC-073-M1-002-01)
- 409: 소진 쿠폰 발급 시도
- 500: 서버 오류
