- title: "수동 마일리지 처리"

폼 필드:
1. 회원 선택 * (Combobox):
   - 검색 후 선택 (이름/전화번호)
   - 선택 시 현재 잔액 표시: "현재 잔액: {balance:,}P"
2. 금액 * (Input, number):
   - suffix: "P"
   - 차감 모드: "차감 후 잔액: {balance-amount:,}P" 실시간 표시
3. 사유 * (Textarea, rows=2):
   - placeholder="적립/차감 사유를 입력해주세요"

차감 시 잔액 부족 경고:
{type==='차감' && amount > balance && (
  <div className="bg-red-50 border border-red-200 rounded-lg p-2 mt-2">
    <p className="text-xs text-red-600">잔액이 부족합니다. (잔액: {balance:,}P)</p>
  </div>
)}

하단 버튼:
- "취소" variant=outline
- "처리" variant=primary
  disabled={amount > balance && type==='차감'}
  onClick={handleProcess}

- POST /mileage-transactions (supabase 가상)
  body: { memberId, type: 'EARN'|'DEDUCT', amount, reason, processedBy: user.id }
- 200: toast.success("마일리지가 처리되었습니다.") (TC-074-F2-02)

사용 컴포넌트:
