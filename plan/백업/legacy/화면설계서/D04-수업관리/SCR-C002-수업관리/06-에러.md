## 역할별 차이

## 상태 고유 인터랙션
1. [🔄 재시도] → `refetch()` 모든 쿼리 → `01-로딩` 전환
2. [← 뒤로] → `router.back()` 또는 `/`
3. 재시도 쿨다운 3초 (스팸 방지)
4. E401 → `/login?redirect=/lessons`
5. E403 → `/forbidden`

## 상태 고유 비즈니스 룰 델타
- 에러 코드 매핑 (마스터 §10)
- 503: warning 톤 (amber)
- Sentry: `captureException({tags:{screen:'SCR-C002',code}})`
- 부분 실패 (stats 실패 + 테이블 성공): 카드만 "—" + 테이블은 정상 (06 상태는 전체 실패 시만)

## 에지 케이스
- stats 실패 + records 성공: 카드만 "—", 테이블 정상 → 02 또는 03
- 네트워크 단절 → 전역 오프라인 배너 (06-에러 병행)

## 🧩 바이브코딩 프롬프트 (델타)
```
상속: 00-기본화면.md §14
상태: lessons-error

델타:
- 전체 에러 조건:
  const isError = lessons.isError || records.isError || stats.isError;
  const code = lessons.error?.code ?? records.error?.code ?? stats.error?.code ?? 'E500001';

- ErrorCard:
  if (isError) return (
    <div role="alert" className="bg-white rounded-xl ring-1 ring-gray-100 p-10 text-center space-y-3">
      <AlertTriangle className="size-12 mx-auto text-rose-500" />
      <h2 className="text-base font-semibold">수업 데이터를 불러오지 못했습니다</h2>
      <p className="text-xs font-mono text-gray-400">[{code}]</p>
      <p className="text-sm text-gray-600">잠시 후 다시 시도해주세요.</p>
      <div className="flex justify-center gap-2 pt-2">
        <Button variant="secondary" onClick={()=>router.back()}><ChevronLeft className="size-4" />뒤로</Button>
        <Button variant="primary" onClick={handleRetry}><RotateCw className="size-4" />재시도</Button>
      </div>
    </div>
  );

- 재시도 쿨다운:
  const [cd, setCd] = useState(0);
  const handleRetry = () => {
    if (cd>0) return; setCd(3);
    qc.invalidateQueries({queryKey:['lessons']});
    qc.invalidateQueries({queryKey:['records']});
    qc.invalidateQueries({queryKey:['lesson-stats']});
  };
  useEffect(() => {
    if (!cd) return;
    const t = setInterval(()=>setCd(c=>Math.max(0,c-1)),1000);
    return () => clearInterval(t);
  }, [cd]);
```
