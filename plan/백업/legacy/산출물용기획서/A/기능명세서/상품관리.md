# 상품관리 기능명세서

## 1. 목적

- 상품 정의와 할인 정책을 운영 기준으로 정리한다.
- 상품, 할인, 전 지점 배포의 책임 주체를 분명히 한다.

## 2. 포함 범위

- 상품 목록
- 상품 등록/수정
- 상품 분류
- 할인 설정
- 전 지점 배포

## 3. 상품 체계

| 분류 | 설명 |
| --- | --- |
| 정기 이용권 | 기간형 이용 상품 |
| PT 이용권 | 횟수형 또는 기간형 PT |
| GX 이용권 | 그룹 수업 상품 |
| 골프 이용권 | 골프 레슨/타석 상품 |
| 옵션 상품 | 락커, 운동복, 기타 부가서비스 |
| 패키지 상품 | 복수 이용권 생성 가능한 묶음 상품 |

## 4. 핵심 기능

- 상품 검색, 분류 필터, 판매 가능 여부 확인
- 상품 등록/수정
- 사용 제한 조건 설정
- 할인 정책 생성/수정
- 본사 상품의 전 지점 배포

## 5. 운영 규칙

- 상품은 판매 단위이고 실제 사용 권한은 이용권으로 생성된다.
- 본사 공통 상품은 본사가 배포한다.
- 지점 전용 상품은 지점에서만 생성/운영한다.
- 할인 정책은 상품 정의와 분리해서 관리한다.
- 판매 중지 상품은 신규 판매는 막되 기존 이력은 유지한다.

## 6. 권한 기준

| 기능 | 최소 권한 |
| --- | --- |
| 상품 조회 | staff |
| 지점 상품 등록/수정 | manager |
| 할인 정책 수정 | manager |
| 전 지점 배포 | primary 이상 |

## 7. 퍼블리싱 포인트

- 상품 목록은 가격과 사용 가능 여부보다 상품 성격이 먼저 읽혀야 한다.
- 패키지 상품은 구성 항목을 카드나 배지 구조로 명확히 보여야 한다.
- 배포 기능은 일반 저장 버튼과 시각적으로 분리한다.
