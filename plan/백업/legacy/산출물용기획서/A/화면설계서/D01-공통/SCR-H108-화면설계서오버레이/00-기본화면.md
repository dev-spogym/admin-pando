# SCR-H108 화면설계서 오버레이 — 기본화면

## 1. 화면 목적

- 내부 작업용 문서 오버레이 화면이다.

## 2. 핵심 기능

- 현재 화면 ID 표시
- 연결 문서 바로가기
- 상태/기능 요약 표시

## 3. 퍼블리싱 포인트

- 운영 사용자용 기능이 아니라 내부 검수용 기능임이 분명해야 한다.
