# SCR-H202 회원 상세 — 기본화면

## 1. 화면 목적

- 회원 한 명의 운영 상태를 종합적으로 확인하고 처리하는 화면이다.

## 2. 레이아웃

```text
ProfileHeader
QuickActionBar
SummaryCards
Tabs
```

- 기본 탭 순서
  - 회원정보
  - 이용권
  - 결제이력
  - 예약/출석
  - 상담/메모

## 3. 핵심 기능

- 홀딩 등록/해제
- 탈퇴 처리
- 결제 등록 이동
- 상담 등록
- 예약 확인

## 4. 상태 규칙

- 회원 상태와 이용권 상태를 분리 노출한다.
- 만료예정 배너는 대표 이용권 기준으로 계산한다.

## 5. 권한 차이

- `readonly`는 조회만 가능
- `staff`는 일반 수정 가능
- `manager` 이상은 홀딩, 탈퇴, 이관 가능

## 6. 퍼블리싱 포인트

- 상단 헤더만 봐도 핵심 상황이 끝나야 한다.
- 탭 수가 많아도 운영 우선 탭은 앞쪽에 고정한다.
