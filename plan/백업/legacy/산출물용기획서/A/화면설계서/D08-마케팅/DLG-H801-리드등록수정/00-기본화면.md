# DLG-H801 리드 등록수정 — 기본화면

## 1. 목적

- 리드를 빠르게 등록하거나 수정하는 다이얼로그다.

## 2. 핵심 필드

- 이름
- 연락처
- 유입경로
- 상태
- 담당 FC
- 메모

## 3. 검증 규칙

- 연락처 필수
- 기존 회원 중복 경고
- `등록완료` 선택 시 회원 전환 연결 확인

## 4. 퍼블리싱 포인트

- 필드는 최소화하고 저장 속도를 우선한다.
- 중복 경고는 일반 validation보다 강한 경고 박스로 처리한다.
