# SCR-H501 상품 목록 — 기본화면

## 1. 화면 목적

- 상품 구조와 판매 가능 상태를 관리하는 화면이다.

## 2. 레이아웃

```text
Header
FilterBar
ProductTable
DetailPanel
```

## 3. 핵심 기능

- 카테고리 필터
- 상품 검색
- 판매 가능 상태 표시
- 상세 패널 확인
- 상품 등록/수정 진입

## 4. 권한 차이

- 지점 상품 수정은 `manager`
- 전 지점 배포는 `primary` 이상

- 패키지 여부와 상품 성격이 표 안에서 바로 읽혀야 한다.
- 상세 패널은 필드 나열보다 운영 요약 중심이 낫다.
