## 1. 락커 관리 (`/locker`)

> 소스: `src/app/(facilities)/locker/page.tsx`
> 참조: SCR-050 (UI-053 ~ UI-056)

### A. 화면 개요

센터 내 모든 락커의 배정 현황을 시각적 그리드와 리스트 뷰로 관리하는 화면이다. 구역(A/B/C) 탭으로 분류하고, 락커 셀 호버 시 기록·이동·회수·배정·고장 액션을 제공한다. 일괄 배정 및 만료임박 일괄 해제 기능을 지원한다.

- 컴포넌트: `Locker` (default export)
- 라우트: `/locker`
- 레이아웃: `AppLayout` > `PageHeader` + `StatCardGrid` + 구역 TabNav + 그리드/리스트 뷰

### G. 모달/팝업

#### G-1. 락커 상세 모달 (LockerDetailModal — UI-056)

- 컴포넌트: `LockerDetailModal`
- `isOpen={!!selectedLockerId && !bulkMode}`
- 내용:
  - 락커 번호 + 구역 + 상태 배지
  - 사용중 시: 배정 회원(클릭→회원상세), 배정일, 만료일(D-Day), 이용 이력, [배정 해제] 버튼
  - 빈 상태 시: `UserPlus` 아이콘 + "현재 배정된 회원이 없습니다." + [회원 배정하기] 버튼

#### G-2. 기록 모달 (history)

- `Modal title="${number}번 락커 이용 이력" size="md"`
- 이력 테이블: 날짜 | 상태(배정/회수 배지) | 회원명
- 비밀번호/메모 편집 영역 (input + textarea + 저장 버튼)
- 이력 데이터: `supabase.from("audit_logs").select(...).eq("target_type", "locker").eq("target_id", locker.id)`

#### G-3. 이동 모달 (move)

- `Modal title="${number}번 락커 이동" size="sm"`
- 번호 입력 → `handleMove()` → `supabase.update({ number: targetNum })`
- toast: `락커가 ${targetNum}번으로 이동되었습니다.`

#### G-4. 회수 확인 (reclaim — ConfirmDialog)

- `title="락커 회수"`
- `description="${number}번 락커를 회수하시겠습니까?\n배정된 회원 정보가 초기화됩니다."`
- `variant="danger"`, `confirmLabel="회수"`

#### G-5. 고장 토글 확인 (broken — ConfirmDialog)

- 고장 처리: `title="고장 처리"`, `description="${number}번 락커를 고장 처리하시겠습니까?"`
- 고장 해제: `title="고장 해제"`, `description="${number}번 락커의 고장 상태를 해제하시겠습니까?"`

#### G-6. 개별 배정 모달 (assign)

- `Modal title="${number}번 락커 회원 배정" size="md"`
- 회원 검색 (autocomplete) + 만료일 입력
- footer: [취소] [배정]

#### G-7. 일괄 배정 모달

- `Modal title="일괄 배정" size="md"`
- 그리드에서 빈 락커 클릭 선택 → 회원 검색 + 만료일 → [{N}개 락커 배정]

#### G-8. 일괄 해제 확인 (ConfirmDialog)

- `title="만료임박 락커 일괄 해제"`
- `description="선택한 ${N}개의 락커를 일괄 해제하시겠습니까?\n해제 시 배정된 회원 정보가 모두 초기화됩니다."`
- `variant="danger"`, `confirmLabel="${N}개 일괄 해제"`

#### D-Day 계산

```typescript
function getDDay(expiryDate?: string): number | null {
  if (!expiryDate) return null;
  const today = new Date(); today.setHours(0, 0, 0, 0);
  const expiry = new Date(expiryDate); expiry.setHours(0, 0, 0, 0);
  return Math.floor((expiry.getTime() - today.getTime()) / (1000 * 60 * 60 * 24));
}

function getDDayLabel(expiryDate?: string): string | null {
  const d = getDDay(expiryDate);
  if (d === null) return null;
  if (d < 0) return `D+${Math.abs(d)}`;
  if (d === 0) return "D-0";
  return `D-${d}`;
}
```
