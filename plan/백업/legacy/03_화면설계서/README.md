### 시설 관리

- [SCR-050] 락커 현황 - 구역별 락커 그리드. 색상으로 상태 구분 (사용중/빈/만료임박)
- [SCR-051] 락커 관리 - 락커 배정/해제/일괄 관리
- [SCR-052] RFID 관리 - RFID 카드 등록/해제/이력 조회
- [SCR-053] 룸 관리 - GX룸/PT룸 등록/수정/삭제. 예약 시간대 설정

### 직원 관리

- [SCR-060] 직원 목록 - 직원 검색/필터/목록 조회
- [SCR-061] 직원 등록 - 신규 직원 등록 폼
- [SCR-062] 급여 현황 - 월별 급여 현황 테이블. 합계/평균 요약
- [SCR-063] 급여명세서 - 직원별 급여명세서 상세. PDF 다운로드

### 커뮤니케이션

- [SCR-070] 메시지 발송 - SMS/카카오톡 메시지 발송. 수신자 선택 + 본문 편집
- [SCR-071] 자동 알림 - 자동 알림 규칙 설정 (만료/생일/미출석 등)
- [SCR-072] 쿠폰 관리 - 쿠폰 생성/발급/이력 관리
- [SCR-073] 마일리지 관리 - 회원 마일리지 적립/사용 이력 조회 및 관리

### 계약 관리

- [SCR-080] 계약 마법사 - 단계별 계약 등록 위자드 (회원→상품→기간→결제→확인)
- [SCR-081] 구독 관리 - 정기 구독 상품 관리. 자동 결제 현황

### 설정

- [SCR-090] 설정 - 시스템 기본 설정. 센터 정보/영업시간/알림 등
- [SCR-091] 권한 설정 - 역할별 메뉴/기능 권한 매트릭스
- [SCR-092] 키오스크 설정 - 키오스크 디스플레이/동작 설정
- [SCR-093] IoT 설정 - 출입문/체성분 기기 연동 설정
- [SCR-094] 지점 관리 - 지점 CRUD. 멀티테넌트 관리
