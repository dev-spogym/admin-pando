# 수업관리 기능명세서

## 1. 목적

- 수업 개설, 예약, 출석, 횟수 차감, 노쇼 처리의 공통 운영 규칙을 정의한다.
- PT, GX, 골프를 하나의 운영 틀 안에서 본다.

## 2. 포함 범위

- 수업 캘린더
- 수업 관리
- 시간표 일괄 등록
- 횟수 관리
- 페널티 관리
- 일정 요청 처리

## 3. 핵심 기능

- 수업 일정 등록/수정/삭제
- 정원과 예약 현황 관리
- 대기예약 자동 승격
- 수업 완료 후 출석/서명 처리
- 횟수 차감
- 노쇼/패널티 기록

## 4. 공통 상태 기준

### 4.1 수업 상태

- 예정
- 마감
- 진행중
- 완료
- 취소
- 승인대기

### 4.2 예약 상태

- 예약완료
- 대기예약
- 취소
- 출석완료
- 노쇼

## 5. 운영 규칙

- 수업은 일정 단위, 예약은 좌석 단위, 출석은 실제 이용 단위다.
- 횟수 차감 기본값은 출석완료 또는 수업완료 후다.
- 대기예약은 자동 승격을 지원한다.
- 골프는 양방향 서명, 일반 PT/GX는 강사 서명 중심 구조를 쓴다.
- 노쇼 차감 여부는 본사 정책 템플릿 기준으로 운영한다.

## 6. 권한 기준

| 기능 | 최소 권한 |
| --- | --- |
| 수업 조회 | staff |
| 수업 등록/수정 | manager |
| 일정 승인 | manager |
| 노쇼 정책 조정 | owner 이상 |

## 7. 퍼블리싱 포인트

- 캘린더는 시간표와 정원 상태가 먼저 읽혀야 한다.
- 예약 상태와 출석 상태를 같은 라벨 체계로 섞지 않는다.
- 강사용 처리 흐름은 회원용 예약 화면보다 더 빠르게 끝나야 한다.
