# DLG-H1001 정책 세트 편집 — 기본화면

## 1. 목적

- 본사가 자동화 정책 세트를 생성하거나 수정하는 다이얼로그다.

## 2. 구성 섹션

- 기본정보
- 적용 범위
- 스텝 리스트
- 지점 수정 허용 범위

## 3. 검증 규칙

- 정책명 필수
- 스텝 최소 1개 이상
- 기준일 중복 경고

## 4. 퍼블리싱 포인트

- 스텝 편집은 표 편집보다 카드 리스트와 인라인 편집 조합이 적합하다.
