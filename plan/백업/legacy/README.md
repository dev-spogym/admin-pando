## 프로젝트 개요

**FitGenie CRM**은 피트니스 센터의 회원 관리, 매출 추적, 회원권 관리, 지점 관리 등 모든 업무를 통합하여 관리하는 클라우드 기반 솔루션입니다.

### 핵심 기능
- 회원 정보 및 이용권 관리
- 지점별 매출 및 통계 조회
- 직원 및 권한 관리
- 자동 알림 및 메시지 발송
- 설정 및 통합 관리

## 문서 관리 정책

### 문서 작성 기준
- 모든 문서는 마크다운(.md) 또는 JSON 형식으로 작성
- 한국어 작성 (기술 용어는 영문 사용)
- 이미지는 상대경로 사용: `../assets/image.png`

### 문서 수정 절차
1. 해당 문서 담당자가 수정
2. 변경 사항을 버전 이력에 기록
3. 관련 문서 동기화

### 링크 규칙
- 같은 폴더 내: `[링크명](파일명.md)`
- 다른 폴더: `[링크명](../폴더명/파일명.md)`
- 외부 링크: `[링크명](https://example.com)`

## FAQ

**Q: 새로운 기능을 추가하려면?**  
A: 05_기능명세서에 기능을 추가하고, 관련 화면(03), UI요소(04), API(06), 테스트케이스(08)를 함께 업데이트하세요.

**Q: 화면 레이아웃을 변경하려면?**  
A: 03_화면설계서에서 화면을 수정한 후, 04_UI요소와 08_QA테스트케이스를 함께 업데이트하세요.

**Q: API 응답 형식이 변경되었으면?**  
A: 06_API명세서를 수정하고, 05_기능명세서의 관련 기능도 함께 업데이트하세요.
