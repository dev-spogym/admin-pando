## 모듈 1: 통합 관제 대시보드 (Unified Control Dashboard)

**목표**: 경영진과 관리자가 센터 운영의 핵심 지표를 실시간으로 모니터링하고, 데이터 기반의 신속하고 정확한 의사결정을 내릴 수 있도록 지원합니다.

|---------|-----------|-------|---------|---------|------------|
| 실시간 매출 현황 (일/주/월, 지점별, 상품별) | 실시간 재무 대시보드 — 일별/월별 손익 현황을 실시간으로 모니터링 | 본사 경영진, 지점 매니저 | 모든 모듈의 핵심 데이터를 집약하여 시각화 | 데이터 기반의 신속한 의사결정, 문제 발생 시 즉각적인 인지 및 대응, 지점별 성과 경쟁 및 동기 부여 | 1단계: 기존 회계 데이터 마이그레이션, 2단계: 통합 시스템 구축 및 테스트, 3단계: 직원 교육 및 모니터링 |

| 지점별 핵심 성과 지표(KPI) 비교 분석 | | | | | |

**목표**: 회원 정보를 통합 관리하고, 회원과의 모든 상호작용을 기록하여 개인화된 서비스를 제공하고 고객 생애 가치(CLV)를 극대화합니다.

|---------|-----------|-------|---------|---------|------------|
| 회원 정보 통합 관리 (기본 정보, 회원권, PT/GX 내역, 상담 기록) | | 지점 매니저, 트레이너, 데스크 직원 | 영업/마케팅, 결제/정산, 예약/스케줄 | 360도 고객 뷰 확보, 개인화된 회원 응대 및 서비스 제공, 상담 누락치 및 이력 관리 | CRM 시스템 구축 및 데이터 통합, 직원 재교육 및 업무 재배치, 셀프서비스 포탈 및 챗봇 개발 |
| 회원 등급 (일반/프리미엄/VIP) 및 포인트 자동 관리 | | | | | |

| 회원별 활동 로그 (출석, 운동, 결제) | | | | | |

## 모듈 3: 영업 및 마케팅 자동화 (Sales & Marketing Automation)

**목표**: 데이터를 기반으로 잠재 고객과 기존 회원을 정교하게 타겟팅하여 자동으로 마케팅 캠페인을 실행하고, 영업 프로세스를 효율화합니다.

> **벤치마킹 사례**: Village Fitness는 Zen Planner의 자동화 기능을 활용하여 주간 상담 예약 건수를 10건에서 30건으로 증가시켰습니다. 이는 리드 관리 및 후속 조치 자동화의 중요성을 보여줍니다.

### 주요 기능

| 주요 기능 | 세부 핵심기능 | 사용자 | 연동 모듈 | 기대 효과 | 구체적 실행 계획 |
|---------|-----------|-------|---------|---------|------------|
| 리드 자동 수집 및 스코어링 (유입 경로별) | | 본사 마케팅팀, 지점 매니저 | 지능형 회원 관리(CRM) | 신규 회원 전환율 15% 향상, 재등록율 25% 향상, 마케팅 인력의 반복 업무 감소 | 마케팅 자동화 도구 도입 및 설정, 고객 세그먼테이션, 캠페인 시나리오 구축, 성과 측정 대시보드 구축 |
| 개인화 마케팅 자동화 (예: 30일 미방문 회원 대상 할인쿠폰 자동 발송) | | | | | |
| 회원권 만료 임박 회원 대상 재등록 유도 캠페인 | | | | | |
| 앱 푸시, SMS, 알림톡 등 멀티 채널 메시지 발송 | | | | | |

### I. 신규회원 유입 강화

#### ① 문의 자동 관리로 상담 누락 방지

|-----|-----------|---------|------------|
| 문의 자동 관리로 상담 누락 방지 | **통합 리드 관리 시스템** — 모든 채널의 문의를 하나의 시스템으로 통합. 자동 유입 경로 태깅 — 전단, 미디어, 전단지, 추천 등 자동 분류. 리드 스코어링 — 문의 내용과 행동 패턴 기반 우선순위 자동 배정. **자동 후속 조치** — 미응답 문의에 대한 자동 리마인드 및 대안 제시. 고객 세그먼트별 자동 응답 캠페인 실행 | | 통합 CRM 시스템 구축, 각 채널별 API 연동 및 자동화 설정, 직원 교육 및 프로세스 정착 |

> **벤치마킹**: Village Fitness 성공 사례 적용 — 자동화 시스템 도입으로 주간 상담 건수 300% 증가 (10건 → 30건). Facebook 광고 연동으로 15분 내 자동 응답 체계 구축
