## 역할별 (super/primary)
- **superAdmin**: 상세 에러(traceId, errorCode, stack snippet) 노출
- **primary**: traceId만, 상세 버튼 비노출
- 나머지는 `04-권한없음` 분기 (이 상태 도달 불가)

## 상태 고유 인터랙션
1. 재시도 → `refetchAll()` → `01-로딩`
2. "자세히" → errorCode/traceId JSON
3. E503001: 30초마다 `/api/health` 폴링
4. Esc: 배너 닫기
5. Sentry issue 링크 (super만)

## 비즈니스 룰 델타
- Retry: `retry: 2`, exponential 1s/4s/9s
- Sentry: 자동 전송 (`traceId, userRole, path`)
- 연속 3회 실패 시 서버에서 Slack 알림 발송
- E503: `Retry-After` 파싱 → 예상 복구 시간

## 에지 케이스
- 일부 지점만 503 → 서버 필터링 or 빈 배열 degrade
- 2번째 실패 시 "지속 문제, IT팀 자동 알림" 안내
- `online` 이벤트 자동 재시도

## 🧩 바이브코딩 프롬프트 (델타)
```
상속: 00-기본화면.md §12 프롬프트 전체 적용
상태: super-error

에러 수집:
const errors = { stats: statsQ.error, branches: branchQ.error, audit: auditQ.error };
const hasCoreError = !!errors.stats;
const primaryCode = extractErrorCode(errors.stats || Object.values(errors).find(Boolean));

403 선 처리:
if (primaryCode === 'E403001') return <ForbiddenScreen role={user.role} />;

상단 에러 배너:
{hasCoreError && (
  <div role="alert" aria-live="assertive"
       className={cn('flex items-start gap-3 rounded-lg border p-4 mb-6',
         ERROR_TONE[primaryCode] === 'warn'
           ? 'bg-amber-50 border-amber-200 text-amber-900'
           : 'bg-red-50 border-red-200 text-red-900')}>
    <AlertCircle className="size-5 shrink-0 mt-0.5" />
    <div className="flex-1">
      <p className="font-medium">슈퍼 대시보드 데이터를 불러오지 못했습니다 ({primaryCode})</p>
      <p className="text-sm mt-1">{ERROR_MESSAGES[primaryCode]}</p>
      {primaryCode === 'E503001' && retryAfter && (
        <p className="text-xs mt-1">예상 복구: {formatKST(new Date(Date.now()+retryAfter*1000),'HH:mm')}</p>
      )}
      {traceId && <p className="text-xs mt-1 font-mono text-gray-600">Trace: {traceId}</p>}
    </div>
    <div className="flex gap-2">
      <button onClick={refetchAll}
        className="h-8 px-3 rounded-md bg-blue-600 hover:bg-blue-700 text-white text-sm inline-flex items-center gap-1">
        <RefreshCw className={cn('size-3.5', isRefetching && 'animate-spin')} /> 재시도
      </button>
      {user.role === 'superAdmin' && (
        <button onClick={() => setShowDetails(v=>!v)}
          className="h-8 px-3 rounded-md border border-gray-300 bg-white text-sm">
          {showDetails ? '숨기기' : '자세히'}
        </button>
      )}
    </div>
  </div>
)}
{showDetails && user.role === 'superAdmin' && (
  <pre className="rounded-lg bg-gray-900 text-gray-100 text-xs p-3 mb-6 overflow-auto max-h-48">
    {JSON.stringify({ errors, traceId, userId: user.id, role: user.role }, null, 2)}
  </pre>
)}

부분 실패:
<section aria-label="통합 KPI">
  {errors.stats
    ? <div className="col-span-full"><WidgetError title="KPI" code={extractErrorCode(errors.stats)} onRetry={statsQ.refetch} /></div>
    : KPI_CARDS.map(...)}
</section>

자동 폴링:
useEffect(() => {
  if (primaryCode !== 'E503001') return;
  const id = setInterval(async () => {
    try { const r = await fetch('/api/health'); if (r.ok) refetchAll(); } catch {}
  }, 30_000);
  return () => clearInterval(id);
}, [primaryCode]);

Sentry:
useEffect(() => {
  if (!hasCoreError) return;
  Sentry.captureException(errors.stats, { tags: { page: 'super-dashboard', role: user.role } });
}, [hasCoreError]);

접근성:
- 배너 role="alert" aria-live="assertive"
- aria-busy={isRefetching}
- reduced-motion 준수
```
