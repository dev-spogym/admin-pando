> ⚠ 이 문서는 D10 본사관리 맥락의 로그인 화면으로, **공통 SCR-100 로그인(`docs/화면설계서/D01-공통/SCR-100-로그인/`)과 동일한 라우트 `/login`을 공유**한다.

## 1. 화면 목적 (D10 델타)

공통 SCR-100 로그인의 본사 맥락 확장:
- 로그인 직후 **8역할 × 지점 컨텍스트 초기화** (tenantId + branchId + isSuperAdmin)
- 역할별 자동 리다이렉트: superAdmin→`/super-dashboard`, trainer→`/calendar`, front→`/attendance`, 그 외→`/`
- 로그인 후 감사 로그(`LOGIN` 기록) + 본사 대시보드 데이터 prefetch (선택)
- 본사 맥락: primary/superAdmin은 **지점 선택 없이** 통합 뷰 진입, owner/manager 이하는 **지점 자동 매핑**

## 8. 비즈니스 룰 (D10 델타)

공통 §8 규칙 상속 +:
1. **지점 목록 로드**: 컴포넌트 마운트 시 `getBranches()` 호출 (public 엔드포인트)
2. **지점 선택 필수 검증**: isSuperAdmin이 아니면 지점 미선택 시 제출 차단
3. **슈퍼관리자 자동 감지**: 이메일 도메인 `@fitgenie.hq` 또는 입력된 아이디에서 판별(클라 UX용) — 실제 판단은 서버
4. **로그인 성공 시 감사 로그**: 서버에서 `AUDIT.LOGIN_SUCCESS` 자동 기록 (userId, ipAddress, userAgent)
5. **실패 시 감사 로그**: `AUDIT.LOGIN_FAILED` + `failCount` 누적 (5회+ 30분 잠금)
6. **로그인 후 prefetch**:
   - primary/super → SCR-091 KPI 7개 prefetch
   - owner/manager → SCR-090 KPI 5개 + 위젯 4개 prefetch
   - trainer → SCR-020 오늘 수업 prefetch
7. **tenantId 일관성**: 다른 tenant 로그인 시 localStorage 완전 초기화
8. **로그인 유지 (`remember`)**:
   - `true`: localStorage `spoGym_rememberedId` (아이디만, 비번 제외)
   - 세션 자체 유지 기간은 서버 정책 (JWT refreshToken)
9. **잠금 상태**: 30분 경과 또는 관리자 수동 해제

## 13. 🧩 바이브코딩 프롬프트 (마스터, D10 델타)

```

━━ D10 본사 맥락 추가 ━━
파일: src/app/(auth)/login/page.tsx (공통 구현 공유)
추가 컴포넌트: src/components/auth/BranchSelect.tsx

━━ 지점 선택 드롭다운 ━━
const [branches, setBranches] = useState<Branch[]>([]);
const [branchesLoading, setBranchesLoading] = useState(true);
const [selectedBranchId, setSelectedBranchId] = useState<number|null>(null);

useEffect(() => {
  setBranchesLoading(true);
  api.get('/branches/public-summary')
    .then(data => setBranches(data))
    .finally(() => setBranchesLoading(false));
}, []);

<FormField label="지점 선택" htmlFor="branch">
  <select id="branch"
    value={selectedBranchId ?? ''}
    onChange={e => setSelectedBranchId(Number(e.target.value) || null)}
    className="h-11 w-full rounded-lg border border-gray-300 bg-white px-3 text-sm">
    <option value="">{branchesLoading ? '지점 로딩 중...' : '지점을 선택하세요'}</option>
    {branches.map(b => <option key={b.id} value={b.id}>{b.name}</option>)}
  </select>
  {!isSuperAdminCandidate && !selectedBranchId && errors.branch && (
    <p className="text-sm text-red-600 mt-1">지점을 선택해주세요</p>
  )}
</FormField>

━━ 로그인 후 역할 분기 ━━
async function onSubmit(data) {
  try {
    const res = await api.post('/auth/login', {
      username: data.id,
      password: data.password,
      branchId: selectedBranchId ?? undefined,
      remember: data.remember,
    });

    const { session, user } = res.data;
    useAuthStore.getState().login(user, session.accessToken);
    if (data.remember) localStorage.setItem('spoGym_rememberedId', data.id);
    clearFailCount();

    // prefetch (옵션)
    prefetchForRole(user.role, user.isSuperAdmin);

    // 역할별 분기
    const redirect = searchParams.get('redirect') ?? redirectAfterLogin(user);
    router.push(redirect);

  } catch (e) {
    incrementFailCount();
    const remaining = MAX_FAIL_COUNT - getFailCount();
    if (remaining <= 0) {
      lockAccount(LOCK_DURATION_MS);
      setErrorCode('E403002');
    } else {
      setErrorCode(e.errorCode || 'E401001');
      setErrorMessage(`이메일 또는 비밀번호가 올바르지 않습니다. (${remaining}회 더 실패 시 계정이 잠깁니다)`);
    }
  }
}

function redirectAfterLogin(user) {
  if (user.isSuperAdmin) return '/super-dashboard';
  if (user.role === 'trainer') return '/calendar';
  if (user.role === 'front') return '/attendance';
  return '/';
}

function prefetchForRole(role, isSuperAdmin) {
  if (isSuperAdmin) {
    queryClient.prefetchQuery({ queryKey:['super-stats'], queryFn:fetchSuperStats });
  } else if (['owner','manager','fc'].includes(role)) {
    queryClient.prefetchQuery({ queryKey:['dashboard-stats'], queryFn:fetchDashboardStats });
  } else if (role === 'trainer') {
    queryClient.prefetchQuery({ queryKey:['today-lessons'], queryFn:fetchTodayLessons });
  } else if (role === 'front') {
    queryClient.prefetchQuery({ queryKey:['today-attendance'], queryFn:fetchTodayAttendance });
  }
}
```
