## 비즈니스 룰 델타
- 정렬 기본 `totalSales DESC` (매출순)
- 1~3위 메달 (`salesRank` 계산)
- 4위+ 숫자 표시
- 출석률 계산: `attendanceCount / (activeMembers * daysInMonth) * 100`, 소수 첫째 자리
- 활성 회원 0인 지점 객단가 0, 출석률 "-"
- MoM 비교 열은 요약 카드에만 표시 (이번달 vs 전월)

## 에지 케이스
- 동점 매출: 같은 순위 (두 지점 모두 🥇), 다음 순위는 건너뜀
- 월 변경 → 일부 지점 데이터 0 → 객단가/출석률 "-"
- windowFocus → stale 재검증 (5분)

## 바이브코딩 프롬프트 (델타)
```
const sortedBySales = [...stats].sort((a,b) => b.totalSales - a.totalSales);
const statsWithRank = sortedBySales.map((s, i) => ({ ...s, salesRank: i+1 }));

const displayStats = useMemo(() => [...statsWithRank].sort((a,b) => {
  const va = a[sortKey] ?? 0;
  const vb = b[sortKey] ?? 0;
  if (typeof va === 'number' && typeof vb === 'number')
    return sortAsc ? va - vb : vb - va;
  return sortAsc ? String(va).localeCompare(String(vb)) : String(vb).localeCompare(String(va));
}), [statsWithRank, sortKey, sortAsc]);

// 메달 렌더
render: (v, row) => (
  <span className="flex items-center gap-2">
    {row.salesRank === 1 && <Medal className="h-4 w-4 text-yellow-500" aria-label="1위"/>}
    {row.salesRank === 2 && <Medal className="h-4 w-4 text-gray-400" aria-label="2위"/>}
    {row.salesRank === 3 && <Medal className="h-4 w-4 text-amber-700" aria-label="3위"/>}
    {row.salesRank >= 4 && <span className="text-xs text-gray-500 tabular-nums w-4 text-center">{row.salesRank}</span>}
    <button onClick={() => router.push(`/branches?focus=${row.id}`)}
      className="font-medium text-gray-900 hover:text-blue-600">{v}</button>
  </span>
)

// 차트 바 너비
const maxSales = Math.max(...data.map(d => d.value));
data.map(d => <div style={{ width: `${(d.value/maxSales)*100}%` }} className="bg-blue-500 h-6 rounded"/>)
```
