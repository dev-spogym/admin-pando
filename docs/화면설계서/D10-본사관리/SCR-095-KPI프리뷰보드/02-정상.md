## 상태 고유 인터랙션
1. **탭 클릭** → `setActiveTab(key)` + `sessionStorage('kpi_preview_last_tab')` 저장 + 캐시 확인 후 즉시/로딩
2. **StatCard hover** → 카드 `ring-primary/30` + 툴팁(지표 설명)
3. **액션 큐 항목 클릭** → `it.href` 있으면 `moveToPage(it.href)`, 없으면 `toast.info("연결된 화면이 없습니다")`
4. **랭킹 "상세 보기"** → `moveToPage(board.moreHref)` (hq→`/branches`, branch→`/staff`, fc→`/members?filter=leads`, trainer→`/members?filter=pt`, operations→`/message/auto-alarm`)
5. **퍼널 단계 hover** → 툴팁(해당 단계 수치 + 전단계 전환율)
6. **인사이트 카드 hover** → `ring-amber-200`
7. **액션 버튼** — 자동알림/Today/매출/회원 — 기능 그대로

## 상태 고유 비즈니스 룰 델타
- 탭 전환 시 이전 데이터 표시 유지 + 새 탭 로딩 인디케이터 작게 표시(탭 라벨 옆 spinner)
- StatCard value가 숫자+단위 조합이면 `tabular-nums`
- 매출 마스킹 (fc/trainer/staff/front): 매출 관련 KPI는 value=`***`, description=`"권한 없음"`
- 퍼널 `rate` 계산: `steps[i].value / steps[0].value * 100` (단, hardcoded rate 우선)
- 재방문 시 마지막 `activeTab` 복원 (단 권한 없는 탭이면 `defaultBoardKey`로 fallback)

## 에지 케이스
- **일부 보드만 비어있음**: 빈 보드는 `03-빈상태` UI로, 다른 탭은 정상 유지
- **super가 탭 열람 중 권한 회수**: 서버 403 → 탭 자동 숨김 + 기본 탭으로 fallback + toast
- **insights 3개 미만**: 제공된 만큼만 렌더, 빈 슬롯 미표시
- **랭킹 delta가 "고위험"처럼 라벨형**: `text-rose-600` 표시(숫자형 delta는 부호 기반 색상)
- **퍼널 rate가 100% 초과(역전환)**: `max:100` 클램프 + 경고 배지 "이상 감지"
