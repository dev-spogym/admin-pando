## 인터랙션
- 스켈레톤은 클릭 불가 (`aria-hidden="true"`)
- 탭 이동은 허용되나 데이터 없이 스켈레톤 유지
- 3초 초과 시 하단 `"지점 정보를 불러오는 중..."` 마이크로카피 표시

## 비즈니스 룰 델타
- 병렬 API 3개 중 1개라도 pending 이면 전체 loading
  - `/branches`
  - `/branches/stats/summary`
  - `/member-transfers?limit=20`
- 10초 초과 시 자동 `05-에러` 전이 + toast "응답이 지연되고 있습니다"

## 에지 케이스
- 네트워크 느린 환경: 3초 시점 "불러오는 중" 문구
- JWT 만료: 전역 인터셉터 → `/login?redirect=/branches`
- stale 캐시 + 백그라운드 refetch: stale data 표시 + 우상단 작은 스피너만
