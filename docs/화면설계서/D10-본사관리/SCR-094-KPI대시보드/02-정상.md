## 비즈니스 룰 델타
- MoM 계산: `prev > 0 ? round(((curr-prev)/prev)*100) : 0`
- **게이지**: target 없으면 "목표를 설정하세요"
- **PT 완료율**: ≥85 mint / <85 peach
- **PT 노쇼율**: ≤5 mint / >5 peach
- **수업 출석률**: ≥80 mint / <80 peach
- **코호트 계산**: `ACTIVE || expiry >= monthEnd 수 / cohortTotal * 100`
- 퍼널 각 단계 전환율 표시 (idx=0 100%, 이후 `curr/prev*100`)

## 에지 케이스
- MoM prev=0 → 0% 표시 ("데이터 없음" 툴팁)
- 활성 회원 0명: 일부 비율 NaN 방지 (`|| 0` fallback)
- 코호트: 미도달 월 `null` → `"-"`
- 목표 저장 직후: 게이지 즉시 반영 (optimistic)
