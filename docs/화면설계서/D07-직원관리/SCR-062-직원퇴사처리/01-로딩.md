## 역할별 차이
- 공통. 권한 없으면 `/forbidden` (마스터 가드).

## 고유 인터랙션
- 중단 불가 (자동 진행).
- 10s 초과 시 "계속 불러오는 중..." 안내.
- Esc: 목록 복귀 (`moveToPage(974)`) — 선택.

## 비즈니스 룰 델타
1. 두 쿼리 병렬: `getStaff` + (preselectedId 있으면) `getStaffById`.
2. 재시도 1회 (React Query `retry:1`).
3. 10s 타임아웃 → `08-에러`.

## 에지 케이스
- staffList가 비어 있으면 (지점에 직원 0명) `08-에러` 대신 Step 1에서 "등록된 직원이 없습니다" 안내.
- preselectedId에 해당하는 직원이 없음 → Step 1에서 "유효하지 않은 직원 ID" + Select는 계속 탐색 가능.
- 전환 직후 quick-cancel: 페이지 이탈 → 쿼리 cancel.

## 바이브코딩 프롬프트 (델타)
```
isLoading === true:
- StepIndicator: 원을 bg-gray-100 animate-pulse 처리
- Step 1 패널: input/select skeleton
- 모든 버튼 disabled

useEffect(() => {
  Promise.all([
    getStaff({ size:100 }),
    preselectedId ? getStaffById(preselectedId) : Promise.resolve(null),
  ]).then(([listRes, one]) => {
    setStaffList(listRes.data.data);
    if (one?.data) { setSelectedStaff(one.data); setSelectedStaffId(one.data.id); }
  }).catch(() => setGlobalError('load-failed'))
    .finally(() => setIsLoading(false));
}, []);
```
