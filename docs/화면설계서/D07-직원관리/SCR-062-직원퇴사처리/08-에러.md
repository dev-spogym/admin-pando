## 고유 인터랙션
- "다시 시도" → `handleSubmit()` 재실행 → `06-제출중` 재진입.
- "목록으로 돌아가기" → 현재 진행 포기 + `moveToPage(974)`.
- 배너 닫기: 배너만 닫고 상태 유지 → 사용자 수동 재시도.

## 비즈니스 룰 델타
1. **에러 분기**:
   - `E404200`: "직원을 찾을 수 없습니다." — 목록 이동.
   - `E409200`: "이미 퇴사 처리된 직원입니다." — 목록 이동.
   - `E422200`: "담당 회원이 있어 처리할 수 없습니다." — Step 2 복귀(isFC면).
   - `E500001`: 재시도 가능.
   - `NETWORK`: 오프라인 배너 + 연결 복구 대기.
   - `E401*`: `/login?redirect=...`.
   - `E403001`: `/forbidden`.
2. **부분 실패 안내**: reassign 성공 + resign 실패 → "담당 회원은 이미 재배정되었습니다. 퇴사 등록만 다시 시도해주세요." 배너.
3. **재시도 쿨다운**: 3회 연속 실패 시 60s 쿨다운(선택).
4. **복구 상태**: 이전 입력 데이터(selectedStaff, resignDate, members 배정, scheduleAction) 모두 보존.

## 에지 케이스
- **재시도 중 성공**: `07-완료` 전이.
- **재시도 중 또 다른 에러**: 새 에러 코드로 배너 갱신.
- **세션 만료 후 재시도**: 401 → /login.
- **멤버 배정 불일치**(재배정 실패 후 members.staffId가 부분 업데이트): 서버에 재조회 권장.

## 바이브코딩 프롬프트 (델타)
```
globalError !== null:
<ErrorBanner tone="error" role="alert" aria-live="assertive"
  className="flex items-start gap-2 rounded-lg border border-red-200 bg-red-50 p-4 text-sm text-red-700">
  <AlertCircle size={18}/>
  <div className="flex-1">
    <p className="font-semibold">
      {globalError.step === 'reassign' ? '담당 회원 재배정에 실패했습니다.'
       : globalError.step === 'resign' ? '퇴사 예정 등록에 실패했습니다.'
       : '처리 중 오류가 발생했습니다.'}
    </p>
    <p className="text-xs text-red-600 mt-1">에러코드: {globalError.code ?? 'UNKNOWN'} · {globalError.message}</p>
    {globalError.step === 'resign' && isFC && (
      <p className="text-xs text-red-500 mt-1">담당 회원은 이미 재배정되었습니다. 퇴사 등록만 다시 시도해주세요.</p>
    )}
  </div>
  <div className="flex flex-col gap-1">
    <Button variant="outline" size="sm" onClick={handleSubmit}>다시 시도</Button>
    <Button variant="ghost" size="sm" onClick={()=>moveToPage(974)}>목록으로</Button>
  </div>
</ErrorBanner>

// 분기 처리 (handleSubmit catch)
if (code === 'E404200') { toast.error('직원을 찾을 수 없습니다.'); moveToPage(974); return; }
if (code === 'E409200') { toast.error('이미 퇴사 처리된 직원입니다.'); moveToPage(974); return; }
if (code === 'E422200' && isFC) { toast.error('담당 회원을 먼저 배정하세요.'); setCurrentStep(2); return; }
if (code === 'E401001') { router.push('/login?redirect=' + encodeURIComponent(pathname+search)); return; }
if (code === 'E403001') { router.push('/forbidden'); return; }
```
