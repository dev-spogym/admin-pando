## [SCR-P001] 상품 관리

### 1. 화면 목적
센터에서 판매하는 이용권, PT, GX 및 기타 상품을 관리한다. 마스터-디테일 패턴으로 목록과 상세/편집을 한 화면에서 처리. 분류 관리 탭에서 상품 카테고리를 CRUD 관리. 상품의 등록, 수정, 삭제, 비활성화, 패키지 구성, 가격 이력 추적을 통합 관리한다.

### 2. 접근 경로
- 사이드바 > 상품 > 상품 관리 (`/products`)
- 내부 pageId: `972`

### 3. 권한별 접근

| 역할 | 접근 | 제한 사항 |
|------|------|-----------|
| 슈퍼관리자 | 전체 접근 | 전 지점 배포 가능 |
| 센터장/매니저 | 전체 접근 | 상품 등록/수정/삭제 가능 |
| 트레이너 | 조회만 | 상품 등록/수정/삭제 불가. `canEditProduct=false` |
| 프론트 | 조회만 | 상품 등록/수정/삭제 불가 |

| 포인트 | pointAccrual | center | ✓(green) / ✗(red) | 패널 닫힘시만 |
| 판매 | salesCountMap | center | `${N}건` 또는 '0건' | 패널 닫힘시만 |
| 상태 | isActive | center | StatusBadge. 사용(mint, dot) / 미사용(default) | 항상 표시 |

**상품 타입 배지 스타일:**
- MEMBERSHIP -> 회원권 (bg-blue-100 text-blue-700)
- LESSON -> 수강권 (bg-green-100 text-green-700)
- RENTAL -> 대여권 (bg-orange-100 text-orange-700)
- GENERAL -> 일반 (bg-gray-100 text-gray-600)

**패키지 배지:** `📦 패키지` (bg-amber-100 text-amber-700). description JSON 파싱하여 isPackage===true인 경우 표시.

| 버튼명 | 위치 | 조건 | 동작 |

| 상품 목록 / 분류 관리 | PageHeader 토글 | 항상 | mainTab 전환 (products / groups) |

**분류 관리 탭 버튼:**

| 버튼명 | 위치 | 동작 |
|--------|------|------|
| 등록 / 수정 저장 | 분류 폼 | `createProductGroup` 또는 `updateProductGroup`. 성공: `toast.success('분류가 등록/수정되었습니다.')`, 실패: `toast.error('등록/수정에 실패했습니다.')`. 분류명 빈값: `toast.error('분류명을 입력하세요.')` |
| 취소 | 분류 폼 (수정시만) | editingGroupId=null, 폼 초기화 |
| 수정 (Edit2) | 분류 목록 행 | editingGroupId 설정 + 폼에 데이터 로드 |
| 삭제 (Trash2) | 분류 목록 행 | `window.confirm('이 분류를 삭제하시겠습니까?')` -> `deleteProductGroup`. 성공: `toast.success('분류가 삭제되었습니다.')` |

#### 3. 필드 테이블

**배포할 상품 선택 섹션:**

| 필드명 | 타입 | 필수 | 옵션 | 유효성 | 에러메시지 |
|--------|------|------|------|--------|------------|

| 필드명 | 타입 | 필수 | 옵션 | 유효성 | 에러메시지 |

## [SCR-P002] 상품 등록/수정 - 레거시 폼

### 1. 화면 목적
레슨북(클래식) 스타일의 상품 등록/수정 폼. `/products/new` (신규), `/products/edit?id=N` (수정). react-hook-form + zod 유효성 검증. 기존 레슨북 시스템과의 호환성을 위한 전통적 폼 인터페이스.

### 2. 접근 경로
- 신규: POS > "상품 등록" 버튼 (`moveToPage(997)`)
- 수정: 상품 목록에서 이동 (`/products/edit?id=N`)
- `/products/edit` 페이지는 `/products/new` 컴포넌트를 재사용 (export default from '../new/page')

### 3. 권한별 접근

| 역할 | 접근 | 제한 사항 |
|------|------|-----------|
| 슈퍼관리자 | 전체 접근 | 없음 |
| 센터장/매니저 | 전체 접근 | 없음 |
| 트레이너 | 접근 불가 | 리다이렉트 (상품 목록) |
| 프론트 | 접근 불가 | 리다이렉트 (상품 목록) |
