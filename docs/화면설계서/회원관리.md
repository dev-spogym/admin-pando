## SCR-M001. 회원 목록

### 1. 화면 목적

센터에 등록된 전체 회원을 조회하고, 필터/검색/정렬을 통해 원하는 회원을 찾아 상세 페이지로 이동하거나 일괄 작업(상태 변경, 메시지 발송, 출석 처리, 관심회원 등록)을 수행한다.

### 2. 접근 경로

| 경로 | 설명 |
|------|------|
| 사이드바 > 회원관리 > 회원 목록 | 주 진입 경로 |
| URL 직접 접근 | `/members` |
| 다른 화면에서 회원 목록 링크 | 대시보드, KPI 센터 등 |

### 3. 권한별 접근

| 역할 | 접근 | 조회 | 등록 | 수정 | 삭제 | 엑셀 다운로드 | 벌크 액션 |
|------|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| primary (최고관리자) | O | O | O | O | O | O | O |
| owner (센터장) | O | O | O | O | O | O | O |
| manager (매니저) | O | O | O | O | X | O | O |
| fc (피트니스코치) | O | O | X | X | X | X | O |
| staff (스태프) | O | O | O | O | X | X | O |
| readonly (조회전용) | X | X | X | X | X | X | X |

### 5. 통계 카드

| # | 카드명 | 데이터 규칙 | 클릭 동작 |
|---|--------|-------------|-----------|
| 1 | 전체 회원 | `stats.total` — branchId 필터 + deletedAt IS NULL 전체 카운트. 아이콘: Users(20px), variant: default | 🆕 상태 서브탭을 "전체"로 전환 |
| 2 | 활성 회원 | `stats.active` — status='ACTIVE' 카운트. 아이콘: UserCheck(20px), variant: mint | 🆕 상태 서브탭을 "활성"으로 전환 |
| 3 | 만료 예정(D-30) | `stats.expiringCount` — status='ACTIVE' AND membershipExpiry가 현재~30일 이내. 아이콘: Clock(20px), variant: peach | 🆕 만료 임박 필터 적용 |
| 4 | 이번달 만료 | `stats.expiredThisMonth` — status='EXPIRED' AND membershipExpiry가 이번달 1일~말일. 아이콘: AlertTriangle(20px), variant: default | 🆕 상태 서브탭을 "만료"로 전환 + 기간 필터 적용 |

### 6. 탭 구조

**메인 탭 (1차 분류)**

| 탭명 | 필터 조건 | 배지 |
|------|-----------|------|
| 회원 전체 | 전체 회원 목록 + 상태 서브탭 + 검색/필터 + 데이터 테이블 | - |
| 보유상품별 | membershipType 기준 그룹핑 카드 + 드롭다운 필터 + 테이블 | - |
| 이용권 목록 | 이용권 중심 테이블 (회원명, 이용권, 시작일, 만료일, D-Day, 상태) | - |

**상태 서브탭 (2차 분류 — "회원 전체" 탭 내)**

| 탭명 | 필터 조건 | 배지 |
|------|-----------|------|
| 전체 | 필터 없음 | stats.total (pill 배지) |
| 활성 | status='ACTIVE' | stats.active |
| 만료 | status='EXPIRED' | stats.expired |
| 미등록 | status='INACTIVE' | stats.inactive |
| 홀딩 | status='HOLDING' | stats.holding |
| 정지 | status='SUSPENDED' | stats.suspended |

배지 스타일: 활성 탭 — `bg-primary text-white`, 비활성 탭 — `bg-surface-tertiary text-content-secondary`

    │    └──┬───┬───┬───┘    │
    │       │   │   │        │
    │       ▼   │   ▼        │

    │  └───┬────┘ └────────┘ │
    │      │                 │
    │      ▼                 │
    │  홀딩 해제 ─────────────┘
    │
    │  ┌──────────┐    ┌───────────┐
    └──│SUSPENDED │    │ WITHDRAWN │
       └──────────┘    └───────────┘

    INACTIVE: 초기 상태 또는 soft delete 시

| ACTIVE | EXPIRED | 만료일 경과 (자동) 또는 수동 변경 |
| ACTIVE | SUSPENDED | 수동 변경 |
| ACTIVE | WITHDRAWN | 탈퇴 처리 |
| HOLDING | ACTIVE | 홀딩 해제 |
| EXPIRED | ACTIVE | 재등록 (상품 재구매) |

1. members UPDATE status='HOLDING'

### 홀딩 해제 (endHolding)
