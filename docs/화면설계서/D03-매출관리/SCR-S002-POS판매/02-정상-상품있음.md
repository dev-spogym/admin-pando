  const isOutOfStock = product.stock === 0;
  return (
    <div
      className={cn(
        "rounded-xl border p-4 cursor-pointer hover:shadow-md transition",
        isOutOfStock && "opacity-50 grayscale cursor-not-allowed"
      )}
      onClick={() => !isOutOfStock && onAdd(product)}
    >
      <p className="font-semibold text-sm">{product.name}</p>
      <p className="text-xs text-gray-500 mt-1">{product.period}</p>
      <div className="mt-3 flex justify-between items-center">
        <span className="text-sm font-bold text-blue-600">
          {formatNumber(product.cardPrice)}원
        </span>
        {isOutOfStock && (
          <span className="text-xs text-red-500 font-semibold">품절</span>
        )}
      </div>
    </div>
  );
};

## 📝 디스크립션

### 사용 시점 (Trigger)
- isLoading=false AND products.length > 0

### 필수 데이터
| 항목 | 테이블 | 조건 |
|------|--------|------|
| 상품 목록 | `products` | branch_id, is_active=true, category |

### 인터랙션 (User Actions)
1. 카테고리 탭 전환 → 해당 카테고리 상품 표시
2. 상품 카드 클릭 → 장바구니 추가 (품절 제외)
3. 검색어 입력 → 상품명 필터링 (로컬 필터)
4. 장바구니 아이템 수량 +/- → 합계 업데이트

### 비즈니스 룰
- 품절(stock=0): opacity-50 grayscale, 클릭 불가
- 장바구니 담기: 동일 상품 재클릭 → 수량 +1
- 가격 유형: 현금가/카드가 구분 (priceType 토글)

### 에지 케이스
- 검색 결과 0건: 카테고리 내 검색 빈 상태
- 상품 12개 초과: 스크롤 허용 (페이지네이션 없음)

### 접근성 (A11y)
- 상품 카드 `role="button"` + `aria-label="{상품명} 장바구니에 추가"`
- 품절 카드 `aria-disabled="true"` + `aria-label="{상품명} 품절"`

### 연결 화면
- `05-장바구니-채워짐` (상품 추가 시)
- SCR-S003 결제처리 (결제하기 클릭)
- DLG-S002 구매자검색 (구매자 선택 버튼)
