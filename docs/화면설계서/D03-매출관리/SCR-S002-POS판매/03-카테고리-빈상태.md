## 📝 디스크립션

### 사용 시점 (Trigger)
- 선택 카테고리에 active 상품이 없는 경우 (E_EMPTY_01)
- 검색어 필터 결과 0건인 경우

### 필수 데이터
| 항목 | 설명 |
|------|------|
| products | 해당 카테고리 결과 = [] |

### 인터랙션 (User Actions)
1. 다른 카테고리 탭 클릭 → 해당 카테고리 상품 로드
2. 검색어 초기화 → 전체 상품 표시

### 비즈니스 룰
- 우측 장바구니: 기존 담긴 상품 유지 (탭 전환과 무관)
- "상품 등록" 링크: 권한 있는 역할에만 표시 (manager 이상)

### 연결 화면
- 다른 탭 → `01-로딩` → `02-정상`
