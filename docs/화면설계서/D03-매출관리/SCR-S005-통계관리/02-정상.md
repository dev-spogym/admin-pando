## 📝 디스크립션

### 사용 시점 (Trigger)
- isLoading=false AND stats.length > 0

### 필수 데이터
| 항목 | 테이블 | 설명 |
|------|--------|------|
| 매출 집계 | `sales` | 월별 그룹핑 |
| 목표 매출 | `sales_targets` (가상) | 월별 목표 |

### 인터랙션 (User Actions)
1. 월 변경 → 재조회
2. 구분 변경 → 테이블 재집계
3. [엑셀 다운로드] → exportToExcel()
4. 테이블 헤더 클릭 → 정렬

### 비즈니스 룰
- 목표 달성률 = 실제 매출 / 목표 매출 × 100
- 목표 미설정 시: "목표 없음" 표시
