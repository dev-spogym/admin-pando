## 📝 디스크립션

### 사용 시점 (Trigger)
- sale 조회 결과 status가 취소가능 상태
- E_ST_CANCELABLE_01: 초록 배지, 전체취소/부분환불 버튼 활성

### 필수 데이터
| 항목 | 테이블 | 설명 |
|------|--------|------|
| 매출 건 | `sales` | saleId |
| 환불 이력 | `refunds` | sale_id 기준 |

### 인터랙션 (User Actions)
1. [전체 취소] → DLG-S013 환불처리 모달
2. [부분환불] → DLG-S015 환불요청 모달

### 비즈니스 룰
- 전체취소: 결제금액 전액 환불
- 부분환불: 환불 가능 잔여금액 내 일부 환불
- 취소 가능 기간: 결제일로부터 30일 (정책)

### 연결 화면
- DLG-S013 환불처리 (전체 취소)
- DLG-S015 환불요청 (부분환불)
