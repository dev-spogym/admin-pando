## 📝 디스크립션

### 사용 시점 (Trigger)
- 통계 집계 완료 AND 데이터 > 0건

### 필수 데이터
| 항목 | 테이블 | 설명 |
|------|--------|------|
| 매출 내역 | `sales` | 기간 내 전체 |
| 집계 결과 | 프론트 계산 | 일별/월별 그룹 |

### 인터랙션 (User Actions)
1. 기간 탭 전환 (일별/주별/월별) → 차트 데이터 변경
2. 날짜 범위 변경 → 재조회
3. 차트 바 hover → 툴팁 (금액 표시)

### 비즈니스 룰
- 전월 대비 증감: (이번달 - 지난달) / 지난달 × 100
- 음수 증감률: 빨간색 (↓), 양수: 초록색 (↑)
