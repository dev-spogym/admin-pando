```
Next.js 15 App Router + TypeScript + Tailwind v4 기반 'use client' 컴포넌트를 작성하라.
화면: DLG-S015 환불요청 — 상태: 열림 (신규 기획)

파일: src/app/payment-cancel/page.tsx 내 RefundRequestModal

<Modal isOpen={isOpen} onClose={onClose} title="환불 요청">
  <div className="p-6 space-y-4">
    {/* 매출 정보 요약 */}
    <div className="bg-gray-50 rounded-lg p-4 space-y-2">
      <div className="flex justify-between text-sm">
        <span className="text-gray-500">회원명</span>
        <span className="font-medium">{sale.member_name ?? '비회원'}</span>
      </div>
      <div className="flex justify-between text-sm">
        <span className="text-gray-500">상품명</span>
        <span className="font-medium">{sale.product_name}</span>
      </div>
      <div className="flex justify-between text-sm">
        <span className="text-gray-500">결제금액</span>
        <span className="font-bold">{formatKRW(sale.amount)}</span>
      </div>
      <div className="flex justify-between text-sm">
        <span className="text-gray-500">결제일</span>
        <span>{sale.paid_at}</span>
      </div>
    </div>

    {/* 환불 유형 */}
    <div>
      <label className="block text-sm font-medium text-gray-700 mb-2">환불 유형</label>
      <div className="flex gap-3">
        <label className="flex items-center gap-2 cursor-pointer">
          <input
            type="radio"
            name="reqType"
            value="full"
            checked={reqType === 'full'}
            onChange={() => { setReqType('full'); setReqAmount(sale.amount); }}
          />
          <span className="text-sm">전액</span>
        </label>
        <label className="flex items-center gap-2 cursor-pointer">
          <input
            type="radio"
            name="reqType"
            value="partial"
            checked={reqType === 'partial'}
            onChange={() => { setReqType('partial'); setReqAmount(0); }}
          />
          <span className="text-sm">부분</span>
        </label>
      </div>
    </div>

    {/* 부분 환불 금액 */}
    {reqType === 'partial' && (
      <div>
        <label className="block text-sm font-medium text-gray-700 mb-1">환불 요청 금액 *</label>
        <input
          type="number"
          value={reqAmount}
          onChange={(e) => setReqAmount(Number(e.target.value))}
          max={sale.amount}
          min={1}
          placeholder="0"
          className="w-full border rounded-lg px-3 py-2 text-sm focus:outline-none focus:ring-2 focus:ring-blue-500"
        />
        <p className="text-xs text-gray-400 mt-1">최대 {formatKRW(sale.amount)}</p>
      </div>
    )}

    {/* 요청 사유 */}
    <div>
      <label className="block text-sm font-medium text-gray-700 mb-1">요청 사유 *</label>
      <textarea
        value={reqReason}
        onChange={(e) => setReqReason(e.target.value)}
        placeholder="환불 요청 사유를 입력하세요"
        rows={3}
        className="w-full border rounded-lg px-3 py-2 text-sm focus:outline-none focus:ring-2 focus:ring-blue-500 resize-none"
      />
    </div>
  </div>

  <div className="p-4 border-t flex gap-3">
    <Button variant="outline" className="flex-1" onClick={onClose}>취소</Button>
    <Button
      variant="primary"
      className="flex-1"
      onClick={handleRequest}
      disabled={!reqReason || (reqType === 'partial' && reqAmount <= 0)}
    >
      환불 요청
    </Button>
  </div>
</Modal>

const [reqType, setReqType] = useState<'full' | 'partial'>('full');
const [reqAmount, setReqAmount] = useState(sale?.amount ?? 0);
const [reqReason, setReqReason] = useState('');
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- E_OPEN_01: SCR-S012 [환불 요청] 버튼 클릭

### 필수 데이터
| 항목 | 설명 |
|------|------|
| sale | 대상 매출 건 (member_name, product_name, amount, paid_at) |
| reqType | 전액/부분 |
| reqAmount | 요청 금액 |
| reqReason | 요청 사유 |

### 인터랙션 (User Actions)
1. 환불 유형 선택
2. 부분 환불 시 금액 입력
3. 요청 사유 입력
4. [환불 요청] → `02-처리중`
5. [취소] / X → 닫힘

### 비즈니스 룰
- 사유 미입력 또는 부분 환불 금액 0 이하 시 [환불 요청] 비활성화
- 부분 금액은 결제금액 초과 불가
- trainer/fc는 요청만 가능, 승인은 owner/manager

### 연결 화면
- `02-처리중`
- 닫힘
