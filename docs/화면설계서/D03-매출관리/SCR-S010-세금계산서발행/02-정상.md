## 📝 디스크립션

### 사용 시점 (Trigger)
- isLoading=false AND taxInvoices.length > 0

### 필수 데이터
| 항목 | 테이블 | 설명 |
|------|--------|------|
| 세금계산서 목록 | `tax_invoices` (가상) | branch_id, 날짜 범위 |

### 인터랙션 (User Actions)
1. 행 클릭 → DLG-S010 세금계산서상세
2. [세금계산서 발행] → DLG-S011 발행 모달
3. [재발행] 버튼 → DLG-S011 재발행 모드
4. 날짜/상태 필터 변경 → 재조회

### 비즈니스 룰
- 세액 = 공급가액 × 10%
- 발행완료 건: 취소 가능 (30일 이내)
- 취소된 건: 재발행 가능

### 연결 화면
- DLG-S010 세금계산서상세 (행 클릭)
- DLG-S011 세금계산서발행 ([발행] / [재발행])
