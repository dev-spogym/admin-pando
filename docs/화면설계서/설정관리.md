## SCR-080 센터 설정

### 1. 화면 목적
센터 기본 정보, 알림 설정, 테마 설정, 물품 관리를 4개 탭으로 분리하여 관리한다. 탭 간 이동 시 저장되지 않은 변경사항 경고를 제공한다.

### 2. 접근 경로
`사이드바 > 설정 > 센터 설정` (`/settings`)

### 3. ASCII 와이어프레임

```
┌─────────────────────────────────────────────────────────┐
│ [PageHeader] 센터 설정                                   │
├─────────────────────────────────────────────────────────┤
│ [ℹ️ 기본정보] [🔔 알림설정] [🎨 테마설정] [📦 물품관리]   │
├─────────────────────────────────────────────────────────┤
│                                                         │
│ (각 탭 콘텐츠 영역)                                      │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                              [저장하기]                   │
└─────────────────────────────────────────────────────────┘
```

### 4. 탭 구조

| 탭 | key | 아이콘 | 설명 |
|----|-----|--------|------|
| 기본정보 | basic | Info | 센터명, 주소, 사업자번호, 연락처, 영업시간 |
| 알림설정 | notification | Bell | 푸시/이메일/SMS 알림 ON/OFF |
| 테마설정 | theme | Palette | 라이트/다크/시스템 모드, 주요 색상, 폰트 크기 |
| 물품 관리 | supplies | Package | 수건/운동복 재고, 일일 제한, 잔여 수량 |

### 5. 기본정보 탭 (FormSection)

| 필드명 | key | 타입 | 아이콘 | 설명 |
|--------|-----|------|--------|------|
| 센터명 | name | text | Building2 | 센터 이름 |
| 설명 | description | textarea | - | 센터 소개 |
| 주소 | address | text | MapPin | 센터 주소 |
| 사업자번호 | businessNumber | text | - | 사업자등록번호 |
| 연락처 | phone | tel | Phone | 대표 전화번호 |
| 평일 영업시간 | openTime / closeTime | time | Clock | 오픈/마감 시간 |
| 주말 영업시간 | weekendOpenTime / weekendCloseTime | time | Clock | 주말 오픈/마감 |
| 업종 | sectors | multi-select | - | 센터 업종 태그 |

### 6. 알림설정 탭

| 카테고리 | 항목 | key | 기본값 |
|----------|------|-----|--------|
| 앱 푸시 | 입장 알림 | pushEntrance | OFF |
| 앱 푸시 | 만료 임박 알림 | pushExpiry | OFF |
| 앱 푸시 | 결제 알림 | pushPayment | OFF |
| 앱 푸시 | 예약 알림 | pushReservation | OFF |
| 이메일 | 주간 리포트 | emailWeeklyReport | OFF |
| 이메일 | 만료 안내 | emailExpiry | OFF |
| SMS | 결제 알림 | smsPayment | OFF |
| SMS | 만료 안내 | smsExpiry | OFF |

각 항목: ToggleSwitch (ON/OFF)

### 7. 테마설정 탭

| 필드 | key | 타입 | 옵션 |
|------|-----|------|------|
| 모드 | mode | 3버튼 | light (Sun), dark (Moon), system (Monitor) |
| 주요 색상 | primaryColor | color picker | 기본: #3B82F6 |
| 강조 색상 | accentColor | color picker | 기본: #10B981 |
| 폰트 크기 | fontSize | 3버튼 | sm, md, lg |

### 8. 물품 관리 탭

| 물품 | key | 항목 |
|------|-----|------|
| 수건 (대) | towelLarge | stock, dailyLimit, remaining |
| 수건 (소) | towelSmall | stock, dailyLimit, remaining |
| 운동복 (상의) | uniformTop | stock, dailyLimit, remaining |
| 운동복 (하의) | uniformBottom | stock, dailyLimit, remaining |

각 물품: 재고 수량(stock), 일일 제한(dailyLimit), 잔여 수량(remaining) 입력

### 9. 저장되지 않은 변경사항 경고

탭 전환 시 isDirty === true이면 경고 다이얼로그 표시:
- "저장되지 않은 변경사항이 있습니다. 이동하시겠습니까?"
- [이동] → 변경사항 버리고 탭 전환
- [취소] → 현재 탭 유지

### 10. 상태별 화면

| 상태 | 표시 |
|------|------|
| 저장 중 | isSaving=true, 버튼 disabled |
| 저장 성공 | saveSuccess=true (3초간 체크 아이콘 표시) |
| 로딩 중 | isLoading=true |

## SCR-081 권한 설정

### 1. 화면 목적
직원 역할별 메뉴 접근 및 기능 사용 권한을 매트릭스 형태로 관리한다. 6종 시스템 역할 + 커스텀 역할 생성/복사/삭제, 권한 충돌 검증 기능을 제공한다.

### 2. 접근 경로
`사이드바 > 설정 > 권한 설정` (`/settings/permissions`)

### 3. ASCII 와이어프레임

### 4. 6종 시스템 역할 (INITIAL_ROLES)

| id | code | name | description | isSystem | 특성 |
|----|------|------|-------------|----------|------|
| 1 | primary | 최고관리자 | 모든 권한 (수정 불가) | Y | 전 메뉴 전 권한 ON, 편집 불가 |
| 2 | owner | 센터장 | 경영/매출 전체 접근 | Y | 편집 가능 |
| 3 | manager | 매니저 | 회원/상품/일정 관리 | Y | 편집 가능 |
| 4 | fc | 피트니스 코치 | 담당 회원/수업 접근 | Y | 편집 가능 |
| 5 | staff | 스태프 | 출석 체크/기본 조회 | Y | 편집 가능 |
| 6 | readonly | 조회전용 | 읽기 전용 전체 | Y | 편집 가능 |

| 매출 | 매출 현황, POS 결제 |

| 설정 | 센터 설정, 키오스크 설정, IoT 설정, 권한 설정 |

**권한 타입 (PermissionType):** access(접근), read(조회), create(생성), update(수정), delete(삭제)

**기본 권한 매트릭스 (역할별 기본값):**

| 센터장 | ARCUD | ARCUD | ARCUD | ARCUD | AR | AR | ARCUD | ARCUD | ARCUD | ARCUD | ARCUD | ARCUD |
| 매니저 | ARCU | ARCU | ARCU | ARCU | AR | AR | ARCU | AR | - | ARCU | ARCU | AR |
| FC | AR | AR | AR | ARC | AR | AR | AR | - | - | AR | AR | - |

*(A=접근, R=조회, C=생성, U=수정, D=삭제)*

### 6. 권한 충돌 검증 로직 (validateConflicts)

| 검증 규칙 | 경고 메시지 |
|----------|------------|
| 삭제 ON + 수정 OFF | "{메뉴명}": 삭제 권한이 있지만 수정 권한이 없습니다. |
| 쓰기 ON + 접근 OFF | "{메뉴명}": 쓰기 권한이 있지만 접근 권한이 없습니다. |
| 쓰기 ON + 조회 OFF | "{메뉴명}": 조회 권한 없이 쓰기 권한이 설정되어 있습니다. |

충돌 발견 시 → DLG-081 충돌 경고 모달 표시 → 무시하고 저장 가능

### 7. 버튼 → 동작 매핑

| 버튼 | 조건 | 동작 |
|------|------|------|
| 역할 추가 (+) | 항상 | DLG-081-C 역할 생성 모달 |
| 초기화 (RotateCcw) | 항상 | DLG-081-R 초기화 확인 → savedPermissions 복원 |
| 변경 사항 저장 (Save) | isDirty && !isPrimary | validateConflicts → 충돌 없으면 doSave / 있으면 DLG-081 |
| 전체 허용 | !isPrimary | 전 메뉴 전 권한 ON |
| 전체 차단 | !isPrimary | 전 메뉴 전 권한 OFF |
| 역할 복사 | !isSystem | DLG-081-CP 역할 복사 모달 |
| 역할 삭제 | !isSystem | DLG-081-D 역할 삭제 확인 |

### 8. 변경 감지

- isDirty: 현재 permissions !== savedPermissions
- 변경된 셀: isChanged(menuId, type) → 해당 셀 하이라이트 표시
- 저장 성공 시: 3초간 "저장되었습니다." 배지 표시 (fade-in slide-in-from-right)

## SCR-082 키오스크 설정

### 1. 화면 목적
키오스크 기본 설정, 화면 커스터마이즈, TTS 메시지, 출입 규칙을 4탭으로 관리한다. 실시간 미리보기(폰 프레임)를 제공한다.

### 2. 접근 경로
`사이드바 > 설정 > 키오스크 설정` (`/settings/kiosk`)

### 3. 탭 구조

| 탭 | key | 아이콘 | 설명 |
|----|-----|--------|------|
| 기본 설정 | basic | Settings | 활성화, 타입, 체크인 방식, 화면 타임아웃, 관리자 PIN |
| 화면 설정 | screen | Monitor | 배경, 로고, 테마 색상, 환영 문구, 공지, 배너 |
| TTS 설정 | tts | Volume2 | 이벤트별 음성 안내 메시지 |
| 출입 규칙 | access | Lock | 허용 시간, 만료자 처리, 중복 체크인, 미수금 처리 |

### 4. 기본 설정 탭

| 필드 | key | 타입 | 기본값 | 설명 |
|------|-----|------|--------|------|
| 키오스크 활성화 | isActive | toggle | ON | - |
| 키오스크 타입 | kioskType | select | typeB | typeA: 입퇴장 전용, typeB: 결제 포함 |
| 체크인 방식 | checkInMethods | multi-checkbox | [qr, rfid, face] | QR, RFID, 얼굴인식 |
| 화면 타임아웃 | screenTimeout | number | 30초 | 무동작 시 대기 화면 전환 |
| 자동 로그아웃 | autoLogout | number | 5분 | 관리자 모드 자동 로그아웃 |
| 관리자 PIN | adminPin | password | 1234 | 관리자 모드 진입 비밀번호 |

### 5. 화면 설정 탭

| 필드 | key | 타입 | 설명 |
|------|-----|------|------|
| 배경 이미지 | bgImage | URL/upload | 대기 화면 배경 |
| 로고 이미지 | logoImage | URL/upload | 센터 로고 |
| 테마 색상 | themeColor | color picker | 기본: #3B82F6 |
| 환영 문구 | welcomeMessage | text | "오늘도 건강한 하루 되세요!" |
| 공지 표시 | showNotice | toggle | ON |
| 공지 내용 | noticeContent | text | - |
| 날씨 표시 | showWeather | toggle | ON |
| 광고 배너 표시 | showAdBanner | toggle | ON |
| 배너 목록 | banners | list (이름+URL) | 추가/삭제 가능 |

### 6. TTS 설정 탭 (6종 기본 메시지)

| # | event | 메시지 | description |
|---|-------|--------|------------|
| 1 | 체크인 성공 | "{이름}님 환영합니다." | 정상 체크인 완료 시 |

### 7. 출입 규칙 탭

### 8. 키오스크 미리보기 (KioskPreview)
폰 프레임 (w-200px, h-360px) 내 실시간 렌더링:
- 배경 이미지 (20% opacity)
- 로고 영역
- 현재 시간 (20px bold)
- 환영 문구
- 체크인 버튼 (테마 색상)
- 공지 영역 (showNotice 시)
- 하단: "타입 A/B" 표시

## SCR-083 IoT/출입 관리

### 1. 화면 목적
센터 IoT 기기(출입문, 체성분, 키오스크, RFID, 카메라) 관리, 게이트 원격 개방, 출입 로그 조회, 출입 규칙 설정, 자동 전원 스케줄을 5탭으로 관리한다.

### 2. 접근 경로
`사이드바 > 설정 > IoT 관리` (`/settings/iot`)

### 3. 탭 구조

| 탭 | key | 아이콘 | 설명 |
|----|-----|--------|------|
| 게이트 관리 | gate | DoorOpen | 게이트 목록 + 원격 개방 |
| IoT 기기 | iot | Cpu | 기기 등록/삭제/연결 테스트 |
| 출입 로그 | log | ClipboardList | 시간별 출입 기록 |
| 출입 규칙 | rule | Settings2 | 게이트별 요일/시간 규칙 |
| 자동 전원 | power | Cpu | 기기별 전원 ON/OFF 스케줄 |

### 4. IoT 기기 테이블 컬럼

| 컬럼 | 설명 |
|------|------|
| 기기명 | 이름 |
| 유형 | 출입문/체성분/키오스크/RFID/카메라 (색상 배지) |
| IP 주소 | ip:port |
| 시리얼 | serial 번호 |
| 펌웨어 | 버전 |
| 상태 | 온라인(녹색)/오프라인(회색)/오류(빨강) StatusBadge |
| 최종 통신 | lastComm 날짜시간 |
| 연결 테스트 | [테스트] 버튼 → loading → success/fail 결과 |

**기기 유형별 색상 (DEVICE_TYPE_COLORS):**

| 유형 | 색상 |
|------|------|
| 출입문 | text-primary bg-primary-light |
| 체성분 | text-accent bg-accent-light |
| 키오스크 | text-information bg-information/10 |
| RFID | text-primary bg-primary-light |
| 카메라 | text-content-secondary bg-surface-secondary |

### 5. 게이트 관리

| 기능 | 설명 |
|------|------|
| 게이트 목록 | 출입문 유형 기기만 필터 표시 |
| 원격 개방 | DLG-083 확인 → 0.4초 후 toast.success("{이름} 게이트가 개방되었습니다." → 3초 후 자동 잠금) |

### 6. 출입 규칙 탭

| 필드 | 설명 |
|------|------|
| 게이트 | 게이트 이름 |
| 요일 | 월~일 다중 선택 |
| 시작 시간 | 06:00 등 |
| 종료 시간 | 23:00 등 |

### 7. 자동 전원 스케줄

| 컬럼 | 설명 |
|------|------|
| 기기명 | 대상 기기 |
| 전원 ON 시간 | 자동 시작 시간 |
| 전원 OFF 시간 | 자동 종료 시간 |
| 활성화 | ON/OFF 토글 |

기본 스케줄:
- 로비 키오스크: 05:50~23:10
- 프로젝터 (타석1): 06:00~23:00
- 프로젝터 (타석2): 06:00~23:00
- 메인 출입구 RFID: 05:55~23:05

## SCR-085 공지사항 관리

### 1. 화면 목적
센터 공지사항을 등록, 수정, 삭제하고, 고정/공개 설정 및 읽음 상태를 관리한다.

### 2. 접근 경로
`사이드바 > 설정 > 공지사항` (`/notices`)

### 3. 테이블 컬럼

| 컬럼 | 렌더링 |
|------|--------|
| 고정 | Pin 아이콘 (isPinned) |
| 제목 | 텍스트 + [NEW] 배지 (미읽음) |
| 작성자 | authorName |
| 작성일 | createdAt (YYYY-MM-DD) |
| 공개 | Eye/EyeOff 아이콘 |
| 관리 | [수정] [삭제] 버튼 |

**정렬:** 고정 → 최신순

### 4. 버튼 → 동작 매핑

| 버튼 | 동작 |
|------|------|
| 공지 등록 (Primary + Plus) | 등록 모달 표시 |
| 수정 (Edit2) | 수정 모달 표시 (기존 데이터 로드) |
| 삭제 (Trash2) | DLG-085 삭제 확인 다이얼로그 |
| 제목 클릭 | 상세 보기 패널 (detailNotice) |

## SCR-086 출석 관리

### 1. 화면 목적
회원의 일별 출석 기록을 조회하고, 수동 출석 체크, 퇴장 처리, 세션 차감 기능을 제공한다.

### 2. 접근 경로
`사이드바 > 출석 관리` (`/attendance`)

### 3. 통계 카드

날짜별 총 출석, 성공, 실패, 재실 인원 표시

### 4. 출석 유형 배지 (ATTENDANCE_TYPE_MAP)

| 유형 | 스타일 | variant |
|------|--------|---------|
| 일반 | info 계열 | info |
| PT | success 계열 | success |
| GX | primary 계열 | peach |
| 수동 | secondary 계열 | default |

### 5. 테이블 컬럼

| 컬럼 | 렌더링 |
|------|--------|
| 시간 | 체크인 시각 |
| 회원명 | 이름 (클릭 → 상세) |
| 출석유형 | 일반/PT/GX/수동 배지 |
| 체크인 방식 | 앱/키오스크/수동 |
| 상태 | 성공(녹색)/실패(빨강) |
| 재실 여부 | 재실/부재 |

| 퇴장 | [퇴장] 버튼 → checkOutMember API |
