## [SCR-C001] 수업/캘린더

### 1. 화면 목적
PT 및 그룹 수업 스케줄을 FullCalendar 기반 캘린더로 관리하고, 수업 등록/수정/삭제, 예약 현황 확인, 드래그앤드롭 일정 이동, 좌석 배치, 미승인 일정 처리를 수행한다. 센터의 모든 수업 일정을 한눈에 파악하는 메인 허브.

### 2. 접근 경로
- 사이드바 > 수업/캘린더 > 캘린더 (`/calendar`)
- 내부 viewId: `969`

### 3. 권한별 접근

| 역할 | 접근 | 제한 사항 |
|------|------|-----------|
| 슈퍼관리자 | 전체 접근 | 전 지점 캘린더 조회 가능 |
| 센터장/매니저 | 전체 접근 | 수업 등록/수정/삭제, 승인/거절 |
| 트레이너 | 제한 접근 | 본인 수업만 수정 가능. 다른 강사 수업 조회만 |
| 프론트 | 조회만 | 수업 등록/수정/삭제 불가 |

### 6. 캘린더 뷰 (4종)

#### 월간 뷰 (`dayGridMonth`)
- 날짜 셀 클릭 시 일간 뷰로 전환
- 각 날짜 셀 우측 상단에 수업 건수 배지 (파란색 원형, 10px 폰트)
- 이벤트: 컴팩트 형태 - 6px 색상 dot + 시간 + 제목
- 미승인 이벤트: 반투명(opacity-70) + `미승인` 텍스트 (8px, amber)
- `dayMaxEvents: 3` 초과 시 "+N more" 링크 -> 팝오버
- 최소 셀 높이: 96px

#### 주간 뷰 (`timeGridWeek`)
- 시간 슬롯: 30분 간격, 06:00~22:00
- 이벤트 블록: 제목 + 분류 태그 + 인원/정원 + 강사 + 장소
- 왼쪽 3px 컬러 보더 (카테고리/강사 색상)
- 드래그앤드롭으로 시간 이동 가능
- 리사이즈로 시간 변경 가능
- 시간 클릭 시 수업 등록 모달 (날짜+시간 사전 입력)

#### 일간 뷰 (`timeGridDay`)
- 주간 뷰와 동일한 시간 그리드
- 단일 날짜 확대 표시
- nowIndicator: 현재 시간 빨간 줄 표시

#### 목록 뷰 (`listWeek`)
- 주 단위 리스트
- 제목 (볼드) + 카테고리 배지 + 미승인 배지 + 강사/장소 + 인원

### 9. 우측 패널 (선택 날짜 수업 리스트)

- 너비: 280px, sticky
- 날짜 네비게이션: `< [날짜 표시] >`
- 일정 수 요약 바: `조회 일정 N건`
- 각 수업 카드:
  - 색상 dot (2px) + 수업명 + 마감/여유 배지
  - 시간 (HH:MM ~ HH:MM)
  - 강사명
  - 인원 바: currentCount/capacity + 프로그레스바 (>=100%: error, >=80%: warning, <80%: success)

### 10. 미승인 일정 알림

- 상단 amber 배너: `미승인 일정 N건이 있습니다.` + [처리하기] 버튼
- Bell 아이콘 + 볼드 건수
- 미승인 이벤트는 캘린더에서 반투명(opacity-70) 표시
- 처리하기 클릭: 일정 요청 처리 페이지(`/schedule-requests`)로 이동

## [SCR-C002] 수업 관리

### 1. 화면 목적
수업 정의(Lesson)를 등록/수정/삭제하고, classes 테이블의 수업 기록을 관리한다. 수업 상태 변경(시작/완료/노쇼/취소), 서명 기반 완료 처리, 노쇼/취소 정책 설정을 포함한다.

### 2. 접근 경로
- 사이드바 > 수업/캘린더 > 수업 관리 (`/lessons`)
- 또는 캘린더 페이지 > "수업 관리" 탭

### 3. 권한별 접근

| 역할 | 접근 | 제한 사항 |
|------|------|-----------|
| 슈퍼관리자 | 전체 접근 | 없음 |
| 센터장/매니저 | 전체 접근 | 없음 |
| 트레이너 | 제한 접근 | 본인 수업만 수정/삭제. 일괄변경 불가 |
| 프론트 | 조회만 | 수업 등록/수정/삭제 불가 |

### 2. 접근 경로
사이드바 > 수업/캘린더 > 시간표 등록 (`/class-schedule`)

### 3. 권한별 접근

| 트레이너 | 접근 불가 | 접근 차단 |
| 프론트 | 접근 불가 | 접근 차단 |

없음
