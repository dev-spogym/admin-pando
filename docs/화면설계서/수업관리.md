## SCR-021 수업/캘린더

### 화면 목적
PT 및 그룹 수업 스케줄을 FullCalendar 기반 캘린더로 관리하고, 수업 등록/수정/삭제, 예약 현황 확인, 드래그&드롭 일정 이동, 좌석 배치, 미승인 일정 처리를 수행한다.

### 접근 경로
사이드바 > 수업관리 > 수업/캘린더 (`/calendar`)

### ASCII 와이어프레임

```
+------------------------------------------------------------------+
| [PageHeader] 수업/캘린더                                          |

+------------------------------------------------------------------+
| [일정표] [수업 관리] [횟수 관리] [페널티 관리] [유효 수업 목록]     |
```

### 5개 탭 구성

| 탭 키 | 탭 라벨 | 라우팅 경로 | 설명 |

### 4개 캘린더 뷰

#### 월간 뷰 (`dayGridMonth`)
- 날짜 셀 클릭 시 일간 뷰로 전환
- 각 날짜 셀 우측 상단에 수업 건수 배지 (파란색 원형, 10px 폰트)
- 이벤트: 컴팩트 형태 - 6px 색상 dot + 시간 + 제목
- 미승인 이벤트: 반투명 + `미승인` 텍스트 (8px, amber)
- `dayMaxEvents: 3` 초과 시 "+N more" 링크 -> 팝오버
- 최소 셀 높이: 96px

#### 주간 뷰 (`timeGridWeek`)
- 시간 슬롯: 30분 간격, 06:00~22:00
- 이벤트 블록: 제목 + 분류 태그 + 인원/정원 + 강사 + 장소
- 왼쪽 3px 컬러 보더 (카테고리/강사 색상)
- 드래그&드롭으로 시간 이동 가능
- 리사이즈로 시간 변경 가능
- 시간 클릭 시 수업 등록 모달 (날짜+시간 사전 입력)

#### 일간 뷰 (`timeGridDay`)
- 주간 뷰와 동일한 시간 그리드
- 단일 날짜 확대 표시
- nowIndicator: 현재 시간 빨간 줄 표시

#### 목록 뷰 (`listWeek`)
- 주 단위 리스트
- 제목 (볼드) + 카테고리 배지 + 미승인 배지 + 강사/장소 + 인원

### 필터 옵션

| 필터 | 타입 | 옵션 |
|------|------|------|
| 강사 | Select | 전체 강사, [DB staff 목록] |

### 우측 패널 (선택 날짜 수업 리스트)

- 너비: 280px, sticky
- 날짜 네비게이션: `< [날짜 표시] >`
- 일정 수 요약 바: `조회 일정 N건`
- 각 수업 카드:
  - 색상 dot (2px) + 수업명 + 마감/여유 배지
  - 시간 (HH:MM ~ HH:MM)
  - 강사명
  - 인원 바: currentCount/capacity + 프로그레스바

### 미승인 일정 알림

- 상단 amber 배너: `미승인 일정 N건이 있습니다.`
- Bell 아이콘 + 볼드 건수
- 미승인 이벤트는 캘린더에서 반투명(opacity-70) 표시

#### 좌석 그리드 상태별 색상

| 상태 | 배경색 | 보더 | 글자색 | 커서 |
|------|--------|------|--------|------|
| 예약됨 | `bg-state-info/20` | `border-state-info` | `text-state-info` | `cursor-not-allowed` |
| 선택 중 | `bg-state-warning/80` | `border-state-warning` | `text-white` | `pointer` |
| 빈 좌석 | `bg-surface-tertiary` | `border-line` | `text-content-secondary` | `pointer` |

#### 예약 현황 바 색상

| 예약률 | 바 색상 | 텍스트 색상 |
|--------|---------|-------------|
| >= 100% | `bg-state-error` | `text-state-error` |
| >= 80% | `bg-state-warning` | `text-state-warning` |
| < 80% | `bg-state-success` | `text-state-success` |

#### 버튼 동작

| 버튼 | 조건 | 동작 |
|------|------|------|
| 승인 | 미승인(pending) 상태 | Supabase approval_status='approved' 업데이트 -> toast |
| 거절 | 미승인(pending) 상태 | Supabase approval_status='rejected' 업데이트 -> toast |
| 삭제 | editable=true | Supabase classes DELETE -> 이벤트 목록에서 제거 -> toast |
| 수정 | editable=true | 상세 모달 닫고 -> 등록 모달 열기 (기존 데이터 프리필) |

#### 닫기 조건

- 조회 버튼: `fetchCounts()` 호출

| 잔여횟수 | `remainCount` | 숫자 + 프로그레스바 (<=20% red, <=50% amber, >50% green) |

| 종료일 | `endDate` | 날짜 or '-' |
| 상태 | `status` | StatusBadge dot (ACTIVE=success, EXPIRED=error, PAUSED=warning) |
