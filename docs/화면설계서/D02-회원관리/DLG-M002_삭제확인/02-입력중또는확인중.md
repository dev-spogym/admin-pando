## 상태 진입 조건
- `confirmText !== ''` (한 글자 이상 입력)
- `confirmText !== '삭제'` 또는 `confirmText === '삭제'` 양쪽 모두 해당

## 비주얼 델타

### 2a. 불일치 (mismatch, "ㅅ" "삭" "삭재" 등)
```
│ 삭제하려면 '삭제'를 입력하세요              │
│ [삭재                                  ]   │ ← border-gray-300
│ ⚠ "삭제"와 정확히 일치해야 합니다            │ ← text-amber-700
│                                           │
│        [취소]  [🗑 삭제하기 (비활성)]       │
```

### 2b. 일치 (match, "삭제")
```
│ 삭제하려면 '삭제'를 입력하세요              │
│ [삭제                                  ]   │ ← border-green-400 bg-green-50
│ ✓ 입력 확인됨                              │ ← text-green-700
│                                           │
│        [취소]  [🗑 삭제하기 (활성, 빨강)]   │ ← bg-red-600
```

| 요소 | mismatch | match |
|---|---|---|
| input border | `border-gray-300 focus:ring-red-500` | `border-green-400 bg-green-50 focus:ring-green-500` |
| 힌트 | "⚠ 정확히 일치해야 합니다" (amber) | "✓ 입력 확인됨" (green) |
| "삭제하기" 버튼 | `disabled bg-red-300` | `enabled bg-red-600` |

## 역할별 차이
- 없음 (입력 로직 공통)

## 상태 고유 인터랙션
1. **input 편집** → `setConfirmText(value)` → 즉시 match/mismatch 계산
2. **Enter 키** (일치 시) → `handleDelete()` 호출 (UX 편의, 선택 적용)
3. **Backspace/Delete 연타** → 빈 문자열 시 `01-열림` 복귀
4. **"삭제하기" 클릭 (match)** → `03-제출중`
5. **취소/X/Esc** → 모달 닫기 + confirmText 초기화

## 비즈니스 룰 델타
1. **엄격 일치**: trim/대소문자 구분 없음이 아니라 **완전 일치** (`=== '삭제'`)
2. **IME 조합 중**: composition 이벤트 중에는 match 계산 보류 (`onCompositionEnd` 이후)
3. **자동완성/붙여넣기 허용**: 접근성 및 실수 방지 (엄격할 필요 없음)
4. **Enter 제출 옵션**: `isMatch === true && e.key === 'Enter' && !mutation.isPending` → handleDelete

## 에지 케이스
- **한글 조합 "ㅅ + ㅏ + ㄱ..."**: `onCompositionEnd` 후 검증
- **양쪽 공백 " 삭제 "**: 비일치 (엄격 규칙)
- **전각 공백/특수문자**: 비일치
- **붙여넣기 후 즉시 제출**: 허용 (의도적 UX)

## 바이브코딩 프롬프트 (델타)
```
상태: typing
isMatch = confirmText === '삭제';
onInputChange:
  <input
    value={confirmText}
    onChange={e=>setConfirmText(e.target.value)}
    onKeyDown={e=>{
      if (e.key === 'Enter' && isMatch && !mutation.isPending) {
        e.preventDefault();
        handleDelete();
      }
    }}
    onCompositionEnd={e=>setConfirmText((e.target as HTMLInputElement).value)}
    className={cn(
      'h-11 w-full rounded-lg border px-3 text-sm focus:outline-none focus:ring-2',
      isMatch ? 'border-green-400 bg-green-50 focus:ring-green-500'
              : 'border-gray-300 focus:ring-red-500'
    )}/>
힌트 영역:
  <p className="text-xs mt-1" aria-live="polite">
    {confirmText === '' ? <span className="text-gray-500">삭제 텍스트 입력 필요</span>
      : isMatch ? <span className="text-green-700">✓ 입력 확인됨</span>
      : <span className="text-amber-700">⚠ "삭제"와 정확히 일치해야 합니다</span>}
  </p>
```
