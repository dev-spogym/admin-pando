## 상태 고유 비주얼 델타
- "삭제" 버튼: `<Loader2 className="size-4 animate-spin" />` + "삭제 중..." + `cursor-not-allowed` + `bg-rose-300`
- "취소" 버튼: `disabled + opacity-50`
- X 버튼: `disabled + opacity-50`
- backdrop `pointer-events-none` 가드(클릭해도 닫히지 않음)

## 상태 고유 인터랙션
1. **ESC**: 무시 (`onEscapeKeyDown preventDefault`)
2. **배경 클릭**: 무시 (`onPointerDownOutside preventDefault`)
3. **재클릭**: 무시 (disabled)

## 상태 고유 비즈니스 룰 델타
- **API**: `DELETE /api/consultations/:id` (body 없음)
- **타임아웃**: 5초 이내 무응답 → 토스트 "네트워크 지연" (선택)
- **낙관적 제거(옵션)**: `queryClient.setQueryData(['consultations', memberId], prev => prev.filter(c => c.id !== id))` + `context.previous` 보관 → 실패 시 롤백

## 역할별 차이
- 없음 (API 계층 응답 기준)

## 에지 케이스
- **응답 누락**: AbortController 5초 타임아웃 → `04-실패`
- **204 정상 응답**: body 없음 → 성공 처리
- **네트워크 끊김**: Axios/fetch reject → `04-실패` 유지
