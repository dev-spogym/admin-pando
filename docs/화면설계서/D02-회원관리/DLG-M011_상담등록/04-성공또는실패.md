# DLG-M011 상담 등록 — 상태 04: 성공 / 실패

> 상속: [`00-기본화면.md`](./00-기본화면.md) — 이 문서는 **변경점(델타)**만 기술합니다.

## 메타
| 항목 | 값 |
|------|----|
| 상태 코드 | `consult-done` |
| 상태 ID | STATE_DONE |
| 다이어그램 노드 | `M3_결과분기.md` → SUCCESS / FAIL |
| 이전 상태 | `03-제출중` (2xx/4xx/5xx 응답) |
| 다음 상태 | Closed (성공) / `02-입력중` (실패 재시도) / Closed (치명 실패) |

## 상태 진입 조건
- `mutation.isPending === false`
- `mutation.isSuccess` or `mutation.isError`

## 상태 고유 비주얼 델타

### 성공 (SUCCESS)
- 토스트: `toast.success("상담 이력이 저장되었습니다.")` (수정 시 "수정되었습니다.") — 3초
- 모달 즉시 닫힘 (별도 성공 화면 없음)
- 상담 목록 refetch → 새 행 표시 (최상단)

### 실패 (FAIL)
- 모달 유지 (단, 403/404 는 닫기)
- 에러 토스트 3초: 에러코드별 메시지
- "저장" 버튼 `02-입력중` 상태로 복귀 (활성)
- `form.formState.errors` 는 유지 (서버 필드 에러 있을 시 매핑)

## 상태 고유 인터랙션

### 성공
- 자동 정리 (사용자 액션 없음)
- 토스트 3초 후 자동 소멸

### 실패
1. **재시도**: "저장" 재클릭 → `03-제출중`
2. **취소**: 모달 닫힘 (dirty 잔존 시 DLG-002 이탈경고)
3. **치명(403/404)**: 토스트 후 자동 닫힘 (2초 지연)

## 상태 고유 비즈니스 룰 델타

### 성공 후처리
```ts
onSuccess: (data) => {
  qc.invalidateQueries({ queryKey: ['consultations', memberId] });
  qc.invalidateQueries({ queryKey: ['member', memberId] }); // 상담 카운트 갱신
  toast.success(editTarget ? '상담 이력이 수정되었습니다.' : '상담 이력이 저장되었습니다.');
  form.reset(data); // dirty=false
  onClose();
}
```

### 실패 에러코드 매핑
| errorCode | 액션 |
|---|---|
| E400001 | 필드 에러 매핑 + 모달 유지 |
| E400105 | 인라인 "2,000자 이내" + 유지 |
| E403001 | 토스트 + 닫기(2s) |
| E404100 | 토스트 "회원을 찾을 수 없습니다" + 닫기 |
| E409101 | 토스트 "현재 상태에서 불가" + 닫기 |
| E500001 | 토스트 + 유지 재시도 |
| NETWORK | 토스트 + 유지 재시도 |

### KPI 기록
- 성공 시 상담 등록 이벤트 발행 → `member_consultations_created` 지표

## 역할별 차이
- `fc`: 실패(E403001) 시 "본인 지점 회원만 상담 가능" 힌트
- `manager+`: 전체 에러 메시지 그대로

## 에지 케이스
- **성공 응답 누락**: `data` undefined → 안전 모드: refetch + 닫기
- **토스트 중복**: React Query 다중 mutation.onSettled 방지 (mutationKey)
- **editTarget 업데이트 후 삭제됨(동시성)**: 404 → 닫기 + 목록 refetch

## 🧩 바이브코딩 프롬프트 (델타)
```
상속: 00-기본화면.md §14 전체
상태: consult-done

onSuccess(data):
  invalidateQueries(['consultations', memberId])
  invalidateQueries(['member', memberId])
  toast.success(editTarget ? '상담 이력이 수정되었습니다.' : '상담 이력이 저장되었습니다.')
  form.reset(data)
  onClose()

onError(e):
  switch (e.errorCode) {
    case 'E403001': toast.error('권한이 없습니다'); setTimeout(onClose, 2000); break;
    case 'E404100': toast.error('회원을 찾을 수 없습니다'); onClose(); break;
    case 'E409101': toast.error('현재 상태에서 작업할 수 없습니다'); onClose(); break;
    default: toast.error(e.message ?? '저장 실패');
  }
```
