## 역할별 차이
모든 역할 동일 EmptyState. CTA는 canCreateLesson true인 경우만 표시.

## 상태 고유 인터랙션
1. [캘린더로 이동] → `router.push('/calendar')`
2. 탭 전환 → 02 또는 04

## 상태 고유 비즈니스 룰 델타
- 필터 적용 후 0건인 경우: 다른 메시지 "선택한 조건에 해당하는 수업 기록이 없습니다" + [필터 초기화] 버튼
- 페이지네이션 숨김 (totalPages=0)

## 에지 케이스
- trainer + 본인 담당 기록 0건 + 다른 강사 기록 존재: trainer 관점 0건 (정상 빈 상태)
- 과거 삭제된 기록: soft delete된 경우 건수에 포함 X
