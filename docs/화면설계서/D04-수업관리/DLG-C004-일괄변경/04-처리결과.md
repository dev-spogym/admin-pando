## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind v4 기반 일괄변경 결과 처리를 작성하라.
화면: DLG-C004 일괄변경 — 상태: 처리 결과

파일: src/components/classes/BulkChangeModal.tsx (handleApply)

처리 중 상태:
- 적용 버튼: disabled + Loader2 스피너
- 폼 필드: pointer-events-none

const handleApply = async () => {
  setIsProcessing(true)
  try {
    const result = await bulkUpdateClasses({ ids: selectedIds, field: selectedOption, value: changeValue })
    if (result.failedIds?.length > 0) {
      // 부분 실패
      toast.warning(`일부 수업 변경에 실패했습니다. (${result.failedIds.length}건)`)
      setFailedIds(result.failedIds)
      // 모달 유지, 실패 목록 표시
    } else {
      toast.success(`${selectedIds.length}개 수업이 변경되었습니다.`)
      onSuccess()
      onClose()
    }
  } catch (error) {
    if (error.status === 403) toast.warning("권한이 없습니다.")
    else toast.error("처리에 실패했습니다.")
  } finally {
    setIsProcessing(false)
  }
}

부분 실패 UI (failedIds.length > 0):
<div className="rounded-lg border border-yellow-200 bg-yellow-50 p-3 mt-3">
  <p className="text-sm text-yellow-800">
    다음 {failedIds.length}개 수업은 변경에 실패했습니다:
  </p>
  <ul className="mt-1 text-xs text-yellow-700 list-disc list-inside">
    {failedIds.map(id => <li key={id}>{id}</li>)}
  </ul>
</div>
```
