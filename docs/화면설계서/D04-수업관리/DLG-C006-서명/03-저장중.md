```
서명 저장 흐름:
const handleConfirm = async () => {
  // 유효성 검사
  if (sigPadRef.current?.isEmpty()) {
    toast.error("서명을 입력해주세요.")
    return
  }
  const strokeCount = sigPadRef.current?.toData()?.length ?? 0
  if (strokeCount < 3) {
    toast.error("서명이 너무 짧습니다. 다시 서명해주세요.")
    return
  }

  setIsSaving(true)
  try {
    const base64 = sigPadRef.current?.toDataURL('image/png')
    await fetch(`/api/lesson-records/${recordId}/status`, {
      method: 'PATCH',
      body: JSON.stringify({ status: 'completed', signature_data: base64 }),
    })
    toast.success("수업이 완료 처리되었습니다.")
    onSuccess()   // DLG-C005 갱신
    onClose()     // DLG-C006 닫힘
  } catch (error) {
    if (error.status === 400) toast.warning("입력값을 확인해주세요.")
    else if (error.status === 403) toast.warning("권한이 없습니다.")
    else toast.error("처리에 실패했습니다.")
  } finally {
    setIsSaving(false)
  }
}

저장 중 UI (isSaving=true):
- 확인 버튼: disabled + Loader2 스피너
- 캔버스, 지우기: pointer-events-none
```
