## 📝 디스크립션

### 사용 시점 (Trigger)
- GET /api/lessons/:id 200 응답 수신

### 표시 데이터
| 항목 | 소스 |
|------|------|
| 수업명 | lesson.class.name |
| 강사명 | lesson.instructor.name |
| 날짜/시간 | lesson.start_at ~ end_at |
| 정원/참가자 | lesson.capacity, current_count |
| 상태 | lesson.status |
| 메모 | lesson.memo (선택) |

### 인터랙션 (User Actions)
1. "수정" 버튼 → onEdit(lessonId) → DLG-C001 수정 모달
2. "삭제" 버튼 → `04-삭제확인`
3. X/ESC/오버레이 클릭 → `01-닫힘`

### 비즈니스 룰
- completed/in_progress: 삭제 버튼 disabled
- fc 타강사 수업: 버튼 없음 (읽기전용)

### 연결 화면
- 수정: DLG-C001 (mode='edit')
- 삭제: `04-삭제확인`
- 닫기: `01-닫힘`
