## 📝 디스크립션

### 사용 시점 (Trigger)
- SCR-I001 "수동 출석 등록" 버튼 클릭
- `hasPermission('attendance:write') === true`

### 필수 데이터
- 현재 시각: `format(new Date(), "yyyy-MM-dd'T'HH:mm")` (datetime-local 기본값)
- 회원 검색: 입력 후 Supabase 조회

### 인터랙션 (User Actions)
1. 회원 검색 입력 → `02-회원검색중`
2. 채널 선택 → selectedChannel 상태 갱신
3. 시간 수정 → 미래 시간 불가 (max 설정)
4. 취소/X/ESC → 모달 닫힘

### 비즈니스 룰
- 출석 시간: 미래 시간 입력 불가 (`max` 현재 시각)
- 채널 기본값 없음 (필수 선택)
- 메모 최대 200자
