오프라인 감지:
useEffect(() => {
  const handleOffline = () => setIsOffline(true)
  const handleOnline = () => { setIsOffline(false); loadData() }
  window.addEventListener('offline', handleOffline)
  window.addEventListener('online', handleOnline)
  return () => {
    window.removeEventListener('offline', handleOffline)
    window.removeEventListener('online', handleOnline)
  }
}, [])

## 📝 디스크립션

### 사용 시점 (Trigger)
- `window 'offline'` 이벤트
- 네트워크 단절 상태에서 페이지 진입

### 필수 데이터
- 캐시된 마지막 출석 데이터 (state 유지)
- isOffline: boolean

### 인터랙션 (User Actions)
- 오프라인 중: 모든 액션 비활성화
- 복구 시: 자동 loadData() + 배너 제거

### 비즈니스 룰
- 오프라인 중 수동 출석 등록 불가 (서버 저장 불가)
- 기존 로드된 데이터는 읽기 가능
- 복구 후 자동 재조회 (사용자 액션 불필요)

### 에지 케이스

- 장시간 오프라인: "마지막 업데이트: N분 전" 타임스탬프 표시
