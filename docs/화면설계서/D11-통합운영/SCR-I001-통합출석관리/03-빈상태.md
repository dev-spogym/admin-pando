## 📝 디스크립션

### 사용 시점 (Trigger)
- 로드 완료 후 `attendance.length === 0`
- 날짜·채널·검색 필터 적용 결과 0건

### 필수 데이터
- KPI 카운트: 0 (표시)
- 현재 필터 값 (빈 메시지 문구 분기용)

### 인터랙션 (User Actions)
1. 수동 출석 등록 버튼 → DLG-I001
2. 필터 초기화 → `01-로딩` → `02-정상` 또는 재빈상태
3. 날짜 변경 → 재조회

### 비즈니스 룰
- KPI 카드는 0으로 유지 표시 (숨기지 않음)
- `attendance:write` 권한 없으면 수동 등록 버튼 숨김
- 실시간 WebSocket 구독 유지 (신규 출석 수신 시 자동 정상 상태 전환)

### 에지 케이스
- 검색어 있고 0건: "검색 결과 없음" 메시지 + 검색 초기화 링크
- 날짜 미래: 빈 상태 + "아직 출석 데이터가 없습니다" 안내

### 연결 화면

- 다음: DLG-I001 / 필터 변경 후 `02-정상`
