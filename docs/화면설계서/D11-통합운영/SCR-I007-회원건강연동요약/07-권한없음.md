## 📝 디스크립션

### 사용 시점 (Trigger)
- 미인증 상태에서 `/members/[id]/health` 직접 접근

### 비즈니스 룰
- SCR-I007은 전 역할 읽기 허용 (403 페이지 없음)
- 쓰기 권한(수기등록/락커배정/알림발송)만 hasPermission으로 버튼 단위 제어
- 미인증: 로그인 후 returnUrl로 자동 복귀

### 연결 화면
- 미인증: `/login?returnUrl=/members/[id]/health`
- 로그인 후: `/members/[id]/health` 복귀
