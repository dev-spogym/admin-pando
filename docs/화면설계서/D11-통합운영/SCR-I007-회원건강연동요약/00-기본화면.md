interface MemberHealthSummary {

## 8. 비즈니스 룰
1. 혈압 수축기 ≥ 140 → `danger` 배지
2. 체지방률 전회 대비 +3% 이상 → `warn`
3. 트레이너: 담당 회원만 조회
4. 측정 7일 이상 경과 → "오래된 데이터" 배지
5. 이상 지표 있는 카드 상단 정렬
