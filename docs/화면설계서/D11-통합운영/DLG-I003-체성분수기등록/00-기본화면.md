## 6. 데이터 계약
```ts
interface ManualBodyCompositionInput {
  memberId: number;
  measuredAt: string;       // ISO date
  weight: number;
  bodyFatPct: number;
  muscleMass: number;
  bmi: number;
  visceralFat?: number;
  skeletalMuscleMass?: number;
  basalMetabolicRate?: number;
  bodyWater?: number;
  staffId: number;
}
```

## 7. 비즈니스 룰
1. 체중·체지방률·근육량·BMI 필수 입력
2. 측정일 미래 날짜 불가
3. 같은 날 중복 등록 시 확인 모달 (DLG-M016 패턴)
4. 저장 후 SCR-I006 목록 갱신
5. 담당자 staffId 자동 기록
