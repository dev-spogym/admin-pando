## 8. 비즈니스 룰
1. D-7 이내 → `expiring-soon` 자동 전이 + 회원 알림
2. 만료 경과 → `overdue` + 회수 대상 배지
3. 연장 시 기존 만료일 기준 + N개월
4. 회수 시 락커 상태 복구, 보증금 반환 처리(매출 도메인 연계)
5. 월 요금 미납 시 `overdue` 처리
6. 신규 배정 시 결제 연동 (D03 매출 POS)
7. 멀티테넌트 branchId
