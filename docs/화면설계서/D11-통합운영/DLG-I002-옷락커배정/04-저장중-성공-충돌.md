## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind v4 기반 'use client' 컴포넌트를 작성하라.
화면: DLG-I002 옷락커배정 — 상태: 저장 중 / 결과 분기

파일: src/components/locker/LockerAssignModal.tsx

submitAssign 함수:
async function submitAssign() {
  setIsSubmitting(true)
  try {
    const { data, error } = await supabase
      .from('clothing')
      .insert({
        locker_id: selectedLocker!.id,
        member_id: memberId,
        branch_id: branchId,
        status: 'active',
        assigned_at: new Date().toISOString(),
        assigned_by: userId,
      })
      .select('*, lockers(*)')
      .single()

    if (error?.code === '23505') {  // 409 충돌
      setConflictError('해당 락커는 이미 배정되었습니다. 다른 락커를 선택해 주세요.')
      setSelectedLocker(null)
      // 빈 락커 목록 재조회
      await loadAvailableLockers()
      return
    }
    if (error) throw error

    // 성공
    toast.success(`락커 ${selectedLocker!.number}번이 ${memberName}님께 배정되었습니다`)
    onSuccess(data)
    onClose()
  } catch (e) {
    toast.error('락커 배정에 실패했습니다')
  } finally {
    setIsSubmitting(false)
  }
}

[저장 중 버튼]
<button disabled={isSubmitting}
  className="flex-1 px-4 py-2 bg-blue-600 text-white text-sm font-medium rounded-lg disabled:opacity-60 flex items-center justify-center gap-2">
  {isSubmitting ? <><svg className="animate-spin w-4 h-4" /> 배정 중...</> : '배정 확인'}
</button>

[충돌 에러 UI]
{conflictError && (
  <div className="mt-3 p-3 bg-amber-50 border border-amber-200 rounded-lg">
    <p className="text-sm text-amber-700">{conflictError}</p>
    <p className="text-xs text-amber-500 mt-1">빈 락커 목록이 갱신되었습니다. 다른 락커를 선택해 주세요.</p>
  </div>
)}

[성공 처리 — 부모 컴포넌트 연동]
onSuccess 콜백:
- SCR-I004: 락커 그리드 셀 상태 변경 (available → occupied)
- SCR-I001: KPI 갱신 (optional)
- SCR-I007: 락커 카드 갱신 (미배정 → 배정됨)
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- 검증 통과 후 배정 확인 클릭

### API 응답 분기 (M3 플로우)
| 응답 | 처리 |
|------|------|
| 200 OK | 성공 토스트 + 모달 닫힘 + 부모 갱신 |
| 409/23505 충돌 | 충돌 경고 + selectedLocker 초기화 + 목록 재조회 |
| 500 | 에러 토스트 + 모달 유지 |

### 부모 컴포넌트 갱신
- SCR-I004 그리드: 해당 셀 occupied 스타일
- SCR-I004 KPI: 사용중 +1, 빈 락커 -1
- SCR-I007: 락커 카드 갱신
