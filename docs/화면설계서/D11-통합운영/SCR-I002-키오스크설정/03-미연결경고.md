## 📝 디스크립션

### 사용 시점 (Trigger)
- `/api/kiosk/ping` 응답 실패 (timeout 또는 오류)
- 30초 폴링에서 연결 끊김 감지
- 키오스크 장치 전원 OFF

### 필수 데이터
- kioskStatus: 'disconnected' | 'offline'
- 마지막 정상 연결 시각

### 인터랙션 (User Actions)
1. "연결 테스트" 버튼 → ping 재시도
2. 설정 폼 수정/저장 가능 (경고 표시 후 진행)
3. 연결 복구 자동 감지 후 배너 제거

### 비즈니스 룰
- 미연결이어도 설정값 저장 가능 (DB 저장, 키오스크 재연결 시 적용)
- 저장 시 경고 모달: "키오스크 미연결 상태입니다. 계속 저장하시겠습니까?"
- 연결 복구 감지 시: toast.success('키오스크 연결 복구') + 배너 제거

### 에지 케이스
- 장시간 미연결: 배너 유지, "마지막 연결: N시간 전" 표시
- 네트워크 정상이나 키오스크 앱 다운: 동일 미연결 처리
