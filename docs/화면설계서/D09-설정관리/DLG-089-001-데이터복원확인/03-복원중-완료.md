## 🧩 바이브코딩 프롬프트

```
복원 실행 및 진행 UI (M3_089001_02~03):

handleRestore: async () => {
  setIsRestoring(true)
  const { data, error } = await fetch(`/api/backup/${targetBackup.id}/restore`, {
    method: 'POST',
    body: JSON.stringify({ scope: restoreScope })
  })

  if (error?.status === 400) {
    toast.error('유효하지 않은 백업 파일입니다.')
    setIsRestoring(false)
    return
  }
  if (error) {
    toast.error('복원 실패. 데이터가 롤백되었습니다.')
    setIsRestoring(false)
    return
  }

  // 진행률 폴링 (5초 간격)
  const pollProgress = setInterval(async () => {
    const { progress } = await getRestoreProgress(data.jobId)
    setRestoreProgress(progress)
    if (progress >= 100) {
      clearInterval(pollProgress)
      // M3_089001_03: 복원 완료
      toast.success('복원이 완료되었습니다.')
      setShowRestoreModal(false)
      window.location.reload()  // 시스템 재로딩
    }
  }, 5000)
}

복원중 모달 UI:
<ModalBody>
  <div className="space-y-3">
    <p className="text-sm text-gray-600">복원을 진행 중입니다. 잠시 기다려 주세요.</p>
    <div className="h-2 w-full bg-gray-100 rounded-full">
      <div
        className="h-full bg-primary rounded-full transition-all"
        style={{ width: `${restoreProgress}%` }}
      />
    </div>
    <p className="text-sm text-center text-gray-500">{restoreProgress}%</p>
  </div>
</ModalBody>
닫기 버튼: 복원중 비활성 (ESC 차단)
```
