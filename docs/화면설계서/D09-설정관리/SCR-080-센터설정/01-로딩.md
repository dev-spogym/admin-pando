## 상태 고유 비즈니스 룰 델타
- 쿼리 타임아웃 10초 → `05-에러` 전환(E500001 with `retryable=true`)
- 최초 진입 시 7일 온보딩 배너는 `02-정상` 진입 후 표시

## 에지 케이스
- **매우 빠른 응답 (<150ms)**: 스켈레톤 플래시 방지 → `skeletonMinDelay=150ms` 가드 후 렌더
- **느린 응답 (>3s)**: 하단에 "네트워크가 느립니다" 힌트 노출(`role="status"`)
- **오프라인 전환**: 로딩 중 오프라인 감지 → 즉시 `07-오프라인`으로
