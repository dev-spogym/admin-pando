## 상태 고유 인터랙션
1. **저장하기 클릭** → `04-저장중`
2. **취소 클릭** → `form.reset(data)` → `02-정상`
3. **탭 클릭 (다른 탭)** → `DLG-080-001` 미저장 경고
4. **사이드바 클릭** → `DLG-080-001` (라우터 가드)
5. **BranchSwitcher 변경** → `DLG-080-001`
6. **Ctrl/Cmd+S** → 저장 단축키
7. **Esc** → 취소 버튼(선택)
8. **추가 수정** → 계속 dirty 유지

## 상태 고유 비즈니스 룰 델타
- `pickDirty()`로 변경된 필드만 diff 저장
- 탭별 변경 추적: `dirtyTabs = Set<TabKey>` → 탭 배지
- 저장 버튼 키보드 순서: 현재 필드 → 취소 → 저장
- 충돌 감지: 저장 시 서버가 `updatedAt` 불일치 → 409 E900007

## 에지 케이스
- **원본 값으로 복귀**: RHF isDirty 유지 → 수동 `form.reset(currentValues)` 가능
- **빈 문자열 vs null**: 서버에서 빈 문자열 → null 치환
- **여러 탭 동시 변경**: 모든 변경 필드 함께 PATCH

## 🧩 바이브코딩 프롬프트 (델타)

```
상속: 00-기본화면.md §14 프롬프트 전체
상태: settings-dirty (변경됨)

이 상태에서만 적용할 변경점:
- isDirty = true
- DirtyBadge 표시
- SavedBadge 강제 false
- 저장/취소 버튼 활성

DirtyBadge:
<div role="status" aria-live="polite"
     className="inline-flex items-center gap-1.5 h-8 px-3 rounded-full text-xs font-medium
                bg-amber-50 text-amber-800 ring-1 ring-amber-200">
  <AlertTriangle className="size-3.5" />
  저장되지 않은 변경사항이 있습니다
</div>

탭 배지:
const dirtyTabs = useMemo(() => computeDirtyTabs(dirtyFields), [dirtyFields]);
<button role="tab">
  <t.Icon className="size-4 mr-1" />
  {t.label}
  {dirtyTabs.has(t.key) && <span className="size-2 rounded-full bg-amber-500 ml-1.5" aria-label="변경됨" />}
</button>

탭 전환 가드:
function tryChangeTab(next) {
  if (isDirty) { setPendingTab(next); setShowUnsavedDialog(true); }
  else setActiveTab(next);
}

라우터 가드:
useEffect(() => {
  if (!isDirty) return;
  const origPush = router.push;
  router.push = ((to, opts) => {
    if (isDirty) { setPendingHref(to); setShowUnsavedDialog(true); return; }
    return origPush(to, opts);
  }) as typeof router.push;
  return () => { router.push = origPush; };
}, [isDirty]);

beforeunload:
useEffect(() => {
  const h = (e: BeforeUnloadEvent) => { if (isDirty) { e.preventDefault(); e.returnValue = ''; } };
  window.addEventListener('beforeunload', h);
  return () => window.removeEventListener('beforeunload', h);
}, [isDirty]);

Ctrl/Cmd+S:
useEffect(() => {
  const h = (e: KeyboardEvent) => {
    if ((e.ctrlKey || e.metaKey) && e.key === 's') {
      e.preventDefault();
      if (isDirty && !isSaving) handleSave();
    }
  };
  window.addEventListener('keydown', h);
  return () => window.removeEventListener('keydown', h);
}, [isDirty, isSaving]);
```
