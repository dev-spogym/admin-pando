## 비즈니스 룰
- `useQueries` 병렬 fetch 3개 (roles, rolePermissions, assignedUsers)
- 첫 렌더 시 `selectedRoleId` = 첫 번째 비-primary 시스템 역할(owner)
- 타임아웃 10초 → `07-에러` 전환

## 에지 케이스
- **빠른 응답 (<150ms)**: skeletonMinDelay 가드
- **partial success**: roles만 200, permissions 실패 → 역할 패널만 렌더 + 매트릭스 영역 에러 박스
