## 🧩 바이브코딩 프롬프트

```
충돌 감지 로직:
detectConflicts(matrix: PermMatrix): ConflictItem[] {
  const conflicts: ConflictItem[] = []
  Object.entries(matrix).forEach(([role, features]) => {
    Object.entries(features).forEach(([feature, perms]) => {
      // 삭제 권한 ON + 쓰기 권한 OFF → 충돌
      if (perms.delete && !perms.write) {
        conflicts.push({ role, feature, type: 'delete_without_write' })
      }
    })
  })
  return conflicts
}

저장 핸들러에서:
const conflicts = detectConflicts(permMatrix)
if (conflicts.length > 0) {
  setConflicts(conflicts)
  setShowConflictModal(true)  // → 02-열림
  return
}
```
