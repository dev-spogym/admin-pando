### 변경 요소 표

| 요소 | 변경 |
|---|---|
| StatCard 5장 | 모두 값 0, `variant='default'`로 회색 단조 톤 |
| 상태별 분포 섹션 | 완전히 숨김 (조건부 렌더) |
| Toolbar | 숨기거나 전체 disabled (검색 무의미) |
| DataTable | EmptyState 카드로 대체 |
| EmptyState | 중앙 정렬, 아이콘 + 제목 + 설명 + CTA |

## 역할별 차이

| 역할 | EmptyState CTA |
|---|---|
| superAdmin/primary/owner/manager/fc | `[+ 리드 등록하기]` 버튼 활성 |
| readonly | CTA 숨김, 안내 문구만 |
| trainer/staff/front | 이 상태 도달 불가(`06-권한없음`으로 분기) |

## 상태 고유 인터랙션
1. **[+ 리드 등록하기] 클릭** → `DLG-070-001` 등록 모드 오픈
2. **등록 성공** → `02-정상-데이터있음` 전환 + toast
3. **페이지 상단 [+ 리드 등록]** 버튼도 동일 동작
4. **엑셀 가져오기 링크**(Phase 2) → 파일 업로드 다이얼로그

## 상태 고유 비즈니스 룰 델타
- StatCard 값은 `0`으로 표시 (null/"-" 금지)
- 전환율/미전환율: `0%` 고정 (분모 0이면 NaN 방지)
- 상태별 분포 섹션은 `total > 0` 조건으로만 렌더
- EmptyState의 `[+ 리드 등록하기]`는 페이지 상단 버튼과 **동일한 `openCreate()` 호출**
- 읽기 전용 계정(`readonly`)은 CTA 버튼 자체를 렌더하지 않고 메시지만 표시

## 에지 케이스
- **필터 적용 후 0건**: 이 상태 아님 → `02-정상`에서 "검색 결과가 없습니다" 표시
- **새 지점 첫 생성**: 멀티테넌트 신규 지점의 경우 자동으로 이 상태
- **모든 리드 삭제 직후**: 마지막 리드 DELETE 성공 → 즉시 이 상태로 전환
- **네트워크 복구 직후 0건**: 오프라인에서 복귀했는데 DB에도 0건인 경우 → 이 상태 (오프라인 아님)
