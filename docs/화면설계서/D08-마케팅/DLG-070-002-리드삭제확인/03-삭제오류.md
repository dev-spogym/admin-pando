```
삭제 오류 처리 (LeadDeleteModal.tsx):

구현:
catch (err: any) {
  if (err.status === 404) {
    toast.error("이미 삭제된 리드입니다.")
    onClose()
  } else {
    toast.error("삭제에 실패했습니다.")
  }
  setIsDeleting(false)
}
```
