## 📝 디스크립션

### 사용 시점 (Trigger)
- SCR-P004 할인 규칙 목록에서 행의 "수정" 버튼 클릭
- 기존 규칙 데이터 프리필

### 필드 정의

| 필드 | 타입 | 필수 | 기본값 |
|------|------|------|--------|
| 규칙명 | text | Y | rule.name |
| 할인 유형 | Select | Y | rule.type |
| 할인 값 | number | Y | rule.value |
| 적용 조건 | Select | Y | rule.condition |
| 적용 기간 | DateRangePicker | 조건부 | rule.start_date ~ rule.end_date |
| 적용 상품 | MultiSelect | N | rule.product_ids |

### 인터랙션 (User Actions)
1. 필드 수정 후 "저장" 클릭 → UPDATE → `03-저장결과`
2. "취소" / X → 변경사항 폐기
