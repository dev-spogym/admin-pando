```
파일: src/components/product/ProductCreateModal.tsx (validation error)

유효성 검사:
  const validate = () => {
    const errs: Record<string, string> = {}
    if (!form.name.trim()) errs.name = '상품명을 입력해 주세요.'
    if (!form.price || Number(form.price) <= 0) errs.price = '올바른 판매가를 입력해 주세요.'
    return errs
  }

에러 표시:
  - 각 필드 하단에 <p className="text-red-500 text-xs mt-1">{errors.필드명}</p>
  - 에러 있는 input에 border-red-500 클래스 적용

등록 클릭 시:
  const handleSubmit = () => {
    const errs = validate()
    if (Object.keys(errs).length > 0) {
      setErrors(errs)
      return
    }
    // 정상 처리
  }
```
