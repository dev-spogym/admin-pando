- `can(role,'create')` false
- 지점에 아직 상품이 등록되지 않음

## 역할별 차이

| 역할 | 표시 |
|---|---|
| manager | 기본 안내 + "분류 관리로 이동" 보조 링크 (조회 가능) |
| fc / staff / front / trainer | 기본 안내만 |
| readonly | 기본 안내 + 푸터에 "읽기 전용 계정" 배지 |

## 상태 고유 인터랙션
- 클릭 가능한 영역: **분류 관리 탭 전환**, **Excel**(빈 다운로드 방지 → 에러 토스트), **필터/검색**(의미 없지만 허용).
- CTA 버튼 없음. 관리자에게 안내만.

## 비즈니스 룰 델타
- Excel 클릭 시 `toast.error('내보낼 상품이 없습니다.')`.
- 전지점배포/등록 버튼은 완전 비렌더 (display 제거, 스페이스 차지 X).
- `sendOnboardingRequest(branchId)` API는 여기에서 자동 트리거 안 함(수동).
- 분류 탭 진입은 허용(조회).

## 에지 케이스
- manager가 임시로 권한 변경되어 상품이 추가됐으나 UI는 아직 refetch 전 → `staleTime` 만료 시 `02-정상` 전환.
- 분류는 존재하나 상품만 0 → 분류 탭은 정상 렌더.
- readonly 계정이 딥링크로 진입 → 동일 화면 + 읽기 전용 배지.
