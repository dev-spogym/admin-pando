    const file = e.target.files?.[0]

    if (file.size > 5 * 1024 * 1024) {
      toast.error('5MB 이하 파일만 업로드 가능합니다.')
      return
    }
    setPreview(URL.createObjectURL(file))
    setSelectedFile(file)

  const { data, error } = await supabase.storage

## 📝 디스크립션

### 사용 시점 (Trigger)
- 상품 등록(SCR-P002) 또는 상세 패널(SCR-P003)에서 "이미지 업로드" 클릭

### 레이아웃 구성
- 드래그앤드롭 + 클릭 업로드 지원
- 파일 선택 시 즉시 미리보기
- 기존 이미지 있을 경우 현재 이미지 표시

### 파일 제약

| 항목 | 제약 |
|------|------|
| 형식 | JPG, PNG, WEBP |
| 크기 | 최대 5MB |
| 개수 | 1개 |

### 인터랙션 (User Actions)
1. 드래그앤드롭 또는 클릭으로 파일 선택
2. 미리보기 확인 후 "업로드" 클릭 → Storage 업로드 → `02-업로드결과`
3. "취소" / X → 닫기

### 비즈니스 룰
- 업로드 성공 시 products.image_url 자동 업데이트
- 5MB 초과 시 선택 즉시 toast.error 표시

### 연결 화면
- 업로드 결과: `02-업로드결과`
- 취소: SCR-P002/P003 복귀
