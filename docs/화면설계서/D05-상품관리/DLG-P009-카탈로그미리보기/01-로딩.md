## 상태 고유 인터랙션
- 로딩 중 디바이스 재클릭: 이전 요청 `abort` + 새 요청
- 로딩 중 ESC: 즉시 모달 언마운트 (쿼리 cancel)
- `staleTime` 경과 전 재오픈: 캐시 hit → 로딩 스킵하고 `02-시뮬표시`

## 비즈니스 룰 델타
- Skeleton 최소 노출 시간 150ms (플리커 방지)
- 3초 초과 시 하단에 `"로딩이 지연되고 있습니다..."` 힌트 출력
- 10초 초과 시 `04-에러` 상태로 강제 전이 (타임아웃)

## 에지 케이스
- super가 지점 미선택 상태에서 오픈 → `enabled:false` 로 `01-로딩` 대신 "지점을 먼저 선택하세요" 메시지
- iframe 프리로드 실패: `onError` 발생 시 즉시 `04-에러`
