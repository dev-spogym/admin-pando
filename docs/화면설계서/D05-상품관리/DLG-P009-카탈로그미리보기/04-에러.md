# DLG-P009 카탈로그 미리보기 — 04 에러 (델타)

| 이전 상태 | `01-로딩` |
| 다음 상태 | `01-로딩`(재시도) / Closed |

## 상태 진입 조건
- HTTP 4xx/5xx 응답
- 네트워크 오프라인 / 타임아웃(>10s)
- iframe `onError` 발생
- E403001 (super가 접근 불가 지점 선택)

## 역할별 차이
- 모든 역할 동일 UI. 단, E403001(지점 접근 불가)은 super/primary에게만 발생 (드롭다운 오선택).

## 상태 고유 인터랙션
- "다시 시도" → `01-로딩` 전이 (queryClient.invalidateQueries + refetch)
- "닫기" / ESC / 배경 → 모달 닫힘
- 5초 내 연속 실패 3회: "문제가 지속되면 관리자에게 문의하세요" 헬퍼 추가

## 비즈니스 룰 델타
- 에러 토스트는 띄우지 않음 (모달 내부 상태로 충분히 전달)
- 에러 상태 유지 중 디바이스 전환은 가능 (UI 레이아웃은 프리뷰 시도 가능)
- 에러 코드별 메시지 매핑:
  - `E403001` → "해당 지점에 접근할 수 없습니다."
  - `E404301` → "공개 상품이 조회되지 않습니다."
  - `E5xx001` → "서버 오류가 발생했습니다."
  - `E-NET`   → "네트워크를 확인해주세요."
  - `E-TIMEOUT` → "응답이 지연되고 있습니다. 다시 시도해주세요."
- E401001/E401002는 DLG-000 세션만료가 글로벌로 가로채므로 본 모달에서는 다루지 않음.

## 에지 케이스
- iframe HTML은 200이나 내부 XHR 500: iframe에서 자체 에러 UI → 모달은 `02-시뮬표시` 유지
- CSP 에러: onError 트리거로 `04-에러` 전이
- super가 primary 권한 없는 branchId 선택 시 E403001 → 메시지 스위칭

## 바이브코딩 프롬프트 (델타)

```
━━ 델타: PreviewErrorState ━━
import { AlertTriangle } from 'lucide-react';

const MSG: Record<string, string> = {
  E403001: '해당 지점에 접근할 수 없습니다.',
  E404301: '공개 상품이 조회되지 않습니다.',
  E5xx001: '서버 오류가 발생했습니다.',
  'E-NET': '네트워크를 확인해주세요.',
  'E-TIMEOUT': '응답이 지연되고 있습니다. 다시 시도해주세요.',
};

export function PreviewErrorState({ code, onRetry, onClose, retryCount }: {
  code: string; onRetry: () => void; onClose: () => void; retryCount: number;
}) {
  return (
    <div role="alert" className="flex flex-col items-center justify-center text-center px-6 py-10 max-w-sm">
      <AlertTriangle className="size-12 text-red-500" aria-hidden />
      <h3 className="mt-4 text-base font-semibold text-red-700">미리보기를 불러오지 못했습니다.</h3>
      <p className="mt-2 text-sm text-gray-600 leading-relaxed">
        {MSG[code] ?? '알 수 없는 오류가 발생했습니다.'}
      </p>
      <code className="mt-2 text-xs text-gray-500 font-mono">errorCode: {code}</code>
      {retryCount >= 3 && (
        <p className="mt-3 text-xs text-gray-500">문제가 지속되면 관리자에게 문의하세요.</p>
      )}
      <div className="mt-5 flex items-center gap-2">
        <button onClick={onRetry}
          className="h-10 px-4 rounded-lg bg-red-600 hover:bg-red-700 text-white text-sm font-medium
                     focus:outline-none focus:ring-2 focus:ring-offset-2 focus:ring-red-500">
          다시 시도
        </button>
        <button onClick={onClose}
          className="h-10 px-4 rounded-lg border border-gray-300 text-gray-700 text-sm font-medium hover:bg-gray-50">
          닫기
        </button>
      </div>
    </div>
  );
}

━━ 사용 ━━
const [retryCount, setRetryCount] = useState(0);
useEffect(() => { if (status === 'error') setRetryCount(n => n + 1); }, [status]);
useEffect(() => { if (status === 'success') setRetryCount(0); }, [status]);

{viewState === 'error' && (
  <PreviewErrorState
    code={(error as any)?.response?.data?.errorCode ?? 'E-NET'}
    retryCount={retryCount}
    onRetry={() => { setRetryCount(0); refetch(); }}
    onClose={onClose}
  />
)}
```
