## 📝 디스크립션

### 사용 시점 (Trigger)
- `E_M1_OPEN_01`: SCR-050 in_use 셀 → "회수" 버튼 클릭

### 필수 데이터
- 락커 번호, 배정 회원명, 만료일

### 인터랙션 (User Actions)
1. "회수 확인" 클릭 → 회수 API 호출
2. "취소" / ESC → 모달 닫힘

### 비즈니스 룰
- 만료일 전 회수도 가능 (강제 회수)
- 회수 후 locker.status = 'available', member_id = null
