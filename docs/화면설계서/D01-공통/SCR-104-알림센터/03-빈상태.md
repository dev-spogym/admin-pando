### 탭별 메시지 분기

| 탭 | 아이콘 | 제목 | 부제 | CTA |
|---|---|---|---|---|
| `all` | `BellOff` | 새 알림이 없습니다 | 중요한 이벤트가 발생하면 이곳에 표시됩니다. | "알림 설정 열기" → /profile?tab=notifications |
| `unread` | `CheckCircle2` | 모두 읽으셨습니다 | 미확인 알림이 없습니다. 잘하고 계세요! | "전체 알림 보기" → activeTab='all' |
| `read` | `Inbox` | 읽은 알림이 없습니다 | 알림을 읽으면 이곳에 누적됩니다. | "미읽음 보기" → activeTab='unread' |

## 상태 고유 인터랙션

1. **"알림 설정 열기" 클릭** (탭=all) → `router.push('/profile?tab=notifications')`
2. **"전체 알림 보기" 클릭** (탭=unread) → `setActiveTab('all')`
3. **"미읽음 보기" 클릭** (탭=read) → `setActiveTab('unread')`
4. **"모두 읽음" 버튼** → `disabled`
5. **실시간 INSERT** → `05-실시간수신` 전환 → 이후 `02-정상`

## 상태 고유 비즈니스 룰 델타

- **tab별 맞춤 메시지**: `activeTab` 기준 제목/부제/CTA 분기
- **"모두 읽음" 버튼 조건부 비활성화**: `unreadCount === 0`이면 `disabled={true}`
- **빈 상태에서도 실시간 구독 유지**: WebSocket 채널 활성
- **CTA는 선택 노출**: Phase 2에서 제거 가능(정책 옵션)

## 에지 케이스

- **전체 삭제 직후 → 빈 상태 전환 시 애니메이션**: `animate-[fadeIn_300ms_ease-out]`
- **탭별 빈 상태 카운트**: `total>0 && filter==='unread' && unreadCount===0` → "모두 읽으셨습니다" 메시지
- **아직 한 번도 알림 발생 안 한 신규 가입자** → 전체 탭 기본 빈 상태 ("알림 설정 열기" CTA 강조)
- **네트워크 회복 직후 0건** → 일반 빈 상태와 동일 처리

## 🧩 바이브코딩 프롬프트 (델타)

```
상속: 00-기본화면.md §14 프롬프트 전체 적용
상태: notification-empty (빈 상태)

이 상태에서만 적용할 변경점:
- items.length === 0 (모든 탭 0건 또는 현재 탭 0건)
- Pagination 렌더 X
- "모두 읽음" disabled (unreadCount === 0)

NotificationEmptyState 컴포넌트:
export function NotificationEmptyState({ tab }: { tab: 'all'|'unread'|'read' }) {
  const router = useRouter();
  const setActiveTab = useActiveTab();
  const variant = {
    all:    { icon: BellOff,      title: '새 알림이 없습니다',
              sub:  '중요한 이벤트가 발생하면 이곳에 표시됩니다.',
              cta:  { label: '알림 설정 열기', onClick: () => router.push('/profile?tab=notifications') } },
    unread: { icon: CheckCircle2, title: '모두 읽으셨습니다',
              sub:  '미확인 알림이 없습니다. 잘하고 계세요!',
              cta:  { label: '전체 알림 보기', onClick: () => setActiveTab('all') } },
    read:   { icon: Inbox,        title: '읽은 알림이 없습니다',
              sub:  '알림을 읽으면 이곳에 누적됩니다.',
              cta:  { label: '미읽음 보기',   onClick: () => setActiveTab('unread') } },
  }[tab];
  const Icon = variant.icon;
  return (
    <div role="status" aria-live="polite"
         className="flex flex-col items-center justify-center rounded-xl border border-dashed border-gray-200 bg-white py-16 px-6 text-center">
      <Icon className="size-16 text-gray-300 mb-4" aria-hidden />
      <h3 className="text-base font-medium text-gray-700">{variant.title}</h3>
      <p className="mt-1 text-sm text-gray-400 max-w-xs">{variant.sub}</p>
      {variant.cta && (
        <Button variant="ghost" size="sm" className="mt-4" onClick={variant.cta.onClick}>
          {variant.cta.label}
        </Button>
      )}
    </div>
  );
}

"모두 읽음" 버튼 조건:
<Button disabled={unreadCount === 0 || isMarkingAll}
        aria-disabled={unreadCount === 0 || isMarkingAll}>모두 읽음</Button>
```
