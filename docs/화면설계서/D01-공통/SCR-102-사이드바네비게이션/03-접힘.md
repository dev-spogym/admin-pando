## 📝 디스크립션

### 사용 시점 (Trigger)
- `02-펼침`에서 접기 버튼 클릭
- localStorage에 접힘 상태 저장 → 재방문 시 유지

### 필수 데이터
- menuItems (동일, 아이콘만 표시)
- isCollapsed = true (localStorage 유지)

### 인터랙션 (User Actions)
1. 아이콘 클릭 → 해당 화면 이동
2. 아이콘 호버 → 메뉴명 tooltip 표시
3. 펼치기(ChevronRight) 버튼 → `02-펼침`

### 비즈니스 룰
- 접힘 상태에서도 역할별 메뉴 필터링 동일 적용
- 메인 콘텐츠 영역 너비 확장 (w-[calc(100%-4rem)])

### 에지 케이스
- 모바일 뷰포트(< 768px) → 사이드바 오버레이 방식으로 전환 (별도 처리)

### 접근성 (A11y)
- `aria-expanded="false"` on 펼치기 버튼
- 각 아이콘 `aria-label="{메뉴명}"` (텍스트 숨겨져 있으므로 필수)
- tooltip `role="tooltip"` `id` 연결
