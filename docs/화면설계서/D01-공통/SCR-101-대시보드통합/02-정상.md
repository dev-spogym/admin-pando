## 상태 고유 인터랙션

1. **StatCard 클릭** → 해당 상세 페이지(`/members?filter=...`, `/sales`, `/attendance` 등)
2. **리스트 항목 클릭** → `/members/detail?id={id}` (readOnly 역할은 비활성 + toast)
3. **"전체보기" 링크** → 필터 메뉴(`/members?filter=unpaid|holding|expiring|birthday`)
4. **차트 호버** → 툴팁(값 + 월/요일/카테고리)
5. **BranchSwitcher 전환** (super/primary만) → 전체 refetch + URL `?branch=` + toast
6. **PeriodFilter 변경** (today/week/month) → stats·차트만 refetch
7. **새로고침 버튼** → `runDailySync` + 전체 refetch, 60초 쿨다운
8. **엑셀 내보내기** → POST `/dashboard/export-excel` → `toast.success(KPI 엑셀 다운로드 완료)`
9. **레이아웃 설정** → 모달 오픈 → 위젯 on/off·순서 저장 → `localStorage`
10. **감사로그 행 클릭** → `/audit-log?id={id}` 또는 대상 엔티티
11. **퀵액션 클릭** → `/attendance`, `/members/new`, `/classes/new`

## 상태 고유 비즈니스 룰 델타

- **변화율 계산**: `delta = prev>0 ? ((curr-prev)/prev*100) : 0`, `formatDelta(d)` → `"+5.3%"` / `"-1.2%"` / `"0.0%"`
- **매출 포맷**: 1억+ `"X.X억"`, 1천만+ `"N,NNN만"`, 그 외 천단위 콤마 + "원"
- **숫자 포맷**: `Intl.NumberFormat('ko-KR')` + 단위("명","건")
- **D-Day**: `Math.ceil((expiry - today)/day) = N` → `D-${N}`
- **홀딩 남은 기간**: 동일 로직, 음수는 "만료"
- **생일자**: `MM-DD` 일치 + `status='ACTIVE'` + 만 나이 계산
- **미수금 연체일**: `today - dueDate` (일 단위)
- **감사로그 상대시간**: `formatDistanceToNowStrict` ko locale
- **refetchOnWindowFocus**: 60초 내 재포커스 시 자동 refetch(조용히)

## 에지 케이스

- **trainer 담당 회원 0명**: KPI는 0명 표시, 리스트 위젯은 해당 위젯만 빈 상태 (03-빈상태 규칙 적용)
- **front 권한 제한**: §C 1개만 노출되어도 정상
- **멀티 지점 owner**: BranchSwitcher 노출(다지점 보유 시), 기본 지점 자동 선택
- **차트 데이터 전부 0**: 차트 내부에서 "데이터 없음" 빈 상태
- **부분 실패**: 일부 위젯 에러 → 해당 위젯만 에러 UI, 나머지 정상 유지
- **refetchOnWindowFocus 중 페이지 이탈**: 쿼리 cancel
- **IoT realtime error 발생**: 06-IoT장애로 동적 전환 (배너만 추가, 나머지는 정상 유지)
