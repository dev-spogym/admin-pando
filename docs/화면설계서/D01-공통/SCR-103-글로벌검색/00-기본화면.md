## 7. 데이터 계약

```ts
interface SearchResult {
  id: string;
  type: 'member'|'product'|'staff'|'sale'|'class'|'setting';

  subtitle?: string;
```

## 8. 비즈니스 룰
1. 최소 2글자 입력 후 검색 시작 (debounce 200ms)
2. 카테고리별 최대 5건씩 표시
3. 역할별 접근 불가 카테고리 결과 제외
4. 최근 검색어 localStorage 저장 (최대 10개)
5. Cmd+K: 검색창 포커스, ESC: 닫기
6. 키보드 ↑↓: 결과 탐색, Enter: 이동

## 11. 접근성
- `role="dialog" aria-modal="true" aria-label="글로벌 검색"`
- 검색 입력 `role="combobox" aria-expanded aria-controls="search-listbox"`
- 결과 목록 `role="listbox" id="search-listbox"`
- 결과 항목 `role="option" aria-selected`
- 키보드 완전 지원 (↑↓, Enter, ESC)

## 12. 진입/이탈
- 진입: `Cmd+K`, 헤더 검색 아이콘
- 이탈: ESC, 결과 선택(해당 페이지로 이동), 오버레이 클릭

```
  const handler = (e: KeyboardEvent) => {
    if ((e.metaKey || e.ctrlKey) && e.key === 'k') {

    }

  if (e.key === 'Enter')     handleSelect(results[selectedIdx]);
```
