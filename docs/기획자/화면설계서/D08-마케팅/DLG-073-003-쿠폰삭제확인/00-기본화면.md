## 목적

쿠폰을 삭제하기 전 최종 확인을 받는 다이얼로그입니다. 이미 발급된 쿠폰이 있는 경우 삭제 영향 범위를 미리 안내합니다.

## 구성요소

- **제목**: "쿠폰 삭제"
- **안내 메시지**: "이 쿠폰을 삭제하시겠습니까? 삭제된 쿠폰은 복구할 수 없습니다."
- **발급 현황 경고**: 이미 발급된 쿠폰이 있을 경우 "N명에게 발급된 쿠폰이 있습니다. 삭제 시 미사용 쿠폰도 함께 무효화됩니다." 경고가 추가로 표시됩니다.
- **삭제 버튼 (위험)**: 쿠폰을 최종 삭제합니다.
- **취소 버튼**: 다이얼로그를 닫고 삭제를 취소합니다.

## 언제 표시되나요

- 쿠폰 관리(SCR-073)에서 쿠폰의 삭제 버튼을 클릭할 때 표시됩니다.

## 주요 흐름

1. 담당자가 삭제할 쿠폰의 삭제 버튼을 클릭합니다.
2. 이 다이얼로그가 표시됩니다. 발급 현황에 따라 경고 메시지가 추가됩니다.
3. "삭제"를 클릭하면 쿠폰이 영구 삭제되고 이미 발급된 미사용 쿠폰도 무효화됩니다.
4. "취소"를 클릭하면 다이얼로그가 닫히고 쿠폰이 유지됩니다.
