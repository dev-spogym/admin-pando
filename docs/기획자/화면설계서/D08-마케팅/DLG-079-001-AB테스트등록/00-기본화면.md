## 목적

두 가지 버전(A안·B안)의 메시지 또는 쿠폰을 비교하는 A/B 테스트를 설정하는 다이얼로그입니다. 대상 분배 비율과 성과 측정 기준을 설정해 데이터 기반 마케팅 의사결정을 지원합니다.

> **현재 상태**: 미구현 기능입니다.

## 구성요소

- **테스트명** (필수): A/B 테스트를 식별하는 이름을 입력합니다.
- **테스트 유형**: 메시지 비교 또는 쿠폰 비교 중 선택합니다.
- **A안 설정**: A버전의 메시지 내용 또는 쿠폰 조건을 설정합니다.
- **B안 설정**: B버전의 메시지 내용 또는 쿠폰 조건을 설정합니다.
- **대상 분배 비율**: A안과 B안에 각각 몇 %의 수신자를 배정할지 설정합니다. (예: 50:50)
- **테스트 기간**: 시작일과 종료일을 설정합니다.
- **성과 측정 기준**: 클릭율 / 전환율 / 매출 기여 중 비교 기준을 선택합니다.
- **저장 버튼**: 테스트를 등록하고 다이얼로그를 닫습니다.
- **취소 버튼**: 등록하지 않고 닫습니다.

## 언제 표시되나요

- A/B 테스트 화면(SCR-079)에서 "테스트 등록" 버튼을 클릭할 때 표시됩니다.

## 주요 흐름

1. 담당자가 SCR-079에서 "테스트 등록" 버튼을 클릭합니다.
2. 이 다이얼로그가 열립니다.
3. 테스트명, 유형, A안·B안 내용, 분배 비율, 기간을 설정합니다.
4. "저장"을 클릭하면 테스트가 등록되고 기간 시작 시 자동으로 실행됩니다.
