## 화면 목적

대규모 회원에게 SMS 또는 카카오 알림톡을 대량으로 발송하고, 발송 채널을 통합 관리하며 비용 현황을 추적하는 화면입니다. 개별 메시지 발송(SCR-071)과 달리 대용량 발송 전용 채널 설정과 발송 이력 관리에 특화됩니다.

> **현재 상태**: 미구현 화면입니다. 향후 개발 예정이며, 현재는 화면 접근이 제한됩니다.

## 주요 구성요소

- **채널 현황 카드**: SMS 발송 건수 / 카카오 알림톡 발송 건수 / 이번 달 총 비용 / 잔여 포인트를 표시합니다.
- **발송 대상 선택**: DLG-078-001을 통해 발송 대상 그룹(회원 등급·이용권 상태·지역 등)을 선택합니다.
- **메시지 템플릿 선택**: 사전 등록된 메시지 템플릿을 선택하거나 직접 작성합니다.
- **발송 예약**: 즉시 또는 특정 일시에 예약 발송합니다.
- **발송 확인**: DLG-078-002로 최종 발송 전 수신자 수와 비용을 확인합니다.
- **발송 이력 목록**: 발송 일시 / 채널 / 수신자 수 / 성공·실패 건수 / 비용 컬럼으로 이력을 조회합니다.
- **비용 현황 그래프**: 월별 발송 비용 추이를 시각화합니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 채널 현황 및 발송 이력 | 대량 발송, 예약 발송, 이력 조회 |
| FC / 트레이너 / 스태프 | 접근 제한 | 대량 발송 채널 관리 불가 |

## 주요 UX 흐름

1. 사이드바 "마케팅 > SMS/카카오 발송"으로 진입합니다.
2. 발송 대상 그룹을 선택합니다(DLG-078-001).
3. 메시지 내용을 작성하거나 템플릿을 선택합니다.
4. 발송 일시를 설정(즉시 또는 예약)합니다.
5. 최종 확인 다이얼로그(DLG-078-002)에서 수신자 수와 비용을 확인하고 발송합니다.
6. 발송 이력에서 결과를 확인합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 미구현 | 현재 화면이 개발되지 않은 상태입니다. "준비 중" 안내 페이지가 표시됩니다. |
| 발송 대기 | 대상·내용 설정 완료 후 발송 버튼 활성화 상태입니다. |
| 발송 중 | 대량 발송 처리 중 진행 표시가 나타납니다. |
| 이력 없음 | 발송 이력이 없을 때 안내 메시지가 표시됩니다. |

## 관련 다이어그램

- DLG-078-001 발송 대상 선택
- DLG-078-002 발송 확인
- SCR-071 메시지 발송 (소규모 발송)
