## 화면 목적

아직 회원으로 등록되지 않은 잠재 고객(문의·상담 대상)을 등록하고, 상담 단계별 진행 상황을 추적해 회원 전환율을 높이는 마케팅 도메인의 핵심 화면입니다. FC는 담당 리드의 후속 상담을 관리하고, 매니저는 전체 리드 전환율을 모니터링합니다.

## 주요 구성요소

- **요약 지표 카드 (5개)**: 전체 리드 수 / 등록 전환 수 / 전환율 / 대기 중(신규+연락완료) / 미전환율을 한눈에 표시합니다.
- **상태별 분포 카드**: 신규·연락완료·상담예정·방문완료·등록완료·미전환·보류 각 상태별 건수를 카드 형태로 표시합니다. (데이터가 0건인 상태는 숨김)
- **상태 필터**: 전체 또는 특정 상담 단계(7종)만 표시합니다.
- **유입경로 필터**: 간판·인터넷·전단지·추천·SNS·카카오톡·전화문의·방문·기타 중 선택합니다.
- **뷰 전환 버튼**: 목록 뷰(테이블)와 칸반 뷰(상태별 컬럼) 사이를 전환합니다.
- **리드 목록 테이블**: 문의일 / 이름 / 연락처 / 유입경로 / 상담 단계 / 담당 FC / 후속 예정일 / 메모 컬럼으로 구성됩니다. 수정·삭제 버튼 제공.
- **칸반 뷰**: 상담 단계를 컬럼으로 배치해 리드의 진행 상황을 한눈에 파악합니다.
- **리드 등록 버튼**: 신규 잠재 고객 등록 다이얼로그(DLG-070-001)를 엽니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 소속 센터 전체 리드 | 리드 등록·수정·삭제, 상태 변경, 칸반 뷰 |
| FC | 소속 센터 전체 리드 (담당 리드 우선) | 리드 등록·수정, 상태 변경 (삭제 불가) |
| 트레이너 / 스태프 | 접근 제한 | 마케팅 도메인 접근 불가 |

## 주요 UX 흐름

1. 사이드바 "마케팅 > 리드 관리"로 진입합니다.
2. 요약 카드와 상태별 분포로 전체 현황을 파악합니다.
3. 필터를 조합하거나 검색(이름·연락처·담당FC)으로 원하는 리드를 찾습니다.
4. 리드 행의 수정 버튼 클릭 시 DLG-070-001이 열려 정보를 수정합니다.
5. 상담 단계가 "등록완료"로 변경되면 해당 리드가 전환율 통계에 반영됩니다.
6. 불필요한 리드는 삭제 버튼 클릭 후 DLG-070-002에서 최종 확인합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 로딩 중 | 테이블 또는 칸반 영역에 로딩 표시가 나타납니다. |
| 정상 목록 | 리드 데이터가 테이블 또는 칸반으로 표시됩니다. |
| 리드 없음 | 필터 없이 0건이면 "등록된 리드가 없습니다" 안내가 표시됩니다. |
| 검색 결과 없음 | 필터·검색 후 0건이면 해당 안내 메시지가 표시됩니다. |
| 칸반 열 비어있음 | 해당 단계의 리드가 없으면 "없음" 텍스트가 표시됩니다. |

## 관련 다이어그램

- DLG-070-002 리드 삭제 확인
