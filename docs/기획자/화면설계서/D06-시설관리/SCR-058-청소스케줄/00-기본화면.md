## 화면 목적

센터 내 구역별 청소 담당자와 정기 청소 일정을 관리하는 화면입니다. 오늘의 청소 완료 여부를 체크하고, 정기 청소(주간·월간) 일정을 등록하며, 청소 완료 이력을 기록합니다. 담당자별 청소 현황을 파악해 시설 위생 관리의 누락을 방지합니다.

> 이 화면은 현재 퍼블리싱이 구현되지 않은 준비 중 기능입니다.

## 주요 구성요소

- **오늘의 청소 현황 카드**: 오늘 전체 청소 구역 수 / 완료 구역 수 / 미완료 구역 수 / 완료율(%)을 표시합니다.
- **청소 구역 체크리스트**: 오늘 청소 예정인 구역 목록을 표시합니다.
  - 표시 항목: 구역명 / 청소 유형(일일·정기) / 담당자 / 예정 시간 / 완료 여부 체크박스 / 완료 시간.
  - 완료 체크 시 완료 시간이 자동 기록됩니다.
- **정기 청소 일정 캘린더·목록**: 주간·월간 정기 청소 일정을 달력 또는 목록으로 표시합니다.
- **청소 스케줄 등록 버튼**: 새 청소 일정과 담당자를 등록합니다.
- **이력 조회**: 기간별 청소 완료 이력을 조회합니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 센터장 | 전체 구역 현황 및 이력 | 스케줄 등록·수정·삭제·담당자 지정 전체 가능 |
| 매니저 | 전체 구역 현황 및 이력 | 스케줄 등록·완료 체크 가능 |
| 스태프 | 담당 구역 현황 | 담당 구역 완료 체크만 가능 |
| 트레이너 / 조회전용 | 현황 조회만 | 변경 불가 |

## 주요 UX 흐름

1. 화면 진입 시 오늘의 청소 현황 카드와 체크리스트가 표시됩니다.
2. 청소 완료 후 해당 구역의 완료 체크박스를 클릭하면 완료 시간이 자동 기록됩니다.
3. **스케줄 등록**: [청소 스케줄 등록] 버튼 클릭 → 구역명·청소 유형·담당자·주기·예정 시간 입력 → 저장.
4. **이력 조회**: 기간을 선택하면 해당 기간의 청소 완료 이력이 표시됩니다.
5. 담당자 미지정 구역이나 오늘 완료되지 않은 구역은 목록 상단에 강조 표시됩니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 완료 | 초록색 체크 아이콘 + 완료 시간 표시, 행 배경 옅은 초록색 |
| 미완료 | 빈 체크박스 표시 |
| 시간 초과 미완료 | 빨간색 강조 표시 (예정 시간 경과) |
| 오늘 일정 없음 | "오늘 예정된 청소 일정이 없습니다" 안내 |
| 전체 완료 | 현황 카드 완료율 100% + 축하 메시지 표시 |
