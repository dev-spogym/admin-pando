## 화면 목적

센터에서 회원에게 대여하는 운동복의 재고 현황과 대여·반납·세탁·파손 상태를 관리하는 화면입니다. QR 코드 스캔을 통해 대여와 반납을 빠르게 처리할 수 있으며, 운동복별 현재 상태와 대여 이력을 한눈에 파악할 수 있습니다.

## 주요 구성요소

- **현황 요약 카드**: 전체 운동복 수 / 대기(사용 가능) 수 / 대여중 수 / 세탁중 수를 4개 카드로 표시합니다.
- **상태 탭**: 전체 · 대기 · 대여중 · 세탁중 · 파손 탭으로 필터링하고, 각 탭에 해당 건수를 표시합니다.
- **검색창**: 운동복 번호 또는 대여 회원명으로 검색합니다.
- **운동복 목록 테이블**: 등록된 운동복 전체를 표시합니다.
  - 표시 항목: 번호 / 사이즈(S·M·L·XL·XXL) / 타입(상의·하의·세트) / 상태 배지 / 대여 회원명 / 대여일 / 반납 예정일 / 메모 / 상태별 액션 버튼.
  - 액션 버튼은 현재 상태에 따라 다르게 표시됩니다.
    - 대기: [세탁] [파손]
    - 대여중: [반납] [세탁]
    - 세탁중: [세탁 완료]
    - 파손: [복구]
- **QR 스캔 버튼**: QR 코드·번호를 입력해 해당 운동복을 빠르게 조회하고 대여·반납을 처리합니다.
- **운동복 등록 버튼**: 새 운동복 항목을 추가합니다.
- **엑셀 다운로드**: 전체 운동복 현황을 엑셀로 내려받습니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 센터장 / 매니저 | 전체 목록 | 등록·상태 변경·QR 스캔 대여·반납·엑셀 전체 가능 |
| 스태프 / 프런트 | 전체 목록 | QR 스캔 대여·반납·상태 변경 가능 (등록 불가) |
| 트레이너 / 조회전용 | 현황 조회만 | 변경 불가 |

## 주요 UX 흐름

1. 화면 진입 시 전체 운동복 목록이 표시됩니다.
2. 상태 탭 또는 검색창을 이용해 원하는 운동복을 찾습니다.
3. **QR 스캔 대여**: [QR 스캔] 클릭 → QR 코드·번호 입력 → 운동복 조회 → 빈 운동복이면 회원 검색 후 [대여 확정] → 완료.
4. **QR 스캔 반납**: [QR 스캔] 클릭 → QR 코드·번호 입력 → 대여중 운동복이면 [반납 처리] → 완료.
5. **상태 직접 변경**: 목록에서 해당 행의 액션 버튼(반납·세탁·파손·복구)을 클릭하면 즉시 상태가 변경됩니다.
6. **운동복 등록**: [운동복 등록] 클릭 → 번호·사이즈·타입·메모 입력 → 저장.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 로딩 중 | 테이블 로딩 상태 표시 |
| 정상 | 운동복 목록 표시 |
| 빈 상태 | "운동복 데이터가 없습니다" 안내 메시지 |
| 대기(AVAILABLE) | 초록색 배지 |
| 대여중(RENTED) | 주황색 배지 + 대여 회원·일자 표시 |
| 세탁중(WASHING) | 회색 배지 |
| 파손(DAMAGED) | 빨간색 배지 |
| QR 조회 실패 | "해당 QR 코드의 운동복을 찾을 수 없습니다" 오류 메시지 |
