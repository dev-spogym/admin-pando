## 화면 목적

사용중인 락커를 회수(배정 해제)하기 전 최종 의사를 확인하는 다이얼로그입니다. 회수 시 배정된 회원 정보가 모두 초기화된다는 점을 명확히 안내합니다.

## 주요 구성요소

- **다이얼로그 제목**: "락커 회수"
- **안내 메시지**: "[락커 번호]번 락커를 회수하시겠습니까? 배정된 회원 정보가 초기화됩니다."
- **회수 버튼**: 위험(danger) 스타일로 표시되어 신중한 선택을 유도합니다.
- **취소 버튼**: 회수 없이 다이얼로그를 닫습니다.

## 주요 UX 흐름

1. 락커 셀 호버 시 [회수] 버튼 클릭 → 다이얼로그 열림.
2. 안내 메시지 확인 후 [회수] 클릭 → 락커가 빈 상태로 초기화 → "락커가 회수되었습니다" 알림.
3. [취소] 클릭 시 변경 없이 닫힘.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 회수 성공 | "락커가 회수되었습니다" 알림 + 그리드 갱신 |
| 회수 실패 | "락커 회수에 실패했습니다" 오류 알림 |
