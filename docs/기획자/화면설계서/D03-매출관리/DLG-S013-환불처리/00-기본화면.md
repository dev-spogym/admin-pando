## 목적

환불 요청이 접수된 건에 대해 환불 금액과 사유를 확정하고 처리를 완료합니다. 전체 환불 또는 부분 환불을 선택할 수 있으며, 처리 내용은 이력으로 저장됩니다.

## 구성요소

| 구성요소 | 내용 |
|---|---|
| 원 매출 정보 | 환불 대상 매출 번호, 결제일, 상품명, 결제 금액 (읽기 전용) |
| 환불 유형 선택 | 전체 환불 또는 부분 환불 중 선택 |
| 환불 금액 입력 | 부분 환불 선택 시 환불할 금액 직접 입력 (전체 환불 시 자동 입력) |
| 환불 사유 선택 | 사전 정의된 사유 목록에서 선택 (고객 변심, 서비스 불만, 중복 결제 등) |
| 사유 직접 입력 | 목록에 없는 경우 직접 입력 가능 |
| 환불 수단 | 환불 방법 선택 (원결제 수단 환불, 현금, 포인트 등) |
| 처리 확인 버튼 | 입력 내용으로 환불 처리 실행 |
| 취소 버튼 | 처리를 중단하고 이전 화면으로 복귀 |

## 언제 표시되나요

- DLG-S015(환불 요청)에서 처리 담당자가 승인 후 환불 처리로 진행할 때
- SCR-S007(환불 관리) 목록에서 미처리 건의 처리 버튼을 클릭했을 때
- DLG-S001(매출 상세)에서 환불 버튼을 클릭했을 때

## 주요 흐름

1. 환불 대상 매출 건을 선택하고 환불 처리 버튼을 누릅니다.
2. 환불 처리 다이얼로그가 열리며 원 매출 정보가 표시됩니다.
3. 환불 유형(전체/부분)을 선택합니다.
4. 환불 금액, 사유, 환불 수단을 입력합니다.
5. 처리 확인 버튼을 눌러 환불을 완료합니다.
6. 처리 완료 후 DLG-S014(환불 상세 결과)가 자동으로 표시됩니다.

## 관련 다이어그램

```

        ▼

        │              │
        │ 취소          │ 처리 확인
        ▼              ▼

```
