> **퍼블리싱 미구현**: 이 화면은 기획·설계 단계이며 아직 화면이 구현되지 않았습니다.

## 화면 목적

과거 매출 데이터를 기반으로 향후 매출을 예측하고, 설정한 목표 대비 달성률을 시각적으로 확인하는 화면입니다. 센터장과 매니저는 이 화면에서 다음 달·분기 매출을 미리 파악해 인력 배치, 상품 프로모션, 목표 조정 등 선제적인 운영 계획을 수립할 수 있습니다.

## 주요 구성요소

- **요약 지표 카드 (4개)**: 이번 달 예측 매출 / 목표 달성률 / 전월 대비 증감 / 이번 분기 누적 매출을 상단에 표시합니다.
- **예측 기간 선택**: 다음 달·다음 분기·연간 예측 중 원하는 기간을 선택합니다.
- **매출 목표 설정**: 기간별 매출 목표 금액을 직접 입력하면 달성률이 실시간으로 업데이트됩니다.
- **매출 예측 차트**: 과거 실적(선 그래프)과 예측값(점선 그래프)을 하나의 차트에 겹쳐 표시합니다. 목표 라인도 함께 표시됩니다.
- **달성률 게이지**: 현재 누적 매출이 목표의 몇 퍼센트인지 프로그레스 바와 숫자로 표시합니다.
- **상품별 예측 기여도**: 어떤 상품이 향후 매출에 가장 큰 기여를 할 것으로 예측되는지 순위 목록으로 제공합니다.
- **월별 예측 테이블**: 월별 예측 매출·목표·달성률·전년 동기 대비 항목을 표 형태로 제공합니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 본사 최고관리자 | 전체 지점 매출 조회·분석 가능 | 전체 지점 예측 조회, 목표 설정 |
| 센터장 / 매니저 | 소속 지점 매출 전체 | 지점 예측 조회, 목표 설정 |
| FC | 담당 회원 결제 처리 가능 | 조회만 가능 |
| 트레이너 / 스태프 | 매출 조회만 (처리 불가) | 조회만 가능 |

## 주요 UX 흐름

1. 사이드바에서 "매출관리 > 매출 예측"을 클릭해 진입합니다.
2. 상단 지표 카드로 이번 달 예측 매출과 목표 달성률을 확인합니다.
3. 예측 기간(다음 달·분기·연간)을 선택합니다.
4. 매출 목표 금액을 입력해 달성률 게이지가 변하는 것을 확인합니다.
5. 예측 차트에서 과거 추세와 예측 궤적을 비교합니다.
6. 상품별 기여도 목록으로 어느 상품에 집중해야 하는지 파악합니다.
7. 월별 예측 테이블에서 상세 수치를 확인합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 로딩 중 | 예측 데이터를 계산하는 동안 차트와 테이블 영역에 로딩 상태를 표시합니다. |
| 정상 표시 | 과거 데이터가 충분할 때 예측 차트와 테이블을 함께 표시합니다. |
| 데이터 부족 | 예측 기반이 될 과거 데이터가 부족할 때 안내 문구를 표시합니다. |
| 목표 미설정 | 목표 금액을 입력하지 않으면 달성률 게이지 대신 입력 안내를 표시합니다. |
| 목표 초과 달성 | 달성률이 100%를 초과하면 게이지 색상이 성공 색으로 변경됩니다. |

## 관련 다이어그램

- 매출 예측 기간 선택 흐름 (다음 달·분기·연간)
- 목표 대비 달성률 계산 및 게이지 표시 흐름
- 과거 실적·예측값·목표 라인 차트 구성 흐름
- 계정 권한별 접근 분기 흐름
