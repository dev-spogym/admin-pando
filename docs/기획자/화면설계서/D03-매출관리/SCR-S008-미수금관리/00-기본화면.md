## 화면 목적

회원이 결제를 완료하지 못한 미수금을 추적하고 관리하는 화면입니다. 미결제·일부결제·연체·완료 상태를 탭으로 구분해 현황을 파악하고, 납입이 이루어지면 상태를 직접 변경합니다. 연체 30일 이상 건을 별도로 집계해 독촉이 필요한 회원을 빠르게 파악할 수 있습니다.

## 주요 구성요소

- **요약 지표 카드 (4개)**: 미수금 총액 / 미수금 건수 / 연체(30일 이상) 건수 / 이번 달 회수액을 상단에 표시합니다. 연체 건수가 있을 때 해당 카드에 경고 강조 테두리가 표시됩니다.
- **상태 탭 (5개)**: 전체 / 미결제 / 일부결제 / 연체 / 완료 탭을 제공하며 각 탭에 해당 건수 배지가 표시됩니다.
- **회원명 검색**: 회원 이름으로 실시간 검색해 원하는 미수금 건을 빠르게 찾습니다.
- **미수금 목록 테이블**: No / 회원명 / 상품명 / 미수금액 / 결제 기한 / 상태 / 메모 / 등록일 / 액션 컬럼을 제공합니다.
  - 미수금액: 붉은색으로 강조 표시합니다.
  - 결제 기한: 연체된 건은 날짜를 붉은색 굵은 글씨로 표시합니다.
  - 상태 배지: 완료(초록) / 연체(빨강) / 일부결제(노랑) / 미결제(기본) 색상으로 구분합니다.
  - 액션 컬럼: 상태 변경 드롭다운과 메모 편집 버튼을 제공합니다.
- **상태 변경**: 현재 상태에 따라 전환 가능한 다음 상태만 드롭다운에 표시됩니다. (미결제→일부결제·완료, 일부결제→완료, 연체→일부결제·완료, 완료→변경 불가)
- **메모 편집 모달**: 메모 버튼을 클릭하면 메모를 입력·수정할 수 있는 모달이 열립니다. "저장" 버튼으로 즉시 반영됩니다.
- **엑셀 내보내기**: 현재 탭과 검색 조건이 적용된 미수금 목록을 엑셀 파일로 내려받습니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 본사 최고관리자 | 전체 지점 매출 조회·분석 가능 | 상태 변경, 메모 편집, 엑셀 내보내기, 전체 탭 사용 |
| 센터장 / 매니저 | 소속 지점 매출 전체 | 상태 변경, 메모 편집, 엑셀 내보내기, 전체 탭 사용 |
| FC | 담당 회원 결제 처리 가능 | 상태 변경, 메모 편집 가능 |
| 트레이너 / 스태프 | 매출 조회만 (처리 불가) | 조회만 가능 (상태 변경·메모 편집 불가) |

## 주요 UX 흐름

1. 사이드바에서 "매출관리 > 미수금 관리"를 클릭해 진입합니다.
2. 상단 지표 카드로 미수금 총액·연체 건수를 파악합니다.
3. 상태 탭을 선택해 원하는 상태의 미수금 건만 표시합니다.
4. 회원명을 검색해 특정 회원의 미수금 건을 찾습니다.
5. 납입이 이루어진 건의 액션 열 드롭다운에서 새 상태를 선택해 업데이트합니다.
6. 메모 버튼을 눌러 독촉 이력이나 납입 협의 내용을 메모로 기록합니다.
7. 엑셀 버튼으로 현재 목록을 파일로 저장합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 로딩 중 | 데이터를 불러오는 동안 테이블 영역에 로딩 상태를 표시합니다. |
| 정상 표시 | 미수금 데이터가 있을 때 지표 카드와 테이블을 표시합니다. |
| 검색 결과 없음 | 회원명 검색 후 결과가 없을 때 안내 문구를 표시합니다. |
| 탭 내 데이터 없음 | 해당 상태의 미수금이 없을 때 빈 테이블 안내를 표시합니다. |
| 메모 편집 모달 열림 | 메모 버튼 클릭 시 모달이 화면 위에 표시됩니다. |
| 오류 | 데이터 조회 또는 업데이트 실패 시 오류 메시지 알림(토스트)을 표시합니다. |

## 관련 다이어그램

- 상태 전환 규칙 흐름 (미결제→일부결제·완료, 연체→일부결제·완료)
- 연체 판단 기준 흐름 (결제 기한 30일 초과)
- 메모 편집 모달 흐름
- 엑셀 내보내기 흐름
- 계정 권한별 접근 분기 흐름
