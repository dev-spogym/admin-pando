## 화면 목적
유사한 규모·업종의 다른 센터와 자사 지점의 KPI를 익명으로 비교 분석하는 화면입니다.
업계 평균 대비 우리 센터의 위치를 파악하여 경쟁력 강화 전략을 수립합니다.

## 주요 구성요소

### 비교 기준 설정
- 비교 대상 센터 규모 선택 (회원 수 기준 소 / 중 / 대)
- 비교 대상 업종 선택 (헬스 / 필라테스 / PT샵 등)
- 비교 지표 선택 (매출 / 회원 유지율 / 출석율 / 신규 회원 등)
- 기간 선택 (월간 / 분기 / 연간)

### 벤치마크 차트
- 우리 지점 수치 vs 업계 평균 vs 상위 25% 비교 막대 차트
- 지표별 우리 지점의 백분위 표시 (예: "상위 35%")
- 개선 여지가 큰 지표 강조 표시

### 인사이트 요약
- 강점 지표 목록 (업계 평균 대비 우수한 항목)
- 개선 필요 지표 목록 (업계 평균 대비 미달 항목)
- 개선을 위한 액션 아이템 제안

## 계정별 차이
| 계정 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 | 전체 지점 벤치마크 비교 | 비교 기준 설정, 내보내기 |
| 센터장 | 소속 지점 벤치마크 | 비교 기준 설정 |
| 매니저 이하 | 접근 불가 | — |

## 주요 UX 흐름
1. 비교 기준(규모, 업종, 지표, 기간)을 선택한다.
2. 분석 실행 버튼 클릭 → 벤치마크 데이터 로드 및 차트 표시.
3. 지표별 차트를 확인하고 인사이트 요약을 검토한다.
4. 내보내기 버튼으로 결과를 PDF로 저장한다.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 초기 | 비교 기준 설정 패널 표시, 차트 비어있음 |
| 로딩 | 벤치마크 데이터 수집 중 (처리 중 표시) |
| 결과 표시 | 차트 및 인사이트 표시 |
| 데이터 부족 | 비교 가능한 동일 규모 센터 데이터 부족 안내 |
