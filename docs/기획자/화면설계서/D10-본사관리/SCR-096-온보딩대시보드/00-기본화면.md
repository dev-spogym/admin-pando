## 화면 목적
신규 지점 또는 센터가 서비스를 처음 사용할 때 초기 설정을 완료할 수 있도록 단계별 가이드를 제공하는 화면입니다.
설정 완료 현황을 한눈에 확인하고, 미완료 항목으로 바로 이동할 수 있습니다.

## 주요 구성요소

### 전체 진행률
- 온보딩 전체 완료율 표시 (퍼센트 게이지)
- 완료 단계 수 / 전체 단계 수 표시

### 온보딩 단계 목록
각 단계 카드에 완료 여부, 단계명, 설명, 이동 버튼 표시:
1. 센터 기본 정보 입력 (SCR-080 연결)
2. 직원 계정 생성 및 역할 배정
3. 상품·이용권 등록
4. 첫 회원 등록
5. 키오스크 설정 (SCR-082 연결)
6. IoT 기기 연결 (SCR-083 연결)
7. 구독 플랜 선택 (SCR-084 연결)

### 빠른 도움말
- 각 단계별 FAQ 링크
- 고객 지원 문의 버튼

## 계정별 차이
| 계정 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 | 전체 지점 온보딩 현황 | 미완료 단계 직접 이동 |
| 센터장 | 소속 지점 온보딩 현황 | 미완료 단계 직접 이동 |
| 매니저 이하 | 접근 불가 | — |

## 주요 UX 흐름
1. 신규 지점 생성 후 이 화면으로 자동 이동한다.
2. 각 단계 카드의 이동 버튼 클릭 → 해당 설정 화면으로 이동.
3. 설정 화면에서 완료 후 돌아오면 해당 단계가 완료 표시로 전환.
4. 모든 단계 완료 시 축하 메시지와 함께 대시보드 이동 버튼 표시.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 초기 | 모든 단계 미완료 상태, 1단계 강조 표시 |
| 진행 중 | 일부 단계 완료, 진행률 업데이트 |
| 전체 완료 | 100% 달성 축하 메시지 및 대시보드 이동 버튼 |
