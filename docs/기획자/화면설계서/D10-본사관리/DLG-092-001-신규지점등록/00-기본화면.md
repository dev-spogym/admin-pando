## 화면 목적
새로운 지점의 기본 정보를 입력하여 시스템에 등록하는 다이얼로그입니다.
등록 완료 후 해당 지점의 관리자 계정이 생성되고 온보딩 절차가 시작됩니다.

## 주요 구성요소
- "신규 지점 등록" 제목
- 지점명 입력란 (필수)
- 지점 코드 입력란 (필수, 시스템 내부 식별자, 중복 불가)
- 대표 연락처 입력란
- 주소 입력란 (주소 검색 버튼 포함)
- 지점 관리자 계정 이메일 입력란 (필수, 초기 관리자 계정 생성용)
- 운영 시작일 선택
- 수용 가능 회원 수 입력 (선택)
- 취소 버튼
- 등록 버튼 (필수 항목 입력 시 활성)

## 주요 UX 흐름
1. 지점 관리 화면에서 신규 지점 등록 버튼 클릭 → 다이얼로그 표시
2. 필수 정보 입력 (지점명, 지점 코드, 관리자 이메일)
3. 주소 검색 버튼 클릭 → 주소 검색 팝업 → 선택한 주소 자동 입력
4. 등록 버튼 클릭 → 지점 생성 → 관리자 계정으로 초대 이메일 발송
5. 등록 완료 후 다이얼로그 닫히고 지점 목록에 새 지점 추가됨
6. 온보딩 대시보드(SCR-096)로 이동 여부 확인 안내

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 빈 폼 표시 |
| 코드 중복 | 이미 사용 중인 지점 코드 입력 시 오류 메시지 |
| 등록 처리 중 | 등록 버튼 클릭 후 처리 중 (버튼 비활성) |
| 등록 완료 | 다이얼로그 닫히고 지점 목록 갱신, 온보딩 이동 안내 표시 |
