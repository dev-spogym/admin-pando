## 화면 목적
위젯을 드래그앤드롭 방식으로 배치하여 자신만의 대시보드를 구성하는 화면입니다.
자주 확인하는 지표와 정보를 원하는 레이아웃으로 배치하여 업무 효율을 높입니다.

## 주요 구성요소

### 위젯 라이브러리 패널 (좌측)
- 추가 가능한 위젯 목록 (카테고리별 분류: 매출 / 회원 / 출석 / 수업 / 공지 / 할 일 등)
- 각 위젯 썸네일 미리보기
- 위젯 검색

### 캔버스 영역 (우측)
- 현재 대시보드 레이아웃 편집 영역
- 위젯 드래그앤드롭으로 배치 및 이동
- 위젯 크기 조절 (모서리 드래그)
- 위젯별 설정 버튼 (위젯 내 표시 기간, 지점 등 세부 설정)
- 위젯별 제거 버튼

### 레이아웃 옵션
- 그리드 열 수 선택 (2 / 3 / 4열)
- 그리드 선 표시/숨김 토글

### 저장 및 관리
- 현재 레이아웃 저장 버튼
- 저장된 레이아웃 불러오기
- 기본값으로 초기화 버튼
- 레이아웃 이름 지정 및 저장 (여러 레이아웃 프리셋 관리)

## 계정별 차이
| 계정 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 | 전체 위젯 라이브러리 | 레이아웃 구성·저장·공유 |
| 센터장 | 지점 관련 위젯 | 레이아웃 구성·저장 |
| 매니저 이하 | 접근 불가 | — |

## 주요 UX 흐름
1. 화면 진입 시 현재 저장된 대시보드 레이아웃을 불러온다.
2. 좌측 라이브러리에서 위젯을 드래그하여 캔버스에 배치.
3. 배치된 위젯을 드래그로 이동하거나 모서리를 드래그로 크기 조정.
4. 위젯 설정 버튼 클릭 → 해당 위젯의 데이터 범위·필터 설정 패널 표시.
5. 저장 버튼 클릭 → 레이아웃 저장 → 대시보드에 즉시 반영.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 저장된 레이아웃 편집 모드 |
| 위젯 드래그 중 | 드래그 대상 위젯 강조, 배치 가능 위치 표시 |
| 수정 중 | 변경 사항 있을 때 저장 버튼 활성 |
| 저장 완료 | 완료 메시지 후 대시보드 반영 |
