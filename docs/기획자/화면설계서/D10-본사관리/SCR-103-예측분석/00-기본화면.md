## 화면 목적
AI 기반 분석을 활용하여 향후 매출, 회원 수, 이탈률 등을 예측하는 화면입니다.
과거 데이터 패턴을 학습하여 미래 운영 계획 수립과 선제적 대응에 활용합니다.

## 주요 구성요소

### 예측 항목 선택
- 예측 지표 선택 (매출 / 신규 회원 수 / 이탈 회원 수 / 출석율)
- 예측 기간 선택 (향후 1개월 / 3개월 / 6개월 / 12개월)
- 대상 지점 선택

### 예측 결과 차트
- 과거 실적 + 예측 값 통합 선 차트 (예측 구간은 점선 또는 밴드로 구분)
- 예측 신뢰 구간 표시 (낙관적 / 기본 / 보수적 시나리오)
- 예측 기반 데이터 기간 안내

### 예측 인사이트
- 예측 근거 주요 요인 설명 (계절성, 최근 추이, 이벤트 영향 등)
- 이탈 위험 회원 예측 목록 (회원 이름, 이탈 위험도 수치)
- 권장 대응 액션 제안

### 시나리오 비교
- 시나리오별(낙관 / 기본 / 보수) 예측 수치 비교 테이블

## 계정별 차이
| 계정 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 | 전체 지점 예측 분석 | 항목·기간 설정, 내보내기 |
| 센터장 | 소속 지점 예측 | 항목·기간 설정 |
| 매니저 이하 | 접근 불가 | — |

## 주요 UX 흐름
1. 예측 지표, 기간, 지점을 선택한다.
2. 예측 실행 버튼 클릭 → AI 분석 처리 중 표시 → 완료 후 차트 표시.
3. 차트에서 마우스 오버 시 해당 시점의 예측 수치와 신뢰 구간 툴팁 표시.
4. 이탈 위험 회원 목록에서 회원 이름 클릭 → 해당 회원 상세 화면으로 이동.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 초기 | 예측 항목 설정 패널 표시 |
| 분석 중 | AI 처리 중 표시 (처리 시간 안내 포함) |
| 결과 표시 | 예측 차트 및 인사이트 표시 |
| 데이터 부족 | 학습에 필요한 최소 데이터 부족 안내 (예: "6개월 이상의 데이터가 필요합니다") |
