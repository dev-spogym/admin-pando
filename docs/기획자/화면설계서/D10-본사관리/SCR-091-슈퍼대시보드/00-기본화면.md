## 화면 목적
슈퍼관리자 전용 화면으로, 전체 지점의 통합 KPI와 지점별 상세 현황을 한 화면에서 파악합니다.
지점 전환 기능을 통해 특정 지점 관리 모드로 빠르게 진입할 수 있습니다.

## 주요 구성요소

### 전사 통합 KPI 카드 (7종)
- 전체 회원 수 (전월 대비 증감률 표시)
- 전체 매출 (전월 대비 증감률 표시)
- 전체 출석 수 (전일 대비 증감률 표시)
- 전체 직원 수
- 이번 달 신규 회원 수
- 만료 예정 회원 수 (30일 이내)
- 만료 회원 수

### 지점별 현황 카드
- 각 지점의 활성 회원 수, 직원 수, 월 매출, 만료 회원 수, 오늘 출석 수
- 지점 운영 상태 배지 (운영 중 / 휴업 / 폐점)
- 지점 전환 버튼 (클릭 시 해당 지점 관리 모드로 전환)
- 지점 상세 보기 버튼

### 최근 감사 로그
- 시스템 전체에서 발생한 최근 10건의 주요 활동 이력

### 상단 액션
- 수동 새로고침 버튼

## 계정별 차이
| 계정 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 | 전체 지점 통합 현황 및 지점별 카드 | 지점 전환, 지점 상세 보기, 새로고침 |
| 일반 관리자 / 직원 | 접근 불가 안내 화면 | 대시보드로 이동 버튼만 사용 |

## 주요 UX 흐름
1. 슈퍼관리자 로그인 후 화면 진입 → 전체 지점 데이터 자동 로드.
2. KPI 카드 로딩 중 스켈레톤 표시 → 완료 후 실제 수치로 전환.
3. 증감률 양수 → 초록 상승 화살표, 음수 → 빨간 하락 화살표.
4. 지점 전환 버튼 클릭 → 해당 지점 모드로 전환 → 본사 대시보드로 이동.
5. 슈퍼관리자 외 접근 시 권한 없음 안내 메시지 표시.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 통합 KPI 및 지점별 카드 정상 표시 |
| 로딩 | 데이터 불러오는 중 (스켈레톤 표시) |
| 권한 없음 | 슈퍼관리자 외 접근 시 권한 없음 안내 표시 |
| 지점 없음 | 등록된 지점이 없을 때 지점 등록 안내 표시 |
