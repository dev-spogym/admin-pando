## 화면 목적
오늘의 할 일 화면에서 새로운 업무 항목을 추가하는 다이얼로그입니다.

## 주요 구성요소
- "할 일 추가" 제목
- 할 일 제목 입력란 (필수)
- 할 일 상세 설명 입력란 (선택)
- 우선순위 선택 (높음 / 보통 / 낮음)
- 담당자 선택 드롭다운 (직원 목록 중 선택, 기본값: 본인)
- 마감 시각 선택 (시간 입력, 선택)
- 취소 버튼
- 추가 버튼 (제목 입력 시 활성)

## 주요 UX 흐름
1. 오늘의 할 일 화면에서 할 일 추가 버튼 클릭 → 다이얼로그 표시
2. 제목 입력 및 선택 항목 설정
3. 추가 버튼 클릭 → 할 일 목록에 즉시 추가, 다이얼로그 닫힘
4. 취소 → 다이얼로그 닫힘

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 빈 폼, 우선순위 기본값 "보통", 담당자 기본값 본인 |
| 제목 미입력 | 추가 버튼 비활성 |
| 추가 완료 | 다이얼로그 닫히고 목록에 새 항목 표시 |
