## 화면 목적

새로운 회원을 시스템에 등록하는 화면입니다. 필수 정보(Step 1)와 추가 정보(Step 2)를 단계별로 입력받아 빠른 등록과 상세 정보 입력을 분리합니다. 등록 완료 후 곧바로 이용권 결제(POS)로 연결해 신규 회원의 서비스 시작을 자연스럽게 유도합니다.

## 주요 구성요소

- **단계 표시기 (Step 1 / Step 2)**: 현재 진행 단계를 상단에 표시. Step 1이 완료되어야 Step 2로 이동 가능.
- **프로필 사진 업로더**: 원형 이미지 영역을 클릭해 사진 등록. 미등록 시 이니셜 아바타 자동 생성.
- **[Step 1] 기본 인적 사항**
  - 이름 (필수), 성별 (필수, 남성/여성 토글)
  - 연락처 (필수) + 중복 확인 버튼 — 동일 연락처 회원이 있으면 경고 표시
  - 회원구분 (일반 / 기명법인 / 무기명법인, 필수)
  - 법인 회원 선택 시 법인 회사명 입력란 추가 노출
  - 생년월일, 키(cm) (선택)
- **[Step 1] 관리 정보**: 담당 FC / 담당 트레이너 / 방문경로 / 유입경로 / 운동 목적 (모두 선택)
- **[Step 2] 추가 연락 정보**: 별칭/닉네임 / 이메일 / 주소(주소 검색 버튼) / 상세 주소 / 회사명 / 출석번호
- **[Step 2] 기타 설정**: 광고성 정보 수신 동의 체크박스, 메모 입력란(최대 500자, 글자 수 표시)
- **하단 고정 버튼**: Step 1에서는 [취소] [다음], Step 2에서는 [이전] [저장]
- **초기화 버튼**: 입력한 내용이 있을 때만 활성화. 확인 후 전체 초기화.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 본사 최고관리자 | 전체 화면 | 회원 등록 완전 접근 |
| 센터장 (owner) | 전체 화면 | 회원 등록 완전 접근 |
| 매니저 | 전체 화면 | 회원 등록 완전 접근 |
| 스태프 | 전체 화면 | 회원 등록 완전 접근 |
| FC / 트레이너 / 프런트 | 접근 불가 | 이 화면 진입 시 접근 제한 안내로 이동 |

## 주요 UX 흐름

1. 회원 목록(SCR-M001)의 "회원 추가" 버튼을 눌러 진입합니다.
2. Step 1에서 이름·성별·연락처·회원구분을 입력하고, 연락처 중복 확인을 완료합니다.
3. "다음" 버튼을 누르면 Step 2로 이동합니다. (필수 항목 미입력 또는 중복 확인 미완료 시 이동 불가)
4. Step 2에서 추가 정보를 선택적으로 입력하고 "저장"을 누릅니다.
5. 저장 성공 시 해당 회원의 상세 페이지(SCR-M004)로 이동하며, "바로 결제" 버튼이 5초간 노출됩니다.
6. 취소 또는 이탈 시, 입력한 내용이 있으면 확인 다이얼로그가 나타납니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 초기 상태 | 모든 입력란 비어 있음, 초기화 버튼 비활성 |
| 입력 중 | 사용자가 값을 입력하기 시작한 상태, 초기화 버튼 활성 |
| 유효성 오류 | 필수 항목 누락 또는 형식 불일치 시 해당 입력란 하단에 오류 메시지 표시 |
| 이미지 업로드 중 | 사진 선택 후 업로드 처리 중 로딩 표시 |
| 법인 회원 모드 | 회원구분에서 기명/무기명 법인 선택 시 법인 회사명 입력란 추가 노출 |
| Step 2 | Step 1 완료 후 추가 정보 입력 단계 |
| 저장 중 | "저장" 클릭 후 처리 중 버튼 비활성 + 로딩 표시 |
| 등록 완료 | 저장 성공 직후, 회원 상세로 자동 이동 |
