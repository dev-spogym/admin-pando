## 목적
키오스크나 QR 체크인 없이 관리자가 직접 회원의 출석을 처리하는 다이얼로그입니다. 기기 오류, 회원 요청 등 예외 상황에 대응합니다.

## 구성요소
- 제목: "수동 출석 처리"
- 처리 대상 회원 정보 표시 (회원명, 회원번호, 현재 이용권)
- 출석 날짜 선택 (기본값: 오늘, 과거 날짜 선택 가능)
- 출석 시간 입력 (기본값: 현재 시간)
- 처리 사유 입력 (텍스트 입력 또는 사유 목록 선택)
- 처리 버튼
- 취소 버튼

## 언제 표시되나요
- 회원 상세 > 출석 탭에서 수동 출석 버튼 클릭 시
- 출석 관리 화면에서 해당 회원의 수동 출석 버튼 클릭 시

## 주요 흐름
1. 관리자가 수동 출석 버튼 클릭 → 다이얼로그 표시, 대상 회원 정보 표시
2. 출석 날짜·시간 확인(또는 수정) 및 사유 입력
3. 처리 버튼 클릭 → 출석 기록 저장, 이용 횟수 차감(횟수권인 경우)
4. 취소 클릭 → 다이얼로그 닫힘, 변경 없음
