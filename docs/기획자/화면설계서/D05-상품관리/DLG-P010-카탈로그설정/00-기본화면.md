## 목적

상품 카탈로그에 표시할 정보 항목과 레이아웃 옵션을 설정하는 모달입니다. 센터의 특성에 맞게 노출 항목을 조절할 수 있습니다.

---

## 구성요소

| 설정 항목 | 설명 |
|-----------|------|
| 가격 표시 여부 | 카탈로그에 가격 노출 여부 |
| 기간/횟수 표시 여부 | 이용 기간 또는 횟수 노출 여부 |
| 상품 유형별 구분 | 유형별 섹션 구분 여부 |
| 레이아웃 선택 | 카드형 / 리스트형 |
| 컬러 테마 | 센터 브랜드 색상 적용 |
| 센터 로고 표시 | 카탈로그 상단 로고 노출 여부 |

하단 버튼: **취소** / **저장**

---

## 언제 표시되나요

- SCR-P005(상품 카탈로그) 화면에서 "카탈로그 설정" 버튼을 클릭했을 때

---

## 주요 흐름

1. 현재 설정값이 표시된 상태로 모달 열림
2. 원하는 옵션 변경
3. "저장" 클릭: 설정 저장 → 카탈로그에 즉시 반영
4. "취소" 클릭: 변경 사항 폐기, 모달 닫힘
