## 목적

상품을 최종 삭제하기 전, 해당 상품에 연결된 이용권 또는 판매 이력을 안내하고 삭제 의사를 재확인하는 다이얼로그입니다. 연결된 데이터가 있을 경우 영향 범위를 명확히 알려줍니다.

---

## 구성요소

- 삭제 대상 상품명 표시
- 연결된 이용권 수 / 판매 이력 안내 (있을 경우)
- 경고 문구: "삭제 후에는 복구할 수 없습니다."
- 하단 버튼: **취소** / **삭제**

---

## 언제 표시되나요

- SCR-P006(상품 비교) 화면에서 상품 삭제를 시도할 때
- 상품 관련 삭제 확인이 필요한 다른 경로에서도 재사용될 수 있습니다.

---

## 주요 흐름

1. 삭제 요청 → 대상 상품 정보 및 연결 데이터 조회
2. 이 다이얼로그에 요약 정보 표시
3. "취소" 클릭: 다이얼로그 닫힘, 작업 중단
4. "삭제" 클릭: 상품 삭제 처리 → 완료 메시지 → 화면 갱신
