## 화면 목적

여름 특가, 신년 이벤트 등 특정 기간에만 적용되는 시즌 특별 가격을 설정하고 관리하는 화면입니다. 기간, 대상 상품, 특가 금액을 지정하여 정해진 기간 동안 자동으로 할인 가격이 적용되도록 합니다.

> 이 화면은 현재 퍼블리싱이 구현되지 않은 계획 단계의 화면입니다.

---

## 주요 구성요소

### 시즌 특가 목록 테이블

| 컬럼 | 내용 |
|------|------|
| 시즌명 | 특가 이벤트 이름 (예: 2026 여름 특가) |
| 적용 기간 | 시작일 ~ 종료일 |
| 대상 상품 | 특가 적용 상품 목록 (N개) |
| 특가 금액/할인율 | 적용되는 특별 가격 또는 할인율 |
| 상태 | 예정 / 진행 중 / 종료 |
| 액션 | 수정 / 삭제 버튼 |

### 상태별 구분

| 상태 | 조건 | 표시 색상 |
|------|------|-----------|
| 예정 | 시작일 이전 | 파란색 |
| 진행 중 | 시작일~종료일 사이 | 초록색 |
| 종료 | 종료일 이후 | 회색 |

### 달력 뷰 (선택)

- 월별 달력에서 시즌 특가 기간을 시각적으로 확인
- 달력과 목록 뷰 전환 가능

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|-----------|--------------|--------------|
| 일반 직원 | 시즌 특가 목록 조회 | 조회만 가능 |
| 상품 편집 권한 보유 직원 | 전체 화면 | 시즌 특가 등록, 수정, 삭제 |
| 관리자 / 슈퍼관리자 | 전체 화면 | 시즌 특가 등록, 수정, 삭제 |

---

## 주요 UX 흐름

1. **시즌 특가 조회**: 화면 진입 → 전체 시즌 특가 목록 표시 (예정/진행/종료 구분)
2. **특가 등록**: "시즌 특가 등록" 버튼 → [DLG-P016-시즌가격등록수정]에서 이벤트명, 기간, 대상 상품, 특가 설정 후 저장
3. **특가 수정**: 목록에서 수정 버튼 클릭 → [DLG-P016-시즌가격등록수정]에서 기존 정보 수정
4. **특가 삭제**: 목록에서 삭제 버튼 클릭 → 삭제 확인 후 제거
5. **진행 중 특가 확인**: 현재 적용 중인 특가는 목록 상단 고정 또는 강조 표시

## 화면 상태

| 상태 | 조건 | 표시 내용 |
|------|------|-----------|
| 로딩 중 | 데이터 불러오는 중 | 테이블 로딩 표시 |
| 빈 상태 | 등록된 시즌 특가 없음 | "등록된 시즌 특가가 없습니다." 안내 |
| 진행 중 있음 | 현재 적용 중 특가 존재 | 상단 배너 또는 강조 표시로 안내 |
| 종료된 이력 | 과거 특가 존재 | 회색 처리로 구분 표시 |
