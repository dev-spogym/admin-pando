## 목적

특정 강사의 기본 정보와 해당 기간 내 담당 수업 목록을 빠르게 확인하는 팝업입니다. 강사 근무 현황 화면에서 개별 강사를 클릭할 때 열리며, 별도 페이지 이동 없이 핵심 정보를 조회할 수 있습니다.

## 구성요소

- **강사 기본 정보**:
  - 이름, 역할(트레이너/GX강사 등), 프로필 사진
  - 연락처, 담당 수업 유형
- **기간 내 수업 요약**:
  - 총 수업 건수, 총 근무 시간
  - 수업 유형별 분포 (PT N건, GX N건 등)
- **담당 수업 목록**:
  - 수업명, 날짜·시간, 장소, 예약 현황, 출석 처리 상태
  - 목록은 최근순 정렬
- **닫기 버튼**

## 언제 표시되나요

- 강사 근무 현황 화면(SCR-C006)에서 강사 행을 클릭했을 때

## 주요 흐름

1. 강사 근무 현황 목록에서 강사 행 클릭 → 팝업 열림
2. 강사 기본 정보 및 기간 내 수업 요약 확인
3. 담당 수업 목록 스크롤하며 개별 수업 현황 파악
4. 닫기 버튼 또는 외부 클릭으로 팝업 닫힘
