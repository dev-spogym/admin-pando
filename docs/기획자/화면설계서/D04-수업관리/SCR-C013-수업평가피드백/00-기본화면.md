> **퍼블리싱 미구현**: 이 화면은 기획 완료 상태이나 아직 개발·퍼블리싱이 진행되지 않았습니다.

## 화면 목적

수업 완료 후 회원이 남긴 평가·후기를 관리자가 조회하는 화면입니다. 강사별·수업별 평균 평점과 텍스트 후기를 확인해 서비스 품질을 개선하고, 우수 강사를 파악하거나 불만 사항에 대응하는 데 활용합니다.

## 주요 구성요소

- **기간 필터**: 조회 기간 선택 (주간/월간/사용자 지정)
- **강사 필터**: 특정 강사 피드백만 표시
- **수업 유형 필터**: PT / GX / 개인레슨 등
- **평점 요약 카드**: 전체 평균 평점, 강사별 평균 평점 랭킹
- **피드백 목록 테이블**:
  - 회원명, 수업명, 강사명, 수업 일시
  - 평점 (별점 1~5)
  - 후기 텍스트
  - 피드백 등록 일시
- **피드백 상세**: 행 클릭 시 후기 전문 표시
- **평점 분포 차트**: 1~5점 분포 막대 그래프 (선택)

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 강사·수업 피드백, 평균 평점 | 조회만 가능 |
| 트레이너 | 본인 수업 피드백만 | 본인 피드백 조회만 가능 |
| FC / 스태프 | 접근 불가 | 이 기능 사용 불가 |

## 주요 UX 흐름

1. 사이드바 > 수업관리 > 수업 평가 피드백 진입
2. 기간 및 강사·유형 필터 설정
3. 평점 요약 카드에서 강사별 평균 평점 비교
4. 피드백 목록에서 낮은 평점 후기 우선 확인
5. 행 클릭으로 후기 전문 열람
6. 불만 사항 발견 시 해당 강사·수업 개선 검토

## 화면 상태

| 상태 | 설명 |
|---|---|
| 피드백 없음 | 해당 기간 피드백 없음 안내 |
| 피드백 있음 | 목록 및 평점 요약 카드 표시 |
| 낮은 평점 강조 | 3점 이하 후기 주황/빨강 강조 표시 |
| 로딩 중 | 스켈레톤 표시 |
