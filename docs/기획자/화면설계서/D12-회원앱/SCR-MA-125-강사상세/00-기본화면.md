## 화면 목적

회원 앱에서 강사의 프로필, 전문 분야, 자격 이력, 담당 수업 목록을 조회한다. 회원이 수업 예약 전 강사 정보를 확인하고 예약으로 바로 이동할 수 있다.

## 주요 구성요소

### 강사 프로필 섹션

| 항목 | 설명 |
|------|------|
| 프로필 사진 | 강사 대표 이미지 |
| 강사명 | 이름 |
| 전문 분야 | 담당 운동 종목 태그 (예: 필라테스, 웨이트, 골프) |
| 소개글 | 강사 자기소개 또는 철학 |
| 경력 | 주요 경력 사항 |
| 자격증 | 보유 자격증 목록 |

### 담당 수업 목록 섹션

해당 강사가 담당하는 수업 목록을 카드 형태로 표시한다.

| 항목 | 설명 |
|------|------|
| 수업명 | 수업 이름 |
| 수업 유형 | 그룹 / 1:1 PT / 골프 레슨 등 |
| 정원 | 최대 수강 인원 |
| 다가오는 일정 | 가장 가까운 수업 일시 |
| 예약 가능 여부 | 잔여 자리 여부 배지 |

### 버튼 (각 수업 항목별)

| 버튼 | 조건 | 설명 |
|------|------|------|
| 예약하기 | 예약 가능 상태 | 해당 수업 예약 화면으로 이동 |
| 대기 신청 | 만석 상태 | 대기 예약 신청 처리 |

## 계정별 차이

회원 앱 화면으로 로그인한 회원 누구나 강사 정보를 조회할 수 있다. 편집 기능은 없으며 관리자 시스템에서만 수정 가능하다.

## 주요 UX 흐름

1. 수업 목록 또는 강사 목록에서 강사 이름·사진 탭 → 강사 상세 진입
2. 프로필 정보 확인
3. 담당 수업 목록 스크롤하여 원하는 수업 확인
4. 예약하기 버튼 → 수업 예약 화면으로 이동
5. 만석 수업 → 대기 신청 버튼 → 대기 예약 처리 → 대기 목록에서 확인 가능

## 화면 상태

| 상태 | 설명 |
|------|------|
| 기본(정보 있음) | 프로필 및 담당 수업 목록 정상 표시 |
| 담당 수업 없음 | '현재 담당 수업이 없습니다.' 안내 |
| 수업 만석 | 해당 수업 '만석' 배지 + 대기 신청 버튼 표시 |
| 로딩 | 정보 조회 중 스켈레톤 표시 |
| 조회 실패 | 오류 안내 및 재시도 유도 |
