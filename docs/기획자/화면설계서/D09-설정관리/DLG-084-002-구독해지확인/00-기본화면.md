## 화면 목적
서비스 구독을 해지하기 전 해지 시 발생하는 변화를 명확히 안내하고 최종 의사를 확인하는 다이얼로그입니다.
충동적인 해지를 방지하고, 해지 후 서비스 이용 조건을 투명하게 공지합니다.

## 주요 구성요소
- "구독 해지 확인" 제목
- 해지 후 서비스 종료 예정일 안내 (현재 결제 기간 만료일)
- 해지 시 사용 불가가 되는 주요 기능 목록
- 데이터 보관 정책 안내 (해지 후 데이터 보관 기간)
- 재가입 시 데이터 복구 가능 여부 안내
- "해지를 계속 진행하겠습니까?" 최종 확인 문구
- 취소 버튼 (구독 유지)
- 구독 해지 확인 버튼

## 주요 UX 흐름
1. 구독 결제 관리 화면에서 구독 해지 버튼 클릭 → 다이얼로그 표시
2. 해지 후 변화 및 데이터 정책 확인
3. 취소 → 다이얼로그 닫힘, 구독 유지
4. 구독 해지 확인 → 해지 처리 → 해지 완료 메시지 및 서비스 종료 예정일 표시

## 화면 상태
| 상태 | 설명 |
|---|---|
| 표시 | 해지 안내 정보 표시 |
| 처리 중 | 해지 확인 버튼 클릭 후 처리 중 (버튼 비활성) |
| 해지 완료 | 다이얼로그 닫히고 구독 화면에 해지 완료 상태 표시 |
