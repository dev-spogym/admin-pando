## 화면 목적
기존 역할의 권한 설정을 그대로 복사하여 새로운 역할을 빠르게 생성하는 다이얼로그입니다.
유사한 권한 구성이 필요한 역할을 처음부터 설정하지 않고 기존 역할을 기반으로 생성합니다.

## 주요 구성요소
- "역할 복사" 제목
- 복사할 원본 역할 선택 드롭다운 (필수)
- 새 역할 이름 입력란 (필수, 기본값: "원본역할명 복사본")
- 새 역할 코드 입력란 (필수)
- 취소 버튼
- 복사 생성 버튼 (원본 역할, 이름, 코드 모두 입력 시 활성)

## 주요 UX 흐름
1. 역할 목록에서 복사 버튼 클릭 → 다이얼로그 표시 (현재 선택 역할이 원본으로 자동 설정)
2. 원본 역할 확인 후 새 역할 이름·코드 입력
3. 복사 생성 버튼 클릭 → 원본 역할의 권한이 그대로 복사된 새 역할 생성
4. 새 역할이 목록에 추가되고 자동으로 선택된 상태로 전환

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 현재 선택 역할이 원본으로 설정된 상태 |
| 입력 완료 | 모든 필수 항목 입력 후 복사 생성 버튼 활성 |
| 이름 중복 | 이미 존재하는 역할명 입력 시 오류 메시지 |
| 생성 완료 | 다이얼로그 닫히고 새 역할 목록에 추가 |
