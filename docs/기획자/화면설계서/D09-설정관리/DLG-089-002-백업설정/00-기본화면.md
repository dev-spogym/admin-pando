## 화면 목적
자동 백업의 실행 주기, 시각, 데이터 보관 기간을 설정하는 다이얼로그입니다.

## 주요 구성요소
- "자동 백업 설정" 제목
- 자동 백업 활성화 여부 토글
- 백업 주기 선택 (매일 / 매주 / 매월)
  - 매주 선택 시: 요일 선택 (월~일)
  - 매월 선택 시: 날짜 선택 (1~28일)
- 백업 실행 시각 선택 (시간 입력)
- 백업 데이터 보관 기간 선택 (30일 / 90일 / 180일 / 무제한)
- 취소 버튼
- 저장 버튼

## 주요 UX 흐름
1. 데이터 백업·복원 화면에서 자동 백업 설정 변경 버튼 클릭 → 다이얼로그 표시
2. 현재 설정값이 자동으로 로드된 상태
3. 변경 후 저장 버튼 클릭 → 설정 저장 완료, 다이얼로그 닫힘
4. 취소 → 다이얼로그 닫힘, 변경 사항 폐기

## 화면 상태
| 상태 | 설명 |
|---|---|
| 표시 | 현재 백업 설정값 로드된 상태 |
| 비활성화 | 자동 백업 OFF 시 하위 설정 항목 비활성(회색) |
| 저장 완료 | 다이얼로그 닫히고 백업 화면에 변경 내용 반영 |
