## 화면 목적
센터 운영 데이터의 백업 주기를 설정하고, 수동 백업을 실행하며, 이전 백업 시점으로 복원하는 화면입니다.
데이터 손실 사고에 대비한 안전망으로 운영되며, 복원 이력을 통해 과거 복원 내역을 추적할 수 있습니다.

## 주요 구성요소

### 자동 백업 설정
- 자동 백업 활성화 여부 토글
- 백업 주기 선택 (매일 / 매주 / 매월)
- 백업 실행 시각 설정
- 백업 데이터 보관 기간 설정 (30일 / 90일 / 180일 / 무제한)
- 자동 백업 설정 변경 버튼 → 백업 설정 다이얼로그(DLG-089-002)

### 수동 백업
- 지금 백업 실행 버튼
- 마지막 백업 일시 표시
- 백업 데이터 다운로드 버튼 (최신 백업 파일)

### 백업 이력
- 백업 일시, 데이터 크기, 백업 방식(자동/수동), 상태 목록
- 각 이력 행에 복원 버튼 및 다운로드 버튼
- 기간 필터 (최근 1개월 / 3개월 / 전체)

### 복원 이력
- 과거 복원 실행 이력 (복원 일시, 복원한 백업 시점, 실행자)

## 계정별 차이
| 계정 | 권한 |
|---|---|
| 슈퍼관리자 | 전체 설정 조회, 백업 실행, 복원 실행 가능 |
| 센터장 | 소속 지점 백업 조회 및 수동 백업 실행 가능 (복원은 슈퍼관리자 승인 필요) |
| 매니저 | 접근 불가 |
| 기타 | 접근 불가 |

## 주요 UX 흐름
1. 화면 진입 시 자동 백업 설정값과 백업 이력 목록을 불러와 표시한다.
2. "지금 백업 실행" 버튼 클릭 → 백업 진행 중 표시 → 완료 후 이력 목록에 추가.
3. 이력 목록의 "복원" 버튼 클릭 → 복원 최종 확인 다이얼로그(DLG-089-001) 표시 → 확인 후 복원 진행.
4. 복원 완료 후 시스템이 재시작되고 선택한 시점의 데이터로 전환된다.
5. 자동 백업 설정 변경 버튼 클릭 → 백업 설정 다이얼로그(DLG-089-002) 표시.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 자동 백업 설정 및 이력 목록 표시 |
| 백업 진행 중 | 수동 백업 실행 중 (버튼 비활성, 진행 표시) |
| 백업 완료 | 완료 메시지 표시 후 이력 목록 갱신 |
| 복원 진행 중 | 복원 실행 중 화면 전체 처리 중 표시 |
| 이력 없음 | 백업 이력이 없을 때 안내 메시지 표시 |
