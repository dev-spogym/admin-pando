## 화면 목적
모든 화면에서 공통으로 사용되는 좌측 메뉴 영역으로, 시스템의 주요 기능으로 이동할 수 있는 네비게이션입니다.
계정 역할에 따라 접근 가능한 메뉴 항목이 다르게 표시됩니다.

## 주요 구성요소
- 브랜드 로고 (클릭 시 대시보드 이동)
- 지점명 표시
- 메뉴 항목 목록 (아이콘 + 텍스트)
- 현재 위치 강조 표시
- 접기/펼치기 버튼 (아이콘만 보기로 전환)

## 메뉴 구성
| 메뉴 | 이동 화면 |
|---|---|
| 대시보드 | 지점 현황 대시보드 |
| 회원관리 | 회원 목록 |
| 매출관리 | 매출 현황 |
| 수업관리 | 수업 캘린더 |
| 상품관리 | 상품 목록 |
| 시설관리 | 락커 관리 |
| 직원관리 | 직원 목록 |
| 마케팅 | 리드 관리 |
| 설정 | 센터 설정 |
| 본사관리 | 본사 대시보드 (권한 있는 계정만) |

## 계정별 차이
| 계정 유형 | 표시되지 않는 메뉴 |
|---|---|
| 슈퍼관리자 | 없음 (전체 접근) |
| 센터장 / 매니저 | 본사관리 일부 |
| 트레이너 | 매출관리, 직원관리, 설정 등 제한 |
| FC / 스태프 | 역할에 맞는 메뉴만 표시 |

## 주요 UX 흐름
1. 원하는 메뉴 클릭 → 해당 화면으로 이동
2. 현재 위치한 메뉴가 강조 표시됨
3. 좁은 화면 또는 접기 버튼 클릭 시 아이콘만 표시
4. 모바일에서는 햄버거 버튼으로 사이드바 열기/닫기

## 화면 상태
| 상태 | 설명 |
|---|---|
| 펼침 | 메뉴명 + 아이콘 모두 표시 |
| 접힘 | 아이콘만 표시 (마우스 올리면 툴팁) |
| 모바일 열림 | 전체화면 오버레이로 표시 |
| 모바일 닫힘 | 숨김 상태 |
