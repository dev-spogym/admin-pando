## 화면 목적
어느 화면에서든 Cmd+/ (Mac) 또는 Ctrl+/ (Windows) 를 누르면 현재 화면의 기획 의도와 설계 내용을 우측 패널로 확인할 수 있는 오버레이입니다.
고객사 및 팀원이 화면을 보면서 기획 내용을 바로 참조할 수 있습니다.

## 주요 구성요소
- 우측에서 슬라이드 인 되는 패널
- 화면명 + 카테고리 배지
- 탭 전환: 기능명세 / 화면설계
- 섹션별 아코디언 (현재 화면 관련 섹션 자동 강조)
- 내부 검색 (Ctrl+F)
- 전체보기 버튼 (패널 전체 화면 확장)
- 닫기 버튼 (또는 ESC)
- 읽기 진행률 표시바

## 주요 UX 흐름
1. Cmd+/ 단축키 → 우측 패널 슬라이드 인
2. 기능명세 탭: 현재 화면의 기획 의도·기능 설명 확인
3. 화면설계 탭: 화면 상태별 설계 내용 확인
4. 관련 섹션이 자동으로 강조·스크롤
5. Cmd+/ 재입력 또는 ESC / 바깥 클릭으로 닫기

## 화면 상태
| 상태 | 설명 |
|---|---|
| 닫힘 | 오버레이 숨김 |
| 로딩 | 문서 불러오는 중 |
| 기능명세 탭 | 기능 설명 표시 |
| 화면설계 탭 | 화면 상태별 설계 표시 |
| 문서 없음 | "문서 준비 중" 안내 |
