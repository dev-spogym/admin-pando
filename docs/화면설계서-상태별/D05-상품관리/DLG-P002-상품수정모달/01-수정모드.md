## 📝 디스크립션

### 사용 시점 (Trigger)
- SCR-P001 목록에서 행의 "수정" 버튼 클릭
- 기존 상품 데이터가 폼에 프리필된 상태

### 필드 정의

| 필드 | 타입 | 필수 | 기본값 |
|------|------|------|--------|
| 상품명 | text | Y | product.name |
| 상품 그룹 | Select | N | product.group_id |
| 판매가 | number | Y | product.price |
| 상품 유형 | Select | Y | product.type |
| 설명 | textarea | N | product.description |
| 판매 상태 | Toggle | Y | product.is_active |

### 인터랙션 (User Actions)
1. 필드 수정 후 "저장" 클릭 → UPDATE → `03-저장결과`
2. "취소" / X → 변경사항 폐기, 모달 닫기

### 비즈니스 룰

- 수정 후 SCR-P001 목록 자동 갱신
