```
구현:
const [panelOpen, setPanelOpen] = useState(false)
const [selectedProduct, setSelectedProduct] = useState<Product | null>(null)

행 클릭 시:
  setSelectedProduct(product)
  setPanelOpen(true)

패널 닫기:
  setPanelOpen(false)
  setSelectedProduct(null)

사용 컴포넌트: AppLayout, DataTable
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- `02-정상-데이터있음` 상태에서 테이블 행 클릭

### 필수 데이터
- `selectedProduct`: 클릭된 행의 product 객체

### 인터랙션 (User Actions)
1. X 버튼 클릭 → 패널 닫기 → `02-정상-데이터있음`
2. ESC 키 → 패널 닫기
3. 테이블 다른 행 클릭 → selectedProduct 교체

### 비즈니스 룰
