## 역할별 차이
없음 (모든 역할 동일한 스켈레톤). 버튼 자체는 역할별 조건부 렌더가 유지되지만 노출된 버튼은 전부 disabled.

## 상태 고유 인터랙션
- 없음. 클릭/키보드 입력 전부 무시(`pointer-events-none` on skeleton overlay).
- `Esc` 키만 허용 (사이드바 `/` 복귀 X, 현재 화면 유지).

## 비즈니스 룰 델타
- **타임아웃 10초**: 응답이 없으면 자동 `05-에러` 전환.
- **최소 로딩 시간 150ms**: Skeleton flash 방지(지연된 resolve는 즉시 `02-정상` 전환).
- 일일 동기화 `runDailySync()`는 이 상태에서 트리거하지 않음(대시보드 전용).

## 에지 케이스
- 한 쿼리만 성공하고 나머지 실패 → `products` 성공 시 `02-정상`으로 전환, 실패한 쿼리는 해당 섹션(분류 탭/판매 컬럼)에서 인라인 에러.
- 네트워크 재연결 이벤트(`online`): React Query 자동 refetch, 스켈레톤 유지.
- Role 판정 실패(세션 만료) → 로딩 중 401 인터셉터가 `/login` 으로 이동.
