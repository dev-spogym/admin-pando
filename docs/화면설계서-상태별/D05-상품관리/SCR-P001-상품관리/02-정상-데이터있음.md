## 📝 디스크립션

### 사용 시점 (Trigger)
- 로딩 완료 + `products.length > 0`

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 통계 카드 | `products` | `branch_id = ?`, 집계 |
| 상품 목록 | `products` | `branch_id = ?`, 탭/필터 조건 |
| 탭 목록 | `product_groups` | `branch_id = ?` |

### 인터랙션 (User Actions)
1. 탭 클릭 → 카테고리 필터 변경
2. 검색/필터 → 목록 재조회
3. 행 클릭 → 우측 상세 패널 오픈
4. 상품 등록 버튼 → DLG or SCR-P002 이동
5. 전지점 배포 버튼 → DLG-P001
6. 상품 가져오기 버튼 → DLG-P005

### 비즈니스 룰
- `hasPermission('canEditProduct') = false` 이면 등록/배포 버튼 숨김
- 페이지네이션: 기본 20건/페이지
- 가격은 `formatNumber` 적용 후 "원" 단위 표시

### 에지 케이스
- 필터 결과 0건: `FilterEmpty` 상태 인라인 표시
- 상품 100개 이상: 페이지네이션 활성

### 접근성 (A11y)
- DataTable `role="grid"`, 행 `role="row"`
- StatusBadge `aria-label="상태: 활성/비활성"`

### 연결 화면
- 패널 오픈: `06-패널열림`
- trainer 역할: `07-읽기전용`
- 에러 발생: `05-에러`
