## 📝 디스크립션

### 사용 시점 (Trigger)
- SCR-P001 상품 목록에서 "상품 등록" 버튼 클릭

### 필드 정의

| 필드 | 타입 | 필수 | 기본값 |
|------|------|------|--------|
| 상품명 | text | Y | 빈값 |
| 상품 그룹 | Select | N | 없음 |
| 판매가 | number | Y | 빈값 |
| 상품 유형 | Select | Y | 단품 |
| 설명 | textarea | N | 빈값 |
| 판매 상태 | Toggle | Y | 판매중 |

### 인터랙션 (User Actions)
1. 폼 입력 후 "등록" 클릭 → INSERT → `03-저장결과`
2. "취소" / X → 모달 닫기

### 비즈니스 룰
- 상품명 + 판매가 필수 입력
- owner, manager만 접근 가능
- 등록 후 SCR-P001 목록 갱신

### 연결 화면
