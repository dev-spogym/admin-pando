| 항목 | 값 |
|---|---|
| 화면 ID | SCR-P002 |
| 상태 코드 | `saving` |
| 다이어그램 노드 | F6 → `STATE_SAVING` |

## 상태 진입 조건

- `form.handleSubmit` zod 검증 통과 후 `mutation.mutate()` 호출
- `mutation.isPending === true`
- 최소 300ms 이상 지속 시 SavingOverlay 표시 (깜빡임 방지)

## 비주얼 델타

```
폼 상단/중단: 그대로 유지 (값 변경 불가, 시각적으로 60% 오버레이)
┌────────────────── 전체 화면 오버레이 ──────────────────┐
│                                                        │
│                    ⟳  저장 중...                       │
│                                                        │
└────────────────────────────────────────────────────────┘
Sticky footer primary: [저장중...] (disabled, Loader2 아이콘)
```

| 요소 | 저장중 값 |
|---|---|
| 모든 input/select/checkbox/radio | `readOnly` 또는 `disabled` |
| sticky footer 모든 버튼 | `disabled`, primary에 Loader2 애니메이션 |
| SavingOverlay | 표시 (`fixed inset-0 bg-white/60 backdrop-blur-[1px] z-40`) |
| 헤더 X 버튼 | disabled (저장 중 이탈 방지) |
| ValidationBanner | 숨김 (이미 통과한 상태) |

## 상태 고유 인터랙션

- 모든 입력 차단 (폼 dirty 변경 무시)
- ESC / 브라우저 뒤로가기 → `beforeunload` 방지 ("저장 중입니다. 잠시만 기다려주세요.")
- 네트워크 타임아웃 10초 초과 시 토스트 "처리가 오래 걸립니다. 잠시만 기다려주세요."

## 상태 고유 비즈니스 룰 (델타)

1. 중복 제출 절대 금지 — mutation 완료 전까지 submit 핸들러는 no-op
2. 성공 시 먼저 `invalidateQueries(['products'])` → toast 표시 → `moveToPage('/products')`
3. 실패 시 분기:
   - 400/422 (zod/비즈니스 룰) → `05-유효성에러` 상태로 전환, 첫 에러 포커스
   - 409 (중복) → name 필드 인라인 에러 + toast
   - 5xx/NETWORK → toast "등록/수정에 실패했습니다." + 폼 유지 (이전 상태로 복귀)
4. 낙관적 업데이트 없음 (`invalidateQueries`로 fresh fetch)

## 에지 케이스

- 사용자가 저장 중 브라우저 탭 닫기 시도 → `beforeunload` 경고
- 저장 중 네트워크 끊김 → mutation `onError('NETWORK')` → toast + 재시도 가능
- 저장 성공 직후 라우트 전환 중단 시도 → 이미 처리 완료, 다음 프레임에 navigate

## 바이브코딩 프롬프트 (델타)

```tsx
const mutation = useMutation({
  mutationFn: (v: ProductLegacyForm) =>
    isEdit ? updateProduct(id!, v) : createProduct(v),
  onSuccess: async () => {
    await qc.invalidateQueries({ queryKey: ['products'] });
    toast.success(isEdit ? '상품이 수정되었습니다.' : '상품이 등록되었습니다.');
    moveToPage('/products');
  },
  onError: (e: ApiError) => {
    if (e.errorCode?.startsWith('E409')) {
      form.setError('name', { message: `'${form.getValues('name')}' 상품명이 이미 존재합니다.` });
    } else if (e.errorCode === 'E400001' || e.errorCode === 'E400002') {
      injectServerErrors(form, e.fields);  // 05-유효성에러 상태 전환
    } else {
      toast.error('등록/수정에 실패했습니다.');
    }
  },
});

useEffect(() => {
  if (!saving) return;
  const handler = (e: BeforeUnloadEvent) => { e.preventDefault(); e.returnValue = ''; };
  window.addEventListener('beforeunload', handler);
  return () => window.removeEventListener('beforeunload', handler);
}, [saving]);

{saving && (
  <div role="status" aria-live="polite" aria-label="저장 중"
       className="fixed inset-0 bg-white/60 backdrop-blur-[1px] flex items-center justify-center z-40">
    <Loader2 className="size-6 animate-spin text-blue-600" aria-hidden />
    <span className="ml-2 text-sm font-medium text-gray-700">저장 중...</span>
  </div>
)}
```
