## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind v4 CRM 관리자 화면을 작성하라.
화면: SCR-P002 상품 등록/수정 — 상태: 저장 중 (saving=true)

파일: src/app/products/new/page.tsx, src/app/products/edit/page.tsx

구현:
const [saving, setSaving] = useState(false)

저장 핸들러:
  setSaving(true)
  try {
    await supabase.from('products').insert/update(...)
    toast.success(editId ? '상품이 수정되었습니다.' : '상품이 등록되었습니다.')
    router.push('/products')
  } catch {
    toast.error('저장에 실패했습니다.')
  } finally {
    setSaving(false)
  }

saving=true 동안:
- "등록"/"저장" 버튼: disabled + "저장 중..." 텍스트
- "취소" 버튼: disabled
- 폼 필드: pointer-events-none opacity-60

사용 컴포넌트: AppLayout, PageHeader, Button
사용 유틸: supabase, toast, useAuthStore
```
