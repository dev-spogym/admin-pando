## 📝 디스크립션

### 사용 시점 (Trigger)
- "등록"/"저장" 버튼 클릭 시 유효성 검사 실패

### 인터랙션 (User Actions)
1. 에러 필드 수정 → 인라인 에러 즉시 해제
2. 재저장 → 유효성 통과 시 `04-저장중`

### 비즈니스 룰

### 에지 케이스

- 배너 자동 소멸: 에러 해소 시 사라짐
