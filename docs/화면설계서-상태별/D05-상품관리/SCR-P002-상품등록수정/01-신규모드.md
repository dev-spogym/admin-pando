## 📝 디스크립션

### 사용 시점 (Trigger)
- `/products/new` 직접 진입
- SCR-P001 "상품 등록" / "첫 상품 등록하기" 버튼 클릭

### 필수 데이터
- `product_groups`: 카테고리 Select 옵션

### 인터랙션 (User Actions)
1. 폼 입력 → 실시간 유효성 표시
2. 취소 버튼 → 변경 있으면 DLG-P002, 없으면 즉시 이동
3. "등록" 버튼 → 유효성 통과 시 `04-저장중`

### 비즈니스 룰
- 상품명 필수 (최대 100자)
- 판매가 필수, 0원 이상
- 유형에 따라 횟수/기간 필드 동적 표시

### 에지 케이스
- 뒤로가기(브라우저) → 변경 감지 시 DLG-P002 트리거

### 연결 화면
- 저장 성공: SCR-P001 목록
