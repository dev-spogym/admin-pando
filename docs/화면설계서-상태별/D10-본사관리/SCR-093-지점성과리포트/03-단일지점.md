### 변경 요소
| 요소 | 변경 |
|---|---|
| 안내 배너 | `role="status"` 중립 정보성 배너 (blue-50) |
| 메달 아이콘 | **미표시** (비교 대상 없음) |
| 바 차트 2개 | 단일 지점 라인 차트(MoM 추이)로 **대체** |
| KPI 4카드 | 지점 합계 = 단일 지점 값 |
| 신규 지점 등록 링크 | primary/owner만 표시 |

## 역할별 차이
| 역할 | 차이 |
|---|---|
| primary/super | 안내 배너 + 신규 지점 등록 링크(`/branches` 이동) |
| owner | "브랜드에 지점 1개만 있습니다." + 본사 문의 |

## 인터랙션
- 월 선택 → refetch
- "+ 신규 지점 등록" 클릭 → `/branches` 이동 (primary/owner)
- 단일 지점 라인 차트: 월별 매출 12개월 추이

## 비즈니스 룰 델타
- 메달 아이콘 숨김 (비교 불가)
- 바 차트 대신 월별 추이 차트 표시
- 엑셀 다운로드 여전히 가능 (단일 행)
- 안내 배너는 새로고침 후에도 유지 (닫기 버튼 없음)

## 에지 케이스
- 해당 월 데이터 0: 안내 배너 유지 + "이달 데이터가 없습니다" 보조 문구
- 신규 지점 등록 직후: `02-정상-복수지점`으로 자동 전이
- primary가 자기 브랜드에 owner 강등 → owner view로 전환
