```
    </p>
  </div>

카운트다운 (기본 60초):
const [countdown, setCountdown] = useState(60)
useEffect(() => {
  const timer = setInterval(() => {
    setCountdown(prev => {
      if (prev <= 1) { clearInterval(timer); setIsLocked(false); return 0 }
      return prev - 1
    })
  }, 1000)
  return () => clearInterval(timer)
}, [])
```
