## 📝 디스크립션

### 사용 시점 (Trigger)
- `E_090_ST_02`: API 로드 성공 + 데이터 있음
- `E_090_ST_07`: 로딩 완료 후 정상 전환

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| KPI 카드 | `kpi_snapshots` | 최신 1건 |
| 지점 요약 | `branches` | status='active' |
| 공지 | `notices` | is_published=true, limit 3 |
| 할일 | `today_tasks` | 오늘 날짜 기준 |

### 인터랙션 (User Actions)
1. StatCard 클릭 → 해당 상세 화면
2. DataTable 행 클릭 → SCR-092 지점관리
3. "더보기" → 각 상세 페이지
4. 공지 항목 클릭 → 공지 상세

### 비즈니스 룰
- 전 역할 읽기 접근 가능
- 데이터는 최대 24시간 캐시 (ISR 또는 revalidate)
- 매출은 만원 단위 formatNumber 적용

### 에지 케이스
- 일부 카드 데이터 null → 0 또는 "-" 표시
- 지점 없음 → DataTable 빈 상태 메시지
