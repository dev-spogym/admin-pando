## 상태 고유 비즈니스 룰 델타
- **변화율 계산**: `delta = (curr - prev) / prev * 100`, prev=0일 때 0% 처리, `formatDelta(d)` 유틸로 `+5.3%` 형식
- **매출 포맷**: 1억+ `"X.X억"`, 1천만+ `"N,NNN만"`, 그 외 천단위 콤마
- **숫자 포맷**: `Intl.NumberFormat('ko-KR')` + 단위 ("명", "건")
- **만료예정 D-Day**: `Math.ceil((expiry - today)/day) = N` → `D-${N}` 표기
- **홀딩 남은 기간**: 동일 로직
- **생일자**: `MM-DD` 일치 + `status='ACTIVE'` + 나이 계산 (만 나이)
- **미수금 연체일**: `today - dueDate`
- **감사로그 상대시간**: date-fns `formatDistanceToNowStrict` ko locale

## 에지 케이스
- **트레이너 담당 회원 0명**: KPI는 0명 표시, 리스트 위젯은 빈 상태로 분기 → 03-빈상태 규칙 적용(해당 위젯만)
- **프론트 데스크**: 권한 허용 경로가 극도로 제한적 → §C 1개만 노출되어도 정상
- **멀티 지점 owner**: BranchSwitcher 노출되지만 첫 로그인 시 기본 지점 자동 선택
- **차트 데이터 모두 0**: 차트 위젯 내부에서 "데이터 없음" 빈 상태 (03-빈상태 세부 참조)
- **부분 실패**: 일부 위젯 에러 → 해당 위젯만 에러 UI, 나머지는 정상 유지
