### 변경 요소
| 요소 | 변경 |
|---|---|
| 폐점 배너 | 상단 amber (`role="status"`, `aria-live="polite"`) |
| 폐점 행 | `opacity-60` + cursor-not-allowed |
| 폐점 행 지점명 | `<span>` text-gray-500 |
| 폐점 행 메뉴 | "보기"만 (읽기 전용) |
| 폐점 상태 배지 | 🔴 폐점 (red-100) |
| KPI 활성 지점 | closed 제외 카운트 |

## 인터랙션
| 요소 | 동작 |
|---|---|
| 폐점 배너 | 읽기 전용 |
| 폐점 행 지점명 | 클릭 불가 + tooltip |
| 폐점 행 "보기" | `/branches/:id/archive` 읽기 전용 상세 |
| 폐점 행 기타 버튼 | disabled |
| 상태 필터 "폐점" | 폐점만 |
| 상태 필터 "운영중/임시휴업" | 폐점 제외 → `02-정상` |

## 비즈니스 룰 델타
- 정렬: 폐점은 하단 배치 (`ORDER BY status ASC, regDate DESC`)
- KPI 총 지점 수는 폐점 포함, 활성 지점 수는 제외
- 매출/회원 합계는 폐점 제외
- 통합 현황 탭: 폐점 제외 집계 + "폐점 N개 제외" 캡션
- 엑셀 export: 폐점 포함 (status 컬럼 구분)

## 에지 케이스
- 폐점 전환 중 타 사용자 동시 작업 → 409 → 경고 toast
- 모든 지점이 폐점: 전체 배너 + "모든 지점이 폐점 상태입니다"
- URL `?status=closed` 직접 입력: 필터 자동 적용
- audit_log `BRANCH_CLOSE` 보존 (복원 히스토리용)

## 바이브코딩 프롬프트 (델타)
```
const hasClosed = branches.some(b => b.status === 'closed');
const activeBranches = branches.filter(b => b.status !== 'closed').length;
const closedCount = branches.filter(b => b.status === 'closed').length;

{hasClosed && (
  <div role="status" aria-live="polite"
    className="rounded-md bg-amber-50 border border-amber-200 p-3 text-sm text-amber-800 flex items-start gap-2">
    <AlertTriangle className="h-5 w-5 flex-none"/>
    <div>
      <strong className="font-semibold">폐점 지점이 포함되어 있습니다 ({closedCount}개)</strong>
      <p className="mt-1 text-xs">폐점 지점은 조회 전용으로 표시되며 수정/전환이 제한됩니다.</p>
    </div>
  </div>
)}

render 컬럼 지점명:
row.status === 'closed' ? (
  <span title="폐점된 지점은 전환할 수 없습니다"
    className="text-gray-500 cursor-not-allowed">
    {v} <span className="text-xs text-gray-400">(폐점)</span>
  </span>
) : (<button onClick={() => switchBranch(row.id, row.name)}
    className="text-blue-600 hover:underline font-medium">{v}</button>)

row className:
className={cn('hover:bg-gray-50',
  row.status === 'closed' && 'opacity-60 cursor-not-allowed hover:bg-transparent')}

메뉴:
row.status === 'closed'
  ? <Button size="sm" variant="ghost" onClick={() => router.push(`/branches/${row.id}/archive`)}>보기</Button>
  : (<>관리/수정/비활성/상태변경</>)

정렬:
const sorted = [...branches].sort((a,b) => {
  if (a.status==='closed' && b.status!=='closed') return 1;
  if (a.status!=='closed' && b.status==='closed') return -1;
  return b.regDate.localeCompare(a.regDate);
});
```
