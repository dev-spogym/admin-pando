## 비즈니스 룰 델타
- 서버 middleware 우선 (403 + `/forbidden?from=/kpi`)
- 클라이언트 fallback: `ForbiddenCard`
- audit_log `PERMISSION_DENIED` 기록 (1분 throttle)
- 사이드바 `getMenuForRole`에서 "KPI 대시보드" 제외
- fc/trainer/staff/front에게는 대신 **"KPI 프리뷰 보드" 메뉴 노출**

## 에지 케이스
- manager → fc 강등: 이전 탭 새로고침 시 06 상태
- JWT 만료 + 권한 없음: `/login` 우선
- readonly: 3초 타이머로 자동 `/` redirect

## 바이브코딩 프롬프트 (델타)
```
// middleware.ts
if (pathname === '/kpi') {
  if (!user) return NextResponse.redirect('/login?redirect=/kpi');
  if (!['primary','superAdmin','owner','manager'].includes(user.role)) {
    logAudit({ action:'PERMISSION_DENIED', userId:user.id, targetType:'page', targetId:'SCR-094' });
    return NextResponse.redirect('/forbidden?from=/kpi');
  }
}

// 클라이언트 fallback
if (!['primary','superAdmin','owner','manager'].includes(user.role)) {
  return (
    <main className="min-h-screen flex items-center justify-center bg-gray-50 px-4">
      <div className="max-w-md w-full bg-white rounded-2xl shadow-sm ring-1 ring-gray-100 p-8 text-center">
        <div className="mx-auto h-16 w-16 rounded-full bg-red-100 flex items-center justify-center mb-4">
          <ShieldAlert className="h-8 w-8 text-red-600"/>
        </div>
        <h1 className="text-xl font-bold text-gray-900">접근 권한이 없습니다</h1>
        <p className="mt-3 text-sm text-gray-600">KPI 대시보드는 매니저 이상만 접근할 수 있습니다.</p>
        <div className="mt-4 rounded-lg bg-gray-50 p-3 text-xs text-gray-600">
          <p>현재 역할: <span className="font-mono">{user.role}</span></p>
          <p>필요 역할: <span className="font-mono">primary · superAdmin · owner · manager</span></p>
        </div>
        <p className="mt-4 text-xs text-blue-700 bg-blue-50 rounded-md p-2">
          대신 <strong>KPI 프리뷰 보드</strong>에서 역할별 핵심 지표를 확인할 수 있어요.
        </p>
        <div className="mt-6 flex gap-2 justify-center">
          <Button variant="primary" onClick={() => router.push('/kpi-preview')}>KPI 프리뷰 보드 보기</Button>
          <Button variant="secondary" onClick={() => router.push('/')}>대시보드로 이동</Button>
        </div>
      </div>
    </main>
  );
}
```
