- 15초 초과 timeout

### 변경 요소
- 상단 PageHeader 유지 (목표 설정 disabled)
- KPI 6섹션 전부 → ErrorCard 교체
- 코호트/퍼널 포함 전체 에러
- 3개 버튼: 다시 시도 / 부분 조회 / 문의하기

## 역할별 차이
- primary: 에러코드 + 요청 ID + 문의 링크
- owner/manager: 에러코드만

## 인터랙션
| 요소 | 동작 |
|---|---|
| [다시 시도] | 전 쿼리 invalidate |
| [부분 조회] | 성공한 쿼리만 렌더 → `03-일부실패` 상태로 우회 전이 |
| [문의하기] | mailto/Intercom + 에러코드/요청ID/타임스탬프 |

## 비즈니스 룰 델타
- React Query `retry: 3, retryDelay: exp backoff`
- Sentry 로그 + Datadog 모니터링 트리거
- 503 시 자동 30초 주기 재시도 (3회)
- 인증 만료 (401): 전역 인터셉터 → `/login`
- 15초 timeout → 에러 상태 전이
- 오프라인 감지 시 별도 문구

## 에지 케이스
- 401: 전역 인터셉터 처리
- 403 권한: `06-권한없음` 직행
- Supabase rate limit (429): "잠시 후 다시 시도해주세요" + 자동 backoff
- 부분 성공 (3개 이상 성공): `03-일부실패`로 판단 우선
- 15개 초과 실패 시만 05

## 바이브코딩 프롬프트 (델타)
```
const failedCount = queries.filter(q => q.isError).length;
const successCount = queries.filter(q => q.isSuccess).length;

if (failedCount >= 15) {
  const firstError = queries.find(q => q.isError)?.error;
  const errorCode = firstError?.errorCode || 'E500001';
  const isOffline = firstError?.type === 'NETWORK';
  const isMaintenance = errorCode === 'E503001';

  useEffect(() => {
    if (isMaintenance && retryCount < 3) {
      const t = setTimeout(() => { refetchAll(); setRetryCount(c => c+1); }, 30_000);
      return () => clearTimeout(t);
    }
  }, [isMaintenance, retryCount]);

  return (
    <div className="rounded-xl bg-rose-50 border border-rose-200 p-8 text-center">
      {isOffline ? <WifiOff className="h-10 w-10 text-rose-600 mx-auto mb-3"/>
       : isMaintenance ? <Construction className="h-10 w-10 text-amber-600 mx-auto mb-3"/>
       : <AlertCircle className="h-10 w-10 text-rose-600 mx-auto mb-3"/>}
      <h2 className="text-lg font-semibold text-gray-900">
        {isOffline ? '네트워크 연결이 끊어졌습니다'
         : isMaintenance ? '서비스 점검 중입니다'
         : 'KPI 데이터를 불러오지 못했습니다'}
      </h2>
      <p className="mt-2 text-sm text-gray-600">
        {errorMessageMap[errorCode] ?? '일시적인 서버 오류입니다.'}
      </p>
      <p className="mt-2 font-mono text-xs text-gray-500">
        에러코드: {errorCode} {user.role==='primary' && `· 요청 ID: ${requestId}`}
      </p>
      <div className="mt-4 flex gap-2 justify-center">
        <Button variant="primary" onClick={() => refetchAll()}>
          <RefreshCw className="h-4 w-4 mr-1"/> 다시 시도
        </Button>
        {successCount > 0 && (
          <Button variant="secondary" onClick={() => setForcePartialRender(true)}>
            부분 조회 ({successCount}/18)
          </Button>
        )}
        {user.role === 'primary' && (
          <Button variant="ghost" onClick={contactSupport}>문의하기</Button>
        )}
      </div>
    </div>
  );
}
```
