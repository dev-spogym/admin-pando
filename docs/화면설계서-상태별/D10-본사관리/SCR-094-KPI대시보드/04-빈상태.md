## 비즈니스 룰 델타
- `totalMembers === 0` && `totalPtSessions === 0` && `totalConsultations === 0` 이면 빈 상태 판단
- 매출 게이지: target 없으면 "목표를 설정하세요", target 있고 current=0이면 0% red
- 목표 설정은 데이터 없어도 가능 (미래 목표)
- 안내 배너는 닫기 불가 (데이터 생길 때까지 유지)
- 새로고침 후 값이 생기면 자동 02로 전이

## 에지 케이스
- 부분 데이터: 회원만 있고 매출 없음 → `02-정상`으로 진입 (0값 허용)
- 완전 0: 이 상태 유지
- manager가 본인 지점에 회원 1명만 있고 PT/매출 0: `02-정상` (회원 1+이면)
- audit_log `EMPTY_KPI_VIEW` 기록 (신규 지점 감지)
