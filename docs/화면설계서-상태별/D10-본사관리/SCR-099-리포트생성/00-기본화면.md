## 1. 화면 목적 (Why)

**정기/수시 운영 리포트를 자동 생성하고 PDF/CSV로 다운로드**하여 본사·경영진 보고에 활용.
- 일간(최근 7일) / 주간(최근 4주) / 월간(최근 3개월) 자동 생성 목록
- 수시 생성 모달: 리포트 유형 + 기간 지정
- 각 리포트: KPI 4개 (총 매출 / 전체 회원 / 신규 회원 / 총 출석)
- 상세 모달에서 CSV 내보내기

## 8. 비즈니스 룰

1. **자동 생성 주기**:
   - 일간: 매일 00:30 UTC (지점별)
   - 주간: 월요일 00:30 UTC
   - 월간: 매월 1일 00:30 UTC
2. **기본 목록**: 일간 7건 + 주간 4건 + 월간 3건 = 최근 14건
3. **수동 생성**:
   - 필드: type, startDate, endDate
   - 검증: startDate ≤ endDate, 최대 365일 범위
4. **상태 전이**: GENERATING → READY (평균 30초) → 다운로드 가능 / FAILED
5. **PDF 포맷**: 표지 + KPI 4카드 + 그래프 + 상세표
6. **CSV**: KPI + 거래 상세
7. **audit_log**: `REPORT_GENERATE`, `EXPORT` 기록
8. **권한별 범위**: primary=전 지점 혼합, owner/manager=본인 지점만

```
        await api.post('/reports/generate', form);

        setGenerateOpen(false);

    <DetailModal report={detailReport}
      onClose={() => setDetailReport(null)}

━━ ReportCard ━━
function ReportCard({ report, onDetail, onDownload }) {
  const typeLabel = { DAILY:'일간', WEEKLY:'주간', MONTHLY:'월간' };
  const typeClass = { DAILY:'bg-blue-50 text-blue-600', WEEKLY:'bg-green-50 text-green-600', MONTHLY:'bg-purple-50 text-purple-600' };
  return (
    <article className="bg-white rounded-xl ring-1 ring-gray-100 p-5 hover:ring-blue-200 transition-all">
      <div className="flex items-center justify-between mb-3">
        <span className={cn('text-xs font-medium px-2 py-0.5 rounded-full', typeClass[report.type])}>
          {typeLabel[report.type]}
        </span>
        <StatusBadge variant={report.status==='READY'?'success':report.status==='GENERATING'?'warning':'error'}>
          {report.status}
        </StatusBadge>
      </div>
      <h3 className="text-base font-semibold text-gray-900">{report.periodLabel}</h3>
      {report.branchName && <p className="text-xs text-gray-500">{report.branchName}</p>}
      <dl className="mt-3 grid grid-cols-2 gap-2 text-sm">
        <div>
          <dt className="text-xs text-gray-500">매출</dt>
          <dd className="tabular-nums font-medium">{formatAmount(report.kpi.totalRevenue)}</dd>
        </div>
        <div>
          <dt className="text-xs text-gray-500">신규 회원</dt>
          <dd className="tabular-nums font-medium">{formatNumber(report.kpi.newMembers)}명</dd>
        </div>
      </dl>
      <div className="mt-4 flex gap-2">
        <Button size="sm" variant="secondary" onClick={onDetail} disabled={report.status !== 'READY'}>상세</Button>
        <Button size="sm" variant="primary" onClick={onDownload} disabled={report.status !== 'READY'}>
          <Download className="h-4 w-4 mr-1"/> PDF
        </Button>
      </div>
    </article>
  );
}
```
