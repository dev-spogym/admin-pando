```
Next.js 15 App Router + TypeScript + Tailwind v4 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-C002 수업관리 — 상태: 수업 기록 탭

수업 기록 탭 DataTable:
- 컬럼: 날짜 | 수업명 | 강사 | 회원 | 상태 | 시작시간 | 종료시간 | 액션
- 상태 배지 (StatusBadge):
  · scheduled: bg-blue-100 text-blue-700 "예정"
  · in_progress: bg-yellow-100 text-yellow-700 "진행중"
  · completed: bg-green-100 text-green-700 "완료"
  · no_show: bg-red-100 text-red-700 "노쇼"
  · cancelled: bg-gray-100 text-gray-500 "취소"
- 액션: Eye 아이콘 버튼 → DLG-C005 수업기록 상세

SearchFilter (수업 기록용):
- 날짜 범위 DatePicker (시작일 ~ 종료일)
- 상태 Select: 전체 | 예정 | 진행중 | 완료 | 노쇼 | 취소
- 강사 Select
- 수업명 검색 input

페이지네이션: 20행씩

데이터:
- SELECT ls.*, c.name as class_name, u.name as instructor_name,
    m.name as member_name
  FROM lesson_schedules ls
  JOIN classes c ON ls.class_id = c.id
  JOIN users u ON ls.instructor_id = u.id
  LEFT JOIN users m ON ls.member_id = m.id
  WHERE ls.branch_id=? AND ls.status IN(?) AND ls.start_at BETWEEN ? AND ?
  ORDER BY ls.start_at DESC

사용 유틸:
- DataTable, StatusBadge, SearchFilter from '@/components/ui'
- formatDate, formatTime from '@/lib/utils'
- Eye from 'lucide-react'
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- "수업 기록" 탭 클릭 후 데이터 로드 완료
- 상태/날짜 필터 적용 후 결과 있음

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 수업 기록 | `lesson_schedules` | branch_id, 날짜범위, 상태 |
| 수업 정보 | `classes` | JOIN |
| 강사/회원 | `users` | JOIN |

### 인터랙션 (User Actions)
1. Eye 버튼 클릭 → DLG-C005 수업기록 상세
2. 상태 필터 → 해당 상태만 표시
3. 날짜 범위 필터 → 해당 기간 기록만 표시
4. 수업 목록 탭 클릭 → `02-수업목록-정상`
