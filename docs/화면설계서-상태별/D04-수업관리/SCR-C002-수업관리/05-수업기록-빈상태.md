## 📝 디스크립션

### 사용 시점 (Trigger)
- 기록 탭에서 특정 필터(날짜/상태/강사) 적용 후 결과 없음
- 신규 지점: 수업 진행 이력 없음

### 인터랙션 (User Actions)
1. 필터 초기화 버튼 → 기본 조건으로 재조회
2. 날짜 범위 변경 → 재조회

### 비즈니스 룰
- 필터 없는 초기 상태에서 빈 상태: "아직 수업 기록이 없습니다." 메시지
- 필터 적용 후 빈 상태: "조건에 맞는 기록이 없습니다." + 초기화 버튼

### 연결 화면
- 필터 초기화: `03-수업기록-정상` 또는 유지
- 목록 탭: `02-수업목록-정상`
