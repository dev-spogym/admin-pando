## 상태 고유 인터랙션
1. `[+ 수업 등록]` 클릭 → `DLG-C003` (create, 빈 폼)
2. PageHeader 상단의 `[+ 수업 등록]` 버튼도 동일 동작 (권한 있으면 중복 노출)
3. 탭 전환 → `03-수업기록-정상` 또는 `05-수업기록-빈상태`

## 상태 고유 비즈니스 룰 델타
- **신규 지점 온보딩**: 지점 최초 가입 시 이 상태 진입 → 온보딩 흐름 연결 (선택적 배너)
- **삭제로 인한 0**: stats도 함께 0 (전체 수업 삭제)
- **검색 결과 0**: `02-정상`의 필터 결과 분기, 본 상태는 아님

## 에지 케이스
- 탭 전환 시 수업 기록은 있을 수 있음 → `03` 상태 정상
- 신규 지점 + 온보딩 완료 전: 배너 "템플릿에서 수업을 불러오시겠어요?" 옵션 (future)
