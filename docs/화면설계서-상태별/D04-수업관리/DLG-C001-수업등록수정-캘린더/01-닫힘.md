## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind v4 기반 모달 컴포넌트를 작성하라.
화면: DLG-C001 수업등록/수정(캘린더) — 상태: 닫힘 (초기)

파일: src/components/classes/LessonCalendarModal.tsx

모달 상태 관리 (부모 SCR-C001에서 제어):
- isOpen: false (초기값)
- mode: 'create' | 'edit'
- selectedLesson: LessonSchedule | null

트리거:
- 수업 등록 버튼 클릭: setIsOpen(true), setMode('create'), setSelectedDate(clickedDate)
- 캘린더 빈 셀 클릭: setIsOpen(true), setMode('create'), setSelectedDate(cellDate)
- 캘린더 이벤트 클릭: setIsOpen(true), setMode('edit'), setSelectedLesson(event.lesson)

모달 컴포넌트 (isOpen=false일 때):
- null 반환 또는 hidden 처리
- createPortal로 body에 마운트

사용 패턴:
const [isOpen, setIsOpen] = useState(false)
const [mode, setMode] = useState<'create'|'edit'>('create')
const [selectedLesson, setSelectedLesson] = useState<LessonSchedule|null>(null)

<LessonCalendarModal
  isOpen={isOpen}
  mode={mode}
  lesson={selectedLesson}
  onClose={() => { setIsOpen(false); setSelectedLesson(null); }}
  onSuccess={() => { setIsOpen(false); refetchCalendar(); }}
/>
```
