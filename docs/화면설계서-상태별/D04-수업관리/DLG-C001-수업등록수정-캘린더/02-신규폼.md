## 상태 고유 인터랙션

1. **템플릿 선택**: `template` 변경 → `name`, `capacity`, `room` 자동 채움 + `isDirty=true`
2. **시작 시간 입력**: `endTime` 비어있으면 `start+1h` 자동 제안 (사용자 덮어쓰기 가능)
3. **요일 토글**: 반복 설정 (월~일). 1개 이상이면 백엔드에서 시리즈 생성
4. **좌석 행/열 > 0**: "총 N석 그리드 생성" 안내 표시 (`aria-live="polite"`)
5. **일정 분류 "없음"**: `setFormScheduleCategory("")`
6. **대상 유형 토글**: '회원'|'비회원'|'직원' → 관련 필드 표시
7. **참여자 검색**: 300ms 디바운스 → `supabase.from('members').ilike` → 드롭다운 → 클릭 시 추가 (max 5)
8. **색상 팔레트 클릭**: `color` 갱신 + 3px ring + 체크 SVG
9. **저장 버튼**: 유효성 통과 + 시간 충돌 확인 → INSERT → `05-저장성공`
10. **취소/ESC/X**: `isDirty` → `04-미저장확인`, 아니면 `01-닫힘`

## 상태 고유 비즈니스 룰 델타

- `isValid` 초기 false (필수 4개 비어있음)
- 시간 충돌 감지: 저장 직전에만 수행 (폼 입력 중에는 계산 X)
- 종료 자동 계산: 시작만 있을 때 `startTime +1h` 자동 제안 (`setValue('endTime', ...)` with `shouldDirty:false`)
- 반복 설정은 저장 시 백엔드 트랜잭션 (1개 INSERT로 여러 row 생성, 이 모달 관점에서는 단일 저장)
- 참여자 최대 5명 UI 상한
- 좌석 `rows*cols > 100` 경고: 실사용 한도 초과 알림(기능명세 10×10)

## 에지 케이스

- **템플릿 삭제됨**: Select에 stale 옵션 → 선택 시 toast "삭제된 템플릿입니다" + reset
- **강사 목록 빈 배열**: 경고 배너 "담당 강사를 먼저 등록하세요" + 저장 disabled
- **빈 셀 클릭이 월간 뷰**: `handleDateClick`이 먼저 `changeView('timeGridDay')`로 이동 후 모달 미오픈 (본 상태 진입 안 됨)
- **같은 시작/종료 시간**: Zod `refine` 실패 → `06-유효성에러`
- **세션 만료(401)**: DLG-000 오픈 + 이 모달 정리
- **반복 0개**: 단일 INSERT
