파일: src/components/classes/LessonCalendarModal.tsx

## 📝 디스크립션

### 사용 시점 (Trigger)
- mode='create' 로 모달 열림 시

### 필수 데이터 (폼 렌더링용)
| 항목 | 소스 |
|------|------|
| 강사 목록 | `users` role=instructor |
| 수업 분류 | enum 고정값 |

### 인터랙션 (User Actions)
1. 필드 입력 → 실시간 상태 업데이트
2. 저장 버튼 → 유효성 검사 → API
3. X/취소 → `04-미저장확인` (변경 있을 때) / 바로 닫힘 (변경 없을 때)
4. ESC 키 → X 버튼과 동일

### 비즈니스 룰
- 필드 유효성: 순차 검증 (수업명 → 강사 → 날짜 → 시간 → 정원)
- 시간 충돌: 동일 강사, 동일 시간 겹침 시 confirm 경고
- 종료 시간 자동 계산 (시작 + 수업시간)

### 에지 케이스
- 강사 목록 로드 실패: Select에 에러 메시지
- 빈 셀 클릭으로 열린 경우: 해당 날짜 기본값 주입

### 연결 화면
- 저장: `05-저장성공`
- 닫기(변경있음): `04-미저장확인`
- 닫기(변경없음): `01-닫힘`
- 유효성 실패: `06-유효성에러`
