## 📝 디스크립션

### 사용 시점 (Trigger)
- 로딩 완료 후 평가 데이터 있음

### 인터랙션 (User Actions)
1. 코멘트 "더보기" → 인라인 확장
2. 공개여부 Toggle → PATCH
3. 검색/필터 적용
4. 강사별 카드 클릭 → 해당 강사 필터 적용

### 비즈니스 룰
- 회원 앱에서 수업 완료 후 평가 제출
- 비공개 평가: 회원 앱에 미노출 (관리자만 조회)
- 평점 3 이하 → 자동 플래그 (빨간 행 하이라이트)

### 에지 케이스
- 코멘트 없는 평가: 별점만 표시
- 익명 평가 옵션 (향후)
