## 📝 디스크립션

### 사용 시점 (Trigger)
- 로딩 완료 후 녹화 데이터 있음

### 필수 데이터
| 블록 | 테이블/스토리지 |
|------|--------|
| 녹화 목록 | `class_recordings` |
| 파일 | Supabase Storage |
| 수업 정보 | `lesson_schedules` JOIN `classes` |

### 인터랙션 (User Actions)
1. 카드 클릭 → 비디오 플레이어 모달
2. Download → 파일 다운로드
3. Trash2 → confirm → 스토리지 + DB 삭제
4. Eye → 공개/비공개 토글
5. 업로드 → 파일 선택 + Supabase Storage 업로드

### 비즈니스 룰
- 공개 녹화: 회원 앱에서 조회 가능 (향후)
- 파일 크기 제한: 500MB per file
- 지원 형식: mp4, mov, webm
- 스토리지 초과: 경고 배너 "저장 공간이 부족합니다."

### 에지 케이스
- 썸네일 없는 경우: Video 아이콘 플레이스홀더
- 파일 URL 만료: 서명된 URL (1시간) 자동 갱신
