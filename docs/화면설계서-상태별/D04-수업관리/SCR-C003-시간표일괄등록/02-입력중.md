```
Next.js 15 App Router + TypeScript + Tailwind v4 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-C003 시간표일괄등록 — 상태: 입력 중

폼 입력 중 상태:

- 필수 필드 미입력 시 에러 없음 (blur 또는 submit 시에만 에러 표시)

- 수업: 필수 선택
- 강사: 필수 선택

생성 예상 건수 계산 (미리보기 전 힌트):
- 기간 + 요일 입력 완료 시: "예상 수업 수: N건" 텍스트 표시 (우측 상단)

사용 유틸:
- react-hook-form 또는 useState 기반 폼
- differenceInCalendarDays from 'date-fns'
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- 폼 필드 최초 입력 시
- 미리보기 클릭 → 유효성 실패 후 수정 중

### 인터랙션 (User Actions)
1. 미리보기 버튼 클릭 → 유효성 체크 → 통과 시 `03-미리보기활성`
2. 요일 체크박스 토글 → 예상 건수 실시간 업데이트
3. 기간 변경 → 예상 건수 갱신

### 비즈니스 룰
- 종료일 < 시작일 입력 불가 (DateRangePicker 제한)
- 최대 기간: 3개월 (90일)
- 예상 건수 0이면 경고: "선택한 기간/요일에 생성 가능한 수업이 없습니다."

### 연결 화면
- 미리보기 통과: `03-미리보기활성`
- 미리보기 실패: 인라인 에러 + 입력 중 유지
