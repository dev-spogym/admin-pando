데이터:
- SELECT w.*, m.name as member_name, ls.start_at, c.name as class_name
  FROM waitlists w
  JOIN users m ON w.member_id=m.id
  JOIN lesson_schedules ls ON w.lesson_schedule_id=ls.id
  JOIN classes c ON ls.class_id=c.id
  WHERE w.branch_id=? ORDER BY w.created_at ASC
