## 📝 디스크립션

### 사용 시점 (Trigger)
- 캘린더에서 수업이 없는 날짜 셀 클릭
- 날짜 이동 후 해당 날짜에 `lesson_schedules` 레코드 없음
- 필터 적용 후 선택 날짜에 조건 맞는 수업 없음

### 필수 데이터
| 블록 | 설명 |
|------|------|
| selectedDate | 클릭한 날짜 (Date 객체) |
| selectedDateLessons | 빈 배열 `[]` |

### 인터랙션 (User Actions)
1. "수업 등록" 버튼 클릭 → DLG-C001 열림 (해당 날짜 자동 세팅)
2. 다른 날짜 클릭 → 패널 데이터 갱신 (수업 있으면 목록 표시)
3. 캘린더 이벤트 클릭 → DLG-C002 열림 (패널 상태 무관)

### 비즈니스 룰
- 빈 상태에서도 캘린더 좌측은 정상 동작
- readonly 역할: 수업 등록 버튼 미노출
- 날짜 헤더는 항상 표시 (빈 상태 여부 무관)

### 에지 케이스
- 필터(강사/분류) 적용으로 빈 상태: "필터 조건에 맞는 수업이 없습니다." 메시지 변형 가능
- 주말(공휴일) 클릭: 동일하게 빈 상태 처리

### 접근성 (A11y)
- 빈 상태 영역 `role="status"`
- 아이콘 `aria-hidden="true"`
- 버튼 `aria-label="선택한 날짜에 수업 등록"`

- 이전: `02-정상-데이터있음`
