## 상태 고유 인터랙션
1. 날짜 `< >` 이동 → 이벤트 있는 날로 가면 02 전환
2. [+ 이 날짜에 수업 등록] → `DLG-C001` (create, date 사전입력)
3. 좌측 캘린더 다른 날짜 클릭 → `selectedDate` 변경

## 상태 고유 비즈니스 룰 델타
- 빈 판정: `events.filter(e=>isSameDay(e.startTime,selectedDate)).length===0`
- CTA 사전 입력: `date = format(selectedDate,'yyyy-MM-dd')`, `startTime:'09:00', endTime:'10:00'`
- 캘린더 전체 빈 ≠ 이 상태 (그 경우 02의 이벤트 0 분기)

## 에지 케이스
- 과거 날짜 선택 → CTA 숨김
- trainer + 다른 강사 수업만 → trainer 관점 0건 → 빈 상태
- 필터 적용 후 0건 → "선택한 필터에 해당하는 수업이 없습니다." (다른 메시지)

## 🧩 바이브코딩 프롬프트 (델타)
```
상속: 02-정상-데이터있음.md
상태: calendar-right-empty

델타:
- RightPanel:
  const eventsOfDate = useMemo(() =>
    events.filter(e => isSameDay(parseISO(e.startTime), selectedDate))
  , [events, selectedDate]);

  const isPastDate = isBefore(selectedDate, startOfToday());
  const isFilterActive = filter.staffId||filter.lessonId||filter.categories.length>0;

  {eventsOfDate.length===0 ? (
    <EmptyState
      icon={<BookOpen className="size-8 text-gray-300" />}
      title={isFilterActive
              ? '선택한 필터에 해당하는 수업이 없습니다.'
              : '해당 날짜에 수업이 없습니다.'}
      action={canCreateClass(role) && !isPastDate && (
        <Button variant="secondary" size="sm"
          onClick={() => openAdd({
            date: format(selectedDate,'yyyy-MM-dd'),
            startTime:'09:00', endTime:'10:00',
          })}>
          + 이 날짜에 수업 등록
        </Button>
      )}
    />
  ) : (
    <ul className="space-y-1">{eventsOfDate.map(renderEventCard)}</ul>
  )}

- EmptyState 공통:
  <div role="status" aria-live="polite"
       className="flex flex-col items-center justify-center py-10 text-center space-y-2">
    {icon}
    <p className="text-sm text-gray-500">{title}</p>
    {action}
  </div>

- 필터 해제:
  {isFilterActive && <Button variant="ghost" size="sm" onClick={resetFilter}>필터 초기화</Button>}
```
