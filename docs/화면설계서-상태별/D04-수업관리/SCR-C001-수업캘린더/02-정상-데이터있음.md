## 📝 디스크립션

### 사용 시점 (Trigger)
- 로딩 완료 후 `lesson_schedules.length > 0`
- 필터 변경 후 재조회 결과 데이터 있음
- 월/주 이동 후 해당 기간에 수업 존재

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 수업 이벤트 | `lesson_schedules` JOIN `classes` | branch_id, 날짜범위, 필터조건 |
| 강사 정보 | `users` | role='instructor' |
| 우측 패널 | `lesson_schedules` | 선택 날짜 |
| 미승인 배너 | `lesson_schedules` | status='pending' |

### 인터랙션 (User Actions)
1. 캘린더 이벤트 클릭 → DLG-C002 일정상세 모달 열림
2. 빈 날짜 셀 클릭 → DLG-C001 수업등록 모달 (해당 날짜 기본값 주입)
3. 수업 등록 버튼 클릭 → DLG-C001 열림
4. 뷰 전환 → 월간/주간/일간 전환
5. 강사 필터 → 해당 강사 수업만 표시
6. 월/주 이동 화살표 → 날짜 범위 이동 + 재fetch

### 비즈니스 룰
- dayMaxEvents=3 초과 시 "+N more" 링크 → 해당 날짜 일간 뷰로 이동
- PT=파란색, GX=보라색, 기타=초록색 이벤트 색상 구분
- 미승인(pending) 이벤트: opacity-70, amber 레이블
- readonly 역할: 등록/수정 버튼 숨김

### 에지 케이스
- 같은 시간 이벤트 3개 초과: +N more 처리
- 이벤트 제목 길면 truncate (max 15자)
- 강사 필터 + 수업분류 필터 동시 적용 가능

### 접근성 (A11y)
- 이벤트 `role="button"` + `aria-label="{수업명} {시간}"`
- 캘린더 그리드 `role="grid"`
- 뷰 전환 탭 `role="tablist"`

### 연결 화면
- 이전: `01-로딩`
- 이벤트 클릭: DLG-C002 일정상세
- 등록 버튼: DLG-C001 수업등록/수정
- 미승인 배너 클릭: `04-미승인배너` 상태
