## 상태 고유 비즈니스 룰 델타
- **카테고리 색상 매핑**: `CATEGORY_COLORS` 7색
- **정원 표시**: ≥100% [마감] rose / 80~99% [여유] amber / <80% [여유] emerald
- **편집 가능**: `isEventEditable`로 과거/진행/2시간내 판정, lock 아이콘 + tooltip
- **드래그 충돌**: `detectConflict` + `window.confirm`, 취소 시 `info.revert()`
- **trainer 제약**: 다른 강사 이벤트 `eventDidMount`에서 opacity-60, editable=false
- **미승인 이벤트**: opacity-70 + 배지
- **useCalendarFilter** 전역 store → 탭 이동 후에도 필터 유지

## 에지 케이스

상속: 00-기본화면.md §14

델타:
- toFullCalendarEvent:
  function toFullCalendarEvent(e: ScheduleEvent): EventInput {
    const color = e.color || CATEGORY_COLORS[e.category ?? '기타'];
    return {
      id: String(e.id), title: e.title, start: e.startTime, end: e.endTime,
      backgroundColor: color+'22', borderColor: color, textColor: '#111827',
      extendedProps: e,
      classNames: [
        e.approvalStatus === 'pending' && 'fc-event-pending',
        role==='trainer' && e.instructorId!==user.id && 'fc-event-foreign',
        !isEventEditable(e) && 'fc-event-locked',
      ].filter(Boolean),
    };
  }

- globals.css:
  .fc-event-pending { opacity: .7; }
  .fc-event-pending::after { content:'미승인'; font-size:8px; color:#d97706; }
  .fc-event-foreign { opacity:.6; pointer-events:none; }
  .fc-event-locked { cursor:not-allowed; }

- eventContent (월간):
  function renderEventContent(arg) {
    const e = arg.event.extendedProps as ScheduleEvent;
    return (
      <div className="flex items-center gap-1 px-1 overflow-hidden">
        <span className="size-1.5 rounded-full" style={{background:e.color}} />
        <span className="text-[10px] tabular-nums text-gray-600">{format(e.startTime,'HH:mm')}</span>
        <span className="text-xs font-semibold text-gray-900 truncate">{e.title}</span>
        {e.approvalStatus==='pending' && <span className="text-[8px] text-amber-600">미승인</span>}
      </div>
    );
  }

- handleEventDrop (optimistic):
  async (info) => {
    const e = info.event.extendedProps as ScheduleEvent;
    if (!canEditEvent(role,e,user.id)) { info.revert(); toast.warning('본인 수업만 수정할 수 있습니다.'); return; }
    if (!isEventEditable(e)) { info.revert(); toast.warning('과거/시작된 수업은 수정할 수 없습니다.'); return; }
    const newStart = info.event.start!.toISOString();
    const newEnd = info.event.end!.toISOString();
    if (detectConflict(e.instructorId, newStart, newEnd, e.id)) {
      if (!window.confirm(`${e.instructorName}의 시간과 겹칩니다. 계속하시겠습니까?`)) { info.revert(); return; }
    }
    try {
      await update({ id: e.id, patch: { startTime: newStart, endTime: newEnd } });
      toast.success('일정이 이동되었습니다.');
    } catch { info.revert(); toast.error('일정 이동에 실패했습니다.'); }
  };

- RightPanel 카드:
  <button onClick={()=>openDetail(e)}
          className="w-full text-left rounded-md hover:bg-gray-50 px-3 py-2 space-y-1">
    <div className="flex items-center gap-2">
      <span className="size-1.5 rounded-full" style={{background:e.color}} />
      <span className="text-sm font-medium truncate">{e.title}</span>
      <Badge variant={e.currentCount>=e.capacity?'rose':'emerald'}>
        {e.currentCount>=e.capacity?'마감':'여유'}
      </Badge>
    </div>
    <div className="text-xs tabular-nums text-gray-600">
      {format(e.startTime,'HH:mm')}~{format(e.endTime,'HH:mm')}
    </div>
    <div className="text-xs text-gray-600">{e.instructorName}  {e.currentCount}/{e.capacity}명</div>
  </button>

- role==='trainer' 초기 필터:
  useEffect(() => {
    if (role==='trainer') setFilter(f=>({...f, staffId:user.id}));
  }, [role, user.id]);
