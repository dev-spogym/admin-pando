## 📝 디스크립션

### 사용 시점 (Trigger)
- 로딩 완료 후 그룹수업 데이터 있음

### 필수 데이터
| 블록 | 테이블 |
|------|--------|
| 그룹수업 목록 | `classes` (type=GX/group) |
| 수업 횟수 | `lesson_schedules` |
| 예약 수 | `lesson_bookings` |

### 인터랙션 (User Actions)
1. 기간 필터 변경 → 재조회
2. 수업명 검색 → 필터링
3. 컬럼 정렬 → 해당 컬럼 기준 정렬

### 비즈니스 룰
- 충원율 80% 이상: 초록, 50~79%: 노랑, 50% 미만: 빨강
- 휴강 수업: 회색 처리, 정렬 하단으로

### 에지 케이스
- 정원 0인 수업: 충원율 N/A 표시
- 이번달 수업 없는 수업: 수업수 0, 충원율 0%

### 연결 화면
- 결과 없음: `03-빈상태`
- 에러: `04-에러`
