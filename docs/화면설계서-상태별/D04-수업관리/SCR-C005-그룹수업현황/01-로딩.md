```
데이터:
- SELECT c.*, COUNT(ls.id) as lesson_count,
    COUNT(lb.id) as booking_count
  FROM classes c
  LEFT JOIN lesson_schedules ls ON c.id=ls.class_id
  LEFT JOIN lesson_bookings lb ON ls.id=lb.lesson_schedule_id
  WHERE c.branch_id=? AND c.type IN ('GX','group')
  GROUP BY c.id
```
