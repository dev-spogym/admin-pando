## 상태 진입 조건
- `mutation.isSuccess === true` → `success` 또는 `partial` 분기
- `mutation.isError === true` → `error`
- `response.failed.length === 0` → `success`
- `response.failed.length > 0 && response.updated > 0` → `partial`
- `response.updated === 0` → `error`

## 상태 고유 인터랙션

### 공통
- X/취소/Esc/Overlay → 모달 닫기 (`isSubmitting=false`이므로 허용)

### partial 전용
- "실패 회원만 선택 유지" 클릭 → `onSuccess({ selectedIds: failed.map(f=>f.id) })` 호출 → 부모가 selectedRows 갱신 → 모달 닫기
- "닫기" → 모달 닫기 + selectedRows 전체 초기화

### error 전용
- "다시 시도" 클릭 → `02-선택중` 복귀 (status, reason 유지) → `mutation.mutate(parsed.data)` 재실행
- "취소" → 모달 닫기

## 상태 고유 비즈니스 룰 델타
1. **성공 자동 닫힘**: 300ms 지연 후 `onOpenChange(false)` — UX 가독성(토스트 체크)
2. **부분 실패 보존**: 성공한 회원은 서버에 커밋됨. 롤백 없음.
3. **감사 로그**: 성공 건별 `AUDIT.MEMBER_STATUS_CHANGE` 저장 (실패는 제외)
4. **토스트**: 성공=`toast.success`, 부분=`toast.warning`, 실패=`toast.error`
5. **`onSuccess` 콜백**: 부모 화면에 `{ updated, failed }` 전달 → BulkActionBar 상태 동기화

## 에지 케이스
- **403 E403001**: 토스트만 + 모달 즉시 닫기 (권한 없으면 재시도 무의미)
- **409 E409101 (상태 충돌)**: 실패 목록 개별 표시
- **422 E422101 (탈퇴 회원)**: 실패 목록 개별 표시 ("탈퇴 회원은 변경 불가")
- **NETWORK**: "네트워크 연결을 확인해주세요" + 재시도 버튼
- **응답 구조 이상 (missing data)**: `E500001`로 폴백
- **동시 발생한 refund 등으로 상태 재변경**: 서버 최신 상태 기반 응답 처리

## 바이브코딩 프롬프트 (델타)
```
mutation.onSuccess = (data) => {
  if (data.failed.length === 0) {
    toast.success(`${data.updated}명의 상태를 '${LABELS[status]}'(으)로 변경했습니다.`);
    queryClient.invalidateQueries({ queryKey: ['members', branchId] });
    onSuccess?.({ updated: data.updated, failed: 0 });
    setTimeout(() => onOpenChange(false), 300);
  } else if (data.updated > 0) {
    setSubState('partial');
    setFailedList(data.failed);
    toast.warning(`${data.updated}명 성공 / ${data.failed.length}명 실패`);
    queryClient.invalidateQueries({ queryKey: ['members', branchId] });
  } else {
    setSubState('error');
    setErrorCode(data.failed[0]?.errorCode ?? 'E500001');
    toast.error('상태 변경에 실패했습니다.');
  }
};
mutation.onError = (err) => {
  setSubState('error');
  setErrorCode(err.code ?? (navigator.onLine ? 'E500001' : 'NETWORK'));
  toast.error(ERROR_MAP[errorCode] ?? '오류가 발생했습니다');
};
```
