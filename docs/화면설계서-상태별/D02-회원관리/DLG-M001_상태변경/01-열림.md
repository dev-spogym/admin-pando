## 메타
| 항목 | 값 |
|------|----|
| 화면 ID | DLG-M001 |
| 상태 코드 | `open` |
| 경로 | `/members` (SCR-M001 위) |
| 역할 | primary / owner / manager |
| 우선순위 | P0 |
| 다이어그램 | `docs/다이어그램/D02_회원관리/DLG/DLG-M001_상태변경/M1_생명주기.md` |

## 🧩 바이브코딩 프롬프트
```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 'use client' 컴포넌트 작성.
화면: DLG-M001 상태 일괄 변경 다이얼로그 — 상태: 열림

컴포넌트: Modal (max-w-sm, 중앙 정렬)

구조:
<Modal open={isOpen} onClose={handleClose} title="상태 변경">
  <p className="text-sm text-gray-500 mb-4">
    선택한 {selectedRows.size}명의 상태를 변경합니다.
  </p>
  <RadioGroup value={selectedStatus} onValueChange={setSelectedStatus}>
    <RadioItem value="ACTIVE">정상 이용 중</RadioItem>
    <RadioItem value="HOLDING">홀딩</RadioItem>
    <RadioItem value="SUSPENDED">이용 정지</RadioItem>
  </RadioGroup>
  <div className="flex gap-2 mt-6 justify-end">
    <Button variant="outline" onClick={handleClose}>취소</Button>
    <Button onClick={handleSubmit}>변경 확인</Button>
  </div>
</Modal>

초기값: selectedStatus = "ACTIVE"
트리거: selectedRows.size > 0 조건 충족 후 열림

사용 유틸: useAuthStore, supabase, toast, lucide-react
```

## 📝 디스크립션

### 사용 시점
- SCR-M001 BulkActionBar에서 "상태 변경" 클릭
- selectedRows.size > 0 필수

### 필수 데이터
| 항목 | 상태 |
|------|------|
| selectedRows | Set<string> (회원 ID 목록) |
| selectedStatus | "ACTIVE" (초기값) |

### 인터랙션
1. RadioGroup 선택 → `02-선택중`
2. 취소/ESC/X → 닫힘
3. 변경 확인 → `03-제출중`

### 비즈니스 룰
- primary/owner/manager만 가능
- selectedRows.size=0이면 버튼 자체가 비활성 (다이얼로그 열리지 않음)

### 접근성
- Modal `role="dialog"`, `aria-modal="true"`, `aria-labelledby="dlg-title"`
- ESC 키로 닫기 지원

### 연결 화면
- SCR-M001 (부모 화면)

### 다이어그램 참조
- 엣지: `E_OPEN_01`
- 상태: `DLG_OPEN`, `INIT`
