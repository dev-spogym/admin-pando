## 📝 디스크립션

### 사용 시점
- 사용자가 `/members` 경로에 진입하고 권한 확인(primary/owner/manager/fc/staff)을 통과한 직후
- 필터 탭 변경 시 데이터를 다시 fetch하는 동안
- BulkAction 완료 후 목록 refetch 중 (부분 로딩과 구분)

### 필수 데이터
| 항목 | 상태 |
|------|------|
| GET /api/members | 요청 중 (isLoading=true) |
| 통계 카드 데이터 | 아직 없음 → 스켈레톤 표시 |

### 인터랙션
- 검색 필터: disabled 상태 (입력 차단)
- 정렬/필터 버튼: disabled
- 회원 등록 버튼: disabled (로딩 중에는 중복 클릭 방지)
- 테이블 행 클릭: 불가

### 비즈니스 룰
- 로딩 시간이 3초 이상 지속되면 "시간이 걸리고 있습니다..." 서브 메시지 표시
- 동시에 여러 fetch가 발생하지 않도록 요청 중복 방지 처리
- readonly 역할은 이 상태에 진입하지 않음 (권한 없음 화면으로 분기)

### 에지 케이스
- 네트워크가 매우 느린 경우: skeleton UI를 유지하며 timeout 메시지 표시
- 로딩 중 뒤로 가기: 요청 취소 후 이전 화면 복귀

### 접근성
- 스켈레톤 컨테이너에 `aria-busy="true"` + `aria-label="회원 목록을 불러오는 중입니다"`
- Loader2 아이콘 `aria-hidden="true"`
