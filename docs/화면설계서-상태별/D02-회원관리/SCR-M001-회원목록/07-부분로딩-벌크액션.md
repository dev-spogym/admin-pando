## 📝 디스크립션

### 사용 시점
- DLG-M001 (상태 변경)에서 "변경 확인" 클릭 후 API 응답 대기 중
- PUT /api/members/bulk-status 요청 진행 중

### 필수 데이터
| 항목 | 상태 |
|------|------|
| selectedRows | Set<string> (선택된 회원 ID들) |
| isBulkLoading | true |
| 기존 members | 그대로 표시 |

### 인터랙션
1. 처리 완료 → refetch → selectedRows 초기화 → `02-정상목록`
2. 처리 실패 → toast.error → `02-정상목록` (선택 유지 또는 초기화)

### 비즈니스 룰
- 처리 중 페이지 이탈 시 경고 다이얼로그 표시
- BulkAction은 primary/owner/manager만 가능

### 에지 케이스
- 대용량 선택(100명+) 처리 지연 시 진행률 표시 고려
- 네트워크 단절 시 재시도 안내

### 접근성
- `aria-busy="true"` on BulkActionBar
- 처리 완료 시 `aria-live="polite"` 결과 고지

### 연결 화면
- 완료: `02-정상목록`

### 다이어그램 참조
- 엣지: `E_NORMAL_BULK_01` (BulkAction 중)
- 상태: `STATE_BULK_LOADING`
- 완료: `E_BULK_DONE_01`
