## 📝 디스크립션

### 사용 시점
- 신규 센터 오픈 직후 회원이 한 명도 없는 상태
- 검색/필터 없이 전체 조회 시 members 테이블이 비어있는 경우

### 필수 데이터
| 항목 | 상태 |
|------|------|
| members | 빈 배열 [] (검색 필터 없음) |
| 통계 카드 | 전부 0 |

### 인터랙션
1. "회원 등록하기" CTA → `/members/new`
2. PageHeader "회원 등록" 버튼 → 동일
3. 탭 변경 → 모든 탭 동일하게 0건 (상태별 탭도 0)

### 비즈니스 룰
- 검색 빈 상태(`empty-search`)와 구별: 이 상태는 검색 조건 없이 0건
- CTA 버튼 표시 조건: primary/owner/manager/staff만 노출 (fc는 직접 등록 불가)
- readonly 역할은 이 화면 자체에 진입 불가

### 에지 케이스
- 이관 등으로 회원이 모두 다른 지점으로 이동한 경우도 이 상태 발생 가능
- fc 역할이 접근하면 CTA 버튼 없이 안내 메시지만 표시

### 접근성
- EmptyState `role="status"`
- CTA 버튼 `aria-label="첫 번째 회원 등록하기"`

### 연결 화면
- 회원 등록: SCR-M002 (`/members/new`)
