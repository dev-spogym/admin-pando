try {

## 📝 디스크립션

### 사용 시점
- GET /api/members/{id} 4xx/5xx 응답
- 존재하지 않는 회원 ID로 접근 (404)
- 네트워크 오류

### 필수 데이터
| 항목 | 상태 |
|------|------|
| error | 오류 객체 |

### 인터랙션
- 자동: toast.error → /members 리다이렉트

### 비즈니스 룰
- 404: "존재하지 않는 회원입니다."
- 403: "접근 권한이 없습니다."
- 500: "서버 오류가 발생했습니다."

### 에지 케이스
- id가 숫자가 아닌 경우: 즉시 /members 이동

### 접근성
- toast.error 스크린리더 고지

### 연결 화면
- SCR-M001 (회원 목록)

### 다이어그램 참조
- 엣지: `E_LOAD_FAIL_01`
- 상태: `STATE_LOAD_ERR`
