## 📝 디스크립션

### 사용 시점
- 회원 구분 Select에서 "기명법인" 또는 "무기명법인" 선택 시

### 필수 데이터
| 항목 | 상태 |
|------|------|
| memberType | 기명법인 또는 무기명법인 |
| corporateName | 필수 입력 대기 |

### 인터랙션
1. 기명법인/무기명법인 선택 → 법인 회사명 필드 슬라이드 표시
2. 일반 선택 → 법인 회사명 필드 숨김 + 값 초기화
3. 나머지: `02-입력중` 동일

### 비즈니스 룰
- 기명법인: 담당자 이름 + 법인명 모두 필수
- 무기명법인: 법인명만 필수 (이름 선택)
- 법인 회원은 이용권 할당 방식이 다를 수 있음 (step2에서 처리)

### 에지 케이스
- 법인명 입력 후 일반으로 변경: corporateName 값 클리어
- 기명↔무기명 전환: 이름 필드 required 속성 동적 변경

### 접근성
- 조건부 필드 `aria-hidden` 토글
