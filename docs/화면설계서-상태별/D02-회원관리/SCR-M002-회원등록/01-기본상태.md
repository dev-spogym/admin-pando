## 📝 디스크립션

### 사용 시점
- `/members/new` 마운트 직후 초기 상태
- 초기화(DLG-M008) 실행 후 Step1으로 복귀한 상태

### 필수 데이터
| 항목 | 상태 |
|------|------|
| 폼 초기값 | 모두 빈값 |
| memberType | 일반 (default) |
| isDirty | false |
| phoneChecked | false |

### 인터랙션
1. 필드 입력 → `02-입력중`으로 전환
2. 취소 버튼 → isDirty=false → `/members` 즉시 이동
3. 법인 선택 → `05-법인상태`

### 비즈니스 룰
- primary/owner/manager/staff만 접근 가능
- fc 역할은 회원 등록 불가 (403 또는 권한 없음 화면)
- 담당FC 기본값: 로그인한 사용자가 fc이면 자동 선택

### 에지 케이스
- 브라우저 뒤로 가기로 재진입 시 폼 리셋 여부 정책 필요
