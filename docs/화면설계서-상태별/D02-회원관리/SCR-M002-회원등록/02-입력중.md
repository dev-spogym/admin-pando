| 항목 | 상태 |

| isDirty | true |
| phoneChecked | false 또는 true |

1. 취소 → DLG-M007 (isDirty=true이므로 확인 필요)
2. 초기화 → DLG-M008

4. 다음 → Step1 통과 시 `06-Step2`

### 비즈니스 룰
