> 🧭 **델타**. 공통 스펙은 [`00-기본화면.md`](./00-기본화면.md) 참조.

|------|----|
| 상태 코드 | `new-dirty` |
| 다이어그램 | `STATE_DIRTY` @ `F6_상태별.md` |

## 진입 조건
- 사용자가 1개 이상 필드 수정하여 `isDirty === true`
- (아직 onBlur 전이면 에러 숨김)

## 비주얼 델타
| 요소 | 변경 |
|---|---|
| [초기화] | `enabled` (isDirty or phoneChecked) |
| [취소] | `enabled` — 클릭 시 **DLG-M007 취소 확인** 모달 |
| 입력 값 | 사용자 입력 반영, 실시간 렌더 |
| 전화번호 필드 | 입력 시 `formatPhone()` 자동 포맷 (예: `0101234…` → `010-1234-…`) |
| 중복확인 버튼 | phone 길이 13 도달 시 `enabled`, `phoneChecked=false`일 때만 |
| Draft 자동 저장 | 5분 주기 localStorage `member:new:draft` 저장 (🆕) |
| `beforeunload` | 활성 — 탭 닫기/새로고침 시 브라우저 확인 |

## 인터랙션
| 액션 | 동작 |
|---|---|
| phone 변경 | `phoneChecked=false, phoneDuplicate=false` 리셋 |
| memberType 변경 | 법인이면 `05-법인상태`, 아니면 companyName 초기화 |
| 프로필 이미지 선택 | `04-이미지업로드중` |
| [초기화] 클릭 | DLG-M008 확인 → reset + `01-기본상태` |
| [취소] 클릭 | DLG-M007 확인 → `/members` |
| [다음] 클릭 | trigger 검증 → 통과면 `06-Step2`, 실패면 `03-유효성에러` |

## 룰 델타
1. 각 필드 `mode: 'onBlur'` 검증, 실시간 `onChange`는 formatPhone만 적용.
2. isDirty 판정: react-hook-form 내장 `formState.isDirty`.
3. 전화번호 변경 시 `phoneChecked/phoneDuplicate` 초기화 (Zod가 아닌 RHF watcher로 구현).
4. Draft 자동 저장: `useEffect(() => { const t = setInterval(() => localStorage.setItem('member:new:draft', JSON.stringify(values)), 300_000); return () => clearInterval(t); }, [values])`.
5. `beforeunload` 리스너:
   ```ts
   useEffect(() => {
     const handler = (e: BeforeUnloadEvent) => { if (isDirty) { e.preventDefault(); e.returnValue = ''; } };
     window.addEventListener('beforeunload', handler);
     return () => window.removeEventListener('beforeunload', handler);
   }, [isDirty]);
   ```

## 에지
- 빠른 탭 이동 (Tab × 여러번) → 검증 트리거 스팸 방지 (onBlur만).
- 한국어 IME 조합 중 Enter → 제출 방지 (기본 form submit 차단).
- 브라우저 자동완성 → onChange 트리거 → isDirty=true → draft 저장 정상 동작.

## 🧩 바이브코딩 (델타)
```
// 전화번호 포맷 + phoneChecked 리셋
const phone = watch('phone');
useEffect(() => { setPhoneChecked(false); setPhoneDuplicate(false); }, [phone]);

// draft 자동 저장
const values = watch();
useDebounceEffect(() => {
  if (isDirty) localStorage.setItem('member:new:draft', JSON.stringify(values));
}, [values, isDirty], 2000);

// beforeunload
usePrompt(isDirty && !isSubmitSuccessful, '작성 중인 내용이 있습니다. 정말 이동하시겠습니까?');
```
