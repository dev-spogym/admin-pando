## 📝 디스크립션

### 사용 시점 (Trigger)
- 로딩 완료 후 `leads.length > 0`

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 통계 카드 | `leads` | branch_id = ? (집계) |
| 리드 목록 | `leads` | branch_id = ?, 필터 조건, 페이지네이션 |

### 인터랙션 (User Actions)
1. "리드 등록" 버튼 → DLG-070-001 열기 (등록 모드)
2. 수정 아이콘 → DLG-070-001 열기 (수정 모드, 기존값 주입)
3. 삭제 아이콘 → DLG-070-002 열기
4. 필터 변경 → 목록 재조회
5. 페이지 이동 → 해당 페이지 로드

### 비즈니스 룰
- 전환율 = 등록완료 건수 / 전체 건수 × 100 (소수점 1자리)
- count > 0 인 상태만 통계 카드 강조 표시
- 삭제는 owner만 가능 (fc는 수정만)
- 후속일정이 오늘 이전이면 amber 하이라이트

### 에지 케이스
- 리드 100건 이상: 페이지네이션 표시
- 필터 조합 결과 0건: 빈 상태 메시지 (테이블 내 inline)

### 접근성 (A11y)
- StatusBadge `aria-label="{상태명}"`
- DataTable `role="grid"`, 행 `role="row"`
