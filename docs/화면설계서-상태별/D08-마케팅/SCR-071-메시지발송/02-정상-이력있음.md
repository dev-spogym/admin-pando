## 📝 디스크립션

### 사용 시점 (Trigger)
- 로딩 완료 + `message_logs.length > 0`

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 발송 이력 | `message_logs` | branch_id = ?, limit 20 |
| 회원 목록 | `members` | DLG-071-001 수신자 검색용 |

### 인터랙션 (User Actions)
1. "검색" 버튼 → DLG-071-001 수신자 검색 모달
2. 변수 버튼 클릭 → textarea에 변수 삽입
3. "메시지 발송" → 내용 검증 후 DLG-071-002 미리보기 모달
4. 예약 토글 ON → DateTimePicker 표시

### 비즈니스 룰
- SMS 최대 80자, 카카오 1000자, 앱푸시 200자
- 수신자 0명 시 발송 버튼 비활성
- 예약 시간은 현재 시각 이후만 선택 가능

### 에지 케이스
- 포인트 잔액 부족: 발송 버튼 클릭 시 toast.error
- 메시지 미입력: 발송 버튼 비활성
