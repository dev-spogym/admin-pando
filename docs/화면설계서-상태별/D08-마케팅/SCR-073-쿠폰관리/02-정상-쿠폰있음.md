## 📝 디스크립션

### 사용 시점 (Trigger)
- 로딩 완료 + `coupons.length > 0`

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 쿠폰 목록 | `coupons` | branchId = ?, isActive = true |

### 인터랙션 (User Actions)
1. "쿠폰 생성" → DLG-073-001 (생성 모드)
2. Edit 아이콘 → DLG-073-001 (수정 모드)
3. Send 아이콘 → DLG-073-002 (발급 모달)
4. Trash2 아이콘 → DLG-073-003 (삭제 확인)

### 비즈니스 룰
- `totalUsed >= totalIssued` → 소진 배지 (TC-073-F6-03)
- `validUntil < today` → 만료 배지 (TC-073-F6-02)
- `validUntil - today <= 7` → amber D-N 경고 (TC-073-F6-04)
- Trash2는 `totalIssued === 0`일 때만 활성

### 에지 케이스
- 쿠폰 코드 중복: DLG-073-001 저장 시 409 처리
- 만료/소진 쿠폰 Send 버튼 비활성
