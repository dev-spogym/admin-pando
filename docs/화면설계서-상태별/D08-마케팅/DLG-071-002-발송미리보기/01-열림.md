## 📝 디스크립션

### 사용 시점 (Trigger)
- SCR-071 "메시지 발송" 버튼 클릭 후 내용 검증 통과

### 필수 데이터
| 블록 | 설명 |
|------|------|
| 메시지 내용 | SCR-071 폼에서 전달 |
| 수신자 목록 | DLG-071-001에서 선택한 회원 |
| 예약 일시 | SCR-071 DateTimePicker 값 |

### 인터랙션 (User Actions)
1. "발송하기" → 즉시/예약 발송 API 호출
2. "취소"/X → 모달 닫힘, 폼 유지

### 비즈니스 룰
- 포인트 잔액 부족: 402 응답 → toast.error("포인트가 부족합니다.") (TC-071-M3)
- 타임아웃: toast.warning

### 에지 케이스
- 발송 중 로딩: 버튼 spinner + disabled
- 402 잔액 부족: 모달 유지, 충전 안내
