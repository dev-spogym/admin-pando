## 📝 디스크립션

### 사용 시점 (Trigger)
- `E_M1_OPEN_01`: SCR-052 행 "삭제" 버튼 클릭

### 검증 로직 (M2)
- 입력값 !== "삭제" → 삭제 버튼 disabled (`E_M1_MISMATCH_01`)
- 입력값 === "삭제" → 삭제 버튼 활성 (`E_M1_MATCH_01`)

### 비즈니스 룰
- owner/manager만 접근 가능
- 연결 회원 있을 경우 경고 표시 (삭제 후 연결 해제)
