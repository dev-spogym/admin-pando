## 📝 디스크립션

### 사용 시점 (Trigger)
- `E_M2_NO_TARGET_01`: 락커 미선택
- `E_M2_SAME_01`: 현재 락커 선택
- `E_M2_OCCUPIED_01`: 이미 사용중인 락커 선택

### 인터랙션 (User Actions)
1. 다른 락커 선택 → 에러 메시지 사라짐
2. 유효한 락커 선택 후 "확인" → `03-성공` 또는 `04-실패`

### 연결 화면
- 다음: 수정 후 `03-성공` / `04-실패`
