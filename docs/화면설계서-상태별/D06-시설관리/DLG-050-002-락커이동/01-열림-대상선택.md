## 📝 디스크립션

### 사용 시점 (Trigger)
- `E_M1_OPEN_01`: SCR-050 in_use 셀 → "이동" 버튼 클릭

### 필수 데이터
- 현재 락커 정보 (number, memberName)
- 사용 가능한 락커 목록 (available 상태만)

### 인터랙션 (User Actions)
1. 대상 락커 Select 선택
2. "확인" 클릭 → 검증 (`E_M2_*`) → 이동 API
3. "취소" / ESC → 모달 닫힘

### 검증 규칙 (M2)
| 에러 코드 | 조건 | 메시지 |
|-----------|------|--------|
| E_M2_NO_TARGET_01 | 미선택 | "이동할 락커를 선택하세요." |
| E_M2_SAME_01 | 현재 락커와 동일 | "다른 락커를 선택하세요." |
| E_M2_OCCUPIED_01 | 대상 사용중 | "사용중인 락커입니다." |
