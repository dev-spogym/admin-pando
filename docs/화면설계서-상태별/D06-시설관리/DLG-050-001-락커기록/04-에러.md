- 메타 저장(`saveMeta`) 실패는 이 상태로 전이하지 않고, `02` 유지 + 토스트 에러

## 상태 고유 비즈니스 룰 델타

1. **에러 분류 & 분기**:
   | 코드 | 표시 메시지 | 액션 |
   |---|---|---|
   | `E403001` (403) | "접근 권한이 없습니다." | 토스트 + 자동 닫기 (본 상태 미노출) |
   | `E404600` (404) | "락커를 찾을 수 없습니다." | 토스트 + 부모 refetch + 자동 닫기 |
   | `E500001` (500) | "이력을 불러오지 못했습니다." | 본 상태 유지, 재시도 가능 |
   | `NETWORK` | "네트워크 오류가 발생했습니다." | 본 상태 유지, 재시도 가능 |
   | `TIMEOUT` (15s) | "응답이 지연되고 있습니다." | 본 상태 유지, 재시도 가능 |
   | `E401002` (401) | (표시 안 함) | DLG-000 우선, 본 모달 정리 |
2. **재시도 쿨다운**: 연속 3회 실패 시 재시도 버튼 10초간 `disabled` + 카운트다운 문구.
3. **에러 공지**: `role="alert" aria-live="assertive"` + 에러 메시지 즉시 스크린리더 공지.
4. **편집 섹션**:
   - `E404600` → 숨김 (락커 없음)
   - 그 외(5xx/NETWORK) → `readOnly` + "연결 복구 후 편집 가능" 헬퍼 텍스트
5. **에러 로깅**: 클라이언트 `console.error('[LockerHistory]', error)` + (옵션) Sentry 리포트. 개인정보 금지.

## 에지 케이스

- 재시도 직후 성공: `02` 또는 `03`으로 정상 전이
- 재시도 직후 다른 에러: 메시지 갱신만, 본 상태 유지
- 모달 닫는 도중 응답 도착: `onClose` 후 state 업데이트 금지 (`ignore` flag)
- 부모 락커가 이미 삭제되어 `E404600` → 부모 `invalidateQueries(['lockers'])` 트리거 후 모달 닫기
- 403 (타 지점 접근 시도): 보안 이벤트 → `AUDIT.UNAUTHORIZED_ACCESS` 서버 기록

## 바이브코딩 프롬프트 (델타)

```
{query.isError && (
  <div role="alert" aria-live="assertive"
       className="rounded-md bg-rose-50 border border-rose-200 p-4 space-y-2">
    <div className="flex items-start gap-2">
      <AlertCircle className="size-4 mt-0.5 text-rose-600 shrink-0" aria-hidden />
      <div className="flex-1 text-sm text-rose-700">
        {getErrorMessage(query.error)}
      </div>
    </div>
    <div className="flex justify-end">
      <button
        type="button"
        onClick={() => query.refetch()}
        disabled={cooldownMs > 0}
        className="h-9 px-3 rounded-md border border-rose-300 text-rose-700 hover:bg-rose-100 text-sm
                   disabled:opacity-50 disabled:cursor-not-allowed">
        {cooldownMs > 0 ? `${Math.ceil(cooldownMs/1000)}초 후 재시도` : '다시 시도'}
      </button>
    </div>
  </div>
)}

{에러 메시지 헬퍼}
function getErrorMessage(err: any) {
  const code = err?.code ?? err?.errorCode;
  switch (code) {
    case 'E404600': return '락커를 찾을 수 없습니다. 목록이 갱신됩니다.';
    case 'E500001': return '이력을 불러오지 못했습니다. 잠시 후 다시 시도해주세요.';
    case 'NETWORK': return '네트워크 오류가 발생했습니다.';
    case 'TIMEOUT': return '응답이 지연되고 있습니다.';
    default:        return '이력을 불러오지 못했습니다.';
  }
}

{404/403 즉시 닫기 훅}
useEffect(() => {
  if (!query.isError) return;
  const code = (query.error as any)?.code;
  if (code === 'E404600') {
    toast.error('락커를 찾을 수 없습니다.');
    qc.invalidateQueries({queryKey:['lockers', branchId]});
    onClose();
  } else if (code === 'E403001') {
    toast.error('접근 권한이 없습니다.');
    onClose();
  }
}, [query.isError]);
```
