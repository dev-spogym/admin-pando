## 📝 디스크립션

### 사용 시점 (Trigger)
- `E_M1_LOAD_OK_01`: 이력 API 200 + 데이터 존재

### 필수 데이터
| 컬럼 | 설명 |
|------|------|
| created_at | 이력 발생 일시 |
| action | 배정/회수/이동/고장/복구 |
| member_name | 관련 회원명 |
| staff_name | 처리 담당자 |

### 인터랙션 (User Actions)
1. 기간 필터 변경 후 "조회" → 해당 기간 이력 재조회
2. 날짜 범위 역순 (시작 > 종료) → 인라인 에러 "날짜 범위를 확인하세요." (`E_M2_DATE_ERR_01`)
3. "닫기" 버튼 또는 ESC → 모달 닫힘 (`E_M3_CLOSE_01`)

### 비즈니스 룰
- 기본 조회: 최근 30건
- 기간 필터: 최대 3개월 범위
- 이력 없는 기간 조회 시 `03-이력없음` 상태로 전환

### 에지 케이스
- 이력 100건 초과: 페이지네이션 (20건씩)
- 액션 유형 미분류: "기타" 배지 (gray)

### 접근성 (A11y)
- 테이블 `role="table"` + `caption="락커 이력"`
- 닫기 버튼 `aria-label="모달 닫기"`

### 연결 화면
- 이전: `01-열림-이력로드중`
- 다음: 닫기 → SCR-050 유지 / 필터 변경 → 재조회
