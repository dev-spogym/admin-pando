## 🧩 바이브코딩 프롬프트

```
모달: DLG-057-002 입고 처리 (신규 기능)

<Modal isOpen={isOpen} onClose={onClose} title={`입고 처리 — ${item.name}`} size="sm">
  <div className="space-y-4">
    {/* 현재 재고 표시 */}
    <div className="p-3 bg-gray-50 rounded-lg flex justify-between items-center">
      <span className="text-sm text-gray-600">현재 재고</span>
      <span className="text-lg font-bold text-gray-800">{item.currentStock} {item.unit}</span>
    </div>

    {/* 입고 수량 */}
    <div>
      <label className="text-sm font-medium text-gray-700 mb-1 block">입고 수량 *</label>
      <div className="flex items-center gap-2">
        <button onClick={() => setQty(Math.max(1, qty - 1))}
          className="w-8 h-8 rounded border border-gray-300 text-gray-600 hover:bg-gray-50">-</button>
        <input
          type="number"
          className={`flex-1 border rounded-md p-2 text-sm text-center ${errors.qty ? 'border-red-300' : 'border-gray-300'}`}
          min={1}
          value={qty}
          onChange={e => setQty(Number(e.target.value))}
        />
        <button onClick={() => setQty(qty + 1)}
          className="w-8 h-8 rounded border border-gray-300 text-gray-600 hover:bg-gray-50">+</button>
      </div>
      {errors.qty && <p className="text-xs text-red-500 mt-1">{errors.qty}</p>}
    </div>

    {/* 입고 후 재고 미리보기 */}
    <div className="p-3 bg-green-50 rounded-lg flex justify-between items-center">
      <span className="text-sm text-green-700">입고 후 재고</span>
      <span className="text-lg font-bold text-green-700">{item.currentStock + qty} {item.unit}</span>
    </div>

    {/* 메모 */}
    <div>
      <label className="text-sm font-medium text-gray-700 mb-1 block">메모</label>
      <input type="text" className="w-full border border-gray-300 rounded-md p-2 text-sm"
        placeholder="입고 관련 메모 (선택)" value={memo} onChange={e => setMemo(e.target.value)} />
    </div>
  </div>

  <div className="flex justify-end gap-2 mt-6">
    <Button variant="outline" onClick={onClose}>취소</Button>
    <Button variant="primary" onClick={handleSubmit} loading={isSubmitting}>입고 확인</Button>
  </div>
</Modal>
```
