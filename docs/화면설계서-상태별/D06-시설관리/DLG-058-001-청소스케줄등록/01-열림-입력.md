### 비즈니스 룰
- 예정일시 min 제한: 현재 시각 이후만 선택 가능 (datetime-local min 속성)
- 담당자 목록은 해당 센터 직원 목록 동적 로드
- 반복 스케줄 설정은 이 모달 범위 밖 (단일 건 등록)

### 에지케이스
- 직원 목록 로드 실패 시: Select placeholder "담당자 로드 실패" + 재시도 안내
- 과거 일시 직접 입력 시: 포커스 아웃 후 즉시 인라인 에러

### A11y
- datetime-local 입력 필드에 aria-label="예정 일시" 부여
- 에러 메시지 role="alert"
