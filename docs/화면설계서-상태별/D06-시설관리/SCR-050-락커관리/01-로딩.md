## 역할별 차이

역할 분기 없음. 모든 역할에 동일한 스켈레톤. 단, 권한 검사 실패 시 즉시 07 상태로 전환되므로 로딩 단계에서는 권한별 차이 렌더 안 됨.

## 상태 고유 인터랙션

- 모든 버튼/인풋 disabled
- ESC 키: 무시
- 새로고침 버튼 중복 클릭: 방지 (`isFetching` 동안 disabled)
- 5초 이상 로딩 지속 시: `role="status" aria-live="polite"`로 "데이터를 불러오는 중..." 스크린리더 안내

## 비즈니스 룰 델타

- 타임아웃: 10초 초과 시 자동으로 `05-에러` 전환 + `NETWORK_TIMEOUT` 토스트
- `runDailySync()` 1회 호출 (세션당) — 만료/임박 자동 전환

## 에지 케이스

| 상황 | 처리 |
|---|---|
| 로딩 중 탭 변경 | 현재 fetch 유지, 새 zone queryKey로 별도 fetch (React Query 자동 dedupe) |
| 로딩 중 오프라인 감지 | 즉시 06 상태 전환 |
| 로딩 중 세션 만료(401) | `/login?redirect=/locker` 리다이렉트 |
| 로딩 중 403 수신 | 07-권한없음 전환 |
