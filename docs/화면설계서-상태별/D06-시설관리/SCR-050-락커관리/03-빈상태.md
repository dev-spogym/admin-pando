## 📝 디스크립션

### 사용 시점 (Trigger)
- API 200 OK + `lockers.length === 0`
- 필터 검색 결과 0건 (이 경우 필터 초기화 CTA 추가)

### 필수 데이터

### 인터랙션 (User Actions)
1. "락커 추가" 버튼 → 락커 추가 모달 (owner/manager만)
2. 필터 결과 없음 → "필터 초기화" 버튼 추가

### 비즈니스 룰
- staff 역할은 "락커 추가" 버튼 비노출 (hasPermission 확인)
- 최초 지점 설정 시 이 상태가 초기 상태

### 에지 케이스
- 검색 필터 적용 후 0건: "검색 결과가 없습니다" + 필터 초기화 버튼
- 락커 추가 후 즉시 `02-정상` 상태로 전환

### 접근성 (A11y)
- 빈 상태 컨테이너 `role="status"` + `aria-label="등록된 락커 없음"`
