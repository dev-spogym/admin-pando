## 📝 디스크립션

### 사용 시점 (Trigger)
- API 응답 200 OK + lockers.length > 0

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 락커 목록 | `lockers` | `branch_id = ?` |
| 회원 정보 | `members` | `id IN (locker.member_id)` |
| 통계 | `lockers` | 상태별 COUNT |

### 인터랙션 (User Actions)
1. 셀 호버 → 액션 버튼 오버레이 표시
2. "배정" 클릭 → DLG-050-004 개별배정 모달
3. "기록" 클릭 → DLG-050-001 락커기록 모달
4. "이동" 클릭 → DLG-050-002 락커이동 모달
5. "회수" 클릭 → DLG-050-003 락커회수확인 모달
6. "고장" 토글 → DLG-050-005 고장토글확인 모달
7. "일괄배정" 버튼 → DLG-050-006
8. "일괄해제" 버튼 → DLG-050-007

### 비즈니스 룰
- expires_at - today <= 7 → expiring 상태 (amber 색상)
- expires_at - today <= 0 → D-0 표시, text-red-600
- broken 셀은 배정 불가 (배정 버튼 비노출)
- FC/코치 역할은 이 화면 접근 불가 (403)

### 에지 케이스
- 전체 만료임박: 통계 카드 amber 강조 + "일괄해제" 버튼 활성화
- 락커 100개 초과: 가상화 스크롤 또는 페이지네이션 적용
- 회원명 길면 truncate 처리

### 접근성 (A11y)
- 각 셀 `role="button"` + `aria-label="락커 {번호} - {상태}"`
- D-Day 배지 `aria-label="만료까지 {N}일"`
- 고장 셀 `aria-label="고장 락커, 사용 불가"`
