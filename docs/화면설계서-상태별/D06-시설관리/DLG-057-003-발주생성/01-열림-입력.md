```
모달: DLG-057-003 발주 생성 (신규 기능)

<Modal isOpen={isOpen} onClose={onClose} title="발주 생성" size="md">
  <div className="space-y-4">
    {/* 부족 품목 목록 — 안전재고 미만 품목 자동 로드 */}
    <div>
      <label className="text-sm font-medium text-gray-700 mb-1 block">발주 품목</label>
      {shortItems.length === 0 ? (
        <p className="text-sm text-gray-400 py-3 text-center">발주 대상 품목이 없습니다.</p>
      ) : (
        <div className="space-y-2">
          {shortItems.map(item => (
            <div key={item.id} className="flex items-center gap-3 p-3 border border-gray-200 rounded-lg">
              <input
                type="checkbox"
                checked={selected.includes(item.id)}
                onChange={() => toggleSelect(item.id)}
                className="w-4 h-4"
              />
              <div className="flex-1">
                <p className="text-sm font-medium text-gray-800">{item.name}</p>
                <p className="text-xs text-gray-500">현재고 {item.currentStock}{item.unit} / 안전재고 {item.safetyStock}{item.unit}</p>
              </div>
              <div className="w-24">
                <input
                  type="number"
                  className={`w-full border rounded-md p-1.5 text-sm text-center ${errors[item.id] ? 'border-red-300' : 'border-gray-300'}`}
                  min={1}
                  value={quantities[item.id] ?? ''}
                  onChange={e => setQuantity(item.id, Number(e.target.value))}
                  placeholder="수량"
                />
                {errors[item.id] && <p className="text-xs text-red-500 mt-0.5">{errors[item.id]}</p>}
              </div>
              <span className="text-xs text-gray-500">{item.unit}</span>
            </div>
          ))}
        </div>
      )}
      {errors.items && <p className="text-xs text-red-500 mt-1">{errors.items}</p>}
    </div>

    {/* 메모 */}
    <div>
      <label className="text-sm font-medium text-gray-700 mb-1 block">메모</label>
      <input
        type="text"
        className="w-full border border-gray-300 rounded-md p-2 text-sm"
        placeholder="발주 관련 메모 (선택)"
        value={memo}
        onChange={e => setMemo(e.target.value)}
      />
    </div>
  </div>

  <div className="flex justify-end gap-2 mt-6">
    <Button variant="outline" onClick={onClose}>취소</Button>
    <Button variant="primary" onClick={handleSubmit} loading={isSubmitting}
      disabled={selected.length === 0}>발주 확인</Button>
  </div>
</Modal>

검증 규칙:
- E_M2_EMPTY_01: 선택된 품목 없음 → "발주할 품목을 선택하세요."
- E_M2_QTY_ZERO_01: 선택 품목 중 수량 0 → "발주 수량을 입력하세요."
```

## 📝 디스크립션

### 검증 규칙 (M2)
| 에러 코드 | 조건 | 메시지 |
|-----------|------|--------|
| E_M2_EMPTY_01 | 선택 품목 없음 | "발주할 품목을 선택하세요." |
| E_M2_QTY_ZERO_01 | 선택 품목 수량 0 이하 | "발주 수량을 입력하세요." |

### 비즈니스 룰
- 모달 오픈 시 안전재고 미만(SHORT/OUT) 품목을 자동으로 목록에 로드
- 체크박스로 개별 품목 선택/해제 가능
- 선택된 품목이 없으면 "발주 확인" 버튼 비활성화
- 발주 수량은 기본값 없음 — 관리자가 직접 입력

### 에지케이스
- 안전재고 미만 품목이 없는 경우: "발주 대상 품목이 없습니다." 메시지 + 버튼 비활성
- 수량 입력 없이 체크만 한 경우: 수량 검증 에러 인라인 표시
