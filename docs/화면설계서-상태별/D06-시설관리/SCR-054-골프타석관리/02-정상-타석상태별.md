## 📝 디스크립션

### 사용 시점 (Trigger)
- `E_F6_OK_01`: API 200 + golf_bays.length > 0

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 타석 | `golf_bays` | `branch_id = ?` |
| 통계 | `golf_bays` | 상태별 COUNT |

### 인터랙션 (User Actions)
1. AVAILABLE "시작" 버튼 → 사용자 입력 후 IN_USE 전환
2. IN_USE "종료" 버튼 → AVAILABLE 전환 + 이용 기록 저장
3. IN_USE "이동" 버튼 → 다른 타석으로 이동 (빈 타석 선택)
4. IN_USE "📺TV" 버튼 → TV 제어 패널 오픈
5. RESERVED "예약 취소" → 확인 모달 후 AVAILABLE 전환
6. MAINTENANCE "점검 해제" → AVAILABLE 전환

### 비즈니스 룰
- 타이머: started_at 기준으로 서버 시간 차이 계산 (클라이언트 드리프트 방지)
- RESERVED: 예약 시간 10분 전 알림 toast
- IN_USE 2시간 초과: 카드 테두리 amber로 변경 + 경고 배지

### 에지 케이스
- 전체 IN_USE: 상단 "모든 타석 사용중" 배너
- 타이머 동기화 오류: "시간 정보 없음" 표시
