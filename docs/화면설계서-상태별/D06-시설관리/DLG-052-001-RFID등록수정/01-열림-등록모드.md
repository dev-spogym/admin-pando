## 📝 디스크립션

### 사용 시점 (Trigger)
- `E_M1_OPEN_01`: SCR-052 "카드 등록" 버튼 클릭

### 필수 데이터
- rfid_cards 테이블 (등록용)
- members/staff 목록 (검색용)

### 인터랙션 (User Actions)
1. RFID 리더기 스캔 → 카드번호 자동 입력 (`E_M1_SCAN_01`)
2. 직접 입력 → `E_M1_MANUAL_01`
3. 사용자 유형 선택 → 회원/직원 검색 필드 표시
4. "저장" → M2 검증 → API 호출

### 검증 규칙 (M2)
| 에러 코드 | 조건 | 메시지 |
|-----------|------|--------|
| E_M2_CARD_EMPTY_01 | 카드번호 미입력 | "카드번호를 입력하세요." |
| E_M2_CARD_DUP_01 | 중복 카드번호 | "이미 등록된 카드번호입니다." |
| E_M2_TYPE_EMPTY_01 | 유형 미선택 | "사용자 유형을 선택하세요." |
