## 🧩 바이브코딩 프롬프트

```
모달: DLG-081-005 역할 복사 — 폼 입력

<Modal open={showCopyRoleModal}>
  <ModalContent className="max-w-md">
    <ModalHeader><ModalTitle>역할 복사</ModalTitle></ModalHeader>
    <ModalBody className="space-y-4">
      <p className="text-sm text-gray-600">
        <strong>{sourceRole.name}</strong> 역할의 권한을 복사하여 새 역할을 생성합니다.
      </p>
      <div>
        <label className="text-sm font-medium">새 역할 이름 <span className="text-red-500">*</span></label>
        <Input value={form.name} onChange={...} placeholder="예: 영업팀장 (복사)" className="mt-1" />
      </div>
      <div>
        <label className="text-sm font-medium">새 역할 코드 <span className="text-red-500">*</span></label>
        <Input value={form.code} onChange={...} placeholder="예: sales_manager_copy" className="mt-1 font-mono" />
        {codeError && <p className="text-xs text-red-500 mt-1">{codeError}</p>}
      </div>
    </ModalBody>
    <ModalFooter className="flex justify-end gap-2">
      <Button variant="outline" onClick={() => setShowCopyRoleModal(false)}>취소</Button>
      <Button disabled={!form.name || !form.code} onClick={handleCopy}>복사 생성</Button>
    </ModalFooter>
  </ModalContent>
</Modal>

handleCopy: async () => {
  // POST /roles with sourceId + 새 이름/코드
  const { data, error } = await supabase.from('custom_roles')
    .insert({ name: form.name, code: form.code, branch_id: branchId })
    .select().single()
  if (error?.code === '23505') { setCodeError('이미 사용 중인 코드'); return }
  if (error) { toast.error('복사 실패'); return }
  // 원본 권한 복사
  await copyPermissions(sourceRole.id, data.id)
  onRoleCreated(data)
  setShowCopyRoleModal(false)
  toast.success('역할이 복사되었습니다.')
}
```
