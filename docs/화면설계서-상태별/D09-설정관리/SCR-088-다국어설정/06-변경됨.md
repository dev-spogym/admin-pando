## 🧩 바이브코딩 프롬프트

```
화면: SCR-088 다국어설정 — 상태: isDirty=true, 미저장 배지 (E_F6_088_08)

변경됨 상태:
- PageHeader 우측: amber 배지 "저장되지 않은 변경사항"
- 저장하기 버튼: variant="primary" 활성

저장 핸들러:
handleSave: async () => {
  const changes = Object.entries(dirtyKeys).map(([key, value]) => ({
    branch_id: branchId, key, value, locale: activeLocale
  }))
  const { error } = await supabase.from('i18n_custom').upsert(changes)
  if (error) { toast.error('번역 저장에 실패했습니다.'); return }
  setSavedData({...customData})
  setDirtyKeys({})
  toast.success('번역이 저장되었습니다.')
}
```
