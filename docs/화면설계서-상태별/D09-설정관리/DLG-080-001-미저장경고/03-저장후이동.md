## 🧩 바이브코딩 프롬프트

```
모달: DLG-080-001 — 상태: 저장 후 이동 처리 (M3_080001_01)

handleSaveAndMove: async () => {
  setIsSavingModal(true)  // 버튼 로딩

  const { error } = await supabase
    .from('settings')
    .upsert({ branch_id: branchId, ...form })

  if (error) {
    // M3_080001_05: 저장 실패 → 토스트, 모달 유지
    toast.error('저장에 실패했습니다.')
    setIsSavingModal(false)
    return
  }

  // M3_080001_04: 저장 성공 → 탭 이동
  setSavedData({ ...form })
  setIsDirty(false)
  toast.success('저장 완료 후 이동합니다.')
  setShowUnsavedModal(false)
  setActiveTab(pendingTab)
  setPendingTab(null)
  setIsSavingModal(false)
}

버튼 로딩 중 UI:
<Button disabled>
  <Loader2 className="mr-2 h-4 w-4 animate-spin" />
  저장 중...
</Button>
```
