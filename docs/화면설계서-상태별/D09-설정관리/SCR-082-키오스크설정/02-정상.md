## 진입 조건

## 비주얼 델타
| 요소 | 상태 |
|---|---|
| 6개 섹션 | 모두 현재 값으로 렌더, 토글 활성 |
| D11 배지 | `'connected'` 초록, `'disconnected'` 빨강 |
| 프린터 배지 | 동일 패턴 |
| 저장 버튼 | disabled (isDirty=false) |
| 프리뷰 패널 | 실시간 현재 값 반영 |
| DirtyBadge / SavedBadge | 비표시 |

## 인터랙션
1. 토글 클릭 → `03-변경됨`
2. 음량 슬라이더 드래그 → `03-변경됨`
3. 색상 선택 → `03-변경됨`
4. 로고 업로드 → `03-변경됨`
5. D11 WebSocket 'disconnected' 이벤트 → `04-D11연결경고` 배너

## 비즈니스 룰 델타
- D11 상태는 WebSocket + polling 둘 다(fail-safe)
- `printerStatus`는 polling만 (60초)
- 프리뷰는 `form.watch()` 구독 → 키 입력당 렌더 (debounce 100ms로 조절 선택)

## 에지 케이스
- D11 feature flag off: IntegrationSection 전체 hidden
- 프린터 미연결: 배지만 '미연결', 섹션 렌더 유지
