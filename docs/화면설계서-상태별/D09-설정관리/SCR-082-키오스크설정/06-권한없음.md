# SCR-082 키오스크 설정 — 상태 06: 권한 없음

> 상속: [`00-기본화면.md`](./00-기본화면.md) — **델타**만 기술.

## 메타
| 항목 | 값 |
|---|---|
| 상태 코드 | `kiosk-forbidden` |
| 이전 | 진입 403 / role 불일치 |
| 다음 | 홈 이동 / 로그아웃 |

## 진입 조건
- 서버 403 E403001
- role이 `['superAdmin','primary','owner']` 아님

## 비주얼 델타
- 전체 영역을 `<ForbiddenState>`로 치환 (SCR-080 06과 동일 패턴, 메시지만 "키오스크 설정은 센터장 이상만 접근 가능")

## 인터랙션
1. "홈으로 이동" → `/`
2. "로그아웃" → `/logout`
3. "설정 홈으로" → `/settings`

## 비즈니스 룰 델타
- 사이드바 키오스크 항목 자체 미노출 (manager 이하)
- `AUDIT.ACCESS_DENIED` 기록
- 3회 접근 시도 시 보안 알림

## 🧩 바이브코딩 프롬프트 (델타)

```
상속: 00-기본화면.md §14

const canAccess = ['superAdmin','primary','owner'].includes(role);
if (!canAccess || errorCode === 'E403001') {
  return (
    <main className="min-h-screen bg-gray-50 flex items-center justify-center p-6">
      <div role="alert" className="max-w-md bg-white rounded-2xl shadow-xl p-8 text-center space-y-4">
        <Lock className="size-12 mx-auto text-red-600" />
        <h1 className="text-xl font-semibold">접근 권한이 없습니다</h1>
        <p className="text-sm text-gray-600">키오스크 설정은 센터장(owner) 이상만 접근할 수 있습니다.</p>
        <div className="flex flex-col gap-2">
          <button onClick={() => moveToPage('/')} className="h-11 rounded-lg bg-blue-600 text-white">홈으로 이동</button>
          <button onClick={() => moveToPage('/settings')} className="h-11 rounded-lg border">설정 홈으로</button>
        </div>
      </div>
    </main>
  );
}

useEffect(() => auditLog('ACCESS_DENIED', { role, path: '/settings/kiosk', branchId }), []);
```
