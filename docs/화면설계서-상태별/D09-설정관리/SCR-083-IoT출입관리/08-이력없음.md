# SCR-083 IoT/출입 관리 — 상태 08: 이력 없음 (Empty Logs)

> 상속: [`00-기본화면.md`](./00-기본화면.md) — **델타**만 기술.

## 메타
| 항목 | 값 |
|---|---|
| 상태 코드 | `iot-empty-logs` |
| 이전 | `02-정상-장비온라인` (해당 날짜 이력 0) |

## 진입 조건
- 선택 날짜의 `accessLogs.length === 0`
- 장비는 온라인 상태

## 비주얼 델타

```

│ 날짜: [2026-04-22] [방식▼] [검색]              │

```

| 요소 | 변경 |
|---|---|
| 이력 테이블 | `<EmptyState>`로 치환 |
| 필터 영역 | 유지 (날짜 변경 가능) |
| 장비 패널 | 변경 없음 |

## 인터랙션
1. **"오늘로 이동"** → `date = today` → 재조회
2. **"다른 날짜"** → DatePicker 오픈
3. 실시간 신규 이벤트 → Supabase realtime 으로 즉시 테이블 렌더

## 비즈니스 룰 델타
- 과거 30일 + 미래 날짜 불가(오늘 이하)
- 필터 조건 관련: 방식 선택 중이면 "필터 조건을 변경해보세요" 안내
- Realtime INSERT 구독 유지: 새 이력 들어오면 02-정상 전환

## 에지 케이스
- 오늘 00:00 직후 진입 → 정상 EmptyState (이력 0)
- 날짜 변경 시 로딩 → `01-로딩`(이력 영역만)
- 필터가 과도하게 좁음 → "필터 조건 완화" 힌트
