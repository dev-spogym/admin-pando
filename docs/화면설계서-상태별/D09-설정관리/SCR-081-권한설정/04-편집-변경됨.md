## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind v4 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-081 권한설정 — 상태: 편집 (체크박스 변경, amber 하이라이트)

파일: src/app/settings/permissions/page.tsx

편집 상태 체크박스 변경 시 (E_F6_081_06):
- 변경된 셀:
  className="ring-2 ring-amber-400 bg-amber-600/10 rounded"
  → 개별 셀 <td> 또는 체크박스 래퍼에 적용

- TC-081-003: 권한 토글 → ring-2 ring-amber 표시

변경 감지:
- changedCells: Set<string> (key: `{role}-{feature}-{action}`)
- handlePermChange(role, feature, action, value):
    const key = `${role}-${feature}-${action}`
    const newMatrix = deepClone(permMatrix)
    newMatrix[role][feature][action] = value
    setPermMatrix(newMatrix)
    if (value !== savedMatrix[role][feature][action]) {
      setChangedCells(prev => new Set([...prev, key]))
    } else {
      setChangedCells(prev => { prev.delete(key); return new Set(prev) })
    }
    setIsDirty(changedCells.size > 0)

셀 렌더링:
<td key={action} className={cn(
  "text-center p-2",
  changedCells.has(`${role}-${feature}-${action}`) && "ring-2 ring-amber-400 bg-amber-600/10 rounded"
)}>
  <Checkbox
    checked={permMatrix[role][feature][action]}
    onCheckedChange={(v) => handlePermChange(role, feature, action, v)}
  />
</td>

저장 버튼: variant="primary", disabled={!isDirty}
초기화 버튼: variant="outline", disabled={!isDirty}
  onClick: setPermMatrix(savedMatrix), setChangedCells(new Set()), setIsDirty(false)
  이벤트: E_F6_081_10 → Normal 복귀
```
