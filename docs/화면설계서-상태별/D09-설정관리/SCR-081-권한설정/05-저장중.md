# SCR-081 권한 설정 — 상태 05: 저장 중

> 상속: [`00-기본화면.md`](./00-기본화면.md) — **델타**만 기술.

## 메타
| 항목 | 값 |
|---|---|
| 상태 코드 | `perm-saving` |
| 이전 | `04-편집-변경됨`(저장 클릭), `DLG-081-002`(무시 저장) |
| 다음 | `06-저장완료`(200) / `07-에러`(4xx/5xx) |

## 진입 조건
- `saveMutation.isPending === true`

## 비주얼 델타

| 요소 | 변경 |
|---|---|
| 저장 버튼 | "저장 중..." + Loader2 + pulse, disabled |
| 초기화/모두허용/모두거부 | disabled |
| 매트릭스 전체 | `pointer-events-none opacity-60` |
| 역할 리스트 | 클릭 차단 |
| DirtyBadge | 유지 |
| 충돌 리스트 | 비표시 (저장 중에는 숨김) |

## 비즈니스 룰 델타
- 변경된 cell만 diff 생성: `changes[]` 배열
- API: `PUT /role-permissions/bulk` with `{ branchId, roleCode, permissions: Record<menuId, Perm>, dirtyKeys: string[] }`
- 타임아웃 15초 → abort → `07-에러` E500001
- 성공 시:
  1. `savedPermissions = { ...savedPermissions, [roleCode]: draft[roleCode] }`
  2. `dirtyCells.clear()`
  3. `06-저장완료` 전환 (배지 3초)
  4. `AUDIT.PERMISSION_UPDATE(roleCode, changes[])`

## 에지 케이스
- **네트워크 불안정**: 재시도는 서버가 idempotent 처리 (dirtyKeys 기반 delta)
- **다른 관리자 동시 저장**: 서버 `updatedAt` 충돌 → 409 → `07-에러` "재로드"
- **저장 중 역할 클릭**: 차단 (pointer-events-none)

## 🧩 바이브코딩 프롬프트 (델타)

```
상속: 00-기본화면.md §14

상태: perm-saving

<button disabled aria-busy="true"
        className={cn(primaryBtn, 'animate-pulse')}>
  <Loader2 className="size-4 animate-spin inline mr-1.5" />
  저장 중...
</button>

<fieldset disabled={isSaving}
          className={cn(isSaving && 'pointer-events-none opacity-60')}>
  {/* 매트릭스 + 역할 리스트 */}
</fieldset>

mutation:
const saveMutation = useMutation({
  mutationFn: (dto: BulkPermDto) => savePermissions(dto),
  onSuccess: (updated) => {
    setSavedPermissions(prev => ({ ...prev, [selectedRoleCode]: updated.permissions }));
    setDirtyCells(new Set());
    setJustSaved(true);
    setTimeout(() => setJustSaved(false), 3000);
    auditLog('PERMISSION_UPDATE', { roleCode: selectedRoleCode, changes: updated.changedKeys });
  },
  onError: (err) => setErrorCode(mapError(err)),
});

async function handleSave() {
  const dto = {
    branchId,
    roleCode: selectedRoleCode,
    permissions: draft[selectedRoleCode],
    dirtyKeys: Array.from(dirtyCells),
    baseUpdatedAt: data.updatedAt,
  };
  const ctrl = new AbortController();
  const t = setTimeout(() => ctrl.abort(), 15_000);
  try { await saveMutation.mutateAsync(dto, { signal: ctrl.signal }); }
  finally { clearTimeout(t); }
}
```
