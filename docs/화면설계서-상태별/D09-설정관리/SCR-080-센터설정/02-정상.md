### 인터랙션 (User Actions)
1. 탭 전환 → isDirty=true 시 DLG-080-001 미저장 경고 트리거
2. 필드 수정 → `03-변경됨` 상태로 전환
3. 저장 버튼 (isDirty=false) → disabled, 클릭 불가

### 비즈니스 룰
- isDirty=false → 저장 버튼 opacity-50 + cursor-not-allowed (이벤트: `E_F6_080_09`)
- 로고 이미지 최대 2MB, image/* 형식만 허용
- 사업자등록번호 10자리 형식 자동 포맷팅

### 에지 케이스
- 설정 미등록 → 기본값으로 초기화된 폼 표시 (Empty 아님, 폼은 동일)
- 이미지 없음 → 기본 플레이스홀더 아이콘
