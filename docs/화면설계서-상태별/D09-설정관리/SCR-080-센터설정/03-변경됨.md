## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind v4 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-080 센터설정 — 상태: 변경됨 (isDirty=true, amber 배지 표시)

파일: src/app/settings/page.tsx

변경됨 상태 UI 추가:
- PageHeader 우측: amber 배지 표시
  · <span className="inline-flex items-center px-2.5 py-0.5 rounded-full text-xs font-medium bg-amber-100 text-amber-800">저장되지 않은 변경사항</span>
- 저장 버튼: variant="primary" disabled={false} (활성화)
  · Button className="bg-primary text-white hover:bg-primary/90"

탭 전환 인터셉터:
- handleTabChange(newTab):
  if (isDirty) → setShowUnsavedModal(true), setPendingTab(newTab)
  else → setActiveTab(newTab)

브라우저 떠나기 경고:
- useEffect: window.addEventListener('beforeunload', handler)
- handler: if (isDirty) e.preventDefault(), e.returnValue = ''

isDirty 계산:
- useMemo: JSON.stringify(form) !== JSON.stringify(savedData)
- 실시간 감지: onChange 핸들러에서 setForm(prev => ({...prev, [field]: value}))

저장 핸들러:
- handleSave: async () => {
    setIsSaving(true)
    const { error } = await supabase.from('settings').upsert({...form, branch_id: branchId})
    if (error) toast.error('저장에 실패했습니다')
    else { setSavedData(form); toast.success('저장되었습니다') }
    setIsSaving(false)
  }
```
