## 🧩 바이브코딩 프롬프트

```
모달: DLG-085-001 공지 수정 — 기존 데이터 로드 (OpenEdit)

수정 모드 진입:
handleEditNotice(notice: Notice) {
  setEditingNotice(notice)
  setForm({
    title: notice.title,
    content: notice.content,
    isPinned: notice.is_pinned,
    sendNotification: false,  // 수정 시 알림 기본 off
    attachmentUrl: notice.attachment_url
  })
  setShowNoticeModal(true)
}

수정 저장:
handleSave: async () => {
  if (editingNotice) {
    // PATCH /notices/{id}  (M3_085001_02)
    const { error } = await supabase.from('notices')
      .update({ title: form.title, content: form.content, is_pinned: form.isPinned })
      .eq('id', editingNotice.id)
    if (error) { toast.error('수정 실패'); return }
    // M3_085001_04: 공지 정보 갱신
    onNoticeUpdated({ ...editingNotice, ...form })
    if (form.sendNotification) triggerX15Notification()  // M3_085001_05
  }
  setShowNoticeModal(false)
  toast.success('공지가 수정되었습니다.')
}

모달 제목: "공지 수정" (editingNotice !== null)
```

- 이벤트: `Closed → OpenEdit`
