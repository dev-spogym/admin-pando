## 메타

| 항목 | 값 |
|------|----|
| 모달 ID | DLG-081-003 |
| 상태 코드 | `saving` |
| 트리거 화면 | SCR-081, SCR-087 |
| 우선순위 | P1 |
| 이전 상태 | `02-열림-폼입력` |
| 다음 상태 | `01-닫힘` (성공) / `02-열림-폼입력` (실패) |

## 🧩 바이브코딩 프롬프트

```
handleCreate: async () => {
  setIsSaving(true)
  const { data, error } = await supabase
    .from('custom_roles')
    .insert({ branch_id: branchId, name: form.name, code: form.code, description: form.description })
    .select().single()

  if (error?.code === '23505') {
    // M3_081003_03: 409 코드 중복
    setCodeError('이미 사용 중인 역할 코드입니다.')
    setIsSaving(false)
    return
  }
  if (error) {
    // M3_081003_04: 500
    toast.error('역할 생성에 실패했습니다.')
    setIsSaving(false)
    return
  }

  // M3_081003_02: 성공
  if (form.baseRole) {
    // M3_081003_05: 기준 역할 권한 복사
    await copyPermissions(form.baseRole, data.id)
  }
  // M3_081003_06: 빈 권한 매트릭스 or 복사된 권한으로 시작
  onRoleCreated(data)  // 역할 목록 갱신 + 자동 선택
  setShowCreateRoleModal(false)
  resetForm()
  toast.success('역할이 생성되었습니다.')
}

저장 중 버튼:
<Button disabled><Loader2 className="mr-2 h-4 w-4 animate-spin" />저장 중...</Button>
```
