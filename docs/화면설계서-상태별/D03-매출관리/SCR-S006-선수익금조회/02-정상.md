## 📝 디스크립션

### 사용 시점 (Trigger)
- isLoading=false AND deferredItems.length > 0

### 필수 데이터
| 항목 | 테이블 | 설명 |
|------|--------|------|
| 활성 계약 | `contracts` | status=서명완료, remaining_count>0 |
| 회원 정보 | `members` | JOIN |
| 상품 정보 | `products` | JOIN |

### 인터랙션 (User Actions)
1. 기준월 변경 → 인식 예정 금액 재계산
2. 테이블 정렬 (잔여선수익금 desc)

### 비즈니스 룰
- 선수익금 = 계약금액 × (잔여횟수 / 전체횟수)
- 만료임박: 만료일 D-30 이내 → warning 배지
- 만료: 만료일 지남 → error 배지 (잔여 있으면 환불 대상)

### 에지 케이스
- 기간제 계약: 일할 계산 (잔여일수 / 전체일수)
- 소진율 100%: 선수익금 0원 (테이블에서 제외)

### 접근성 (A11y)
- ProgressBar `role="progressbar"` + `aria-valuenow={소진율}`
