```
Next.js 15 App Router + TypeScript + Tailwind v4 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-S002 POS판매 — 상태: 품절 카드 포함 정상 그리드

파일: src/app/pos/page.tsx (상품 그리드 내 품절 처리)

품절 카드 스타일:
const ProductCard = ({ product, onAdd }) => {
  const isOutOfStock = product.stock === 0;
  
  return (
    <div
      className={cn(
        "relative rounded-xl border p-4 transition",
        isOutOfStock
          ? "opacity-50 grayscale cursor-not-allowed bg-gray-50"
          : "cursor-pointer hover:shadow-md hover:border-blue-300"
      )}
      onClick={() => {
        if (isOutOfStock) return;
        onAdd(product);
      }}
      role="button"
      aria-disabled={isOutOfStock}
      aria-label={isOutOfStock ? `${product.name} 품절` : `${product.name} 장바구니에 추가`}
    >
      {isOutOfStock && (
        <div className="absolute top-2 right-2">
          <span className="bg-red-100 text-red-600 text-xs font-bold px-2 py-0.5 rounded-full">
            품절
          </span>
        </div>
      )}
      <p className={cn("font-semibold text-sm", isOutOfStock && "text-gray-400")}>
        {product.name}
      </p>
      <p className="text-xs text-gray-400 mt-1">{product.period}</p>
      <div className="mt-3">
        <span className={cn(
          "text-sm font-bold",
          isOutOfStock ? "text-gray-300" : "text-blue-600"
        )}>
          {formatNumber(product.cardPrice)}원
        </span>
      </div>
    </div>
  );
};

조건:
- product.stock === 0 → 품절 처리
- product.stock === null → 재고 무제한 (정상 표시)
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- 상품 그리드에 stock=0 상품이 존재하는 경우 (E_SOLDOUT_01)
- 정상 상품과 품절 상품이 혼재

### 필수 데이터
| 항목 | 조건 |
|------|------|
| products | stock=0 포함 |

### 인터랙션 (User Actions)
1. 품절 카드 클릭: 아무 동작 없음 (cursor-not-allowed)
2. 정상 카드 클릭: 장바구니 추가

### 비즈니스 룰
- stock=null: 재고 무제한 → 정상 카드
- stock=0: 품절 → 비활성
- stock>0: 정상 → 클릭 가능 (stock 감소 로직은 결제 완료 시)

### 접근성 (A11y)
- `aria-disabled="true"` 품절 카드
- 키보드: Enter/Space로 클릭 시 품절이면 무시

### 연결 화면
- 정상 카드 클릭 → `05-장바구니-채워짐`
