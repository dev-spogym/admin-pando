## 📝 디스크립션

### 사용 시점 (Trigger)
- E_OPEN_01: SCR-S009 [할부 등록] 버튼 클릭

### 필수 데이터
| 항목 | 설명 |
|------|------|
| selectedMember | 회원 검색으로 선택한 회원 |
| form | 상품명, 총금액, 할부 개월, 첫 납입일 |

### 인터랙션 (User Actions)
1. [검색] → 회원 검색 모달 (BuyerSearchModal 재사용)
2. 상품명 / 총금액 / 할부 개월 입력
3. 월 납입액 자동 계산 표시
4. [등록] → `02-유효성검사` (E_SUBMIT_01)
5. [닫기] / X → 닫힘

### 비즈니스 룰
- 필수값 (회원, 상품명, 총금액, 할부 개월) 미입력 시 [등록] 비활성화
- 월 납입액 = Math.ceil(total_amount / total_months)
- 할부 개월: 2~24개월 선택

### 연결 화면
- `02-유효성검사` → `03-처리중`
- 닫힘
