## 📝 디스크립션

### 사용 시점 (Trigger)
- isLoading=false AND installments.length > 0

### 필수 데이터
| 항목 | 테이블 | 설명 |
|------|--------|------|
| 할부 목록 | `installments` (가상) | branch_id |
| 회원 정보 | `members` | JOIN (이름) |

### 인터랙션 (User Actions)
1. 행 클릭 → DLG-S007 할부상세 (납입 스케줄 표시)
2. [할부 등록] → DLG-S009 할부등록 모달
3. 상태/날짜 필터 변경 → 재조회
4. [엑셀 다운로드] → exportToExcel()

### 비즈니스 룰
- 연체: next_due_date < 오늘 AND status === ACTIVE → OVERDUE 처리
- 완납: paid_months === total_months → COMPLETED
- 잔여금액 = total_amount - (paid_months × monthly_amount)

### 에지 케이스
- 연체 행: bg-red-50 + 연체 배지
- 완납 행: text-gray-400 (비활성 표시)
