## 📝 디스크립션

### 사용 시점 (Trigger)
- paymentMethod === '마일리지' AND mileageBalance < totalAmount
- E_VALID_MILEAGE_LACK_01 엣지

### 필수 데이터
| 항목 | 테이블 | 설명 |
|------|--------|------|
| mileageBalance | `members` | buyer.mileage |
| totalAmount | — | 장바구니 합계 |

### 인터랙션 (User Actions)
1. 결제수단 변경 → 마일리지 오류 해제
2. [구매자 검색] → DLG-S002 (회원 미선택 시)

### 비즈니스 룰
- 마일리지 잔액 조회: buyer 선택 시 실시간 조회
- 부족금액 표시: totalAmount - mileageBalance

### 연결 화면
- DLG-S002 구매자검색 (회원 변경)
- `01-대기` (결제수단 변경)
