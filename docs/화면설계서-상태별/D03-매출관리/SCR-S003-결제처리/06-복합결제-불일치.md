## 역할별 차이
전 역할 동일.

## 상태 고유 인터랙션
- 카드/현금/마일리지 입력 `onChange` → 실시간 합계 재계산
- 입력 debounce 100ms(깜빡임 방지)
- [자동 채우기] 버튼 → `현금 = finalAmount - 카드 - 마일리지`(음수면 0 fallback)
- 합계 일치 시 자동 `02-유효성-통과` 전이
- mileage 입력 > memberMileage → `05-마일리지부족` 전이

## 상태 고유 비즈니스 룰 델타
- 입력값은 정수(원 단위), 소수 무시 (clamp `Math.max(0, Math.floor(v))`)
- 각 입력 max는 finalAmount (부분이 전체 이상이 되면 자동 조정 X, error만 표시)
- 카드+현금+마일리지 = finalAmount 일 때만 isValid=true
- 마일리지=0이면 회원 미선택이어도 OK
- 마일리지>0이면 회원 필수(없으면 info 배너: "마일리지 사용은 회원 검색 후 이용 가능합니다.")

## 에지 케이스
- 초과(합계 > finalAmount) → `text-state-warning` "N원 초과"
- 부족(합계 < finalAmount) → `text-state-error` "N원 부족"
- 음수 입력 방지 → 0으로 clamp
- 자동 채우기 후 합계 일치 → 즉시 `02` 전이(300ms 이후)
- finalAmount 변경(할인 추가) → 자동 재검증

## 바이브코딩 프롬프트 (델타)
```
상태: pay-mixed-mismatch. 마스터 프롬프트에서 다음만 override.
const sum = mixed.card + mixed.cash + mixed.mileage;
const diff = finalAmount - sum;
const matchType = diff === 0 ? 'ok' : diff > 0 ? 'short' : 'over';

<MixedPaymentPanel>
  <MixedRow label="카드"     value={mixed.card}    onChange={v=>setMixed({...mixed, card: clamp(v)})} />
  <MixedRow label="현금"     value={mixed.cash}    onChange={v=>setMixed({...mixed, cash: clamp(v)})} />
  <MixedRow label="마일리지" value={mixed.mileage} suffix="P" max={memberMileage ?? Infinity}
            onChange={v=>setMixed({...mixed, mileage: clamp(Math.min(v, memberMileage ?? v))})} />
  <div className="border-t pt-2 space-y-1 text-xs">
    <div className="flex justify-between"><span>입력 합계</span><span className="tabular-nums">{formatKRW(sum)}</span></div>
    <div className="flex justify-between"><span>결제 금액</span><span className="tabular-nums">{formatKRW(finalAmount)}</span></div>
    <div role="status" aria-live="polite"
      className={cn('flex justify-between font-semibold',
        matchType==='ok' && 'text-state-success',
        matchType==='short' && 'text-state-error',
        matchType==='over' && 'text-state-warning')}>
      {matchType==='ok' && <><span>일치</span><span>✓</span></>}
      {matchType==='short' && <><span>부족</span><span>{formatKRW(diff)}</span></>}
      {matchType==='over' && <><span>초과</span><span>{formatKRW(-diff)}</span></>}
    </div>
  </div>
  {matchType!=='ok' && (
    <button type="button" onClick={autoFillCash} className="text-xs text-primary underline">
      자동 채우기 (현금)
    </button>
  )}
</MixedPaymentPanel>

- 결제 확인 버튼 disabled=true
- matchType==='ok' && 회원/잔액 조건 OK 즉시 02 전이
- autoFillCash: setMixed({...mixed, cash: Math.max(0, finalAmount - mixed.card - mixed.mileage)})
```
