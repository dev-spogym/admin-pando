## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind v4 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-S003 결제처리 — 상태: 복합결제 금액 불일치 (E_VALID_MIXED_MISMATCH_01)

파일: src/app/pos/payment/page.tsx (paymentMethod==='복합결제' && cardAmount+cashAmount !== totalAmount)

복합결제 금액 불일치 UI:
const mixedTotal = cardAmount + cashAmount;
const diff = totalAmount - mixedTotal;
const isOver = diff < 0;

{paymentMethod === '복합결제' && (
  <div className={cn(
    "mt-3 p-3 rounded-lg text-sm font-medium",
    mixedTotal === totalAmount
      ? "bg-green-50 text-green-700"
      : "bg-red-50 text-red-700"
  )}>
    <div className="flex justify-between">
      <span>입력 합계</span>
      <span>{formatKRW(mixedTotal)}</span>
    </div>
    <div className="flex justify-between">
      <span>결제 금액</span>
      <span>{formatKRW(totalAmount)}</span>
    </div>
    {mixedTotal !== totalAmount && (
      <div className="flex justify-between mt-1 font-bold border-t pt-1">
        <span>{isOver ? '초과' : '부족'}</span>
        <span>{formatKRW(Math.abs(diff))} {isOver ? '초과' : '부족'}</span>
      </div>
    )}
    {mixedTotal === totalAmount && (
      <div className="flex items-center gap-1 mt-1">
        <CheckCircle className="h-4 w-4" />
        <span>금액이 일치합니다.</span>
      </div>
    )}
  </div>
)}

자동 채우기 버튼 (현금 입력란 옆):
<button
  className="text-xs text-blue-500 underline ml-2"
  onClick={() => setCashAmount(totalAmount - cardAmount)}
>
  자동 채우기
</button>

결제 확인 버튼: disabled (mixedTotal !== totalAmount)
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- paymentMethod === '복합결제' AND cardAmount + cashAmount !== totalAmount
- E_VALID_MIXED_MISMATCH_01

### 인터랙션 (User Actions)
1. 카드/현금 금액 입력 → 실시간 합계 계산
2. [자동 채우기] → cashAmount = totalAmount - cardAmount
3. 금액 일치 시 → `02-유효성통과`

### 비즈니스 룰
- 실시간 검증 (onChange)
- 초과/부족 금액 표시

### 연결 화면
- 금액 일치 → `02-유효성통과` → DLG-S003
