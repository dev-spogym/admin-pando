```
유효성 검사 로직:
const isValid = useMemo(() => {
  if (!paymentMethod) return false;
  if (paymentMethod === '마일리지') {
    return buyer !== null && mileageBalance >= totalAmount;
  }
  if (paymentMethod === '복합결제') {
    return cardAmount + cashAmount === totalAmount;
  }
  return true;
}, [paymentMethod, buyer, mileageBalance, totalAmount, cardAmount, cashAmount]);

복합결제 금액 일치 표시:
{paymentMethod === '복합결제' && (
  <div className={cn(
    "mt-2 text-sm font-medium",
    cardAmount + cashAmount === totalAmount
      ? "text-green-600"
      : "text-red-500"
  )}>
    {cardAmount + cashAmount === totalAmount
      ? "금액이 일치합니다."
      : `${formatKRW(Math.abs(totalAmount - cardAmount - cashAmount))} ${
          cardAmount + cashAmount > totalAmount ? '초과' : '부족'
        }`
    }
  </div>
)}

결제 확인 버튼 (활성 상태):
<Button
  className="w-full"
  variant="primary"
  disabled={!isValid}
  onClick={() => setShowConfirmModal(true)}
>
  결제 확인
</Button>

활성 상태 시각:
- 버튼: bg-blue-600 hover:bg-blue-700 (isValid=true)
- 선택된 결제수단 라디오: border-blue-500 bg-blue-50
```
