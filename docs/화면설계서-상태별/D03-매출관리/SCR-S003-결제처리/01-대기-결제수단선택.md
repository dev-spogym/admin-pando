## 📝 디스크립션

### 사용 시점 (Trigger)
- SCR-S002에서 [결제하기] 클릭 후 장바구니 데이터 전달받아 진입
- paymentStatus === 'idle'

### 필수 데이터
| 항목 | 설명 |
|------|------|
| cartItems | SCR-S002에서 전달된 장바구니 |
| buyer | 선택된 구매자 (null 가능) |
| totalAmount | 장바구니 합계 |

### 인터랙션 (User Actions)
1. 결제수단 선택 (카드/현금/마일리지/복합결제)
2. 복합결제: 금액 분배 입력
3. 마일리지: 회원 마일리지 조회 → 잔액 표시
4. [결제 확인] → DLG-S003 열림

### 비즈니스 룰
- 장바구니 없이 진입: `07-장바구니없음` 에러 전환
- 마일리지 선택: buyer 필수 (비회원 불가)
- 복합결제: 카드+현금 합계 = totalAmount 일치 필요

### 에지 케이스
- 직접 URL 접근 (장바구니 없음): SCR-S002로 리다이렉트
