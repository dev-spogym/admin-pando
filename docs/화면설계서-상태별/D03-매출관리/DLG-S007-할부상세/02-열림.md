<Modal isOpen={isOpen} onClose={onClose} title="할부 상세">

## 📝 디스크립션

### 사용 시점 (Trigger)
- E_LOAD_OK_01: installment 상세 로드 완료
- 납입 스케줄 표시

### 필수 데이터
| 항목 | 설명 |
|------|------|
| installment | 할부 건 상세 + schedule 배열 |

### 인터랙션 (User Actions)
1. [납입 처리] → DLG-S008 중첩 (E_NAV_PAY_01)
2. [닫기] / X / ESC → 닫힘

### 비즈니스 룰
- 완납(COMPLETED) / 연체(OVERDUE): [납입 처리] 버튼 숨김 또는 disabled
- 납부 완료 회차: 초록 배경 + CheckCircle
- 미납 회차: 회색 배경 + Clock

### 연결 화면
- DLG-S008 납입처리 (중첩)
- 납입 완료 후 → 스케줄 갱신 (E_REFRESH_OK_01)
