검색 로직 (debounce 300ms):
useEffect(() => {
  const timer = setTimeout(async () => {
    if (searchQuery.length < 1) { setSearchResults([]); return; }
    setIsSearching(true);
    const { data } = await supabase
      .from('members')
      .select('id, name, phone, membership')
      .eq('branch_id', branchId)
      .or(`name.ilike.%${searchQuery}%,phone.ilike.%${searchQuery}%`)
      .limit(20);
    setSearchResults(data ?? []);
    setIsSearching(false);
  }, 300);
  return () => clearTimeout(timer);
}, [searchQuery]);

## 📝 디스크립션

### 사용 시점 (Trigger)
- E_OPEN_01: SCR-S002에서 [구매자 검색] 버튼 클릭
- 검색 입력 자동 포커스

### 필수 데이터
| 항목 | 테이블 | 조건 |
|------|--------|------|
| 회원 목록 | `members` | branch_id, 검색어 |

### 인터랙션 (User Actions)
1. 검색어 입력 → debounce 300ms → 실시간 검색
2. 회원 행 클릭 → buyer 설정 → 모달 자동 닫힘 (E_SELECT_01)
3. [비회원으로 진행] → buyer=null → 닫힘
4. X / ESC → 닫힘 (buyer 미설정)

### 비즈니스 룰
- 최소 1글자 입력 시 검색 시작
- 검색 결과 최대 20건
- 비회원 판매: buyer=null로 장바구니 진행

- 닫힘: SCR-S002 (buyer 설정 또는 null)
