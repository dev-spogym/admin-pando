## 📝 디스크립션

### 사용 시점 (Trigger)
- SCR-S001 최초 진입 시 (isLoading=true)
- 필터/탭 변경으로 데이터 재조회 시 (isLoading=true)
- 날짜 범위 변경 후 재조회 시

### 필수 데이터
| 항목 | 테이블 | 비고 |
|------|--------|------|
| 없음 | — | 로딩 중이므로 데이터 없음 |

### 인터랙션 (User Actions)
- 로딩 중에는 탭 클릭/필터 입력 비활성 (pointer-events-none)
- PageHeader 버튼(엑셀 다운로드, POS 이동)은 disabled 처리

### 비즈니스 룰
- 로딩 timeout: 15초 초과 시 에러 상태로 전환
- 탭 기본값: TAB-001 매출내역

### 에지 케이스
- 권한에 따라 스켈레톤 레이아웃 동일 (RBAC는 데이터 로드 후 적용)
- 네트워크 지연 3초 이상: 추가 안내 없음 (스켈레톤 유지)

### 접근성 (A11y)
- `role="status"` + `aria-busy="true"` 컨테이너
- `aria-label="매출 현황 데이터 로딩 중"` 스피너/스켈레톤 영역
