## 📝 디스크립션

### 사용 시점 (Trigger)
- TAB-002~007 클릭 시 해당 탭 데이터가 0건
- TAB-006 환불내역: status='환불' 건 없음
- TAB-007 미납내역: unpaid > 0 건 없음

### 필수 데이터
| 항목 | 설명 |
|------|------|
| sales (전체) | StatCard 합계 계산용 (필터 미적용) |
| tabFilteredData | 현재 탭 필터 적용 결과 = 0건 |

### 인터랙션 (User Actions)
1. 다른 탭 클릭 → 해당 탭 데이터 로드
2. SearchFilter 변경 → 재조회

### 비즈니스 룰
- StatCardGrid는 탭 필터 무관하게 전체 집계값 표시
- 탭 빈 상태에서도 엑셀 다운로드: disabled

### 에지 케이스
- TAB-001(매출내역)이 빈 상태라면 `03-빈상태`와 동일 처리

### 접근성 (A11y)
- 탭 패널 `role="tabpanel"` + `aria-label="{탭명} 데이터 없음"`

### 연결 화면
- 이전: `02-정상` (탭 전환)
- 다음: 탭 변경 또는 필터 변경
