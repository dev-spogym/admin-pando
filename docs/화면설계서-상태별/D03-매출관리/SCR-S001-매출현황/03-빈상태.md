## 📝 디스크립션

### 사용 시점 (Trigger)
- 선택 날짜 범위에 매출 데이터가 0건인 경우
- trainer 역할로 담당 건이 없는 경우

### 필수 데이터
| 항목 | 설명 |
|------|------|
| sales | 조회 결과 0건 배열 |

### 인터랙션 (User Actions)
1. [신규 결제 POS로 이동] 버튼 → `/pos`
2. 날짜 범위 변경 → 재조회
3. 탭 전환 → 빈 상태 유지 또는 데이터 있으면 정상 전환

### 비즈니스 룰
- StatCard는 0원으로 표시 (숨기지 않음)
- 탭은 모두 클릭 가능 (해당 탭 데이터도 0건이면 같은 빈 상태)
- 엑셀 다운로드 버튼: disabled 처리

### 에지 케이스
- trainer 자동 필터 적용 → 본인 담당 건 없음: 같은 빈 상태 + "담당 매출 데이터가 없습니다" 안내

### 접근성 (A11y)
- 빈 상태 컨테이너 `role="status"` + `aria-label="매출 데이터 없음"`
