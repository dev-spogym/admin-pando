## 📝 디스크립션

### 사용 시점 (Trigger)
- isLoading=false AND unpaidItems.length > 0

### 필수 데이터
| 항목 | 테이블 | 조건 |
|------|--------|------|
| 미수금 목록 | `sales` | unpaid > 0, branch_id |
| 회원 정보 | `members` | JOIN (이름) |

### 인터랙션 (User Actions)
1. TabNav 탭 전환 → 상태별 필터링 (로컬)
2. Pencil 아이콘 클릭 → DLG-S005 메모편집 모달
3. 날짜/검색어 필터 → 재조회
4. [엑셀 다운로드] → exportToExcel()

### 비즈니스 룰
- 연체: dueDate < 오늘 AND status !== PAID → 행 bg-red-50
- 부분납부: 0 < paid_amount < total_amount
- 납부완료(PAID)는 목록에서 제외 (unpaid > 0 조건)

### 에지 케이스
- 연체 30일 이상: 별도 강조 (빨간 텍스트)
- 탭 전환 시 해당 탭 0건 → 탭 빈 상태 메시지

### 접근성 (A11y)
- 연체 행 `aria-label="연체: {회원명} {금액}"`
- 메모 버튼 `aria-label="메모 편집: {회원명}"`
