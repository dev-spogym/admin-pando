```
const fetchSaleDetail = async (saleId: number) => {
  setIsDetailLoading(true);
  const { data, error } = await supabase
    .from('sales')
    .select('*')
    .eq('id', saleId)
    .single();
  if (error) {
    toast.error('데이터를 불러올 수 없습니다.');
    setDetailError(error);
    onClose(); // 모달 닫기
  } else {
    setSaleDetail(data);
  }
  setIsDetailLoading(false);
};

aria-busy="true" aria-label="매출 상세 데이터 로딩 중"
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- SCR-S001 매출 행 클릭 → E_OPEN_01 발생
- GET /api/sales/:id 요청 중

### 필수 데이터
| 항목 | 설명 |
|------|------|
| saleId | 클릭된 행의 ID |

### 인터랙션 (User Actions)
- X 버튼: 로딩 중 닫기 가능 (요청 취소)
- ESC: 닫기 가능

### 비즈니스 룰
- 로딩 timeout 5초 → 에러 토스트 + 모달 닫힘
- saleId가 없는 경우 즉시 에러
