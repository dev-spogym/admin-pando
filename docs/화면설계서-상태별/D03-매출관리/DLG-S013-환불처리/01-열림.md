```
Next.js 15 App Router + TypeScript + Tailwind v4 기반 'use client' 컴포넌트를 작성하라.
화면: DLG-S013 환불처리 — 상태: 열림 (신규 기획)

파일: src/app/refunds/page.tsx 내 RefundProcessModal

모달 크기: 440px × 480px

<Modal isOpen={isOpen} onClose={onClose} title="환불 처리">
  <div className="p-6 space-y-4">
    {/* 환불 정보 요약 */}
    <div className="bg-gray-50 rounded-lg p-4 space-y-2">
      <div className="flex justify-between text-sm">
        <span className="text-gray-500">회원명</span>
        <span className="font-medium">{refund.member_name}</span>
      </div>
      <div className="flex justify-between text-sm">
        <span className="text-gray-500">상품명</span>
        <span className="font-medium">{refund.product_name}</span>
      </div>
      <div className="flex justify-between text-sm">
        <span className="text-gray-500">원 결제금액</span>
        <span className="font-medium">{formatKRW(refund.original_amount)}</span>
      </div>
    </div>

    {/* 환불 유형 */}
    <div>
      <label className="block text-sm font-medium text-gray-700 mb-2">환불 유형</label>
      <div className="flex gap-3">
        <label className="flex items-center gap-2 cursor-pointer">
          <input
            type="radio"
            name="refundType"
            value="full"
            checked={refundType === 'full'}
            onChange={() => { setRefundType('full'); setRefundAmount(refund.original_amount); }}
            className="text-blue-600"
          />
          <span className="text-sm">전액 환불</span>
        </label>
        <label className="flex items-center gap-2 cursor-pointer">
          <input
            type="radio"
            name="refundType"
            value="partial"
            checked={refundType === 'partial'}
            onChange={() => { setRefundType('partial'); setRefundAmount(0); }}
            className="text-blue-600"
          />
          <span className="text-sm">부분 환불</span>
        </label>
      </div>
    </div>

    {/* 환불 금액 (부분 환불 시) */}
    {refundType === 'partial' && (
      <div>
        <label className="block text-sm font-medium text-gray-700 mb-1">환불 금액 *</label>
        <input
          type="number"
          value={refundAmount}
          onChange={(e) => setRefundAmount(Number(e.target.value))}
          max={refund.original_amount}
          min={1}
          placeholder="0"
          className="w-full border rounded-lg px-3 py-2 text-sm focus:outline-none focus:ring-2 focus:ring-blue-500"
        />
        <p className="text-xs text-gray-400 mt-1">최대 {formatKRW(refund.original_amount)}</p>
      </div>
    )}

    {/* 환불 사유 */}
    <div>
      <label className="block text-sm font-medium text-gray-700 mb-1">환불 사유 *</label>
      <textarea
        value={refundReason}
        onChange={(e) => setRefundReason(e.target.value)}
        placeholder="환불 사유를 입력하세요"
        rows={3}
        className="w-full border rounded-lg px-3 py-2 text-sm focus:outline-none focus:ring-2 focus:ring-blue-500 resize-none"
      />
    </div>
  </div>

  <div className="p-4 border-t flex gap-3">
    <Button variant="outline" className="flex-1" onClick={onClose}>취소</Button>
    <Button
      variant="danger"
      className="flex-1"
      onClick={handleConfirm}
      disabled={!refundReason || (refundType === 'partial' && refundAmount <= 0)}
    >
      환불 확인
    </Button>
  </div>
</Modal>

const [refundType, setRefundType] = useState<'full' | 'partial'>('full');
const [refundAmount, setRefundAmount] = useState(refund?.original_amount ?? 0);
const [refundReason, setRefundReason] = useState('');
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- E_OPEN_01: SCR-S007 [환불 처리] 버튼 클릭

### 필수 데이터
| 항목 | 설명 |
|------|------|
| refund | 환불 요청 건 (member_name, product_name, original_amount) |
| refundType | 전액/부분 |
| refundAmount | 환불 금액 |
| refundReason | 환불 사유 |

### 인터랙션 (User Actions)
1. 환불 유형 선택 (전액/부분)
2. 부분 환불 시 금액 입력
3. 환불 사유 입력
4. [환불 확인] → `02-처리중`
5. [취소] / X → 닫힘

### 비즈니스 룰
- 사유 미입력 또는 부분 환불 시 금액 0 이하면 [환불 확인] 비활성화
- 부분 환불 금액은 원 결제금액 초과 불가
- owner, manager만 접근

### 연결 화면
- `02-처리중`
- 닫힘
