## 📝 디스크립션

### 사용 시점 (Trigger)
- 프로필 로드 완료 후 기본 상태
- 편집 모드에서 저장 또는 취소 후 복귀

### 필수 데이터
| 항목 | 테이블 | 비고 |
|------|--------|------|
| 사용자 정보 | `users` | id 기준 |
| 소속 지점 | `branches` JOIN | branch_id |
| 마지막 로그인 | `audit_logs` | event='LOGIN', 최신 1건 |

### 인터랙션 (User Actions)
1. "편집" 버튼 → `03-편집모드`
2. "비밀번호 변경" → SCR-106
3. 2FA 토글 → 활성화/비활성화 확인 모달

### 비즈니스 룰
- 이메일은 읽기 전용 (변경 불가, 관리자만 가능)
- superAdmin은 소속 지점 "전체" 표시
- 역할 정보 읽기 전용 (관리자 권한 필요)

### 에지 케이스
- 아바타 이미지 없음 → 이름 이니셜 원형 배지 표시
- 전화번호 없음 → "-" 표시

### 접근성 (A11y)
- 프로필 영역 `role="region"` `aria-label="내 프로필"`
- 편집 버튼 `aria-label="프로필 편집"`

### 연결 화면
- 편집: `03-편집모드`
- 비밀번호 변경: SCR-106
