## 📝 디스크립션

### 사용 시점 (Trigger)
- 이메일 링크 클릭 → 서버 토큰 검증 실패
  - `expired`: 30분 초과
  - `invalid`: 토큰 형식 오류 또는 이미 사용된 토큰

### 필수 데이터
- tokenState (서버 응답 기반)

### 인터랙션 (User Actions)
1. "재설정 링크 다시 받기" → `01-이메일입력`
2. "로그인으로 돌아가기" → SCR-100

### 비즈니스 룰
- 토큰은 1회 사용 후 무효화
- 이미 사용된 토큰 접근 → invalid 상태

### 에지 케이스
- URL에 토큰 파라미터 없음 → `01-이메일입력`으로 자동 전환

### 접근성 (A11y)
- `role="alert"` `aria-live="assertive"`

### 연결 화면
- 재요청: `01-이메일입력`
- 로그인: SCR-100
