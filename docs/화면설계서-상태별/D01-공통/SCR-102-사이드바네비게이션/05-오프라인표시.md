## 📝 디스크립션

### 사용 시점 (Trigger)
- `window.offline` 이벤트 수신
- `navigator.onLine === false`

### 필수 데이터
- navigator.onLine 상태

### 인터랙션 (User Actions)
1. 메뉴 클릭 → 오프라인 상태 토스트 표시 (화면 이동은 허용, 데이터 로드 실패)
2. 네트워크 복구 → 표시기 자동 제거

### 비즈니스 룰
- 오프라인 중 메뉴 이동 자체는 허용 (클라이언트 라우팅)
- 데이터 로드 실패는 각 화면에서 처리

### 에지 케이스
- 간헐적 오프라인(< 1초) → 감지 후 즉시 복구 표시

### 접근성 (A11y)
