## 📝 디스크립션

### 사용 시점 (Trigger)
- 메뉴 로딩 완료 후 기본 상태
- `03-접힘`에서 펼치기 토글 클릭

### 필수 데이터
| 항목 | 출처 | 비고 |
|------|------|------|
| 메뉴 구성 | hasPermission 기반 필터 | 역할별 상이 |
| 현재 경로 | usePathname() | 활성 메뉴 하이라이트 |
| 사용자 정보 | useAuthStore | 하단 프로필 |

### 인터랙션 (User Actions)
1. 메뉴 항목 클릭 → 해당 화면 이동
2. 접기 버튼(ChevronLeft) → `03-접힘`
3. 하단 프로필 → SCR-105 프로필/계정

### 비즈니스 룰
- 역할별 메뉴 필터링:
  - staff: 매출 관리 메뉴 숨김
  - fc: 직원 관리 메뉴 숨김
  - superAdmin: 지점 전환 드롭다운 추가 표시
- 현재 경로 기반 활성 메뉴 하이라이트
- 사이드바 상태(펼침/접힘) localStorage 유지

### 에지 케이스
- 접근 불가 메뉴 직접 URL 접근 → 403 에러 페이지
- 메뉴 항목 20개 초과 → 스크롤 (overflow-y-auto)

### 접근성 (A11y)
- `<nav>` role, `aria-label="주 내비게이션"`
- 활성 메뉴 `aria-current="page"`
- 접기 버튼 `aria-label="사이드바 접기"` `aria-expanded="true"`

### 연결 화면
- 각 메뉴 항목 → 해당 도메인 화면
- 프로필: SCR-105
