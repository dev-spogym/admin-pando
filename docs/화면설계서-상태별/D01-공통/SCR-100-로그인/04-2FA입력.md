## 상태 진입 조건
- 로그인 성공 1차 인증 통과, 서버가 `needs2FA=true` 반환
- 서버가 임시 토큰(`tempToken`) 발급하여 클라이언트로 전달
- 이 상태는 **최대 5분(300초)** 유효. 초과 시 `01-기본` 복귀 + 배너 "인증 시간이 초과되었습니다"

## 상태 고유 인터랙션
1. **자동 포커스**: 진입 시 첫 번째 OTP 칸에 focus
2. **자동 이동**: 숫자 입력 시 다음 칸으로 자동 포커스 이동 (`onInput`)
3. **Backspace**: 값 있을 땐 현재 칸 지움, 빈 칸에서 누르면 이전 칸으로 이동
4. **붙여넣기**: 6자리 코드 일괄 붙여넣기 지원 (`onPaste` → `split`)
5. **6자리 완성**: 자동 제출 (별도 버튼 클릭 없이)
6. **인증 확인 버튼**: 명시적 제출도 지원
7. **코드 재전송**: `POST /auth/2fa/resend` 호출, 60초 쿨다운 후 재활성화
8. **취소**: `01-기본` 복귀, tempToken 폐기

## 상태 고유 비즈니스 룰 델타
- **OTP 형식**: 숫자 6자리
- **만료**: 서버 JWT tempToken `exp=now+300s`. 클라이언트 동일 카운트다운
- **재전송 쿨다운**: 60초. 쿨다운 중 링크 `disabled`
- **오입력 3회**: OTP 무효 연속 3회 → tempToken 무효화 → `01-기본` 복귀 + "인증 실패. 처음부터 다시 시도하세요" (E401005)
- **서버 호출**: `POST /auth/2fa/verify { tempToken, otp }`
- **성공 응답**: 세션 발급 → 역할별 대시보드 이동
- **실패 응답(`E401003`)**: OTP 필드 흔들림 애니메이션 + 값 초기화 + 에러 배너

## 에지 케이스
- **만료 후 제출**: 서버가 `E401004` 반환 → "인증 시간이 초과되었습니다" 배너 + `01-기본`
- **페이지 새로고침**: tempToken 휘발 → `01-기본` 복귀
- **브라우저 뒤로가기**: 취소로 간주 → `01-기본` 복귀
- **여러 탭 동시 인증**: 하나 성공 시 `storage` 이벤트로 다른 탭도 대시보드 이동

OTP 입력 로직:
const [otpDigits, setOtpDigits] = useState<string[]>(Array(6).fill(''));
const otpRefs = useRef<(HTMLInputElement | null)[]>([]);
const [attempts, setAttempts] = useState(0);

function handleChange(i: number, value: string) {
  if (!/^\d?$/.test(value)) return;
  const next = [...otpDigits]; next[i] = value; setOtpDigits(next);
  if (value && i < 5) otpRefs.current[i+1]?.focus();
  if (next.join('').length === 6) verifyOtp({ preventDefault() {} } as any);
}

function handleKey(i: number, e: React.KeyboardEvent<HTMLInputElement>) {
  if (e.key === 'Backspace' && !otpDigits[i] && i > 0) otpRefs.current[i-1]?.focus();
  if (e.key === 'ArrowLeft'  && i > 0) otpRefs.current[i-1]?.focus();
  if (e.key === 'ArrowRight' && i < 5) otpRefs.current[i+1]?.focus();
}

function handlePaste(e: React.ClipboardEvent) {
  const text = e.clipboardData.getData('text').replace(/\D/g,'').slice(0,6);
  if (text.length === 6) {
    setOtpDigits(text.split(''));
    otpRefs.current[5]?.focus();
    e.preventDefault();
  }
}

만료 카운트다운:
useEffect(() => {
  const expiresAt = Date.now() + 300_000;
  const id = setInterval(() => {
    const r = Math.max(0, Math.ceil((expiresAt - Date.now())/1000));

검증 API:
async function verifyOtp(e) {
  e.preventDefault();
  setIsVerifying(true);
  const { data, error } = await api.post('/auth/2fa/verify',
    { tempToken, otp: otpDigits.join('') });
  if (error?.errorCode === 'E401003') {
    setOtpDigits(Array(6).fill(''));
    otpRefs.current[0]?.focus();
    setErrorCode('E401003');
    const next = attempts + 1;
    setAttempts(next);
    if (next >= 3) { setStep('credentials'); setErrorCode('E401005'); }
  } else if (data) {
    useAuthStore.setSession(data.session, data.user);
    moveToPage(roleToPath(data.user.role));
  }
  setIsVerifying(false);
}

재전송:
async function resend() {
  await api.post('/auth/2fa/resend', { tempToken });
  setResendCooldown(60);
  const id = setInterval(() => {
    setResendCooldown(c => { if (c <= 1) { clearInterval(id); return 0; } return c-1; });
  }, 1000);
}
