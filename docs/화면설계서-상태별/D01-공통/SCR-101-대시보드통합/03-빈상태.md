## 📝 디스크립션

### 사용 시점 (Trigger)
- 신규 지점 생성 직후 (데이터 전혀 없음)
- 모든 KPI 값이 0인 경우
- 선택 기간에 해당하는 데이터 없음

### 필수 데이터
- 모든 KPI 쿼리 반환값 = 0 또는 빈 배열

### 인터랙션 (User Actions)
1. "회원 등록하기" CTA → SCR-M002
2. "수업 개설하기" CTA → SCR-C003
3. 기간 필터 변경 → 다른 기간 데이터 조회

### 비즈니스 룰
- 빈 상태에서도 사이드바 네비게이션 전체 활성
- KPI 카드는 0값으로 렌더링 (스켈레톤 아님)
- staff 권한은 CTA 버튼 중 일부 숨김 (hasPermission)

### 에지 케이스
- 기간 필터 '오늘'에는 데이터 없지만 '이번달'에는 있음 → 필터 변경 유도 안내
- 신규 지점 초기 상태 → "시작하기" 가이드 우선 표시

### 접근성 (A11y)
- 빈 상태 영역 `role="status"` `aria-label="데이터 없음"`
- CTA 버튼 명확한 레이블

### 연결 화면
- 회원 등록: SCR-M002
- 수업 개설: SCR-C003
