## 📝 디스크립션

- `window.navigator.onLine === false`
- API 요청 시 네트워크 에러 (`TypeError: Failed to fetch`)
- 대시보드 정상 표시 중 네트워크 단절

### 필수 데이터
- `navigator.onLine` 상태
- 캐시된 이전 데이터 (있을 경우 희미하게 표시)
