## 📝 디스크립션

### 사용 시점 (Trigger)
- 폼 편집 중(isDirty=true) 이탈 시도:
  - 뒤로가기 버튼
  - 사이드바 메뉴 클릭
  - 브라우저 탭 닫기 (beforeunload)
  - URL 직접 변경

### 필수 데이터
| 항목 | 출처 | 비고 |
|------|------|------|
| isDirty | 폼 상태 | 변경사항 존재 여부 |
| destination | 라우터 이벤트 | 이탈 목적지 |

### 인터랙션 (User Actions)
1. "나가기" → 변경사항 버리고 destination 이동
2. "계속 편집" / ESC / 배경 / X → 편집 화면 복귀

### 비즈니스 룰
- isDirty=false 상태에서 이탈 → 모달 없이 즉시 이동
- 브라우저 탭 닫기(beforeunload) → 브라우저 기본 확인 다이얼로그 사용 (커스텀 모달 불가)
- 폼 저장 성공 → isDirty=false → 이탈 경고 없음

### 에지 케이스
- DLG-004 저장 확인 중 이탈 시도 → DLG-002 우선
- 로그아웃 버튼 클릭 → DLG-002 없이 DLG-001 로그아웃 확인 표시

### 접근성 (A11y)
- `role="alertdialog"` `aria-modal="true"` `aria-labelledby="leave-warning-title"`
- 모달 열림 시 "나가기" 버튼에 포커스
- 닫힘 시 이탈 시도한 요소로 포커스 복원

### 연결 화면
- 나가기: `02-이탈허용` → 목적지 화면
- 유지: 현재 편집 화면
