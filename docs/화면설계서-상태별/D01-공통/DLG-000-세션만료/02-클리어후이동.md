## 🧩 바이브코딩 프롬프트
```
Next.js 15 App Router + TypeScript + Tailwind 기반 'use client' 컴포넌트를 작성하라.
화면: DLG-000 세션만료 — 상태: 클리어 후 로그인 이동

파일: src/components/common/SessionExpiredModal.tsx (handleConfirm 핸들러)

처리 순서:
1. useAuthStore.reset() → Zustand 스토어 초기화
2. localStorage.removeItem('supabase.auth.token')
3. sessionStorage.clear()
4. supabase.auth.signOut() (fire-and-forget, 결과 무시)
5. router.replace('/login?session=expired')
6. 모달 닫힘 (isOpen = false)

데이터:
- 모달 닫힘 후 처리
- /login?session=expired → SCR-100에서 "세션이 만료되어 로그아웃되었습니다" 토스트

인터랙션:
- 즉시 처리 후 리다이렉트 (로딩 UI 없음)

사용 유틸:
- useAuthStore, supabase, router
```
