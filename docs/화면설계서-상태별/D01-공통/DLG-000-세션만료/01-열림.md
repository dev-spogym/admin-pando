## 📝 디스크립션

### 사용 시점 (Trigger)
- API 인터셉터에서 401 Unauthorized 응답 감지
- Supabase 세션 만료 이벤트 (`onAuthStateChange → SIGNED_OUT`)
- 자동 트리거 (사용자 액션 없이)

### 필수 데이터
- 없음 (모달 자체는 정적)

### 인터랙션 (User Actions)
1. "다시 로그인" → 세션 클리어 + `/login`
2. X 버튼 → 동일 (`02-클리어후이동`)
3. ESC → 무시 (강제 로그인)
4. 배경 클릭 → 무시

### 비즈니스 룰
- 배경 스크롤 잠금 (`document.body.style.overflow = 'hidden'`)
- ESC 키 이벤트 preventDefault
- 모달 중복 열림 방지 (전역 상태 플래그)
- 세션 만료 중 진행 중인 폼 작업 → 데이터 손실 허용 (경고 없음)
