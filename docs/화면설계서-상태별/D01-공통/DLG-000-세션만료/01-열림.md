# DLG-000 세션만료 — 상태 01: 열림 (오픈 직후)

> 상속: [`00-기본화면.md`](./00-기본화면.md) — 이 문서는 **변경점(델타)**만 기술합니다.

## 메타
| 항목 | 값 |
|------|----|
| 상태 코드 | `session-expired-open` |
| 상태 ID | STATE_OPEN |
| 다이어그램 노드 | `M1_생명주기.md` → STATE_OPEN |

| 다음 상태 | `02-클리어후이동` (버튼 클릭) |

## 상태 진입 조건
- `useSessionStore.open === true` 로 전환된 직후
- `isExpiredDialogOpen` 가드를 통과한 최초 오픈 이벤트 (중복 오픈 아님)
- 현재 경로가 `/login`, `/reset-password`, `/forbidden`, `/not-found` 가 **아님**

## 상태 고유 비주얼 델타
**기본 렌더 그대로**. 마스터(00) 레이아웃이 전체 그대로 보여지는 기준 상태.
- backdrop + 모달 카드 모두 렌더됨
- 버튼 활성(`bg-blue-600`) 상태, 로딩 스피너 없음
- 타이틀/본문 모두 기본 문구
- 아이콘은 `Lock + amber`

## 상태 고유 인터랙션
1. **오픈 직후**: Primary 버튼에 `autoFocus` → 키보드 사용자도 즉시 Enter 로 진행 가능
2. **ESC 키**: `preventDefault()` 로 차단 (강제 모달)
3. **배경 클릭**: 핸들러 없음 (차단)
4. **Tab 키**: 포커스가 Primary 버튼에서 순환만 하도록 `focus trap` 적용
5. **페이지 뒤로가기(popstate)**: 동작하나, 세션 상태는 여전히 무효 → 다음 API 요청에서 다시 다이얼로그 오픈

## 역할별 차이
- 역할별 차이 없음 — 모든 역할이 동일한 UI/카피
- 단, `reason` 에 따른 본문 문구 분기:
  - `E401002` 또는 미지정 → "보안을 위해 일정 시간 이상 활동이 없어 자동 로그아웃되었습니다."
  - `E401003` → "로그인 정보가 유효하지 않습니다."
  - `idle-timeout` → "일정 시간 동안 활동이 없어 자동 로그아웃되었습니다."

## 상태 고유 비즈니스 룰 델타
- 오픈 시점에 **다른 토스트/모달 자동 정리**: `toast.dismissAll()`, 열린 모달 닫기
- `document.body.style.overflow = 'hidden'` 으로 배경 스크롤 잠금
- `redirectPath` 가 미지정이면 현재 경로를 저장 (SSR 안전: `typeof window !== 'undefined'` 가드)
- 멀티탭: 다른 탭이 이미 `/login` 이라면 조용히 오픈 생략 (store 오픈 자체를 막음)

## 에지 케이스
- **이미 /login 에서 수동 signOut 중**: 다이얼로그 오픈 금지 (무한 리다이렉트 방지)
- **오프라인 상태에서 401**: 인터셉터가 오프라인 에러로 분류하면 다이얼로그 대신 토스트 ("네트워크 연결을 확인해주세요")
- **401 E401001(인증 실패)**: 이 에러코드는 **로그인 폼 에러**이지 세션 만료가 아니므로 다이얼로그 오픈 금지
- **다중 탭 동시 만료**: `storage` 이벤트 수신 탭은 중복 오픈 가드에 걸려 1회만 처리

## 🧩 바이브코딩 프롬프트 (델타)
```
상속: 00-기본화면.md §14 프롬프트 전체 적용
상태: session-expired-open (열림)

이 상태에서만 적용할 변경점:
- loading=false (버튼 활성)
- reason에 따른 카피 스위칭만 수행 (마스터 프롬프트의 handleGoLogin 이전 단계)
- 오픈 이펙트:
  useEffect(() => {
    if (open) {
      btnRef.current?.focus();
      toast.dismissAll?.();
      document.body.style.overflow = 'hidden';
    }
    return () => { document.body.style.overflow = ''; };
  }, [open]);

- 탭 동기화 broadcast 수신자 활성화
  useEffect(() => {
    const onStorage = (e: StorageEvent) => {
      if (e.key === 'logout_broadcast') useSessionStore.getState().openExpired('manual');
    };
    window.addEventListener('storage', onStorage);
    return () => window.removeEventListener('storage', onStorage);
  }, []);
```
