## 📝 디스크립션

### 사용 시점 (Trigger)
- `02-정상`에서 전체화면(Maximize2) 버튼 클릭

### 필수 데이터
- specContent (동일)
- isFullscreen = true

### 인터랙션 (User Actions)
1. Minimize2 버튼 / ESC → `02-정상` 복귀
2. X 버튼 → 오버레이 완전 닫힘
3. 뷰어 스크롤/줌 정상 동작

### 비즈니스 룰
- 전체화면 중 사이드바/헤더 접근 불가
- 전체화면 상태 브라우저 히스토리에 추가 안 함

### 에지 케이스
- 전체화면 중 새로고침 → 오버레이 닫힘 (상태 휘발)

### 접근성 (A11y)
- 전체화면 진입 시 포커스 헤더로 이동
- ESC 키 `aria-keyshortcuts="Escape"` 안내
- `aria-label="전체화면 모드 화면설계서"`

### 연결 화면
- 복귀: `02-정상`
