## 📝 디스크립션

### 사용 시점 (Trigger)
- 저장 API → 200/201 성공 응답

### 필수 데이터
- afterSaveAction, destination

### 인터랙션 (User Actions)
- 없음 (자동 처리)

### 비즈니스 룰
- dirty 상태 초기화 (이탈 경고 비활성화)
- stay: 폼 읽기 모드 전환 또는 편집 계속 (화면별 정책)
- navigate: 이탈 경고 없이 목적지 이동 (dirty=false이므로)
- Zustand store / React Query 캐시 갱신

### 에지 케이스
- navigate 목적지 없음 → stay 폴백

### 접근성 (A11y)
- 토스트 `role="status"` `aria-live="polite"`

### 연결 화면
- stay: 현재 편집/읽기 화면
- navigate: 목적지 화면
