## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind v4 기반 'use client' 컴포넌트를 작성하라.
화면: DLG-I001 수동출석등록 — 상태: 회원 검색 드롭다운

파일: src/components/attendance/ManualAttendanceModal.tsx (검색 드롭다운 부분)

검색 로직:
const [searchQuery, setSearchQuery] = useState('')
const [searchResults, setSearchResults] = useState<Member[]>([])
const [isSearching, setIsSearching] = useState(false)
const [selectedMember, setSelectedMember] = useState<Member | null>(null)

useEffect(() => {
  if (searchQuery.length < 2) { setSearchResults([]); return }
  const timer = setTimeout(async () => {
    setIsSearching(true)
    const { data } = await supabase
      .from('members')
      .select('id, name, phone, member_number')
      .eq('branch_id', branchId)
      .or(`name.ilike.%${searchQuery}%,phone.ilike.%${searchQuery}%,member_number.ilike.%${searchQuery}%`)
      .limit(10)
    setSearchResults(data ?? [])
    setIsSearching(false)
  }, 300) // debounce 300ms
  return () => clearTimeout(timer)
}, [searchQuery])

[검색 드롭다운 UI]
{(searchResults.length > 0 || isSearching) && (
  <div className="absolute top-full left-0 right-0 mt-1 bg-white border border-gray-200 rounded-lg shadow-lg z-10 max-h-48 overflow-y-auto">
    {isSearching ? (
      <div className="px-4 py-3 text-sm text-gray-400 flex items-center gap-2">
        <svg className="animate-spin w-4 h-4" /> 검색 중...
      </div>
    ) : searchResults.map(member => (
      <button
        key={member.id}
        onClick={() => handleSelectMember(member)}
        className="w-full px-4 py-2.5 text-left hover:bg-blue-50 flex items-center justify-between group"
      >
        <div>
          <p className="text-sm font-medium text-gray-800">{member.name}</p>
          <p className="text-xs text-gray-400">{member.phone} · #{member.member_number}</p>
        </div>
        <svg ... className="w-4 h-4 text-gray-300 group-hover:text-blue-500" />
      </button>
    ))}
  </div>
)}

검색 결과 없음:
{searchQuery.length >= 2 && !isSearching && searchResults.length === 0 && (
  <div className="absolute top-full left-0 right-0 mt-1 bg-white border border-gray-200 rounded-lg shadow-lg z-10 px-4 py-3">
    <p className="text-sm text-gray-400">검색 결과가 없습니다</p>
  </div>
)}

회원 선택 완료:
handleSelectMember = (member: Member) => {
  setSelectedMember(member)
  setSearchQuery(member.name) // 입력창에 이름 표시
  setSearchResults([])         // 드롭다운 닫기
}

선택된 회원 표시:
{selectedMember && (
  <div className="flex items-center gap-2 mt-2 px-3 py-2 bg-blue-50 rounded-lg">
    <div className="w-6 h-6 rounded-full bg-blue-200 flex items-center justify-center text-xs text-blue-700 font-bold">
      {selectedMember.name[0]}
    </div>
    <div className="flex-1">
      <p className="text-sm font-medium text-blue-800">{selectedMember.name}</p>
      <p className="text-xs text-blue-500">{selectedMember.phone}</p>
    </div>
    <button onClick={() => { setSelectedMember(null); setSearchQuery('') }}
      className="text-blue-400 hover:text-blue-600">
      <svg ... className="w-4 h-4" /> {/* X 아이콘 */}
    </button>
  </div>
)}
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- 회원 검색 Input에 2글자 이상 입력
- debounce 300ms 후 Supabase 조회

### 필수 데이터
- `members` WHERE `branch_id=?` AND (name/phone/member_number ILIKE)
- 최대 10건 표시

### 인터랙션 (User Actions)
1. 검색어 입력 → 드롭다운 표시
2. 회원 클릭 → 선택 완료 (드롭다운 닫힘)
3. 선택 취소 X → selectedMember = null

### 비즈니스 룰
- 2글자 미만: 드롭다운 미표시
- debounce 300ms (API 과호출 방지)
- 검색 결과 없음: "검색 결과가 없습니다" 표시
- 선택 후 이름 입력창 표시 + 파란 선택 배지 표시

### 에지 케이스
- 동명이인: 전화번호/회원번호로 구분
- 탈퇴 회원: 검색 결과에서 제외 (`status != '탈퇴'`)

### 연결 화면
- 이전: `01-열림-초기`
- 다음: 선택 완료 → `01-열림-초기` (채널 선택으로 이동)
