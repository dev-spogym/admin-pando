```
화면: DLG-I001 수동출석등록 — 상태: 저장 중

파일: src/components/attendance/ManualAttendanceModal.tsx

저장 중 상태:
const [isSubmitting, setIsSubmitting] = useState(false)

submitAttendance 함수:
async function submitAttendance(force = false) {
  setIsSubmitting(true)
  try {
    const { data, error } = await supabase
      .from('attendance')
      .insert({
        member_id: selectedMember!.id,
        branch_id: branchId,
        channel: selectedChannel,
        checked_in_at: attendanceTime,
        memo,
        source: 'manual',
        created_by: userId,
        force,
      })
      .select()
      .single()

    if (error?.code === '23505') {  // unique constraint (중복)
      setDuplicateWarning(true)
      return
    }
    if (error) throw error

    toast.success(`${selectedMember!.name}님 수동 출석이 등록되었습니다`)
    onSuccess(data)
    onClose()
  } catch (e) {
    toast.error('출석 등록에 실패했습니다')
    setApiError(String(e))
  } finally {
    setIsSubmitting(false)
  }
}

[저장 중 버튼 상태]
<button
  disabled={isSubmitting}
  className="flex-1 px-4 py-2 bg-blue-600 text-white text-sm font-medium rounded-lg
             disabled:opacity-60 flex items-center justify-center gap-2"
>
  {isSubmitting ? (
    <><svg className="animate-spin w-4 h-4" /> 등록 중...</>
  ) : '저장'}
</button>

- 저장 중 취소 버튼 비활성화 (isSubmitting)
- 폼 전체 pointer-events-none
- 모달 X 버튼 비활성화 (닫기 차단)
```
