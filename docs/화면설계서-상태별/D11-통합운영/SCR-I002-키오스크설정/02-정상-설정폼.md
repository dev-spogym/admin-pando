- 키오스크 연결 상태 정상

### 필수 데이터
| 블록 | 소스 | 설명 |
|------|------|------|
| 설정값 | `kiosk_sessions` 또는 API | 현재 설정 |
| 연결 상태 | `/api/kiosk/ping` | 30초 폴링 |

### 인터랙션 (User Actions)
1. 출석 방식 체크 → isDirty = true
2. 저장 버튼 → `04-저장중` 상태
3. 연결 상태 배지 클릭 → 핑 재시도

- 얼굴인식 체크 해제 시 섹션 3 비활성화
