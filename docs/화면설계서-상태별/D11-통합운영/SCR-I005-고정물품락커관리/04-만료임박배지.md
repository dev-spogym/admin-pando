## 📝 디스크립션

### 사용 시점 (Trigger)
- `expires_at BETWEEN NOW() AND NOW() + INTERVAL '7 days'` 존재
- 페이지 진입 시 클라이언트 계산

### 필수 데이터
- 만료임박 목록: D-7 이내 배정
- D-day 값: `differenceInDays(expires_at, today)`

### 인터랙션 (User Actions)
1. "해당 목록 보기" → 필터 'expiring' 자동 선택
2. "연장" 버튼 → 연장 날짜 선택 모달
3. 연장 완료: 해당 행 amber → 정상 스타일 전환

### 비즈니스 룰
- D-3 이내: 배지 색상 amber → red로 강조 상승
- 연장 단위: 1개월 / 3개월 / 6개월 선택
- 만료 경과 시 `05-회수대상배지`로 자동 전환 (다음 진입 시)

### 에지 케이스
- 만료임박 + 회수대상 동시 존재: 두 배지 모두 표시
- D-0 (당일): "오늘 만료" 별도 강조

### 연결 화면
- 이전: `02-정상`
- 다음: 연장 → 배지 제거 / 만료 경과 → `05-회수대상배지`
