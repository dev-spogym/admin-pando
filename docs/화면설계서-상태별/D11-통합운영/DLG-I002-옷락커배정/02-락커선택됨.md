파일: src/components/locker/LockerAssignModal.tsx (선택 상태 UI)

선택된 락커 강조:
- 선택된 셀: border-blue-500 bg-blue-50 text-blue-700 ring-2 ring-blue-500
- 나머지 셀: 정상 표시 (다른 락커 선택 변경 가능)

[선택 요약 배너 — 하단 고정]
<div className="flex items-center gap-3 p-3 bg-blue-50 border border-blue-200 rounded-lg mt-4">

  </div>

## 📝 디스크립션

### 사용 시점 (Trigger)
- `selectedLocker !== null`

### 인터랙션 (User Actions)
1. 다른 락커 클릭 → selectedLocker 교체
2. "선택 취소" → selectedLocker = null
3. 출석 없음 경고 → "계속 배정" 클릭 시 proceedWithoutAttendance=true
4. "배정 확인" → `04-저장중`

### 비즈니스 룰
- 출석 없는 회원 배정: 경고 표시 후 선택 허용 (M2 플로우)
- 선택 변경: 기존 선택 즉시 해제 후 새 락커 선택
- "배정 확인" 비활성: 출석 없음 + 미동의 상태

### 연결 화면
- 이전: `01-열림-락커목록`
- 다음: `04-저장중` / `01-열림-락커목록` (선택 취소)
