## 📝 디스크립션

### 사용 시점 (Trigger)
- SCR-I001 빈 락커 셀 클릭 (회원 pre-fill)
- SCR-I004 빈 락커 셀 클릭
- SCR-I007 "락커 배정" 버튼 클릭

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 빈 락커 목록 | `lockers` | `branch_id=?`, `type='clothing'`, `status='available'` |
| 회원 정보 | Props (memberId, memberName) | 부모에서 전달 |

### 인터랙션 (User Actions)
1. 존 탭 전환 → 필터링
2. 락커 셀 클릭 → `02-락커선택됨`
3. "배정 확인" → 검증 → `03-검증에러` 또는 `04-저장중`
4. 취소/X/ESC → 모달 닫힘

### 비즈니스 룰
- 빈 락커만 표시 (사용중/점검중 숨김)
- 점검중 락커: 그리드에 회색 표시 + cursor-not-allowed
- 회원 정보 읽기 전용 (수정 불가)
