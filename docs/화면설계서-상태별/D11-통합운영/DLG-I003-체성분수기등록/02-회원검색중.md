```
화면: DLG-I003 체성분수기등록 — 상태: 회원 검색 드롭다운

DLG-I001과 동일한 패턴:

- SELECT id, name, phone, member_number FROM members WHERE branch_id=? AND ... LIMIT 10

{isSearching && <스피너 />}

선택 완료:

[선택된 회원 + 체성분 히스토리 힌트]
{selectedMember && (

회원 선택 후 마지막 측정 조회:
const { data: lastMeasurement } = await supabase

  .eq('member_id', m.id)
  .order('measured_at', { ascending: false })
  .limit(1)
  .single()
```
