## 📝 디스크립션

### 사용 시점 (Trigger)
- SCR-I006 "수기 등록" 버튼 (회원 미pre-fill)
- SCR-I007 InBody 카드 "수기 등록" (memberId pre-fill)

### 필수 데이터
- 현재 날짜: 측정일 기본값
- 회원 정보: prefill 또는 검색

### 인터랙션 (User Actions)
1. 회원 검색 (prefill 없으면) → `02-회원검색중`
2. 날짜 변경 → 미래 날짜 불가 (max 오늘)
3. 수치 입력 → 범위 검증 (step 0.1)
4. 저장 → 검증 → `04-저장중`

### 비즈니스 룰
- 체중 필수, 나머지 선택
- 측정일: 미래 날짜 불가

- 중복 날짜 체크는 검증 시 처리
