## 📝 디스크립션

### 사용 시점 (Trigger)
- 로딩 완료 + 락커 데이터 정상
- 탭 전환 (전체/사용중/빈/점검중)

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 락커 목록 | `lockers` JOIN `clothing` JOIN `members` | `branch_id=?`, `type='clothing'` |
| IoT 상태 | `iot_devices` | `type='locker_controller'` |

### 인터랙션 (User Actions)
1. 빈 락커 셀 클릭 → DLG-I002 (배정 모달)
2. 사용중 셀 클릭 → 배정 상세 모달 (반납/연장 버튼)
3. 탭 전환 → 클라이언트 사이드 필터 (재조회 없음)
4. 엑셀 다운로드 → GET /api/locker/export

### 비즈니스 룰
- 점검중 락커 클릭 불가 (cursor-not-allowed)
- 만료 7일 이내: amber 강조 + "D-N" 배지
