## 📝 디스크립션

### 사용 시점 (Trigger)
- `devices.every(d => d.status !== 'online')` 감지
- 정전, 네트워크 전체 장애 등

### 필수 데이터
- 전체 장비 목록 + 각 오류 상세
- 영향 받는 출석 채널 정보

### 인터랙션 (User Actions)
1. "전체 재시작" → POST /api/iot/restart-all
2. 개별 카드 "재시작" 버튼
3. SCR-I001로 이동 → 수동 출석 등록 안내

### 비즈니스 룰
- 전체 오프라인 시 manager 이상 역할에 즉시 알림 발송
- 앱/키오스크 QR 출석은 서버 처리로 영향 없음 명시
- 복구 감지: toast.success('IoT 장비 복구') + 배너 제거

### 에지 케이스
- 전체 재시작 실패: 관리자 직접 확인 CTA
- 복구 중 일부만 복구: `05-장비오류배지` 상태로 전환
