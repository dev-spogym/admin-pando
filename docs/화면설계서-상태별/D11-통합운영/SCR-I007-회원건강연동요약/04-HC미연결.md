## 📝 디스크립션

### 사용 시점 (Trigger)
- `healthData === null` 또는 `healthData.connected === false`
- HC API 응답에서 연동 없음 확인

### 필수 데이터
- HC 연동 상태: `health_data.connected` (가상 테이블)
- 나머지 카드 데이터: 정상 로드

### 인터랙션 (User Actions)
1. "연결 안내 메시지 발송" → POST 알림 API
2. 나머지 카드 정상 인터랙션

### 비즈니스 룰
- HC 미연결은 에러가 아닌 정보 상태 (info 배지)
- 알림 발송 권한: `member:notify` (fc 이상)
- HC 연동 후 페이지 새로고침 시 `02-정상`으로 전환

### 에지 케이스
- HC API 타임아웃: 미연결과 동일 처리 (timeout = disconnected)
- 알림 발송 실패: toast.error('메시지 발송 실패')

### 연결 화면
- 이전: `02-정상` (HC 미연결 감지)
- 다음: 회원 HC 연동 후 새로고침 → `02-정상`
