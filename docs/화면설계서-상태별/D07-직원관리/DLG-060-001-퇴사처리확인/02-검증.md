## 🧩 바이브코딩 프롬프트

```
검증 핸들러:
  const handleValidate = () => {
    if (!confirmText) {
      setConfirmError('확인 텍스트를 입력하세요.')
      return false
    }
    if (confirmText !== '퇴사처리') {
      setConfirmError('"퇴사처리"를 정확히 입력하세요.')
      return false
    }
    setConfirmError(null)
    return true
  }

상태별 input 스타일:
  // 빈 값 blur
  className="border-red-500 bg-red-50 focus:ring-red-400"
  에러 메시지: "확인 텍스트를 입력하세요."

  // 불일치
  className="border-red-500 bg-red-50 focus:ring-red-400"
  에러 메시지: '"퇴사처리"를 정확히 입력하세요.'

  // 일치 (실시간 onChange 체크)
  className="border-green-500 bg-green-50 focus:ring-green-400"
  "퇴사 처리 확정" 버튼 활성화

실시간 검증 (onChange):
  onChange={e => {
    setConfirmText(e.target.value)
    if (e.target.value === '퇴사처리') setConfirmError(null)
  }}

"퇴사 처리 확정" 버튼:
  disabled={confirmText !== '퇴사처리' || isProcessing}
  → confirmText === '퇴사처리'일 때만 활성 (disabled 해제)
```
