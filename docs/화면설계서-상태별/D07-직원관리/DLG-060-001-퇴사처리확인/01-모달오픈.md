  <Modal

## 📝 디스크립션

### 사용 시점 (Trigger)
- SCR-060에서 selectedRows.size > 0 → "퇴사 처리" 버튼 클릭
- `hasPermission('staff:resign')` true인 경우만 모달 표시

### 필수 표시 항목
- 대상 직원 목록 (이름, 부서, 직책)
- 확인 텍스트 입력 필드 ("퇴사처리" 정확히 입력)
- 되돌릴 수 없다는 경고 메시지

### 인터랙션 (User Actions)
1. 확인 텍스트 입력 → `02-검증` 상태
2. "취소" 클릭 → 모달 닫힘, SCR-060 선택 상태 유지
3. ESC 키 → 모달 닫힘
4. 모달 외부 클릭 → 닫힘 (선택적)

### 비즈니스 룰
- manager 역할: 퇴사 처리 버튼 자체가 hidden → 이 모달 도달 불가
- 이미 RESIGNED 직원이 선택에 포함된 경우: 모달 열기 전 경고 toast
- 대상이 1명 이상이어야 모달 오픈 가능

### 접근성 (A11y)
- `role="dialog"` + `aria-modal="true"` + `aria-labelledby="modal-title"`
- 모달 오픈 시 첫 번째 인터랙티브 요소(입력 필드)로 포커스 이동
- ESC 키 닫기

### 연결 화면
- 이전: SCR-060 `08-선택있음`
- 다음: `02-검증` / 닫힘 → SCR-060
