## 📝 디스크립션

### 사용 시점 (Trigger)
- SCR-060 selectedRows.size > 0 → "상태 변경" 클릭
- primary / owner / manager 접근 가능

### 상태 옵션 매트릭스
| 변경할 상태 | 사유 필요 | 접근 역할 |
|------------|-----------|-----------|
| ACTIVE | X | primary / owner / manager |
| ON_LEAVE | O | primary / owner / manager |
| TRANSFERRED | O | primary / owner |
| LOCKED | X | primary / owner |

### 인터랙션 (User Actions)
1. 상태 선택 → 현재 상태와 동일 시 경고 메시지
2. ON_LEAVE / TRANSFERRED 선택 → 사유 입력 필드 표시
3. "취소" → 모달 닫힘
4. "변경 확인" → `02-검증` → `03-결과분기`

### 비즈니스 룰
- 복수 선택 시 현재 상태 표시 생략 (다를 수 있음)
- RESIGNED 상태로 변경 불가 (퇴사 처리는 DLG-060-001 전용)
- manager는 TRANSFERRED 변경 불가

### 접근성 (A11y)
- `role="dialog"` + `aria-modal="true"`
- 라디오 그룹 `role="radiogroup"` + `aria-label="변경할 상태 선택"`

### 연결 화면
- 이전: SCR-060 `08-선택있음`
- 다음: `02-검증` / 닫힘
