## 2. 검증 규칙

| 조건 | 메시지 | UI |
|---|---|---|
| 상태 미선택 | `"변경할 상태를 선택하세요"` | 라디오 그룹 `aria-invalid` + 헬퍼 |
| 단일 대상 + 동일 상태 | `"현재와 동일한 상태입니다"` | 버튼 disabled + 인라인 경고 |

| 사유 500자 초과 | `"사유는 500자 이하로 입력하세요"` | 카운터 빨간색 |

## 3. Zod 스키마

```ts
const schema = z.discriminatedUnion('status', [
  z.object({ status: z.literal('ACTIVE') }),
  z.object({
    status:  z.literal('ON_LEAVE'),
    startAt: z.string().min(1, '휴직 시작일을 입력하세요'),
    endAt:   z.string().min(1, '휴직 종료일을 입력하세요'),
    reason:  z.string().min(5, '사유는 5자 이상 입력하세요').max(500),
  })
  .refine(v => v.startAt <= v.endAt, { message: '종료일은 시작일 이후여야 합니다', path: ['endAt'] })
  .refine(v => v.startAt >= todayIso(), { message: '시작일은 오늘 이후여야 합니다', path: ['startAt'] }),
  z.object({
    status: z.literal('TRANSFERRED'),
    reason: z.string().min(5).max(500),
  }),
  z.object({
    status: z.literal('LOCKED'),
    reason: z.string().min(5).max(500),
  }),
]);
```

| 요소 | 실패 | 성공 |
|---|---|---|
| input border | `border-red-300` + `aria-invalid="true"` | `border-gray-300` |

| 버튼 disabled | `true` | `false` |

## 5. 동일 상태 가드

```ts
const isDuplicate = targets.length === 1 && targets[0].staffStatus === selectedStatus;
const isValid = form.formState.isValid && !isDuplicate;

{isDuplicate && (
  <p role="alert" className="text-xs text-amber-600">
    현재와 동일한 상태입니다. 다른 상태를 선택하세요.
  </p>

```

## 6. 인터랙션 델타

- 라디오 변경 시 다른 필드(사유·기간) 에러 초기화 + 조건부 렌더 전환
- Textarea onChange 중 실시간 카운터 갱신 (`{reason.length}/500`)
- Enter(submit 버튼 외)는 기본 제출 방지 (`onKeyDown preventDefault`)
- 제출 버튼 클릭 직후 Zod 실패 시 첫 에러 필드로 포커스 이동(`setFocus(firstErrorField)`)

---

## 7. 역할별 차이

- manager 가 submit 한 상태가 `TRANSFERRED`/`LOCKED` 면 클라이언트 단에서 차단 (옵션 미노출이지만 DOM 조작 방어): Zod refine으로 거부 + `E403001` 메시지.

---

## 8. 에지 케이스

- **DateRange 모두 동일일**: 1일 휴직 허용 (start===end OK)
- **타임존 경계**: 클라이언트 자정 이후 제출 → 서버가 `today` 재계산 (UTC+9 기준)
- **브라우저 자동완성**: date input 에 이전 값 자동 주입 시 refine 재실행

## 9. 바이브코딩 프롬프트 (델타)

```
상태: dlg-statuschange-validating
변경점:
- zodResolver 연결 + mode: 'onBlur'
- isDuplicate 계산식 추가:
  const isDuplicate = targets.length === 1 && targets[0].staffStatus === form.watch('status');
- submit 가드:
  const onSubmit = form.handleSubmit(
    (v) => { if (isDuplicate) return; mutation.mutate(v); },
    (errors) => {
      const first = Object.keys(errors)[0];
      form.setFocus(first as any);
    }

- Textarea 카운터:

    {reason.length}/500
  </p>
- manager 방어:
  if (user.role === 'manager' && ['TRANSFERRED','LOCKED'].includes(v.status)) {
    toast.error(ERROR_MESSAGE.E403001);

  }
```
