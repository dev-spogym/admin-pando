## 📝 디스크립션

### 사용 시점 (Trigger)
- 역할 Select에서 값 선택 시 (`watchRole` 값 존재)
- 수정 모드에서 기존 역할 preloaded → 진입 즉시 표시

### 인터랙션 (User Actions)
1. 역할 Select 변경 → 미리보기 카드 내용 즉시 업데이트
2. 역할 선택 해제(초기화): 미리보기 숨김

### 비즈니스 룰
- 미리보기는 정보 제공 목적 (실제 권한 변경은 저장 후 적용)
- 수정 모드에서 역할 변경 후 저장하지 않으면 권한 변경 없음

### 에지 케이스
- manager가 OWNER 역할 부여 시도: 저장 시 서버에서 403 반환 (manager는 OWNER 생성 불가)
- 자기 자신의 역할 변경: 저장 시 409 또는 별도 경고

### 접근성 (A11y)
- 미리보기 카드 `aria-live="polite"` (역할 변경 시 내용 변경 알림)
- 권한 목록 `<ul>` + `<li>` 시맨틱 마크업

### 연결 화면
- 이전/다음: `01-등록모드` 또는 `03-수정모드` (인라인 상태)
