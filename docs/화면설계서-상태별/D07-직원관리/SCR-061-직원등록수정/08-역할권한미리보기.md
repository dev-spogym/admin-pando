## 고유 인터랙션
- role 변경 즉시 Preview 재렌더 (react-hook-form `watch('role')`).
- `aria-live="polite"`로 스크린리더에 새 역할 공지.
- Preview 박스 클릭: 상세 권한 매트릭스 모달 오픈(옵션, 추후 확장).
- 키보드 포커스: 박스 자체는 접근 불필요 (정보 전달만).

## 비즈니스 룰 델타
1. **단순 UI 가이드**: 실제 권한은 서버(RLS)에서 결정. Preview는 교육용.
2. **다국어**: `ROLE_PERMISSIONS`를 i18n 번들로 추후 분리.
3. **역할 없음 / 미정의 role**: Preview 박스 숨김.
4. **애니메이션**: 역할 변경 시 `animate-in fade-in duration-150`.

## 에지 케이스
- 역할이 미지원 문자열(예: DB 한글만 온 경우): `ROLE_PERMISSIONS[key]` undefined → Preview 숨김.
- 역할 강제 업데이트 (수정 모드 로드): 기존 role 그대로 Preview.
- 잦은 변경으로 인한 깜박임: `duration-150` 정도면 허용 수준.

## 바이브코딩 프롬프트 (델타)
```
const watchedRole = watch('role');
const info = ROLE_PERMISSIONS[watchedRole];

{info && (
  <div aria-live="polite"
       className="mt-2 bg-primary-light border border-primary/20 rounded-input p-4
                  animate-in fade-in duration-150">
    <h3 className="text-sm font-semibold text-primary flex items-center gap-1.5">
      <Shield size={14}/> {info.label} 권한
    </h3>
    <p className="text-xs text-content-secondary mt-1">{info.desc}</p>
    <div className="flex flex-wrap gap-1.5 mt-2">
      {info.perms.map((p) => (
        <span key={p}
              className="inline-flex px-2 py-0.5 rounded-full bg-white border border-primary/20 text-xs text-primary">
          {p}
        </span>
      ))}
    </div>
  </div>
)}
```
