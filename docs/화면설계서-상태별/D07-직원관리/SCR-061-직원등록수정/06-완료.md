# SCR-061 직원 등록/수정 — 06 완료 (저장 성공) (델타)

> 이 문서는 `00-기본화면.md`의 **델타**입니다.

## 메타
| 항목 | 값 |
|---|---|
| 상태 코드 | `staff-form-success` |
| 상태 ID | 06 |
| 다이어그램 노드 | F6 > SUCCESS / F9 TOAST_SUCCESS |
| 이전 상태 | `04-저장중` |
| 다음 상태 | SCR-060 `/staff` 목록 자동 이동 |

## 진입 조건
- 저장 API 200 응답
- 감사 로그 INSERT 완료(실패해도 UX는 성공 유지)

## 비주얼 델타
- 짧은 체크 마크 확인 애니메이션(선택) 후 toast 표시 + 즉시 목록 이동.
- 폼 자체는 1~2 프레임 노출 후 unmount.

| 변경 요소 | From(04) → To(06) |
|---|---|
| 저장 버튼 | "저장 중..." | "완료!" + CheckCircle (옵션, 300ms 표시) |
| toast | — | `toast.success("직원이 성공적으로 등록되었습니다.")` or `"직원 정보가 수정되었습니다."` |
| 화면 | 폼 | 0~300ms 유지 후 `moveToPage(974)` → SCR-060 |

## 역할별 차이
- 공통. 토스트 메시지 동일.

## 고유 인터랙션
- 자동 리다이렉트: `moveToPage(974)` (직원 목록).
- 목록에서는 새로 등록된 직원이 상단(이름 정렬에 따라) 또는 하이라이트 표시(선택 3초).

## 비즈니스 룰 델타
1. **성공 후 네비게이션**: 항상 `/staff`로 이동. 체류 옵션 제공하지 않음(흐름 단순화).
2. **감사 로그**:
   - 등록: `AUDIT.STAFF_CREATE` + afterValue `{name, role, branchId}`.
   - 수정: `AUDIT.STAFF_UPDATE` + beforeValue/afterValue diff (선택).
   - 추가: role 변경 시 `STAFF_ROLE_CHANGE`, salary 변경 시 `STAFF_SALARY_CHANGE`.
3. **목록 갱신**: SCR-060의 React Query `invalidateQueries(['staff-list'])` 실행(전역 이벤트 또는 mutation onSuccess).
4. **하이라이트**: 새 id 전달 → 목록에서 3초 `bg-primary/10 animate-pulse` 후 자연스럽게 소멸.

## 에지 케이스
- **감사 로그 실패**: 메인 성공 유지. 콘솔 warning + Sentry.
- **리다이렉트 실패**(예: `moveToPage` 예외): `router.push('/staff')` fallback.
- **네트워크 복구 직후 재저장**: 정상 처리 + 중복 레코드 방지는 전화번호 UNIQUE로.

## 바이브코딩 프롬프트 (델타)
```
200 응답 후:
toast.success(editId ? '직원 정보가 수정되었습니다.' : '직원이 성공적으로 등록되었습니다.');

// 감사 로그 (비동기, 실패 무시)
try {
  if (editId) await createAuditLog({ action:'STAFF_UPDATE', targetType:'staff', targetId:editId });
  else await createAuditLog({ action:'STAFF_CREATE', targetType:'staff', targetId:data.id,
                              afterValue:{ name:payload.name, role:payload.role } });
} catch { /* swallow */ }

// 목록 캐시 무효화
queryClient.invalidateQueries({ queryKey:['staff-list'] });

// 이동 + 하이라이트용 쿼리 전달
moveToPage(974, { highlight: editId ?? data?.id });
// SCR-060: ?highlight=123 있을 때 해당 row 3초 애니메이션
```
