## 📝 디스크립션

### 사용 시점 (Trigger)
- SCR-060에서 "직원 등록" 버튼 클릭
- URL: `/staff/new` (editId 없음)

### 필수 데이터
| 필드 | 필수 여부 | 검증 |
|------|-----------|------|
| 이름 | 필수 | 2자 이상 |
| 연락처 | 필수 | 010-xxxx-xxxx 패턴 |
| 부서 | 필수 | 1자 이상 |
| 직책 | 필수 | 1자 이상 |
| 역할 | 필수 | enum 선택 |
| 입사일 | 필수 | 오늘 이전 날짜 |

### 인터랙션 (User Actions)
1. 역할 Select 변경 → 역할 권한 미리보기 카드 업데이트
2. 취소 버튼: `isDirty=true` → DLG-061-001, `isDirty=false` → 즉시 목록 이동
3. 저장하기 → 검증 통과 시 `04-저장중` 전환

### 비즈니스 룰
- 동일 연락처 직원 중복 체크 (저장 시 서버 409 응답)
- 급여 입력은 선택사항, 미입력 시 0으로 저장
- owner 역할 직원: center_id 연결 필수

### 에지 케이스
- 검증 실패 후 저장 시도: `05-인라인에러` 전환
- 브라우저 새로고침: `isDirty=false` 초기화 (미저장 데이터 사라짐)

### 접근성 (A11y)
- 필수 필드 `aria-required="true"`
- 에러 메시지 `aria-describedby` 연결
