## 고유 인터랙션
- "다시 시도" → `handleSubmit(onSubmit)` 재실행 → `04-저장중` 재진입.
- 배너 닫기(X): 배너만 닫고 폼 유지.
- 필드 재편집 후 재제출.

## 비즈니스 룰 델타
1. **에러 분류**:
   - `E400200/E400103/E400104`: 필드 인라인 에러(`setError`)로 변환 → `05-인라인에러`.
   - `E409100`(전화번호 중복) → 인라인.
   - `E409200`(이미 퇴사) → dialog + 안내.
   - `E422200`(담당 회원 존재) → dialog + SCR-062 권유.
   - `E500001/E503001` → 에러 배너 + 재시도.
   - `NETWORK` → 오프라인 배너 + 자동 재시도 안내.
   - `E401*` → `/login?redirect=현재URL`.
   - `E403001` → `/forbidden`.
2. **폼 유지**: 사용자 입력 전부 보존. 재시도 시 동일 payload.
3. **재시도 쿨다운**: 네트워크 에러 3회 연속 실패 시 30s 쿨다운(선택).

## 에지 케이스
- **부분 성공**: 직원 INSERT는 성공했으나 감사 로그 실패 → UX 성공 + 로그만 warning.
- **타임아웃 후 늦은 성공 응답**: 이미 `07`로 전이된 후 응답 도착 시 무시 (AbortController로 취소됨).
- **중복 ID 회피**: 서버가 UNIQUE 위반 → E409 → 인라인.
- **감사 로그 API 500**: 메인 성공 유지.

## 바이브코딩 프롬프트 (델타)
```
catch (e: any) {
  const code = e?.code ?? 'UNKNOWN';
  const msg = e?.message ?? '알 수 없는 오류';

  // 필드 매핑 분기
  if (code === 'E409100' || code === 'E400103')
    setError('contact', { type:'server', message:'이미 등록된 전화번호입니다.' });
  else if (code === 'E400104')
    setError('email', { type:'server', message:'올바른 이메일 형식이 아닙니다.' });
  else if (code === 'E401001' || code === 'E401002') {
    router.push(`/login?redirect=${encodeURIComponent(pathname)}`); return;
  } else if (code === 'E403001') {
    router.push('/forbidden'); return;
  } else if (code === 'E409200') {
    openDialog({ title:'이미 퇴사 처리된 직원', description:'이미 퇴사된 직원은 수정할 수 없습니다.', onOk:()=>moveToPage(974) });
    return;
  } else if (code === 'E422200') {
    openDialog({ title:'담당 회원 존재', description:'담당 회원이 있어 처리할 수 없습니다. 퇴사 위자드를 이용하세요.',
                 onOk:()=>moveToPage('/staff/resignation', {staffId: editId}) });
    return;
  } else {
    // 시스템 에러 → 배너
    setApiError({ code, message: msg });
    toast.error((editId ? '수정 실패: ' : '저장 실패: ') + msg);
  }
  setIsSaving(false);
}

{apiError && (
  <ErrorBanner tone="error" role="alert" aria-live="assertive"
    className="mb-4 flex items-start gap-2 rounded-lg border border-red-200 bg-red-50 p-3 text-sm text-red-700">
    <AlertCircle size={16}/>
    <div className="flex-1">
      <p className="font-medium">저장에 실패했습니다.</p>
      <p className="text-xs mt-0.5">에러코드: {apiError.code} · {apiError.message}</p>
    </div>
    <Button variant="outline" size="sm" onClick={handleSubmit(onSubmit)}>다시 시도</Button>
    <Button variant="ghost" size="sm" onClick={()=>setApiError(null)}>닫기</Button>
  </ErrorBanner>
)}
```
