## 🧩 바이브코딩 프롬프트

```
확정 가능 여부 검증 (모달 오픈 시 + 확정 버튼 클릭 시):

  const validateConfirm = (): boolean => {
    // 1. 급여 데이터 없음
    if (payrollList.length === 0) {
      toast.error('이 월의 급여 데이터가 없습니다.')
      return false
    }

    // 2. 전원 이미 확정
    if (pendingList.length === 0) {
      toast.warning('모든 직원이 이미 확정되었습니다.')
      setConfirmModalOpen(false)
      return false
    }

    return true
  }

"급여 확정" 버튼 disabled 조건:
  disabled={
    payrollList.length === 0 ||
    pendingList.length === 0 ||
    isConfirming
  }

버튼 비활성 시 경고 표시 (모달 내부):
  // 전원 확정:
  <p className="text-sm text-yellow-700">모든 직원이 이미 확정되었습니다.</p>

  // 데이터 없음:
  <p className="text-sm text-red-700">이 월의 급여 데이터가 없습니다. 확정할 수 없습니다.</p>
```

### 검증 규칙
| 조건 | 메시지 | 처리 |
