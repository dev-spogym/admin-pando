선택 핸들러:
  const toggleRow = (id: string) => {

      const next = new Set(prev)
      next.has(id) ? next.delete(id) : next.add(id)
      return next
    })
  }
  const toggleAll = () => {
    selectedRows.size === staffList.length
      ? setSelectedRows(new Set())
      : setSelectedRows(new Set(staffList.map(s => s.id)))
  }

## 📝 디스크립션

### 사용 시점 (Trigger)
- 데이터 로드 직후 초기 상태 (`selectedRows.size === 0`)
- 선택 해제 후 (체크박스 다시 해제)
- 전체 해제 (헤더 체크박스 클릭)

### 인터랙션 (User Actions)
1. 개별 체크박스 클릭 → `toggleRow(id)` → `08-선택있음`
2. 헤더 전체 선택 체크박스 → `toggleAll()` → `08-선택있음`
3. 행 이름 클릭 (체크박스와 무관) → 직원 상세/수정

### 비즈니스 룰
- 선택 없음 상태에서 3개 액션 버튼 모두 disabled
- disabled 버튼 클릭 시 아무 동작 없음 (pointer-events-none 또는 onClick guard)
- 탭 전환/검색 시 selectedRows 초기화

### 에지 케이스
- 탭 변경: `setSelectedRows(new Set())` 자동 호출
- 검색어 변경: 선택 초기화
- 전체 선택 후 필터로 일부 숨겨진 경우: hidden 행의 selectedRows 유지 여부 → 초기화 권장

### 접근성 (A11y)
- disabled 버튼 `aria-disabled="true"` (실제 disabled 속성도 추가)
- 툴팁 `role="tooltip"` id 연결
