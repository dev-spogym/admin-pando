```
Next.js 14 App Router + TypeScript + Tailwind CSS 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-063 직원 근태 관리 — 상태: 월별 요약 뷰

파일: src/app/staff/attendance/page.tsx

조건: viewMode='monthly'

월별 데이터 fetch:
  const startOfMonth = format(selectedMonth, 'yyyy-MM-01')
  const endOfMonth = format(endOfMonthFn(selectedMonth), 'yyyy-MM-dd')
  const { data: monthlyData } = await supabase
    .from('staff_attendance')
    .select('staff_id, date, clock_in, clock_out, status')
    .gte('date', startOfMonth)
    .lte('date', endOfMonth)

상단:
- TabNav: [일별 | 월별 (active)]
- MonthPicker: 연월 선택 (yyyy년 MM월)

집계 테이블 (직원별 월간 요약):
  컬럼:
  - 이름
  - 부서
  - 출근일수 (clock_in 있는 날 count)
  - 결근일수 (영업일 - 출근일수)
  - 지각횟수 (clock_in > 기준시간, 예: 09:10)
  - 총 근무시간 (sum of clock_out - clock_in, "XXX시간 YY분")
  - 초과근무 (총 근무시간 - 기준 근무시간)
  - 상세보기 (버튼 → 해당 직원 일별 뷰로 이동)

집계 계산:
  const summary = staffList.map(staff => {
    const staffRecords = monthlyData.filter(r => r.staff_id === staff.id)
    const workedDays = staffRecords.filter(r => r.clock_in).length
    const totalMinutes = staffRecords.reduce((sum, r) => {
      if (r.clock_in && r.clock_out) {
        return sum + differenceInMinutes(new Date(r.clock_out), new Date(r.clock_in))
      }
      return sum
    }, 0)
    return { ...staff, workedDays, totalMinutes, ... }
  })

엑셀 다운로드 버튼:
  <Button variant="outline" onClick={downloadExcel}>
    엑셀 다운로드
  </Button>
  // 월별 요약 데이터를 xlsx 또는 CSV로 export

사용 컴포넌트:
- AppLayout, PageHeader, DataTable, TabNav, Button
- supabase, formatNumber
- date-fns (format, endOfMonth, differenceInMinutes)
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- `02-일별뷰`에서 "월별" 탭 클릭
- `viewMode === 'monthly'`

### 필수 데이터
| 테이블 | 조건 |
|--------|------|
| `staff_attendance` | `date BETWEEN startOfMonth AND endOfMonth` |
| `staff` | ACTIVE 직원 전체 |

### 인터랙션 (User Actions)
1. MonthPicker 변경 → 해당 월 재fetch
2. "일별" 탭 클릭 → `02-일별뷰` 복귀
3. "상세보기" → 해당 직원 일별 뷰 (`selectedStaffId` 필터 적용)
4. "엑셀 다운로드" → CSV/xlsx export

### 비즈니스 룰
- 영업일 계산: 주말 제외 (공휴일 미포함, 향후 확장)
- 초과근무 기준: 1일 8시간 (480분)
- 결근: 영업일 중 clock_in 없는 날

### 에지 케이스
- 당월 데이터 없는 직원: 모든 값 0 표시
- 직원 퇴사 후 해당 월 데이터: 표시하되 "(퇴사)" 표기

### 접근성 (A11y)
- DataTable `aria-label="직원별 월간 근태 요약"`
- MonthPicker `aria-label="조회 월 선택"`

### 연결 화면
- 이전: `02-일별뷰`
- 다음: `02-일별뷰` (탭 전환)
