## 고유 인터랙션
- 모든 버튼 disabled. 키보드 이벤트 preventDefault.
- 창 닫기: `beforeunload` 경고.
- 진행 중 스피너 + 상태 텍스트 변화.

## 비즈니스 룰 델타
1. **순차 호출**:
   - (FC) `reassignMembers` → `scheduleResignation` → `createAuditLog`.
   - (비FC) `scheduleResignation` → `createAuditLog`.
2. **재배정 실패 시**: 퇴사 등록 중단. 롤백은 수행하지 않음(재배정 API는 이미 부분 적용 가능성 존재 — 추후 서버 트랜잭션 래핑 권장).
3. **퇴사 등록 실패**: 재배정은 유지된 상태. 사용자에게 안내 후 재시도 가능.
4. **타임아웃**: 각 호출 30초. 이후 AbortError.
5. **다중 mutation 원자성**: 현재는 클라 측 순차. 추후 Supabase Edge Function으로 이관 권장.

## 에지 케이스
- **부분 성공**: reassign 성공 + resign 실패 → toast + `08-에러` + "이미 재배정된 회원은 유지됨" 안내.
- **네트워크 단절**: 첫 번째 호출에서 실패 → `08` + retry.
- **감사 로그 실패**: 메인 성공 → UX 성공.
- **세션 만료 중 제출**: 401 → `/login?redirect=/staff/resignation?staffId={id}` (진행 데이터 손실 — 추후 draft 저장 기능).

## 바이브코딩 프롬프트 (델타)
```
isSubmitting === true:
<div className="pointer-events-none opacity-60">
  {/* ... 기존 Step 4 요약 ... */}
</div>

<div className="flex justify-between">
  <Button variant="secondary" disabled><ArrowLeft size={16}/> 이전</Button>
  <Button variant="danger" disabled aria-busy="true">
    <Loader2 className="animate-spin" size={16}/> 처리 중...
  </Button>
</div>

// 진행 상태 텍스트 (선택)
const [progressMsg, setProgressMsg] = useState('');
async function handleSubmit() {
  setIsSubmitting(true);
  try {
    if (isFC) {
      setProgressMsg('담당 회원 재배정 중...');
      const r = await reassignMembers({ assignments });
      if (!r.ok) throw Object.assign(new Error(r.message), { step:'reassign' });
    }
    setProgressMsg('퇴사 예정 등록 중...');
    const r2 = await scheduleResignation(selectedStaffId!, { resignScheduledAt:resignDate, resignReason });
    if (!r2.ok) throw Object.assign(new Error(r2.message), { step:'resign' });
    await createAuditLog({ action:'STAFF_RESIGN_SCHEDULE', targetType:'staff', targetId:selectedStaffId!,
                          afterValue:{ resignScheduledAt:resignDate } });
    toast.success('퇴사 예정이 등록되었습니다.');
    moveToPage(974);  // /staff
  } catch (e:any) {
    if (e.step === 'reassign') toast.error(`담당 회원 재배정 실패: ${e.message}`);
    else toast.error(`퇴사 예정 등록 실패: ${e.message ?? '알 수 없는 오류'}`);
    setGlobalError(e);  // 08-에러 표시
  } finally {
    setIsSubmitting(false);
    setProgressMsg('');
  }
}
```
