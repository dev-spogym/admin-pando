## 1. 레슨북 (LessonBook)

### 장점

#### 상품관리 — 한 화면 자유도
- 한 화면에서 모든 상품 속성을 제어 가능 (뎁스 없음)
- 요일별/시간별/일자별 이용 제한 설정 가능
- 주중권/주말권 등 장소·시간에 따른 가격 차별화
- 상품 클릭 시 바로 옆에 상세 표시 (마스터-디테일)
- "상품 관리가 너무 좋다. 이걸 똑같이 피트니스에도 만들어 달라" — 이지훈 대표

#### 수업/레슨 관리 — 진위 여부 확인
- 수업 완료 시 **회원 서명(사인)** 필수 → 수업 진행 증빙
- 사인 없으면 수업 인정 안됨 (정책으로 강제 가능)
- 하루 지나면 자동으로 수업 완료 처리 (노쇼도 진행으로 전환)
- 프로(강사)와 다이렉트 소통 가능
- "이 사람 정말 레슨 했어? 믿을 수 있는 자료가 레슨북밖에 없어" — 고척점

#### 스케줄 관리
- 레슨 예약 + 시설 예약 통합
- 모바일 앱에서 예약 → 대기 → 현황 확인 가능
- 현재 이용자 / 예약자 / 대기자 실시간 표시
- "바깥에서 모바일로 예약해서 대기에 걸려 있어. 현장에 와서 기다리지 않아도 된다" — 고척점

#### 노쇼/취소 정책
- 수업 N시간 전 취소 제한 설정 가능 (3시간 전 등)
- 노쇼 = 수업 진행으로 카운트 (횟수 차감)
- 취소와 결석이 별도 분리되어 관리

#### 키오스크 연동
- 타석 예약 → 키오스크 자동 제어
- 오픈/마감 시 키오스크·프로젝터 전원 자동 제어 (고척점)
- 시간 종료 시 자동 종료

#### 통계 — 엑셀 친화적
- 모든 데이터가 한 화면에서 쭉 내려가는 형태
- 엑셀로 긁어와서 가공하기 매우 편리
- 데일리/상품별/인원별/그룹별/금액별 시트 제공
- "모든 데이터 값이 그냥 쭉 내려가기 때문에 갖다 붙여서 가공하는 게 너무 좋아요" — 고척점

#### 기타
- 프로(강사) 입장에서 만든 앱 → 강사 채택률 높음
- 골프장 프로들이 이미 대부분 사용 중 → 자연스러운 도입
- 초기 도입비 300~600만원, 월 15만원 (문자 비용 포함)
- 1년 무료, 키오스크 제공

### 단점
- 골프에 특화 → 피트니스 기능 부족
- GX/그룹수업 관리 미흡
- 매출 통계가 과다할 정도로 많음 (정리 필요)
- 프로젝터 전원 제어는 고척점만 가능 (다른 지점은 수동)

## 2. 온핏 (OnFit)

### 장점

#### UX 편의성 — 매출 중심
- FitGenie CRM과 함께 성장하여 매출 관리에 최적화
- 입장 알림, 만료 임박 회원 표시 등 매출 촉진 기능 우수
- "매출이나 이런 부분에 대해서 좀 최적화돼 있는 부분이 그나마 온핏" — 대치점
- 심플한 UI → 신규 직원 적응 빠름

#### 회원 관리
- 메모 기능이 직관적이고 편리
- 즐겨찾기 회원 방문 시 **담당 FC 핸드폰에 알림** 전송
- 만료 임박 회원 자동 표시 (색상 구분)
- 전 지점 통합 회원 검색 가능
- "즐겨찾기 표시하면 회원님 방문 시 제 핸드폰에 문자. 이거 되게 중요한 기능" — 대치점 FC

#### 입장/퇴실 시스템
- NFC 밴드 기반 입장/퇴실
- 랜덤 락커 배정 → 사물함 점유 방지
- 입장 시 락커번호 영수증 출력 (대치점)
- 방문 이력 실시간 표시
- "입장하기를 이렇게 하니까 스트레스가 좀 줄었다" — 대치점

#### GX 예약
- 그룹수업 예약 시스템 (앱 연동)
- 횟수권 자동 차감
- "GX는 온핏이 낫지도 모르겠네" — 고척점

#### 안정성
- 네트워크 끊겨도 로컬에서 동작 (오프라인 모드)
- 대규모 회원 처리 안정적 (2천명 이상)
- "인터넷 끊겨도 하나가 죽어도 하나는 살 수 있는 정도로 만들어 놨더라" — 판교점

#### 락커 관리
- NFC 밴드로 자동 락커 배정 (랜덤)
- 사용 중/빈 락커 실시간 파악
- 홀수 먼저/윗칸 먼저 배정 설정 가능 (판교점)
- 개인 사물함, 골프 사물함 분리 관리

### 단점
- 상품 관리 시 뎁스가 깊음 (2단계 이상 들어가야 설정)
- "상품 요일/시간 설정하려면 버튼을 눌러야 팝업이 뜨고... 불편하다" — 고척점
- GX 예약 일괄 변경 불편 (하나씩 수동 수정)
- N시간 전 자동 예약 오픈 기능 없음
- 어드민 권한 설정이 본사 의존적 (지점에서 직접 못 풀어줌)
- 메인 센터 이동(지점 변경) 버그 존재
- 수업 출석 체크 자동화 없음 (수동)

## 3. 브로제이 (BroJ)

### 장점

#### 얼굴 인식 출입
- 빠르고 정확한 얼굴 인식 (10명 동시 인식 가능)
- 핸드폰 번호 입력 방식도 병행 가능
- "인식률이 꽤 좋네요. 상당히 빠르더라" — 대치점
- 입장/퇴실 자동 체크

#### 골프 타석 관리
- 대기열 시스템 (꼬리물기 예약)
- 실시간 남은 시간 표시
- 자동 전원 ON/OFF (오픈/마감 시간 기반)
- 타석 이동 기능
- "프로그램 자체를 이용 못하게 잠금 → 시간 끝나면 자동 종료" — 대치점

#### 무인화 특화
- 입장~퇴실 전과정 비대면
- 키오스크 + 얼굴인식 기반 무인 운영
- 문 설치 등 물리적 출입 통제 연동

#### 앱 연동
- 회원 전용 앱에서 예약/확인
- 1일권 구매 가능 (일부 매장)

### 단점
- 매출 관리 기능 약함
- "매출 관리는 온핏이 더 편하다" — 대치점
- 회원 메모 확인이 불편 → 온핏을 병행 사용하는 이유
- 대규모 회원(2천명+) 시 시스템 에러 발생
- "500~600명 기준으로 만들었는데 2천명이니까 뻥나는 거" — 대치점
- 설치 시 매장 규모 고려 부족
- 피트니스 전용이 아닌 범용 솔루션 → 세부 기능 약함
- 온보딩 자료/가이드 부족 (영상만 있음)

## 6. 현장의 목소리 (핵심 인용)

### 상품관리
> "한 화면에서 다 제어할 수 있잖아요. 온핏은 뎁스가 두 단계를 들어가. 버튼을 눌러야지만 팝업이 떠요. 그래서 불편하다고 느끼는 것 같고 기능은 있는 것 같거든요. 문제는 찾아야 되니" — 고척점

### 수업 신뢰성
> "내가 이 사람 정말 레슨 했어? 믿을 수 있는 자료가 없어. 근데 레슨북은 사인 안 하면 수업한 걸로 인정 안 해줘" — 고척점

### 통합 필요성
> "온핏과 브로제이와 레슨북을 쓰는 매장이 각기 다르면서 제어가 안 되니까 후가공하는 엑셀이 많아지는 거고, 점장들은 엑셀만 하고 앉아 있고" — 고척점

### FC 알림
> "즐겨찾기 표시하면 회원님 방문 시 제 핸드폰에 문자가 와요. 이거 되게 중요한 기능인데. 회원권 만료가 얼마 안 남으셨다면 먼저 안내 해드려야 되는데" — 대치점 FC

### 표준화 필요
> "지점마다 관리자들이 갖고 있는 정보로 다 잘 되고 있대요. 근데 매뉴얼화가 전혀 안 되어 있어요. 통일이 안 된다" — 고척점

### 최종 목표
> "레슨북의 장점과 온핏과 브로제이의 장점 긁어와서 하이브리드 만드는 게 우리의 최종 목적" — 고척점
