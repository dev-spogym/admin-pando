## 3. 네비게이션 구조

### 3.1 하단 탭 바 (Bottom Tab Navigation)
```
[ 홈 ] [ 출석 ] [ 예약 ] [ 내정보 ] [ 더보기 ]
  MA-010  MA-030   MA-040   MA-050    MA-070
```

### 3.2 주요 플로우

#### 출석 플로우
```
홈 (MA-010) → QR 체크인 (MA-030) → 체크인 완료 팝업
                                    → 출석 이력 (MA-031)
```

#### 예약 플로우
```
수업 목록 (MA-040) → 수업 상세 (MA-041) → 예약 확인 팝업
                                         → 내 예약 목록 (MA-042)
                                         → 예약 취소 확인 팝업
```

#### 구매 플로우
```
상품 목록 (MA-060) → 상품 상세 (MA-061) → 결제 (MA-062)
                                         → 쿠폰 선택 (MA-055)
                                         → 결제 완료 → 이용권 활성화
```

#### 내 정보 플로우
```
프로필 (MA-050) → 이용권 (MA-051)
               → 체성분 (MA-052)
               → 결제 이력 (MA-053)
               → 락커 (MA-054)
               → 쿠폰함 (MA-055)
               → 마일리지 (MA-056)
```
