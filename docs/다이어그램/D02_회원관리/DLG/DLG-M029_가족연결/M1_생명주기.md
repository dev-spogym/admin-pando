## 1. 목적

DLG-M029 가족 회원 연결 다이얼로그의 열기/닫기/완료 생명주기를 명세한다. 🆕 미구현 기능.

## 2. 트리거/전제조건

- 회원 상세 > "가족 연결" 버튼 클릭

## 3. 다이어그램

```mermaid
flowchart TD
  TRIGGER([회원 상세 가족 연결 버튼 클릭])

  TRIGGER --> DLG_OPEN[DLG-M029 가족 회원 연결 🆕]
  DLG_OPEN --> FLD_MEMBER[가족 회원 member search 필수]
  DLG_OPEN --> FLD_REL[관계 select 필수 기본값 배우자]

  FLD_MEMBER --> USER_ACT{사용자 액션}
  FLD_REL --> USER_ACT

  USER_ACT -->|취소|CLOSE[모달 닫기]
  USER_ACT -->|연결 클릭|VALID{필드 유효성}

  VALID -->|회원 미선택|ERR_MEMBER[회원을 선택하세요]
  ERR_MEMBER --> DLG_OPEN

  VALID -->|통과|API_CALL[🆕]

  API_CALL -->|200|TOAST_OK((toast 가족 관계가 연결되었습니다))
  TOAST_OK --> CLOSE_OK[모달 닫기 + 가족 목록 갱신]

  API_CALL -->|오류|TOAST_ERR((toast 연결에 실패했습니다))
  TOAST_ERR --> DLG_OPEN

  classDef newFeature fill:#F3E5F5,stroke:#9C27B0,stroke-dasharray:5 5
  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20

  classDef system fill:#EDE7F6,stroke:#5E35B2

  class DLG_OPEN,FLD_MEMBER,FLD_REL newFeature
  class TOAST_OK,CLOSE_OK success
  class TOAST_ERR,ERR_MEMBER error
  class API_CALL system
```
