## 1. 목적

DLG-M007 작성 취소 확인 다이얼로그의 열기/닫기/완료 생명주기를 명세한다.

## 2. 트리거/전제조건

- 회원 등록/수정 > "취소" 버튼 클릭
- (폼 변경사항 존재)
