## 1. 목적

DLG-M027 주소 검색 다이얼로그의 열기/닫기/완료 생명주기를 명세한다.

## 2. 트리거/전제조건

- 회원 등록/수정 Step 2 > "주소 검색" 버튼 클릭

## 3. 다이어그램

```mermaid
flowchart TD
  TRIGGER([주소 검색 버튼 클릭])

  TRIGGER -->|E_OPEN_01| DLG_OPEN[DLG-M027 주소 검색 모달]
  DLG_OPEN --> INIT[검색어 빈값 초기화]

  INIT --> USER_ACT{사용자 액션}

  USER_ACT -->|E_SEARCH_01 검색어 입력| SEARCH[SAMPLE_ADDRESSES 검색]
  SEARCH -->|E_RESULT_01 결과 있음| RESULT_LIST[주소 목록 표시]
  SEARCH -->|E_EMPTY_01 결과 없음| RESULT_EMPTY[검색 결과가 없습니다 00000]

  RESULT_LIST -->|E_SELECT_01 주소 클릭| AUTO_FILL[address 필드 자동 입력]
  AUTO_FILL   -->|E_CLOSE_AUTO_01| CLOSE[모달 자동 닫기]

  USER_ACT -->|E_X_01 X 버튼| CLOSE

  classDef modal   fill:#FFF3E0,stroke:#E65100,color:#BF360C
  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
  classDef system  fill:#EDE7F6,stroke:#5E35B2

  class DLG_OPEN,INIT modal
  class AUTO_FILL,CLOSE success
  class SEARCH,RESULT_LIST,RESULT_EMPTY system
```

## 4. 엣지 설명

| 엣지 ID | 출발 | 도착 | 조건 |
|---------|------|------|------|
| E_OPEN_01 | 주소 검색 버튼 | 모달 열기 | - |
| E_SEARCH_01 | 검색어 입력 | SAMPLE_ADDRESSES 검색 | - |
| E_SELECT_01 | 주소 클릭 | 자동 입력 | - |
| E_CLOSE_AUTO_01 | 자동 입력 | 모달 닫기 | 선택 시 자동 |
| E_X_01 | X 버튼 | 모달 닫기 | - |
