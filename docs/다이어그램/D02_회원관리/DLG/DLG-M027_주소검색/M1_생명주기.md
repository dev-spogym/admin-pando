## 1. 목적

DLG-M027 주소 검색 다이얼로그의 열기/닫기/완료 생명주기를 명세한다.

## 2. 트리거/전제조건

- 회원 등록/수정 Step 2 > "주소 검색" 버튼 클릭

## 3. 다이어그램

```mermaid
flowchart TD
  TRIGGER([주소 검색 버튼 클릭])

  TRIGGER --> DLG_OPEN[DLG-M027 주소 검색 모달]
  DLG_OPEN --> INIT[검색어 빈값 초기화]

  INIT --> USER_ACT{사용자 액션}

  USER_ACT -->|검색어 입력|SEARCH[SAMPLE_ADDRESSES 검색]
  SEARCH -->|결과 있음|RESULT_LIST[주소 목록 표시]
  SEARCH -->|결과 없음|RESULT_EMPTY[검색 결과가 없습니다 00000]

  RESULT_LIST -->|주소 클릭|AUTO_FILL[address 필드 자동 입력]
  AUTO_FILL --> CLOSE[모달 자동 닫기]

  USER_ACT -->|X 버튼|CLOSE

  classDef modal fill:#FFF3E0,stroke:#E65100,color:#BF360C
  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
  classDef system fill:#EDE7F6,stroke:#5E35B2

  class DLG_OPEN,INIT modal
  class AUTO_FILL,CLOSE success
  class SEARCH,RESULT_LIST,RESULT_EMPTY system
```
