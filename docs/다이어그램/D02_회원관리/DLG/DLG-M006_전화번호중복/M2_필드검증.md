## 1. 목적

DLG-M006 중복확인 버튼 활성화 조건과 인라인 결과 표시 검증을 명세한다.

## 2. 트리거/전제조건

- 연락처 필드 입력 중

## 3. 다이어그램

```mermaid
flowchart TD
  FLD_PHONE[연락처 입력 필드]

  FLD_PHONE -->|입력변경|RESET[인라인 초기화]
  RESET --> CHECK_LEN{phone}

  CHECK_LEN -->|충족|BTN_ACTIVE[중복확인 버튼 활성]
  CHECK_LEN -->|미충족|BTN_DISABLED[중복확인 버튼 비활성]

  BTN_ACTIVE -->|확인완료|BTN_DISABLED2[재입력 전까지 버튼 비활성]

  classDef system fill:#EDE7F6,stroke:#5E35B2
  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
  classDef error fill:#FFEBEE,stroke:#C62828,color:#B71C1C

  class FLD_PHONE,CHECK_LEN,RESET system
  class BTN_ACTIVE success
  class BTN_DISABLED,BTN_DISABLED2 error
```
