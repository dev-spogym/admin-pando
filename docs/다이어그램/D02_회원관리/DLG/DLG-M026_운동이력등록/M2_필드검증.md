## 3. 다이어그램

flowchart TD
  DLG[DLG-M026 열림]

  DLG --> FLD_DATE[날짜 date 필수 기본=오늘]
  DLG --> FLD_NAME[운동명 text 필수]
  DLG --> FLD_OPT[세트/횟수/중량/시간/거리/메모 선택]

  FLD_DATE --> CHECK_ALL{AND 있음}
  FLD_NAME --> CHECK_ALL

  CHECK_ALL --> BTN_ACTIVE[저장 버튼 활성]

  classDef system fill:#EDE7F6,stroke:#5E35B2

  class DLG,FLD_DATE,FLD_NAME,FLD_OPT,CHECK_ALL system
