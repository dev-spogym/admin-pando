## 1. 목적

DLG-M009 인라인 폼의 필드별 유효성 검증을 명세한다.

## 2. 트리거/전제조건

- 상담메모 탭 인라인 폼 표시 상태

## 3. 다이어그램

```mermaid
flowchart TD
  FORM[인라인 메모 폼]

  FORM --> FLD_CAT[카테고리 select w-40 기본=상담]
  FORM --> FLD_CONTENT[내용 textarea 3행 필수]

  FLD_CAT -->|선택|CAT_OK[카테고리 저장]
  FLD_CONTENT -->|입력|CHECK_CONTENT{trim}

  CHECK_CONTENT -->|입력됨|BTN_ACTIVE[메모 저장 버튼 활성]
  CHECK_CONTENT -->|비어있음|BTN_DISABLED[메모 저장 버튼 disabled]

  BTN_ACTIVE -->|클릭|SUBMIT[API 호출]
  BTN_DISABLED --> NO_ACTION[동작 없음]

  classDef system fill:#EDE7F6,stroke:#5E35B2
  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
  classDef error fill:#FFEBEE,stroke:#C62828,color:#B71C1C

  class FORM,FLD_CAT,FLD_CONTENT,CHECK_CONTENT system
  class CAT_OK,BTN_ACTIVE,SUBMIT success
  class BTN_DISABLED,NO_ACTION error
```

## 4. 엣지 설명

| 출발 | 도착 | 조건 |
|------|------|------|
| 내용 확인 | 버튼 활성 | trim |
| 내용 확인 | 버튼 비활성 | 비어있음 |
| 버튼 활성 | API | 클릭 |
