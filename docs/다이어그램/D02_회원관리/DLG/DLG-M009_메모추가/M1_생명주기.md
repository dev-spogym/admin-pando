## 1. 목적

DLG-M009 메모 추가 인라인 폼의 트리거/저장/완료 생명주기를 명세한다.

## 2. 트리거/전제조건

- 상담메모 탭 > "메모 저장" 버튼 클릭 (인라인 폼, 모달 아님)

## 3. 다이어그램

```mermaid
flowchart TD
  TAB([상담메모 탭 활성화])

  TAB --> INLINE_FORM[인라인 메모 입력 폼 표시]
  INLINE_FORM --> FILL[카테고리 선택 + 내용 입력]

  FILL --> BTN_SAVE{메모 저장 버튼}

  BTN_SAVE -->|내용 비어있음|BTN_DISABLED[버튼 disabled]
  BTN_SAVE -->|내용 입력됨|API_CALL[동작]

  API_CALL -->|200|TOAST_OK((toast 메모가 저장되었습니다))
  TOAST_OK --> RESET[폼 초기화 카테고리=상담 내용=빈값]
  RESET --> REFETCH[메모 목록 refetch]

  API_CALL -->|오류|TOAST_ERR((toast 메모 저장 실패 error))

  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
  classDef error fill:#FFEBEE,stroke:#C62828,color:#B71C1C
  classDef system fill:#EDE7F6,stroke:#5E35B2

  class TOAST_OK,RESET,REFETCH success
  class BTN_DISABLED,TOAST_ERR error
  class INLINE_FORM,BTN_SAVE,API_CALL system
```
