## 1. 목적

DLG-M023 이관 확인 다이얼로그의 열기/닫기/완료 생명주기를 명세한다.

## 2. 트리거/전제조건

- 회원 이관(SCR-M005) > "이관 실행" 버튼 클릭

## 3. 다이어그램

```mermaid
flowchart TD
  TRIGGER([SCR-M005 이관 실행 버튼 클릭])

  TRIGGER --> DLG_OPEN[DLG-M023 이관 확인 max-w-sm]
  DLG_OPEN --> DISPLAY[회원명 지점명 이관유형 사유 표시]

  DISPLAY --> USER_ACT{사용자 액션}

  USER_ACT -->|취소 오버레이클릭|CLOSE[모달 닫기]
  USER_ACT -->|이관 확인 클릭|API_CALL[+]

  API_CALL -->|200|TOAST_OK((toast name 회원이 으로 이관되었습니다))
  TOAST_OK --> REDIRECT[/ 이동]

  API_CALL -->|오류|TOAST_ERR((toast 이관에 실패했습니다))
  TOAST_ERR --> CLOSE

  classDef modal fill:#FFF3E0,stroke:#E65100,color:#BF360C
  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20

  class DLG_OPEN,DISPLAY modal
  class TOAST_OK,REDIRECT success
  class TOAST_ERR error
  class API_CALL system
```
