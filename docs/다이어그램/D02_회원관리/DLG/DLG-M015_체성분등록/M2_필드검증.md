## 1. 목적

DLG-M015의 필드별 유효성 검증 흐름을 명세한다.

## 2. 트리거/전제조건

- DLG-M015 열린 상태

## 3. 다이어그램

```mermaid
flowchart TD
  DLG[DLG-M015 열림]

  DLG --> FLD_DATE[측정일 date 필수]
  DLG --> FLD_WEIGHT[체중 number 0.1 필수 20~300kg]
  DLG --> FLD_MUSCLE[골격근량 number 0.1 필수 5~80kg]
  DLG --> FLD_PBF[체지방률 number 0.1 필수 3~60%]

  FLD_DATE -->|비어있음|ERR_DATE[날짜를 입력하세요]
  FLD_WEIGHT -->|범위초과|ERR_WEIGHT[체중을 올바르게 입력하세요 20~300kg]
  FLD_MUSCLE -->|범위초과|ERR_MUSCLE[골격근량을 올바르게 입력하세요 5~80kg]
  FLD_PBF -->|범위초과|ERR_PBF[체지방률을 올바르게 입력하세요 3~60%]

  FLD_DATE --> AUTO_CALC[BMI + 체지방량 자동계산]
  FLD_WEIGHT --> AUTO_CALC
  FLD_PBF --> AUTO_CALC

  AUTO_CALC -->|모두 유효|BTN_ACTIVE[저장 버튼 활성]
  AUTO_CALC -->|오류 존재|BTN_DISABLED[저장 버튼 disabled]

  classDef system fill:#EDE7F6,stroke:#5E35B2
  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
  classDef error fill:#FFEBEE,stroke:#C62828,color:#B71C1C

  class DLG,FLD_DATE,FLD_WEIGHT,FLD_MUSCLE,FLD_PBF,AUTO_CALC system
  class BTN_ACTIVE success
  class ERR_DATE,ERR_WEIGHT,ERR_MUSCLE,ERR_PBF,BTN_DISABLED error
```

| 출발 | 도착 | 조건 |
|------|------|------|
| 측정일 | 에러 | 비어있음 |
| 체중 | 에러 | 20~300 범위 초과 |
| 골격근량 | 에러 | 5~80 범위 초과 |
| 체지방률 | 에러 | 3~60 범위 초과 |
| 자동계산 | 버튼 활성 | 모두 유효 |
