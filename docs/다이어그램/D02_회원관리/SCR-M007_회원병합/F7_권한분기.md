## 1. 목적

SCR-M007에서 역할별 접근 및 액션 가능 범위를 명세한다. 🆕 미구현 기능.

## 2. 트리거/전제조건

- 사용자가 로그인 상태이다.

## 3. 다이어그램

```mermaid
flowchart LR
  USER([사용자])
  USER --> ROLE{역할 확인}

  ROLE --> PRIM[primary]
  ROLE --> OWN[owner]
  ROLE --> MGR[manager]
  ROLE --> FC[fc]
  ROLE --> STF[staff]
  ROLE --> RO[readonly]

  PRIM -->|접근+병합 실행|FULL[전체 기능 🆕]
  OWN -->|접근+병합 실행|FULL

  MGR -->|접근 X|BLOCKED[접근 차단]
  FC -->|접근 X|BLOCKED
  STF -->|접근 X|BLOCKED
  RO -->|접근 X|BLOCKED

  classDef newFeature fill:#F3E5F5,stroke:#9C27B0,stroke-dasharray:5 5
  classDef rbacBlocked fill:#F5F5F5,stroke:#9E9E9E,color:#616161
  classDef system fill:#EDE7F6,stroke:#5E35B2

  class FULL newFeature
  class BLOCKED rbacBlocked
  class ROLE system
```

## 4. 엣지 설명

| 출발 | 도착 | 조건 | |---------|------|------|------| | | primary | 전체 기능 | 접근 허용 | | | owner | 전체 기능 | 접근 허용 | | | manager | 접근 차단 | 권한 없음 | | | fc | 접근 차단 | 권한 없음 | | | staff | 접근 차단 | 권한 없음 | | | readonly | 접근 차단 | 권한 없음 |
