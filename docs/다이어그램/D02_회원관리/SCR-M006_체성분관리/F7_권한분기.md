## 1. 목적

SCR-M006에서 역할별 접근 및 액션 가능 범위를 명세한다.

## 2. 트리거/전제조건

- 사용자가 로그인 상태이다.

## 3. 다이어그램

```mermaid
flowchart LR
  USER([사용자])

  USER --> ROLE{역할 확인}

  ROLE --> PRIM[primary]
  ROLE --> OWN[owner]
  ROLE --> MGR[manager]
  ROLE --> FC[fc]
  ROLE --> STF[staff]
  ROLE --> RO[readonly]

  PRIM -->|접근+조회+추가|FULL_ACCESS[전체 기능]
  OWN -->|접근+조회+추가|FULL_ACCESS
  MGR -->|접근+조회+추가|FULL_ACCESS
  FC -->|접근+조회+추가|FULL_ACCESS

  STF -->|접근 X|BLOCKED[접근 차단]
  RO -->|접근 X|BLOCKED

  FULL_ACCESS --> ACT_VIEW[체성분 데이터 조회]
  FULL_ACCESS --> ACT_ADD[측정 추가 DLG-M015]
  FULL_ACCESS --> ACT_GOAL[목표 설정 DLG-M017 🆕]

  classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
  classDef rbacBlocked fill:#F5F5F5,stroke:#9E9E9E,color:#616161
  classDef system fill:#EDE7F6,stroke:#5E35B2
  classDef newFeature fill:#F3E5F5,stroke:#9C27B0,stroke-dasharray:5 5

  class FULL_ACCESS,ACT_VIEW,ACT_ADD screen
  class ACT_GOAL newFeature
  class BLOCKED rbacBlocked
  class ROLE system
```
