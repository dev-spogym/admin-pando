## 1. 목적

SCR-M006의 데이터 건수 및 로딩/에러 상태별 UI 분기를 명세한다.

## 2. 트리거/전제조건

- SCR-M006 마운트 시점

## 3. 다이어그램

```mermaid
flowchart TD
  MOUNT([SCR-M006 마운트])

  MOUNT --> STATE_LOAD{데이터 로딩 상태}

  STATE_LOAD -->|fetch 중|UI_LOADING[스피너 표시]
  STATE_LOAD -->|API 오류|UI_ERR[에러 상태 표시]
  STATE_LOAD -->|완료|DATA_COUNT{데이터 건수}

  DATA_COUNT -->|0건|UI_ZERO[통계 카드 하이픈 차트 미표시 테이블 빈 상태]
  DATA_COUNT -->|1건|UI_ONE[통계 카드 현재값만 차트 미표시 최소 2건 필요]
  DATA_COUNT -->|2건+|UI_FULL[통계 카드 change 라인 차트 테이블 전체]

  UI_FULL --> CHART_STATE{차트 데이터 상태}
  CHART_STATE -->|모든 메트릭 OFF|UI_CHART_EMPTY[차트 영역 빈 상태]
  CHART_STATE -->|하나 이상 ON|UI_CHART_LINES[라인 차트 표시]

  classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
  classDef system fill:#EDE7F6,stroke:#5E35B2
  classDef error fill:#FFEBEE,stroke:#C62828,color:#B71C1C
  classDef info fill:#E0F7FA,stroke:#00838F

  class UI_FULL,UI_CHART_LINES screen
  class STATE_LOAD,DATA_COUNT,CHART_STATE system
  class UI_ERR error
  class UI_LOADING,UI_ZERO,UI_ONE,UI_CHART_EMPTY info
```
