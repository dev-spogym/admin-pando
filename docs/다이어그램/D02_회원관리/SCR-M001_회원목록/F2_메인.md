## 1. 목적

SCR-M001의 정상 시나리오(Happy Path)를 명세한다. 진입→필터→검색→행 선택→상세 이동 또는 벌크 액션의 전체 흐름. 성공/검증실패/시스템에러 3갈래 분기 강제.

## 2. 전제조건

- primary 중 하나로 로그인되어 있다.
- SCR-M001에 진입 완료 후 데이터 로드가 성공한 상태이다.
