## 1. 목적

SCR-M001에서 발생하는 모든 토스트 메시지(성공/경고/에러/정보)의 발생 조건을 명세한다. 피드백 TC 원천.

## 2. 전제조건

- SCR-M001 회원 목록이 표시된 상태이다.
