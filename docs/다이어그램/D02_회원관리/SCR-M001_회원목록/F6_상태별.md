## 1. 목적

SCR-M001의 로딩/빈/에러/권한없음 등 UI 상태별 화면 분기를 명세한다. UI 상태 TC 원천.

## 2. 전제조건

- 사용자가 SCR-M001 진입을 시도한 상태이다.
