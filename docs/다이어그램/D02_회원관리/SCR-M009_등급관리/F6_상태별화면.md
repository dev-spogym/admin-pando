## 3. 다이어그램

```mermaid
flowchart TD
  MOUNT([SCR-M009 마운트 🆕])

  MOUNT --> LOAD_STATE{로딩 상태}

  LOAD_STATE -->|fetch 중|UI_LOADING[스피너]

  ROLE_STATE -->|primary/owner|UI_FULL[등급 카드 수정버튼 + 테이블 수동변경 + 규칙 설정]
  ROLE_STATE -->|manager|UI_VIEW[등급 카드 조회 + 테이블 조회 — 수정/변경 불가]

  UI_FULL --> EDIT_STATE{등급 수정 상태}

  UI_FULL --> SAVE_STATE{저장 상태}
```
