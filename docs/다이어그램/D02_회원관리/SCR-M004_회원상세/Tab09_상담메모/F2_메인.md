## 1. 목적

상담메모 탭(SCR-M004-09)의 메모 작성 폼 + 목록 표시 + 수정/삭제 플로우를 정의한다.

## 2. 전제조건

- tab=memo 활성, member_memos 데이터 로드 완료

## 3. 다이어그램

```mermaid
flowchart TD
  TAB[Tab09 상담메모 활성] --> FORM[새 메모 작성 폼\n카테고리 드롭다운 상담/특이사항/일반\ntextarea 3행]

  FORM --> SAVE_BTN{메모 저장 버튼\n! → disabled}
  SAVE_BTN -->|": 내용 있음 클릭"|[동작]

  DLG_M010 -->|": 취소"|MEMO_CARDS
```

## 4. 엣지 설명

| 조건/액션 | 결과 |
|-----------|------|
| 내용 있음, 저장 클릭 | |
| 성공 | 토스트 + 목록 갱신 |
| 실패 | 에러 토스트 |
| 메모 없음 | 빈 상태 |
| 메모 있음 | 카드 목록 |
| 수정 버튼 | 인라인 편집 모드 |
| 수정 저장 성공 | 토스트 + 카드 업데이트 |
| 삭제 버튼 | DLG-M010 |
| 삭제 확인 | 토스트 + 목록 갱신 |
| 삭제 취소 | 모달 닫기 |
