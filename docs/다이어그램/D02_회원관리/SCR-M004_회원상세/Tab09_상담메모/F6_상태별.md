## 1. 목적

상담메모 탭의 데이터/수정모드 상태별 화면 분기를 정의한다.

## 2. 전제조건

- Tab09 상담메모 활성

## 3. 다이어그램

```mermaid
flowchart TD
  TAB[Tab09 상담메모] --> LIST{판단}
  LIST -->|": 0"|EMPTY[아이콘+작성된 메모가 없습니다.]
  LIST -->|": 1+"|CARDS[메모 카드 목록]

  CARDS --> MODE{수정 모드}
  MODE -->|": 일반 모드"|NORMAL[텍스트 표시\n수정/삭제 버튼]
  MODE -->|": 수정 모드"|EDIT[Textarea 전환\n저장/취소 버튼]

  classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
  classDef system fill:#EDE7F6,stroke:#5E35B2
  class TAB,CARDS,NORMAL screen
  class LIST,MODE system
```

## 4. 엣지 설명

| 조건 | 화면 | |---------|------|------| | | 메모 없음 | 빈 상태 메시지 | | | 메모 있음 | 카드 목록 | | | 일반 모드 | 텍스트 + 수정/삭제 버튼 | | | 수정 모드 | Textarea + 저장/취소 버튼 |
