## 1. 목적

레슨 탭의 데이터 유무 및 서명 상태별 화면 분기를 정의한다.

## 2. 전제조건

- Tab10 레슨 활성

## 3. 다이어그램

```mermaid
flowchart TD
  TAB[Tab10 레슨] --> LIST{lessons}
  LIST -->|": 0"|EMPTY[아이콘+레슨 기록이 없습니다.]
  LIST -->|": 1+"|CARDS[레슨 카드 목록]

  CARDS --> SIGN{서명 상태}
  SIGN -->|": 서명 완료"|SIGNED[서명됨 배지]
  SIGN -->|": 미서명"|UNSIGNED[미서명 배지]

  classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
  classDef system fill:#EDE7F6,stroke:#5E35B2
  class TAB,CARDS,SIGNED,UNSIGNED screen
  class LIST,SIGN system
```
