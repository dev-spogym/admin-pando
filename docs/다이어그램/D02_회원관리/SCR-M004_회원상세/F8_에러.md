| 엣지 ID | 에러 유형 | 복구 경로 |

| E_F8_03 | 403 권한없음 | /members 리다이렉트 |
| E_F8_04 | 404 회원없음 | 빈 화면 |
| E_F8_05 | 500 서버오류 | 에러 토스트 + /members |
| E_F8_07 | 네트워크 타임아웃 | 재시도 or /members |
