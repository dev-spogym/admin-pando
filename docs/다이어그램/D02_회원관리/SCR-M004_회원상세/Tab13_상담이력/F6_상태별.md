## 1. 목적

상담이력 탭의 데이터 유무 및 전환 결과 상태별 화면 분기를 정의한다.

## 2. 전제조건

- Tab13 상담이력 활성

## 3. 다이어그램

```mermaid
flowchart TD
  TAB[Tab13 상담이력] --> LIST{consultings}
  LIST -->|": 0"|EMPTY[아이콘+상담 이력이 없습니다.]
  LIST -->|": 1+"|TABLE[DataTable 10컬럼]

  TABLE --> RESULT{결과 유형}
  RESULT -->|": 전환"|CONVERTED[전환 배지 초록]
  RESULT -->|": 미전환"|NOT_CONVERTED[미전환 배지 회색]
  RESULT -->|": 재상담"|FOLLOW_UP[재상담 배지 파랑]

  classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
  classDef system fill:#EDE7F6,stroke:#5E35B2
  class TAB,TABLE screen
  class CONVERTED success
  class LIST,RESULT system
```
