## 1. 목적

회원정보 탭의 데이터 상태별(필드 값 있음/없음/특이사항/메모) 화면 분기를 정의한다.

## 2. 전제조건

- Tab01 회원정보 활성

## 3. 다이어그램

```mermaid
flowchart TD
  TAB[Tab01 회원정보] --> FIELD_STATE{필드 값 상태}
  FIELD_STATE -->|": 모든 필드 값 있음"|FULL[모든 필드 정상 표시]
  FIELD_STATE -->|": 일부 필드 null"|PARTIAL[null 필드 - 표시\n나머지 정상]

  TAB --> SPEC{판단}
  SPEC -->|": 있음"|SPEC_SHOW[주황 배경 특이사항 박스]
  SPEC -->|": 없음"|SPEC_HIDE[특이사항 박스 미표시]

  TAB --> MEMO{메모 내용}
  MEMO -->|": 있음"|MEMO_SHOW[메모 내용 표시]
  MEMO -->|": 없음"|MEMO_EMPTY[메모가 없습니다.]

  classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
  classDef system fill:#EDE7F6,stroke:#5E35B2
  classDef warning fill:#FFF8E1,stroke:#F9A825,color:#F57F17
  class TAB,FULL,MEMO_SHOW screen
  class FIELD_STATE,SPEC,MEMO system
  class SPEC_SHOW warning
```

## 4. 엣지 설명

| 조건 | 화면 | |---------|------|------| | | 모든 필드 값 있음 | 정상 표시 | | | 일부 필드 null | null 필드 "-" | | | 있음 | 주황 특이사항 박스 | | | 없음 | 박스 미표시 | | | 메모 있음 | 메모 표시 | | | 메모 없음 | "메모가 없습니다." |
