## 1. 목적

회원정보 탭에서 사용 가능한 버튼과 액션을 정의한다. 탭 자체에는 수정 폼이 없고 프로필 헤더의 수정 버튼으로 SCR-M003 이동.

## 2. 전제조건

- Tab01 회원정보 활성

## 3. 다이어그램

```mermaid
flowchart LR

  B_EDIT -->|"E_F3_01: canEdit=true"| EDIT_PAGE[/members/edit?id=memberId]
  B_EDIT -->|"E_F3_02: canEdit=false"| BLOCKED[권한없음]
```

## 4. 엣지 설명

| 엣지 ID | 버튼 | 조건 | 동작 |
|---------|------|------|------|
| E_F3_01 | 수정 | canEdit=true | /members/edit 이동 |
| E_F3_02 | 수정 | canEdit=false | 권한없음 |
| E_F3_03 | 기본정보 섹션 헤더 | 항상 | 섹션 접기/펼치기 |
| E_F3_04 | 운영정보 섹션 헤더 | 항상 | 섹션 접기/펼치기 |
