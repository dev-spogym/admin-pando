## 1. 목적

회원정보 탭에서 사용 가능한 버튼과 액션을 정의한다. 탭 자체에는 수정 폼이 없고 프로필 헤더의 수정 버튼으로 SCR-M003 이동.

## 2. 전제조건

- Tab01 회원정보 활성

## 3. 다이어그램

```mermaid
flowchart LR
  TAB[Tab01 회원정보] --> B_EDIT[수정 버튼\n프로필 헤더 위임\ 조건]
  B_EDIT -->|": "|EDIT_PAGE[동작]
  B_EDIT -->|": "|BLOCKED[권한없음]

  TAB --> SEC_COLLAPSE[기본정보 섹션 접기/펼치기\ncollapsible 토글]
  SEC_COLLAPSE -->|": 클릭"|TOGGLE[섹션 접힘/펼침 상태 전환]

  TAB --> SEC_OPS_COLLAPSE[운영정보 섹션 접기/펼치기]
  SEC_OPS_COLLAPSE -->|": 클릭"|TOGGLE2[섹션 접힘/펼침 상태 전환]

  classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
  classDef rbacBlocked fill:#F5F5F5,stroke:#9E9E9E,color:#616161
  classDef system fill:#EDE7F6,stroke:#5E35B2
  class TAB,EDIT_PAGE screen
  class BLOCKED rbacBlocked
  class TOGGLE,TOGGLE2 system
```
