## 1. 목적

체성분 탭(SCR-M004-08)의 통계 카드 + 라인차트 + 테이블 플로우를 정의한다.

## 2. 전제조건

- tab=body 활성, 데이터 로드 완료

## 4. 엣지 설명

| 조건 | 결과 |
|------|------|
| 데이터 0건 | 통계 카드 -, 차트/테이블 빈 상태 |
| 데이터 1건 | change 없음, 차트 미표시 |
| 데이터 2건+ | 전체 표시 |
| 메트릭 토글 | 라인 on/off |
| 측정 추가 클릭 | DLG-M015 열기 |
| 동일 날짜 | DLG-M016 덮어쓰기 확인 |
| 신규 날짜 | 직접 저장 |
| 덮어쓰기 확인 | 기존 데이터 갱신 |
| 덮어쓰기 취소 | DLG-M015 복귀 |
