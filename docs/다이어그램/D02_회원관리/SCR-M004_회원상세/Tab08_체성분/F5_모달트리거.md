## 1. 목적

체성분 탭에서 트리거되는 DLG-M015/M016 모달 생명주기를 정의한다.

## 2. 전제조건

- Tab08 체성분 활성

## 3. 다이어그램

```mermaid
flowchart TD
  TAB[Tab08 체성분] --> DLG_M015

  DLG_M015[DLG-M015 체성분 등록\n트리거: 측정 추가]
  DLG_M015 --> FIELDS[측정일 / 체중 / 골격근량 / 체지방률\n유효범위: 체중 20-300 골격근 5-80 체지방 3-60]
  FIELDS -->|": 저장 클릭"|DATE_CHK{동일 날짜 존재}

  SAVE -->|": 성공"|OK((토스트: 체성분이 등록되었습니다.))
```
