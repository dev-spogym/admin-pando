## 1. 목적

SCR-M004 회원 상세 화면에 진입 가능한 모든 경로를 정의한다. 진입 시 memberId 파라미터 유효성 검증 및 권한 확인 분기를 포함한다.

## 2. 전제조건

- 로그인 세션 유효
- memberId 파라미터 존재
