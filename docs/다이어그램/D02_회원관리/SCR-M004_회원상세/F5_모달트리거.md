## 1. 목적

SCR-M004 루트(프로필 헤더/하단 상태관리/위험구역)에서 트리거되는 모달 전체를 트리 구조로 정의한다. 탭별 모달은 각 탭의 F5에서 정의한다.

## 2. 전제조건

- SCR-M004 데이터 로드 완료
