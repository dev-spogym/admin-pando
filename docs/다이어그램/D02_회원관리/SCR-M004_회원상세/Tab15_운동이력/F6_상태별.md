## 1. 목적

운동이력 탭의 데이터 유무 및 날짜 그룹 상태별 화면 분기를 정의한다.

## 2. 전제조건

- Tab15 운동이력 활성

## 3. 다이어그램

```mermaid
flowchart TD
  TAB[Tab15 운동이력] --> LIST{records}
  LIST -->|": 0"|EMPTY[아이콘+운동 기록이 없습니다.]
  LIST -->|": 1+"|GROUPS[날짜별 그룹\n내림차순 정렬]

  GROUPS --> GROUP[날짜 헤더\n해당 날짜 운동 카드 목록]
  GROUP --> FIELDS[운동명 / 세트 / 횟수\n무게 / 시간 / 메모]

  classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
  classDef system fill:#EDE7F6,stroke:#5E35B2
  class TAB,GROUPS,GROUP,FIELDS screen
  class LIST system
```

## 4. 엣지 설명

| 조건 | 화면 |
|------|------|
| 기록 없음 | 빈 상태 메시지 |
| 기록 있음 | 날짜별 그룹 내림차순 |
