## 1. 목적

운동프로그램 탭(SCR-M004-14)의 프로그램 카드 조회 + 등록/수정/삭제 플로우를 정의한다.

## 2. 전제조건

- tab=program 활성
- 운동프로그램 데이터 로드 완료

## 3. 다이어그램

