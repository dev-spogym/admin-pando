## 1. 목적

SCR-M010에서 모달/패널이 열리는 트리거 조건을 명세한다. 🆕 미구현 기능.

## 2. 트리거/전제조건

- SCR-M010 렌더링 완료

## 3. 다이어그램

```mermaid
flowchart TD
  SCR_M010[SCR-M010 세그먼트 관리 🆕]

  SCR_M010 --> TRIG_ADD[세그먼트 추가 버튼 클릭]
  SCR_M010 --> TRIG_EDIT[테이블 행 수정 클릭]
  SCR_M010 --> TRIG_DEL[테이블 행 삭제 클릭]

  TRIG_ADD -->|primary/owner|MODAL_BUILDER[조건 빌더 패널 열기 🆕]
  TRIG_EDIT -->|primary/owner|MODAL_BUILDER
  TRIG_DEL -->|primary/owner|MODAL_CONFIRM[삭제 확인 모달]

  MODAL_BUILDER -->|취소/X|SCR_M010
  MODAL_BUILDER -->|저장|API_SAVE[POST/]
  API_SAVE -->|200|CLOSE_BUILDER[패널 닫기 + 갱신]
  API_SAVE -->|4xx/5xx|ERR_BUILDER[에러 메시지 표시]

  MODAL_CONFIRM -->|확인|API_DEL[동작]
  MODAL_CONFIRM -->|취소|SCR_M010
  API_DEL -->|200|CLOSE_CONFIRM[모달 닫기 + 갱신]
  API_DEL -->|5xx|ERR_DEL[삭제 실패 toast]

  classDef newFeature fill:#F3E5F5,stroke:#9C27B0,stroke-dasharray:5 5
  classDef modal fill:#FFF3E0,stroke:#E65100,color:#BF360C

  class SCR_M010,MODAL_BUILDER newFeature

```
