## 3. 다이어그램

```mermaid
flowchart TD
  API{-->|": 200"|Success[정책 저장 완료]
  API -->|": 400"|BadReq[toast 입력값을 확인해주세요.]
  API -->|": 403"|PermErr[toast 권한이 없습니다.]
  API -->|": 500"|SysErr[toast 처리에 실패했습니다.]

  Success --> BranchStorage[lesson_policy 갱신]
  BranchStorage --> CloseModal[모달 닫힘]
  CloseModal --> Toast_S[toast 정책이 저장되었습니다.]

  BadReq --> Stay[모달 유지]
  PermErr --> Stay
  SysErr --> Stay

  classDef system fill:#EDE7F6,stroke:#5E35B2
```
