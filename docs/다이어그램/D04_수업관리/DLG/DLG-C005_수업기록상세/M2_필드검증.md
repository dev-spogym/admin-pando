## 1. 목적
DLG-C005에서 상태 전환 버튼의 활성화 조건을 정의한다.

## 2. 전제조건
- DLG-C005 열림 상태

## 3. 다이어그램

```mermaid
flowchart TD
  StatusCheck{현재 lesson_status} -->|": scheduled"|ShowStart[수업시작 버튼 활성\n노쇼 버튼 활성\n취소 버튼 활성]
  StatusCheck -->|": in_progress"|ShowComplete[완료처리 버튼 활성\n취소 버튼 활성]
  StatusCheck -->|": completed"|ReadOnly[모든 버튼 비활성\n서명 이미지 표시]
  StatusCheck -->|": no_show"|ReadOnly
  StatusCheck -->|": cancelled"|ReadOnly

  ShowStart -->|": 상태전환 시도"|RoleCheck{역할 확인}
  ShowComplete -->|": 상태전환 시도"|RoleCheck

  RoleCheck -->|": fc 본인"|Allow
  RoleCheck -->|": fc 타강사"|Blocked[toast 권한이 없습니다.]
```
