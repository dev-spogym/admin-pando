## 1. 목적
DLG-C005 수업기록 상세 모달의 생명주기와 하위 모달 체인을 정의한다.

## 2. 전제조건
- SCR-C002 수업기록 테이블에서 Eye 버튼 클릭

## 3. 다이어그램

```mermaid
flowchart TD
  Trigger[Eye 상세 버튼 클릭] --> Open[DLG-C005 열림\n480px md]
  Open --> LoadDetail{LoadDetail -->|": 200"|Render[상세 렌더링\n수업명/강사/회원/상태/날짜/서명]
  LoadDetail -->|": 500"|LoadErr[toast 로드 실패]

  Render -->|": 수업시작 버튼"|StartUpdate{PATCH lesson_status=in_progress}

  Render -->|": 노쇼 버튼"|NoShowUpdate{PATCH lesson_status=no_show}
  Render -->|": 취소 버튼"|CancelUpdate{PATCH lesson_status=cancelled}

  StartUpdate -->|": 200"|StartOK[toast 수업이 시작되었습니다. + 갱신]
```

## 4. 엣지 설명

| 버튼 | 동작 | |------|------| | 수업시작 | lesson_status=in_progress | | 완료처리 | DLG-C006 서명 체인 | | 노쇼 | lesson_status=no_show + 모달 닫힘 | | 취소 | lesson_status=cancelled + 모달 닫힘 |
