## 1. 목적
DLG-C012 횟수 조정 API 결과 분기를 정의한다.

## 2. 전제조건
- 유효성 통과 후 API 호출

## 3. 다이어그램

```mermaid
flowchart TD
  API{-->|": 200"|Success[조정 완료]
  API -->|": 400"|BadReq[toast 입력값을 확인해주세요.]
  API -->|": 403"|PermErr[toast 권한이 없습니다.]
  API -->|": 500"|SysErr[toast 처리에 실패했습니다.]

  Success --> CloseModal[모달 닫힘]
  CloseModal --> RefreshList[SCR-C007 목록 갱신]
  RefreshList --> Toast_S[toast 횟수가 조정되었습니다.]

  BadReq --> Stay[모달 유지]
  PermErr --> Stay
  SysErr --> Stay

  classDef system fill:#EDE7F6,stroke:#5E35B2
  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
  classDef error fill:#FFEBEE,stroke:#C62828,color:#B71C1C

  class API system
```
