## 3. 다이어그램

```mermaid
flowchart TD
  API{-->|": 200"|Success[강사 상세 렌더링]
  API -->|": 404"|NotFound[toast 강사를 찾을 수 없습니다.\n모달 닫힘]
  API -->|": 403"|PermErr[toast 권한이 없습니다.]
  API -->|": 500"|SysErr[toast 처리에 실패했습니다.]

  Success --> ShowTabs[탭 렌더링\n수업목록 / 근무통계]
  NotFound --> CloseModal[모달 닫힘]
  PermErr --> CloseModal
  SysErr --> StayModal[모달 유지 + 재시도 버튼]

  classDef system fill:#EDE7F6,stroke:#5E35B2
  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
  classDef error fill:#FFEBEE,stroke:#C62828,color:#B71C1C
  classDef warning fill:#FFF8E1,stroke:#F9A825,color:#F57F17
  classDef modal fill:#FFF3E0,stroke:#F57C00,color:#E65100
  class API system
  class Success,ShowTabs success
  class SysErr error
  class NotFound,PermErr warning
  class CloseModal,StayModal modal
```

| 응답 | 동작 |
|------|------|
| 200 | 탭 렌더링 |
| 404/403 | 경고 + 닫힘 |
| 500 | 에러 + 재시도 |
