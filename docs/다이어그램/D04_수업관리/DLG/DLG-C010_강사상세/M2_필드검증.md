## 1. 목적
DLG-C010은 읽기전용 모달이므로 데이터 로드 유효성만 정의한다.

## 2. 전제조건
- DLG-C010 열림

## 3. 다이어그램

```mermaid
flowchart TD
  LoadData[데이터 로드] --> CheckData{강사 데이터 존재?}
  CheckData -->|": 존재"|RenderOK[정상 렌더링]
  CheckData -->|": 없음 404"|NotFound[toast 강사 정보를 찾을 수 없습니다.\n모달 닫힘]
```
