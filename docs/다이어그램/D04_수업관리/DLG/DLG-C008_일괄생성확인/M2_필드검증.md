## 1. 목적
DLG-C008은 확인 모달이므로 SCR-C003에서 이미 검증된 데이터를 요약 표시한다.
추가 검증은 최소화하며 확인 단계의 체크만 정의한다.

## 2. 전제조건
- SCR-C003 유효성 통과 후 DLG-C008 열림
