## 1. 목적
SCR-C001의 필터(강사/수업명/정원상태/분류), 뷰 전환(월/주/일/목록), 날짜 네비게이션의 조합 플로우를 정의한다.

## 2. 전제조건
- SCR-C001 진입, 캘린더 데이터 로드 완료

## 3. 다이어그램

```mermaid
flowchart TD
  CAL[SCR-C001 캘린더 활성]

  subgraph 뷰전환
    CAL --> ViewSelect{뷰 선택}
    ViewSelect -->|": 월"|Month[\n날짜셀 수업건수 배지]
    ViewSelect -->|": 주"|Week[\n30분 슬롯 06~22시]
    ViewSelect -->|": 일"|Day[\]
    ViewSelect -->|": 목록"|List[\n주단위 리스트]
  end

  subgraph 날짜네비게이션
    CAL --> NavBtn{날짜 이동}
    NavBtn -->|": 오늘"|Today[오늘 날짜 포커스]
    NavBtn -->|": 이전"|Prev[이전 기간 이동]
    NavBtn -->|": 다음"|Next[다음 기간 이동]
  end

  subgraph 필터바
    CAL --> FilterInst[강사 필터 Select\n전체/강사목록]
    CAL --> FilterName[수업명 필터 Select\n전체/수업목록]
    CAL --> FilterCap[정원상태 ButtonGroup\n전체/여유/마감]
    CAL --> FilterCat[분류 MultiSelect Pill\n상담/OT/체성분/방문/수업/PT/기타]
  end

  FilterInst --> Combine{필터 조합}
  FilterName --> Combine
  FilterCap --> Combine
  FilterCat --> Combine

  Combine -->|": 결과 있음"|ReRender[캘린더 이벤트\n재렌더링]
  Combine -->|": 결과 없음"|EmptyCal[해당 필터 이벤트 없음\n빈 캘린더]

  subgraph 우측패널
    CAL --> PanelNav{우측 패널 날짜}
    PanelNav -->|": 이전"|PanelPrev[이전 날짜 수업목록]
    PanelNav -->|": 다음"|PanelNext[다음 날짜 수업목록]
    PanelNav -->|": 해당날짜 수업없음"|PanelEmpty[해당 날짜에\n수업이 없습니다.]
  end

  classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
  classDef system fill:#EDE7F6,stroke:#5E35B2
  classDef info fill:#E0F7FA,stroke:#00838F
  class CAL,Month,Week,Day,List screen
  class Combine,ReRender,FilterInst,FilterName,FilterCap,FilterCat system
  class EmptyCal,PanelEmpty info
```
