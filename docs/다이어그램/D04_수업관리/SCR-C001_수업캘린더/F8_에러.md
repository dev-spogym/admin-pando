## 1. 목적
SCR-C001에서 발생 가능한 에러코드별 분기, 재시도/복구 경로를 정의한다.

## 2. 전제조건
- SCR-C001 진입 또는 인터랙션 도중

## 4. 엣지 설명

| 에러 유형 | 에러코드 | 동작 | 복구 경로 |
|-----------|---------|------|-----------|
| 데이터 로드 실패 | 500 | 에러 토스트 | 재시도 |
| 세션 만료 | 401 | 로그인 이동 | 재로그인 |
| 드래그 충돌 | 409 | 경고 토스트 + 원위치 복원 | - |
| 드래그 권한 없음 | 403 | 경고 토스트 + 원위치 복원 | - |
| 과거 수업 수정 | - | 경고 토스트 | - |
| 2시간 이내 수정 | - | 경고 토스트 | - |
