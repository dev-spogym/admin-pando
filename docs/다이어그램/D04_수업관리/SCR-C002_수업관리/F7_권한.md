## 2. 전제조건
- 로그인 완료

## 3. 다이어그램

```mermaid
flowchart LR

  RoleCheck -->|": primary/owner"|FullAccess[슈퍼관리자/센터장\n수업 등록/수정/삭제\n일괄변경\n노쇼/취소정책 설정\n엑셀 다운로드]

  RoleCheck -->|": manager"|ManagerAccess[매니저\n수업 등록/수정/삭제\n일괄변경\n노쇼/취소정책 설정\n엑셀 다운로드]

  RoleCheck -->|": fc 트레이너"|FCAccess[트레이너\n본인 수업 수정/삭제만\n일괄변경 불가\n조회/상세/서명 가능]
```
