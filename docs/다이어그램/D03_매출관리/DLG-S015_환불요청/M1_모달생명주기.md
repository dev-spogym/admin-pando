## 1. 목적
DLG-S015 부분환불 요청 모달(🆕)의 열기/닫기 생명주기를 표현한다.

## 2. 전제조건
- SCR-S012에서 부분환불 버튼 클릭
- 또는 DLG-S014에서 추가환불 클릭

## 3. 다이어그램

```mermaid
flowchart TD
    CLOSED([모달 닫힘]) -->|": SCR-S012 부분환불 클릭"|LOAD[결제 정보 로드\n잔여 환불 가능금액 조회]
    CLOSED -->|": DLG-S014 추가환불 클릭"|LOAD

    LOAD --> OPEN[DLG-S015 열림 🆕\n440px × 480px\n부분환불 입력 폼]
    LOAD --> ERR_TOAST[에러 토스트]

    OPEN -->|": 환불요청 제출"|VALIDATE[입력 검증]
    OPEN -->|": X"|CLOSED
    OPEN -->|": 취소"|CLOSED

    VALIDATE --> PROCESS[부분환불 처리]
    VALIDATE --> ERR_INLINE[인라인 오류]

    PROCESS --> SUCCESS([성공 → DLG-S014 표시])
    PROCESS --> ERR_TOAST2[에러 토스트]

    classDef newFeature fill:#F3E5F5,stroke:#7B1FA2,color:#4A148C,stroke-dasharray:4 4
    classDef system fill:#EDE7F6,stroke:#5E35B2
    classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
    classDef error fill:#FFEBEE,stroke:#C62828,color:#B71C1C

    class OPEN newFeature
    class LOAD,VALIDATE,PROCESS system
    class SUCCESS success
    class ERR_TOAST,ERR_INLINE,ERR_TOAST2 error
```
