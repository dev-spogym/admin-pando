## 1. 목적
POS 판매의 Happy Path — 상품 선택 → 장바구니 담기 → 구매자 검색 → 결제하기 흐름. 성공/검증실패/시스템에러 3갈래 분기 포함.

## 2. 전제조건
- SCR-S002 진입 완료, 상품 목록 로드됨
