## 1. 목적
SCR-S002의 카테고리 탭 필터, 상품명 검색, 상품 정렬 흐름을 표현한다.

## 2. 전제조건
- SCR-S002 진입 완료, 상품 목록 로드됨

## 3. 다이어그램

```mermaid
flowchart TD
    S002[SCR-S002 POS 판매]

    S002 --> TAB_FILTER[카테고리 탭 필터\n이용권/PT/GX/기타]
    S002 --> SEARCH_BOX[상품명 검색\n실시간 매칭]

    TAB_FILTER -->|": 이용권 클릭"|CAT_MEMBER[이용권 상품만 표시]
    TAB_FILTER -->|": PT 클릭"|CAT_PT[PT 상품만 표시]
    TAB_FILTER -->|": GX 클릭"|CAT_GX[GX 상품만 표시]
    TAB_FILTER -->|": 기타 클릭"|CAT_ETC[기타 상품만 표시]

    SEARCH_BOX -->|": 입력"|SEARCH_FILTER[실시간 필터\n상품명 포함 검색]
    SEARCH_FILTER -->|": 결과 있음"|GRID_UPDATE[상품 그리드 업데이트]
    SEARCH_FILTER -->|": 결과 없음"|GRID_EMPTY[해당 카테고리에 상품이 없습니다.]

    CAT_MEMBER --> SORT_APPLY[정렬 적용\n등록일 내림차순 기본\n품절 상품 하단]
    CAT_PT --> SORT_APPLY
    CAT_GX --> SORT_APPLY
    CAT_ETC --> SORT_APPLY
    GRID_UPDATE --> SORT_APPLY

    SORT_APPLY --> GRID_RENDER[그리드 렌더링\n2~4열 반응형]

    classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
    classDef system fill:#EDE7F6,stroke:#5E35B2
    classDef warning fill:#FFF8E1,stroke:#F9A825,color:#F57F17
    classDef info fill:#E0F7FA,stroke:#00838F

    class S002 screen
    class SEARCH_FILTER,SORT_APPLY system
    class GRID_EMPTY warning
    class GRID_UPDATE,GRID_RENDER,CAT_MEMBER,CAT_PT,CAT_GX,CAT_ETC info
```

## 4. 엣지 설명

| 출발 | 도착 | 설명 |
|------|------|------|
| TAB_FILTER | CAT_MEMBER | 이용권 탭 클릭 |
| SEARCH_BOX | SEARCH_FILTER | 실시간 검색 |
| SEARCH_FILTER | GRID_UPDATE | 검색 결과 표시 |
| SEARCH_FILTER | GRID_EMPTY | 결과 없음 |
