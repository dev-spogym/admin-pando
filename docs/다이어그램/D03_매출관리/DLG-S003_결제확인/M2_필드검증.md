## 2. 전제조건

## 3. 다이어그램

```mermaid
flowchart TD
    OPEN[DLG-S003 열림] --> VALIDATE{결제 정보 검증}

    VALIDATE -->|": 장바구니 유효"|AMOUNT_CHECK{금액 검증}
    VALIDATE -->|": 장바구니 비어있음"|ERR_CART[장바구니 오류\n상품을 선택해주세요.]

    PAYMENT_CHECK -->|": 수단 선택됨"|CONFIRM_READY[확인 버튼 활성]
    PAYMENT_CHECK -->|": 수단 미선택"|ERR_METHOD[결제수단 미선택 오류]

    classDef system fill:#EDE7F6,stroke:#5E35B2
```
