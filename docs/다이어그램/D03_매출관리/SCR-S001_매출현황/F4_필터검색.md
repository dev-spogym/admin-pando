## 1. 목적
SCR-S001의 필터, 검색, 정렬, 페이지네이션 각 액션과 조합 시나리오를 표현한다.

## 2. 전제조건
- SCR-S001 진입 완료, 기본 필터: 이번달, TAB-001

## 3. 다이어그램

```mermaid
flowchart TD
    S001[SCR-S001 매출 현황]

    S001 --> FILTER_DATE[날짜 범위 변경\n시작일/종료일 선택]
    S001 --> FILTER_PRESET[기간 프리셋\n오늘/이번주/이번달]
    S001 --> FILTER_TYPE[유형 멀티셀렉트\n이용권/PT/상품/기타]
    S001 --> FILTER_STATUS[상태 멀티셀렉트\n완료/환불/미납]
    S001 --> SEARCH_INPUT[검색어 입력\n구매자명 또는 상품명]
    S001 --> SORT_COL[컬럼 헤더 클릭\n정렬 토글]
    S001 --> PAGE_NAV[페이지 네비게이션\n 2 3 ... N >]
    S001 --> PAGE_SIZE[표시 건수 변경\n20/50/100]

    FILTER_DATE -->|": 날짜 변경"|APPLY_FILTER[필터 적용\ 재계산]
    FILTER_PRESET --> APPLY_FILTER
    FILTER_TYPE --> APPLY_FILTER
    FILTER_STATUS --> APPLY_FILTER

    SEARCH_INPUT -->|": 300ms debounce"|SEARCH_PROC[검색 처리\n구매자명/상품명 포함 검색]
    SEARCH_PROC -->|": 결과 있음"|TABLE_UPDATE[테이블 업데이트\n검색 결과 표시]
    SEARCH_PROC -->|": 결과 없음"|EMPTY[빈 데이터 상태\n매출 데이터가 없습니다.]

    APPLY_FILTER -->|": 결과 있음"|TABLE_UPDATE
    APPLY_FILTER -->|": 결과 없음"|EMPTY

    SORT_COL -->|": 구매일 클릭"|SORT_TOGGLE[정렬 토글\n오름/내림차순]
    SORT_COL -->|": 금액 클릭"|SORT_TOGGLE
    SORT_COL -->|": 상태 클릭"|SORT_TOGGLE

    PAGE_NAV -->|": 페이지 이동"|PAGE_LOAD[해당 페이지 데이터 표시]
    PAGE_SIZE -->|": 건수 변경"|PAGE_RESET[1페이지로 초기화 + 재표시]

    TABLE_UPDATE --> SUMMARY_BAR[하단 요약바 업데이트\n총매출/순매출/현금/카드/...]

    classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1

    classDef warning fill:#FFF8E1,stroke:#F9A825,color:#F57F17
    classDef info fill:#E0F7FA,stroke:#00838F

    class S001 screen
    class APPLY_FILTER,SEARCH_PROC,SORT_TOGGLE system
    class EMPTY warning
    class TABLE_UPDATE,SUMMARY_BAR,PAGE_LOAD,PAGE_RESET info
```

## 4. 엣지 설명

| 출발 | 도착 | 설명 | |---------|------|------|------| | | FILTER_DATE | APPLY_FILTER | 날짜 범위 변경 시 자동 필터 | | | FILTER_PRESET | APPLY_FILTER | 기간 프리셋 버튼 클릭 | | | SEARCH_INPUT | SEARCH_PROC | 300ms debounce 후 검색 | | | SEARCH_PROC | TABLE_UPDATE | 검색 결과 있음 | | | SEARCH_PROC | EMPTY | 검색 결과 없음 | | | APPLY_FILTER | EMPTY | 필터 결과 없음 | | | SORT_COL | SORT_TOGGLE | 구매일 정렬 토글 | | | PAGE_NAV | PAGE_LOAD | 페이지 이동 | | | PAGE_SIZE | PAGE_RESET | 표시 건수 변경 → 1페이지 초기화 |
