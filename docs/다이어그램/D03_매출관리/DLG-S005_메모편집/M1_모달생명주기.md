## 1. 목적
DLG-S005 메모편집 모달의 열기/닫기 생명주기를 표현한다.

## 2. 전제조건
- SCR-S008 미수금관리에서 메모 버튼 클릭

## 3. 다이어그램

```mermaid
flowchart TD
    CLOSED([모달 닫힘]) -->|": 메모 버튼 클릭"|LOAD[기존 메모 로드\]
    LOAD --> OPEN[모달 열림\n400px × 360px\n메모 텍스트 표시]
    LOAD -->|": 메모 없음"|OPEN_EMPTY[모달 열림\n빈 텍스트 영역]

    OPEN -->|": 저장 버튼"|SAVE[동작]
    OPEN_EMPTY -->|": 저장 버튼"|SAVE
    OPEN -->|": X 버튼"|CLOSED
    OPEN_EMPTY --> CLOSED

    SAVE -->|": 저장 성공"|CLOSED
    SAVE -->|": 저장 실패"|ERR_TOAST[에러 토스트]

    classDef modal fill:#FFF3E0,stroke:#F57C00,color:#E65100
    classDef system fill:#EDE7F6,stroke:#5E35B2
    classDef error fill:#FFEBEE,stroke:#C62828,color:#B71C1C

    class OPEN,OPEN_EMPTY modal
    class LOAD,SAVE system
    class ERR_TOAST error
```

## 4. 엣지 설명

| 출발 | 도착 | 설명 | |---------|------|------|------| | | CLOSED | LOAD | 메모 버튼 클릭 → 기존 메모 로드 | | | OPEN | SAVE | 저장 버튼 클릭 | | | SAVE | CLOSED | 저장 성공 → 닫힘 | | | SAVE | ERR_TOAST | 저장 실패 |
