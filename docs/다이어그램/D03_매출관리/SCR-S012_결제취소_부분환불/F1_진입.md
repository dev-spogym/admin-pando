## 1. 목적
SCR-S012 결제취소/부분환불 화면으로의 진입 경로와 초기 데이터 로드 흐름을 표현한다.

## 2. 전제조건
- 로그인 완료
- 역할: primary, owner, manager (trainer/fc는 제한적 접근)
