## 1. 목적
SCR-S012에서 발생하는 모든 토스트/피드백 메시지의 발생 조건을 표현한다.

## 2. 전제조건
- SCR-S012 진입 완료

## 3. 다이어그램

```mermaid
flowchart TD
    S012[SCR-S012 결제취소/부분환불]

    S012 --> EVT_CANCEL_OK[전체취소 성공]
    S012 --> EVT_PARTIAL_OK[부분환불 성공 🆕]
    S012 --> EVT_APPROVAL_SENT[승인 요청 발송]
    S012 --> EVT_REJECTED[취소 반려]

    EVT_CANCEL_OK --> TOAST_S_CANCEL((성공 토스트\n결제가 취소되었습니다.))
```
