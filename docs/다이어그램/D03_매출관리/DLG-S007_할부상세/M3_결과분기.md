## 1. 목적
DLG-S007에서 마지막 회차 납입 완료 시 완납 처리 분기를 표현한다.

## 2. 전제조건
- DLG-S007에서 납입 처리 완료

## 3. 다이어그램

```mermaid
flowchart TD
    PAY_DONE[납입 처리 완료] --> LAST_CHECK{마지막 회차?}

    LAST_CHECK -->|": 아님"|PARTIAL_DONE[부분 납입 완료\n스케줄 갱신]
    LAST_CHECK -->|": 마지막 회차"|FULL_DONE[완납 처리\ninstallments 상태 COMPLETED]

    FULL_DONE --> NOTIFY[완납 알림 발송\/앱 푸시]
    FULL_DONE --> CLOSED([모달 닫힘\-S009 목록 갱신])

    classDef newFeature fill:#F3E5F5,stroke:#7B1FA2,color:#4A148C,stroke-dasharray:4 4
    classDef system fill:#EDE7F6,stroke:#5E35B2
    classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20

    class FULL_DONE newFeature
    class LAST_CHECK system
    class PARTIAL_DONE,NOTIFY success
```
