## 1. 목적
SCR-S003에서 역할별 접근 제한을 표현한다.

## 2. 전제조건
- 로그인 완료

## 3. 다이어그램

```mermaid
flowchart LR
    AUTH{역할 분기}

    AUTH -->|": primary"|FULL[전체 접근\n할인 적용 가능]
    AUTH -->|": trainer"|TRAINER[접근 가능\n할인 적용 불가]
    AUTH -->|": readonly"|BLOCKED[접근 차단]

    FULL --> ACT_FULL[모든 결제수단\n할인 정책 선택\n결제 실행]
    TRAINER --> ACT_TRAINER[모든 결제수단\n할인 UI 비활성\n결제 실행]
    BLOCKED --> ACT_BLOCK[접근 차단]

    classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
    classDef system fill:#EDE7F6,stroke:#5E35B2
    classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
    classDef warning fill:#FFF8E1,stroke:#F9A825,color:#F57F17
    classDef rbacBlocked fill:#F5F5F5,stroke:#9E9E9E,color:#616161

    class AUTH system
    class ACT_FULL success
    class ACT_TRAINER warning
    class ACT_BLOCK rbacBlocked
```
