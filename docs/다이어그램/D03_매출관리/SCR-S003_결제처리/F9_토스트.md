## 1. 목적
SCR-S003에서 발생하는 모든 토스트/피드백 메시지 발생 조건을 표현한다.

## 2. 전제조건
- SCR-S003 진입 완료

## 3. 다이어그램

```mermaid
flowchart TD
    S003[SCR-S003 결제 처리]

    S003 --> EVT_MILEAGE[마일리지 적립]
    S003 --> EVT_PAY_ERR[결제 실패]
    S003 --> EVT_MILEAGE_ERR[마일리지 차감 실패]
```
