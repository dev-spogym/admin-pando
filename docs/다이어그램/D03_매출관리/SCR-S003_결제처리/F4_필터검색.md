## 2. 전제조건
- SCR-S003 진입 완료

## 3. 다이어그램

```mermaid
flowchart TD
    S003[SCR-S003 결제 처리]

    S003 --> MEMBER_SEARCH[회원 검색 입력\n이름 또는 전화번호]
    S003 --> DISCOUNT_SELECT[할인 정책 선택 🆕\ndiscount_policies 드롭다운]
```
