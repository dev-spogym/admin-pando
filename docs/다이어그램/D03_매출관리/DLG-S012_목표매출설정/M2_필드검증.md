## 1. 목적
DLG-S012 목표 금액 입력 필드 검증 규칙을 표현한다.

## 2. 전제조건
- DLG-S012 열림 상태

## 3. 다이어그램

```mermaid
flowchart TD
    INPUT[목표금액 입력] --> VALIDATE{금액 검증}

    VALIDATE -->|": 금액"|MONTH_CHECK{대상 월 선택 확인}
    VALIDATE -->|": 0 이하"|ERR_ZERO[0원 불가]
    VALIDATE -->|": 100억 초과"|ERR_OVER[최대 금액 초과]

    MONTH_CHECK -->|": 선택됨"|VALID[저장 버튼 활성]
    MONTH_CHECK -->|": 미선택"|ERR_MONTH[대상 월 선택 필수]

    classDef system fill:#EDE7F6,stroke:#5E35B2
    classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
    classDef error fill:#FFEBEE,stroke:#C62828,color:#B71C1C

    class VALIDATE,MONTH_CHECK system
    class VALID success
    class ERR_ZERO,ERR_OVER,ERR_MONTH error
```
