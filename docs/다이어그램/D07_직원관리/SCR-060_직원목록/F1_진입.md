## 1. 목적

SCR-060 직원 목록 화면에 진입할 수 있는 모든 경로를 명세한다. 진입 TC의 Given 조건 원천.

## 2. 전제조건

- 사용자가 로그인 상태이다.
- 세션이 유효하다.
- 접근 가능 역할: primary(슈퍼관리자), owner(최고관리자), manager(센터장). 매니저/FC/스태프는 접근 불가.
