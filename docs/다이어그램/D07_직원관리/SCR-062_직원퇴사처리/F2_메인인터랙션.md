## 1. 목적

SCR-062 4단계 위자드 전체 흐름. FC/비FC 분기 포함. 성공/검증실패/시스템에러 3갈래 분기 강제.

## 2. 전제조건

- SCR-062 진입 완료, 직원 목록 로드 완료 상태이다.
