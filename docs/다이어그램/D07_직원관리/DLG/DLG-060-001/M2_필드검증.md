## 3. 다이어그램

```mermaid
flowchart TD
  INPUT([확인 텍스트 입력])
  INPUT -->|blur|VALIDATE{입력값 검증}

  VALIDATE -->|"퇴사처리" 일치|VALID[확인 버튼 활성화]
  VALIDATE -->|빈 값|ERR_EMPTY[인라인 에러\n확인 텍스트를 입력하세요.]
  VALIDATE -->|불일치|ERR_MISMATCH[인라인 에러\n"퇴사처리"를 정확히 입력하세요.]

  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
  classDef error fill:#FFEBEE,stroke:#C62828,color:#B71C1C
  classDef system fill:#EDE7F6,stroke:#5E35B2

  class VALID success
  class ERR_EMPTY,ERR_MISMATCH error
  class VALIDATE system
```
