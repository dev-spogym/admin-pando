## 도메인 특이사항

- **근태 의존 급여**: 근태(체크인/아웃) 데이터가 급여 자동계산(A9)의 입력값
- **상태(S8)**: EmployeeStatus — ACTIVE / ON_LEAVE / RESIGNED / TRANSFERRED / LOCKED
- **명세서 발송**: 이메일/카카오톡 이중 채널 (X08 시퀀스 참조)
- **퇴사 위자드**: FC 역할은 담당 회원 재배정 Step 2 추가
- **🆕 미구현**: SCR-066 인사평가, SCR-067 교육이력, SCR-068 복리후생, SCR-069 계약서 관리
