## 목적
센터 설정 화면(SCR-080)으로 진입 가능한 모든 경로를 정의한다.

## 전제조건
- 로그인 상태
- 역할: primary(슈퍼관리자) 또는 owner(센터장)

## 다이어그램

```mermaid
flowchart TD
    SB([사이드바 > 설정 > 센터 설정]) --> Auth{권한 확인}
    URL([직접 URL: /settings]) --> Auth
    NAV([타 화면 설정 링크]) --> Auth
    NOTIF([알림 센터 설정 링크]) --> Auth

    Auth -->|": primary 또는 owner"|SCR080[SCR-080 센터 설정\n기본정보 탭 활성]
    Auth -->|": manager 이하"|Blocked[권한없음\n/settings 접근 불가]
    Auth -->|": 비로그인"|Login[SCR: 로그인 화면\n리다이렉트]

    Blocked --> Toast_Block((권한없음 토스트))
    Login --> LoginPage[/login?redirect=/settings]
```

| 타 화면 링크 | Auth | 설정 바로가기 클릭 |

## TC 후보
- TC-080-001: primary 로그인 → /settings 접근 → 기본정보 탭 로드
- TC-080-NEG-001: manager 로그인 → /settings 직접 접근 → 권한없음 토스트
- TC-080-NEG-002: 비로그인 → /settings 접근 → /login 리다이렉트
