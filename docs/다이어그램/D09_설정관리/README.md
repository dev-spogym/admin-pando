## 도메인 특이사항

- **접근 권한**: 슈퍼관리자~센터장만 접근 (매니저 이하 차단). F7 권한 분기에서 primary/owner 중심으로 상세화.
- **센터 설정 (SCR-080)**: 4개 탭(기본정보/알림/테마/물품), isDirty 상태 관리, 탭 전환 미저장 경고(DLG-080-001)
- **권한 설정 (SCR-081)**: 6역할 × 9그룹 × 22메뉴 × 5권한 매트릭스. primary는 수정 불가. 충돌 검증 3규칙.
- **키오스크 설정 (SCR-082)**: RFID/QR/전화번호/얼굴인식 체크인 방식, 표시/음성/보안 섹션. D11 통합운영과 연결.
- **IoT/출입 관리 (SCR-083)**: 게이트 장비, 출입 정책, CCTV, 환경 센서. D11 통합운영 연결.
- **구독/결제 관리 (SCR-084)**: Starter/Professional/Enterprise 플랜, 사용량 게이지, 결제 이력.
- **공지사항 관리 (SCR-085)**: 등록/고정/알림 발송. 전파는 X15 시퀀스 연결.
- **출석 관리 (SCR-086)**: 수동 체크인/아웃, RFID/QR/수동 방식, 날짜/유형 필터.
