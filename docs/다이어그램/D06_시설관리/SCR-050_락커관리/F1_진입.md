# F1 진입 플로우 — SCR-050 락커 관리

## 1. 목적
락커 관리 화면(`/locker`)으로 진입 가능한 모든 경로를 정의하여 진입 TC의 원천으로 활용한다.

## 2. 전제조건
- 로그인 상태
- 역할: 슈퍼관리자/센터장/매니저/스태프 (FC·코치는 접근 불가)
