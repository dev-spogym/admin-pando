# F7 권한(RBAC) 분기 플로우 — SCR-051 사물함 배정 관리

## 1. 목적
6개 역할별 접근·배정·해제 권한 분기를 정의한다.

## 2. 다이어그램

```mermaid
flowchart LR
    Login([로그인 사용자]) --> RoleCheck{역할 확인}

    RoleCheck -->|": primary/owner"|RolePO[슈퍼관리자/센터장\n전체 권한]
    RoleCheck -->|": manager"|RoleMgr[매니저\n배정·해제 가능]
    RoleCheck -->|": staff"|RoleStaff[스태프\n배정·해제 가능]
    RoleCheck -->|": fc/coach"|RoleFC[FC/코치\n접근 불가]
    RoleCheck -->|": readonly"|RoleRO[읽기전용\n조회만]

    RolePO --> A_FULL[배정O 해제O 추가O 동기화O 엑셀O]
    RoleMgr --> A_MGR[배정O 해제O 동기화O 엑셀O]
    RoleStaff --> A_STAFF[배정O 해제O 엑셀O]
    RoleFC --> A_BLOCKED[접근X → 403]
    RoleRO --> A_RO[조회O\n배정·해제 버튼 disabled]

    classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
    classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1

    class A_FULL success
    class A_MGR,A_STAFF screen
    class A_RO warning
    class A_BLOCKED error
    class RoleCheck system
```
