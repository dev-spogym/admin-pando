## 도메인 특이사항

- **락커 상태전이**: S11 LockerStatus (`docs/다이어그램/20_상태전이도/20-11_LockerStatus`) 연동
- **자동화 연계**: A03 이용권 자동만료, A04 홀딩자동연장 (`docs/다이어그램/40_자동화_크론/`)
- **IoT 연계**: 밴드/카드(SCR-052) → D11 통합운영 IoT 연동
- **크론**: 락커 만료임박 알림 A01, 만료 자동처리 A03
- **🆕 장비점검(SCR-056)**: D-7 점검 예정 알림, 보증만료 D-30 알림, 장비별 점검이력 추적
- **🆕 소모품재고(SCR-057)**: 안전재고 미달 자동 발주 알림, 재고 상태 자동 판정(정상/주의/부족/품절)
- **🆕 청소스케줄(SCR-058)**: 체크리스트 전체 완료 시 자동 완료 전환, 구역별 청소이력 관리
