# E14 — 파일 업로드 실패

## 1. 개요

| 항목 | 내용 |
|------|------|
| 에러코드 | E400003 / E500001 (파일 처리) |
| HTTP | 400 / 500 |
| 발생 모듈 | 회원(프로필/체성분), 전자계약, 공지(첨부) |
| 영향 화면 | SCR-M002 회원 등록, SCR-M006 체성분, SCR-075 전자계약 |

## 2. 발생 조건

- 파일 크기 초과 (E400003)
- 허용되지 않는 파일 형식
- 서버 저장 실패
- 네트워크 단절 중 업로드
- 악성 파일 감지

## 4. 복구/재시도 전략

| 상황 | 전략 |
|------|------|
| 크기/형식 오류 | 클라이언트에서 사전 차단, 안내 메시지 |
| 서버 오류 | 재시도 버튼, 최대 3회 |
| 악성 파일 | 즉시 차단, 보안 경고 |
| 네트워크 단절 | 네트워크 복구 후 재시도 |

## 5. 사용자 노출 메시지

| 상황 | 메시지 |
|------|--------|
| E400003 크기 초과 | 허용된 범위를 초과했습니다 (최대 {N}MB) |
| 형식 오류 | 허용된 파일 형식이 아닙니다 (JPG, PNG, PDF) |
| 악성 파일 | 보안 위협이 감지되어 업로드가 차단되었습니다 |
| 서버 오류 | 파일 업로드 중 오류가 발생했습니다. 다시 시도해주세요 |
