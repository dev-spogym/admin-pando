# M1 생명주기 플로우 — DLG-002 이탈 경고

## 목적
폼 편집 중 이탈 시도 → 경고 모달 → 이탈/유지 분기 생명주기를 정의한다.

## 다이어그램

```mermaid
flowchart TD
    TRIGGER([이탈 시도 감지\n뒤로가기/메뉴클릭/탭닫기]) --> DIRTY_CHECK{미저장 변경사항 있음?}

    DIRTY_CHECK -->|": 없음"|ALLOW_NAV[이탈 허용\n바로 이동]
    DIRTY_CHECK -->|": 있음"|OPEN[DLG-002 이탈 경고 모달 열림]
```
