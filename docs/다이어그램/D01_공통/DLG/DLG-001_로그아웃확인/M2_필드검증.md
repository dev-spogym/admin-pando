# M2 필드검증 플로우 — DLG-001 로그아웃 확인

## 목적
로그아웃 확인 모달은 입력 필드가 없으므로 검증 로직 없음. 미저장 변경사항 경고 조건을 정의한다.

## 다이어그램

```mermaid
flowchart TD
    OPEN([DLG-001 열림]) --> CHECK{미저장 변경사항 확인}

    CHECK -->|": 미저장 없음"|NORMAL_MODAL[일반 로그아웃 확인 모달]
    CHECK -->|": 미저장 있음"|WARN_MODAL[경고 문구 추가\n저장되지 않은 변경사항이 있습니다]

    NORMAL_MODAL -->|": 확인"|LOGOUT[로그아웃]
    WARN_MODAL -->|": 확인 (저장 없이)"|LOGOUT
    WARN_MODAL -->|": 취소"|STAY[현재 화면 유지]

    classDef modal fill:#FFF3E0,stroke:#F57C00,color:#E65100
```
