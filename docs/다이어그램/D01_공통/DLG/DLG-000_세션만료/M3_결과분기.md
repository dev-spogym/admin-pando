# M3 결과분기 플로우 — DLG-000 세션만료 모달

## 목적
세션만료 모달 액션 결과에 따른 분기(재로그인/닫기)와 로그인 후 원래 경로 복원을 정의한다.

## 다이어그램

```mermaid
flowchart TD
    DLG_000[DLG-000 세션만료 모달] --> ACTION{사용자 액션}

    ACTION -->|": 재로그인 버튼"|SAVE_REDIRECT[현재 경로 저장\]
    ACTION -->|": 닫기 버튼"|CLEAR_ONLY[세션 클리어만\n경로 저장 없음]

    SAVE_REDIRECT --> SCR_100[SCR-100 로그인]
    CLEAR_ONLY --> SCR_100

    SCR_100 -->|": 로그인 성공 + 있음"|RESTORE[저장된 경로로 복원]
    SCR_100 -->|": 로그인 성공 + 없음"|DASH[SCR-101 대시보드]

    RESTORE --> ORIGINAL[원래 화면 표시]
```
