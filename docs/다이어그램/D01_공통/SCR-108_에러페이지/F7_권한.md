# F7 권한(RBAC) 분기 플로우 — SCR-108 에러 페이지

## 목적
에러 페이지는 역할 무관 표시되며, 복구 버튼 목적지가 역할별로 분기된다.

```mermaid
flowchart LR

    ROLE -->|": 모든 역할"|ALL[에러 페이지 공통 표시]

    ALL -->|": 403 에러 복구"|RECOVER_403{역할별 복구 목적지}
    RECOVER_403 -->|": "|DASH_FULL[SCR-101 대시보드]
    RECOVER_403 -->|": fc/staff"|DASH_LIMITED[SCR-101 대시보드 (제한 뷰)]

    ALL -->|": 500/503 에러 복구"|RECOVER_500[SCR-101 대시보드]

    classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
```
