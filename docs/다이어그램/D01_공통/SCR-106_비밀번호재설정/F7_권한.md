# F7 권한(RBAC) 분기 플로우 — SCR-106 비밀번호 재설정

## 목적
비밀번호 재설정은 모든 역할이 자신의 계정에 대해 수행 가능하며, 관리자는 타인 비밀번호 초기화 권한을 갖는다.

## 다이어그램

```mermaid
flowchart LR
    ROLE{역할 확인}

    ROLE -->|": 모든 역할"|SELF_RESET[자신의 비밀번호 재설정\n항상 허용]
    ROLE -->|": "|ADMIN_RESET[타 계정 비밀번호 초기화\n직원 관리 화면에서]

    SELF_RESET --> ACT_SELF[자기 비밀번호 변경 허용]
    ADMIN_RESET --> ACT_ADMIN[타인 비밀번호 초기화 이메일 발송]

    ACT_ADMIN -->|": fc/staff가 타인 초기화 시도"|BLOCKED[권한없음 토스트]

    classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
```
