# F1 진입 플로우 — SCR-107 화면설계서 오버레이

## 목적
화면설계서 오버레이 진입 경로(단축키/헤더 버튼)와 초기 로드를 정의한다.

## 다이어그램

```mermaid
flowchart TD
    TRIGGER([오버레이 진입]) --> OPEN_METHOD{진입 방법}

    OPEN_METHOD -->|": 헤더 설계서 버튼"|OVERLAY[화면설계서 오버레이 열림]
    OPEN_METHOD -->|": 단축키 Shift+D"|OVERLAY
    OPEN_METHOD -->|": URL 파라미터 ?spec=1"|OVERLAY

    OVERLAY --> LOAD{설계서 데이터 로드}
    LOAD -->|": 로딩 중"|STATE_LOADING[스켈레톤 로딩]
    LOAD -->|": 로드 성공"|STATE_NORMAL[설계서 오버레이 표시]
    LOAD -->|": 파일 없음"|STATE_EMPTY[설계서 없음 안내]
    LOAD -->|": 로드 실패"|STATE_ERROR[에러 상태]

    OVERLAY -->|": ESC / 닫기 버튼"|CLOSE[오버레이 닫힘]

    classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
    classDef system fill:#EDE7F6,stroke:#5E35B2

    class OVERLAY,STATE_NORMAL,STATE_LOADING screen
    class OPEN_METHOD,LOAD system
    class STATE_ERROR error
    class STATE_EMPTY,CLOSE rbacBlocked
```
