# F6 상태별 화면 플로우 — SCR-107 화면설계서 오버레이

## 목적
오버레이 로딩/정상/빈/에러 상태별 UI 분기를 정의한다.

## 다이어그램

```mermaid
flowchart TD
    ENTER([오버레이 열림]) --> LOAD{설계서 로드}

    LOAD -->|": 로딩 중"|STATE_LOADING[스켈레톤]
    LOAD -->|": 로드 성공"|STATE_NORMAL[설계서 표시]
    LOAD -->|": 파일 없음"|STATE_EMPTY[설계서 없음\n업로드 안내]
    LOAD -->|": 로드 실패"|STATE_ERROR[에러 + 재시도]

    STATE_NORMAL -->|": 전체화면 진입"|STATE_FULLSCREEN[전체화면 모드]
    STATE_FULLSCREEN -->|": 전체화면 해제"|STATE_NORMAL
    STATE_ERROR -->|": 재시도"|LOAD

    classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
    classDef error fill:#FFEBEE,stroke:#C62828,color:#B71C1C

    class STATE_NORMAL,STATE_LOADING,STATE_FULLSCREEN screen
    class STATE_ERROR error
    class STATE_EMPTY rbacBlocked
    class LOAD system
```
