# F1 진입 플로우 — SCR-103 글로벌 검색

## 목적
글로벌 검색 오버레이 열기 경로(단축키/사이드바/헤더)와 초기 상태를 정의한다.

## 다이어그램

```mermaid
flowchart TD
    TRIGGER([검색 트리거]) --> OPEN_METHOD{열기 방법}

    OPEN_METHOD -->|": Cmd+K / Ctrl+K"|OVERLAY[글로벌 검색 오버레이 열림]
    OPEN_METHOD -->|": 헤더 검색 아이콘 클릭"|OVERLAY
    OPEN_METHOD -->|": 사이드바 검색창 포커스"|OVERLAY

    OVERLAY --> INIT_STATE{초기 상태}
    INIT_STATE -->|": 최근 검색 있음"|RECENT[최근 검색어 표시]
    INIT_STATE -->|": 최근 검색 없음"|EMPTY_HINT[검색 힌트 표시\n회원명, 화면명 등]

    OVERLAY -->|": ESC / 오버레이 외부 클릭"|CLOSE[오버레이 닫힘]

    classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
    classDef system fill:#EDE7F6,stroke:#5E35B2
    classDef rbacBlocked fill:#F5F5F5,stroke:#9E9E9E,color:#616161
    class OVERLAY screen
    class OPEN_METHOD,INIT_STATE,RECENT system
    class EMPTY_HINT,CLOSE rbacBlocked
```
