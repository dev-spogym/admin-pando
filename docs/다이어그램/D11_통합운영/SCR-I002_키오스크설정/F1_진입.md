# F1 진입 플로우 — SCR-I002 키오스크 설정

## 목적
키오스크 설정 화면(`/settings/kiosk`)으로 진입 가능한 모든 경로를 정의한다.

## 전제조건
- 로그인 세션 유효
- 설정 > 키오스크 설정 접근 권한 (superAdmin/primary/owner만 가능)
