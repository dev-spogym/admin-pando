```mermaid
flowchart TD
    SUBMIT[저장 버튼 클릭] --> V_MEMBER{회원 선택 여부}
    V_MEMBER -->|": 미선택"|ERR_MEMBER[회원을 선택하세요]
    V_MEMBER -->|": 선택됨"|V_CHANNEL{채널 선택 여부}

    V_CHANNEL -->|": 미선택"|ERR_CHANNEL[채널을 선택하세요]

    V_MEMBERSHIP -->|": 만료/정지"|WARN_MEMBER[회원권 만료 경고\n강제 등록 가능]
    V_MEMBERSHIP -->|": 유효"|PASS[검증 통과 → API 호출]

    ERR_MEMBER --> FOCUS_MEMBER[회원 검색 필드 포커스]
    ERR_CHANNEL --> FOCUS_CHANNEL[채널 선택 포커스]
```
