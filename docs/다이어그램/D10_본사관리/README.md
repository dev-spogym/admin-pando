## 도메인 개요

본사관리 도메인은 슈퍼관리자(primary) 전용 통합 관제 기능을 제공한다.
지점 드릴다운(본사 대시보드 → 지점 리스트 → 지점 상세), KPI 롤업,
리포트 자동 생성(크론 A06), 전 도메인 감사로그(DF10), 공지 전파(X15)가 핵심이다.

## 도메인 특이사항

- **본사→지점 드릴다운**: SCR-090(지점별) → SCR-092(지점목록) → SCR-091(전지점통합)
- **KPI 롤업**: X09 시퀀스 — 지점 데이터 → 본사 집계 → 주간 리포트 자동 발행(A06)
- **감사로그**: 전 도메인 이벤트 수집 (DF10 연결), 21종 액션 타입
- **슈퍼관리자 전용**: SCR-091, 지점 생성/폐점, 권한 변경 — primary만 접근 (RBAC F7)
- **X15 공지 전파**: 본사 → 전 지점 공지 발행 → 읽음 집계 연결
- **미구현(🆕)**: SCR-101 커스텀대시보드빌더, SCR-102 벤치마크, SCR-103 예측분석, SCR-104 NPS설문
