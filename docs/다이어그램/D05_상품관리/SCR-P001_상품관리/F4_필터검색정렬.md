# F4 필터/검색/정렬 플로우 — SCR-P001 상품 관리

## 목적
필터 조합, 검색, 정렬 조작에 따른 상품 목록 갱신 흐름을 정의한다. 가격 이력/할인 중첩/재고 차감 참조는 F8에서 상세화.
