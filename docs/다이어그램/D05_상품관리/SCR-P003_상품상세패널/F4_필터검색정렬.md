# F4 가격이력/할인중첩/재고차감 상세화 — SCR-P003 상품 상세 패널

## 목적
단일 패널이므로 필터/검색 대신 F4에서 가격이력 추적·할인 중첩·재고 차감 로직을 상세화한다.

## 다이어그램

```mermaid
flowchart TD
    subgraph 가격이력_추적
        SAVE([저장 클릭]) --> PriceChanged{가격 변경됨?}
        PriceChanged -->|": 변경 있음"|AuditLog[\=product\naction=\nbefore/after 가격 기록]
        PriceChanged -->|": 변경 없음"|NoLog[이력 미기록]
        AuditLog --> HistoryView[DLG-P003 가격이력\n최대 20건 타임라인 표시]
    end

    subgraph 할인중첩_연결
        PRODUCT([상품 저장 완료]) --> DiscountLink{할인 정책 연결?}
        DiscountLink -->|": 할인 적용"|DiscountCalc[SCR-P004 할인설정 참조\n정률/정액 중첩 적용\ 결제 시 자동 계산]
        DiscountLink -->|": 없음"|NoDiscount[원가 그대로 판매]
    end

    subgraph 재고차감_연결
        SELL([POS 판매 시]) --> StockCheck{재고 관리 대상?}
        StockCheck -->|": GENERAL/RENTAL 타입"|StockDec[inventory_logs OUT\ 차감\-P007 재고관리 반영]
        StockCheck -->|": MEMBERSHIP/LESSON"|NoStock[재고 차감 없음\n이용권 개시만]
    end

    classDef system fill:#EDE7F6,stroke:#5E35B2
    classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
    classDef info fill:#E0F7FA,stroke:#00838F
    class PriceChanged,DiscountLink,StockCheck system
    class AuditLog,HistoryView,DiscountCalc,StockDec success
    class NoLog,NoDiscount,NoStock info
```

| TC-P003-F4-02 | positive | 가격 변경 없이 저장 | 저장 클릭 | audit_log 미기록 |
