# F4 필터/검색/정렬 + 재고차감 이력 상세화 — SCR-P007 재고 관리 🆕

## 목적
재고 목록의 필터·정렬과 함께, 계획서 요구사항에 따라 **재고차감(inventory_logs)** 이력 조회 흐름을 상세화한다.

## 다이어그램

```mermaid
flowchart TD
```
