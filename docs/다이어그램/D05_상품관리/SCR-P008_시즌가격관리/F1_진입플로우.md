# F1 진입 플로우 — SCR-P008 시즌 가격 관리 🆕

## 목적

## 전제조건
- 사이드바 메뉴 또는 SCR-P001/P003에서 진입
- RBAC: primary/owner/manager만 접근, trainer/front 차단
- 시즌 가격 = 특정 기간(성수기/비수기 등)에 적용되는 별도 단가 규칙
