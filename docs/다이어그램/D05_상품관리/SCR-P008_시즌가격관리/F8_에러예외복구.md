# F8 에러/예외/복구 플로우 — SCR-P008 시즌 가격 관리 🆕

## 목적
시즌 가격 관리 화면의 에러·예외 상황과 복구 경로를 정의하며, **가격이력(audit_log)**, **할인중첩(discount stacking)** 관련 에러를 상세화한다.
