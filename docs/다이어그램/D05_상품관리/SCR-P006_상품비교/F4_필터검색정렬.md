# F4 필터/검색/정렬 + 가격이력/할인중첩/재고차감 — SCR-P006 상품 비교 🆕

## 목적
비교 화면 내 필터·정렬과 함께, 계획서 요구사항에 따라 **가격이력(audit_log)**, **할인중첩(discount stacking)**, **재고차감(inventory_logs)** 연산 흐름을 상세화한다.

## 전제조건
- 비교 테이블 렌더링 완료 상태
- audit_log 테이블: 상품 가격 변경 이력 보관
- discount_rules 테이블: 복수 할인 규칙(우선순위, 중첩 여부)
- inventory_logs 테이블: 재고 차감/복구 트랜잭션
