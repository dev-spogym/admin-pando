# F8 에러/예외/복구 플로우 — SCR-P006 상품 비교 🆕

## 목적
비교 화면에서 발생하는 에러·예외 상황과 복구 경로를 상세화한다.  
계획서 요구사항에 따라 **가격이력(audit_log)**, **할인중첩(discount stacking)**, **재고차감(inventory_logs)** 관련 에러도 포함한다.
