# F2 메인 인터랙션 플로우 — SCR-P006 상품 비교 🆕

## 목적
상품 비교 화면의 핵심 인터랙션(상품 추가/제거, 비교 테이블 갱신, 최저가 하이라이트)을 정의한다.

## 전제조건
- 2~4개 상품이 선택된 상태로 진입
- 비교 항목: 가격(1회당/1일당), 할인 적용가, 재고, 유효기간, 이용권 연동 여부
