## 1. 목적

리드 관리 화면의 정상 Happy Path — 목록 조회, 리드 등록/수정/삭제, 뷰 전환 흐름을 TC 원천으로 제공한다.

## 2. 전제조건

- 로그인 상태 (primary)
- `/` 경로 진입
