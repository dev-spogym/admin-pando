## 1. 목적

SCR-071의 로딩/빈/에러/권한없음/오프라인 상태 분기를 UI 상태 TC 원천으로 제공한다.

## 2. 전제조건

- /message 경로 진입

## 3. 다이어그램

```mermaid
flowchart TD
  ENTER([/message 진입]) --> AUTH{권한 확인}
  AUTH -->|E_071_F6_BLOCKED readonly|STATE_BLOCKED[권한없음]
```
