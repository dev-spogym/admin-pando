### D. 7개 KPI 카드 상세

| # | label | value | icon | change 계산 | changeLabel |
|---|-------|-------|------|-------------|-------------|
| 1 | 전체 회원 | `${formatNumber(totalMembers)}명` | Users | `mom(totalMembers, prevMonthMembers)` | 전월 대비 |
| 2 | 전체 매출 | `${formatAmount(totalRevenue)}원` | DollarSign | `mom(totalRevenue, prevMonthRevenue)` | 이번달 전월 대비 |
| 3 | 전체 출석 | `${formatNumber(todayAttendance)}명` | UserCheck | `mom(todayAttendance, yesterdayAttendance)` | 어제 대비 |
| 4 | 전체 직원 | `${formatNumber(totalStaff)}명` | LayoutDashboard | 0 | 전월 대비 |
| 5 | 이번달 신규 | `${formatNumber(newMembers)}명` | ArrowUpRight | 0 | 이번달 |
| 6 | 만료 예정 (30일) | `${formatNumber(expiringMembers)}명` | Clock | 0 | 30일 이내 |
| 7 | 만료 회원 | `${formatNumber(expiredMembers)}명` | AlertTriangle | 0 | 이번달 |

**MoM 계산 함수**:
```typescript
const mom = (curr: number, prev: number): number =>
  prev > 0 ? Math.round(((curr - prev) / prev) * 100) : 0;
```

### L. 비즈니스 로직

1. **전 지점 통합 집계**: 지점별 병렬 조회 (`Promise.all`) → 합산
2. **전월 회원 수 근사치**: `totalMembers - newMembers + expiredMembers` (정확한 월말 스냅샷 없음)
3. **MoM 성장률**: `((당월 - 전월) / 전월) * 100`, 전월=0이면 0%
4. **금액 포맷**: `formatAmount()` — 1억 이상 → `"X.X억"`, 1만 이상 → `"X만"`

### D. 랜덤 배정 알고리즘

```typescript
// 시드 기반 결정적 셔플 (동일 입력 → 동일 결과)
function seededShuffle<T>(items: T[], seedValue: string): T[] {
  const result = [...items];
  let seed = hashSeed(seedValue);  // 문자열 → 숫자 해시
  for (let i = result.length - 1; i > 0; i -= 1) {
    seed = (seed * 1664525 + 1013904223) >>> 0;
    const j = seed % (i + 1);
    [result[i], result[j]] = [result[j], result[i]];
  }
  return result;
}

// 시드 문자열: "{dateKey}-{branchName}-{taskRole}"
const shuffled = seededShuffle(TASK_POOL, `${dateKey}-${branchName}-${taskRole}`);

// 역할+지점 필터 → 슈퍼관리자 18건, 일반 12건
const filtered = shuffled.filter(task => {
  const roleMatch = taskRole === "hq" ? true : task.assigneeRole === taskRole || task.assigneeRole === "branch";
  const branchMatch = !branchName || branchName === "전체 지점" ? true : task.branchName === branchName;
  return roleMatch && branchMatch;
});
return filtered.slice(0, taskRole === "hq" ? 18 : 12);
```

### L. 비즈니스 로직

1. **로그인 실패 카운트**: `incrementFailCount()` → localStorage에 실패 횟수 저장 → 5회 이상 시 30분 잠금
2. **잠금 자동 해제**: `getLoginLockState()` → `lockedUntil` 시간 경과 시 자동 해제 + failCount 초기화
3. **로그인 성공 시**: `clearFailCount()` → 실패 횟수/잠금 상태 초기화
4. **아이디 저장**: `rememberMe=true` → `localStorage.setItem(REMEMBER_KEY, id)`, 다음 방문 시 자동 복원
5. **로그인 성공 에러 메시지**: `"이메일 또는 비밀번호가 올바르지 않습니다. (${remaining}회 더 실패 시 계정이 잠깁니다)"`
6. **슈퍼관리자 처리**: `isSuperAdmin=true` → branchId 없을 수 있음, 선택한 지점 또는 빈 문자열 사용
7. **authStore 저장**: `authLogin({ id, name, email, role, branchId, branchName, tenantId, isSuperAdmin, currentBranchId }, accessToken)`
8. **로그인 후 이동**: `moveToPage(966)` → 대시보드
