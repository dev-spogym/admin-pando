## 1. 상품 목록 (`/products`)

> 소스: `src/app/products/page.tsx`

### A. 화면 개요

센터에서 판매하는 모든 상품(이용권, PT/GX 수강권, 대여권, 일반 상품)을 마스터-디테일 레이아웃으로 조회하고 관리하는 화면이다. 최상위 탭으로 **상품 목록**(`mainTab === 'products'`)과 **분류 관리**(`mainTab === 'groups'`)를 전환할 수 있다.

- 컴포넌트: `ProductList` (default export)
- 라우트: `/products`
- 레이아웃: `AppLayout` > `PageHeader` + `StatCardGrid` + 필터바 + 마스터-디테일 영역
- 권한: `hasFeature(authUser.role, 'productEdit', authUser.isSuperAdmin)` — 상품 등록/수정 버튼 표시 조건
- 슈퍼관리자 전용: 전 지점 배포 버튼 (`isSuperAdmin`)

패널 열린 상태에서는 요일별 ✓/✗, 시간, 홀딩/양도/포인트, 판매 건수 컬럼이 숨겨진다. 표시 컬럼:

| 상품명 | 타입 배지 + 패키지 배지 + 이름 |

| 카드가 | `formatKRW(cardPrice)` 또는 `-` |
| 기간 | `{duration}일` 또는 `-` |
| 횟수 | `{sessions}회` 또는 `-` |
| 상태 | StatusBadge `사용`/`미사용` |

| 전체 지점 체크박스 | checkbox | `deployAllBranches` 토글 |

**검증:**

- 배포할 상품 0개 → `toast.error('배포할 상품을 선택하세요.')` → 진행 차단
- 대상 지점 0개 → `toast.error('배포할 지점을 선택하세요.')` → 진행 차단

#### G-2. 배포 확인 ConfirmFlow

- 컴포넌트: `ConfirmFlow`
- `isOpen={showDeployConfirm}`
- `isLoading={deploying}`
- steps:
  ```
  title: '전 지점 배포 확인'
  description: `${deploySelectedIds.size}개 상품을 ${지점수}개 지점에 배포합니다.
                이미 같은 이름의 상품이 있는 지점은 건너뜁니다.`
  ```

**배포 로직 (`handleDeploy`):**

1. 대상 지점 목록 결정 (전체 또는 선택). 현재 지점(`getBranchId()`) 제외
2. 선택 상품 목록 결정 (`deploySelectedIds`)
3. 이중 루프: 지점 × 상품
   - 중복 체크: `supabase.from('products').select('id').eq('branchId', branchId).eq('name', prod.name).maybeSingle()`
   - 중복이면 `skipCount++`, 건너뜀
   - 중복 아니면 INSERT (id, branchId, createdAt 제외 복사) → `successCount++`
4. 완료 toast: `배포 완료: ${successCount}건 추가, ${skipCount}건 건너뜀`

#### G-3. 분류 삭제 확인

- `window.confirm('이 분류를 삭제하시겠습니까?')` (네이티브)
- 확인 시 `deleteProductGroup(id)` 호출

### K-1. localStorage 이력 구조

- 키: `discount_policy_history`
- 형태: `HistoryEntry[]` (JSON 문자열)
- 최대 50건 유지 (FIFO — 최신 우선)

### K-2. 이력 저장 함수

```typescript
const HISTORY_KEY = 'discount_policy_history';

function loadHistory(): HistoryEntry[] {
  if (typeof window === 'undefined') return [];
  try {
    return JSON.parse(localStorage.getItem(HISTORY_KEY) ?? '[]') as HistoryEntry[];
  } catch { return []; }
}

function saveHistory(entry: HistoryEntry) {
  const prev = loadHistory();
  const next = [entry, ...prev].slice(0, 50);
  localStorage.setItem(HISTORY_KEY, JSON.stringify(next));
}
```
