## SCR-MA-200. 강사 홈

### 1. 화면 목적

로그인 직후 트레이너가 오늘의 수업 현황과 주요 알림을 한눈에 파악하는 대시보드 화면.

### 2. 접근 경로

- 앱 실행 → 로그인 완료 → 강사 홈 (기본 탭)
- 하단 탭바 > 홈 아이콘

### 3. 역할별 접근

| 역할 | 접근 | 비고 |
|------|------|------|
| trainer | O | 기본 홈 |
| golf_trainer | O | 동일 레이아웃, 수업 유형만 다름 |
| member | X | 회원 홈으로 리다이렉트 |

### 4. 와이어프레임

```
┌─────────────────────────────┐
│  🔔(3)            FitGenie  │  ← 헤더 (알림 배지)

│  2026년 4월 17일 (금)        │  ← 오늘 날짜
├─────────────────────────────┤
```

### 5. 구성 요소 상세

| 요소 | 타입 | 설명 |
|------|------|------|
| 알림 배지 | Badge | 미확인 알림 수 표시, 최대 99+ |
| 인사말 | Text | 강사 이름 + 오늘 날짜 (YYYY년 M월 D일 (요일)) |
| 요약 카드 — 오늘 수업 | KPI Card | 오늘 날짜 기준 전체 수업 건수 |
| 요약 카드 — 출석 | KPI Card | completed 상태 수업의 참석 회원 수 |
| 요약 카드 — 노쇼 | KPI Card | 오늘 노쇼 처리된 회원 수, 빨간색 강조 |
| 요약 카드 — 예약대기 | KPI Card | pending 상태 예약 건수 |
| 수업 타임라인 | List | 오늘 수업을 시간 오름차순 정렬, 각 카드 탭 시 SCR-MA-212로 이동 |
| 수업 카드 상태 색상 | Badge | 예정(파랑) / 진행중(초록) / 완료(회색) / 노쇼(빨강) / 취소(줄그음) |

### 6. 인터랙션

- 알림 아이콘 탭 → SCR-MA-251 알림 화면으로 이동
- 요약 카드 탭 → 해당 필터가 적용된 SCR-MA-211 수업 목록으로 이동
- 수업 카드 탭 → SCR-MA-212 수업 상세로 이동
- Pull-to-refresh → 전체 데이터 재조회

### 7. 상태별 화면

| 상태 | 화면 처리 |
|------|-----------|
| loading | 스켈레톤 UI (카드 영역 shimmer) |
| empty (오늘 수업 없음) | 타임라인 영역에 "오늘 예정된 수업이 없습니다" 안내 텍스트 |
| error | 토스트 메시지 "데이터를 불러오지 못했습니다. 다시 시도해 주세요." + 재시도 버튼 |
| success | 정상 렌더링 |

### 2. 접근 경로

- SCR-MA-210 캘린더 수업 블록/카드 탭
- SCR-MA-211 수업 목록 카드 탭

### 3. 역할별 접근

|------|------|------|

| golf_trainer | △ | SCR-MA-312 쌍방서명 화면으로 대체 |
