## SCR-MA-400. FC 홈

### 1. 화면 목적

FC(Fitness Consultant)가 앱 진입 시 당일 업무 현황을 한눈에 파악하고 주요 액션으로 빠르게 이동하도록 지원한다.

### 2. 접근 경로

- 앱 실행 → 로그인 완료 → 역할: fc → FC 홈 자동 이동
- 하단 탭 바 > [홈] 탭

### 3. 역할별 접근

| 역할 | 접근 | 비고 |
|------|------|------|
| fc | 허용 | 기본 진입 화면 |
| staff | 불가 | SCR-MA-500으로 리다이렉트 |
| member | 불가 | 회원 홈으로 리다이렉트 |
| admin | 불가 | 관리자 화면으로 리다이렉트 |

### 4. 와이어프레임 (ASCII art)

```
┌─────────────────────────────┐
│  FitGenie              🔔 ⚙️ │
│  안녕하세요, 이상담님 👋      │
│  2026년 4월 17일 (금)        │
├─────────────────────────────┤
│  ┌───────────┐ ┌───────────┐│
│  │  오늘 상담  │ │  담당 회원  ││
│  │    3 건    │ │   45 명   ││
│  └───────────┘ └───────────┘│
│  ┌───────────┐ ┌───────────┐│
│  │  만료 예정  │ │  신규 리드  ││
│  │    7 명    │ │   2 건    ││
│  └───────────┘ └───────────┘│
├─────────────────────────────┤
│  오늘 상담 일정               │
│  ─────────────────────────  │
```

### 5. 구성 요소 상세

#### 5-1. 상단 헤더

| 요소 | 내용 |
|------|------|
| 앱명 | FitGenie 로고 텍스트 |
| 인사말 | "안녕하세요, {FC 이름}님 👋" |
| 날짜 | 오늘 날짜 (YYYY년 M월 D일 (요일)) |
| 알림 아이콘 | 미읽음 알림 수 Badge 표시, 탭 시 SCR-MA-451 이동 |
| 설정 아이콘 | 탭 시 SCR-MA-452 이동 |

#### 5-2. 요약 카드 (2x2 그리드)

| 카드명 | 표시 값 | 탭 시 이동 |
|--------|---------|----------|
| 오늘 상담 | 당일 배정된 상담 건수 | SCR-MA-410 (예정 탭) |
| 담당 회원 | 내 담당 활성 회원 수 | SCR-MA-420 |
| 만료 예정 | D-30 이내 만료 예정 회원 수 | SCR-MA-430 |
| 신규 리드 | 미처리 신규 리드 수 | SCR-MA-410 (전체 탭) |

#### 5-3. 오늘 상담 일정 리스트

- 당일 상담 건을 시간 오름차순으로 최대 5건 노출
- 각 행: 시간 | 회원명 | 상담 유형
- [전체 보기] 링크 탭 시 SCR-MA-410 이동

#### 5-4. 만료 임박 알림 영역

- D-7 이내 만료 예정 회원을 만료일 오름차순으로 최대 3건 노출
- 각 행: 회원명 — D-{N} ({만료일})
- [상담 등록하기] 탭 시 SCR-MA-411 (회원 선택 프리필)

#### 5-5. 하단 탭 바

| 탭 | 아이콘 | 이동 화면 |
|----|--------|---------|
| 홈 | 집 모양 | SCR-MA-400 |
| 상담 | 말풍선 | SCR-MA-410 |
| 회원 | 사람 | SCR-MA-420 |
| 성과 | 차트 | SCR-MA-440 |
| 더보기 | 점 3개 | 바텀시트 메뉴 |

### 6. 인터랙션

- 화면 진입 시 API 호출 후 각 카드 숫자 업데이트 (로딩 중 Skeleton UI)
- Pull-to-refresh 지원
- 요약 카드 탭 시 해당 화면으로 이동 (전환 애니메이션: slide-left)
- 만료 임박 알림 영역 — 회원 행 탭 시 SCR-MA-421 (해당 회원 상세) 이동

## SCR-MA-410. 리드/상담 예정 목록

### 1. 화면 목적

FC가 자신이 담당하는 모든 상담 건(리드 포함)을 조회하고 필터/정렬하여 업무를 관리한다.

### 2. 접근 경로

- 하단 탭 바 > [상담] 탭
- SCR-MA-400 요약 카드 > [오늘 상담] / [신규 리드] 탭

### 3. 역할별 접근

#### 5-1. 탭 바

| 탭 | 필터 조건 |
|----|---------|
| 전체 | 상태 무관 전체 |
| 예정 | status = scheduled |
| 완료 | status = completed |
| 취소/노쇼 | status = cancelled or no_show |

#### 5-2. 필터

| 필터 | 옵션 |
|------|------|
| 유형 | 전체 / 상담 / OT / 체험 / 재등록상담 |

#### 5-3. 상담 카드

| 필드 | 내용 |
|------|------|
| 회원명 | 텍스트 |
| 상담 유형 | 텍스트 |
| 상담 채널 | 텍스트 (방문/전화/카카오톡/DM/SNS/기타) |
| 일시 | YYYY-MM-DD HH:mm |
| 상태 배지 | 예정(Blue) / 완료(Green) / 취소(Gray) / 노쇼(Red) |
| 탭 액션 | SCR-MA-412 이동 |

#### 5-4. FAB 버튼

- 우하단 고정, 원형 [+ ✏️] 버튼
- 탭 시 SCR-MA-411 이동

### 6. 인터랙션

- 탭 전환 시 즉시 목록 필터링 (API 재호출)
- 무한 스크롤 (페이지당 20건)

- Pull-to-refresh 지원

### 5. 구성 요소 상세

#### 5-1. 회원 검색 필드

| 항목 | 내용 |
|------|------|
| 입력 방식 | 텍스트 입력 (이름 또는 연락처) |
| 자동완성 | 2자 이상 입력 시 드롭다운 (최대 5건) |
| 드롭다운 항목 | 회원명 + 연락처 뒤 4자리 |
| 선택 후 | 칩(Chip) 형태로 표시, X 버튼으로 제거 |
| 미등록 리드 | "신규 리드로 등록" 옵션 표시 |
| 필수 여부 | 필수 (*) |

#### 5-2. 상담 유형 Select

| 옵션 | 값 |
|------|----|
| 상담 | consultation |
| OT | orientation |
| 체험상담 | trial |
| 재등록상담 | re-registration |

#### 5-3. 상담 채널 Select

| 옵션 | 값 |
|------|----|
| 방문 | visit |
| 전화 | phone |
| 카카오톡 | kakao |
| DM | dm |
| SNS | sns |
| 기타 | other |

#### 5-4. 상담 일시 DateTimePicker

- 기본값: 현재 날짜 + 현재 시각 (30분 단위 반올림)
- 미래 날짜 선택 가능 (예약 상담)
- 과거 날짜도 허용 (소급 등록)

#### 5-5. 상담 내용 Textarea

| 항목 | 내용 |
|------|------|
| 최소 글자 수 | 10자 |
| 최대 글자 수 | 500자 |
| 글자 수 카운터 | 현재 글자 수 / 최대 글자 수 실시간 표시 |
| 필수 여부 | 필수 (*) |

#### 5-6. 상담 상태 Select

| 옵션 | 값 | 비고 |
|------|----|----|
| 예정 | scheduled | 상담 결과 필드 비활성화 |
