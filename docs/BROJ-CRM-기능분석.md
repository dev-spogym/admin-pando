## 4. 우리만 가지고 있는 강점

1. **전자계약/서명** — 종이 계약서 대체
2. **급여 관리 + PDF 명세서** — 직원 급여 체계
3. **체성분 측정 히스토리** — BMI/체지방 추적
4. **RBAC 6단계 권한** — 세분화된 접근 제어
5. **멀티테넌트 아키텍처** — 통합 대시보드/지점 비교
6. **감사 로그** — 보안/컴플라이언스

## 5. 구현 우선순위 제안

### Phase 1 (즉시 — 1~2주)
- 미수금 관리 페이지 신규
- 환불 내역 전용 탭
- 매출 상품별/결제수단별 분석 탭 추가
- 락커 기록/이동/회수/고장 액션
- 회원 목록 일괄 상태변경
- 만료 상품 숨기기 필터

### Phase 2 (중기 — 2~4주)
- 수업 관리 (CRUD + 강사 배정 + 정원)
- 횟수 관리 (수강권 횟수 차감)
- 일정 통계 페이지
- 페널티 관리
- POS 현금가/카드가 분리
- 우측 퀵메뉴 패널

### Phase 3 (장기 — 1~2개월)
- 회원권/수강권 상품 타입 분리
- 정기 구독 (자동결제)
- 마켓 판매 연동
- 다지점 회원권/수강권
- 운동복 관리
