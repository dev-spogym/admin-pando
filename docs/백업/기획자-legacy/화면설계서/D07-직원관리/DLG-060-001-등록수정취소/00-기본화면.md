## 목적

직원 등록 또는 수정 중에 취소 버튼을 눌렀을 때, 입력한 내용이 저장되지 않고 사라진다는 사실을 사용자에게 미리 알려 실수로 인한 데이터 손실을 방지합니다.

## 구성요소

- **제목**: "작성 중인 내용이 있습니다"
- **안내 메시지**: "취소하면 입력한 내용이 저장되지 않습니다. 정말 취소하시겠습니까?"
- **계속 작성 버튼**: 다이얼로그를 닫고 폼으로 돌아갑니다.
- **취소 확인 버튼**: 폼을 닫고 직원 목록(SCR-060)으로 이동합니다.

## 언제 표시되나요

- 직원 목록(SCR-060) 화면에서 직원 등록 또는 수정 폼이 열려 있는 상태에서 "취소" 버튼을 클릭할 때 표시됩니다.
- 폼에 아무것도 입력하지 않은 상태에서 취소하면 이 다이얼로그 없이 바로 닫힐 수 있습니다.

## 주요 흐름

1. 사용자가 직원 등록 또는 수정 폼에서 일부 내용을 입력합니다.
2. "취소" 버튼을 클릭하면 이 다이얼로그가 표시됩니다.
3. "계속 작성"을 선택하면 다이얼로그가 닫히고 폼이 그대로 유지됩니다.
4. "취소 확인"을 선택하면 폼이 닫히고 직원 목록 화면으로 돌아갑니다.
