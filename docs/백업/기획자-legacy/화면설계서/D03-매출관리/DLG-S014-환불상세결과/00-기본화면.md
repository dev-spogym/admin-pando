## 목적

환불 처리가 완료된 직후 처리 결과를 즉시 확인합니다. 환불 금액, 처리 시각, 환불 수단 등 처리 완료 내역을 보여주어 직원과 고객 모두 결과를 명확히 인지할 수 있도록 합니다.

## 구성요소

| 구성요소 | 내용 |
|---|---|
| 처리 완료 안내 | "환불 처리가 완료되었습니다" 상태 메시지 |
| 환불 번호 | 시스템이 부여한 환불 고유 번호 |
| 환불 금액 | 실제 환불 처리된 금액 (크게 강조) |
| 환불 수단 | 환불이 이루어진 방법 |
| 처리 일시 | 환불 처리가 완료된 정확한 날짜 및 시각 |
| 처리 담당자 | 환불을 처리한 직원 이름 |
| 원 매출 정보 | 환불 대상이었던 원 매출 번호 및 상품명 |
| 확인 버튼 | 결과 확인 후 다이얼로그를 닫고 이전 화면으로 복귀 |

## 언제 표시되나요

- DLG-S013(환불 처리)에서 처리 확인 버튼을 눌러 환불이 성공적으로 완료된 직후 자동으로 표시

## 주요 흐름

1. DLG-S013 환불 처리에서 처리 확인 버튼을 클릭합니다.
2. 시스템이 환불을 처리합니다.
3. 환불 상세 결과 다이얼로그가 자동으로 표시됩니다.
4. 직원이 처리 결과를 확인합니다.
5. 확인 버튼을 눌러 환불 관리 목록 또는 매출 현황 화면으로 돌아갑니다.

## 관련 다이어그램

```
[DLG-S013 환불 처리]
        │ 처리 완료
        ▼
[DLG-S014 환불 상세 결과]
        │ 확인
        ▼
[SCR-S007 환불 관리 목록]
```
