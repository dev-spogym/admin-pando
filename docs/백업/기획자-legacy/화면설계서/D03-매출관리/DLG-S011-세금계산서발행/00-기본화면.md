## 목적

특정 매출 건에 대한 세금계산서를 발행합니다. 공급받는자의 사업자 정보와 이메일을 입력하여 세금계산서를 생성하고 전송합니다.

## 구성요소

| 구성요소 | 내용 |
|---|---|
| 원 매출 정보 | 발행 대상 매출 번호, 결제일, 금액 (읽기 전용) |
| 공급받는자 사업자번호 | 거래처 사업자등록번호 입력 |
| 공급받는자 상호 | 거래처 법인명 또는 상호 입력 |
| 대표자명 | 거래처 대표자 이름 입력 |
| 업태·업종 | 거래처 업태 및 업종 입력 |
| 이메일 주소 | 세금계산서를 수신할 이메일 주소 입력 |
| 발행 일자 | 세금계산서 발행 기준일 (기본값: 오늘) |
| 공급가액 | 부가세 제외 금액 (자동 계산) |
| 부가세액 | 공급가액의 10% (자동 계산) |
| 발행 버튼 | 입력 내용으로 세금계산서 발행 및 이메일 전송 |
| 취소 버튼 | 발행을 취소하고 이전 화면으로 복귀 |

## 언제 표시되나요

- SCR-S010(세금계산서 발행) 화면에서 새 발행 버튼을 클릭했을 때
- DLG-S001(매출 상세)에서 세금계산서 발행 버튼을 클릭했을 때

## 주요 흐름

1. 매출 상세 또는 세금계산서 발행 화면에서 발행 버튼을 선택합니다.
2. 세금계산서 발행 다이얼로그가 열리며 원 매출 금액이 자동 입력됩니다.
3. 공급받는자 정보(사업자번호, 상호, 이메일 등)를 입력합니다.
4. 공급가액과 부가세액이 자동 계산되어 표시됩니다.
5. 발행 버튼을 눌러 세금계산서를 발행하고 이메일로 전송합니다.
6. 발행 완료 후 DLG-S010(세금계산서 상세)에서 결과를 확인할 수 있습니다.

## 관련 다이어그램

```
[SCR-S010 세금계산서 발행] 또는 [DLG-S001 매출 상세]
        │ 새 발행 클릭
        ▼
[DLG-S011 세금계산서 발행]
        │              │
        │ 취소          │ 발행
        ▼              ▼
[이전 화면]    [DLG-S010 세금계산서 상세] (발행 결과 확인)
```
