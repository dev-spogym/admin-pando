## 목적

POS 결제 진행 시 구매 회원을 빠르게 찾아 연결합니다. 이름, 연락처, 회원번호 등으로 검색하여 해당 회원의 결제 이력과 잔여 수강권을 함께 확인할 수 있습니다.

## 구성요소

| 구성요소 | 내용 |
|---|---|
| 검색 입력창 | 이름, 연락처(전화번호), 회원번호 입력 |
| 검색 버튼 | 입력한 조건으로 검색 실행 |
| 검색 결과 목록 | 이름, 연락처, 회원 등급, 마지막 방문일 표시 |
| 회원 선택 버튼 | 목록에서 특정 회원을 결제 대상으로 선택 |
| 비회원 처리 버튼 | 회원 연결 없이 비회원으로 결제 진행 |
| 닫기 버튼 | 검색을 취소하고 POS 화면으로 복귀 |

## 언제 표시되나요

- SCR-S002(POS 판매) 화면에서 구매자 선택 버튼을 눌렀을 때
- 결제 진행 중 구매자를 변경하고자 할 때

## 주요 흐름

1. POS 판매 화면에서 구매자 선택 버튼을 누릅니다.
2. 구매자 검색 다이얼로그가 열립니다.
3. 이름, 연락처, 회원번호 중 하나를 입력하고 검색합니다.
4. 검색 결과 목록에서 해당 회원을 확인합니다.
5. 회원을 선택하면 POS 화면에 구매자로 연결됩니다.
6. 검색 결과가 없으면 비회원으로 처리하거나 신규 회원 등록을 안내합니다.

## 관련 다이어그램

```
[SCR-S002 POS 판매]
        │ 구매자 선택 클릭
        ▼
[DLG-S002 구매자 검색]
        │ 검색 → 회원 선택
        ▼
[SCR-S002 POS 판매] (구매자 연결 완료)
        │
        ▼
[DLG-S003 결제 확인]
```
