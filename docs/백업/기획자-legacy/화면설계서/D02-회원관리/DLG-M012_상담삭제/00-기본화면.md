## 목적
등록된 상담 기록을 삭제하기 전, 복구가 불가능함을 안내하고 관리자의 최종 확인을 받는 다이얼로그입니다.

## 구성요소
- 제목: "상담 기록을 삭제하시겠습니까?"
- 안내 메시지: "삭제된 상담 기록은 복구할 수 없습니다."
- 삭제 대상 상담 일시 및 유형 표시
- 확인 버튼 (삭제 실행)
- 취소 버튼 (다이얼로그 닫기)

## 언제 표시되나요
- 회원 상세 > 상담 탭에서 특정 상담 기록의 삭제 버튼을 클릭했을 때

## 주요 흐름
1. 관리자가 상담 기록 삭제 버튼 클릭 → 다이얼로그 표시
2. 확인 → 상담 기록 삭제 완료, 다이얼로그 닫힘, 목록에서 해당 기록 제거
3. 취소 → 다이얼로그 닫힘, 기록 유지
