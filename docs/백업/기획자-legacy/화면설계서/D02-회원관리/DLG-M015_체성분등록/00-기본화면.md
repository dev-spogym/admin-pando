## 목적
체성분 측정 기기와 연동되지 않는 환경에서, 관리자가 회원의 체성분 측정값을 수동으로 입력하여 건강 데이터를 기록하는 다이얼로그입니다.

## 구성요소
- 제목: "체성분 등록"
- 측정일 선택 (날짜 선택, 기본값: 오늘)
- 체중 입력 (kg)
- 체지방률 입력 (%)
- 근육량 입력 (kg)
- 체수분율 입력 (%, 선택 사항)
- BMI 자동 계산 표시 (체중·신장 기반)
- 등록 버튼
- 취소 버튼

## 언제 표시되나요
- 회원 상세 > 체성분 탭에서 체성분 등록 버튼을 클릭했을 때
- 같은 날 이미 등록된 데이터가 있는 경우 DLG-M016(체성분 덮어쓰기) 다이얼로그로 분기

## 주요 흐름
1. 관리자가 체성분 등록 버튼 클릭 → 다이얼로그 표시
2. 측정일 선택 및 측정값 입력
3. 등록 버튼 클릭 → 체성분 데이터 저장 완료, 다이얼로그 닫힘, 체성분 그래프 즉시 갱신
4. 취소 → 다이얼로그 닫힘, 저장 없음
