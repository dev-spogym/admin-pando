## 목적
가족 관계인 회원들을 서로 연결하는 다이얼로그입니다. 연결된 가족 회원은 일부 혜택(패밀리 이용권, 동반 출석 등)을 함께 사용할 수 있습니다.

## 구성요소
- 제목: "가족 연결"
- 현재 회원 정보 표시 (읽기 전용)
- 연결할 회원 검색 (이름 또는 전화번호로 검색)
- 검색 결과 목록 (회원명, 회원번호, 연락처)
- 선택된 연결 대상 회원 표시
- 가족 관계 선택 (배우자, 부모, 자녀, 형제·자매, 기타)
- 연결 버튼
- 취소 버튼

## 언제 표시되나요
- 회원 상세 > 기본정보 탭의 가족 연결 버튼 클릭 시
- 관리자 또는 해당 권한을 가진 직원에게 표시

## 주요 흐름
1. 관리자가 가족 연결 버튼 클릭 → 다이얼로그 표시
2. 연결할 회원 검색 → 검색 결과에서 대상 회원 선택
3. 가족 관계 선택
4. 연결 버튼 클릭 → 두 회원 간 가족 관계 설정, 각 회원 상세에 가족 정보 표시
5. 취소 클릭 → 다이얼로그 닫힘, 변경 없음
