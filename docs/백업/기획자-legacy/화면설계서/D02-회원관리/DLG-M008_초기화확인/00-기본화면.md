## 목적
회원 등록 또는 수정 화면에서 입력 폼을 초기화하기 전, 지금까지 입력한 내용이 모두 지워짐을 안내하고 관리자의 의사를 확인하는 다이얼로그입니다.

## 구성요소
- 제목: "입력 내용을 초기화하시겠습니까?"
- 안내 메시지: "입력하신 모든 내용이 초기 상태로 돌아갑니다."
- 확인 버튼 (폼 초기화 실행)
- 취소 버튼 (다이얼로그 닫기, 입력 유지)

## 언제 표시되나요
- 회원 등록 또는 수정 화면에서 초기화(리셋) 버튼을 클릭했을 때
- 하나 이상의 필드에 값이 입력된 상태일 때만 다이얼로그 표시
- 모든 필드가 비어 있으면 다이얼로그 없이 초기화

## 주요 흐름
1. 관리자가 초기화 버튼 클릭 → 다이얼로그 표시
2. 확인 → 모든 입력 필드 초기 상태로 리셋
3. 취소 → 다이얼로그 닫힘, 입력 내용 유지
