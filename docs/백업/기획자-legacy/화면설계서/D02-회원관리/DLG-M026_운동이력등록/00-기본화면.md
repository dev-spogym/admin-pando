## 목적
회원이 수행한 운동 내역을 관리자 또는 트레이너가 직접 입력하여 기록하는 다이얼로그입니다. 앱이나 기기를 사용하지 않은 경우에도 운동 이력을 수동으로 남길 수 있습니다.

## 구성요소
- 제목: "운동 이력 등록"
- 운동 날짜 선택 (기본값: 오늘)
- 운동 종목 선택 또는 직접 입력
- 운동 시간 입력 (시작 시간, 종료 시간 또는 총 운동 시간)
- 세부 운동 내역 입력 (운동명, 세트, 반복 수, 무게 — 행 추가/삭제 가능)
- 특이사항 메모 입력 (선택)
- 저장 버튼
- 취소 버튼

## 언제 표시되나요
- 회원 상세 > 운동 이력 탭에서 이력 등록 버튼 클릭 시
- 트레이너 또는 관리자 권한을 가진 사용자만 표시

## 주요 흐름
1. 트레이너가 이력 등록 버튼 클릭 → 다이얼로그 표시
2. 운동 날짜·종목·시간·세부 내역 입력
3. 저장 클릭 → 운동 이력 저장, 운동 이력 탭에 반영
4. 취소 클릭 → 다이얼로그 닫힘, 변경 없음
