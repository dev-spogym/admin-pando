## 화면 목적
KPI 대시보드에서 지점별 월 매출 목표를 설정하는 다이얼로그입니다.
목표 수치 설정 후 KPI 카드에서 실적 대비 달성률을 확인할 수 있습니다.

## 주요 구성요소
- "매출 목표 설정" 제목
- 대상 지점 선택 (전체 지점 일괄 또는 개별 지점 선택)
- 목표 적용 기간 선택 (특정 월 지정 또는 기간 범위)
- 지점별 월 매출 목표 금액 입력란
  - 전체 일괄 입력 시: 동일 금액을 모든 지점에 적용
  - 개별 입력 시: 지점별 목표 금액 각각 입력
- 전년 동기 실적 참고 표시 (입력 참고용)
- 취소 버튼
- 저장 버튼

## 주요 UX 흐름
1. KPI 대시보드에서 매출 목표 설정 버튼 클릭 → 다이얼로그 표시
2. 대상 지점 및 적용 기간 선택
3. 목표 금액 입력 (전년 동기 실적을 참고하여 입력)
4. 저장 버튼 클릭 → 목표 저장 → KPI 대시보드 달성률 즉시 갱신
5. 취소 → 다이얼로그 닫힘

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 현재 설정된 목표 금액이 채워진 상태 |
| 미설정 | 목표가 없는 지점은 빈 입력란 |
| 저장 처리 중 | 저장 버튼 클릭 후 처리 중 |
| 저장 완료 | 다이얼로그 닫히고 KPI 대시보드 갱신 |
