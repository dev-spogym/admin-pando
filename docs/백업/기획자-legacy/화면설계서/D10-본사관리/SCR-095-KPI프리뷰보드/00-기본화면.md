## 화면 목적
KPI 대시보드에 표시할 카드 항목과 레이아웃을 미리 구성하고 확인하는 화면입니다.
표시할 지표 카드를 선택하고 배치 순서를 조정하여 자신만의 KPI 뷰를 커스터마이징합니다.

## 주요 구성요소

### 카드 선택 패널 (좌측)
- 추가 가능한 KPI 카드 목록 (매출, 회원, 출석, 유지율, 신규 가입, 이탈, 쿠폰 사용 등)
- 각 카드의 미리보기 썸네일
- 추가 버튼

### 레이아웃 미리보기 (우측)
- 현재 선택된 KPI 카드 배치 미리보기
- 드래그앤드롭으로 카드 순서 변경
- 각 카드별 제거 버튼

### 레이아웃 옵션
- 열 수 선택 (2열 / 3열 / 4열)
- 카드 크기 선택 (소 / 중 / 대)

### 저장 및 적용
- 현재 구성 저장 버튼 (KPI 대시보드에 적용)
- 기본값으로 초기화 버튼

## 계정별 차이
| 계정 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 | 전체 KPI 카드 선택 가능 | 레이아웃 구성 및 저장 |
| 센터장 | 소속 지점 관련 KPI 카드 | 레이아웃 구성 및 저장 |
| 매니저 이하 | 접근 불가 | — |

## 주요 UX 흐름
1. 화면 진입 시 현재 저장된 KPI 보드 구성을 불러온다.
2. 좌측 카드 목록에서 카드 추가 → 미리보기 영역에 추가됨.
3. 미리보기 영역에서 드래그앤드롭으로 순서 조정.
4. 저장 버튼 클릭 → KPI 대시보드(SCR-094)에 변경 내용 즉시 반영.
5. 기본값 초기화 버튼 클릭 → 기본 제공 카드 구성으로 복원.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 저장된 KPI 보드 구성 표시 |
| 수정 중 | 카드 추가·제거·순서 변경 후 저장 버튼 활성 |
| 저장 완료 | 완료 메시지 표시 후 KPI 대시보드에 반영 |
