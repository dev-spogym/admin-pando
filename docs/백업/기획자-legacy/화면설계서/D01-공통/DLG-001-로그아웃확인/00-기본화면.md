## 화면 목적
사용자가 로그아웃 버튼을 눌렀을 때, 실수로 누른 경우를 방지하기 위해 한 번 더 의사를 확인하는 다이얼로그입니다.

## 주요 구성요소
- "로그아웃 하시겠습니까?" 확인 메시지
- 로그아웃 확인 버튼
- 취소 버튼

## 주요 UX 흐름
1. 로그아웃 클릭 → 이 다이얼로그 표시
2. 확인 → 세션 종료 후 로그인 화면으로 이동
3. 취소 → 다이얼로그 닫고 현재 화면 유지

## 화면 상태
| 상태 | 설명 |
|---|---|
| 표시 | 확인 메시지 표시 |
| 처리 중 | 로그아웃 처리 중 (버튼 비활성) |
