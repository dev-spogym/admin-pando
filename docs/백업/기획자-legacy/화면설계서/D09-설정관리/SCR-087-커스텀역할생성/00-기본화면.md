## 화면 목적
기본 제공 역할(최고관리자·센터장·매니저 등) 외에 센터 운영 특성에 맞는 맞춤형 역할을 생성·관리하는 화면입니다.
역할 이름과 설명을 정의하고, 필요한 권한 조합을 직접 구성합니다.

## 주요 구성요소

### 커스텀 역할 목록
- 현재 생성된 커스텀 역할 목록 (역할명, 설명, 배정된 직원 수, 생성일)
- 새 역할 만들기 버튼
- 역할별 수정·삭제 버튼

### 역할 생성·수정 폼
- 역할 이름 입력 (필수)
- 역할 설명 입력
- 기존 역할에서 권한 복사 선택 (선택적, 출발점 설정)
- 권한 매트릭스 (SCR-081과 동일 구조로 메뉴별 권한 선택)

### 역할 배정 현황
- 해당 역할이 배정된 직원 목록 미리보기

## 계정별 차이
| 계정 | 권한 |
|---|---|
| 슈퍼관리자 | 전체 커스텀 역할 생성·수정·삭제 가능 |
| 센터장 | 소속 지점 커스텀 역할 생성·수정·삭제 가능 |
| 매니저 | 접근 불가 |
| 기타 | 접근 불가 |

## 주요 UX 흐름
1. 화면 진입 시 생성된 커스텀 역할 목록을 표시한다.
2. 새 역할 만들기 버튼 클릭 → 역할 이름·설명 입력 및 권한 매트릭스 구성 화면으로 전환.
3. 기존 역할에서 복사 선택 시 선택한 역할의 권한이 매트릭스에 자동 로드된다.
4. 저장 → 역할 목록에 추가, 권한 설정(SCR-081)에서 선택 가능.
5. 역할 삭제 시 해당 역할이 배정된 직원이 있으면 역할 재배정 다이얼로그(DLG-087-001) 표시.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 커스텀 역할 목록 표시 |
| 역할 없음 | 생성된 커스텀 역할이 없을 때 안내 메시지 표시 |
| 생성 폼 | 새 역할 작성 중 |
| 수정 폼 | 기존 역할 수정 중 |
| 저장 완료 | 완료 메시지 표시 후 목록으로 복귀 |
