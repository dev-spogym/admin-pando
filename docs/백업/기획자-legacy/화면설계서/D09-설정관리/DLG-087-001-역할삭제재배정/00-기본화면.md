## 화면 목적
삭제하려는 역할에 배정된 직원이 있을 때, 해당 직원들을 다른 역할로 일괄 재배정한 후 역할을 삭제하는 다이얼로그입니다.

## 주요 구성요소
- "역할 삭제 — 직원 재배정" 제목
- 삭제 대상 역할명 및 배정 직원 수 표시
- 배정된 직원 목록 (이름, 현재 역할)
- 재배정할 역할 선택 드롭다운 (삭제 대상 역할 제외한 전체 역할 선택 가능)
- 취소 버튼
- 재배정 후 역할 삭제 버튼

## 주요 UX 흐름
1. 역할 삭제 시도 → 배정 직원이 있는 경우 이 다이얼로그로 이동
2. 재배정할 역할 선택
3. 재배정 후 역할 삭제 버튼 클릭 → 직원 일괄 역할 변경 → 기존 역할 삭제
4. 취소 → 다이얼로그 닫힘, 역할 삭제 취소

## 화면 상태
| 상태 | 설명 |
|---|---|
| 표시 | 배정 직원 목록 및 재배정 역할 선택 |
| 역할 미선택 | 재배정 역할 선택 전 삭제 버튼 비활성 |
| 처리 중 | 재배정 및 삭제 처리 중 (버튼 비활성) |
| 완료 | 다이얼로그 닫히고 역할 목록 갱신 |
