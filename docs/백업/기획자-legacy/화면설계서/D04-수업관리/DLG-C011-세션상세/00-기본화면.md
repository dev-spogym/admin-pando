## 목적

개별 PT 세션에서 진행된 운동 내용과 강사 노트를 상세하게 기록하고 조회하는 팝업입니다. 세션별로 어떤 운동을 몇 세트·몇 회·몇 kg으로 진행했는지 기록해 두면 회원의 운동 이력을 추적하고 다음 세션 계획에 활용할 수 있습니다.

## 구성요소

- **세션 기본 정보**: 수업명(회원명), 날짜·시간, 담당 강사
- **운동 프로그램**: 해당 세션에 배정된 운동 프로그램명
- **운동 기록 목록**:
  - 동작명
  - 세트 수
  - 반복 횟수 (또는 시간)
  - 중량 (kg)
  - 완료 여부 체크박스
  - 동작별 메모
- **강사 노트**: 세션 전체에 대한 강사 코멘트, 특이사항, 다음 세션 제안
- **회원 컨디션 기록**: 당일 회원 컨디션 (좋음/보통/나쁨), 부상·통증 여부
- **저장 버튼** (기록 입력 시) / **닫기 버튼** (조회 시)

## 언제 표시되나요

- 운동 프로그램 관리 화면(SCR-C010)에서 특정 세션 기록 클릭 시
- 수업 기록 상세(DLG-C005)에서 "세션 기록 보기" 버튼 클릭 시
- 유효 수업 목록(SCR-C011)에서 수업 완료 후 세션 기록 입력 버튼 클릭 시

## 주요 흐름

1. 수업 완료 후 "세션 기록 입력" 클릭 → 팝업 열림
2. 동작별 실제 수행 세트·횟수·중량 입력
3. 강사 노트 및 회원 컨디션 입력
4. 저장 클릭 → 세션 이력에 기록 저장
5. 조회 모드: 기존 기록 표시, 닫기로 팝업 닫힘
