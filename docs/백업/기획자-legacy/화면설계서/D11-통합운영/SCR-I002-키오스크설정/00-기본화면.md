## 화면 목적

지점별 키오스크 운영 정책을 설정한다. 출석 인식 방식(QR·RFID·얼굴인식), 대기 화면 구성, 음성 안내(TTS), 출입 규칙, 출석 후 락커 처리 정책을 한 화면에서 관리한다.

## 주요 구성요소

### 탭 구조

| 탭 | 설명 |
|----|------|
| 기본 설정 | 키오스크 활성 여부, 체크인 수단(QR·RFID·얼굴인식), 직원 출석 허용 여부 |
| 화면 설정 | 배경 이미지, 센터 로고, 환영 문구, 공지 메시지 등 키오스크 화면 구성 |
| TTS 설정 | 대기·성공·실패·만료 임박 상황별 음성 안내 문구 관리 |
| 출입 규칙 | 출입 허용 시간, 만료 회원 허용 여부, 중복 출석 방지 시간 설정 |
| 락커 후처리 | 출석 완료 후 락커 처리 정책, 자동 배정 존, 미배정 안내 문구 |

### 기본 설정 탭 주요 항목

| 항목 | 설명 |
|------|------|
| 키오스크 활성 | 지점 키오스크 사용 여부 ON/OFF |
| 체크인 수단 | QR, RFID, 얼굴인식 복수 선택 가능 |
| 직원 출석 허용 | 직원도 키오스크로 출퇴근 처리 가능 여부 |
| 얼굴인식 사용 | 얼굴인식 모듈 활성/비활성 |

### 락커 후처리 탭 주요 항목

| 항목 | 선택 옵션 |
|------|-----------|
| 출석 후 처리 정책 | 출석만 완료 / 수동 배정 대기 / 자동 배정 시도 |
| 자동 배정 존 | 자동 배정 시 탐색할 락커 존 선택 |
| 미배정 안내 문구 | 자동 배정 실패 시 키오스크 화면에 표시할 안내 문구 |

### TTS 설정 탭 주요 항목

| 이벤트 | 설명 |
|--------|------|
| 대기 | 체크인 대기 중 반복 안내 |
| 성공 | 체크인 성공 시 환영 음성 |
| 실패 | 체크인 실패 시 안내 음성 |
| 만료 임박 | 이용권 만료 임박 안내 음성 |

### 버튼

| 버튼 | 설명 |
|------|------|
| 저장 | 현재 탭의 설정값 저장 |
| 연결 테스트 | 연결된 키오스크 기기 상태 확인 |
| 미리보기 | 현재 화면 설정을 키오스크 화면 기준으로 미리보기 |
| 배너 업로드 | 키오스크 배경/로고 이미지 등록 |
| TTS 재생 | 이벤트별 음성 안내 문구 샘플 재생 |

## 계정별 차이

| 계정 유형 | 차이 |
|-----------|------|
| 센터 관리자 | 전체 탭 설정 변경 가능 |
| 프런트 직원 | 조회만 가능, 설정 변경 불가 |
| 본사 관리자 | 전 지점 키오스크 설정 조회 가능, 지점별 수정은 해당 지점 관리자가 수행 |

## 주요 UX 흐름

1. 탭 선택 → 해당 설정 항목 표시
2. 항목 수정 → 저장 버튼 클릭 → 성공/실패 알림 표시
3. 락커 후처리 탭: '자동 배정 시도' 선택 시 자동 배정 존 선택 영역 활성화
4. TTS 탭: 문구 입력 후 재생 버튼으로 음성 미리 확인
5. 화면 설정 탭: 배너 업로드 후 미리보기 버튼으로 키오스크 화면 확인

## 화면 상태

| 상태 | 설명 |
|------|------|
| 기본(설정값 로드됨) | 현재 저장된 설정값 표시 |
| 수정 중 | 변경된 항목 하이라이트, 저장 버튼 활성화 |
| 저장 성공 | 토스트 '설정이 저장되었습니다.' 표시 |
| 저장 실패 | 오류 안내 및 재시도 유도 |
| 키오스크 미연결 | 연결 상태 경고 배지 표시 |
