## 목적

특정 상품의 가격이 언제, 누구에 의해, 얼마에서 얼마로 변경되었는지 이력을 시간순으로 확인하는 다이얼로그입니다. 가격 정책의 투명성을 확보하고 분쟁 발생 시 근거 자료로 활용할 수 있습니다.

---

## 구성요소

- 대상 상품명 표시
- 가격 변경 이력 타임라인
  - 변경 일시
  - 변경 전 가격 → 변경 후 가격
  - 변경한 직원 이름

하단 버튼: **닫기**

---

## 언제 표시되나요

- SCR-P003(상품 상세 패널) 수정 모드에서 "가격 변경 이력" 섹션을 클릭했을 때
- 이력이 없는 신규 상품에서는 "변경 이력이 없습니다." 메시지 표시

---

## 주요 흐름

1. 상품 상세 패널에서 가격 이력 섹션 클릭
2. 해당 상품의 가격 변경 이력 불러오기
3. 타임라인 형태로 이력 표시
4. "닫기" 버튼 또는 모달 외부 클릭으로 닫힘
