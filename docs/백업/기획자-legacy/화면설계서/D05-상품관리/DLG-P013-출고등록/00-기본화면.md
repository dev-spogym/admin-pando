## 목적

실물 상품의 출고 수량을 처리하는 모달입니다. 판매, 사용, 폐기 등 출고 사유와 수량을 기록하며, 처리 시 재고 수량이 자동으로 차감됩니다.

---

## 구성요소

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 상품명 | 자동 입력 | 출고 처리할 상품 (변경 불가) |
| 현재 재고 | 자동 표시 | 현재 보유 수량 (참고용) |
| 출고 수량 | 필수 | 출고할 개수 (현재 재고 이하) |
| 출고 사유 | 필수 | 판매 / 사용 / 폐기 / 기타 중 선택 |
| 출고일 | 필수 | 실제 출고 날짜 (기본값: 오늘) |
| 메모 | 선택 | 특이사항 기록 |

하단 버튼: **취소** / **출고 처리**

---

## 언제 표시되나요

- SCR-P007(재고 관리) 목록에서 특정 상품의 "출고" 버튼을 클릭했을 때

---

## 주요 흐름

1. 대상 상품명, 현재 재고 수량이 표시된 상태로 모달 열림
2. 출고 수량, 사유, 날짜 입력
3. "출고 처리" 버튼 클릭
   - 수량이 현재 재고를 초과할 경우: "재고 수량이 부족합니다." 안내
   - 정상 처리 시: 재고 수량 차감 + 이력 기록 + 모달 닫힘 + 완료 메시지
4. "취소" 클릭: 모달 닫힘
