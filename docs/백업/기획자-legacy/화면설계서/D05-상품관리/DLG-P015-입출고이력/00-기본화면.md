## 목적

특정 상품의 입고·출고·재고조정 이력을 시간순으로 확인하는 모달입니다. 재고 변동 내역을 추적하여 재고 불일치의 원인을 파악하는 데 활용합니다.

---

## 구성요소

- 조회 기간 선택 (시작일 ~ 종료일)
- 이력 유형 필터: 전체 / 입고 / 출고 / 조정
- 입출고 이력 테이블

| 컬럼 | 내용 |
|------|------|
| 날짜 | 처리 날짜 |
| 구분 | 입고 / 출고 / 조정 |
| 수량 | 변동 수량 (+/- 표시) |
| 잔여 재고 | 처리 후 남은 수량 |
| 사유 | 출고 사유 또는 조정 사유 |
| 처리자 | 담당 직원 이름 |

하단 버튼: **닫기**

---

## 언제 표시되나요

- SCR-P007(재고 관리) 목록에서 특정 상품의 "이력" 버튼을 클릭했을 때

---

## 주요 흐름

1. 대상 상품명이 표시된 상태로 모달 열림
2. 기본 기간(최근 1개월) 이력 표시
3. 기간 또는 유형 필터 변경 시 이력 갱신
4. 이력 확인 후 "닫기" 클릭으로 모달 닫힘
