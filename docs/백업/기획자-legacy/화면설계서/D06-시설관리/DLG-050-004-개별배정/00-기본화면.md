## 화면 목적

특정 빈 락커에 회원을 개별적으로 배정하는 다이얼로그입니다. 회원 검색으로 대상 회원을 선택하고 만료일을 설정해 배정을 완료합니다.

## 주요 구성요소

- **다이얼로그 제목**: "[락커 번호]번 락커 회원 배정"
- **회원 검색 자동완성**: 회원명 또는 연락처를 입력하면 자동완성 목록이 나타납니다. 선택 시 회원명이 표시됩니다.
- **만료일 입력**: 날짜 입력 필드로 배정 만료일을 설정합니다.
- **배정 버튼**: 선택한 회원과 만료일로 배정을 완료합니다.
- **취소 버튼**: 변경 없이 다이얼로그를 닫습니다.

## 주요 UX 흐름

1. 빈 락커 셀 호버 시 [배정] 버튼 클릭 → 다이얼로그 열림.
2. 회원 검색창에 이름·연락처 입력 → 자동완성 목록에서 회원 선택.
3. 만료일을 입력합니다.
4. [배정] 클릭 → "[락커 번호]번 락커가 [회원명]님께 배정되었습니다" 알림.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 회원 미선택 | [배정] 버튼 비활성화 |
| 배정 성공 | "[N]번 락커가 [회원명]님께 배정되었습니다" 알림 |
| 배정 실패 | "락커 배정에 실패했습니다" 오류 알림 |
| 회원 없음 검색 결과 | "검색 결과가 없습니다" 자동완성 안내 |
