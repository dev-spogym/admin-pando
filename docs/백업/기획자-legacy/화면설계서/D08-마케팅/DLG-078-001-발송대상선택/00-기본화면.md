## 목적

SMS/카카오 대량 발송(SCR-078) 시 발송 대상 회원 그룹을 조건별로 선택하는 다이얼로그입니다. 세분화된 조건으로 적합한 수신자만 선별해 효율적인 대량 발송을 지원합니다.

> **현재 상태**: 미구현 기능입니다.

## 구성요소

- **그룹 선택**: 전체 회원 / 활성 회원 / 만료 회원 / 특정 등급 회원 / 장기 미방문 등 사전 정의된 그룹을 선택합니다.
- **추가 조건 필터**: 성별, 연령대, 이용권 유형, 마지막 방문일 범위 등 세부 조건을 추가합니다.
- **예상 수신자 수**: 설정한 조건에 해당하는 회원 수가 실시간으로 표시됩니다.
- **확인 버튼**: 선택한 대상을 발송 화면에 적용하고 다이얼로그를 닫습니다.
- **취소 버튼**: 선택 내용을 반영하지 않고 닫습니다.

## 언제 표시되나요

- SMS/카카오 발송 화면(SCR-078)에서 "발송 대상 선택" 버튼을 클릭할 때 표시됩니다.

## 주요 흐름

1. 담당자가 SCR-078에서 "발송 대상 선택" 버튼을 클릭합니다.
2. 이 다이얼로그가 열립니다.
3. 그룹과 추가 조건을 선택하면 예상 수신자 수가 실시간으로 업데이트됩니다.
4. "확인"을 클릭하면 선택된 대상이 발송 화면에 적용됩니다.
