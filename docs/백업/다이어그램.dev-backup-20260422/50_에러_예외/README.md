## 에러 분류 체계

### HTTP 상태코드별 분류

| HTTP | 에러 유형 | 처리 방식 |
|------|-----------|-----------|
| 400 | Bad Request — 입력값 오류 | 인라인 필드 에러 / 토스트 |
| 401 | Unauthorized — 인증 실패 | 로그인 리다이렉트 |
| 402 | Payment Required — 결제 실패 | 모달 다이얼로그 |
| 403 | Forbidden — 권한 없음 | 토스트 알림 |
| 404 | Not Found — 리소스 없음 | 토스트 알림 |
| 409 | Conflict — 중복/충돌 | 토스트 + 안내 |
| 422 | Unprocessable — 비즈니스 로직 위반 | 모달 또는 토스트 |
| 500 | Internal Server Error | 전역 에러 토스트 |
| 503 | Service Unavailable | 전역 토스트 + 재시도 버튼 |
