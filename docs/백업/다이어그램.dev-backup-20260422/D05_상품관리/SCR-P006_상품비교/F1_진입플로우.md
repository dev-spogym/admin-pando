# F1 진입 플로우 — SCR-P006 상품 비교 🆕

## 목적
상품 비교 화면(SCR-P006)으로 진입하는 모든 경로를 정의하고, 선행 조건 및 초기 렌더링 상태를 명시한다.

## 전제조건
- SCR-P001 상품 관리 목록에서 비교 버튼 클릭 또는 SCR-P003 상품 상세 패널에서 비교 추가로 접근
- 최대 4개 상품 선택 가능, 최소 2개 이상 선택 시 비교 활성화
- RBAC: 모든 역할 조회 가능
