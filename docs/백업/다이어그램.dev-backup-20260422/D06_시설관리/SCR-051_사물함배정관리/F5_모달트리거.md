# F5 모달 트리거 트리 + 배정→결제→만료→연장 — SCR-051 사물함 배정 관리

사물함 배정의 전체 생명주기: 배정 → 결제 연동 → 만료 감지 → 연장 처리 플로우를 상세화한다.
