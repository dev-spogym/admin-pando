## 1. 목적

SCR-M003에 존재하는 모든 버튼/액션을 노드로 표현하고 각각의 동작 흐름을 명세한다. SCR-M002와 동일 컴포넌트, isEditRoute 플래그 분기.

## 2. 전제조건

- SCR-M003 회원 수정 화면이 기존 데이터 pre-fill 완료 상태이다.
