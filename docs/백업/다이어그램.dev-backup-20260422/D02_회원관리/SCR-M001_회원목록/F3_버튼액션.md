## 1. 목적

SCR-M001에 존재하는 모든 버튼/액션을 노드로 표현하고 각각의 동작 흐름을 명세한다. 버튼별 TC 원천.

## 2. 전제조건

- SCR-M001 회원 목록 화면이 정상 표시 상태이다.
