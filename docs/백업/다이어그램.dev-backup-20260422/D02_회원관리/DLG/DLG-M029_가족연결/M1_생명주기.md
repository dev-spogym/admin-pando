## 1. 목적

DLG-M029 가족 회원 연결 다이얼로그의 열기/닫기/완료 생명주기를 명세한다. 🆕 미구현 기능.

## 2. 트리거/전제조건

- 회원 상세 > "가족 연결" 버튼 클릭

## 3. 다이어그램

```mermaid
flowchart TD
  TRIGGER([회원 상세 가족 연결 버튼 클릭])

  TRIGGER -->|E_OPEN_01| DLG_OPEN[DLG-M029 가족 회원 연결 🆕]
  DLG_OPEN --> FLD_MEMBER[가족 회원 member search 필수]
  DLG_OPEN --> FLD_REL[관계 select 필수 기본값 배우자]

  FLD_MEMBER --> USER_ACT{사용자 액션}
  FLD_REL    --> USER_ACT

  USER_ACT -->|E_CANCEL_01 취소| CLOSE[모달 닫기]
  USER_ACT -->|E_CONFIRM_01 연결 클릭| VALID{필드 유효성}

  VALID -->|E_INVALID_01 회원 미선택| ERR_MEMBER[회원을 선택하세요]
  ERR_MEMBER -->|E_STAY_01| DLG_OPEN

  VALID -->|E_VALID_01 통과| API_CALL[POST /api/members/:id/family 🆕]

  API_CALL -->|E_FAIL_01 오류| TOAST_ERR((toast.error 연결에 실패했습니다))
  TOAST_ERR -->|E_STAY_02| DLG_OPEN
```
