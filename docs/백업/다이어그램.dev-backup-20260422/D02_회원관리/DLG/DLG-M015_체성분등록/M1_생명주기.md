## 1. 목적

DLG-M015 체성분 측정 추가 다이얼로그의 열기/닫기/완료 생명주기를 명세한다.

## 2. 트리거/전제조건

- 체성분 탭 > "측정 추가" 버튼 클릭

## 3. 다이어그램

```mermaid
flowchart TD
  TRIGGER([체성분 탭 측정 추가 버튼 클릭])

  TRIGGER -->|E_OPEN_01| DLG_OPEN[DLG-M015 체성분 측정 추가 max-w-440px]
  DLG_OPEN --> INIT[측정일/체중/골격근량/체지방률 빈값 초기화]

  INIT --> USER_ACT{사용자 액션}

  USER_ACT -->|E_CANCEL_01 취소| CLOSE[모달 닫기]
```

## 4. 엣지 설명

| 엣지 ID | 출발 | 도착 | 조건 |
|---------|------|------|------|
| E_OPEN_01 | 측정 추가 | 모달 열기 | - |
| E_SUBMIT_01 | 저장 | 날짜 중복 확인 | 필수 충족 |
| E_DATE_DUP_01 | 날짜 확인 | 덮어쓰기 모달 | 중복 존재 |
| E_DATE_NEW_01 | 날짜 확인 | INSERT API | 새 데이터 |
| E_SUCCESS_01 | API | toast.success | 200 |
| E_FAIL_01 | API | toast.error | 오류 |
