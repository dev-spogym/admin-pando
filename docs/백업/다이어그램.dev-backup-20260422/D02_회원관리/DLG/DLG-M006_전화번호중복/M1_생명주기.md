## 1. 목적

DLG-M006 전화번호 중복 확인(인라인)의 트리거/결과 생명주기를 명세한다.

## 2. 트리거/전제조건

- 회원 등록/수정 > 연락처 필드 입력 후 "중복확인" 버튼 클릭
- phone.length >= 13 AND phoneChecked === false

## 3. 다이어그램

