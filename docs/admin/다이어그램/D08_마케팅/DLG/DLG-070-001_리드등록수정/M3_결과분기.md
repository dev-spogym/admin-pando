flowchart TD

  RESP -->|E_M3_TIMEOUT Timeout|ERR_TIMEOUT[toast 타임아웃\n재시도 안내]

  OK_CREATE --> SCR_070_REFRESH[SCR-070 목록 갱신]
  OK_UPDATE --> SCR_070_REFRESH
