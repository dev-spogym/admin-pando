## 1. 목적

메시지 발송 Happy Path — 수신자 선택, 채널 선택, 내용 작성, 미리보기, 발송/예약 흐름을 TC 원천으로 제공한다.

## 2. 전제조건

- SCR-071 렌더링 완료

## 3. 다이어그램

```mermaid
flowchart TD
  SCR_071[SCR-071 메시지 발송] --> SEL_RECV[수신자 선택\n그룹 버튼 or 검색]

  SEL_RECV -->|그룹 버튼 클릭|GROUP_SEL{그룹 선택}
  SEL_RECV -->|대상 검색 클릭|DLG_001[DLG-071-001 수신자 검색]

  API_SEND -->|E_071_F2_SEND_OK 200|T_OK((toast\n메시지가 발송되었습니다))
```
