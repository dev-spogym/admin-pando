## 1. 목적

SCR-071 진입 경로(사이드바/URL직접/알림/대시보드)와 권한 분기를 TC 원천으로 제공한다.

## 2. 전제조건

- 로그인 상태

## 3. 다이어그램

```mermaid
flowchart TD
  SIDEBAR([사이드바 > 마케팅 > 메시지 발송]) --> AUTH
  URL([URL 직접 입력 /message]) --> AUTH
  DASH([대시보드 바로가기]) --> AUTH
  NOTIF([알림 센터 링크]) --> AUTH

  AUTH{권한 확인} -->|E_071_F1_AUTH_OK primary|LOAD[메시지 화면 로드\n보유 포인트 + 발신 설정]
  AUTH -->|E_071_F1_AUTH_FAIL readonly|BLOCKED[접근 차단]

  LOAD -->|E_071_F1_LOAD_OK 200|SCR_071[SCR-071 메시지 발송\n탭: 메시지 전송 / 발송 이력]
```
