| 세션 만료 | 401 | 로그인 페이지 |
| 중복 수업명 | 409 | 경고 토스트 |
| 상태 전환 불가 | 400 | 경고 토스트 |
| 일괄변경 부분실패 | - | 경고 토스트 |
