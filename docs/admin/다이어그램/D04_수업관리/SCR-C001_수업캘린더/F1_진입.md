## 1. 목적
SCR-C001 수업/캘린더 화면으로 진입할 수 있는 모든 경로를 정의한다.

## 2. 전제조건
- 사용자가 로그인된 상태
- 수업관리 도메인 접근 권한 보유

## 3. 다이어그램

```mermaid
flowchart TD
  SB([사이드바 > 수업/캘린더 > 캘린더]) --> Auth{권한 확인}
  URL([직접 URL /calendar 접근]) --> Auth
  DASH([대시보드 위젯 > 오늘 수업]) --> Auth
  NOTIF([알림 센터 > 수업 관련 알림]) --> Auth
  SEARCH([글로벌 검색 > 수업명 결과]) --> Auth
  REQ([SCR-C009 일정요청처리 > 처리하기]) --> Auth

  Auth -->|": readonly 또는 권한 없음"|Blocked[권한없음 페이지]

  SCR_C001 --> TabSchedule[일정표 탭 활성]
```
