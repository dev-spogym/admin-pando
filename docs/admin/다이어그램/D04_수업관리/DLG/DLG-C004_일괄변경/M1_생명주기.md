## 1. 목적
DLG-C004 일괄변경 모달의 생명주기를 정의한다.

## 2. 전제조건
- SCR-C002에서 1개 이상 수업 선택 후 일괄변경 버튼 클릭

## 3. 다이어그램

```mermaid
flowchart TD
```
