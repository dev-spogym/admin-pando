## 1. 목적
DLG-C008 일괄생성 API 결과 분기를 정의한다.

## 2. 전제조건
- 확인 클릭 후 API 호출

## 3. 다이어그램

```mermaid
flowchart TD
  API{판단} -->|": 200 전체성공"|FullSuccess[전체 생성 완료]
  API -->|": 207 부분성공"|PartialSuccess[일부 생성 완료\n충돌 목록 표시]
  API -->|": 403"|PermErr[toast 권한이 없습니다.]
  API -->|": 500"|SysErr[toast 처리에 실패했습니다.]

  FullSuccess --> CloseAll[DLG-C008 닫힘\-C003 폼 리셋]
  CloseAll --> Toast_S[toast N개 수업이 생성되었습니다.]

  PartialSuccess --> ShowConflict[충돌 수업 목록 표시\n생성 성공 수업 수 + 실패 수 표시]
  ShowConflict -->|": 확인"|CloseAll

  PermErr --> Stay[모달 유지]
  SysErr --> Stay

  classDef system fill:#EDE7F6,stroke:#5E35B2
  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20

  class API system
  class FullSuccess,CloseAll,Toast_S success
  class SysErr error
  class PartialSuccess,PermErr,ShowConflict warning
  class Stay modal
```

## 4. 엣지 설명

| 응답 | 동작 |
|------|------|
| 200 | 전체 성공 → 닫힘 + success |
| 207 | 부분 성공 → 충돌 목록 표시 |
| 403/500 | 에러/경고 + 모달 유지 |
