## 1. 목적
DLG-C002 일정상세 모달의 열림/닫힘 생명주기를 정의한다.

## 2. 전제조건
- SCR-C001 캘린더 이벤트 클릭

## 3. 다이어그램

```mermaid
flowchart TD
  Trigger[캘린더 이벤트 클릭] --> Open[DLG-C002 열림\n400px sm]
  Open --> LoadDetail{LoadDetail -->|": 200"|Render[상세 정보 렌더링\n수업명/강사/시간/정원/참가자]
  LoadDetail -->|": 500"|LoadErr[toast 데이터 로드 실패]

  Render -->|": X/ESC/오버레이"|Close[모달 닫힘]
  Render -->|": 수정 버튼"|OpenEdit[DLG-C001 수정 모달 열림]
  Render -->|": 삭제 버튼"|DeleteConfirm[삭제 확인 다이얼로그]
  DeleteConfirm -->|": 확인"|DeleteAPI{DeleteAPI -->|": 200"|DeleteOK[모달 닫힘 + 캘린더 갱신]
  DeleteAPI -->|": 500"|DeleteErr[toast 삭제 실패]

  classDef modal fill:#FFF3E0,stroke:#F57C00,color:#E65100
  classDef system fill:#EDE7F6,stroke:#5E35B2

  class Open,Render,DeleteConfirm modal
  class Trigger,LoadDetail,DeleteAPI system

  class LoadErr,DeleteErr error
```
