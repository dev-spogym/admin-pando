## 1. 목적
DLG-C014 수동 페널티 등록 모달의 생명주기를 정의한다.

## 2. 전제조건
- SCR-C008 페널티관리에서 페널티 등록 버튼 클릭

## 3. 다이어그램

```mermaid
flowchart TD
  Trigger[페널티 등록 버튼] --> Open[DLG-C014 열림\n480px md]

  SaveAPI -->|": 500"|SysErr[toast 처리 실패]
```
