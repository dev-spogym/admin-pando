## 1. 목적
DLG-C013 이력 로드 및 엑셀 다운로드 결과 분기를 정의한다.

## 2. 전제조건
- API 호출 후

## 3. 다이어그램

```mermaid
flowchart TD
  HistoryAPI{-->|": 200"|HistoryOK[이력 테이블 렌더링]
  HistoryAPI -->|": 404"|NotFound[toast 세션을 찾을 수 없습니다.]
  HistoryAPI -->|": 500"|SysErr[toast 처리에 실패했습니다.]

  ExcelAPI{-->|": 200"|ExcelOK[toast N건 엑셀 다운로드]
  ExcelAPI -->|": 500"|ExcelErr[toast 다운로드 실패]

  NotFound --> CloseModal[모달 닫힘]
  SysErr --> StayModal[모달 유지 + 재시도]

  classDef system fill:#EDE7F6,stroke:#5E35B2
  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
  classDef error fill:#FFEBEE,stroke:#C62828,color:#B71C1C
  classDef warning fill:#FFF8E1,stroke:#F9A825,color:#F57F17
  classDef modal fill:#FFF3E0,stroke:#F57C00,color:#E65100
  class HistoryAPI,ExcelAPI system
  class HistoryOK,ExcelOK success
  class SysErr,ExcelErr error
  class NotFound warning
  class CloseModal,StayModal modal
```

## 4. 엣지 설명

| 응답 | 동작 |
|------|------|
| 200 | 이력 렌더링 |
| 404 | 경고 + 닫힘 |
| 500 | 에러 + 재시도 |
