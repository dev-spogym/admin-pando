# F6 상태별 화면 플로우 — SCR-051 사물함 배정 관리

## 1. 목적
로딩/빈/에러/오프라인 상태와 셀 5가지 상태를 정의한다.

## 2. 다이어그램

```mermaid
flowchart TD
    Entry([SCR-051 진입]) --> Auth{권한 확인}
    Auth -->|": FC/코치"|Blocked[403 권한없음]
    Auth -->|": 허용"|Loading[로딩 상태\]

    Loading -->|": 200 데이터있음"|Normal[정상 상태]
    Loading -->|": 200 데이터없음"|Empty[빈 상태\n락커 없음 + 추가 버튼]
    Loading -->|": 5xx"|ErrorState[에러 상태 + 재시도]
    Loading -->|": 네트워크 없음"|Offline[오프라인 배너]

    ErrorState -->|": 재시도"|Loading
    Offline -->|": 복구"|Loading
```

## 4. 엣지 설명

| 상태 | 전환 조건 |
|------|-----------|
| → 403 | FC/코치 역할 |
| → 정상 | 200 + 데이터 |
| → 빈상태 | 200 + 0건 |
| → 에러 | 5xx |
| → 오프라인 | 네트워크 없음 |
