## 1. 목적

SCR-061 정상 시나리오. 폼 입력→검증→저장 흐름. 성공/검증실패/시스템에러 3갈래 분기 강제.

## 2. 전제조건

- SCR-061 진입 완료 상태이다 (등록 또는 수정 모드).
