## 1. 목적
DLG-S002에서 회원 선택 결과를 SCR-S002로 전달하는 분기를 표현한다.

## 2. 전제조건
- DLG-S002에서 회원 선택 완료

## 3. 다이어그램

```mermaid
flowchart TD
    [회원 행 클릭] --> VALIDATE{회원 상태 확인}

    VALIDATE -->|": 활성 회원"|SET_BUYER[buyer 상태 설정\n이름/ID/연락처]
    VALIDATE -->|": 탈퇴/휴면 회원"|INACTIVE_WARN[비활성 회원 경고\n계속 진행하시겠습니까?]

    INACTIVE_WARN -->|": 확인"|SET_BUYER
    INACTIVE_WARN -->|": 취소"|BACK_SEARCH[검색 화면 유지]

    SET_BUYER --> CLOSED([DLG 닫힘\-S002 buyer 표시])

    classDef modal fill:#FFF3E0,stroke:#F57C00,color:#E65100
    classDef system fill:#EDE7F6,stroke:#5E35B2
    classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
    classDef warning fill:#FFF8E1,stroke:#F9A825,color:#F57F17

    class CLOSED modal
    class VALIDATE system
    class SET_BUYER success
    class INACTIVE_WARN warning
```

## 4. 엣지 설명

| 출발 | 도착 | 설명 |
|------|------|------|
| | VALIDATE | 회원 선택 → 상태 확인 |
| VALIDATE | SET_BUYER | 활성 회원 → 즉시 설정 |
| VALIDATE | INACTIVE_WARN | 비활성 회원 → 경고 |
| INACTIVE_WARN | SET_BUYER | 경고 확인 → 설정 진행 |
| SET_BUYER | CLOSED | buyer 설정 → 모달 닫힘 |
