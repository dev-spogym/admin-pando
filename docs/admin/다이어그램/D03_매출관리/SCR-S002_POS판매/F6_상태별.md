## 1. 목적
SCR-S002의 로딩/빈/에러/품절/장바구니 비어있음 등 모든 UI 상태를 표현한다.

## 2. 전제조건
- SCR-S002 진입 시도

## 3. 다이어그램

```mermaid
flowchart TD
    ENTRY([SCR-S002 진입]) --> PROD_LOAD{상품 로드 상태}

    PROD_LOAD -->|": "|SKEL[로딩 상태\n스켈레톤 카드 6개]
    PROD_LOAD -->|": 데이터 있음"|NORMAL[정상 상태\n상품 그리드 표시]
    PROD_LOAD -->|": 카테고리 빈값"|CAT_EMPTY[카테고리 빈 상태\ 아이콘\n해당 카테고리에 상품이 없습니다.]
    PROD_LOAD -->|": fetch 실패"|ERR_STATE[에러 상태\n빈 목록 표시\n별도 에러 UI 없음]

    NORMAL --> SOLDOUT_STATE{품절 상품}
    SOLDOUT_STATE -->|": stock=0"|SOLDOUT_CARD[품절 카드\nopacity-50 grayscale\ncursor-not-allowed]
    SOLDOUT_STATE -->|": stock"|NORMAL_CARD[정상 카드\n클릭 가능]

    NORMAL --> CART_STATE{장바구니 상태}
    CART_STATE -->|": 비어있음"|EMPTY_CART[빈 장바구니\ 아이콘\n장바구니가 비어 있습니다. opacity-50]
    CART_STATE -->|": 아이템 있음"|CART_FILLED[장바구니 채워짐\n총 합계 표시\n결제하기 활성]

    classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1

    class PROD_LOAD,SOLDOUT_STATE,CART_STATE system
```

## 4. 엣지 설명

| 출발 | 도착 | 설명 |
|------|------|------|
| PROD_LOAD | SKEL | 로딩 중 스켈레톤 6개 |
| PROD_LOAD | NORMAL | 정상 로드 |
| PROD_LOAD | CAT_EMPTY | 카테고리 상품 없음 |
| PROD_LOAD | ERR_STATE | fetch 실패 → 빈 목록 |
| SOLDOUT_STATE | SOLDOUT_CARD | 품절 → 비활성 카드 |
| CART_STATE | EMPTY_CART | 장바구니 비어있음 |
| CART_STATE | CART_FILLED | 아이템 있음 → 결제 활성 |
