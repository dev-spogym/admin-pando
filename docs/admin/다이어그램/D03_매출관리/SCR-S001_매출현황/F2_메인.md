## 1. 목적
SCR-S001 매출 현황의 7개 탭 전환 및 핵심 인터랙션 Happy Path를 표현한다. 탭 전환, 행 클릭, 버튼 동작의 3갈래 분기(성공/검증실패/시스템에러)를 포함한다.

## 2. 전제조건
- 로그인 완료, 매출 현황 진입 완료
- 기본 탭: TAB-001 매출내역

## 4. 엣지 설명
