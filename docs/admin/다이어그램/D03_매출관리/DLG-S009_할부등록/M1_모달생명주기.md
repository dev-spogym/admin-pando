## 1. 목적
DLG-S009 할부등록 모달(🆕)의 열기/닫기 생명주기를 표현한다.

## 2. 전제조건
- SCR-S009 할부결제관리에서 할부 등록 버튼 클릭

## 3. 다이어그램

```mermaid
flowchart TD
    CLOSED([모달 닫힘]) -->|": 할부 등록 클릭"|OPEN[DLG-S009 열림 🆕\n480px × 560px\n신규 할부 등록 폼]

    OPEN -->|": 등록 버튼"|VALIDATE[입력 검증]
    OPEN -->|": X 버튼"|CLOSED
    OPEN -->|": 취소"|CLOSED

    VALIDATE -->|": 검증 통과"|SAVE[동작]
    VALIDATE -->|": 검증 실패"|ERR_INLINE[인라인 오류 표시]

    SAVE -->|": 등록 성공"|SUCCESS([성공 토스트\n모달 닫힘\-S009 갱신])
    SAVE -->|": 서버 오류"|ERR_TOAST[에러 토스트]

    classDef newFeature fill:#F3E5F5,stroke:#7B1FA2,color:#4A148C,stroke-dasharray:4 4

    classDef error fill:#FFEBEE,stroke:#C62828,color:#B71C1C

    class OPEN newFeature
```

## 4. 엣지 설명

| 출발 | 도착 | 설명 |
|------|------|------|
| CLOSED | OPEN | 할부 등록 버튼 클릭 |
| OPEN | VALIDATE | 등록 버튼 → 검증 |
| VALIDATE | SAVE | 검증 통과 → API |
| SAVE | SUCCESS | 등록 성공 |
