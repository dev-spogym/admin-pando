## 1. 목적
DLG-S012 목표매출설정 모달(🆕)의 열기/닫기 생명주기를 표현한다.

## 2. 전제조건
- SCR-S011 매출예측에서 목표 설정 버튼 클릭

## 3. 다이어그램

```mermaid
flowchart TD
    CLOSED([모달 닫힘]) -->|": 목표 설정 클릭"|LOAD[기존 목표 로드\ 현재월]
    LOAD --> OPEN[DLG-S012 열림 🆕\n400px × 400px\n목표금액 입력]
    LOAD -->|": 목표 미설정"|OPEN_EMPTY[DLG-S012 열림\n빈 폼]

    OPEN -->|": 저장"|VALIDATE[금액 검증]
    OPEN_EMPTY --> VALIDATE
    OPEN --> CLOSED
    OPEN_EMPTY --> CLOSED

    VALIDATE --> UPSERT[UPSERT]
    VALIDATE --> ERR[금액 오류]

    UPSERT --> SUCCESS([성공 토스트\n모달 닫힘\-S011 갱신])
    UPSERT --> ERR_TOAST[에러 토스트]

    classDef newFeature fill:#F3E5F5,stroke:#7B1FA2,color:#4A148C,stroke-dasharray:4 4
    classDef system fill:#EDE7F6,stroke:#5E35B2
    classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
    classDef error fill:#FFEBEE,stroke:#C62828,color:#B71C1C

    class OPEN,OPEN_EMPTY newFeature
    class VALIDATE,UPSERT system
    class SUCCESS success
    class ERR,ERR_TOAST error
```

## 4. 엣지 설명

| 출발 | 도착 | 설명 |
|------|------|------|
| CLOSED | LOAD | 목표 설정 버튼 클릭 |
| LOAD | OPEN | 기존 목표 있음 |
| LOAD | OPEN_EMPTY | 목표 미설정 |
| UPSERT | SUCCESS | 저장 성공 |
