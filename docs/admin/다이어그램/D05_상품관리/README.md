## 도메인 연결점

- **상품 → 결제**: `` → `sales.` (POS 판매, SCR-S002)
- **상품 → 이용권**: `` → `memberships.` (회원 이용권 개시)
- **상품 가격 이력**: `audit_log` (, action='') → SCR-P003 가격이력
- **할인 중첩**: `discount_policies` → POS 결제 시 적용 (D3 매출관리)
- **재고 차감**: `inventory_logs` (type='OUT') → POS 판매 시 자동 차감
- **시즌 가격**: `seasonal_pricing` → 자동화 크론 A3 (매일 가격 적용/해제)
- **카탈로그**: `product_catalog` → 회원 앱 / 키오스크 노출 (D11 통합운영)
