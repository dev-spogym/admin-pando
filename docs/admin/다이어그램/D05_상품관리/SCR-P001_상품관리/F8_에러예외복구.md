# F8 에러/예외/복구 플로우 — SCR-P001 상품 관리

## 목적
SCR-P001에서 발생 가능한 모든 에러와 복구 경로를 정의한다. 가격이력 변경/할인 중첩/재고 차감 에러는 이 화면의 트리거 포인트를 포함한다.
