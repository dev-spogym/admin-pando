## 컨벤션

- **YAML 메타 헤더**: 각 파일 최상단 필수
- **액터 표기**: `actor`(사람), `participant`(시스템/서비스)
- **메시지 번호**: 각 화살표 앞 `1.`, `2.` 순번 + `%% E_` 주석
- **alt**: 정상 흐름 + 최소 1개 대안 흐름
- **TC 후보**: positive 최소 2개, negative 최소 2개
