## 1. 목적

출석이력 탭의 데이터 상태별 화면 분기를 정의한다.

## 2. 전제조건

- Tab03 출석이력 활성

## 3. 다이어그램

```mermaid
flowchart TD
  TAB[Tab03 출석이력] --> DATA{attendances 데이터}
  DATA -->|": 없음"|EMPTY[StatCard 0\n히트맵 빈 셀\n출석 기록이 없습니다.]
  DATA -->|": 있음"|NORMAL[StatCard 정상\n히트맵 셀 채워짐\n리스트 출력]

  classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
  classDef system fill:#EDE7F6,stroke:#5E35B2
  class TAB,NORMAL screen
  class DATA system
```

## 4. 엣지 설명

| 조건 | 화면 |
|------|------|
| 출석 없음 | 0값 StatCard, 빈 히트맵, 빈 상태 메시지 |
| 출석 있음 | 정상 히트맵 + 리스트 |
