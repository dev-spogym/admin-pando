## 1. 목적

신체정보 탭의 데이터 유무 및 차트 표시 상태별 화면 분기를 정의한다.

## 2. 전제조건

- Tab11 신체정보 활성

## 3. 다이어그램

```mermaid
flowchart TD
  TAB[Tab11 신체정보] --> LIST{records}
  LIST -->|": 0"|EMPTY[아이콘+신체정보가 없습니다.]
  LIST -->|": 1"|ONE[DataTable 1행\ 점 1개]
  LIST -->|": 2+"|MULTI[DataTable 다수\ 라인 표시]

  classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
  classDef system fill:#EDE7F6,stroke:#5E35B2
  class TAB,ONE,MULTI screen
  class LIST system
```

## 4. 엣지 설명

| 조건 | 화면 |
|------|------|
| 기록 없음 | 빈 상태 메시지 |
| 기록 1건 | 테이블 1행 + 차트 점 1개 |
| 기록 2건+ | 테이블 + 라인차트 |
