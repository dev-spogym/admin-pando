## 1. 목적

신체정보 탭은 별도 다이얼로그 없이 인라인 폼으로 처리하므로 트리거되는 모달이 없다.

## 2. 전제조건

- Tab11 신체정보 활성

## 3. 다이어그램

```mermaid

```

## 4. 엣지 설명

해당 없음 — 신체정보 탭은 인라인 폼을 사용하며 별도 모달을 트리거하지 않는다.
