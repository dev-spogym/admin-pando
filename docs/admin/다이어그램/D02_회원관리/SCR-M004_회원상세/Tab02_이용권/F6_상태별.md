## 1. 목적

이용권 탭의 계약 상태(없음/이용중/만료)별 화면 분기를 정의한다.

## 2. 전제조건

- Tab02 이용권 활성

## 3. 다이어그램

```mermaid
flowchart TD
  TAB[Tab02 이용권] --> CNT{판단}
  CNT -->|": 0"|EMPTY[빈 상태\n등록된 이용권이 없습니다.\n+ 신규 구매 버튼]
  CNT -->|": 1+"|LIST[카드 목록 렌더링]

  LIST --> CARD_STATE{contract}
  CARD_STATE -->|": ACTIVE"|ACTIVE_CARD[이용중 배지 success\n더보기: 연장/양도/환불]
  CARD_STATE -->|": 그외"|OTHER_CARD[상태 배지 default\n더보기: 환불만]

  classDef screen fill:#E3F2FD,stroke:#1976D2,color:#0D47A1
  classDef system fill:#EDE7F6,stroke:#5E35B2
  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
  class TAB,LIST screen
  class CNT,CARD_STATE system
  class ACTIVE_CARD success
```

## 4. 엣지 설명

| 조건 | 화면 |
|------|------|
| 없음 | 빈 상태 메시지 + 구매 버튼 |
| 있음 | 카드 목록 |
| ACTIVE 계약 | 이용중 배지, 연장/양도/환불 메뉴 |
| 비활성 계약 | 상태 배지, 환불만 표시 |
