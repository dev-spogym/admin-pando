## 2. 전제조건

## 3. 다이어그램

```mermaid
flowchart TD
  ACTIONS[Tab02 이용권 액션] --> T_OK[성공 토스트 success]
  ACTIONS --> T_ERR[에러 토스트 error]

  T_OK -->|": 연장 성공"|TS1((이용권이 연장되었습니다.))

  T_ERR -->|": 연장 실패"|TE1((연장 처리에 실패했습니다.))
  T_ERR -->|": 🆕 양도 실패"|TE2((양도 처리에 실패했습니다.))
  T_ERR -->|": 환불 실패"|TE3((환불 처리에 실패했습니다.))

  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
```
