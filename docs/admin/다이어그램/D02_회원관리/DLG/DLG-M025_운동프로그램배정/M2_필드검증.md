## 1. 목적

DLG-M025의 프로그램 선택 유효성 검증을 명세한다.

## 2. 트리거/전제조건

- DLG-M025 열린 상태

## 3. 다이어그램

```mermaid
flowchart TD
  DLG[DLG-M025 열림]

  DLG --> RADIO_LIST[프로그램 radio list 선택 필수]

  RADIO_LIST -->|선택|CHECK{있음}
  CHECK -->|있음|BTN_ACTIVE[배정 버튼 활성]
  CHECK -->|없음|BTN_DISABLED[배정 버튼 비활성 OR assigning=true]

  BTN_ACTIVE -->|클릭 assigning=false|SUBMIT[API 호출]
  BTN_DISABLED --> NO_ACTION[동작 없음]

  classDef system fill:#EDE7F6,stroke:#5E35B2
  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
  classDef error fill:#FFEBEE,stroke:#C62828,color:#B71C1C

  class DLG,RADIO_LIST,CHECK system
  class BTN_ACTIVE,SUBMIT success
  class BTN_DISABLED,NO_ACTION error
```
