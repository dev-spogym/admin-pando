## 1. 목적

DLG-M014 결제 상세 조회 다이얼로그의 열기/닫기 생명주기를 명세한다.

## 2. 트리거/전제조건

- 결제이력/결제내역 탭 > "상세" 버튼 클릭

## 3. 다이어그램

```mermaid
flowchart TD
  TRIGGER([결제이력 상세 버튼 클릭])

  TRIGGER --> DLG_OPEN[DLG-M014 결제 상세 max-w-400px]
  DLG_OPEN --> DISPLAY[결제 정보 표시 전용 key-value]

  DISPLAY --> USER_ACT{사용자 액션}

  USER_ACT -->|닫기 버튼|CLOSE[모달 닫기]
  USER_ACT -->|X 버튼|CLOSE
  USER_ACT -->|ESC 키|CLOSE

  classDef modal fill:#FFF3E0,stroke:#E65100,color:#BF360C
  classDef system fill:#EDE7F6,stroke:#5E35B2

  class DLG_OPEN,DISPLAY modal
  class CLOSE system
```

## 4. 엣지 설명

| 출발 | 도착 | 조건 |
|------|------|------|
| 상세 버튼 | 모달 열기 | - |
| 닫기 버튼 | 모달 닫기 | - |
| X 버튼 | 모달 닫기 | p-xs rounded-full |
| ESC 키 | 모달 닫기 | - |
