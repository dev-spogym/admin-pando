## 1. 목적

DLG-M008 폼 초기화 확인 다이얼로그의 열기/닫기/완료 생명주기를 명세한다.

## 2. 트리거/전제조건

- 회원 등록/수정 > "초기화" 버튼 클릭

## 3. 다이어그램

```mermaid
flowchart TD
  TRIGGER([초기화 버튼 클릭])

  TRIGGER --> DLG_OPEN[DLG-M008 ConfirmDialog 열기]

  DLG_OPEN --> USER_ACT{사용자 액션}

  USER_ACT -->|취소 클릭|CLOSE[모달 닫기 폼 유지]
  USER_ACT -->|초기화 클릭|RESET_ACTION[reset + / 초기화 + Step1 이동]

  RESET_ACTION --> TOAST_OK((toast 입력 내용이 초기화되었습니다))
  TOAST_OK --> CLOSE2[모달 닫기]

  classDef modal fill:#FFF3E0,stroke:#E65100,color:#BF360C
  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
  classDef system fill:#EDE7F6,stroke:#5E35B2

  class DLG_OPEN modal
  class TOAST_OK,CLOSE2 success
  class RESET_ACTION,CLOSE system
```

## 4. 엣지 설명

| 출발 | 도착 | 조건 |
|------|------|------|
| 초기화 버튼 | 모달 열기 | - |
| 취소 | 모달 닫기 | - |
| 초기화 | reset 실행 | 클릭 |
| reset | toast | - |
| toast | 모달 닫기 | - |
