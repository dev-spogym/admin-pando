## 1. 목적

DLG-M008은 입력 필드 없는 ConfirmDialog이며 초기화 후 상태 변경 사항을 명세한다.

## 2. 트리거/전제조건

- DLG-M008 열린 상태

## 3. 다이어그램

```mermaid
flowchart TD

  DLG --> BTN_RESET[초기화 danger 항상 활성]
  DLG --> BTN_CANCEL[취소 secondary 항상 활성]

  BTN_RESET -->|클릭|EXEC_RESET[reset 실행]

  BTN_CANCEL -->|클릭|CLOSE[모달 닫기]
```
