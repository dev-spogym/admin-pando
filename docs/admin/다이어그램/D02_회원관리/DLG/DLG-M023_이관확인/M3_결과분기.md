## 1. 목적

DLG-M023 이관 API 응답별 결과 분기를 명세한다.

## 2. 트리거/전제조건

- () API 호출 후 ( + )

## 3. 다이어그램

```mermaid
flowchart TD
  API[+]

  API -->|200|SUCCESS[이관 성공]
  API -->|오류|FAIL[이관 실패]

  SUCCESS --> TOAST_OK((toast name 회원이 으로 이관되었습니다))
  TOAST_OK --> REDIRECT[/ 이동]

  FAIL --> TOAST_ERR((toast 이관에 실패했습니다))
  TOAST_ERR --> CLOSE[모달 닫기]

  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
  classDef error fill:#FFEBEE,stroke:#C62828,color:#B71C1C
  classDef system fill:#EDE7F6,stroke:#5E35B2

  class SUCCESS,TOAST_OK,REDIRECT success
  class FAIL,TOAST_ERR,CLOSE error
  class API system
```
