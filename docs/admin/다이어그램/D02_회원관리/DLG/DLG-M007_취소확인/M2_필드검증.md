## 1. 목적

DLG-M007은 입력 필드 없는 ConfirmDialog이므로 조건만 명세한다.

## 2. 트리거/전제조건

- 취소 버튼 클릭 시점

## 3. 다이어그램

```mermaid
flowchart TD
  BTN_CANCEL[취소 버튼 클릭]

  BTN_CANCEL --> CHECK_DIRTY{확인}
  CHECK_DIRTY -->|true|SHOW_DLG[DLG-M007 표시]
  CHECK_DIRTY -->|false|SKIP[모달 없이 / 이동]

  SHOW_DLG --> BTN_CONTINUE[계속 작성 secondary]
  SHOW_DLG --> BTN_EXIT[취소하기 primary]

  classDef system fill:#EDE7F6,stroke:#5E35B2
  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20

  class CHECK_DIRTY system
  class SHOW_DLG,BTN_CONTINUE,BTN_EXIT,SKIP success
```

## 4. 엣지 설명

| 출발 | 도착 | 조건 |
|------|------|------|
| 확인 | 모달 표시 | true |
| 확인 | / 이동 | false |
