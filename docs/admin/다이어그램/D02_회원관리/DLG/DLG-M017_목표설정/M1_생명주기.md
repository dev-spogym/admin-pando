## 1. 목적

DLG-M017 목표 설정 다이얼로그의 열기/닫기/완료 생명주기를 명세한다. 🆕 미구현 기능.

## 2. 트리거/전제조건

- 체성분 탭 > "목표 설정" 버튼 클릭

## 3. 다이어그램

```mermaid
flowchart TD
  TRIGGER([체성분 탭 목표 설정 버튼 클릭])

  TRIGGER --> DLG_OPEN[DLG-M017 목표 설정 🆕]
  DLG_OPEN --> INIT[현재값 기본값 로드 체중/체지방률/골격근량/기한]

  INIT --> USER_ACT{사용자 액션}

  USER_ACT -->|취소|CLOSE[모달 닫기]
  USER_ACT -->|저장|API_CALL[동작]

  API_CALL -->|200|TOAST_OK((toast 목표가 저장되었습니다))
  TOAST_OK --> POST_OK[모달 닫기 + 차트 갱신]

  API_CALL -->|오류|TOAST_ERR((toast 저장에 실패했습니다))
  TOAST_ERR --> DLG_OPEN

  classDef newFeature fill:#F3E5F5,stroke:#9C27B0,stroke-dasharray:5 5
  classDef success fill:#E8F5E9,stroke:#2E7D32,color:#1B5E20
  classDef error fill:#FFEBEE,stroke:#C62828,color:#B71C1C
  classDef system fill:#EDE7F6,stroke:#5E35B2

  class DLG_OPEN,INIT newFeature
  class TOAST_OK,POST_OK success
  class TOAST_ERR error
  class API_CALL system
```

## 4. 엣지 설명

| 출발 | 도착 | 조건 |
|------|------|------|
| 목표 설정 버튼 | 모달 열기 | - |
| 취소 | 모달 닫기 | - |
| 저장 | API | 클릭 |
| API | toast | 200 |
| API | toast | 오류 |
