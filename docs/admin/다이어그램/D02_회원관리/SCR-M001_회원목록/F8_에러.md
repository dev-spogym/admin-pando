## 1. 목적

SCR-M001에서 발생 가능한 모든 에러/예외 케이스와 복구 경로를 명세한다. 네거티브 TC 원천.

## 2. 전제조건

- SCR-M001 진입 또는 조작 중 오류가 발생한 상태이다.

## 4. 엣지 설명 테이블

| 출발 | 도착 | 조건 |
|------|------|------|
| API | 네트워크 에러 | fetch 실패, 오프라인 |
| API | 세션 만료 | 401 Unauthorized |
| API | 권한없음 | 403 Forbidden |
| API | 서버 오류 | 500 Internal Server Error |
| API | 타임아웃 | 요청 시간 초과 |
| 네트워크 에러 | API 재호출 | 다시 시도 클릭 |
| 서버 오류 | API 재호출 | 다시 시도 클릭 |
| 타임아웃 | API 재호출 | 다시 시도 클릭 |
| 세션 만료 | 로그인 | 자동 리다이렉트 |
| 엑셀 전체 조회 | 현재 페이지만 | 전체 조회 실패 시 폴백 |
| 엑셀 전체 실패 | 실패 토스트 | 폴백도 실패 |
| 상태 변경 | 실패 토스트 | API 오류 |
| 출석 처리 | 실패 토스트 | API 오류 |
| 즐겨찾기 | 실패 토스트 | API 오류 |
