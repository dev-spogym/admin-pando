# 자동화/크론 다이어그램 인덱스

> 각 다이어그램은 "언제 / 무엇이 / 어떤 조건에서 / 어떤 결과로" 실행되는지 명확히 기술한다.
> TC "Given 자동화" 상태를 검증하는 원천 문서.

## TC 연계 안내

- 각 자동화 프로세스는 대응하는 TC의 **Given(사전 상태)** 을 자동으로 만드는 역할
- 자동화 TC 검증 시 크론 트리거를 수동으로 실행하거나 Mock 처리
- 엣지 ID `E_A{번호}_{출발}_{도착}_{일련}` 형식
