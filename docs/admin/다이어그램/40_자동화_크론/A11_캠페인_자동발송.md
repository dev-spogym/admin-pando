# A11 — 캠페인 자동 발송

## 1. 개요

| 항목 | 내용 |
|------|------|
| 트리거 | 크론 + 조건 기반 — 매시간 정각 |
| 대상 엔티티 | Campaign, Member, Message |
| 조건 | 발송 조건 충족 + 발송 시간 도래 |
| 결과 | 조건 일치 회원에게 메시지 자동 발송 |
| 관련 화면 | SCR-071 메시지 발송, SCR-072 자동 알림 설정 |

## 2. 발생 조건

- `Campaign = ACTIVE`
- `Campaign. <= NOW()`
- 세그먼트/태그 기반 대상자 필터
- 1인당 동일 캠페인 중복 발송 방지 (cooldown 기간)

## 3. 다이어그램

```mermaid
flowchart TD
    CRON([크론 트리거<br/>매시간 정각]) --> QUERY[발송 예정 캠페인 조회<br/>ACTIVE + <=NOW]

    QUERY -->|없음|EMPTY[발송 예정 없음<br/>종료]
    QUERY -->|있음|CAMP_LOOP[캠페인 순회]

    CAMP_LOOP --> TARGET_QUERY[대상자 조회<br/>세그먼트/태그 필터]
    TARGET_QUERY -->|없음|CAMP_SKIP[캠페인 스킵<br/>대상자 없음]
    TARGET_QUERY -->|있음|MEMBER_LOOP[회원 순회]

    MEMBER_LOOP --> OPTOUT_CHECK{수신 거부?}
    OPTOUT_CHECK -->|거부|MEM_SKIP[회원 스킵]
    OPTOUT_CHECK -->|수신|COOLDOWN_CHECK{쿨다운<br/>기간 이내?}

    COOLDOWN_CHECK -->|이내|COOL_SKIP[중복 방지 스킵]
    COOLDOWN_CHECK -->|이후|BUILD_MSG[메시지 빌드<br/>개인화 치환]

    BUILD_MSG --> CHANNEL{발송 채널}
    CHANNEL -->|SMS|SMS[EXT: SMS 발송]
    CHANNEL -->|카카오|KAKAO[EXT: 카카오 발송]
    CHANNEL -->|푸시|PUSH[EXT: 앱 푸시 발송]

    SMS & KAKAO & PUSH -->|성공|LOG_SEND[발송 이력 저장]
    SMS & KAKAO & PUSH -->|실패|RETRY{재시도<br/>?}
    RETRY --> CHANNEL
    RETRY -->|초과|LOG_FAIL[발송 실패 로그]

    MEM_SKIP & COOL_SKIP & LOG_SEND & LOG_FAIL --> NEXT_MEM{다음 회원?}
    NEXT_MEM --> MEMBER_LOOP
    NEXT_MEM -->|완료|UPDATE_CAMP[Campaign 발송 통계 업데이트<br/> /]

    CAMP_SKIP & UPDATE_CAMP --> NEXT_CAMP{다음 캠페인?}
    NEXT_CAMP --> CAMP_LOOP
    NEXT_CAMP -->|완료|END([크론 완료])

    classDef cron fill:#E0F2F1,stroke:#00695C
    classDef system fill:#EDE7F6,stroke:#5E35B2

    class CRON,END cron
    class QUERY,CAMP_LOOP,OPTOUT_CHECK,COOLDOWN_CHECK,CHANNEL,NEXT_MEM,NEXT_CAMP system
    class SMS,KAKAO,PUSH external

    class LOG_FAIL,RETRY error
    class MEM_SKIP,COOL_SKIP,CAMP_SKIP,EMPTY warning
```

## 4. 복구/재시도 전략

| 상황 | 전략 |
|------|------|
| 채널 발송 실패 | 최대 2회 재시도, 실패 로그 |
| 개인화 치환 실패 | 기본 메시지로 대체 발송 |
| 쿨다운 위반 | 중복 발송 방지 스킵 |
| 캠페인 대량 실패 | 관리자 알림, 일시 정지 가능 |
