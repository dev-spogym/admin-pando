# F2 메인 인터랙션 플로우 — SCR-100 로그인

## 목적
로그인 정상 흐름: 이메일/비밀번호 입력 → 2FA → 세션 생성 → 대시보드 (X29 시퀀스 참조)

## 다이어그램

```mermaid
flowchart TD
```
