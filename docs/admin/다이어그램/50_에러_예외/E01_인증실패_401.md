# E01 — 인증 실패 (401)

## 1. 개요

| 항목 | 내용 |
|------|------|
| 에러코드 | E401001 / E401002 / E401003 / E401004 |
| HTTP | 401 Unauthorized |
| 발생 모듈 | 공통 (인증) |
| 영향 화면 | SCR-100 로그인, 전체 인증 필요 화면 |

## 2. 발생 조건

| 에러코드 | 조건 |
|----------|------|
| E401001 | username/password 불일치 |
| E401002 | JWT 만료 (exp 초과) |
| E401003 | JWT 서명 불일치 또는 위조 |
| E401004 | (초기 로그인) |

## 4. 복구/재시도 전략

| 에러 | 복구 경로 |
|------|-----------|
| E401001 | 재입력 유도, 5회 실패 시 E403002 계정 잠금 |
| E401002 | 리프레시 토큰으로 자동 재발급 → 실패 시 로그인 |
| E401003 | 강제 로그아웃 → 로그인 |
| E401004 | 비밀번호 변경 화면 강제 이동 |

## 5. 사용자 노출 메시지

| 에러코드 | 사용자 메시지 |
|----------|-------------|
| E401001 | 아이디 또는 비밀번호가 일치하지 않습니다 |
| E401002 | 세션이 만료되었습니다. 다시 로그인해주세요 |
| E401003 | 인증 정보가 유효하지 않습니다 |
| E401004 | 비밀번호를 변경해주세요 |
