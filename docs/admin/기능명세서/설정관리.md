> 이 문서는 고객사 검토용 기획 문서입니다. 개발 구현 내용은 포함되지 않습니다.

---

## 권한/직무 해석 원칙

- 관리자웹 권한 키는 `superAdmin / primary / owner / manager / fc / staff / readonly`를 기준으로 해석한다.
- 본문에서 사용하는 `최고관리자`, `센터장`, `매니저`, `FC`, `스태프`는 운영 라벨 또는 예시 명칭으로 병기될 수 있다.
- `트레이너`, `골프강사`, `프런트`는 직원 직무 또는 운영 주체 표현이며, 이 문서의 기본 권한 템플릿과 동일 개념이 아니다.
- 실제 메뉴 접근 범위는 `SCR-081 권한 설정` 또는 `SCR-087 커스텀 역할 생성`에서 정의한 매핑 기준으로 결정한다.

## 1. 센터 설정

### 화면 목적
센터 운영에 필요한 모든 기본 정보를 한 곳에서 관리하는 설정 허브입니다.
기본정보, 알림 수신 방식, 화면 테마, 물품 재고를 탭별로 구분하여 관리합니다.

### 주요 기능

**기본정보 탭**
- 센터 로고 등록
- 센터명, 사업자등록번호, 센터 소개 입력
- 대표 연락처 및 주소 등록 (주소 검색 지원)
- 업종 선택 (헬스, 필라테스, PT샵, 골프, 요가, 태권도, 크로스핏, 복싱, 수영, 사우나, 기타 중 복수 선택)
- 평일 및 주말 운영시간 설정

**알림설정 탭**
- 앱 푸시 알림 ON/OFF (입장 알림, 만료 임박, 결제 완료, 수업 예약·취소)
- 이메일 알림 ON/OFF (주간 리포트, 만료 예정 안내)
- SMS 알림 ON/OFF (결제 완료, 만료 임박)

**테마설정 탭**
- 화면 모드 선택 (라이트 / 다크 / 시스템 설정)
- 메인 컬러 및 보조 컬러 설정
- 글자 크기 선택 (작게 / 기본 / 크게)

**물품 관리 탭**
- 물품별 재고 수량, 1일 발급 한도, 현재 잔여 수량 입력 관리
- 관리 물품: 수건(대), 수건(소), 운동복(상의), 운동복(하의)

### 계정별 차이
| 계정 | 권한 |
|---|---|
| 슈퍼관리자 | 전체 탭 조회 및 수정 |
| 센터장 | 전체 탭 조회 및 수정 |
| 매니저 | 알림설정 탭 조회만 가능 |
| 기타 | 접근 불가 |

### 비즈니스 규칙
- 변경 사항이 없으면 저장 버튼이 비활성 상태로 표시된다.
- 저장하지 않은 상태에서 다른 탭으로 이동하면 미저장 경고 다이얼로그가 표시된다.
- 저장 완료 메시지는 3초 후 자동으로 사라진다.

## 2. 권한 설정

### 화면 목적
관리자웹의 권한 템플릿별로 메뉴 접근 및 기능 사용 권한을 설정하는 화면입니다.
직원관리에서 등록하는 직무(트레이너, 골프강사, 프런트 등)와는 별도로, 기본 제공 권한 템플릿과 센터 맞춤형 역할의 메뉴별 권한을 매트릭스 형태로 관리합니다.

### 주요 기능
- 기본 제공 권한 템플릿 7종 조회 및 수정 (`superAdmin`, `primary`, `owner`, `manager`, `fc`, `staff`, `readonly`)
- 새 역할 생성, 기존 역할 복사, 역할 삭제
- 역할별 메뉴 × 권한(접근/조회/생성/수정/삭제) 매트릭스 편집
- 전체 허용 / 전체 거부 일괄 처리
- 권한 충돌 자동 감지 및 경고
- 마지막 저장 상태로 초기화

### 메뉴 그룹 (9그룹)
| 그룹 | 포함 메뉴 |
|---|---|
| 회원 | 회원 목록, 회원 상세, 회원 등록·수정 |
| 수업 | 수업·캘린더 |
| 매출 | 매출 현황, POS 결제 |
| 상품 | 상품 관리 |
| 직원 | 직원 관리 |
| 급여 | 급여 관리 |
| 시설 | 락커 관리, 운동룸 관리, 밴드·카드 관리 |
| 메시지 | 메시지 발송, 쿠폰 관리, 마일리지 관리 |
| 설정 | 센터 설정, 키오스크 설정, IoT 설정, 권한 설정 |

### 권한 항목 (5종)
| 권한 | 의미 |
|---|---|
| 접근 | 해당 메뉴 화면을 열 수 있는 권한 |
| 조회 | 메뉴 내 데이터를 볼 수 있는 권한 |
| 생성 | 새로운 데이터를 추가할 수 있는 권한 |
| 수정 | 기존 데이터를 변경할 수 있는 권한 |
| 삭제 | 데이터를 삭제할 수 있는 권한 |

### 계정별 차이
| 계정 | 권한 |
|---|---|
| 슈퍼관리자 | 전체 역할 권한 조회 및 수정 |
| 센터장 | 소속 지점 역할 권한 조회 및 수정 |
| 매니저 이하 | 접근 불가 |

### 비즈니스 규칙
- `superAdmin` 권한 템플릿은 모든 권한이 고정이며 수정할 수 없다.
- 직원 직무값은 이 화면에서 직접 관리하지 않으며, 직원관리에서 등록한 직무에 필요한 권한 템플릿을 별도 매핑한다.
- 삭제 권한이 있으나 수정 권한이 없는 경우 충돌 경고가 표시된다.
- 쓰기 권한(생성·수정·삭제)이 있으나 접근 또는 조회 권한이 없는 경우 충돌 경고가 표시된다.
- 역할에 배정된 직원이 있는 경우 역할 삭제 전 직원 재배정이 필요하다.

## 3. 키오스크 설정

### 화면 목적
센터 입구 키오스크의 화면 구성, 출석 방식, 표시 메시지를 설정합니다.

### 주요 기능
- 출석 인식 방법 설정 (QR코드, 회원번호 입력, 얼굴 인식 중 복수 선택)
- 기본 출석 방식 지정
- 키오스크 표시 언어 선택
- 배경 화면 및 센터 로고 표시 여부 설정
- 환영 메시지 및 출석 완료 메시지 문구 설정
- 회원이 키오스크에서 사용할 수 있는 기능 ON/OFF (이용권 잔여 조회, 수업 예약 조회, 락커 번호 확인, 공지사항 조회)
- 현재 설정 기준 키오스크 화면 미리보기

### 계정별 차이
| 계정 | 권한 |
|---|---|
| 슈퍼관리자 | 전체 설정 조회 및 수정 |
| 센터장 | 소속 지점 설정 조회 및 수정 |
| 매니저 | 조회만 가능 |
| 기타 | 접근 불가 |

### 비즈니스 규칙
- 설정 저장 시 연결된 키오스크 기기에 즉시 반영된다.
- 미리보기는 설정 변경 시 실시간으로 업데이트된다.

## 4. IoT 출입 관리

### 화면 목적
센터에 설치된 IoT 출입 장치를 등록·관리하고 출입 방식과 알림을 설정합니다.

### 주요 기능
- 등록된 IoT 기기 목록 조회 (기기명, 설치 위치, 연결 상태, 마지막 통신 시각)
- 기기 연결 상태 확인 (정상 / 오프라인 / 오류)
- 오프라인 기기 재연결 시도
- 새 기기 등록
- 출입 수단별 활성화 ON/OFF (스마트 밴드, QR코드, 얼굴 인식, 핀번호)
- 동일 회원 재입장 허용 간격 설정
- 출입 이력 보관 기간 설정
- 입장·퇴장·비정상 출입 시도 알림 발송 여부 설정

### 계정별 차이
| 계정 | 권한 |
|---|---|
| 슈퍼관리자 | 전체 기기 조회 및 설정 수정 |
| 센터장 | 소속 지점 기기 조회 및 설정 수정 |
| 매니저 | 연결 상태 조회만 가능 |
| 기타 | 접근 불가 |

### 비즈니스 규칙
- 출입 수단 설정 저장 시 연결된 모든 기기에 동시 반영된다.
- 기기가 오프라인 상태이면 재연결 버튼을 통해 수동으로 연결을 시도할 수 있다.

## 5. 구독 결제 관리

### 화면 목적
센터가 현재 이용 중인 서비스 구독 플랜과 결제 수단을 확인하고 관리합니다.

### 주요 기능
- 현재 구독 플랜 정보 조회 (플랜명, 다음 결제 예정일, 금액)
- 전체 플랜 비교 및 플랜 변경
- 구독 해지
- 결제 수단 등록·변경·삭제
- 과거 청구 이력 조회 (날짜, 플랜명, 금액, 상태)
- 청구 이력 기간 필터 (최근 3개월 / 6개월 / 1년)
- 영수증 PDF 다운로드

### 계정별 차이
| 계정 | 권한 |
|---|---|
| 슈퍼관리자 | 전체 구독 정보 조회 및 관리 |
| 센터장 | 소속 지점 구독 정보 조회 및 관리 |
| 매니저 이하 | 접근 불가 |

### 비즈니스 규칙
- 플랜 변경 전 변경 내용 및 요금 차액을 안내한다.
- 구독 해지 시 현재 결제 기간 만료일까지 서비스가 유지된다.
- 해지 후 데이터 보관 기간은 해지 안내 다이얼로그에서 명확히 고지한다.

## 6. 공지사항 관리

### 화면 목적
센터 내 공지사항을 작성·수정·삭제하고 게시 대상과 게시 기간을 설정합니다.

### 주요 기능
- 공지사항 목록 조회 (제목, 게시 대상, 게시 기간, 상태, 작성자, 작성일)
- 상태별 필터 (전체 / 게시 중 / 예정 / 종료)
- 키워드 검색 (제목 기준)
- 공지 등록·수정·삭제
- 게시 대상 설정 (전체 / 특정 역할 선택)
- 게시 기간 설정 (시작일시·종료일시, 무기한 설정 가능)
- 상단 고정 여부 설정

### 계정별 차이
| 계정 | 권한 |
|---|---|
| 슈퍼관리자 | 전체 지점 공지 조회·작성·수정·삭제 |
| 센터장 | 소속 지점 공지 조회·작성·수정·삭제 |
| 매니저 | 소속 지점 공지 조회·작성·수정 (삭제 불가) |
| 기타 | 수신 대상 포함 시 조회만 가능 |

### 비즈니스 규칙
- 삭제한 공지는 복구할 수 없다.
- 게시 종료일시가 지난 공지는 자동으로 "종료" 상태로 전환된다.
- 상단 고정 공지는 목록 최상단에 표시된다.

## 7. 출석 관리 설정

### 화면 목적
회원 출석을 인정하는 기준과 자동 처리 규칙을 설정합니다.

### 주요 기능
- 출석 인정 시작·마감 시각 설정 (수업 시작 기준 N분 전·후)
- 주말·공휴일 출석 인정 여부 설정
- 지각 판단 기준 시간 설정
- 지각 횟수 누적 알림 기준 설정
- IoT 기반 자동 출석 처리 활성화 여부
- 수업 없이 자유 이용 입장 시 출석 인정 여부
- 동일 회원 하루 중복 출석 처리 방식 (첫 입장만 / 매 입장마다)
- 출석 이력 보관 기간 설정

### 계정별 차이
| 계정 | 권한 |
|---|---|
| 슈퍼관리자 | 전체 지점 설정 조회 및 수정 |
| 센터장 | 소속 지점 설정 조회 및 수정 |
| 매니저 | 조회만 가능 |
| 기타 | 접근 불가 |

### 비즈니스 규칙
- 자동 출석 처리 활성화 시 IoT 출입 관리(4번)에서 기기가 연결되어 있어야 정상 동작한다.
- 출석 인정 시간 외 입장 기록은 출석으로 처리되지 않는다.

## 8. 커스텀 역할 생성 (미구현)

### 화면 목적
기본 제공 역할 외에 센터 운영 특성에 맞는 맞춤형 역할을 생성·관리합니다.

### 주요 기능
- 커스텀 역할 목록 조회 (역할명, 설명, 배정 직원 수, 생성일)
- 새 역할 생성 (이름·설명 입력, 기존 역할에서 권한 복사 선택 가능)
- 역할별 권한 매트릭스 편집
- 역할 수정·삭제
- 역할 삭제 시 배정된 직원을 다른 역할로 재배정

### 계정별 차이
| 계정 | 권한 |
|---|---|
| 슈퍼관리자 | 전체 커스텀 역할 생성·수정·삭제 |
| 센터장 | 소속 지점 커스텀 역할 생성·수정·삭제 |
| 매니저 이하 | 접근 불가 |

### 비즈니스 규칙
- 생성된 커스텀 역할은 권한 설정(2번) 화면에서 선택 가능하다.
- 배정된 직원이 있는 역할을 삭제하려면 직원 재배정이 선행되어야 한다.

## 9. 다국어 설정 (미구현)

### 화면 목적
시스템 전체 언어와 국가·지역 표시 형식을 설정합니다.

### 주요 기능
- 관리자 화면 언어 선택 (한국어, 영어, 일본어, 중국어 간체)
- 키오스크 화면 언어 선택 (복수 선택 가능)
- 날짜·시간·화폐 표시 형식 선택
- 시간대(Timezone) 설정
- 공지사항 자동 번역 활성화 여부

### 계정별 차이
| 계정 | 권한 |
|---|---|
| 슈퍼관리자 | 전체 설정 조회 및 수정 |
| 센터장 | 소속 지점 설정 조회 및 수정 |
| 매니저 | 조회만 가능 |
| 기타 | 접근 불가 |

### 비즈니스 규칙
- 관리자 화면 언어 변경 저장 시 화면 전체가 즉시 선택한 언어로 전환된다.
- 키오스크 언어 변경 저장 시 연결된 키오스크 기기에 즉시 반영된다.

## 10. 데이터 백업·복원 (미구현)

### 화면 목적
운영 데이터를 안전하게 보관하기 위한 자동 백업 주기를 설정하고, 필요 시 이전 시점으로 데이터를 복원합니다.

### 주요 기능
- 자동 백업 활성화·비활성화
- 백업 주기 설정 (매일 / 매주 / 매월)
- 백업 실행 시각 및 데이터 보관 기간 설정 (30일 / 90일 / 180일 / 무제한)
- 수동 백업 즉시 실행
- 최신 백업 데이터 다운로드
- 백업 이력 목록 조회 (날짜, 크기, 방식, 상태)
- 이력별 복원 실행 및 다운로드
- 복원 이력 조회 (복원 일시, 복원 시점, 실행자)

### 계정별 차이
| 계정 | 권한 |
|---|---|
| 슈퍼관리자 | 전체 설정 조회, 백업 실행, 복원 실행 |
| 센터장 | 백업 조회 및 수동 백업 실행 (복원은 슈퍼관리자 승인 필요) |
| 매니저 이하 | 접근 불가 |

### 비즈니스 규칙
- 데이터 복원은 현재 데이터를 덮어쓰며 되돌릴 수 없다.
- 복원 전 확인 텍스트 입력("복원")을 통해 실수 방지 절차를 거친다.
- 복원 완료 후 시스템이 재시작된다.
