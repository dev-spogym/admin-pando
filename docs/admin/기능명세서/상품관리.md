## 권한/직무 해석 원칙

- 관리자웹 권한 키는 `superAdmin / primary / owner / manager / fc / staff / readonly`를 기준으로 해석한다.
- 본문에서 보이는 `최고관리자`, `센터장`, `매니저`, `FC`, `스태프`는 운영 라벨 또는 예시 명칭으로 병기될 수 있다.
- 상품 판매를 담당하는 `트레이너`, `프런트` 같은 표현은 판매/응대 주체를 뜻하며, 관리자웹 기본 권한 키와 동일 개념이 아니다.
- 실제 메뉴 접근 범위는 `SCR-081 권한 설정` 또는 커스텀 역할 매핑 기준으로 결정한다.

## 1. 상품 관리

**화면**: SCR-P001 | **경로**: `/products`

### 화면 목적

센터에서 판매하는 모든 상품(이용권, PT 수강권, GX 수강권, 대여권, 일반 상품 등)을 한눈에 파악하고 관리하는 메인 화면입니다.

### 주요 기능

#### 상단 통계 카드

| 카드 | 내용 |
|------|------|
| 전체 상품 수 | 등록된 상품 총 개수 |
| 활성 상품 수 | 현재 판매 중인 상품 수 |
| 비활성 상품 수 | 판매 중지된 상품 수 |
| 카테고리 수 | 등록된 상품 분류 개수 |

#### 3단계 필터

| 단계 | 유형 | 옵션 |
|------|------|------|
| 1단계 | 상품 유형 탭 | 전체 / 회원권 / 수강권 / 대여권 / 일반 |
| 2단계 | 상품 대분류 | 전체 / 이용권 / PT / GX / 상품 / 서비스 |
| 3단계 | 종목·세부종목 필터 | 2단계 선택값에 따라 동적 표시. GX 선택 시 요가 / 필라테스 / 스피닝 / 줌바 / 에어로빅 / GX 기타 |

- 상품명 검색 기능 제공

#### 상품 마스터 분류 데이터 모델

상품 마스터의 분류 체계는 **2단계 구조**를 정본으로 합니다.

| 단계 | 기획 필드 | 저장 기준 | 설명 |
|------|-----------|-----------|------|
| 1단계 | 상품 대분류 | `category` | 상품의 큰 유형. `MEMBERSHIP` / `PT` / `GX` / `PRODUCT` / `SERVICE` |
| 2단계 | 종목·세부종목 | `subCategory` 또는 `productGroupId` | 대분류 안에서 실제 운영·통계에 쓰는 세부 분류 |

GX 상품은 2단계 세부종목을 필수로 선택합니다.

| 1단계 `category` | 2단계 세부종목 옵션 |
|------|------|
| `GX` | 요가 / 필라테스 / 스피닝 / 줌바 / 에어로빅 / GX 기타 |

운영 규칙:

- `category=GX`인 상품은 반드시 2단계 세부종목을 가져야 합니다.
- GX 세부종목은 위 6개 값으로 고정하며, 매출통계와 KPI 대시보드의 GX 세부 종목 분포 집계 기준으로 사용합니다.
- 기존 `productGroupId`가 있는 구현에서는 `product_groups.name`을 2단계 세부종목으로 해석합니다.
- 신규 DB 확장 시에는 `subCategory` 필드를 명시적으로 추가하거나 `productGroupId`가 위 6개 GX 그룹 중 하나를 참조하도록 제한합니다.

#### 상품 목록 테이블

| 컬럼 | 내용 |
|------|------|
| 상품명 | 유형 배지 + 패키지 배지 + 상품명 |
| 현금가 | 현금 결제 시 가격 |
| 카드가 | 카드 결제 시 가격 |
| 기간 | 이용 가능 기간 |
| 횟수 | 이용 가능 횟수 |
| 요일 (월~일) | 이용 가능 요일 (체크/엑스) |
| 시간 | 이용 가능 시간대 |
| 홀딩 | 일시정지 가능 여부 |
| 양도 | 양도 가능 여부 |
| 포인트 | 포인트 적립 여부 |
| 판매 | 판매 건수 |
| 상태 | 사용 / 미사용 |

> 우측 상세 패널이 열리면 요일/시간/홀딩/양도/포인트/판매 컬럼은 자동으로 숨겨집니다.

#### 분류 관리 탭

- 분류명, 정렬 순서, 활성 여부 등록
- 분류 목록 확인 및 수정·삭제

### 계정별 권한

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|-----------|--------------|--------------|
| 일반 직원 | 상품 목록 조회, 통계 | 조회만 가능 |
| 상품 편집 권한 보유 직원 | 전체 화면 | 상품 등록·수정·삭제, Excel 내보내기 |
| 슈퍼관리자 | 전체 화면 + 전 지점 배포 | 상품 관리 전체 + 지점 간 배포 |

### 주요 UX 흐름

1. 화면 진입 → 전체 상품 목록 표시
2. 필터/검색으로 원하는 상품 찾기
3. 상품 행 클릭 → 우측 상세 패널 열림 (SCR-P003)
4. "+ 상품 등록" 버튼 → 등록 패널 열림 (SCR-P002)
5. "분류 관리" 탭 → 상품 분류 관리
6. "전 지점 배포" 버튼 (슈퍼관리자) → DLG-P001-전지점배포
7. "Excel" 버튼 → 현재 필터된 목록 엑셀 파일로 다운로드

### 화면 상태

| 상태 | 조건 | 표시 내용 |
|------|------|-----------|
| 로딩 중 | 데이터 불러오는 중 | 목록 영역 로딩 표시 |
| 빈 상태 | 상품 없음 또는 필터 결과 없음 | "상품이 없습니다." + 등록 링크(권한 있는 경우) |
| 정상 | 상품 목록 존재 | 상품 목록 테이블 표시 |
| 패널 열림 | 상품 선택 또는 등록 버튼 클릭 | 목록 55% + 우측 패널 45% |

## 2. 상품 등록/수정

**화면**: SCR-P002 | **경로**: `/products/new` 또는 `/products/edit`

### 화면 목적

새 상품을 등록하거나 기존 상품을 수정하는 패널입니다. 상품 구분에 따라 필요한 항목이 동적으로 표시됩니다.

### 상품 구분별 표시 항목

| 구분 | 추가 표시 항목 |
|------|----------------|
| 레슨 | 레슨 시간, 레슨 유효기간, 수업구분 (개인/정규클래스) |
| 이용 | 이용구분 (기간/횟수/포인트), 기간 또는 횟수 입력 |
| 락커 | 기간 입력 |
| 판매 | 태그 입력 |

### 기본 입력 항목

| 항목 | 필수 | 설명 |
|------|------|------|
| 상품 대분류 | 필수 | `MEMBERSHIP` / `PT` / `GX` / `PRODUCT` / `SERVICE` 중 선택 |
| 종목·세부종목 | 조건부 필수 | 대분류별 세부 분류. `GX` 선택 시 요가 / 필라테스 / 스피닝 / 줌바 / 에어로빅 / GX 기타 중 필수 선택 |
| 상품명 | 필수 | 센터 내 중복 불가 |
| 금액 | 필수 | 현금가 기준 (0 초과) |
| 설명 | 선택 | 상품 부가 설명 |
| 태그 | 선택 | 검색·분류용 자유 입력 |

### 이용 요일·시간 설정

- 요일별 체크박스로 이용 가능 요일 지정
- 체크된 요일에 시작·종료 시간 지정 (10분 단위)

### 옵션 설정

| 옵션 | 설명 |
|------|------|
| 예약 가능 | 시설/수업 예약 허용 |
| 시설 이용 | 시설 입장 허용 |
| 홀딩 가능 | 이용권 일시정지 허용 |
| 양도 가능 | 이용권 타인 양도 허용 |
| 포인트 적립 | 구매 시 포인트 적립 |
| 회원 직접 휴회 | 회원이 앱에서 직접 휴회 신청 가능 |
| 키오스크 노출 | 키오스크 화면에 상품 노출 |
| 활성 상태 | 판매 활성/비활성 |

### 조건부 상세 옵션

| 항목 | 표시 조건 | 선택지 |
|------|-----------|--------|
| 시설 이용 시간 | 시설 이용 체크 시 | 기본 / 오전 / 오후 / 종일 |
| 예약 오픈일 | 예약 가능 체크 시 | 당일 / 1일 전 / 3·7·14·30일 전 |
| 예약 시간 간격 | 예약 가능 체크 시 | 10~60분 (10분 단위) |
| 휴회 횟수 | 회원 직접 휴회 체크 시 | 1회 / 2회 / 3회 |
| 휴회 기간 | 회원 직접 휴회 체크 시 | 3일 / 7일 / 15일 / 30일 / 60일 |
| 판매 유형 | 항상 | 전체 / 카운터 / 키오스크 / 온라인 |
| 일일 횟수 제한 | 횟수 이용구분 시 | 1~10회 |

### 패키지 구성

- "패키지 상품으로 등록" 체크 시 활성화
- 구성 상품 선택 및 추가
- 합산 정가 자동 계산, 패키지 할인가 별도 입력

### 계정별 권한

| 계정 유형 | 접근 여부 | 할 수 있는 것 |
|-----------|-----------|--------------|
| 상품 편집 권한 없는 직원 | 진입 불가 | 없음 |
| 상품 편집 권한 보유 직원 | 가능 | 등록, 수정, 삭제 |
| 슈퍼관리자 | 가능 | 등록, 수정, 삭제 |

### 화면 상태

| 상태 | 표시 내용 |
|------|-----------|
| 등록 모드 | 헤더 "상품 등록", 저장 버튼 "등록", 삭제 버튼 없음 |
| 수정 모드 | 헤더 "상품 수정" + 상태 배지, 저장 버튼 "저장", 삭제 버튼 표시 |
| 저장 중 | 저장 버튼 로딩 상태, 입력 비활성화 |
| 삭제 확인 중 | 하단 인라인 확인 버튼 표시 |
| 비활성화 확인 중 | 이용 중 회원 수 안내 + 미사용 전환 확인 |

## 3. 상품 상세 패널

**화면**: SCR-P003 | **경로**: `/products/detail` (우측 슬라이드 패널)

### 화면 목적

상품 목록에서 상품을 클릭하면 우측에 슬라이드로 나타나는 상세 정보 패널입니다.

### 표시 정보

| 영역 | 내용 |
|------|------|
| 기본 정보 | 상품명, 상품 대분류, 종목·세부종목, 가격, 기간/횟수, 수업구분, 이용구분 |
| 요일·시간 | 이용 가능 요일 및 시간대 |
| 옵션 현황 | 예약/시설/홀딩/양도/포인트/키오스크 등 활성 여부 |
| 패키지 구성 | 구성 상품 목록, 개별 가격, 합산 정가, 패키지 할인가 |
| 가격 변경 이력 | 변경 일시, 변경 전→후 가격, 변경자 |
| 판매 현황 | 누적 판매 건수 |

### 계정별 권한

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|-----------|--------------|--------------|
| 일반 직원 | 상세 정보 조회 | 조회만 가능 |
| 상품 편집 권한 보유 직원 | 전체 정보 + 수정/삭제 버튼 | 수정, 삭제 |

### 화면 상태

| 상태 | 표시 내용 |
|------|-----------|
| 정상 조회 | 전체 상세 정보 |
| 가격 이력 로딩 | "불러오는 중..." 텍스트 |
| 삭제 확인 중 | 하단 인라인 확인 버튼 |
| 비활성화 확인 중 | 이용 중 회원 수 안내 + 미사용 전환 확인 |

## 4. 할인 설정

**화면**: SCR-P004 | **경로**: `/discount-settings`

### 화면 목적

상품 판매 시 적용할 할인 정책을 등록·수정·삭제하고 변경 이력을 확인하는 화면입니다.

### 할인 유형

| 유형 | 설명 | 예시 |
|------|------|------|
| 정률 | 정가의 일정 비율 할인 | 10% 할인 |
| 정액 | 정해진 금액 할인 | 50,000원 할인 |

### 할인 정책 구성 항목

| 항목 | 필수 | 설명 |
|------|------|------|
| 할인명 | 필수 | 예: "재등록 10% 할인" |
| 할인 유형 | 필수 | 정률(%) / 정액(원) |
| 할인 값 | 필수 | 할인율 또는 할인 금액 |
| 최소 계약 기간 | 선택 | N일 이상 계약 시에만 적용 |
| 최대 할인금액 | 선택 | 할인 적용 한도 금액 |
| 활성 상태 | 선택 | 기본값: 활성 |

### 목록 표시 항목

| 컬럼 | 내용 |
|------|------|
| 할인명 | 정책 이름 |
| 유형 | 정률 / 정액 배지 |
| 할인 값 | 할인율(%) 또는 금액(원) |
| 조건 | 최소 계약 기간 (없으면 "-") |
| 한도 | 최대 할인금액 (없으면 "-") |
| 상태 | 활성 / 비활성 |
| 액션 | 수정 / 삭제 버튼 |

### 변경 이력

- 등록·수정·삭제 내역을 최근 순으로 타임라인 표시
- 이력이 없으면 타임라인 섹션 자체 비표시

### 계정별 권한

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|-----------|--------------|--------------|
| 일반 직원 | 할인 목록 + 변경 이력 | 조회만 가능 |
| 관리자 이상 | 전체 화면 | 등록, 수정, 삭제 |

### 화면 상태

| 상태 | 표시 내용 |
|------|-----------|
| 로딩 중 | 테이블 로딩 표시 |

## 5. 상품 카탈로그
