# D08 마케팅 기능명세서

> **프로젝트**: Pando CRM
> **도메인**: D08 마케팅
> **작성일**: 2026-04-26
> **대상**: 기획자·고객사
> **포함 화면**: SCR-070 ~ SCR-079, DLG-070-001 ~ DLG-079-002

---

## 도메인 개요

마케팅 도메인은 잠재 고객(리드) 관리부터 회원 메시지 발송, 자동 알림, 쿠폰·마일리지 운영, 전자 계약까지 회원 유치와 리텐션에 필요한 모든 마케팅 업무를 지원합니다. SCR-076(캠페인 관리), SCR-077(리퍼럴 프로그램), SCR-078(SMS/카카오 대량 발송), SCR-079(A/B 테스트)는 향후 개발 예정 화면으로 현재 미구현 상태입니다.

## 권한/직무 해석 원칙

- 관리자웹 권한 키는 `superAdmin / primary / owner / manager / fc / staff / readonly`를 기준으로 해석한다.
- 본문에서 보이는 `센터장`, `매니저`, `FC`, `스태프`는 운영 라벨 또는 예시 명칭으로 병기될 수 있다.
- `트레이너`, `프런트`는 마케팅 협업 주체나 현장 직무를 뜻하며, 관리자웹 기본 권한 키와 동일 개념이 아니다.
- 실제 메뉴 접근 범위는 `SCR-081 권한 설정` 또는 커스텀 역할 매핑 기준으로 결정한다.

---

## 계정 권한 요약

| 계정 유형 | 접근 가능 화면 | 주요 권한 |
|---|---|---|
| 센터장 / 매니저 | 전체 마케팅 화면 | 리드 관리, 메시지 발송, 쿠폰·마일리지 전체 관리, 전자 계약 |
| FC | 리드 관리, 메시지 발송, 전자 계약 | 담당 리드 관리, 담당 회원 메시지 발송, 계약 작성 |
| 트레이너 / 스태프 | 접근 제한 | 마케팅 도메인 접근 불가 |

## 기능 목록

### 1. 리드 관리 (SCR-070)

**목적**: 아직 회원이 아닌 잠재 고객을 등록하고 상담 단계별 진행 상황을 추적해 회원 전환율을 높입니다.

**주요 기능**:
- 전체 리드 수, 전환 수, 전환율, 대기 건수, 미전환율을 요약 카드로 표시합니다.
- 신규·연락완료·상담예정·방문완료·등록완료·미전환·보류 7개 상태별 분포를 카드로 표시합니다.
- 상태, 유입경로 필터와 이름·연락처·담당FC 검색을 조합해 원하는 리드를 찾습니다.
- 목록 뷰(테이블)와 칸반 뷰(상태별 컬럼) 사이를 전환할 수 있습니다.
- 신규 리드를 등록하거나 기존 리드를 수정합니다 (DLG-070-001).
- 리드 삭제 전 확인 다이얼로그(DLG-070-002)가 표시됩니다.
- 리드 상태가 "등록완료"로 변경되면 회원 전환율 통계에 자동 반영됩니다.
- 유입경로는 간판·인터넷·전단지·추천·SNS·카카오톡·전화문의·방문·기타 9종을 지원합니다.

**화면**: SCR-070 리드 관리
**관련 다이얼로그**: DLG-070-001 리드 등록/수정, DLG-070-002 리드 삭제 확인

---

### 2. 메시지 발송 (SCR-071)

**목적**: 회원에게 SMS, 카카오 알림톡, 앱 푸시 메시지를 직접 작성해 발송하고 이력을 관리합니다.

**주요 기능**:
- "메시지 전송" 탭과 "발송 이력" 탭으로 구성됩니다.
- 잔여 발송 포인트를 화면에 표시합니다.
- 수신자를 조건 그룹(전체·활성·만료 등) 또는 개별 검색(DLG-071-001)으로 지정합니다.
- 알림톡·SMS/LMS·앱 푸시 중 발송 채널을 선택합니다.
- 수신자 수와 채널에 따른 예상 비용이 실시간으로 표시됩니다.
- 메시지 내용에 회원명·만료일 등 변수를 삽입할 수 있습니다.
- 발송 전 미리보기(DLG-071-002)로 실제 수신 화면을 확인합니다.
- 즉시 발송 또는 원하는 일시에 예약 발송합니다.
- 발송 이력 탭에서 과거 발송 결과(성공·실패 건수 포함)를 조회합니다.
- 수신자 미선택, 메시지 미입력, 포인트 부족 시 발송 버튼이 비활성화됩니다.

**화면**: SCR-071 메시지 발송
**관련 다이얼로그**: DLG-071-001 수신자 검색, DLG-071-002 발송 미리보기

---

### 3. 자동 알림 설정 (SCR-072)

**목적**: 이용권 만료 임박, 생일, 장기 미방문 등 조건 충족 시 자동으로 메시지가 발송되는 규칙을 설정합니다.

**주요 기능**:
- 고객 관련 자동 알림(이용권 만료 임박·생일 축하·장기 미방문·등록 감사·첫 방문 환영)과 상품 관련 자동 알림(만료 당일·홀딩 종료·결제 완료·환불 처리) 규칙을 토글로 관리합니다.
- 각 규칙의 "상세 설정"(DLG-072-001)에서 발송 시점·채널·메시지 내용을 세부 설정합니다.
- 전체 자동 알림을 한 번에 활성화하거나 비활성화합니다.
- 한 번 설정하면 조건이 충족되는 회원에게 자동으로 발송됩니다.

**화면**: SCR-072 자동 알림 설정
**관련 다이얼로그**: DLG-072-001 알림 규칙 편집

---

### 4. 쿠폰 관리 (SCR-073)

**목적**: 할인 쿠폰을 생성·관리하고 특정 회원 또는 그룹에게 발급해 마케팅 효과를 높입니다.

**주요 기능**:
- 전체 쿠폰 수, 발급 건수, 사용 건수, 평균 사용률을 요약 카드로 표시합니다.
- 쿠폰을 생성(DLG-073-001)할 때 쿠폰명, 할인 유형(정액·정률), 유효 기간, 발급 수량 제한, 1인 사용 제한을 설정합니다.
- 생성된 쿠폰을 특정 회원 또는 그룹에게 발급합니다(DLG-073-002).
- 쿠폰 삭제 전 확인 다이얼로그(DLG-073-003)가 표시되며, 발급된 미사용 쿠폰도 함께 무효화됩니다.
- 상태(활성·만료·중단)로 쿠폰을 필터링합니다.
- 만료된 쿠폰의 발급 버튼은 자동으로 비활성화됩니다.

**화면**: SCR-073 쿠폰 관리
**관련 다이얼로그**: DLG-073-001 쿠폰 생성/수정, DLG-073-002 쿠폰 발급, DLG-073-003 쿠폰 삭제 확인

---

### 5. 마일리지 관리 (SCR-074)

**목적**: 회원별 마일리지 잔액과 적립·차감 이력을 관리하고 적립 규칙을 설정해 회원 재방문을 유도합니다.

**주요 기능**:
- 총 발행·사용·소멸 예정·유통 잔액을 요약 카드로 표시합니다.
- 회원별 마일리지 잔액, 총 적립, 총 사용, 최근 적립일을 목록으로 조회합니다.
- 특정 회원 선택 시 날짜·구분·금액·사유·처리자 단위의 상세 이력을 확인합니다.
- 관리자가 직접 마일리지를 적립하거나 차감합니다(DLG-074-002). 사유 입력이 필수이며 이력에 기록됩니다.
- 결제 금액 대비 적립률, 특별 이벤트 적립 조건, 소멸 정책을 설정합니다(DLG-074-001).
- 조회 결과를 엑셀 파일로 내려받습니다.

**화면**: SCR-074 마일리지 관리
**관련 다이얼로그**: DLG-074-001 적립 규칙 편집, DLG-074-002 수동 적립·차감

---

### 6. 전자 계약 (SCR-075)

**목적**: 회원과의 이용 계약을 전자 방식으로 작성하고 서명을 받아 계약 이력으로 관리합니다.

**주요 기능**:
- 이용권·PT·필라테스 등 계약 유형을 선택합니다.
- 계약 대상 회원을 검색해 지정합니다.
- 계약 기간, 이용 서비스, 금액, 결제 조건 등 핵심 항목을 입력합니다.
- 입력된 정보가 반영된 계약서 전문을 미리보기로 확인합니다.
- 회원이 화면에서 직접 서명하거나, 서명 링크를 전송해 원격 서명을 받습니다.
- 서명 완료 후 계약이 자동으로 저장되어 이력에서 조회할 수 있습니다.
- 완료된 계약서를 이메일·앱으로 재발송할 수 있습니다.

**화면**: SCR-075 전자 계약

---

### 7. 캠페인 관리 (SCR-076) — 미구현

**목적**: 마케팅 캠페인을 생성하고 대상 세그먼트와 실적을 통합 관리합니다.

**주요 기능 (예정)**:
- 캠페인명, 목표(신규 유치·재등록·브랜드 인지), 대상 세그먼트, 기간, 채널을 설정합니다(DLG-076-001).
- 도달 수·클릭 수·전환 수·ROI 등 실적을 추적합니다.
- 준비·진행·종료 상태로 캠페인을 관리합니다.

**화면**: SCR-076 캠페인 관리 (미구현)
**관련 다이얼로그**: DLG-076-001 캠페인 등록, DLG-076-002 캠페인 삭제 확인

---

### 8. 리퍼럴 프로그램 (SCR-077) — 미구현

**목적**: 기존 회원이 지인을 추천해 신규 회원을 유치하면 추천인·피추천인 모두에게 혜택을 제공하는 프로그램을 운영합니다.

**주요 기능 (예정)**:
- 이벤트명, 추천인·피추천인 혜택, 혜택 지급 시점, 기간을 설정합니다(DLG-077-001).
- 추천 건수, 성공 전환 건수, 지급 혜택 합계를 추적합니다.
- 추천인/피추천인 이력과 혜택 지급 여부를 확인합니다.

**화면**: SCR-077 리퍼럴 프로그램 (미구현)
**관련 다이얼로그**: DLG-077-001 리퍼럴 이벤트 등록, DLG-077-002 리퍼럴 이벤트 삭제 확인

---

### 9. SMS/카카오 대량 발송 (SCR-078) — 미구현

**목적**: 대규모 회원에게 SMS 또는 카카오 알림톡을 대량으로 발송하고 채널과 비용을 통합 관리합니다.

**주요 기능 (예정)**:
- 발송 대상 그룹을 조건별로 선택합니다(DLG-078-001).
- 메시지 템플릿 선택 또는 직접 작성 후 즉시·예약 발송합니다.
- 발송 전 최종 확인(DLG-078-002)에서 수신자 수와 비용을 확인합니다.
- 발송 이력과 월별 비용 추이를 조회합니다.

**화면**: SCR-078 SMS/카카오 대량 발송 (미구현)
**관련 다이얼로그**: DLG-078-001 발송 대상 선택, DLG-078-002 발송 확인

---

### 10. A/B 테스트 (SCR-079) — 미구현

**목적**: 두 가지 버전의 메시지나 쿠폰을 비교 실험해 더 효과적인 마케팅 전략을 선택합니다.

**주요 기능 (예정)**:
- 테스트명, 유형(메시지·쿠폰), A안·B안 내용, 대상 분배 비율, 기간, 성과 측정 기준을 설정합니다(DLG-079-001).
- 도달 수·클릭율·전환율·매출 기여를 A안·B안 나란히 비교합니다.
- 효과가 우수한 버전을 선택해 정식 마케팅에 적용합니다.

**화면**: SCR-079 A/B 테스트 (미구현)
**관련 다이얼로그**: DLG-079-001 A/B 테스트 등록, DLG-079-002 A/B 테스트 삭제 확인

## 주요 비즈니스 규칙

1. **리드 전환율 자동 집계**: 리드 상태가 "등록완료"로 변경될 때 전환율 통계가 자동으로 업데이트됩니다.
2. **포인트 기반 발송 비용**: 메시지 발송은 보유 포인트에서 비용이 차감되며, 포인트 부족 시 발송이 차단됩니다.
3. **쿠폰 삭제 시 연쇄 무효화**: 쿠폰 삭제 시 이미 발급된 미사용 쿠폰도 함께 무효화됩니다.
4. **마일리지 수동 조정 이력 의무 기록**: 수동 적립·차감 시 사유 입력이 필수이며 처리자·일시가 이력에 기록됩니다.
5. **전자 서명 완료 후 계약 확정**: 회원 서명이 완료된 시점에 계약이 확정되어 이력에 저장됩니다.
6. **미구현 화면 안내**: SCR-076~079는 현재 미구현 상태로, 해당 메뉴 접근 시 "준비 중" 안내 페이지가 표시됩니다.
