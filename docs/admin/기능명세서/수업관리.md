## D04 공통 데이터 기준

### 강습 세션 유형

- `수업 유형`은 운영 화면의 넓은 분류로 `PT`, `GX`, `개인레슨`, `기타` 등을 사용한다.
- KPI 산출용 `강습 세션 유형`은 별도 세부 분류로 관리하며, `PT`, `필라테스`, `스트레칭`, `테라피` 4종으로 고정한다.
- 화면 표시에서는 `PT`를 `P.T`, `필라테스`를 `필라`로 축약 표기할 수 있지만, 데이터 기준값은 `PT`, `필라테스`, `스트레칭`, `테라피`로 통일한다.
- 강습 세션수 KPI는 실제 수업/출석/일정 원장에 저장된 `강습 세션 유형`을 공식 원천으로 집계한다.
- `[주간월간보고]` 주차별 시트 `N5(강습)` 및 `달성` 시트 `G열(강습)`, `[강습업무보고]` 주간보고 `O열(수업수)`는 4종 분해 공식 원천이 아니라, 월간 합계 검산 및 운영 보조 원천으로 사용한다.

## 1. 수업 캘린더

### 화면 목적

PT 및 그룹 수업 전체 일정을 캘린더 형태로 조회하고 관리하는 메인 수업 화면입니다. 월·주·일·목록 뷰를 자유롭게 전환하며, 강사별·수업 유형별 색상으로 일정을 직관적으로 파악합니다. 수업 블록을 클릭하거나 드래그해 빠르게 등록·수정할 수 있으며, 강습 세션은 KPI 집계를 위해 `PT`, `필라테스`, `스트레칭`, `테라피` 중 하나의 세션 유형을 함께 관리합니다.

### 주요 기능

- **뷰 전환**: 월 / 주 / 일 / 목록 뷰 전환. 뷰에 따라 표시 밀도 조절
- **탭 내비게이션**: 일정표 / 수업 관리 / 횟수 관리 / 페널티 관리 / 유효 수업 목록 — 각 탭이 전용 화면으로 연결
- **필터**: 강사, 수업명, 정원 상태(전체/여유/마감), 일정 분류(상담·OT·체성분·방문·수업·PT·기타), 강습 세션 유형(PT·필라테스·스트레칭·테라피) 복수 선택
- **색상 범례**: 강사별·수업 유형별·일정 분류별 색상 코드를 화면에 표시해 한눈에 구분
- **수업 등록**: 상단 "수업 등록" 버튼 또는 캘린더 빈 슬롯 클릭으로 수업 등록 팝업 진입
- **수업 상세**: 수업 블록 클릭 시 예약 현황·강사·시간·장소를 빠르게 확인하는 상세 팝업 표시
- **드래그&드롭**: 수업 블록을 드래그해 시간 이동, 하단 드래그로 시간 연장
- **우측 패널**: 선택 날짜의 수업 목록을 카드로 표시 (수업명·강사·예약 현황·정원 바)
- **미승인 일정 알림**: 미승인 일정이 있을 때 상단 경고 배너로 안내, 승인·거절 처리 가능
- **스케줄 일괄 변경**: 반복 수업 전체 또는 이후 일정을 한 번에 수정

### 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 강사 일정, 모든 수업 유형 | 수업 등록·수정·삭제, 일괄 변경, 미승인 일정 승인 |
| 트레이너 | 본인 담당 수업 중심 (전체 조회 가능) | 담당 수업 등록·수정, 드래그 이동 |
| FC / 스태프 | 전체 수업 일정 조회 | 조회만 가능 |

### 주요 UX 흐름

1. 사이드바 > 수업관리 > 수업/캘린더 진입, 이번 달 월 뷰 기본 표시
2. 뷰 전환 버튼으로 월/주/일/목록 전환
3. 강사·수업명·분류 필터로 원하는 일정만 표시
4. 수업 블록 클릭 → 일정 상세 팝업에서 정보 확인 및 수정 진입
5. 빈 슬롯 클릭 또는 "수업 등록" 버튼 → 수업 등록 팝업
6. 드래그&드롭으로 시간 이동, 블록 하단 드래그로 시간 연장
7. 우측 패널에서 해당 날짜 수업 카드 빠르게 확인

### 화면 상태

| 상태 | 설명 |
|---|---|
| 기본(월 뷰) | 이번 달 전체 수업 블록 표시, 일별 최대 3건 + 더보기 링크 |
| 필터 적용 | 선택한 강사·수업명·분류에 맞는 수업만 표시 |
| 빈 일정 | 해당 기간 수업 없음 안내 |
| 미승인 일정 존재 | 상단 주황색 경고 배너 + 건수 표시 |
| 시간 충돌 감지 | 겹치는 일정 있을 때 경고 표시 후 저장 차단 또는 강제 저장 선택 |

## 2. 수업 관리

### 화면 목적

등록된 수업 전체를 목록으로 조회하고 상태별로 필터링하는 화면입니다. 개별 수업의 출석 현황을 확인하고 수업 정보를 수정하거나 취소 처리할 수 있습니다.

### 주요 기능

- **필터**: 강사, 장소 드롭다운으로 목록 좁히기
- **상태 필터**: 전체 / 예정 / 진행중 / 완료 / 취소 버튼 토글
- **수업 목록**: 수업명, 유형, 강습 세션 유형, 강사, 장소, 날짜·시간, 예약 현황(현재/정원), 상태 배지 표시
- **출석 현황**: 수업별 출석자 수·노쇼 수 한눈에 확인
- **수업 등록**: 상단 우측 버튼으로 새 수업 등록 팝업 진입
- **수업 수정**: 목록 행 클릭으로 수업 등록/수정 팝업 진입
- **수업 기록 조회**: 완료 수업 행에서 출석·서명 기록 상세 팝업 확인

### 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 수업 목록, 모든 강사 수업 | 수업 등록·수정·취소, 출석 처리 |
| 트레이너 | 본인 담당 수업 목록 | 담당 수업 수정, 출석 처리 |
| FC / 스태프 | 전체 수업 목록 조회 | 조회만 가능 |

### 주요 UX 흐름

1. 수업 캘린더 탭에서 "수업 관리" 탭 클릭 또는 `/lessons` 직접 진입
2. 강사·장소 필터 및 상태 버튼으로 원하는 수업 좁히기
3. 수업 행 클릭 → 수업 등록/수정 팝업에서 상세 정보 확인 및 수정
4. 완료된 수업 행에서 수업 기록 상세 팝업으로 출석·서명 기록 확인

## 3. 시간표 일괄 등록

### 화면 목적

반복되는 수업 일정을 한 번에 대량으로 등록하는 화면입니다. 기간·요일·시간·강사·장소를 설정하면 지정 기간 동안의 일정이 자동 생성됩니다.

### 주요 기능

- **기간 설정**: 시작일 ~ 종료일 날짜 선택
- **요일 선택**: 월~일 중 복수 선택 가능
- **시간 설정**: 시작·종료 시간 (30분 단위)
- **수업 선택**: 수업 템플릿 선택 시 수업명·정원·시간 자동 완성
- **강사·장소·정원 설정**: 각 드롭다운 또는 직접 입력
- **예외 날짜 제외**: 공휴일·휴관일 등 특정 날짜 제외 처리
- **미리보기**: 생성될 일정 건수와 날짜 목록을 등록 전에 확인
- **충돌 감지**: 기존 일정과 겹치는 시간대 있으면 충돌 날짜 목록 경고 표시
- **일괄 생성**: 최종 확인 팝업 후 일정 대량 생성

### 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 강사, 전체 수업 옵션 | 일괄 등록 실행 |
| 트레이너 | 본인 이름만 강사로 선택 가능 | 본인 담당 수업 일괄 등록 |
| FC / 스태프 | 접근 불가 | 이 기능 사용 불가 |

### 주요 UX 흐름

1. 적용 기간·요일·시간 설정
2. 수업 템플릿·강사·장소 선택
3. 제외 날짜 추가
4. 미리보기에서 생성 예정 건수·날짜 확인
5. "일괄 생성" → 최종 확인 팝업 → 확인 후 일정 생성 및 캘린더 이동

## 4. 그룹 수업 템플릿

### 화면 목적

그룹 수업의 기본 정보(수업명, 유형, 정원, 기본 장소, 색상 등)를 템플릿으로 저장·관리하는 화면입니다. 템플릿을 사전에 구성해 두면 수업 등록 시 반복 입력 없이 자동 완성됩니다.

### 주요 기능

- **템플릿 목록**: 수업명, 유형, 정원, 기본 장소, 색상 칩, 활성 여부 표시
- **활성/비활성 전환**: 비활성화 시 수업 등록 선택 목록에서 숨김
- **템플릿 등록·수정**: 등록 버튼 또는 행 클릭으로 입력 팝업 진입
- **템플릿 삭제**: 사용 중인 수업 일정이 없는 경우에만 삭제 가능

### 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 템플릿 목록 | 등록·수정·삭제·활성화 전환 |
| 트레이너 | 활성 템플릿 조회 | 조회만 가능 |
| FC / 스태프 | 활성 템플릿 조회 | 조회만 가능 |

## 5. 그룹 수업 현황

### 화면 목적

그룹 수업별 예약 현황·출석률·잔여 자리를 한눈에 파악하는 현황판 화면입니다. 인기 수업과 빈자리가 있는 수업을 빠르게 파악해 운영 결정(수업 추가·취소·정원 조정)에 활용합니다.

### 주요 기능

- **필터**: 기간(주간/월간), 수업 유형, 강사
- **현황 카드/테이블**: 수업명, 유형, 강사, 요일·시간, 정원, 예약 수, 잔여 자리, 출석률, 노쇼 수
- **잔여 자리 시각화**: 여유(초록) / 마감 임박(주황) / 마감(빨강) 색상 바
- **정렬**: 출석률 높은 순, 잔여 자리 적은 순, 예약 많은 순

## 6. 강사 근무 현황

### 화면 목적

강사별 수업 배정 현황과 근무 시간을 요약해 파악하는 화면입니다. 업무 부담이 편중된 강사가 없는지 점검하고 수업 재배정 의사 결정에 활용합니다.

### 주요 기능

- **기간 필터**: 주간 / 월간 / 사용자 지정 기간
- **강사 현황**: 강사별 수업 건수, 총 근무 시간, 수업 유형별 분포, 강습 세션 유형별 분포, 출석 처리 완료율
- **트레이너별 OT 현황**: `[강습업무보고]` 기준 월별 총 OT, O.T 배정 합계, O.T 1, O.T 2를 트레이너별로 표시
- **OT 1/2차 진행 상태**: `[OT관리-강습]`의 OT 1·OT 2 일자와 진행 여부를 기준으로 완료·예정 건을 분리 표시
- **요약 통계**: 평균 근무 시간, 최다 수업 강사, 수업 없는 강사
- **강사 행 클릭**: 강사 상세 팝업에서 담당 수업 목록과 OT 상세 확인

### 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 강사 근무 현황, 시간 합계 | 조회, 강사 상세 팝업 확인 |
| 트레이너 | 본인 근무 현황만 | 조회만 가능 |
| FC / 스태프 | 전체 강사 근무 현황 | 조회만 가능 |

### OT 데이터 기준

| 지표 | 소스 파일 | 탭 / 셀 | 산출 기준 |
|---|---|---|---|
| 트레이너별 O.T 배정 합계 | `[강습업무보고]` | 주간보고 E열 | 트레이너별 월 배정 회원 수 합계 |
| 트레이너별 O.T 1 완료 | `[강습업무보고]`, `[OT관리-강습]` | 주간보고 F열, OT현황 Q열 | 해당 주 완료된 1차 OT 수 |
| 트레이너별 O.T 2 완료 | `[강습업무보고]`, `[OT관리-강습]` | 주간보고 G열, OT현황 T열 | 해당 주 완료된 2차 OT 수 |
| OT 1/2차 예정 | `[OT관리-강습]` | OT현황 Q/T열 | 진행 값이 `예정`인 1차·2차 OT |
| OT 거절 건 | `[OT관리-강습]` | OT현황 V열 | `거부+보류`를 함께 집계 |

### 비즈니스 규칙

- 강사 근무 현황의 OT 지표는 수업 완료율과 분리해 영업 파이프라인 지표로 표시합니다.
- 트레이너별 O.T 배정 합계는 배정 편중과 공백을 확인하는 개인 KPI이며, 본사 KPI 센터의 트레이너 개인 KPI와 동일 원천을 사용합니다.
- OT 등록 전환율은 본사 KPI 기준과 동일하게 `등록 / OT 진행`으로 해석하며, `OT 진행`의 세부 산식은 본사관리 문서의 확정 기준을 따릅니다.
- OT 이월률의 분모는 우선 `OT배정 인원`으로 기록하지만, `OT 진행` 기준과 구분이 필요한 항목이므로 추후 사용자 확인 후 최종 확정합니다.

## 14. 출석 QR 체크인

> 퍼블리싱 완료 / 데이터 미연동

### 화면 목적

회원이 QR 코드를 스캔해 수업 출석을 직접 처리하는 화면입니다. 프런트 데스크 태블릿에 설치해 회원이 자율적으로 체크인하도록 운영합니다.

### 주요 기능

- **QR 스캔**: 카메라로 QR 코드 스캔 → 출석 자동 등록
- **수업 선택**: 체크인 대상 수업 선택 (진행 예정 수업 목록)
- **실시간 출석 현황**: 총 예약·출석 완료·미출석 인원, 출석 회원 목록
- **수동 체크인**: QR 스캔 실패 시 이름·연락처 검색으로 수동 처리
- **체크인 성공 피드백**: 회원명 + 수업명 표시, 시각·청각 피드백

### 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 수업 QR 체크인 화면 | 수업 선택, 수동 체크인 |
| 트레이너 | 담당 수업 QR 체크인 화면 | 담당 수업 체크인 운영 |
| FC / 스태프 | 전체 수업 QR 체크인 화면 | 수업 선택, 수동 체크인 |

## 15. 수업 녹화 관리

> 퍼블리싱 완료 / 데이터 미연동

### 화면 목적

완료된 수업의 녹화 파일을 업로드하고 담당 회원에게 공유 설정하는 화면입니다. 회원이 수업을 복습하거나 피드백 자료로 활용할 수 있습니다.

### 주요 기능

- **녹화 파일 목록**: 수업명, 강사명, 수업 일시, 파일 크기, 업로드 일시, 공유 상태, 공유 대상 회원 수
- **파일 업로드**: 드래그&드롭 또는 파일 선택, 연결할 완료 수업 선택
- **공유 설정**: 공유 대상 회원 선택, 공개 기간 설정 (무기한 / 특정 날짜까지)
- **공개 기간 만료**: 기간 지난 파일 자동 비공개 전환
- **파일 삭제**: 삭제 확인 후 처리

### 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 녹화 파일 목록 | 업로드·삭제·공유 설정 |
| 트레이너 | 본인 수업 녹화 파일 | 업로드·삭제·본인 수업 공유 설정 |
| FC / 스태프 | 접근 불가 | 이 기능 사용 불가 |
