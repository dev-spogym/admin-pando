# D07 직원관리 기능명세서

> **프로젝트**: Pando CRM
> **도메인**: D07 직원관리
> **작성일**: 2026-04-26
> **대상**: 기획자·고객사
> **포함 화면**: SCR-060 ~ SCR-065, DLG-060-001 ~ DLG-064-002

---

## 도메인 개요

직원관리 도메인은 센터에 소속된 직원의 정보를 등록·관리하고, 출퇴근 근태를 기록하며, 급여를 계산하고 확정하는 모든 업무를 지원합니다. 센터장과 매니저는 인력 현황을 실시간으로 파악하고, 퇴사 처리 및 급여 확정 등 주요 인사 업무를 시스템 내에서 처리할 수 있습니다. 직원 본인은 본인의 정보와 급여 명세서를 직접 조회할 수 있습니다.

## 권한/직무 해석 원칙

- 관리자웹 권한 키는 `superAdmin / primary / owner / manager / fc / staff / readonly`를 기준으로 해석한다.
- 이 문서에서 등록·조회하는 `직무`는 직원의 인사/운영 분류값이며, 관리자웹 메뉴 접근 권한과 동일 개념이 아니다.
- 본문에서 보이는 `센터장`, `매니저`, `FC`, `트레이너`, `스태프`, `프런트`는 직원 직무 또는 운영 라벨로 해석한다.
- 실제 관리자웹 메뉴 접근 권한은 `SCR-081 권한 설정` 또는 커스텀 역할 매핑 기준으로 별도 관리한다.

## 계정 권한 요약

| 계정 유형 | 접근 가능 화면 | 주요 권한 |
|---|---|---|
| 센터장 | 전체 직원관리 화면 | 직원 등록·수정·퇴사 처리, 급여 확정, 명세서 발송, 엑셀 내보내기 |
| 매니저 | 전체 직원관리 화면 | 직원 등록·수정, 근태 기록 보정, 급여 편집 (확정 권한 제외) |
| FC / 트레이너 / 스태프 / 프런트 | 본인 정보 및 급여 명세서만 | 본인 정보 조회, 본인 근태 조회, 본인 급여 명세서 조회 |

## 기능 목록

### 1. 직원 목록 조회 (SCR-060)

**목적**: 센터에 소속된 전체 직원 현황을 한눈에 파악하고 원하는 직원을 빠르게 찾습니다.

**주요 기능**:
- 전체 직원 수 / 재직 / 휴직 / 퇴사 인원을 요약 카드로 표시합니다.
- 이름 또는 연락처로 직원을 검색합니다.
- 직무(센터장·매니저·FC·트레이너·스태프·프런트)와 재직 상태(재직·휴직·퇴사)로 필터링합니다.
- 직원명 클릭 시 상세 정보로 이동합니다.
- 1명 이상 선택 시 메시지 전송·상태 변경·퇴사 처리·명단 다운로드 버튼이 활성화됩니다.
- 목록을 엑셀 파일로 내려받을 수 있습니다.

**화면**: SCR-060 직원 목록

---

### 2. 직원 등록 및 수정 (SCR-061)

**목적**: 신규 직원을 시스템에 등록하거나 기존 직원 정보를 수정합니다.

**주요 기능**:
- 직원 기본 정보(이름, 연락처, 이메일, 직무, 직급, 입사일)를 입력합니다.
- 담당 서비스(PT, 필라테스 등)를 지정합니다.
- 급여 형태(월급제·시급제·건별)와 기본급을 설정합니다.
- 시스템 로그인 계정 생성 여부를 선택합니다.
- 캘린더 표시 색상을 지정합니다.
- 이름과 직무는 필수 입력 항목입니다.
- 저장 완료 시 직원 목록으로 돌아갑니다.
- 입력 중 취소 시 이탈 확인 다이얼로그(DLG-061-001)가 표시됩니다.

**화면**: SCR-061 직원 등록/수정
**관련 다이얼로그**: DLG-061-001 등록 폼 취소 확인

---

### 3. 직원 퇴사 처리 (SCR-062)

**목적**: 퇴사 예정 직원의 퇴사일을 설정하고 미완료 업무를 인수인계한 뒤 계정을 비활성화합니다.

**주요 기능**:
- 퇴사 확정일과 퇴사 예정일을 별도로 입력합니다.
- 퇴사 사유를 내부 관리용으로 기록합니다.
- 해당 직원이 담당하던 회원·수업·계약을 인계받을 담당자를 지정합니다.
- 퇴사 처리 완료 시 시스템 계정이 즉시 비활성화되어 로그인이 차단됩니다.
- 처리 완료 전 최종 확인 다이얼로그(DLG-060-002)가 표시됩니다.
- 처리 완료 후 직원 상태가 "퇴사"로 변경됩니다.

**화면**: SCR-062 직원 퇴사 처리
**관련 다이얼로그**: DLG-060-002 퇴사 처리 확인

---

### 4. 직원 근태 관리 (SCR-063)

**목적**: 직원별 출퇴근 기록을 조회하고 지각·결근 현황을 파악해 급여 산정의 기초 데이터로 활용합니다.

**주요 기능**:
- 조회 기간(월 단위 또는 날짜 범위)과 대상 직원을 선택해 조회합니다.
- 정상 출근 일수, 지각 횟수, 결근 횟수, 총 근무 시간을 요약 카드로 표시합니다.
- 날짜별 출퇴근 시각과 상태(정상·지각·결근·조퇴)를 목록으로 표시합니다.
- 관리자가 오류 기록을 수동으로 보정하거나 누락 기록을 추가합니다.
- 직원별 월 총 근무 시간과 출근율을 비교하는 집계 요약을 제공합니다.
- 조회 결과를 엑셀 파일로 내려받습니다.
- 직원 본인은 자신의 근태 기록만 조회할 수 있으며 수정은 불가합니다.

**화면**: SCR-063 직원 근태 관리

---

### 5. 급여 관리 (SCR-064)

**목적**: 직원별 당월 급여를 계산하고 수당·공제 항목을 편집한 뒤 최종 급여를 확정합니다.

**주요 기능**:
- 지급 월을 선택하면 직원별 급여 데이터가 자동으로 계산됩니다.
- 근태 데이터와 연동되어 기본급에 수당이 자동 반영됩니다.
- 수당·공제 편집 다이얼로그(DLG-064-001)에서 항목별 금액을 수동 조정합니다.
- 개별 또는 전체 직원의 급여를 확정합니다. 확정 전 확인 다이얼로그(DLG-064-002)가 표시됩니다.
- 확정된 급여는 급여 명세서(SCR-065)에서 조회 가능해집니다.
- 당월 내에는 확정 취소 후 재편집이 가능합니다.
- 전체 급여 내역을 엑셀 파일로 내려받습니다.
- 급여 확정 권한은 센터장만 보유하며, 매니저는 편집까지만 가능합니다.

**화면**: SCR-064 급여 관리
**관련 다이얼로그**: DLG-064-001 급여 상세 편집, DLG-064-002 급여 확정 확인

---

### 6. 급여 명세서 (SCR-065)

**목적**: 확정된 급여 명세서를 조회하고 직원에게 발송합니다.

**주요 기능**:
- 월 단위로 급여 명세서를 조회합니다.
- 직원별 기본급, 수당 상세, 공제 상세, 실지급액을 확인합니다.
- 개별 또는 일괄 발송(이메일·앱 알림)이 가능합니다.
- 발송 완료된 명세서에 "발송완료" 표시가 나타납니다.
- 명세서를 인쇄하거나 PDF 파일로 저장합니다.
- 직원 본인은 본인의 급여 명세서만 조회·인쇄할 수 있습니다.
- 급여가 확정되지 않은 월은 "급여 미확정" 안내가 표시됩니다.

**화면**: SCR-065 급여 명세서

## 주요 비즈니스 규칙

1. **직무와 권한 분리**: 직원 등록/수정에서 입력하는 `직무`는 인사/운영 분류값이며, 실제 관리자웹 메뉴 접근 권한은 `SCR-081 권한 설정`에서 별도로 매핑합니다.
2. **관리자 권한 기반 접근**: 직원 목록·등록·수정·퇴사 처리는 센터장·매니저 권한을 가진 관리자만 가능하며, 일반 직원은 본인 정보만 조회할 수 있습니다.
3. **퇴사 처리 이중 확인**: 퇴사 처리는 "퇴사처리" 텍스트를 직접 입력해야 최종 버튼이 활성화됩니다.
4. **급여 확정 권한 분리**: 급여 편집은 매니저 이상이 가능하지만, 최종 확정은 센터장만 가능합니다.
5. **급여 확정 후 명세서 생성**: 급여 확정 완료 시에만 해당 월의 급여 명세서가 생성되어 발송 가능해집니다.
6. **근태 수동 보정**: 출퇴근 기록의 수동 보정은 관리자만 가능하며, 보정 이력이 기록됩니다.
