## 1. 락커 관리

> 참조 화면: [SCR-050](../화면설계서/D06-시설관리/SCR-050-락커관리/00-기본화면.md)

### 기능 개요

센터 내 모든 락커의 배정 현황과 상태를 시각적 그리드로 관리합니다. 구역(A/B/C)별 탭으로 분리하여 한눈에 현황을 파악하고, 개별 락커에 회원을 배정하거나 회수·이동·고장 처리를 수행합니다. 만료 임박 락커 일괄 해제와 여러 락커 일괄 배정 기능을 지원합니다.

### 주요 기능 목록

| 기능 | 설명 |
|---|---|
| 현황 요약 | 전체·사용중·만료 임박·고장 락커 수를 카드로 표시 |
| 구역 탭 전환 | A·B·C 구역별로 그리드 분리 조회 |
| 락커 검색 | 락커 번호 또는 배정 회원명으로 해당 락커 검색 |
| 락커 배치 그리드 | 실제 배치와 동일한 격자 형태로 상태별 색상 구분 |
| 이력 조회 | 특정 락커의 배정·회수 이력 및 비밀번호·메모 확인·수정 |
| 락커 이동 | 배정 정보를 다른 락커 번호로 이전 |
| 락커 회수 | 사용중 락커의 회원 배정 정보 초기화 |
| 개별 배정 | 빈 락커에 회원 검색 후 만료일 설정해 배정 |
| 고장 처리/해제 | 락커 상태를 고장으로 설정하거나 복구 |
| 만료 임박 일괄 해제 | 만료 7일 이내 락커를 선택해 한 번에 회수 |
| 일괄 배정 | 여러 빈 락커를 한 명의 회원에게 동시 배정 |
| 엑셀 다운로드 | 전체 락커 현황을 엑셀 파일로 내려받기 |

### 락커 상태 구분

| 상태 | 설명 | 표시 색상 |
|---|---|---|
| 빈 락커 | 배정된 회원이 없는 상태 | 회색 |
| 사용중 | 회원이 배정되어 이용 중인 상태 | 파란색 |
| 만료 임박 | 만료일이 7일 이내로 남은 사용중 락커 | 주황색 |
| 고장 | 사용 불가 상태로 처리된 락커 | 빨간색 |

### 그리드 셀 액션

| 액션 | 표시 조건 |
|---|---|
| 기록 | 모든 락커 (이력·비밀번호·메모 조회) |
| 이동 | 모든 락커 (번호 변경) |
| 회수 | 사용중·만료 임박 락커 |
| 배정 | 빈 락커 |
| 고장 처리/해제 | 모든 락커 (현재 상태에 따라 처리·해제 전환) |

### 계정별 기능 범위

| 계정 유형 | 조회 | 배정·회수·이동 | 고장 처리 | 일괄 작업 | 엑셀 |
|---|---|---|---|---|---|
| 슈퍼관리자 / 센터장 | O | O | O | O | O |
| 매니저 | O | O | O | O | O |
| 스태프 / 프런트 | O | O (배정·회수) | X | X | O |

## 2. 사물함 배정 관리

> 참조 화면: [SCR-051](../화면설계서/D06-시설관리/SCR-051-사물함배정관리/00-기본화면.md)

### 기능 개요

일일·개인·골프 사물함의 실시간 이용 현황을 그리드로 시각화하고, 회원 검색 자동완성으로 빈 사물함에 즉시 배정합니다. 락커 관리(SCR-050)의 장기 배정과 달리 당일 또는 단기 소지품 보관 목적의 사물함을 빠르게 운영합니다.

### 주요 기능 목록

| 기능 | 설명 |
|---|---|
| 현황 요약 | 총 사물함·사용 가능·사용중·시간 초과·상태 비정상 수 표시 |
| 유형 탭 전환 | 일일·개인·골프 사물함 탭별 분리 조회 |
| 회원 검색 자동완성 | 회원명·연락처·회원번호 입력 시 자동완성 목록 표시 |
| 추천 사물함 | 사용 가능 사물함 중 번호 가장 작은 것을 자동 추천 |
| 사물함 배정 | 선택 회원 + 선택 사물함 + 만료일 입력 후 배정 |
| 만료 사물함 일괄 해제 | 시간 초과(overtime) 상태 사물함 전체 한 번에 해제 |
| 상태 동기화 | 실시간 사물함 상태 새로고침 |
| 엑셀 다운로드 | 현재 사물함 현황을 엑셀로 내려받기 |

### 사물함 상태 구분

| 상태 | 설명 | 표시 색상 |
|---|---|---|
| 사용 가능 | 빈 사물함 | 회색 (클릭하여 선택 가능) |
| 사용중 | 회원이 이용 중인 사물함 | 파란색 |
| 시간 초과 | 이용 시간이 지난 사물함 | 주황색 + 경고 아이콘 깜빡임 |
| 상태 비정상 | 오류·고장 등 비정상 상태 | 빨간색 (클릭 불가) |

### 배정 프로세스

1. 회원 검색창에 이름·연락처·회원번호 입력 → 자동완성 목록에서 선택.
2. 추천 사물함을 사용하거나 그리드에서 직접 빈 사물함 선택.
3. 만료일 확인·수정 후 [배정하기] 클릭.
4. 배정 완료 후 3초간 성공 메시지 표시 → 그리드 자동 갱신.

### 계정별 기능 범위

| 계정 유형 | 조회 | 배정·해제 | 일괄 해제 | 락커 추가 |
|---|---|---|---|---|
| 슈퍼관리자 / 센터장 / 매니저 | O | O | O | O |
| 스태프 / 프런트 | O | O | X | X |
| 트레이너 / 조회전용 | O | X | X | X |

## 3. 밴드/카드 관리

> 참조 화면: [SCR-052](../화면설계서/D06-시설관리/SCR-052-밴드카드관리/00-기본화면.md)

### 기능 개요

센터에서 사용하는 RFID 밴드와 카드를 등록·관리하고, 회원 또는 직원과 연결합니다. 카드 번호 스캔(또는 수동 입력), 사용자 연결, 분실·교체·삭제 처리를 지원하며 사용 이력을 기간별로 조회합니다.

### 주요 기능 목록

| 기능 | 설명 |
|---|---|
| 현황 요약 | 전체·활성·미사용/해제/분실 카드 수 표시 |
| 카드 목록 조회 | 전체 RFID 카드·밴드 목록을 테이블로 표시 |
| 검색 및 상태 필터 | 카드 번호·회원명 검색 및 상태(활성/분실/해제) 필터 |
| 신규 등록 | 카드 번호 스캔 또는 수동 입력 후 회원·직원 연결 |
| 정보 수정 | 기존 카드의 연결 회원·사물함 정보 수정 |
| 분실 처리 | 해당 카드 상태를 "분실"로 변경 |
| 이력 조회 | 기간 선택 후 해당 카드의 사용 이력 조회 |
| 카드 삭제 | 카드 정보 영구 삭제 (연결 회원 관계도 해제) |
| 엑셀 다운로드 | 전체 카드 목록을 엑셀로 내려받기 |

### 카드 상태 구분

| 활성 | 회원에게 배정되어 정상 사용 중 | 초록색 |

| 해제 | 배정 해제되어 미사용 상태인 카드 | 회색 |

### 계정별 기능 범위

|---|---|---|---|---|---|
| 슈퍼관리자 / 센터장 / 매니저 | O | O | O | O | O |

## 4. 운동룸 관리

> 참조 화면: [SCR-053](../화면설계서/D06-시설관리/SCR-053-운동룸관리/00-기본화면.md)

### 기능 개요

센터 내 운동룸(GX룸·PT룸·스피닝룸·필라테스룸 등)의 현황을 카드 뷰 또는 목록 뷰로 관리합니다. 각 룸의 운영 상태와 오늘의 시간대별 예약 혼잡도를 시각적으로 확인하고, 룸 등록·수정·삭제 및 상태 전환을 수행합니다.

### 주요 기능 목록

| 기능 | 설명 |
|---|---|
| 현황 요약 | 전체·운영중·점검/고장/미사용·GX/PT 룸 수 표시 |
| 뷰 전환 | 카드 보기와 목록 보기 전환 |
| 유형 필터 | 전체·GX·PT·스피닝·필라테스·기타 중 선택 필터 |
| 예약 혼잡도 슬롯 바 | 07:00~20:00 시간대를 색상 바로 시각화 (여유·혼잡·만석·빈 시간) |
| 운영 상태 전환 | 버튼 클릭으로 운영중→점검중→고장→운영중 순환 변경 |
| 룸 등록 | 룸명·유형·수용 인원·설명을 입력해 새 운동룸 추가 |
| 룸 수정 | 기존 운동룸 정보 변경 |
| 룸 삭제 | 운동룸 영구 삭제 |

### 룸 운영 상태 구분

| 상태 | 설명 | 카드 표시 |
|---|---|---|
| 운영중 | 정상 운영 중 | 초록색 배지, 일반 카드 |
| 점검중 | 일시 점검으로 예약 불가 | 주황색 배지, 투명도 80% |
| 고장 | 사용 불가 고장 상태 | 빨간색 배지, 빨간색 테두리 강조 |
| 미사용 | 운영 전 또는 폐쇄 상태 | 회색 배지 |

### 수용 인원 퀵 선택 값

1명 · 2명 · 6명 · 10명 · 14명 · 16명 · 20명 · 22명

### 계정별 기능 범위

| 계정 유형 | 조회 | 등록·수정 | 삭제 | 상태 전환 |

## 5. 골프 타석 관리

> 참조 화면: [SCR-054](../화면설계서/D06-시설관리/SCR-054-골프타석관리/00-기본화면.md)

### 기능 개요

골프 연습장 타석의 실시간 이용 현황을 관리합니다. 타석별 사용자·담당 프로·남은 시간(카운트다운)을 실시간으로 표시하고, 이용 시작·종료·타석 간 이동·점검 처리를 수행합니다. 대기열 관리와 자동 배정 기능으로 원활한 타석 운영을 지원합니다.

### 주요 기능 목록

| 기능 | 설명 |
|---|---|
| 현황 요약 | 전체·사용중·대기·예약·대기열 인원 수 표시 |
| 타석 그리드 | 전체 타석을 4열 카드로 실시간 표시 |
| 남은 시간 카운트다운 | 사용중 타석의 남은 시간을 1분 단위로 실시간 감소 표시 |
| 이용 시작 | 빈 타석에 회원명 입력 후 이용 시작 및 타이머 가동 |
| 이용 종료 | 사용중 타석 종료 시 타석 초기화 + 대기열 자동 배정 |
| 타석 이동 | 사용중 회원 정보와 남은 시간을 다른 타석으로 이전 |
| 점검 처리/해제 | 타석 상태를 점검으로 설정하거나 복구 |
| 프로젝터 제어 | 사용중 타석의 프로젝터 on/off 토글 |
| 대기열 관리 | 대기 중인 회원 목록을 순서대로 표시 |
| 대기열 자동 배정 | 타석 종료 시 대기열 첫 번째 회원을 해당 타석에 자동 배정 |
| 시간 종료 알림 | 남은 시간 0이 되면 경고 알림 팝업 표시 |

### 타석 상태 구분

| 상태 | 설명 | 카드 색상 |

### 기능 개요

센터에서 회원에게 대여하는 운동복의 재고 현황과 대여·반납·세탁·파손 상태를 관리합니다. QR 코드 스캔으로 대여와 반납을 빠르게 처리하고, 운동복별 현재 상태를 한눈에 파악합니다.

## 7. 장비 점검 일정

> 참조 화면: [SCR-056](../화면설계서/D06-시설관리/SCR-056-장비점검일정/00-기본화면.md)

센터 내 운동 장비의 정기 점검 일정과 수리 이력을 관리합니다. 장비별 다음 점검 예정일을 설정하고 점검 완료 결과를 기록하며, 고장·수리 접수부터 완료까지 이력을 추적해 시설 안전을 유지합니다.

### 주요 기능 목록

| 기능 | 설명 |

| 점검 임박 강조 | 다음 점검 예정일 7일 이내 장비 주황색 강조 표시 |

| 수리 접수 | 고장 장비 수리 접수 (담당자·예상 완료일 등록) |
| 수리 완료 기록 | 수리 완료 후 결과·비용 기록 + 장비 상태 정상 복구 |
