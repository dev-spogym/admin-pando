## 화면 목적

특정 락커에 배정된 회원 정보를 다른 락커 번호로 이전하는 다이얼로그입니다. 회원의 요청이나 관리 목적으로 락커 번호를 변경할 때 사용합니다.

## 주요 구성요소

- **다이얼로그 제목**: "[현재 락커 번호]번 락커 이동"
- **현재 락커 정보**: 현재 락커 번호와 배정된 회원명을 읽기 전용으로 표시합니다.
- **이동할 번호 입력**: 변경할 락커 번호를 입력하는 필드입니다.
- **확인 버튼**: 입력한 번호로 락커를 이동합니다.
- **취소 버튼**: 변경 없이 다이얼로그를 닫습니다.

## 주요 UX 흐름

1. 락커 셀 호버 시 [이동] 버튼 클릭 → 다이얼로그 열림.
2. 이동할 락커 번호를 입력합니다.
3. [확인] 클릭 → "[이동 번호]번 락커로 이동되었습니다" 알림 후 그리드 갱신.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 번호 미입력 | [확인] 비활성 또는 "이동할 락커 번호를 입력해주세요" 오류 |
| 이동 성공 | "락커가 [N]번으로 이동되었습니다" 알림 |
| 이동 실패 | "락커 이동에 실패했습니다" 오류 알림 |
