## 화면 목적

센터에서 사용하는 RFID 밴드와 카드를 등록·관리하고 회원 또는 직원과 연결하는 화면입니다. 회원이 락커·출입문·시설을 이용할 때 사용하는 RF 장치의 전체 발급 현황을 관리하며, 분실 신고·교체·삭제 처리를 수행할 수 있습니다.

## 주요 구성요소

- **현황 요약 카드**: 전체 카드 수 / 활성(사용중) 수 / 미사용·해제·분실 합계 수를 3개 카드로 표시합니다.
- **검색 및 상태 필터**: 카드 번호 또는 회원명으로 검색하고, 상태(활성/분실/해제)로 필터링합니다.
- **카드 목록 테이블**: 등록된 모든 RFID 카드·밴드의 목록을 표시합니다.
  - 표시 항목: 순번 / 등록일 / 카드 번호 / 연결 회원명 / 연락처 / 사용자 유형(회원·직원) / 상태 / 배정일 / 연결 사물함
  - 회원명 클릭 시 해당 회원 상세 화면으로 이동합니다.
  - 각 행에 이력 보기 · 분실 처리 · 수정 · 삭제 버튼이 있습니다.
- **신규 등록 버튼**: RFID 카드·밴드를 새로 등록합니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 센터장 / 매니저 | 전체 카드 목록 및 이력 | 등록·수정·분실 처리·삭제·이력 조회 전체 가능 |
| 스태프 / 프런트 | 전체 카드 목록 조회 | 등록·분실 처리 가능 (삭제 불가) |
| 트레이너 / 조회전용 | 목록 조회만 | 변경 불가 |

## 주요 UX 흐름

1. 화면 진입 시 전체 카드 목록이 표시됩니다.
2. 검색창에 카드 번호 또는 회원명을 입력하거나 상태 필터를 선택해 원하는 카드를 찾습니다.
3. **신규 등록**: [신규 등록] 버튼 클릭 → 카드 번호 스캔 또는 수동 입력 → 사용자 유형 선택 → 회원 검색 연결 → 연결 사물함 입력(선택) → 저장.
4. **분실 처리**: 해당 카드 행의 [분실 처리] 버튼 클릭 → 상태가 "분실"로 변경됩니다.
5. **이력 조회**: [이력] 버튼 클릭 → 기간을 선택해 해당 카드의 사용 이력을 조회합니다.
6. **삭제**: [삭제] 버튼 클릭 → 삭제 확인 다이얼로그 → 확인 시 영구 삭제.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 정상 (데이터 있음) | 카드 목록 테이블 표시 |
| 검색·필터 결과 없음 | 테이블 빈 상태 안내 메시지 표시 |
| 활성 카드 | "활성" 초록색 배지 표시 |
| 분실 카드 | "분실" 빨간색 배지 표시 |
| 해제 카드 | "해제" 회색 배지 표시 |
