## 화면 목적

새 운동룸을 등록하거나 기존 운동룸의 정보를 수정하는 다이얼로그입니다. 룸명·유형·수용 인원·설명을 입력해 운동룸을 관리합니다.

## 주요 구성요소

- **다이얼로그 제목**: "운동룸 등록" (신규) 또는 "운동룸 수정" (수정)
- **룸명 입력**: 필수 항목. 운동룸의 이름을 입력합니다.
- **유형 선택**: GX / PT / 스피닝 / 필라테스 / 기타 중 하나를 버튼으로 선택합니다.
- **수용 인원**: 숫자를 직접 입력하거나 퀵 선택 버튼(1·2·6·10·14·16·20·22명)으로 빠르게 설정합니다.
- **룸 설명**: 룸 용도·특징을 간단히 입력합니다 (선택 항목).
- **등록하기/수정 완료 버튼**: 입력 내용을 저장합니다.
- **취소 버튼**: 변경 없이 다이얼로그를 닫습니다.

## 주요 UX 흐름

1. [새 운동룸 등록] 클릭 또는 카드의 [수정] 클릭 → 다이얼로그 열림.
2. 룸명을 입력하고 유형을 선택합니다.
3. 수용 인원을 직접 입력하거나 퀵 선택 버튼을 클릭합니다.
4. 설명을 입력(선택)한 뒤 [등록하기/수정 완료]를 클릭합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 신규 등록 | 모든 필드 빈 상태로 열림 |
| 수정 | 기존 룸 정보가 미리 채워진 상태로 열림 |
| 룸명 미입력 | [저장] 버튼 비활성화 또는 유효성 오류 메시지 |
| 등록 성공 | "운동룸이 등록되었습니다" 알림 |
| 수정 성공 | "운동룸이 수정되었습니다" 알림 |
