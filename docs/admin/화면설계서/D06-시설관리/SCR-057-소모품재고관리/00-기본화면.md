## 화면 목적

샴푸·린스·바디워시·타월·비누 등 센터 운영에 필요한 소모품의 재고 현황을 관리하는 화면입니다. 현재 재고 수량을 실시간으로 파악하고, 입고·출고를 기록하며, 재고가 기준 수량 이하로 떨어진 품목을 식별해 적시에 발주할 수 있도록 지원합니다.


## 주요 구성요소

- **현황 요약 카드**: 전체 품목 수 / 재고 부족 품목 수 / 이번 달 총 출고량 / 발주 대기 건수를 표시합니다.
- **재고 부족 알림 영역**: 기준 수량(안전 재고) 이하로 떨어진 품목을 상단에 별도로 강조 표시합니다.
- **소모품 목록 테이블**: 등록된 소모품 전체를 표시합니다.
  - 표시 항목: 품목명 / 카테고리 / 단위 / 현재 재고 / 안전 재고 기준 / 최근 입고일 / 최근 출고일 / 재고 상태 / 액션 버튼.
  - 현재 재고가 안전 재고 기준 이하인 경우 빨간색으로 강조합니다.
- **소모품 등록 버튼**: 새 소모품 항목을 추가합니다.
- **입출고 처리 버튼**: 입고 또는 출고 수량을 기록합니다.
- **발주 생성 버튼**: 재고 부족 품목에 대해 발주를 생성합니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 센터장 | 전체 목록 및 이력 | 등록·수정·삭제·입출고·발주 생성 전체 가능 |
| 매니저 | 전체 목록 및 이력 | 입출고 처리·발주 생성 가능 (삭제 불가) |
| 스태프 | 목록 및 이력 조회 | 출고 기록만 가능 |
| 트레이너 / 조회전용 | 현황 조회만 | 변경 불가 |

## 주요 UX 흐름

1. 화면 진입 시 전체 소모품 목록과 재고 부족 알림이 표시됩니다.
2. **출고 기록**: 소모품을 사용할 때 해당 품목의 [입출고] 버튼 → 출고 탭 → 수량 입력 → 저장.
3. **입고 기록**: 소모품이 도착하면 [입출고] 버튼 → 입고 탭 → 수량·공급업체 입력 → 저장.
4. **발주 생성**: 재고 부족 품목에서 [발주 생성] 버튼 클릭 → 발주 수량·공급업체 입력 → 발주서 생성.
5. **소모품 등록**: [소모품 등록] 버튼 → 품목명·카테고리·단위·안전 재고 기준 입력 → 저장.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 정상 재고 | 초록색 배지 표시 |
| 재고 부족 (안전 재고 이하) | 빨간색 배지 + 현재 재고 수량 강조 표시 |
| 재고 없음 | 빨간색 강조 + "발주 필요" 안내 |
| 발주 대기 중 | 주황색 배지 표시 |
| 품목 없음 | "등록된 소모품이 없습니다" 안내 + [소모품 등록] 버튼 |
