## 화면 목적

특정 RFID 카드 또는 밴드의 과거 사용 이력을 기간별로 조회하는 다이얼로그입니다. 언제, 어디서 해당 카드가 사용되었는지 확인할 수 있습니다.

## 주요 구성요소

- **다이얼로그 제목**: "RFID 이력 조회 — [카드 번호]"
- **기간 선택**: 조회 시작일과 종료일을 선택합니다.
- **조회 버튼**: 선택한 기간의 이력을 불러옵니다.
- **이력 테이블**: 등록일 / 카드 번호 / 회원명 / 상태를 시간 역순으로 표시합니다.
- **닫기 버튼**: 다이얼로그를 닫습니다.

## 주요 UX 흐름

1. 카드 행의 [이력] 버튼 클릭 → 다이얼로그 열림.
2. 기간을 선택하고 [조회]를 클릭합니다.
3. 이력 테이블에서 사용 기록을 확인합니다.
4. [닫기]로 다이얼로그를 닫습니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 이력 있음 | 기간별 사용 이력 행으로 표시 |
| 이력 없음 | "해당 기간의 이력이 없습니다" 안내 메시지 |
