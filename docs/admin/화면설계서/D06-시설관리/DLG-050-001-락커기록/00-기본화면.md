## 화면 목적

특정 락커의 과거 사용 이력(배정·회수 기록)을 시간순으로 조회하고, 해당 락커의 비밀번호와 메모를 확인·수정하는 다이얼로그입니다.

## 주요 구성요소

- **다이얼로그 제목**: "[락커 번호]번 락커 이용 이력"
- **이력 테이블**: 날짜 / 상태(배정·회수 배지) / 회원명을 시간 역순으로 표시합니다.
- **비밀번호 편집**: 해당 락커에 설정된 비밀번호를 확인하고 변경할 수 있습니다.
- **메모 편집**: 해당 락커에 대한 관리자 메모를 입력·수정할 수 있습니다.
- **저장 버튼**: 비밀번호·메모 변경 사항을 저장합니다.
- **닫기 버튼**: 다이얼로그를 닫고 락커 배치도로 돌아갑니다.

## 주요 UX 흐름

1. 락커 셀 호버 시 [기록] 버튼 클릭 → 다이얼로그 열림.
2. 이력 테이블에서 해당 락커의 과거 배정·회수 기록을 확인합니다.
3. 필요 시 비밀번호 또는 메모를 수정하고 [저장]을 클릭합니다.
4. [닫기]로 다이얼로그를 닫습니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 이력 있음 | 날짜·상태·회원명이 행으로 표시 |
| 이력 없음 | "이용 이력이 없습니다" 안내 메시지 |
| 저장 성공 | "비밀번호/메모가 저장되었습니다" 알림 |
| 저장 실패 | "저장에 실패했습니다" 오류 알림 |
