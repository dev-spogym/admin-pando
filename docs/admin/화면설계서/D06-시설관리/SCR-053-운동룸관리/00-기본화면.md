## 화면 목적

센터 내 운동룸(GX룸·PT룸·스피닝룸·필라테스룸 등)의 현황과 예약 슬롯을 관리하는 화면입니다. 각 룸의 운영 상태를 확인하고, 오늘의 시간대별 예약 혼잡도를 시각적으로 파악할 수 있습니다. 새 운동룸을 등록하거나 기존 룸 정보를 수정·삭제하고, 운영 상태를 빠르게 전환할 수 있습니다.

## 주요 구성요소

- **현황 요약 카드**: 전체 룸 수 / 운영중 수 / 점검·고장·미사용 합계 수 / GX룸 수·PT룸 수를 4개 카드로 표시합니다.
- **뷰 전환 탭**: 카드 보기와 목록 보기를 전환합니다.
- **유형 필터**: 전체 · GX · PT · 스피닝 · 필라테스 · 기타로 필터링합니다.
- **카드 보기**: 유형별로 묶어 각 룸을 카드 형태로 표시합니다.
  - 카드 내용: 룸명 / 유형 배지 / 수용 인원 / 게이트 / 운영 상태 배지 / 룸 설명 / 오늘 시간대별 예약 현황 슬롯 바 / 수정·삭제 버튼.
  - 슬롯 바: 07:00~20:00 시간대를 색상으로 표시합니다 (여유: 초록 / 혼잡: 주황 / 만석: 빨강 / 빈 시간: 회색).
  - 운영 상태 전환 버튼(⚙): 운영중 → 점검중 → 고장 → 운영중 순서로 순환 전환합니다.
- **목록 보기**: 룸 목록을 테이블 형태로 표시합니다.
- **새 운동룸 등록 버튼**: 새 운동룸을 등록합니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 센터장 | 전체 룸 현황 | 등록·수정·삭제·상태 전환 전체 가능 |
| 매니저 | 전체 룸 현황 | 상태 전환·수정 가능 (삭제 불가) |
| 스태프 / 프런트 | 전체 룸 현황 조회 | 상태 전환만 가능 |
| 트레이너 / 조회전용 | 현황 조회만 | 변경 불가 |

## 주요 UX 흐름

1. 화면 진입 시 전체 운동룸을 유형별로 묶어 카드 형태로 표시합니다.
2. 유형 필터를 선택하면 해당 유형의 룸만 표시됩니다.
3. **룸 등록**: [새 운동룸 등록] 클릭 → 룸명·유형·수용 인원·설명 입력 → 저장.
4. **룸 수정**: 카드의 [수정] 버튼 클릭 → 정보 수정 → 저장.
5. **룸 삭제**: 카드의 [삭제] 버튼 클릭 → 삭제 확인 다이얼로그 → 완료.
6. **상태 전환**: 카드의 [⚙] 버튼을 클릭할 때마다 운영중→점검중→고장→운영중 순서로 상태가 변경됩니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 로딩 중 | 헤더·카드 영역 스켈레톤 표시 |
| 정상 | 유형별 룸 카드 목록 표시 |
| 고장 룸 | 카드 테두리와 배경이 빨간색 계열로 강조 표시 |
| 점검중 룸 | 카드 투명도 80% + 주황색 테두리 / 예약 슬롯에 "점검 중 - 예약 불가" 표시 |
| 운동룸 없음 | "등록된 운동룸이 없습니다" 안내 + [새 운동룸 등록] 버튼 |
