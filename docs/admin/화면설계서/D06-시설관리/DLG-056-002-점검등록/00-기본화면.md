## 화면 목적

운동 장비의 점검을 완료한 후 점검 결과를 기록하는 다이얼로그입니다. 점검 완료일·점검자·점검 결과를 입력하면 다음 점검 예정일이 자동으로 갱신됩니다.

## 주요 구성요소

- **다이얼로그 제목**: "점검 등록 — [장비명]"
- **점검 완료일**: 실제 점검이 완료된 날짜를 입력합니다 (기본값: 오늘).
- **점검자**: 점검을 수행한 담당자 이름을 입력합니다.
- **점검 결과**: 정상 / 이상 발견 중 선택합니다.
- **이상 내용**: 점검 결과가 "이상 발견"일 경우 이상 내용을 입력합니다.
- **다음 점검 예정일**: 점검 주기에 따라 자동 계산되어 표시됩니다. 수동 변경도 가능합니다.
- **저장 버튼**: 점검 기록을 저장합니다.
- **취소 버튼**: 변경 없이 다이얼로그를 닫습니다.

## 주요 UX 흐름

1. 장비 행의 [점검 등록] 버튼 클릭 → 다이얼로그 열림.
2. 점검 완료일·점검자·점검 결과를 입력합니다.
3. 이상이 발견된 경우 이상 내용을 추가로 입력합니다.
4. [저장] 클릭 → 점검 이력 기록 + 다음 점검 예정일 자동 갱신.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 정상 결과 | 다음 점검 예정일이 주기에 따라 자동 계산 |
| 이상 발견 | 이상 내용 입력 필드 활성화 + 수리 접수 권고 안내 |
| 저장 성공 | 다이얼로그 닫힘 + 장비 목록 갱신 |
