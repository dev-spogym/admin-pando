## 화면 목적

골프 연습장 내 타석의 실시간 이용 현황을 관리하는 화면입니다. 타석별 사용자·프로·남은 이용 시간을 실시간으로 확인하고, 이용 시작·종료·타석 간 이동·점검 처리를 수행합니다. 대기 중인 회원 목록과 자동 배정 기능으로 빈 타석이 생기면 다음 대기자에게 즉시 안내할 수 있습니다.

## 주요 구성요소

- **현황 요약 카드**: 전체 타석 수 / 사용중 / 대기(빈 타석) / 예약 / 대기열 인원을 5개 카드로 표시합니다.
- **타석 그리드**: 전체 타석을 4열 격자로 표시합니다.
  - 각 타석 카드 내용: 타석 번호 / 상태 배지 / 이용 회원명 / 담당 프로명 / 남은 시간 카운트다운 / 프로젝터 on/off 토글 / 시작·종료·이동·점검 버튼.
  - 타석 상태: 대기(초록) / 사용중(파랑) / 예약(노랑) / 점검(회색).
  - 남은 시간이 0이 되면 알림이 표시됩니다.
- **대기열 패널**: 화면 하단에 현재 대기 중인 회원 목록을 순서대로 표시합니다.
  - 각 항목: 순서 번호 / 회원명 / 대기 시작 시간 / 예상 대기 시간.
  - 타석이 종료되면 첫 번째 대기자가 자동으로 해당 타석에 배정됩니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 센터장 / 매니저 | 전체 타석 현황 + 대기열 | 시작·종료·이동·점검·프로젝터 제어 전체 가능 |
| 스태프 / 프런트 | 전체 타석 현황 + 대기열 | 시작·종료·점검 가능 |
| 트레이너 | 담당 타석 현황 조회 | 프로젝터 제어만 가능 |
| 조회전용 | 현황 조회만 | 변경 불가 |

## 주요 UX 흐름

1. 화면 진입 시 전체 타석의 실시간 현황이 그리드로 표시됩니다.
2. **이용 시작**: 빈 타석의 [시작] 버튼 클릭 → 회원명 입력 → 이용 시작 (타이머 카운트다운 시작).
3. **이용 종료**: 사용중 타석의 [종료] 버튼 클릭 → 타석 초기화 → 대기열 첫 번째 회원이 있으면 자동 배정.
4. **타석 이동**: 사용중 타석의 [이동] 버튼 클릭 → 이동할 대상 타석 선택 → 회원 정보와 남은 시간이 새 타석으로 이동.
5. **점검 처리**: 빈 타석의 [점검] 버튼 클릭 → 점검 상태로 전환. 점검중 타석의 [점검 해제]로 복구.
6. **프로젝터 제어**: 사용중 타석의 프로젝터 on/off 버튼으로 프로젝터 상태를 토글합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 대기(빈 타석) | 초록색 카드 + [시작] 버튼 활성화 |
| 사용중 | 파란색 카드 + 회원명·프로명·남은 시간 표시 + [종료][이동] 버튼 |
| 예약 | 노란색 카드 + 예약 회원명 표시 |
| 점검 | 회색 카드 + [점검 해제] 버튼 |
| 시간 종료 | 화면 상단에 "⏰ [타석명] 시간 종료! ([회원명])" 경고 알림 팝업 |
| 대기열 없음 | 대기열 패널에 "대기 중인 회원이 없습니다" 안내 |
