## 화면 목적

RFID 카드 또는 밴드 정보를 영구 삭제하기 전 최종 의사를 확인하는 다이얼로그입니다. 삭제 시 연결된 회원과의 관계가 끊어지며 복구할 수 없다는 점을 명확히 안내합니다.

## 주요 구성요소

- **다이얼로그 제목**: "카드 삭제"
- **안내 메시지**: "카드 ID: [카드 번호]를 삭제하시겠습니까? 삭제된 정보는 복구할 수 없으며, 연결된 회원과의 관계가 끊어집니다."
- **삭제 버튼**: 위험(danger) 스타일로 표시됩니다.
- **취소 버튼**: 변경 없이 다이얼로그를 닫습니다.

## 주요 UX 흐름

1. 카드 행의 [삭제] 버튼 클릭 → 다이얼로그 열림.
2. 안내 메시지에서 삭제 대상 카드 번호와 경고를 확인합니다.
3. [삭제] 클릭 → 카드 정보 영구 삭제 → 목록에서 제거.
4. [취소] 클릭 시 변경 없이 닫힘.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 삭제 성공 | 목록에서 해당 카드 행 제거 |
| 삭제 실패 | 오류 알림 표시 |
