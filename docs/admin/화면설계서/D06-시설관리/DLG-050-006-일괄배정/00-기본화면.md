## 화면 목적

여러 개의 빈 락커를 한 명의 회원에게 동시에 배정하는 다이얼로그입니다. 그리드에서 원하는 빈 락커를 여러 개 선택한 뒤 회원 정보와 만료일을 입력해 한 번에 배정합니다.

## 주요 구성요소

- **다이얼로그 제목**: "일괄 배정"
- **선택된 락커 수 안내**: "빈 락커를 클릭하여 선택하세요. 현재 [N]개 선택됨" 안내 배너.
- **회원 검색 자동완성**: 배정할 회원을 검색합니다.
- **만료일 입력**: 일괄 배정 공통 만료일을 설정합니다.
- **[N]개 락커 배정 버튼**: 선택한 락커를 모두 배정합니다. 회원·만료일·선택 락커가 모두 준비된 경우에만 활성화됩니다.
- **취소 버튼**: 선택 해제 및 일괄 배정 모드를 종료합니다.

## 주요 UX 흐름

1. [일괄 배정] 버튼 클릭 → 일괄 배정 모드 진입 + 안내 배너 표시.
2. 그리드에서 빈 락커 셀을 클릭해 선택 (여러 개 선택 가능).
3. 이 다이얼로그에서 회원 검색 후 선택, 만료일 입력.
4. [[N]개 락커 배정] 클릭 → "[N]개 락커가 [회원명]님께 배정되었습니다" 알림.
