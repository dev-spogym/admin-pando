## 화면 목적

선택한 여러 만료 임박 락커를 한 번에 해제하기 전 최종 의사를 확인하는 다이얼로그입니다. 해제 시 모든 선택 락커의 회원 배정 정보가 초기화된다는 점을 명확히 안내합니다.

## 주요 구성요소

- **다이얼로그 제목**: "만료 임박 락커 일괄 해제"
- **안내 메시지**: "선택한 [N]개의 락커를 일괄 해제하시겠습니까? 해제 시 배정된 회원 정보가 모두 초기화됩니다."
- **[N]개 일괄 해제 버튼**: 위험(danger) 스타일로 표시됩니다.
- **취소 버튼**: 변경 없이 다이얼로그를 닫습니다.

## 주요 UX 흐름

1. 일괄 해제 모드에서 만료 임박 락커를 선택한 뒤 [일괄 해제 실행] 클릭 → 다이얼로그 열림.
2. 안내 메시지에서 선택한 락커 수와 경고 내용을 확인합니다.
3. [[N]개 일괄 해제] 클릭 → "[N]개 락커가 일괄 해제되었습니다" 알림.
4. [취소] 클릭 시 변경 없이 닫힘.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 일괄 해제 성공 | "[N]개 락커가 일괄 해제되었습니다" 알림 + 그리드 갱신 |
| 일괄 해제 실패 | "일괄 해제에 실패했습니다" 오류 알림 |
