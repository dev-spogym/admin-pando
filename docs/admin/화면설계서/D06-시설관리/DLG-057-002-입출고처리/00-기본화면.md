## 화면 목적

소모품의 입고 또는 출고 수량을 기록하는 다이얼로그입니다. 처리 유형(입고/출고)을 선택하고 수량을 입력하면 현재 재고 수량이 자동으로 갱신됩니다.

## 주요 구성요소

- **다이얼로그 제목**: "입출고 처리 — [품목명]"
- **처리 유형 선택**: 입고 / 출고 탭 또는 버튼으로 선택합니다.
- **현재 재고 표시**: 처리 전 현재 재고 수량을 읽기 전용으로 표시합니다.
- **처리 수량 입력**: 입고 또는 출고할 수량을 입력합니다.
- **처리 일자**: 실제 입출고 일자를 입력합니다 (기본값: 오늘).
- **공급업체/사유**: 입고 시 공급업체, 출고 시 사용 사유를 입력합니다 (선택 항목).
- **처리 후 예상 재고**: 입력한 수량 반영 시 예상 재고 수량을 미리 표시합니다.
- **저장 버튼**: 입출고 기록을 저장합니다.
- **취소 버튼**: 변경 없이 다이얼로그를 닫습니다.

## 주요 UX 흐름

1. 소모품 행의 [입출고] 버튼 클릭 → 다이얼로그 열림.
2. 입고 또는 출고 탭을 선택합니다.
3. 수량·일자·사유를 입력합니다.
4. [저장] 클릭 → 재고 수량 갱신 + 이력 기록.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 수량 미입력 | [저장] 버튼 비활성화 |
| 출고 시 재고 부족 | 경고 메시지 표시 ("현재 재고보다 많은 수량을 출고할 수 없습니다") |
| 저장 성공 | 다이얼로그 닫힘 + 재고 수량 갱신 |
