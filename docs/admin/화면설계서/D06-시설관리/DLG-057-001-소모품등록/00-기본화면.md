## 화면 목적

새로운 소모품 항목을 재고 관리 시스템에 등록하는 다이얼로그입니다. 품목명·카테고리·단위·안전 재고 기준 수량을 설정합니다.

## 주요 구성요소

- **다이얼로그 제목**: "소모품 등록"
- **품목명 입력**: 필수. 소모품의 이름을 입력합니다 (예: 샴푸 500ml, 면 타월).
- **카테고리 선택**: 욕실용품 / 청소용품 / 비품 / 기타 중 선택합니다.
- **단위 입력**: 재고를 측정하는 단위를 입력합니다 (예: 개, 박스, L).
- **안전 재고 기준**: 재고 부족 알림을 받을 기준 수량을 입력합니다.
- **초기 재고 수량**: 등록 시 초기 보유 수량을 입력합니다.
- **공급업체**: 주요 공급업체 정보를 입력합니다 (선택 항목).
- **저장 버튼**: 소모품 항목을 등록합니다.
- **취소 버튼**: 변경 없이 다이얼로그를 닫습니다.

## 주요 UX 흐름

1. [소모품 등록] 버튼 클릭 → 다이얼로그 열림.
2. 품목명·카테고리·단위·안전 재고 기준·초기 수량을 입력합니다.
3. [저장] 클릭 → 소모품 목록에 추가.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 필수 항목 미입력 | [저장] 버튼 비활성화 또는 유효성 오류 메시지 |
| 등록 성공 | 다이얼로그 닫힘 + 목록에 새 품목 추가 |
