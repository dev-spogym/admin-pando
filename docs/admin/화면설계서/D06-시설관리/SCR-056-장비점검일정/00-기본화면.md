## 화면 목적

센터 내 운동 장비의 정기 점검 일정과 수리 이력을 관리하는 화면입니다. 장비별로 다음 점검 예정일을 설정하고, 점검 완료 후 결과를 기록하며, 고장·수리 접수부터 완료까지의 이력을 추적합니다. 점검이 지연되거나 수리가 필요한 장비를 한눈에 파악해 시설 안전을 유지하는 것이 목적입니다.


## 주요 구성요소

- **현황 요약 카드**: 전체 장비 수 / 점검 예정(7일 이내) 수 / 수리 중 수 / 정상 운영 수를 표시합니다.
- **장비 목록 테이블**: 등록된 모든 장비를 표시합니다.
  - 표시 항목: 장비명 / 장비 유형 / 설치 위치 / 마지막 점검일 / 다음 점검 예정일 / 점검 주기 / 현재 상태(정상·점검 예정·수리중·고장) / 액션 버튼.
  - 다음 점검 예정일이 7일 이내인 장비는 주황색으로 강조 표시합니다.
  - 수리 중인 장비는 빨간색으로 강조 표시합니다.
- **장비 등록 버튼**: 새 장비 항목을 추가합니다.
- **점검 등록 버튼**: 장비별 점검 완료 기록을 입력합니다.
- **수리 접수 버튼**: 고장·이상 장비의 수리를 접수합니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 센터장 | 전체 장비 목록 및 이력 | 장비 등록·수정·삭제·점검 등록·수리 접수 전체 가능 |
| 매니저 | 전체 장비 목록 및 이력 | 점검 등록·수리 접수 가능 (장비 삭제 불가) |
| 스태프 | 장비 목록 및 이력 조회 | 점검 완료 기록만 가능 |
| 트레이너 / 조회전용 | 현황 조회만 | 변경 불가 |

## 주요 UX 흐름

1. 화면 진입 시 전체 장비 목록이 다음 점검 예정일 기준으로 표시됩니다.
2. 점검 예정이 임박한 장비를 확인하고 [점검 등록] 버튼을 클릭해 점검 완료 결과를 기록합니다.
3. 고장이 발견된 경우 [수리 접수] 버튼을 클릭해 수리 내용과 담당자를 등록합니다.
4. 수리 완료 후 수리 결과를 기록하면 장비 상태가 정상으로 복구됩니다.
5. 새 장비 도입 시 [장비 등록] 버튼으로 장비 정보와 점검 주기를 설정합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 정상 운영 | 초록색 배지 표시 |
| 점검 예정 임박 (7일 이내) | 주황색 배지 + 날짜 강조 표시 |
| 점검 기한 초과 | 빨간색 배지 + 경고 표시 |
| 수리 중 | 빨간색 배지 + 수리 접수일 표시 |
| 고장 | 빨간색 배지 + 사용 금지 안내 |
| 장비 없음 | "등록된 장비가 없습니다" 안내 + [장비 등록] 버튼 |
