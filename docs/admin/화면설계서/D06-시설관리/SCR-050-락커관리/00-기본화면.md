## 화면 목적

센터 내 모든 락커의 배정 현황과 상태를 한눈에 파악하고 관리하는 화면입니다. 구역별(A/B/C) 그리드 배치도로 각 락커의 사용 여부·만료 임박·고장 상태를 시각적으로 확인하고, 회원 배정·이동·회수·고장 처리를 즉시 수행할 수 있습니다. 만료 임박 락커를 한 번에 일괄 해제하거나 여러 빈 락커를 한 명의 회원에게 일괄 배정하는 기능도 지원합니다.

## 주요 구성요소

- **현황 요약 카드**: 전체 락커 수 / 사용중 수 / 만료 임박 수 / 고장 수를 화면 상단에 4개 카드로 표시합니다.
- **구역 탭**: A구역 · B구역 · C구역으로 탭을 분리하여 원하는 구역만 집중해서 볼 수 있습니다.
- **락커 배치 그리드**: 실제 락커 배치와 동일한 형태의 격자 화면으로 각 락커를 색상으로 구분합니다.
  - 사용중: 파란색 계열 / 만료 임박(7일 이내): 주황색 계열 / 고장: 빨간색 계열 / 빈 락커: 회색
  - 각 셀에 락커 번호, 회원명(배정 시), 남은 일수를 표시합니다.
  - 셀에 마우스를 올리면 기록 · 이동 · 회수 · 배정 · 고장 처리 버튼이 나타납니다.
- **색상 범례**: 그리드 하단에 상태별 색상 의미를 안내합니다.
- **검색창**: 락커 번호 또는 배정된 회원명으로 빠르게 해당 락커를 찾을 수 있습니다.
- **만료 임박 일괄 해제**: 탭 우측에 버튼을 클릭하면 만료 임박 락커를 선택해 한 번에 회수할 수 있습니다.
- **일괄 배정**: 여러 빈 락커를 선택한 뒤 한 명의 회원에게 동시에 배정합니다.
- **엑셀 다운로드**: 현재 락커 전체 현황을 엑셀 파일로 내려받습니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 센터장 | 전체 구역 현황, 모든 락커 상세 | 배정·이동·회수·고장 처리·일괄 배정·일괄 해제·엑셀 다운로드 전체 가능 |
| 매니저 | 전체 구역 현황, 모든 락커 상세 | 배정·이동·회수·고장 처리·일괄 작업 가능 |
| 스태프 / 프런트 | 전체 구역 현황 조회 | 배정·회수·기록 조회 가능 (이동·일괄 작업 불가) |
| 트레이너 / 조회전용 | 현황 조회만 | 변경 불가 |

## 주요 UX 흐름

1. 화면 진입 시 현황 요약 카드와 A구역 그리드가 기본으로 표시됩니다.
2. 구역 탭을 클릭하거나 검색창에 락커 번호·회원명을 입력해 원하는 락커를 찾습니다.
3. 락커 셀에 마우스를 올리면 해당 락커에서 할 수 있는 액션 버튼이 표시됩니다.
4. 빈 락커: [배정] 버튼 클릭 → 회원 검색 후 선택 → 만료일 입력 → 배정 완료.
5. 사용중 락커: [회수] 버튼 클릭 → 확인 다이얼로그 → 회수 완료.
6. 만료 임박 락커 일괄 처리: [만료 임박 일괄 해제] 클릭 → 해당 락커들을 클릭해 선택 → [일괄 해제 실행] → 확인 다이얼로그 → 완료.
7. 일괄 배정: [일괄 배정] 버튼 클릭 → 빈 락커를 클릭해 선택 → 회원·만료일 입력 → 한 번에 배정.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 로딩 중 | 락커 데이터를 불러오는 동안 중앙에 안내 메시지 표시 |
| 정상 (데이터 있음) | 구역별 그리드와 현황 카드 표시 |
| 검색 결과 없음 | 검색어에 해당하는 락커가 없을 때 "검색 결과가 없습니다" 안내 |
| 일괄 해제 모드 | 주황색 안내 배너 표시 — "만료 임박 락커를 클릭하여 선택하세요" |
| 일괄 배정 모드 | 파란색 안내 배너 표시 — "빈 락커를 클릭하여 선택하세요" |
| 고장 락커 | 해당 셀이 반투명 사선 처리 + "고장" 텍스트 표시 |
| 만료 임박 락커 | 주황색 배경 + 남은 일수 표시 (만료일 도래 또는 경과 시 빨간색 굵은 글씨) |
