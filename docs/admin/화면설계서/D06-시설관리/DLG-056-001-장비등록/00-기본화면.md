## 화면 목적

새로운 운동 장비를 시스템에 등록하는 다이얼로그입니다. 장비명·유형·설치 위치·점검 주기를 입력해 이후 정기 점검 일정이 자동으로 생성됩니다.

## 주요 구성요소

- **다이얼로그 제목**: "장비 등록"
- **장비명 입력**: 필수. 장비의 이름을 입력합니다 (예: 트레드밀 1번, 레그프레스 A).
- **장비 유형 선택**: 유산소기구 / 웨이트기구 / GX장비 / 기타 중 선택합니다.
- **설치 위치 입력**: 해당 장비가 위치한 구역·위치를 입력합니다.
- **점검 주기 설정**: 정기 점검 주기를 일 단위로 입력합니다 (예: 30일, 90일).
- **첫 점검 예정일**: 최초 점검 예정일을 날짜로 입력합니다.
- **메모**: 추가 설명을 입력합니다 (선택 항목).
- **저장 버튼**: 장비를 등록합니다.
- **취소 버튼**: 변경 없이 다이얼로그를 닫습니다.

## 주요 UX 흐름

1. [장비 등록] 버튼 클릭 → 다이얼로그 열림.
2. 장비명·유형·위치·점검 주기·첫 점검 예정일을 입력합니다.
3. [저장] 클릭 → 장비 목록에 추가 + 다음 점검 예정일 자동 생성.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 필수 항목 미입력 | [저장] 버튼 비활성화 또는 유효성 오류 메시지 |
| 등록 성공 | 다이얼로그 닫힘 + 목록에 새 장비 추가 |
