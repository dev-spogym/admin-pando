## 목적

자동 계산된 급여 항목 중 수당, 공제 등 일부 항목을 관리자가 직접 조정해야 할 때 사용하는 편집 다이얼로그입니다. 기본급은 직원 정보에서 가져오고, 이 다이얼로그에서는 부가 항목만 수정합니다.

## 구성요소

- **직원 정보 요약**: 편집 대상 직원의 이름·역할·지급 월을 다이얼로그 상단에 표시합니다.
- **수당 항목 목록**: 식대 / 교통비 / 특별 수당 / 성과 수당 등 각 항목별 금액을 입력 필드로 표시합니다. 항목 추가 버튼으로 새 수당 항목을 추가할 수 있습니다.
- **공제 항목 목록**: 국민연금 / 건강보험 / 고용보험 / 기타 공제 항목별 금액을 입력 필드로 표시합니다.
- **실지급액 미리보기**: 수당·공제 입력값에 따라 실지급액이 실시간으로 계산되어 하단에 표시됩니다.
- **저장 버튼**: 변경 내용을 저장하고 다이얼로그를 닫습니다.
- **취소 버튼**: 변경 내용을 저장하지 않고 다이얼로그를 닫습니다.

## 언제 표시되나요

- 급여 관리 화면(SCR-064)에서 특정 직원 행의 "편집" 버튼을 클릭할 때 표시됩니다.

## 주요 흐름

1. 관리자가 SCR-064에서 편집할 직원의 "편집" 버튼을 클릭합니다.
2. 이 다이얼로그가 열리며 현재 수당·공제 항목과 금액이 표시됩니다.
3. 필요한 항목의 금액을 수정하거나 새 항목을 추가합니다.
4. 하단 실지급액 미리보기를 확인합니다.
5. "저장"을 클릭하면 변경 내용이 반영되고 다이얼로그가 닫힙니다.
6. "취소"를 클릭하면 변경 내용이 무시되고 다이얼로그가 닫힙니다.
