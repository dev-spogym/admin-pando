## 목적

직원 등록/수정 전용 화면(SCR-061)에서 작성 중인 폼을 취소할 때, 입력 중인 데이터가 저장되지 않는다는 사실을 한 번 더 확인시켜 실수로 인한 데이터 손실을 방지합니다.

## 구성요소

- **제목**: "페이지를 벗어나시겠습니까?"
- **안내 메시지**: "작성 중인 내용이 저장되지 않습니다. 페이지를 이동하시겠습니까?"
- **계속 작성 버튼**: 다이얼로그를 닫고 현재 폼으로 돌아갑니다.
- **이동 확인 버튼**: 폼을 저장하지 않고 직원 목록(SCR-060)으로 이동합니다.

## 언제 표시되나요

- 직원 등록/수정 화면(SCR-061)에서 "취소" 버튼을 클릭할 때 표시됩니다.
- 브라우저 뒤로 가기 또는 사이드바 메뉴를 클릭해 페이지를 이탈하려 할 때도 표시될 수 있습니다.

## 주요 흐름

1. 사용자가 SCR-061에서 직원 정보를 입력하는 중 "취소"를 클릭하거나 다른 메뉴로 이동을 시도합니다.
2. 이 다이얼로그가 표시됩니다.
3. "계속 작성"을 선택하면 다이얼로그가 닫히고 폼이 유지됩니다.
4. "이동 확인"을 선택하면 저장 없이 직원 목록(SCR-060)으로 이동합니다.
