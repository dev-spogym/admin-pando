## 화면 목적

직원별 당월 급여를 계산하고 수당·공제 내역을 검토한 뒤 최종 급여를 확정하는 화면입니다. 근태 데이터와 연동되어 기본급에 각종 수당이 자동 반영되며, 관리자가 항목을 수동으로 편집한 후 일괄 확정할 수 있습니다.

## 주요 구성요소

- **지급 월 선택**: 급여를 처리할 대상 월을 선택합니다. 기본값은 당월입니다.
- **요약 지표 카드**: 전체 지급 예정 금액 / 확정 완료 인원 / 미확정 인원 / 이전 월 대비 증감을 표시합니다.
- **직원별 급여 테이블**: 직원명 / 역할 / 기본급 / 수당 합계 / 공제 합계 / 실지급액 / 확정 상태 컬럼으로 구성됩니다.
- **수당·공제 편집**: 각 직원 행의 편집 버튼 클릭 시 DLG-064-001이 열려 항목별 금액을 수동으로 조정합니다.
- **급여 확정 버튼**: 개별 또는 전체 직원의 급여를 확정합니다. 확정 전 DLG-064-002 확인 다이얼로그가 표시됩니다.
- **확정 취소**: 이미 확정된 급여도 당월 내에는 확정 취소 후 재편집할 수 있습니다.
- **엑셀 내보내기**: 당월 급여 내역 전체를 엑셀 파일로 내려받습니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 | 소속 센터 전체 직원 급여 | 수당·공제 편집, 급여 확정, 확정 취소, 엑셀 내보내기 |
| 매니저 | 소속 센터 전체 직원 급여 | 수당·공제 편집 (확정 권한은 센터장만) |
| 직원 본인 | 본인 급여 정보만 | 조회만 가능 |
| FC / 트레이너 / 스태프 | 접근 불가 | 급여 관리 화면 진입 불가 |

## 주요 UX 흐름

1. 사이드바 "직원 관리 > 급여 관리"로 진입합니다.
2. 지급 월을 선택하면 해당 월 직원별 급여 데이터가 로드됩니다.
3. 근태 데이터 기반으로 기본급과 수당이 자동 계산되어 표시됩니다.
4. 수정이 필요한 직원의 편집 버튼을 클릭해 수당·공제 항목을 조정합니다(DLG-064-001).
5. 검토 완료 후 "급여 확정" 버튼을 클릭하면 확정 확인 다이얼로그(DLG-064-002)가 표시됩니다.
6. 확정 후 해당 직원은 급여 명세서(SCR-065)에서 내역을 조회할 수 있습니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 로딩 중 | 급여 데이터를 계산하는 동안 로딩 표시가 나타납니다. |
| 미확정 상태 | 급여 항목이 표시되고 편집·확정 버튼이 활성화됩니다. |
| 확정 완료 | 해당 행이 "확정" 배지로 표시되며 편집 버튼이 비활성화됩니다. |
| 데이터 없음 | 해당 월에 재직 중인 직원이 없을 때 안내 메시지가 표시됩니다. |
| 확정 처리 중 | 확정 버튼이 로딩 상태로 변경되며 중복 클릭이 방지됩니다. |
