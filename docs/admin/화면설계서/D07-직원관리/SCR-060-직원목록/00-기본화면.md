## 화면 목적

센터에 소속된 모든 직원의 현황을 한눈에 파악하고, 직무·재직 상태별로 빠르게 찾을 수 있는 직원관리 도메인의 핵심 허브 화면입니다. 본사 관계자가 `전체 지점(통합)`을 선택한 경우에는 전 지점 직원이 합산 조회되며 각 직원의 소속 지점이 함께 표시됩니다. 센터장은 소속 지점 인력 현황을 조회하고 신규 직원을 등록하며, 매니저는 특정 직원에게 메시지를 전송하거나 퇴사 처리를 진행합니다. 이 화면에서 다루는 값은 직원의 `직무`이며, 실제 관리자웹 메뉴 접근 권한은 `SCR-081 권한 설정`에서 별도로 관리합니다.

## 주요 구성요소

- **요약 지표 카드 (4개)**: 전체 직원 수 / 재직 중 인원 / 휴직 중 인원 / 퇴사 인원을 한눈에 표시합니다. 카드 클릭 시 해당 상태로 자동 필터링됩니다.
- **검색 바**: 직원 이름 또는 연락처로 통합 검색합니다.
- **직무 필터**: 센터장 / 매니저 / FC / 트레이너 / 스태프 / 프런트 중 선택해 해당 직무 직원만 표시합니다.
- **소속 지점 필터**: 본사 통합 모드에서만 표시되며, 전 지점 직원 목록 중 특정 지점 직원만 좁혀 봅니다.
- **재직 상태 필터**: 재직 / 휴직 / 퇴사 중 선택해 필터링합니다.
- **일괄 작업 버튼 바**: 직원을 1명 이상 체크박스 선택하면 메시지 전송 / 상태 변경 / 퇴사 처리 / 명단 다운로드 버튼이 활성화됩니다.
- **직원 목록 테이블**: 번호 / 직원명(클릭 시 상세 이동) / 소속 지점 / 직무 배지 / 연락처 / 입사일 / 재직 상태 배지 / 메모 / 더보기 메뉴 컬럼으로 구성됩니다.
- **페이지네이션**: 한 페이지에 10건씩 표시하며 페이지 번호로 이동합니다.
- **직원 등록 버튼**: 신규 직원 등록 화면(SCR-061)으로 이동합니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 본사 최고관리자 / 슈퍼관리자 | `전체 지점(통합)` 선택 시 전 지점 직원 목록과 지점별 소속 | 지점 전환, 전체 직원 검색·필터, 명단 다운로드, 권한 범위 내 직원 관리 |
| 센터장 / 매니저 | 소속 센터 전체 직원 목록 | 직원 등록, 상태 변경, 퇴사 처리, 메시지 전송, 명단 다운로드, 급여 확정 |
| FC / 트레이너 / 스태프 / 프런트 | 자신의 정보만 조회 가능 (동료 목록 열람 불가) | 본인 정보 조회만 가능 |
| 직원 본인 | 본인 프로필 및 급여 명세서 | 본인 정보 조회 |

## 주요 UX 흐름

1. 사이드바에서 "직원 관리"를 클릭해 직원 목록 화면으로 진입합니다.
2. 요약 카드로 전체 인원 현황을 파악하고, 검색 또는 필터를 조합해 원하는 직원을 찾습니다.
3. 직원명을 클릭하면 해당 직원의 상세 정보 화면으로 이동합니다.
4. 체크박스로 직원을 선택하면 일괄 작업 버튼이 활성화되어 메시지 전송 또는 퇴사 처리를 진행합니다.
5. 우측 상단 "직원 등록" 버튼으로 신규 직원 등록(SCR-061)으로 이동합니다.
6. 퇴사 처리 버튼 클릭 시 확인 다이얼로그(DLG-060-002)가 표시됩니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 로딩 중 | 데이터를 불러오는 동안 테이블 자리에 로딩 표시를 보여줍니다. |
| 정상 목록 | 직원이 1명 이상 존재할 때 테이블과 페이지네이션을 표시합니다. |
| 검색 결과 없음 | 검색·필터 적용 후 결과가 0건이면 "직원 데이터가 없습니다." 안내를 표시합니다. |
| 전체 직원 없음 | 필터 없이 0건이면 직원 등록 안내 메시지를 함께 표시합니다. |
| 오류 | 데이터 로드 실패 시 오류 메시지와 다시 시도 안내를 표시합니다. |
