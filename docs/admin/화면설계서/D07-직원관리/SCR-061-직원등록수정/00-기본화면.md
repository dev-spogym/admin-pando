## 화면 목적

신규 직원을 시스템에 등록하거나 기존 직원의 정보를 수정하는 화면입니다. 이름·연락처·직무 등 기본 정보와 함께 담당 서비스, 급여 형태, 시스템 계정 생성 여부까지 한 화면에서 설정합니다. 등록된 직원은 직원 목록(SCR-060)에 즉시 반영됩니다. 여기서 입력하는 `직무`는 인사/운영 분류값이며, 실제 관리자웹 메뉴 접근 권한은 `SCR-081 권한 설정`에서 별도로 매핑합니다.

## 주요 구성요소

- **기본 정보 섹션**: 직원명(필수) / 연락처 / 이메일 / 직무(필수, 센터장·매니저·FC·트레이너·스태프·프런트) / 직급 / 입사일을 입력합니다.
- **담당 서비스 섹션**: 해당 직원이 담당할 서비스 항목(예: PT, 필라테스 등)을 선택합니다.
- **급여 정보 섹션**: 급여 형태(월급제·시급제·건별 등)와 기본급을 입력합니다.
- **계정 설정 섹션**: 시스템 로그인 계정 생성 여부를 선택합니다. 계정 생성 시 이메일 주소가 로그인 ID로 사용됩니다.
- **캘린더 색상 선택**: 수업 캘린더에 표시될 직원 고유 색상을 지정합니다.
- **저장 / 취소 버튼**: 저장 시 유효성 검사 후 목록으로 돌아갑니다. 취소 클릭 시 DLG-061-001이 표시됩니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 입력 폼 | 직원 등록 및 수정, 계정 생성 |
| FC / 트레이너 / 스태프 / 프런트 | 본인 정보 조회만 | 본인 정보 수정 불가 (관리자 권한 필요) |

## 주요 UX 흐름

1. 직원 목록(SCR-060)에서 "직원 등록" 버튼을 클릭해 빈 폼으로 진입합니다. (신규 등록)
2. 또는 목록 테이블의 직원명 클릭 후 수정 버튼을 눌러 기존 데이터가 채워진 폼으로 진입합니다. (수정)
3. 필수 항목(이름, 직무)을 입력하고 나머지 항목을 작성합니다.
4. "저장" 버튼을 클릭하면 유효성 검사 후 저장됩니다.
5. 저장 성공 시 직원 목록(SCR-060)으로 돌아갑니다.
6. 입력 중 "취소"를 클릭하면 이탈 확인 다이얼로그(DLG-061-001)가 표시됩니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 신규 등록 | 모든 필드가 빈 상태로 표시됩니다. |
| 수정 | 기존 직원 정보가 각 필드에 채워진 상태로 표시됩니다. |
| 저장 중 | 저장 버튼이 로딩 상태로 변경되며 중복 클릭이 방지됩니다. |
| 유효성 오류 | 필수 항목 미입력 시 해당 필드 하단에 오류 메시지가 표시됩니다. |
| 저장 완료 | 성공 알림 후 직원 목록(SCR-060)으로 이동합니다. |

## 관련 다이어그램

- SCR-060 직원 목록으로 복귀
- DLG-061-001 등록/수정 취소 확인
