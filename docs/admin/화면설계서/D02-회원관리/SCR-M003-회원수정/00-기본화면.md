## 화면 목적

이미 등록된 회원의 기본 정보를 수정하는 화면입니다. 회원 등록(SCR-M002)과 동일한 입력 구조를 사용하되, 기존 데이터가 자동으로 채워진 상태에서 시작합니다. 수정 완료 후 회원 상세(SCR-M004)로 돌아가 변경 결과를 바로 확인할 수 있습니다.

## 주요 구성요소

- **단계 표시기 (Step 1 / Step 2)**: 등록 화면과 동일한 2단계 구조. 기존 값이 각 단계에 미리 채워져 있습니다.
- **프로필 사진**: 기존 사진이 표시되며, 클릭으로 새 사진으로 교체 가능.
- **[Step 1] 기본 인적 사항**: 이름 / 성별 / 연락처 + 중복 확인 / 회원구분 / 법인 회사명(법인인 경우) / 생년월일 / 키. 기존 연락처는 이미 확인된 것으로 처리되어 변경 시에만 중복 확인 필요.
- **[Step 1] 관리 정보**: 담당 FC / 담당 트레이너 / 방문경로 / 유입경로 / 운동 목적
- **[Step 2] 추가 연락 정보**: 별칭/닉네임 / 이메일 / 주소 / 상세 주소 / 회사명 / 출석번호
- **[Step 2] 기타 설정**: 광고성 정보 수신 동의 / 메모(최대 500자)
- **하단 고정 버튼**: Step 1에서는 [취소] [다음], Step 2에서는 [이전] [저장]
- **초기화 버튼**: 수정된 내용을 원래 값으로 되돌립니다 (빈 폼이 아닌 원본 데이터로 복원).

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 본사 최고관리자 | 전체 화면 | 회원 정보 수정 완전 접근 |
| 센터장 (owner) | 전체 화면 | 회원 정보 수정 완전 접근 |
| 매니저 | 전체 화면 | 회원 정보 수정 완전 접근 |
| 스태프 | 전체 화면 | 회원 정보 수정 완전 접근 |
| FC | 접근 불가 | 담당 회원이라도 수정 불가, 접근 제한 안내로 이동 |
| 트레이너 / 프런트 | 접근 불가 | 접근 제한 안내로 이동 |

## 주요 UX 흐름

1. 회원 상세(SCR-M004) 프로필 영역의 "수정" 버튼을 눌러 진입합니다.
2. 기존 회원 정보가 각 입력란에 자동으로 채워진 상태에서 시작합니다.
3. 수정이 필요한 항목을 변경하고 "다음" → "저장"을 누릅니다.
4. 저장 성공 시 "회원 정보가 수정되었습니다" 알림과 함께 회원 상세(SCR-M004)로 이동합니다.
5. 수정 내용이 있는 상태에서 취소를 누르면 확인 다이얼로그가 나타납니다.
6. 초기화를 누르면 수정 전 원래 값으로 복원됩니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 데이터 불러오는 중 | 기존 회원 정보를 조회하는 동안 로딩 화면 표시 |
| 초기값 채워짐 | 기존 데이터가 입력란에 모두 채워진 상태, 아직 수정 없음 |
| 수정 중 | 사용자가 값을 변경하기 시작한 상태 |
| 유효성 오류 | 필수 항목 누락 또는 형식 오류 시 해당 입력란에 오류 메시지 표시 |
| Step 2 | Step 1 완료 후 추가 정보 수정 단계 |
| 저장 중 | 저장 버튼 클릭 후 처리 중 |
| 불러오기 실패 | 회원 정보를 불러올 수 없는 경우(회원 없음 등) 오류 안내 + 목록으로 이동 버튼 |
