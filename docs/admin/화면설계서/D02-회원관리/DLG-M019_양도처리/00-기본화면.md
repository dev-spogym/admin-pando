## 목적
현재 회원의 이용권을 다른 회원에게 양도하는 처리를 확인하는 다이얼로그입니다. 양도 대상 회원을 검색하고 선택하여 이용권을 이전합니다.

## 구성요소
- 제목: "양도 처리"
- 양도할 이용권 정보 표시 (이용권명, 잔여 기간, 잔여 횟수)
- 양도 대상 회원 검색 (이름 또는 전화번호로 검색)
- 검색 결과 목록 (회원명, 회원번호, 연락처)
- 선택된 양도 대상 회원 표시
- 양도 사유 입력 (선택)
- 처리 버튼
- 취소 버튼

## 언제 표시되나요
- 회원 상세 > 이용권 탭에서 양도 버튼 클릭 시
- 잔여 기간 또는 잔여 횟수가 남아 있는 이용권에 한해 표시

## 주요 흐름
1. 관리자가 양도 버튼 클릭 → 다이얼로그 표시, 이용권 정보 표시
2. 양도 대상 회원 검색 → 검색 결과에서 대상 회원 선택
3. 사유 입력 후 처리 클릭 → 이용권 소유자 변경, 양도 이력 기록
4. 취소 클릭 → 다이얼로그 닫힘, 변경 없음
