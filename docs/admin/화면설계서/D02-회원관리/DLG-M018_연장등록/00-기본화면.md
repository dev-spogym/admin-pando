## 목적
이용권의 만료일을 연장 처리하는 다이얼로그입니다. 관리자가 연장 기간과 사유를 입력하면 만료일이 자동으로 재산정됩니다.

## 구성요소
- 제목: "연장 등록"
- 현재 만료일 표시 (읽기 전용)
- 연장 방식 선택 (기간 연장 / 날짜 직접 지정)
  - 기간 연장 선택 시: 연장 일수 또는 개월 수 입력
  - 날짜 직접 지정 선택 시: 새 만료일 날짜 선택
- 연장 후 만료일 자동 계산 표시
- 연장 사유 입력 (텍스트 입력 또는 사유 목록 선택)
- 등록 버튼
- 취소 버튼

## 언제 표시되나요
- 회원 상세 > 이용권 탭에서 연장 등록 버튼 클릭 시
- 현재 활성 이용권이 있는 회원에게만 표시

## 주요 흐름
1. 관리자가 연장 등록 버튼 클릭 → 다이얼로그 표시, 현재 만료일 표시
2. 연장 방식 선택 및 기간/날짜 입력 → 새 만료일 자동 계산·표시
3. 사유 입력 후 등록 클릭 → 만료일 갱신, 이용권 이력에 연장 내역 기록
4. 취소 클릭 → 다이얼로그 닫힘, 변경 없음
