## 목적
회원을 탈퇴 처리하기 전, 개인정보 마스킹 처리 및 서비스 이용 중단 사실을 안내하고 관리자의 최종 확인을 받는 다이얼로그입니다.

## 구성요소
- 제목: "탈퇴 처리하시겠습니까?"
- 경고 메시지: "탈퇴 처리 시 개인정보가 마스킹되며 서비스 이용이 중단됩니다."
- 탈퇴 사유 선택 (드롭다운: 본인 요청, 장기 미이용, 기타)
- 대상 회원명 표시
- 확인 버튼 (탈퇴 실행)
- 취소 버튼 (다이얼로그 닫기)

## 언제 표시되나요
- 회원 상세 화면에서 탈퇴 처리 버튼을 클릭했을 때
- 회원 목록에서 일괄 탈퇴 처리를 선택했을 때

## 주요 흐름
1. 관리자가 탈퇴 처리 버튼 클릭 → 다이얼로그 표시
2. 탈퇴 사유 선택
3. 확인 → 탈퇴 처리 완료, 개인정보 마스킹 적용, 회원 상태 '탈퇴'로 변경
4. 취소 → 다이얼로그 닫힘, 탈퇴 없음
