## 목적
트레이너가 회원에게 맞춤 운동 프로그램을 배정하는 다이얼로그입니다. 기존에 등록된 프로그램 목록에서 선택하거나 새로 구성하여 배정합니다.

## 구성요소
- 제목: "운동 프로그램 배정"
- 프로그램 선택 방식 탭 (기존 프로그램 선택 / 직접 구성)
  - 기존 프로그램 선택: 프로그램 목록 (프로그램명, 운동 목적, 기간, 담당 트레이너)
  - 직접 구성: 운동 종목, 세트, 반복 수, 무게 등 입력
- 배정 시작일 선택
- 배정 종료일 선택 (선택)
- 담당 트레이너 지정 (선택)
- 배정 메모 입력 (선택)
- 배정 버튼
- 취소 버튼

## 언제 표시되나요
- 회원 상세 > 운동 프로그램 탭에서 프로그램 배정 버튼 클릭 시
- 트레이너 또는 관리자 권한을 가진 사용자만 표시

## 주요 흐름
1. 트레이너가 프로그램 배정 버튼 클릭 → 다이얼로그 표시
2. 기존 프로그램 선택 또는 직접 구성 후 배정 기간 설정
3. 배정 클릭 → 프로그램 배정 완료, 회원의 운동 프로그램 탭에 반영
4. 취소 클릭 → 다이얼로그 닫힘, 변경 없음
