## 목적
관리자가 회원의 마일리지를 수동으로 적립하거나 차감하는 다이얼로그입니다. 시스템 오류 보정, 이벤트 지급 등의 용도로 사용합니다.

## 구성요소
- 제목: "마일리지 조정"
- 현재 마일리지 잔액 표시 (읽기 전용)
- 조정 유형 선택 (적립 / 차감)
- 조정 포인트 입력 (숫자 입력)
- 조정 후 예상 잔액 자동 계산 표시
- 조정 사유 입력 (필수, 텍스트 입력 또는 사유 목록 선택)
- 확인 버튼
- 취소 버튼

## 언제 표시되나요
- 회원 상세 > 마일리지 탭에서 마일리지 조정 버튼 클릭 시
- 관리자 권한을 가진 사용자에게만 표시

## 주요 흐름
1. 관리자가 마일리지 조정 버튼 클릭 → 다이얼로그 표시, 현재 잔액 표시
2. 조정 유형(적립/차감) 선택 및 포인트 입력 → 조정 후 예상 잔액 자동 계산
3. 사유 입력 후 확인 클릭 → 마일리지 반영, 조정 이력 기록
4. 취소 클릭 → 다이얼로그 닫힘, 변경 없음
