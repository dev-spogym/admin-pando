## 목적
관리자가 특정 회원에게 쿠폰을 수동으로 적용하는 다이얼로그입니다. 쿠폰 코드를 직접 입력하거나 발급 가능한 쿠폰 목록에서 선택하여 적용합니다.

## 구성요소
- 제목: "쿠폰 적용"
- 쿠폰 적용 방식 선택 탭 (코드 입력 / 목록 선택)
  - 코드 입력: 쿠폰 코드 입력 필드 + 조회 버튼
  - 목록 선택: 적용 가능한 쿠폰 목록 (쿠폰명, 혜택 내용, 유효기간)
- 선택된 쿠폰 상세 정보 표시 (할인 유형, 할인액/율, 사용 조건)
- 쿠폰 적용 대상 이용권 선택 (해당 시)
- 적용 버튼
- 취소 버튼

## 언제 표시되나요
- 회원 상세 > 이용권 또는 결제 탭에서 쿠폰 적용 버튼 클릭 시
- 결제 진행 중 쿠폰 적용 단계에서 표시

## 주요 흐름
1. 관리자가 쿠폰 적용 버튼 클릭 → 다이얼로그 표시
2. 코드 직접 입력 또는 목록에서 쿠폰 선택 → 쿠폰 정보 표시
3. 적용 버튼 클릭 → 해당 회원에게 쿠폰 적용, 할인 반영
4. 취소 클릭 → 다이얼로그 닫힘, 변경 없음
