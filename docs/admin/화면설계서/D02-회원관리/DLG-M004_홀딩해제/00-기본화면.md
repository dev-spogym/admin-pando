## 목적
현재 홀딩 중인 이용권을 재개할 때, 관리자의 의사를 최종 확인하는 다이얼로그입니다. 해제 시 잔여 홀딩 기간이 이용권 만료일에서 차감될 수 있음을 안내합니다.

## 구성요소
- 제목: "홀딩을 해제하시겠습니까?"
- 안내 메시지: 현재 홀딩 기간, 해제 후 이용권 만료일 변경 안내
- 확인 버튼 (홀딩 해제 실행)
- 취소 버튼 (다이얼로그 닫기)

## 언제 표시되나요
- 회원 상세 > 이용권 탭에서 홀딩 해제 버튼을 클릭했을 때
- 이용권 상태가 '홀딩'인 경우에만 버튼 활성화

## 주요 흐름
1. 관리자가 홀딩 해제 버튼 클릭 → 다이얼로그 표시
2. 변경될 이용권 만료일 미리보기 표시
3. 확인 → 홀딩 해제 처리 완료, 이용권 상태 '활성'으로 변경
4. 취소 → 다이얼로그 닫힘, 홀딩 유지
