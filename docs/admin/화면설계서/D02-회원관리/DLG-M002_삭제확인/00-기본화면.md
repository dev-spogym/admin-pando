## 목적
회원 데이터를 삭제하기 전, 되돌릴 수 없는 작업임을 명확히 고지하고 관리자의 최종 의사를 확인하는 다이얼로그입니다.

## 구성요소
- 제목: "회원을 삭제하시겠습니까?"
- 경고 메시지: "삭제된 데이터는 복구할 수 없습니다."
- 대상 회원명 표시
- 확인 버튼 (삭제 실행)
- 취소 버튼 (다이얼로그 닫기)

## 언제 표시되나요
- 회원 목록에서 삭제 버튼을 클릭했을 때
- 회원 상세 화면에서 삭제 버튼을 클릭했을 때

## 주요 흐름
1. 관리자가 삭제 버튼 클릭 → 다이얼로그 표시
2. 확인 → 회원 데이터 삭제 처리 후 다이얼로그 닫힘, 목록 화면으로 이동
3. 취소 → 다이얼로그 닫힘, 삭제 없음
