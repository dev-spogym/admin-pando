## 목적
회원을 다른 지점으로 이관하기 전에 최종 확인을 받는 다이얼로그입니다. 이관 후에는 원 지점에서 해당 회원 정보를 더 이상 관리하지 않음을 안내합니다.

## 구성요소
- 제목: "지점 이관 확인"
- 이관 대상 회원 정보 표시 (회원명, 현재 지점)
- 이관 대상 지점 표시 (이관될 지점명)
- 이관 시 함께 이전되는 정보 안내 (이용권, 출석 이력, 체성분 등)
- 이관 후 변경 사항 안내 메시지
- 확인(이관 처리) 버튼
- 취소 버튼

## 언제 표시되나요
- 회원 상세 > 기본정보 탭의 지점 이관 버튼 클릭 후, 이관 대상 지점을 선택한 직후
- 지점 이관 처리의 마지막 단계에서 표시

## 주요 흐름
1. 이관 대상 지점 선택 완료 → 이관 확인 다이얼로그 표시
2. 이관 정보 및 안내 사항 확인
3. 확인 클릭 → 회원 소속 지점 변경, 해당 지점 관리자에게 이관 알림 발송
4. 취소 클릭 → 다이얼로그 닫힘, 이관 취소
