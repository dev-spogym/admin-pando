## 목적
작성된 메모를 삭제하기 전, 복구가 불가능함을 안내하고 관리자의 최종 확인을 받는 다이얼로그입니다.

## 구성요소
- 제목: "메모를 삭제하시겠습니까?"
- 안내 메시지: "삭제된 메모는 복구할 수 없습니다."
- 삭제 대상 메모 내용 미리보기 (일부 텍스트)
- 확인 버튼 (삭제 실행)
- 취소 버튼 (다이얼로그 닫기)

## 언제 표시되나요
- 회원 상세 > 메모 탭에서 특정 메모의 삭제 버튼을 클릭했을 때

## 주요 흐름
1. 관리자가 메모 삭제 버튼 클릭 → 다이얼로그 표시
2. 확인 → 메모 삭제 완료, 다이얼로그 닫힘, 목록에서 해당 메모 제거
3. 취소 → 다이얼로그 닫힘, 메모 유지
