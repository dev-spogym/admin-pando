## 목적
회원 등록 또는 수정 시 이미 다른 회원에게 등록된 전화번호를 입력하면, 중복 사실을 안내하여 동일 번호로 복수 회원이 생성되는 것을 방지하는 다이얼로그입니다.

## 구성요소
- 제목: "이미 등록된 전화번호입니다"
- 안내 메시지: 해당 번호로 등록된 기존 회원명 표시
- 기존 회원 바로가기 버튼 (선택)
- 확인 버튼 (다이얼로그 닫기 후 입력 필드로 포커스 복귀)

## 언제 표시되나요
- 회원 등록 화면에서 전화번호 입력 후 중복 확인을 실행했을 때
- 회원 수정 화면에서 전화번호를 다른 회원의 번호로 변경하려 할 때

## 주요 흐름
1. 관리자가 전화번호 입력 후 저장 또는 중복 확인 클릭 → 중복 감지 시 다이얼로그 표시
2. 기존 회원 바로가기 클릭 → 해당 회원 상세 페이지로 이동
3. 확인 클릭 → 다이얼로그 닫힘, 전화번호 입력 필드 초기화 후 재입력 유도
