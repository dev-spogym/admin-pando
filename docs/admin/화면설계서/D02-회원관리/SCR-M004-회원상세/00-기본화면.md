## 화면 목적

개별 회원의 모든 정보를 15개 탭으로 분류해 조회하고 관리하는 회원관리 도메인의 중심 화면입니다. 화면 상단의 회원 프로필 요약과 주요 액션 버튼이 스크롤 중에도 고정되어 어느 탭에서든 즉시 접근 가능합니다. 회원 상태(정상이용 / 홀딩 / 만료 / 정지 / 비활성)에 따라 표시되는 배너, 활성화되는 버튼, 안내 메시지가 완전히 달라지며, 회원의 전체 라이프사이클을 단일 페이지에서 관리합니다.

## 주요 구성요소

- **이동 경로 표시**: 화면 상단에 "회원 목록 > 회원명 상세"로 현재 위치 안내.
- **만료 알림 배너**: 본사 정책 세트의 step 결과가 존재할 때 상단에 고정 표시. 색상과 강조 수준은 지점에 적용된 정책의 긴급도 기준을 따른다.
- **회원 프로필 요약 (고정 영역)**: 프로필 사진 / 관심회원 별표 / 회원명 / 등급 배지 / 상태 배지 / 만료일까지 남은 일수 / 연락처 / 이메일. 하단에 미수금·마일리지·이용권 계약 수·락커 번호 요약 표시.
- **주요 액션 버튼**: 수동출석 / 수정 / 상품구매 / 메시지 / 지점이관 / 탈퇴. 회원 상태와 계정 권한에 따라 버튼 활성/비활성이 결정됩니다.
- **15개 탭**: 회원정보 · 이용권 · 출석이력 · 결제이력 · 결제내역 · 예약내역 · 상세내역 · 체성분 · 상담메모 · 레슨 · 신체정보 · 종합평가 · 상담이력 · 운동프로그램 · 운동이력
- **회원 상태 관리 영역**: 정상이용 중이면 "홀딩 처리" 버튼, 홀딩 중이면 "홀딩 해제" 버튼, 정지 상태이면 권한자에게만 "정지 해제" 버튼 표시.
- **위험 구역**: 권한이 있는 계정에게만 화면 최하단에 표시. 회원 삭제 버튼 포함. 삭제는 되돌릴 수 없음을 명시.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 본사 최고관리자 | 전체 탭 + 전체 버튼 + 위험 구역 | 수정·이관·탈퇴·삭제·홀딩·정지해제 등 모든 기능 |
| 센터장 (owner) | 전체 탭 + 전체 버튼 + 위험 구역 | 모든 기능 + 정지 해제 |
| 매니저 | 전체 탭 | 수동출석·수정·상품구매·메시지·탈퇴·홀딩 (이관·삭제·정지해제 불가) |
| FC (피트니스 코치) | 전체 탭 (결제내역 조회만) | 수동출석·상품구매·메시지·관심회원 (수정·이관·탈퇴 불가) |
| 스태프 | 대부분 탭 (결제내역 제외) | 수동출석·수정·상품구매·메시지 (이관·탈퇴·삭제·홀딩 불가) |
| 트레이너 / 프런트 / 조회전용 | 접근 불가 | 이 화면 진입 차단 |

## 주요 UX 흐름

1. 회원 목록(SCR-M001)에서 회원명을 클릭하거나, 대시보드 위젯에서 특정 회원을 선택해 진입합니다.
2. 회원 프로필 요약과 상태 배너로 현재 상황을 파악합니다.
3. 탭을 선택해 이용권 현황, 출석 이력, 결제 내역 등 원하는 정보를 확인합니다.
4. 필요한 액션 버튼(수동출석·메시지·상품구매 등)을 클릭해 작업을 수행합니다.
5. 홀딩이나 탈퇴 등 상태 변경은 해당 버튼을 클릭하면 확인 다이얼로그가 나타나 처리합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 로딩 중 | 회원 정보를 불러오는 동안 스켈레톤 화면 표시 |
| 정상이용 중 (ACTIVE) | 만료 배너 없음, 모든 액션 버튼 활성, "홀딩 처리" 버튼 표시 |
| 홀딩 중 (HOLDING) | 보라색 배너로 홀딩 기간 및 사유 표시, "홀딩 해제" 버튼 표시 |
| 만료 (EXPIRED) | 빨간색 재등록 유도 배너 표시, 상품구매 버튼 강조, 수동출석·이관 불가 |
| 이용 정지 (SUSPENDED) | 주황색 정지 사유 배너 표시, 대부분 액션 차단, 권한자에게만 "정지 해제" 버튼 |
| 비활성 (INACTIVE) | 회색 배너로 읽기 전용 안내, 권한자에게 "복구" 버튼 표시 |
| 회원 없음 | 잘못된 접근이거나 삭제된 회원일 때 "회원을 찾을 수 없습니다" + 목록으로 이동 버튼 |
