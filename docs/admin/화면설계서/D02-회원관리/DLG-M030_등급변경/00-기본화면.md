## 목적
관리자가 회원의 등급을 수동으로 변경하는 다이얼로그입니다. 자동 등급 산정 기준과 관계없이 특별한 사유가 있을 때 등급을 조정합니다.

## 구성요소
- 제목: "등급 변경"
- 현재 등급 표시 (읽기 전용)
- 변경할 등급 선택 (등급 목록 — 예: 일반, 실버, 골드, VIP 등)
- 각 등급별 혜택 요약 안내 (선택 시 표시)
- 변경 사유 입력 (필수, 텍스트 입력 또는 사유 목록 선택)
- 변경 적용일 선택 (기본값: 즉시 적용)
- 확인 버튼
- 취소 버튼

## 언제 표시되나요
- 회원 상세 > 기본정보 탭의 등급 변경 버튼 클릭 시
- 관리자 권한을 가진 사용자에게만 표시

## 주요 흐름
1. 관리자가 등급 변경 버튼 클릭 → 다이얼로그 표시, 현재 등급 표시
2. 변경할 등급 선택 → 해당 등급의 혜택 안내 표시
3. 사유 및 적용일 입력
4. 확인 클릭 → 등급 변경 완료, 변경 이력 기록, 회원에게 등급 변경 알림 발송(선택)
5. 취소 클릭 → 다이얼로그 닫힘, 변경 없음
