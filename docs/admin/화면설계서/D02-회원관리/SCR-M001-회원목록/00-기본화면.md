## 화면 목적

센터에 등록된 전체 회원을 한눈에 조회하고, 다양한 조건으로 원하는 회원을 찾아 상세 페이지로 이동하거나 일괄 작업을 수행하는 회원관리 도메인의 핵심 허브 화면입니다. FC는 오늘 만료 예정 회원을 검색해 리텐션 상담을 진행하고, 매니저는 장기 미방문 회원에게 메시지를 발송하며, 센터장은 전체 회원 데이터를 엑셀로 내려받아 분석에 활용합니다.

## 주요 구성요소

- **요약 지표 카드 (4개)**: 전체 회원 수 / 활성 회원 수 / 정책상 조치 대상 회원 수 / 이번 달 만료 수를 한눈에 표시. 카드 클릭 시 해당 상태로 자동 필터링됩니다.
- **메인 탭**: 회원 전체 / 보유상품별 / 이용권 목록으로 보기 방식 전환.
- **상태 필터 탭**: 전체 · 활성 · 만료 · 미등록 · 홀딩 · 정지를 건수 배지와 함께 표시. 클릭 시 해당 상태의 회원만 표시.
- **검색 및 필터 바**: 이름·연락처 통합 검색, 계약상품·성별·최종만료일·최근방문일 필터. 추가 필터로 관심회원만 보기, 미방문 기간, 회원구분, 유입경로, 만료 숨기기를 제공.
- **일괄 작업 바**: 회원을 2명 이상 선택하면 화면 하단에 자동으로 노출. 상태 변경 / 메시지 전송 / 출석 처리 / 관심회원 등록 / 삭제 버튼 제공.
- **회원 목록 테이블**: 관심회원 별표 / 상태 배지 / 세그먼트 배지 / 회원명(상세 이동) / 성별 / 생년월일 / 연락처 / 이용권 / 만료일 / 등록일 컬럼 구성.
- **페이지 이동**: 한 페이지에 20 / 50 / 100건 선택 가능, 페이지 번호 이동.
- **엑셀 내보내기**: 현재 필터를 적용한 전체 데이터를 엑셀 파일로 내려받습니다.
- **회원 추가 버튼**: 신규 회원 등록 화면(SCR-M002)으로 이동.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 본사 최고관리자 | 전체 브랜드·모든 지점 회원 / 지점 전환 드롭다운 | 엑셀 내보내기, 회원 추가, 상태 변경, 삭제 등 모든 기능 |
| 센터장 (owner) | 소속 브랜드 지점 회원 전체 | 엑셀 내보내기, 회원 추가, 상태 변경, 삭제 |
| 매니저 | 소속 지점 회원 전체 | 엑셀 내보내기, 회원 추가, 상태 변경 (삭제 불가) |
| FC (피트니스 코치) | 소속 지점 전체 회원 (기본: 담당 회원 우선) | 메시지 전송, 출석 처리, 관심회원 등록 (등록·수정·삭제·엑셀 불가) |
| 트레이너 | 담당 회원만 | 조회만 (별도 캘린더 화면 우선 진입) |
| 스태프 | 소속 지점 회원 전체 | 회원 추가, 출석 처리, 관심회원 (삭제·엑셀 불가) |
| 프런트 | 소속 지점 회원 전체 | 조회 및 출석 처리만 |

## 주요 UX 흐름

1. 사이드바에서 "회원 관리 > 회원 목록"을 클릭하거나, 대시보드 지표 카드를 클릭해 진입합니다.
2. 요약 지표 카드로 전체 현황을 파악하고, 상태 탭 또는 검색·필터를 조합해 원하는 회원을 찾습니다.
3. 특정 회원명을 클릭하면 회원 상세(SCR-M004)로 이동합니다.
4. 여러 회원을 체크박스로 선택하면 하단에 일괄 작업 바가 나타나 상태 변경·메시지·출석 처리 등을 일괄 처리합니다.
5. 권한이 있는 역할은 우측 상단 "회원 추가" 버튼으로 신규 등록(SCR-M002)으로 이동합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 로딩 중 | 데이터를 불러오는 동안 테이블 자리에 스켈레톤(뼈대 화면)을 표시 |
| 정상 목록 | 200 응답 + 회원 1명 이상 존재 시 테이블 및 페이지네이션 표시 |
| 검색 결과 없음 | 필터/검색 적용 후 결과가 0건일 때 "검색 결과가 없습니다" 안내 |
| 전체 회원 없음 | 필터 없이 0건일 때 "등록된 회원이 없습니다" + 회원 추가 안내 |
| 오류 | 서버 또는 네트워크 오류 시 오류 메시지와 "다시 시도" 버튼 표시 |
| 접근 불가 | 권한 없는 계정 진입 시 접근 제한 안내 화면으로 전환 |
| 일괄 작업 진행 중 | 벌크 액션 처리 중 테이블 유지 + 진행 스피너 표시 |
