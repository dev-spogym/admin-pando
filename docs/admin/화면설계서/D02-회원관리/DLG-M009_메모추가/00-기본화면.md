## 목적
회원에 대한 상담 내용, 특이사항, 관리자 전달사항 등을 메모로 기록하는 다이얼로그입니다. 중요도를 표시하여 우선순위 높은 메모를 구분할 수 있습니다.

## 구성요소
- 제목: "메모 추가"
- 메모 내용 입력 (텍스트 영역, 최대 글자 수 표시)
- 중요도 선택 (일반 / 중요 토글 또는 라디오 버튼)
- 등록 버튼
- 취소 버튼

## 언제 표시되나요
- 회원 상세 > 메모 탭에서 메모 추가 버튼을 클릭했을 때

## 주요 흐름
1. 관리자가 메모 추가 버튼 클릭 → 다이얼로그 표시
2. 메모 내용 입력 및 중요도 선택
3. 등록 버튼 클릭 → 메모 저장 완료, 다이얼로그 닫힘, 메모 목록 즉시 갱신
4. 취소 → 다이얼로그 닫힘, 메모 저장 없음
