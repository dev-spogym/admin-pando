## 화면 목적

특정 회원을 현재 소속 지점에서 다른 지점으로 이관하는 화면입니다. 회원이 거주지 이동이나 개인 사정으로 이용 지점을 변경할 때, 기존 이용권과 이력 데이터를 그대로 유지한 채 새로운 지점으로 소속을 변경합니다. 이관 처리 후 회원은 새 지점의 관리 대상이 됩니다.

## 주요 구성요소

- **이관 대상 회원 정보**: 이관할 회원의 이름·연락처·현재 이용권·소속 지점을 요약 표시. 회원 상세에서 넘어온 경우 자동으로 채워집니다.
- **이관 대상 지점 선택**: 이관할 지점을 드롭다운으로 선택. 현재 소속 지점은 선택 불가.
- **이관 전 확인 정보**:
  - 이관 후에도 유지되는 항목: 이용권 잔여일·잔여 횟수·결제 이력·출석 이력·상담 메모
  - 이관 후 변경되는 항목: 소속 지점·담당 FC·담당 트레이너(새 지점 담당자로 재배정 필요)
- **이관 사유 입력**: 이관 이유를 간략하게 입력 (선택 항목, 내부 기록용)
- **확인 버튼 및 안내 문구**: "이관 후 현재 지점에서 해당 회원을 조회할 수 없습니다" 등 주의 사항 명시.
- **이관 확인 다이얼로그**: 이관 버튼 클릭 시 최종 확인 다이얼로그(DLG-M023) 표시 후 처리.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 본사 최고관리자 | 전체 지점 목록 선택 가능 | 이관 처리 가능 |
| 센터장 (owner) | 소속 브랜드 내 지점 목록 | 이관 처리 가능 |
| 매니저 | 접근 불가 | 이 기능 사용 불가 |
| FC / 트레이너 / 스태프 / 프런트 | 접근 불가 | 이 기능 사용 불가 |

## 주요 UX 흐름

1. 회원 상세(SCR-M004) 프로필 영역의 "지점이관" 버튼을 눌러 진입합니다.
2. 이관할 회원 정보가 자동으로 표시됩니다.
3. 이관 대상 지점을 드롭다운에서 선택합니다.
4. 이관 후 변경 사항을 확인하고, 선택적으로 이관 사유를 입력합니다.
5. "이관" 버튼을 클릭하면 최종 확인 다이얼로그가 나타납니다.
6. 확인하면 이관이 완료되고, 회원 목록(SCR-M001)으로 이동합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 초기 상태 | 이관 대상 회원 정보 표시, 지점 미선택 상태 |
| 지점 선택됨 | 대상 지점 선택 후 이관 버튼 활성화 |
| 처리 중 | 이관 처리 중 버튼 비활성 + 로딩 표시 |
| 이관 불가 상태 | 만료 회원이나 정지 회원 이관 시도 시 경고 안내 |
| 이관 완료 | 처리 성공 후 회원 목록으로 이동 |
