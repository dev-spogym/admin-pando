## 목적
회원이 이용권을 일시적으로 사용하지 못하는 상황(부상, 출장 등)에서, 관리자가 이용권 사용을 잠시 멈추는 홀딩을 등록하는 다이얼로그입니다.

## 구성요소
- 제목: "홀딩 등록"
- 홀딩 시작일 입력 (날짜 선택)
- 홀딩 종료일 입력 (날짜 선택)
- 홀딩 사유 입력 (텍스트 입력, 선택 또는 직접 입력)
- 홀딩 일수 자동 계산 표시
- 등록 버튼
- 취소 버튼

## 언제 표시되나요
- 회원 상세 > 이용권 탭에서 홀딩 등록 버튼을 클릭했을 때
- 현재 활성 이용권이 존재하는 회원에게만 표시

## 주요 흐름
1. 관리자가 홀딩 등록 버튼 클릭 → 다이얼로그 표시
2. 시작일·종료일 선택 → 홀딩 일수 자동 계산
3. 사유 입력 후 등록 버튼 클릭 → 홀딩 처리 완료, 이용권 만료일 자동 연장
4. 취소 → 다이얼로그 닫힘, 변경 없음
