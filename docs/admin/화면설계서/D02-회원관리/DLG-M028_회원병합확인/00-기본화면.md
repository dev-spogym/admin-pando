## 목적
중복으로 생성된 회원 계정을 하나로 합치기 전에 최종 확인을 받는 다이얼로그입니다. 병합 후 데이터 합산 결과를 미리보기로 제공하여 실수를 방지합니다.

## 구성요소
- 제목: "회원 병합 확인"
- 기준 회원 정보 표시 (병합 후 유지될 계정 — 회원명, 회원번호, 가입일)
- 대상 회원 정보 표시 (병합 후 삭제될 계정 — 회원명, 회원번호, 가입일)
- 병합 후 데이터 합산 미리보기
  - 이용권: 기준 + 대상 이용권 목록
  - 출석 이력: 통합 건수
  - 마일리지: 합산 잔액
  - 결제 이력: 통합 건수
- 주의 안내 메시지 ("병합 후 대상 계정은 삭제되며 복구할 수 없습니다")
- 확인(병합 처리) 버튼
- 취소 버튼

## 언제 표시되나요
- 회원 목록 또는 회원 상세에서 병합 대상 회원을 선택한 후 병합 버튼 클릭 시
- 관리자 권한을 가진 사용자에게만 표시

## 주요 흐름
1. 병합 대상 회원 선택 완료 → 병합 확인 다이얼로그 표시, 합산 미리보기 제공
2. 미리보기 내용 확인
3. 확인 클릭 → 대상 회원 데이터를 기준 회원에 통합, 대상 계정 비활성화
4. 취소 클릭 → 다이얼로그 닫힘, 병합 취소
