## 목적
회원과의 상담 내용을 체계적으로 기록하는 다이얼로그입니다. 상담 유형, 내용, 다음 상담 예정일 등을 입력하여 상담 이력을 관리합니다.

## 구성요소
- 제목: "상담 등록"
- 상담 일시 (자동 입력, 수정 가능)
- 상담 유형 선택 (전화, 방문, 문자, 기타)
- 상담 내용 입력 (텍스트 영역)
- 다음 상담 예정일 선택 (날짜 선택, 선택 사항)
- 담당자 표시 (현재 로그인 관리자 자동 입력)
- 등록 버튼
- 취소 버튼

## 언제 표시되나요
- 회원 상세 > 상담 탭에서 상담 등록 버튼을 클릭했을 때

## 주요 흐름
1. 관리자가 상담 등록 버튼 클릭 → 다이얼로그 표시
2. 상담 유형 선택 및 내용 입력
3. 필요 시 다음 상담 예정일 선택
4. 등록 버튼 클릭 → 상담 기록 저장 완료, 다이얼로그 닫힘, 목록 즉시 갱신
5. 취소 → 다이얼로그 닫힘, 저장 없음
