## 목적
트레이너가 회원의 체력 수준, 자세, 목표 달성도 등을 종합적으로 평가하여 기록하는 다이얼로그입니다. 정기적인 평가 기록을 통해 회원의 운동 진행 상황을 추적합니다.

## 구성요소
- 제목: "종합 평가 등록"
- 평가 날짜 선택 (기본값: 오늘)
- 체력 평가 항목 (근력, 유연성, 심폐지구력 등 각 항목별 점수 또는 등급 선택)
- 자세 평가 항목 (척추, 골반, 어깨 등 주요 부위별 평가)
- 목표 달성도 (목표 대비 현재 달성 수준, 퍼센트 또는 등급)
- 종합 의견 입력 (자유 텍스트)
- 다음 평가 예정일 설정 (선택)
- 저장 버튼
- 취소 버튼

## 언제 표시되나요
- 회원 상세 > 평가 탭에서 평가 등록 버튼 클릭 시
- 트레이너 또는 관리자 권한을 가진 사용자만 표시

## 주요 흐름
1. 트레이너가 평가 등록 버튼 클릭 → 다이얼로그 표시
2. 각 평가 항목 입력 및 종합 의견 작성
3. 저장 클릭 → 평가 기록 저장, 평가 탭에 이력 반영
4. 취소 클릭 → 다이얼로그 닫힘, 변경 없음
