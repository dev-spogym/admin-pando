## 목적
회원 등록 또는 수정 작업 중 취소 버튼을 클릭했을 때, 입력된 내용이 저장되지 않고 사라질 수 있음을 안내하여 실수로 인한 데이터 손실을 방지하는 다이얼로그입니다.

## 구성요소
- 제목: "작업을 취소하시겠습니까?"
- 안내 메시지: "입력하신 내용이 저장되지 않습니다."
- 확인 버튼 (작업 취소 실행, 이전 화면으로 이동)
- 계속 작업 버튼 (다이얼로그 닫기, 작업 유지)

## 언제 표시되나요
- 회원 등록 화면에서 취소 버튼을 클릭했을 때
- 회원 수정 화면에서 변경 사항이 있는 상태에서 취소 버튼을 클릭했을 때
- 변경 사항이 없는 경우 다이얼로그 없이 즉시 이전 화면으로 이동

## 주요 흐름
1. 관리자가 작업 중 취소 버튼 클릭 → 다이얼로그 표시
2. 확인 → 입력 내용 폐기, 이전 화면(목록 또는 상세)으로 이동
3. 계속 작업 → 다이얼로그 닫힘, 작업 화면 유지
