## 목적
같은 날짜에 체성분 기록이 이미 존재할 때, 기존 데이터를 새 데이터로 덮어쓸지 확인하는 다이얼로그입니다. 실수로 중복 기록이 생기는 것을 방지합니다.

## 구성요소
- 제목: "체성분 덮어쓰기"
- 안내 메시지: "[날짜]에 이미 등록된 체성분 기록이 있습니다. 덮어쓰시겠습니까?"
- 기존 기록 요약 표시 (측정일, 체중, 체지방률 등 주요 수치)
- 확인(덮어쓰기) 버튼
- 취소 버튼

## 언제 표시되나요
- 회원 상세 > 체성분 탭에서 체성분 등록 시
- 입력한 측정일에 이미 체성분 데이터가 존재하는 경우

## 주요 흐름
1. 체성분 등록 다이얼로그에서 날짜 입력 후 저장 클릭 → 중복 날짜 감지 → 덮어쓰기 다이얼로그 표시
2. 확인(덮어쓰기) 클릭 → 기존 기록 삭제 후 새 데이터 저장, 다이얼로그 닫힘
3. 취소 클릭 → 다이얼로그 닫힘, 체성분 등록 화면으로 복귀(입력값 유지)
