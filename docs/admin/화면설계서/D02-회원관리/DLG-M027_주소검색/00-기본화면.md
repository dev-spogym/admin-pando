## 목적
회원 등록 또는 정보 수정 시 우편번호 기반 주소를 검색하여 선택하는 팝업입니다. 정확한 주소 입력을 돕고 오타를 줄입니다.

## 구성요소
- 제목: "주소 검색"
- 검색어 입력 필드 (도로명, 지번, 건물명 등)
- 검색 버튼
- 검색 결과 목록 (우편번호, 도로명 주소, 지번 주소)
- 결과 없음 안내 메시지 (검색 결과가 없을 때)
- 상세 주소 입력 필드 (동·호수 등, 주소 선택 후 활성화)
- 확인 버튼
- 닫기 버튼

## 언제 표시되나요
- 회원 등록 화면의 주소 입력 영역에서 주소 검색 버튼 클릭 시
- 회원 정보 수정 화면의 주소 입력 영역에서 주소 검색 버튼 클릭 시

## 주요 흐름
1. 주소 검색 버튼 클릭 → 팝업 표시
2. 검색어 입력 후 검색 버튼 클릭 → 결과 목록 표시
3. 목록에서 주소 선택 → 상세 주소 입력 필드 활성화
4. 상세 주소 입력 후 확인 클릭 → 팝업 닫힘, 회원 등록·수정 화면의 주소 필드에 자동 입력
5. 닫기 버튼 클릭 → 팝업 닫힘, 주소 변경 없음
