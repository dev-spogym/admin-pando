## 목적
결제 목록에서 특정 결제 건의 상세 정보를 팝업으로 확인하는 다이얼로그입니다. 결제 수단, 이용권 내용, 할인 내역 등 전체 결제 정보를 한눈에 조회합니다.

## 구성요소
- 제목: "결제 상세"
- 결제 기본 정보: 결제일, 결제 번호, 결제 상태
- 이용권 정보: 이용권명, 기간, 횟수
- 금액 정보: 정가, 할인 금액, 실결제 금액
- 결제 수단: 카드, 현금, 계좌이체 등
- 환불 이력 (환불 건이 있는 경우)
- 닫기 버튼

## 언제 표시되나요
- 회원 상세 > 결제 탭에서 특정 결제 건의 상세 보기 버튼을 클릭했을 때
- 결제 목록의 행을 클릭했을 때 (설정에 따라)

## 주요 흐름
1. 관리자가 결제 상세 버튼 클릭 → 해당 결제 정보가 채워진 다이얼로그 표시
2. 내용 확인
3. 닫기 버튼 클릭 → 다이얼로그 닫힘
