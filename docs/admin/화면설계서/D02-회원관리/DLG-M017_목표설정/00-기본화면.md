## 목적
회원의 운동 목표(목표 체중, 체지방률, 골격근량 등)를 입력·수정하는 다이얼로그입니다. 목표 수치를 기록해 두면 이후 체성분 변화와 비교하여 달성 여부를 추적할 수 있습니다.

## 구성요소
- 제목: "목표 설정"
- 목표 체중 입력 (kg, 소수점 첫째 자리)
- 목표 체지방률 입력 (%, 소수점 첫째 자리)
- 목표 골격근량 입력 (kg, 소수점 첫째 자리, 선택)
- 목표 달성 기간 입력 (날짜 선택 또는 개월 수 입력, 선택)
- 메모 입력 (자유 텍스트, 선택)
- 저장 버튼
- 취소 버튼

## 언제 표시되나요
- 회원 상세 > 체성분 탭에서 목표 설정 버튼 클릭 시
- 기존 목표가 있는 경우 현재 값이 미리 채워진 상태로 표시

## 주요 흐름
1. 관리자 또는 트레이너가 목표 설정 버튼 클릭 → 다이얼로그 표시
2. 목표 수치 입력 후 저장 클릭 → 목표 저장, 체성분 탭에 목표 수치 반영
3. 취소 클릭 → 다이얼로그 닫힘, 변경 없음
