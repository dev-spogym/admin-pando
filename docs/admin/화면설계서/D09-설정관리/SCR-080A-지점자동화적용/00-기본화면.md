# SCR-080A 지점 자동화 적용 — 기본화면

> **퍼블리싱 현황**: 퍼블리싱 완료 / 데이터 미연동. 현재 UI는 목업·임시 데이터 기준이며 본사 정책 세트 적용 데이터 연동은 후속 단계입니다.

## 1. 화면 목적

- 지점이 본사 정책 세트를 선택 적용하는 화면이다.

## 2. 레이아웃

```text
PageHeader
PolicySummaryCards
ScopeTabs
PolicySelector
StepTable
ActionFooter
```

## 3. 핵심 기능

- 상품군별 정책 세트 선택
- 스텝별 활성/비활성
- 테스트 발송
- 적용 저장

## 4. 운영 규칙

- 지점은 새 정책 세트를 만들지 않는다.
- 지점은 본사가 허용한 스텝만 끌 수 있다.
- 지점은 정책 템플릿 본문을 수정하지 않는다.

## 5. 퍼블리싱 포인트

- 단순 토글 목록이 아니라 현재 적용 중인 정책 요약이 먼저 보여야 한다.
- 상품군, 스텝, 채널의 3단 구조가 읽혀야 한다.
