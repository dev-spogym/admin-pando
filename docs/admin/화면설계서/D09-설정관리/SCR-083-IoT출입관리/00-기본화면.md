## 화면 목적
센터에 설치된 IoT 출입 장치를 등록·관리하고 출입 방식을 설정하는 화면입니다.
밴드, QR코드, 얼굴 인식 등 다양한 출입 수단의 활성화 여부와 연결된 기기 상태를 한눈에 파악하고 제어합니다.

## 주요 구성요소

### 연결 기기 현황
- 등록된 IoT 기기 목록 (기기명, 설치 위치, 연결 상태, 마지막 통신 시각)
- 연결 상태 배지: 정상 / 오프라인 / 오류
- 기기별 재연결 시도 버튼
- 새 기기 등록 버튼

### 출입 방식 설정
- 출입 수단별 활성화 ON/OFF
  - 스마트 밴드 (RFID)
  - QR코드 스캔
  - 얼굴 인식
  - 핀번호 입력
- 중복 출입 방지 시간 설정 (동일 회원 재입장 허용 간격)
- 출입 이력 보관 기간 설정

### 출입 알림 설정
- 입장 알림 발송 여부 (회원 앱 푸시)
- 퇴장 알림 발송 여부 (회원 앱 푸시)
- 비정상 출입 시도 알림 발송 여부 (관리자 알림)

## 계정별 차이
| 계정 | 권한 |
|---|---|
| 슈퍼관리자 | 전체 기기 조회 및 설정 수정 가능 |
| 센터장 | 소속 지점 기기 조회 및 설정 수정 가능 |
| 매니저 | 기기 연결 상태 조회만 가능 |
| 기타 | 접근 불가 |

## 주요 UX 흐름
1. 화면 진입 시 등록된 기기 목록과 현재 연결 상태를 자동으로 불러온다.
2. 기기가 오프라인이면 오렌지 배지로 표시되며, 재연결 버튼을 클릭하면 연결 시도 상태로 전환된다.
3. 출입 수단 ON/OFF 토글 변경 후 저장하면 연결된 모든 기기에 설정이 반영된다.
4. 새 기기 등록 버튼 클릭 시 기기 등록 절차(기기 코드 입력 → 설치 위치 지정)가 진행된다.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 기기 목록 및 현재 설정값 표시 |
| 수정 중 | 설정 변경 시 저장 버튼 활성 |
| 기기 재연결 중 | 해당 기기 행에 처리 중 표시 |
| 기기 미등록 | 등록된 기기가 없을 때 안내 메시지 및 기기 등록 가이드 표시 |
| 저장 완료 | 완료 메시지 표시 후 자동 해제 |
