## 화면 목적
회원의 출석을 인정하는 기준과 자동 처리 규칙을 설정하는 화면입니다.
출석 인정 시간대, 지각 판단 기준, 자동 출석 처리 조건 등을 센터 운영 방침에 맞게 구성합니다.

## 주요 구성요소

### 출석 인정 시간 설정
- 출석 인정 시작 시각 (수업 시작 기준 N분 전부터 인정)
- 출석 인정 마감 시각 (수업 시작 기준 N분 후까지 인정)
- 주말/공휴일 출석 인정 여부

### 지각 기준 설정
- 지각 판단 기준 시간 (수업 시작 후 N분 이내 입장 → 지각)
- 지각 횟수 누적 시 알림 발송 기준 (N회 초과 시 자동 알림)

### 자동 출석 규칙
- 자동 출석 처리 활성화 여부 (IoT 출입 기록 기반 자동 출석 인정)
- 수업 없이 자유 이용 입장 시 출석 인정 여부
- 하루 중 동일 회원 출석 중복 처리 방식 (첫 입장만 / 매 입장마다)

### 출석 기록 보관 설정
- 출석 이력 보관 기간 (개월 단위)

## 계정별 차이
| 계정 | 권한 |
|---|---|
| 슈퍼관리자 | 전체 지점 설정 조회 및 수정 가능 |
| 센터장 | 소속 지점 설정 조회 및 수정 가능 |
| 매니저 | 조회만 가능 |
| 기타 | 접근 불가 |

## 주요 UX 흐름
1. 화면 진입 시 현재 저장된 출석 설정값을 불러와 표시한다.
2. 각 항목 변경 후 저장 버튼 클릭 → 변경 내용 저장 완료 메시지 표시.
3. 자동 출석 처리를 활성화하면 IoT 출입 설정(SCR-083)과 연동 안내 메시지가 표시된다.
4. 저장하지 않고 페이지 이탈 시 미저장 경고(DLG-080-001)가 표시된다.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 저장된 설정값 표시, 저장 버튼 비활성 |
| 수정 중 | 변경 발생 시 저장 버튼 활성 |
| 저장 완료 | 완료 메시지 표시 후 자동 해제 |
| IoT 미연결 | 자동 출석 활성화 시 IoT 기기 미연결 안내 표시 |
