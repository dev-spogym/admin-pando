## 화면 목적
센터가 현재 이용 중인 서비스 구독 플랜과 결제 수단을 확인하고 관리하는 화면입니다.
플랜 업그레이드·다운그레이드, 결제 수단 변경, 과거 청구 이력 조회를 처리합니다.

## 주요 구성요소

### 현재 구독 정보
- 현재 플랜명 및 주요 혜택 요약
- 다음 결제 예정일 및 금액
- 플랜 변경 버튼 (업그레이드 / 다운그레이드)
- 구독 해지 버튼

### 플랜 비교 섹션
- 이용 가능한 전체 플랜 목록 카드 형태 표시
- 각 플랜별 포함 기능 항목, 월 요금, 회원 수 한도
- 현재 플랜 강조 표시
- 타 플랜 선택 시 "이 플랜으로 변경" 버튼 활성

### 결제 수단 관리
- 현재 등록된 결제 수단 (카드사, 카드번호 마지막 4자리, 유효기간)
- 결제 수단 변경 버튼
- 결제 수단 삭제 버튼

### 청구 이력
- 과거 결제 내역 목록 (날짜, 플랜명, 금액, 상태)
- 영수증 다운로드 버튼 (PDF)
- 기간 필터 (최근 3개월 / 6개월 / 1년)

## 계정별 차이
| 계정 | 권한 |
|---|---|
| 슈퍼관리자 | 전체 구독 정보 조회 및 관리 가능 |
| 센터장 | 소속 지점 구독 정보 조회 및 관리 가능 |
| 매니저 | 접근 불가 |
| 기타 | 접근 불가 |

## 주요 UX 흐름
1. 화면 진입 시 현재 구독 정보와 청구 이력을 자동으로 불러온다.
2. 플랜 변경 버튼 클릭 → 플랜 변경 확인 다이얼로그(DLG-084-001) 표시 (변경 후 요금 차액 안내 포함).
3. 구독 해지 버튼 클릭 → 구독 해지 최종 확인 다이얼로그(DLG-084-002) 표시.
4. 결제 수단 변경 버튼 클릭 → 결제 수단 등록 페이지(PG사 연동)로 이동.
5. 영수증 다운로드 클릭 → 해당 월 PDF 영수증 자동 다운로드.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 현재 구독 정보 및 청구 이력 표시 |
| 로딩 | 구독 정보 불러오는 중 (스켈레톤 표시) |
| 플랜 없음 | 구독 이력이 없을 때 플랜 선택 안내 화면 표시 |
| 결제 수단 없음 | 등록된 결제 수단이 없을 때 등록 안내 표시 |
| 해지 완료 | 해지 완료 후 해지 확인 메시지 및 만료일 표시 |
