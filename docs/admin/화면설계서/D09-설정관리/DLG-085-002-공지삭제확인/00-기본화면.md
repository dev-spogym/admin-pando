## 화면 목적
공지를 삭제하기 전 최종 확인을 받는 다이얼로그입니다.
삭제된 공지는 복구할 수 없으므로 실수로 인한 삭제를 방지합니다.

## 주요 구성요소
- "공지를 삭제하시겠습니까?" 제목
- 삭제 대상 공지 제목 표시
- "삭제한 공지는 복구할 수 없습니다." 안내 문구
- 취소 버튼
- 삭제 확인 버튼

## 주요 UX 흐름
1. 공지 목록에서 삭제 버튼 클릭 → 다이얼로그 표시
2. 취소 → 다이얼로그 닫힘, 공지 유지
3. 삭제 확인 → 공지 삭제 처리 → 목록에서 제거

## 화면 상태
| 상태 | 설명 |
|---|---|
| 표시 | 삭제 대상 공지명 및 경고 문구 표시 |
| 처리 중 | 삭제 확인 버튼 클릭 후 처리 중 |
| 삭제 완료 | 다이얼로그 닫히고 목록 갱신 |
