## 화면 목적
권한 설정 화면에서 현재 변경 중인 내용을 마지막 저장 상태로 되돌리기 전 의도를 최종 확인하는 다이얼로그입니다.

## 주요 구성요소
- "권한을 초기화하시겠습니까?" 제목
- "마지막 저장 상태로 되돌아갑니다. 이 작업은 취소할 수 없습니다." 안내 메시지
- 취소 버튼
- 초기화 확인 버튼

## 주요 UX 흐름
1. 권한 설정 화면 상단의 초기화 버튼 클릭
2. 다이얼로그 표시
3. 취소 → 다이얼로그 닫힘, 현재 변경 내용 유지
4. 초기화 확인 → 마지막 저장 상태로 권한 매트릭스 복원, 다이얼로그 닫힘

## 화면 상태
| 상태 | 설명 |
|---|---|
| 표시 | 초기화 확인 대기 |
| 처리 완료 | 초기화 후 다이얼로그 닫히고 권한 매트릭스 갱신 |
