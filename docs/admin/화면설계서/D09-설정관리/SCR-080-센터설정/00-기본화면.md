## 화면 목적
센터 운영에 필요한 기본 정보를 한 곳에서 관리하는 설정 허브 화면입니다.
센터명·주소·연락처 등 기본정보, 알림 수신 방식, 화면 테마, 물품 재고를 탭별로 구분하여 관리합니다.

## 주요 구성요소

### 기본정보 탭
- 센터 로고 업로드 영역
- 센터명 (필수)
- 사업자등록번호
- 센터 소개
- 대표 연락처
- 주소 (주소 검색 버튼 포함)
- 업종 선택 (헬스, 필라테스, PT샵, 골프, 요가 등 11종 다중 선택)
- 평일 운영시간 (오픈~마감)
- 주말 운영시간 (오픈~마감)
- 저장 버튼 (변경 사항이 있을 때만 활성화)

### 알림설정 탭
- 앱 푸시 알림 항목별 ON/OFF 스위치 (입장 알림, 만료 임박, 결제 완료, 수업 예약·취소)
- 이메일 알림 항목별 ON/OFF 스위치 (주간 리포트, 만료 예정 안내)
- SMS 알림 항목별 ON/OFF 스위치 (결제 완료, 만료 임박)

### 테마설정 탭
- 화면 모드 선택 (라이트 / 다크 / 시스템 설정)
- 메인 컬러 색상 선택
- 보조 컬러 색상 선택
- 글자 크기 선택 (작게 / 기본 / 크게)

### 물품 관리 탭
- 물품별 재고 수량 입력 (수건 대·소, 운동복 상·하의)
- 1일 발급 한도 설정
- 현재 잔여 수량 입력

## 계정별 차이
| 계정 | 권한 |
|---|---|
| 슈퍼관리자 | 전체 탭 조회 및 수정 가능 |
| 센터장 | 전체 탭 조회 및 수정 가능 |
| 매니저 | 알림설정 탭만 조회 가능 (수정 불가) |
| 기타 | 접근 불가 |

## 주요 UX 흐름
1. 화면 진입 시 저장된 센터 설정 정보를 불러와 각 탭에 표시한다.
2. 정보 수정 후 저장 버튼 클릭 → 변경 사항 저장 완료 메시지 표시 (3초 후 자동 사라짐).
3. 저장하지 않고 다른 탭으로 이동 시 미저장 경고 다이얼로그(DLG-080-001) 표시.
4. 주소 검색 버튼 클릭 시 주소 검색 팝업 실행 후 선택한 주소가 자동 입력된다.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 저장된 설정값 표시, 저장 버튼 비활성 |
| 수정 중 | 변경 사항 발생 시 저장 버튼 활성, 상단에 "저장되지 않은 변경사항" 경고 배지 표시 |
| 저장 중 | 저장 버튼 처리 중 (버튼 비활성) |
| 저장 완료 | "변경사항이 저장되었습니다" 메시지 표시 (3초 후 자동 해제) |
| 탭 이동 경고 | 미저장 상태에서 탭 전환 시 경고 다이얼로그 표시 |
