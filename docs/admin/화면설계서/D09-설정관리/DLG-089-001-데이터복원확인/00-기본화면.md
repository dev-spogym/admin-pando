## 화면 목적
선택한 백업 시점으로 데이터를 복원하기 전 최종 확인을 받는 다이얼로그입니다.
복원은 현재 데이터를 덮어쓰는 되돌릴 수 없는 작업이므로 신중한 확인이 필요합니다.

## 주요 구성요소
- "데이터를 복원하시겠습니까?" 제목
- 복원 대상 백업 시점 표시 (날짜·시간)
- "복원 후 현재 데이터는 사라지며, 이 작업은 되돌릴 수 없습니다." 경고 문구
- 복원 완료 후 시스템 재시작 안내
- 복원 확인을 위한 텍스트 입력 확인란 (예: "복원" 입력 후 버튼 활성)
- 취소 버튼
- 복원 확인 버튼 (텍스트 입력 완료 시 활성)

## 주요 UX 흐름
1. 백업 이력에서 복원 버튼 클릭 → 다이얼로그 표시
2. 경고 내용 확인 후 확인 텍스트 입력
3. 취소 → 다이얼로그 닫힘
4. 복원 확인 → 복원 진행 중 표시 → 완료 후 시스템 재시작

## 화면 상태
| 상태 | 설명 |
|---|---|
| 표시 | 복원 대상 시점 및 경고 표시 |
| 텍스트 미입력 | 복원 확인 버튼 비활성 |
| 텍스트 입력 완료 | 복원 확인 버튼 활성 |
| 복원 진행 중 | 전체 화면 처리 중 표시 |
| 복원 완료 | 시스템 재시작 후 로그인 화면으로 이동 |
