# DLG-080A-001 정책 적용 확인 — 기본화면

## 1. 목적

- 지점 정책 적용 전 변경 내용을 최종 확인하는 다이얼로그다.

## 2. 표시 항목

- 적용 대상 상품군
- 선택 정책 세트명
- 비활성화한 스텝 목록
- 테스트 발송 여부

## 3. 퍼블리싱 포인트

- 일반 저장 확인이 아니라 운영 변경 요약에 집중해야 한다.
