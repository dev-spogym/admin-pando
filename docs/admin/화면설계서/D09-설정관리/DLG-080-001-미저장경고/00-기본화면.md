## 화면 목적
설정 화면에서 변경 사항을 저장하지 않은 상태로 탭을 이동하거나 페이지를 떠나려 할 때 표시되는 경고 다이얼로그입니다.
실수로 작업 내용을 잃지 않도록 사용자에게 선택 기회를 제공합니다.

## 주요 구성요소
- 경고 아이콘 및 "저장되지 않은 변경사항" 제목
- "탭을 이동하면 변경 사항이 사라집니다" 안내 메시지
- 취소 버튼 (현재 탭 유지)
- 저장 후 이동 버튼
- 저장하지 않고 이동 버튼

## 주요 UX 흐름
1. 사용자가 변경 사항이 있는 상태에서 다른 탭 클릭 또는 페이지 이탈 시도
2. 다이얼로그 자동 표시
3. 취소 → 현재 탭 유지, 다이얼로그 닫힘
4. 저장 후 이동 → 현재 탭 변경 사항 저장 → 대상 탭으로 이동
5. 저장하지 않고 이동 → 변경 사항 폐기 → 대상 탭으로 이동

## 화면 상태
| 상태 | 설명 |
|---|---|
| 표시 | 미저장 상태에서 이탈 시도 시 다이얼로그 열림 |
| 저장 후 이동 처리 중 | 저장 중 버튼 비활성 |
