## 화면 목적
관리자웹의 권한 템플릿별로 메뉴 접근 및 기능 사용 권한을 설정하는 화면입니다.
직원관리에서 등록하는 직무(트레이너·골프강사·프런트 등)와는 별도로, 기본 제공 권한 템플릿과 센터 맞춤형 역할을 추가하고 각 템플릿/역할이 어떤 메뉴에서 무엇을 할 수 있는지를 매트릭스 형태로 한눈에 확인하고 조정합니다.

## 주요 구성요소

### 권한 템플릿/역할 목록 패널 (좌측)
- 기본 제공 권한 템플릿 목록: 슈퍼관리자(`superAdmin`), 최고관리자(`primary`), 센터장(`owner`), 매니저(`manager`), FC(`fc`), 스태프(`staff`), 조회전용(`readonly`)
- `트레이너`, `골프강사`, `프런트`는 이 패널의 기본 권한 템플릿이 아니라 직원 직무값으로 관리합니다.
- 각 역할 카드에 해당 역할이 배정된 직원 수 표시
- 새 역할 추가 버튼 (+)
- 역할 카드별 복사·삭제 액션

### 권한 매트릭스 패널 (우측)
- 좌측에서 선택한 역할의 메뉴별 권한 표시
- 메뉴 그룹: 회원, 수업, 매출, 상품, 직원, 급여, 시설, 메시지, 설정 (총 9그룹)
- 권한 항목: 접근 / 조회 / 생성 / 수정 / 삭제 (5종)
- 전체 허용 / 전체 거부 버튼
- 권한 충돌 감지 경고 영역

### 상단 액션 버튼
- 초기화 버튼 (마지막 저장 상태로 되돌리기)
- 변경 사항 저장 버튼 (변경 사항이 있을 때만 활성)

## 계정별 차이
| 계정 | 권한 |
|---|---|
| 슈퍼관리자 | 전체 역할 조회 및 권한 수정 가능 |
| 센터장 | 소속 지점 역할 권한 조회 및 수정 가능 |
| 매니저 | 접근 불가 |
| 기타 | 접근 불가 |

## 주요 UX 흐름
1. 화면 진입 시 권한 템플릿/역할 목록이 좌측에 표시되고, 첫 번째 권한 템플릿(`superAdmin`)의 권한이 자동 선택되어 우측 매트릭스에 표시된다.
2. 좌측에서 역할 선택 → 해당 역할의 권한 매트릭스가 우측에 표시된다.
3. `superAdmin` 권한 템플릿은 모든 권한이 고정(수정 불가)으로 표시된다.
4. 권한 체크박스 변경 → 충돌 여부 실시간 감지 → 충돌 발생 시 경고 영역에 내용 표시.
5. 저장 버튼 클릭 → 충돌이 있으면 충돌 경고 다이얼로그(DLG-081-002) 표시 후 무시하고 저장 또는 취소 선택.
6. + 버튼 클릭 → 역할 생성 다이얼로그(DLG-081-003) 표시.
7. 역할 삭제 클릭 → 삭제 확인 다이얼로그(DLG-081-004) 표시.
8. 역할 복사 클릭 → 역할 복사 다이얼로그(DLG-081-005) 표시.
9. 초기화 버튼 클릭 → 초기화 확인 다이얼로그(DLG-081-001) 표시.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 역할 선택 후 권한 매트릭스 조회 |
| 수정 중 | 권한 변경 시 "저장되지 않은 변경 사항" 경고 배지 표시, 저장 버튼 활성 |
| `superAdmin` 선택 | 매트릭스 전체 체크박스 비활성(읽기 전용) |
| 충돌 감지 | 권한 충돌 경고 영역에 충돌 내용 목록 표시 |
| 저장 완료 | "저장되었습니다" 메시지 표시 (3초 후 자동 해제) |
