## 화면 목적
센터 입구에 설치된 키오스크 화면의 구성과 동작 방식을 설정하는 화면입니다.
회원이 키오스크를 통해 출석 체크를 하는 방식, 화면에 표시할 메시지와 배경, 사용 가능한 기능 항목을 관리합니다.

## 주요 구성요소

### 출석 방식 설정
- 출석 인식 방법 선택: QR코드 / 회원번호 입력 / 얼굴 인식 (복수 선택 가능)
- 기본 출석 방식 지정 (키오스크 첫 화면에 표시되는 방식)

### 화면 구성 설정
- 키오스크 표시 언어 선택
- 대기 화면 배경 이미지 또는 색상 설정
- 센터 로고 표시 여부
- 환영 메시지 텍스트 입력 (회원 이름 자동 삽입 태그 지원)
- 출석 완료 메시지 텍스트 입력

### 기능 표시 설정
- 키오스크에서 회원이 사용할 수 있는 기능 항목 ON/OFF
  - 이용권 잔여 조회
  - 수업 예약 조회
  - 락커 번호 확인
  - 공지사항 조회

### 미리보기
- 현재 설정값 기준 키오스크 화면 미리보기 표시

## 계정별 차이
| 계정 | 권한 |
|---|---|
| 슈퍼관리자 | 전체 설정 조회 및 수정 가능 |
| 센터장 | 소속 지점 키오스크 설정 조회 및 수정 가능 |
| 매니저 | 조회만 가능 |
| 기타 | 접근 불가 |

## 주요 UX 흐름
1. 화면 진입 시 현재 저장된 키오스크 설정값을 불러와 표시한다.
2. 항목 변경 시 우측 미리보기가 실시간으로 업데이트된다.
3. 저장 버튼 클릭 → 키오스크 기기에 설정이 즉시 반영된다.
4. 저장하지 않고 페이지 이탈 시 미저장 경고(DLG-080-001)가 표시된다.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 저장된 설정값 표시 |
| 수정 중 | 변경 사항 발생 시 저장 버튼 활성, 미리보기 실시간 반영 |
| 저장 중 | 저장 버튼 처리 중 (버튼 비활성) |
| 저장 완료 | 완료 메시지 표시 후 자동 해제 |
| 키오스크 미연결 | 연결된 키오스크 기기가 없을 때 안내 메시지 표시 |
