## 화면 목적
권한 설정 화면에서 새로운 역할을 생성하는 다이얼로그입니다.
역할 이름과 설명을 입력하고, 선택적으로 기존 역할의 권한을 복사하여 출발점으로 사용할 수 있습니다.

## 주요 구성요소
- "새 역할 만들기" 제목
- 역할 이름 입력란 (필수)
- 역할 코드 입력란 (시스템 내부 식별자, 필수)
- 역할 설명 입력란 (선택)
- 기존 역할에서 권한 복사 선택 드롭다운 (선택, 비워두면 빈 권한으로 시작)
- 취소 버튼
- 저장 버튼 (이름·코드 입력 시 활성)

## 주요 UX 흐름
1. 권한 설정 화면에서 + 버튼 클릭 → 다이얼로그 표시
2. 역할 이름·코드 입력 (기존 역할에서 복사 선택 시 해당 역할 권한 기반으로 시작)
3. 저장 → 새 역할이 좌측 역할 목록에 추가, 자동으로 선택된 상태로 전환
4. 취소 → 다이얼로그 닫힘

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 빈 폼 표시 |
| 입력 중 | 역할 이름·코드 입력 후 저장 버튼 활성 |
| 이름 중복 | 이미 사용 중인 역할 이름 입력 시 오류 메시지 표시 |
| 저장 완료 | 다이얼로그 닫히고 새 역할 목록에 추가 |
