## 화면 목적

회원별 체성분, 운동 이력, 건강 목표, 외부 건강 앱(Health Connect) 연동 상태를 하나의 화면에서 통합 요약한다. 관리자가 회원의 건강 상태를 빠르게 파악하고 상담 근거로 활용할 수 있다.

> **퍼블리싱 현황**: 퍼블리싱 완료 / 데이터 미연동. 현재 UI는 목업·임시 데이터 기준이며 Health Connect·회원 건강 데이터 연동은 후속 단계입니다.

## 주요 구성요소

### 요약 카드 영역

| 카드 | 설명 |
|------|------|
| 최근 출석 | 해당 회원의 마지막 체크인 시각 |
| 오늘 옷 락커 | 당일 배정된 락커 번호 또는 미배정 표시 |
| 고정 락커 | 보유 중인 고정 물품 락커 번호 또는 없음 |
| 최근 InBody | 체중·골격근량·체지방률 최신 수치 (확정 데이터 기준) |
| 최근 7일 활동량 | 걸음수 합계, 운동 일수 (Health Connect 연동 시) |

### 체성분 이력 섹션

| 항목 | 설명 |
|------|------|
| 측정일 | 체성분 측정 날짜 |
| 체중 변화 | 이전 측정 대비 증감 표시 |
| 골격근량 / 체지방률 | 핵심 지표 그래프 |
| 입력 채널 | 수기 / 파일 업로드 / 실시간 API |

### 운동 이력 섹션

| 항목 | 설명 |
|------|------|
| 최근 수업 이력 | 출석한 수업명, 날짜, 강사 |
| 운동 프로그램 | 현재 진행 중인 운동 프로그램 |
| 목표 달성률 | 월간 운동 목표 대비 달성 비율 |

### Health Connect 연동 상태 위젯

| 항목 | 설명 |
|------|------|
| 연동 상태 | 연결됨 / 동기화 대기 / 미연결 배지 |
| 최근 동기화 시각 | 마지막으로 데이터가 수신된 시각 |
| 활동량 비교 | 센터 출석 운동 vs 외부 활동량 비교 |
| 안내 문구 | Android 하이브리드 앱 경유 연동임을 명시 |

### 상담 메모 바로가기

회원의 건강·운동 관련 상담 메모를 빠르게 조회·작성할 수 있는 링크 또는 인라인 영역.

## 계정별 차이

| 계정 유형 | 차이 |
|-----------|------|
| 센터 관리자 | 전체 건강 요약 조회 및 상담 메모 작성 가능 |
| 담당 강사 | 담당 회원의 체성분·운동 이력 조회 가능 |
| 프런트 직원 | 최근 출석·락커 요약 카드만 조회 가능 |
| 회원 본인 | 회원 앱에서 자신의 건강 요약 조회 (별도 앱 화면) |

## 주요 UX 흐름

1. 회원 목록 또는 회원 상세에서 '건강 요약' 진입
2. 요약 카드 → 각 카드 클릭 시 상세 섹션으로 스크롤 이동
3. 체성분 이력 섹션 → 기간 필터로 조회 범위 변경
4. Health Connect 미연동 상태 → 회원 앱 연동 안내 문구 표시
5. 상담 메모 바로가기 → 해당 회원 상담 탭으로 이동

## 화면 상태

| 상태 | 설명 |
|------|------|
| 기본(데이터 있음) | 요약 카드 및 이력 섹션 정상 표시 |
| Health Connect 미연동 | 연동 위젯 '미연결' 배지, 앱 연동 안내 표시 |
| InBody 데이터 없음 | '측정 데이터가 없습니다.' 안내 |
| 운동 이력 없음 | '운동 이력이 없습니다.' 안내 |
| 로딩 | 데이터 조회 중 스켈레톤 표시 |
