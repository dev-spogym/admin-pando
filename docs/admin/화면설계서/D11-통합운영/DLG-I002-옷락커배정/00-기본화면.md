## 화면 목적

출석 완료 후 옷 락커가 미배정 상태인 회원에게 빈 락커를 선택하여 배정한다. 통합 출석 관리 또는 옷 락커 운영 관리 화면에서 호출되는 팝업 형태이다.

## 주요 구성요소

### 팝업 구조

모달 팝업 형태. 출석 관리(SCR-I001) 또는 옷 락커 운영 관리(SCR-I004)에서 오버레이로 표시된다.

### 회원 정보 표시 영역 (읽기 전용)

| 항목 | 설명 |
|------|------|
| 회원명 | 배정 대상 회원 이름 |
| 출석 시각 | 해당 회원의 체크인 시각 |
| 회원권 유형 | 일반 / VIP 등 |
| 고정 락커 | 보유 중인 고정 물품 락커 번호 (있는 경우) |

### 락커 선택 영역

빈 락커 목록을 번호 순으로 표시하며, 선택 가능한 락커를 클릭하여 배정한다.

| 상태 | 설명 |
|------|------|
| 선택 가능 | 클릭하여 배정 |
| 사용 중 | 선택 불가 (회색 처리) |
| 점검 중 | 선택 불가 (경고 표시) |

### 버튼

| 버튼 | 설명 |
|------|------|
| 배정 | 선택한 락커를 해당 회원에게 배정 |
| 취소 | 팝업 닫기, 배정하지 않음 |

## 계정별 차이

| 계정 유형 | 차이 |
|-----------|------|
| 센터 관리자 | 모든 회원에 대해 락커 배정 가능 |
| 프런트 직원 | 락커 배정 가능 |
| 강사 | 락커 배정 불가 |

## 주요 UX 흐름

1. 출석 목록(SCR-I001) 또는 락커 운영(SCR-I004)에서 '락커 배정' 버튼 클릭
2. 팝업 열림 → 대상 회원 정보 자동 표시
3. 빈 락커 목록에서 번호 클릭하여 선택
4. 배정 버튼 클릭 → 배정 완료
5. 성공 시 팝업 닫힘 + 출석 목록 및 락커 그리드 즉시 갱신
6. 빈 락커 없는 경우 → '배정 가능한 락커가 없습니다.' 안내

## 화면 상태

| 상태 | 설명 |
|------|------|
| 기본(락커 선택 대기) | 빈 락커 목록 표시, 배정 버튼 비활성 |
| 락커 선택됨 | 선택 락커 강조 표시, 배정 버튼 활성화 |
| 빈 락커 없음 | '배정 가능한 락커가 없습니다.' 안내 |
| 배정 성공 | 팝업 닫힘 + 토스트 '락커가 배정되었습니다.' |
| 배정 실패 | 인라인 오류 메시지 표시 |

## 관련 다이어그램

- SCR-I001: 통합 출석 관리 (진입점)
- SCR-I004: 옷 락커 운영 관리 (진입점 및 락커 상태 갱신)
