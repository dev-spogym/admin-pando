## 화면 목적

상품 계약 기반으로 회원에게 장기 배정하는 개인 소지품 보관 락커를 관리한다. 배정 현황 조회, 신규 배정, 만료 연장, 회수를 처리하며 계약 만료 예정 회원을 사전에 관리한다.

## 주요 구성요소

### 상단 현황 카드

| 항목 | 설명 |
|------|------|
| 전체 고정 락커 | 등록된 고정 물품 락커 총 수량 |
| 배정 중 | 현재 회원에게 배정된 수량 |
| 만료 예정 | 7일 이내 만료 예정 수량 |
| 회수 대기 | 계약 만료 후 미회수 수량 |

### 락커 유형 탭

| 탭 | 설명 |
|----|------|
| 개인 물품 | 일반 소형 개인 소지품 보관 락커 |
| 골프 | 골프백 보관용 대형 락커 |
| 프리미엄 | 계약 기반 프리미엄 락커 |

### 배정 현황 테이블

| 컬럼명 | 설명 |
|--------|------|
| 락커 번호 | 락커 식별 번호 |
| 회원명 | 배정된 회원 이름 |
| 상품명 | 연결된 락커 상품명 |
| 배정일 | 최초 배정 날짜 |
| 만료일 | 계약 만료 날짜 |
| 상태 | 사용 중 / 만료 예정 / 만료 / 회수 대기 |
| 관리 | 연장 / 회수 버튼 |

### 버튼

| 버튼 | 설명 |
|------|------|
| 배정하기 | 회원 + 락커 + 만료일 신규 배정 |
| 연장 | 선택 락커 계약 만료일 연장 |
| 회수 | 회원 연결 제거 (이력은 보존) |
| 상태 동기화 | 만료 상태 재계산 및 갱신 |
| 엑셀 다운로드 | 현재 배정 목록 엑셀 파일 다운로드 |

### 검색·필터

| 필터 | 옵션 |
|------|------|
| 상태 | 전체, 사용 중, 만료 예정, 만료, 회수 대기 |
| 만료 기간 | 이번 주, 이번 달, 직접 선택 |
| 검색 | 회원명, 락커 번호, 상품명 |

## 계정별 차이

| 계정 유형 | 차이 |
|-----------|------|
| 센터 관리자 | 배정, 연장, 회수, 상태 동기화 전체 가능 |
| 프런트 직원 | 배정 및 연장 가능, 회수는 관리자 승인 필요 |
| 강사 | 조회만 가능 |

## 주요 UX 흐름

1. 화면 진입 → 기본 탭(개인 물품) 배정 현황 표시
2. 만료 예정 카드 → 클릭 시 만료 예정 필터 자동 적용
3. 배정하기 → 회원 검색 → 락커 번호 선택 → 만료일 입력 → 저장
   - 락커 상품 미보유 회원은 배정 불가 안내 표시
4. 연장 버튼 → 새 만료일 입력 팝업 → 저장
5. 회수 버튼 → 확인 팝업 → 회수 처리 (이력 보존)
6. 상태 동기화 → 만료 여부 재계산 후 상태 배지 갱신

## 화면 상태

| 상태 | 설명 |
|------|------|
| 기본(배정 있음) | 배정 테이블 정상 표시 |
| 만료 예정 있음 | 만료 예정 행 강조 표시 |
| 빈 상태 | '배정된 락커가 없습니다.' 안내 |
| 락커 없음 | '등록된 고정 락커가 없습니다.' 안내 |
| 회수 대기 있음 | 회수 대기 행 경고 색상 강조 |
