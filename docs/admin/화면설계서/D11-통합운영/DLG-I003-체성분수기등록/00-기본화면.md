## 화면 목적

InBody 등 측정 장비 없이 관리자가 직접 체성분 수치를 수동으로 입력하여 등록한다. 측정지 원본을 보고 수기로 전산화하거나, 장비 연동이 불가한 환경에서 임시 등록 시 사용한다.

## 주요 구성요소

### 팝업 구조

모달 팝업 형태. 체성분 통합 관리(SCR-I006) 화면에서 오버레이로 표시된다.

### 입력 항목

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 회원 검색 | 필수 | 이름 또는 회원번호로 회원 선택 |
| 측정일시 | 필수 | 기본값: 현재 시각 (수정 가능) |
| 체중 (kg) | 필수 | 소수점 1자리 입력 |
| 골격근량 (kg) | 선택 | 소수점 1자리 입력 |
| 체지방량 (kg) | 선택 | 소수점 1자리 입력 |
| 체지방률 (%) | 선택 | 소수점 1자리 입력 |
| BMI | 선택 | 소수점 1자리 입력 |
| 기초대사량 (kcal) | 선택 | 정수 입력 |
| 체수분 (L) | 선택 | 소수점 1자리 입력 |
| 비고 | 선택 | 수기 등록 사유 또는 메모 |

### 버튼

| 버튼 | 설명 |
|------|------|
| 등록 | 입력값 유효성 검사 후 저장 |
| 취소 | 팝업 닫기, 입력값 초기화 |

## 계정별 차이

| 계정 유형 | 차이 |
|-----------|------|
| 센터 관리자 | 모든 회원 체성분 수기 등록 가능 |
| 프런트 직원 | 체성분 수기 등록 가능 |
| 강사 | 담당 회원에 한해 수기 등록 가능 |

## 주요 UX 흐름

1. 체성분 통합 관리(SCR-I006) → '수기 등록' 버튼 클릭 → 팝업 열림
2. 회원 검색 → 자동완성 목록에서 선택
3. 측정일시 확인 (필요 시 수정)
4. 체중 등 수치 입력 (체중은 필수, 나머지는 선택)
5. 등록 버튼 클릭 → 저장
6. 성공 시 팝업 닫힘 + 측정 결과 목록 즉시 갱신
7. 동일 회원·동일 측정 시각의 중복 데이터는 '검수 대기' 상태로 저장되며 안내 메시지 표시

## 화면 상태

| 상태 | 설명 |
|------|------|
| 기본(입력 대기) | 입력 항목 비어있음, 등록 버튼 비활성 |
| 회원 선택됨 | 회원 정보 표시, 나머지 항목 입력 활성화 |
| 필수 항목 입력됨 | 등록 버튼 활성화 |
| 중복 데이터 감지 | '동일 시각 데이터가 있습니다. 검수 대기로 저장됩니다.' 안내 |
| 등록 성공 | 팝업 닫힘 + 토스트 '체성분이 등록되었습니다.' |
| 등록 실패 | 인라인 오류 메시지 표시 |
