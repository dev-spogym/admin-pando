## 화면 목적

출석과 연동되는 당일용 옷 락커를 실시간으로 운영한다. 락커 그리드에서 사용 현황을 한눈에 파악하고, 출석 완료 후 미배정 상태인 회원에게 락커를 배정하거나 회수할 수 있다.

## 주요 구성요소

### 상단 현황 카드

| 항목 | 설명 |
|------|------|
| 전체 락커 수 | 지점 등록 옷 락커 총 수량 |
| 사용 중 | 현재 배정된 락커 수 |
| 미배정 대기 | 출석 완료 후 락커 미배정 인원 수 |
| 점검 중 | 고장·청소 처리된 락커 수 |

### 락커 그리드

락커를 번호 순서로 격자 형태로 표시하며, 상태별 색상 배지로 구분한다.

| 상태 | 설명 |
|------|------|
| 빈 락커 | 배정 가능 상태 |
| 사용 중 | 회원 배정 완료 |
| 만료 임박 | 당일 마감 시간 근접 |
| 반납 지연 | 퇴실 후 미반납 상태 |
| 점검 중 | 고장 또는 청소 처리 중 |

### 미배정 회원 패널

화면 우측 또는 하단 패널로 오늘 출석 후 락커 미배정 상태인 회원 목록을 표시한다.

| 항목 | 설명 |
|------|------|
| 회원명 | 미배정 대기 회원 이름 |
| 출석 시각 | 체크인 완료 시각 |
| 회원권 유형 | 일반 / VIP 등 |
| 배정 버튼 | 해당 회원에게 빈 락커 배정 |

### 버튼

| 버튼 | 설명 |
|------|------|
| 미배정 회원 보기 | 미배정 패널 열기/닫기 |
| 회원 배정 | 선택 회원과 선택 락커 배정 처리 |
| 회수 | 배정된 락커에서 회원 연결 제거 |
| 고장 처리 | 해당 락커를 점검 상태로 변경 |
| 복구 | 점검 상태 락커를 정상 복구 |
| 일괄 해제 | 만료·반납 지연 락커 일괄 회수 |

## 계정별 차이

| 계정 유형 | 차이 |
|-----------|------|
| 센터 관리자 | 전체 기능 사용 가능 (배정, 회수, 고장 처리, 일괄 해제) |
| 프런트 직원 | 배정 및 회수 가능, 일괄 해제는 관리자 승인 필요 |
| 강사 | 조회만 가능 |

## 주요 UX 흐름

1. 화면 진입 → 락커 그리드 + 상단 현황 카드 표시
2. 미배정 회원 있음 → 상단 카드 '미배정 대기' 수치 강조 → 패널 열기 유도
3. 미배정 회원 선택 → 빈 락커 선택 → 배정 버튼 클릭 → 배정 완료
4. 락커 클릭 → 사용 중이면 배정 회원 정보 팝업 표시
5. 사용 중 락커 → 회수 버튼 클릭 → 확인 후 회수 처리
6. 고장 락커 → 복구 버튼 클릭 → 정상 상태 복원
7. 일괄 해제 → 대상 락커 목록 확인 팝업 → 일괄 처리

## 화면 상태

| 상태 | 설명 |
|------|------|
| 기본(정상 운영) | 그리드 + 현황 카드 정상 표시 |
| 미배정 대기 있음 | 미배정 카드 강조, 패널 자동 열기 유도 |
| 락커 전체 사용 중 | '빈 락커 없음' 안내 표시 |
| 일괄 처리 중 | 처리 중 로딩 표시 |
| 빈 상태(락커 미등록) | '등록된 락커가 없습니다.' 안내 |
