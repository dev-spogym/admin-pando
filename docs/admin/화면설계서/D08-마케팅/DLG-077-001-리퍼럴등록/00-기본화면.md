## 목적

새로운 회원 추천(리퍼럴) 이벤트를 등록하는 다이얼로그입니다. 추천인과 피추천인에게 각각 제공할 혜택과 이벤트 기간을 설정합니다.

> **현재 상태**: 미구현 기능입니다.

## 구성요소

- **이벤트명** (필수): 리퍼럴 이벤트를 식별하는 이름을 입력합니다.
- **추천인 혜택**: 추천에 성공한 기존 회원에게 제공할 혜택(마일리지·쿠폰·무료 이용일 등)을 설정합니다.
- **피추천인 혜택**: 추천을 통해 등록한 신규 회원에게 제공할 혜택을 설정합니다.
- **혜택 지급 시점**: 피추천인 등록 완료 즉시 또는 첫 결제 완료 시점 중 선택합니다.
- **이벤트 기간**: 시작일과 종료일을 설정합니다.
- **저장 버튼**: 이벤트를 등록하고 다이얼로그를 닫습니다.
- **취소 버튼**: 등록하지 않고 닫습니다.

## 언제 표시되나요

- 리퍼럴 프로그램(SCR-077)에서 "이벤트 등록" 버튼을 클릭할 때 표시됩니다.

## 주요 흐름

1. 담당자가 SCR-077에서 "이벤트 등록" 버튼을 클릭합니다.
2. 이 다이얼로그가 열립니다.
3. 이벤트명, 추천인·피추천인 혜택, 기간을 입력합니다.
4. "저장"을 클릭하면 이벤트가 등록되고 회원 앱에 리퍼럴 코드가 발급됩니다.
