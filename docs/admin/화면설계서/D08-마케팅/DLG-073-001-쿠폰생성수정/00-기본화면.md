## 목적

새로운 할인 쿠폰을 생성하거나 기존 쿠폰 정보를 수정하는 다이얼로그입니다. 할인 유형과 금액, 유효 기간, 사용 조건 등 쿠폰의 핵심 항목을 설정합니다.

## 구성요소

- **쿠폰명** (필수): 쿠폰을 식별하는 이름을 입력합니다.
- **할인 유형** (필수): 정액 할인(원) 또는 정률 할인(%) 중 선택합니다.
- **할인 금액·율** (필수): 선택한 유형에 따라 할인 금액 또는 할인율을 입력합니다.
- **최소 구매 금액**: 쿠폰 사용 가능한 최소 결제 금액 조건을 설정합니다. (선택)
- **유효 기간**: 쿠폰 사용 가능 시작일과 종료일을 설정합니다.
- **발급 수량 제한**: 총 발급 가능 수량을 제한할 경우 숫자를 입력합니다. (선택)
- **1인 사용 제한**: 1인당 사용 가능 횟수를 제한합니다. (기본: 1회)
- **저장 버튼**: 쿠폰을 저장하고 다이얼로그를 닫습니다.
- **취소 버튼**: 저장하지 않고 닫습니다.

## 언제 표시되나요

- 쿠폰 관리(SCR-073)에서 "쿠폰 생성" 버튼을 클릭할 때 (빈 폼으로 열림).
- 쿠폰 목록의 수정 버튼을 클릭할 때 (기존 데이터가 채워진 상태로 열림).

## 주요 흐름

1. 담당자가 "쿠폰 생성" 또는 수정 버튼을 클릭합니다.
2. 이 다이얼로그가 열립니다.
3. 쿠폰명, 할인 유형, 금액·율, 유효 기간을 입력합니다.
4. "저장"을 클릭하면 쿠폰이 저장되고 목록에 표시됩니다.
5. 필수 항목이 비어있으면 오류 메시지가 표시되고 저장이 차단됩니다.
