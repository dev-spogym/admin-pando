## 목적

캠페인을 삭제하기 전 최종 확인을 받는 다이얼로그입니다. 진행 중인 캠페인을 삭제할 경우 발송이 중단된다는 사실을 안내합니다.

> **현재 상태**: 미구현 기능입니다.

## 구성요소

- **제목**: "캠페인 삭제"
- **안내 메시지**: "이 캠페인을 삭제하시겠습니까? 진행 중인 캠페인은 삭제 즉시 중단됩니다."
- **삭제 버튼 (위험)**: 캠페인을 최종 삭제합니다.
- **취소 버튼**: 다이얼로그를 닫고 삭제를 취소합니다.

## 언제 표시되나요

- 캠페인 관리(SCR-076)에서 캠페인의 삭제 버튼을 클릭할 때 표시됩니다.

## 주요 흐름

1. 담당자가 삭제할 캠페인의 삭제 버튼을 클릭합니다.
2. 이 다이얼로그가 표시됩니다.
3. "삭제"를 클릭하면 캠페인이 삭제되고 목록에서 제거됩니다.
4. "취소"를 클릭하면 다이얼로그가 닫힙니다.
