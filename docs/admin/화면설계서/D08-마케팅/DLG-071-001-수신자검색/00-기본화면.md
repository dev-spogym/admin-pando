## 목적

메시지 발송 시 수신자를 이름·연락처·이용권 상태 등 다양한 조건으로 검색해 발송 대상을 정밀하게 선택할 수 있는 팝업입니다.

## 구성요소

- **검색 입력 필드**: 이름 또는 연락처로 회원을 검색합니다.
- **조건 필터**: 이용권 상태(활성·만료·홀딩), 성별, 회원 등급 등으로 필터링합니다.
- **검색 결과 목록**: 이름 / 연락처 / 이용권 상태 / 마지막 방문일 컬럼으로 표시됩니다.
- **체크박스 선택**: 수신자를 개별 또는 전체 선택합니다.
- **선택 완료 버튼**: 선택한 회원을 수신자 목록에 추가하고 팝업을 닫습니다.
- **취소 버튼**: 선택 내용을 반영하지 않고 팝업을 닫습니다.

## 언제 표시되나요

- 메시지 발송 화면(SCR-071)에서 수신자 선택 영역의 "대상 검색" 버튼을 클릭할 때 표시됩니다.

## 주요 흐름

1. 담당자가 메시지 발송 화면에서 "대상 검색" 버튼을 클릭합니다.
2. 이 팝업이 열리며 전체 회원 목록이 기본 표시됩니다.
3. 이름·연락처 검색 또는 필터를 조합해 원하는 수신자를 찾습니다.
4. 체크박스로 선택한 뒤 "선택 완료"를 클릭합니다.
5. 선택된 회원이 메시지 발송 화면의 수신자 목록에 추가됩니다.
