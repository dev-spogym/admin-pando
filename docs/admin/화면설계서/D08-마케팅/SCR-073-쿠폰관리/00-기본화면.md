## 화면 목적

할인 쿠폰을 생성·관리하고 특정 회원 또는 그룹에게 발급하는 화면입니다. 쿠폰별 발급 현황과 사용률 통계를 통해 쿠폰 마케팅 효과를 파악하고 운영 전략을 조정합니다.

## 주요 구성요소

- **요약 지표 카드**: 전체 쿠폰 수 / 발급 건수 / 사용 건수 / 평균 사용률을 표시합니다.
- **쿠폰 목록 테이블**: 쿠폰명 / 할인 유형(정액·정률) / 할인 금액·율 / 유효 기간 / 발급 수 / 사용 수 / 사용률 / 상태(활성·만료·중단) 컬럼으로 구성됩니다.
- **쿠폰 생성 버튼**: DLG-073-001을 열어 새 쿠폰을 생성합니다.
- **발급 버튼**: 특정 쿠폰을 선택한 뒤 DLG-073-002를 통해 회원에게 발급합니다.
- **수정 버튼**: 쿠폰 정보를 수정합니다 (DLG-073-001 재사용).
- **삭제 버튼**: 쿠폰 삭제 전 DLG-073-003 확인 다이얼로그를 표시합니다.
- **상태 필터**: 활성 / 만료 / 중단 쿠폰을 선택해 필터링합니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 소속 센터 전체 쿠폰 목록 | 쿠폰 생성·수정·삭제, 회원 발급 |
| FC | 쿠폰 목록 조회 | 발급만 가능 (생성·삭제 불가) |
| 트레이너 / 스태프 | 접근 제한 | 쿠폰 관리 불가 |

## 주요 UX 흐름

1. 사이드바 "마케팅 > 쿠폰 관리"로 진입합니다.
2. 요약 카드로 전체 쿠폰 현황을 파악합니다.
3. "쿠폰 생성" 버튼을 클릭해 DLG-073-001에서 새 쿠폰을 만듭니다.
4. 생성된 쿠폰의 "발급" 버튼을 클릭해 DLG-073-002에서 특정 회원이나 그룹에게 발급합니다.
5. 불필요한 쿠폰은 삭제 버튼 클릭 후 DLG-073-003에서 최종 확인합니다.
6. 사용률 통계로 쿠폰 마케팅 효과를 분석합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 로딩 중 | 쿠폰 목록 로딩 중 로딩 표시가 나타납니다. |
| 정상 목록 | 쿠폰 목록이 테이블에 표시됩니다. |
| 쿠폰 없음 | "등록된 쿠폰이 없습니다" 안내와 함께 생성 버튼이 표시됩니다. |
| 만료 쿠폰 | 유효 기간이 지난 쿠폰은 "만료" 배지로 표시되고 발급 버튼이 비활성화됩니다. |

## 관련 다이어그램

- DLG-073-001 쿠폰 생성/수정
- DLG-073-002 쿠폰 발급
- DLG-073-003 쿠폰 삭제 확인
