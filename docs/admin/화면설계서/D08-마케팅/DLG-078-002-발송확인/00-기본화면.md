## 목적

대량 메시지 발송 전 수신자 수와 예상 비용을 최종 확인하는 다이얼로그입니다. 대량 발송은 비용이 발생하고 취소가 어렵기 때문에 명확한 확인 절차를 제공합니다.

> **현재 상태**: 미구현 기능입니다.

## 구성요소

- **발송 요약**: 채널(SMS·알림톡) / 수신자 수 / 메시지 내용 미리보기를 표시합니다.
- **예상 비용**: 수신자 수와 채널에 따른 총 예상 비용과 차감 후 잔여 포인트를 표시합니다.
- **발송 일시**: 즉시 발송 또는 예약 발송 일시를 표시합니다.
- **발송 확인 버튼**: 최종 발송을 실행합니다.
- **취소 버튼**: 발송을 취소하고 작성 화면으로 돌아갑니다.

## 언제 표시되나요

- SMS/카카오 발송 화면(SCR-078)에서 "발송" 버튼을 클릭할 때 표시됩니다.

## 주요 흐름

1. 담당자가 SCR-078에서 발송 설정을 완료하고 "발송" 버튼을 클릭합니다.
2. 이 다이얼로그가 표시되며 발송 요약과 예상 비용이 표시됩니다.
3. 잔여 포인트가 부족하면 경고 메시지가 표시되고 발송 버튼이 비활성화됩니다.
4. "발송 확인"을 클릭하면 발송이 실행됩니다.
5. "취소"를 클릭하면 작성 화면으로 돌아갑니다.
