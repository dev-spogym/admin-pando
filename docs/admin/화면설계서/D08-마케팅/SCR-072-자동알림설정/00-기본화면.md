## 화면 목적

본사 정책 세트로 내려온 만료 step과, 생일·장기 미방문 같은 지점 운영 이벤트 알림을 관리하는 화면입니다. 지점은 본사 기준 step을 따르면서 허용된 범위 안에서 운영용 알림 정책을 적용합니다.

## 주요 구성요소

- **고객 관련 알림 목록**: 본사 정책 세트 기준 만료 step / 생일 축하 / 장기 미방문 / 등록 감사 / 첫 방문 환영 등 고객 이벤트 기반 알림 규칙을 표시합니다.
- **상품 관련 알림 목록**: 이용권 만료 당일 / 홀딩 종료 / 결제 완료 / 환불 처리 등 상품 이벤트 기반 알림 규칙을 토글로 표시합니다.
- **규칙별 토글 스위치**: 각 알림 규칙의 활성화/비활성화를 토글 스위치로 제어합니다. 본사 `필수` step은 비활성화할 수 없습니다.
- **상세 설정 버튼**: 각 규칙 우측의 "상세 설정" 버튼 클릭 시 DLG-072-001이 열려 지점이 조정 가능한 채널·메시지·발송 옵션을 세부 설정합니다. 본사 소유 기준일 step은 읽기 전용으로 표시됩니다.
- **전체 자동 알림 ON/OFF**: 모든 자동 알림을 한 번에 활성화하거나 비활성화합니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 자동 알림 규칙 목록 | 규칙 활성화/비활성화, 상세 설정 편집 |
| FC / 트레이너 / 스태프 | 접근 제한 | 자동 알림 설정 불가 |

## 주요 UX 흐름

1. 사이드바 "마케팅 > 메시지 발송 > 자동 알림 설정" 또는 메시지 발송 화면(SCR-071)의 자동 알림 섹션 "상세 설정" 버튼으로 진입합니다.
2. 본사에서 내려온 step과 지점 운영 이벤트 목록을 확인하고, 허용된 규칙만 활성화합니다.
3. 규칙 우측 "상세 설정"을 클릭해 DLG-072-001에서 지점이 조정 가능한 채널과 메시지 내용을 설정합니다.
4. 저장 후 조건이 충족되면 자동으로 메시지가 발송됩니다.
5. 특정 규칙을 중단하려면 해당 토글을 끕니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 알림 활성 | 토글이 켜진 규칙은 활성 상태로 표시됩니다. |
| 알림 비활성 | 토글이 꺼진 규칙은 회색으로 표시됩니다. |
| 전체 비활성 | 전체 ON/OFF가 꺼지면 모든 규칙이 일시 중단됩니다. |

## 관련 다이어그램

- DLG-072-001 알림 규칙 편집
- SCR-071 메시지 발송
