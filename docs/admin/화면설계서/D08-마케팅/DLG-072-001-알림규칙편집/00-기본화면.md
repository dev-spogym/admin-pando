## 목적

자동 알림 설정(SCR-072)에서 특정 알림 규칙의 지점 적용 옵션, 발송 채널, 메시지 내용을 세부적으로 편집하는 다이얼로그입니다.

## 구성요소

- **규칙명**: 편집 중인 알림 규칙 이름을 상단에 표시합니다. (예: "본사 만료 step D-20 알림", "장기 미방문 알림")
- **발송 시점 설정**: 본사 소유 만료 step이면 기준일을 읽기 전용으로 표시합니다. 지점 자체 운영 이벤트 규칙이면 이벤트 기준 전·당일·후 숫자를 입력합니다.
- **발송 채널 선택**: 알림톡 / SMS / 앱 푸시 중 사용할 채널을 선택합니다.
- **발송 시각**: 자동 발송 시각을 시·분 단위로 설정합니다.
- **메시지 내용**: 자동 발송할 메시지 내용을 입력합니다. 회원명·만료일 등 변수를 삽입할 수 있습니다.
- **저장 버튼**: 설정을 저장하고 다이얼로그를 닫습니다.
- **취소 버튼**: 변경 내용을 저장하지 않고 닫습니다.

## 언제 표시되나요

- 자동 알림 설정 화면(SCR-072)에서 특정 알림 규칙의 "상세 설정" 버튼을 클릭할 때 표시됩니다.

## 주요 흐름

1. 관리자가 SCR-072에서 편집할 알림 규칙의 "상세 설정" 버튼을 클릭합니다.
2. 이 다이얼로그가 열리며 현재 설정값이 표시됩니다.
3. 발송 시점(읽기 전용 또는 편집 가능), 채널, 시각, 메시지 내용을 규칙 유형에 맞게 수정합니다.
4. "저장"을 클릭하면 변경 내용이 반영되고 다이얼로그가 닫힙니다.
5. "취소"를 클릭하면 이전 설정이 유지됩니다.
