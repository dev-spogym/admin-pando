## 화면 목적

특정 목표(신규 회원 유치, 재등록 유도, 브랜드 인지도 향상 등)를 위한 마케팅 캠페인을 생성하고 실적을 추적하는 화면입니다. 대상 회원 세그먼트를 지정하고 메시지·쿠폰·이벤트를 캠페인 단위로 묶어 통합 관리합니다.

> **현재 상태**: 미구현 화면입니다. 향후 개발 예정이며, 현재는 화면 접근이 제한됩니다.

## 주요 구성요소

- **캠페인 목록**: 캠페인명 / 목표 / 대상 세그먼트 / 기간 / 상태(준비·진행·종료) / 실적 요약 컬럼으로 구성됩니다.
- **캠페인 생성 버튼**: DLG-076-001을 열어 새 캠페인을 등록합니다.
- **실적 추적 패널**: 특정 캠페인의 도달 수 / 클릭 수 / 전환 수 / 비용 대비 효과(ROI)를 표시합니다.
- **캠페인 수정**: 진행 전 캠페인의 내용을 수정합니다.
- **캠페인 삭제**: DLG-076-002를 통해 삭제를 확인합니다.
- **상태 필터**: 준비 중 / 진행 중 / 종료 캠페인을 필터링합니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 캠페인 목록 및 실적 | 캠페인 생성·수정·삭제, 실적 조회 |
| FC / 트레이너 / 스태프 | 접근 제한 | 캠페인 관리 불가 |

## 주요 UX 흐름

1. 사이드바 "마케팅 > 캠페인 관리"로 진입합니다.
2. "캠페인 생성"(DLG-076-001)으로 목표·대상·기간·채널을 설정합니다.
3. 캠페인이 시작되면 메시지·쿠폰이 자동 또는 수동으로 발송됩니다.
4. 실적 추적 패널에서 효과를 모니터링합니다.
5. 종료 후 결과를 분석해 다음 캠페인 전략에 반영합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 미구현 | 현재 화면이 개발되지 않은 상태입니다. "준비 중" 안내 페이지가 표시됩니다. |
| 캠페인 없음 | 등록된 캠페인이 없을 때 생성 안내가 표시됩니다. |
| 진행 중 | 진행 중인 캠페인은 실적이 실시간으로 업데이트됩니다. |

## 관련 다이어그램

- DLG-076-002 캠페인 삭제 확인
