## 목적

리드를 삭제하기 전 최종 확인을 받아 실수로 인한 데이터 손실을 방지합니다. 삭제된 리드는 복구할 수 없으므로 명확한 의사 확인이 필요합니다.

## 구성요소

- **제목**: "리드 삭제"
- **안내 메시지**: "이 리드를 삭제하시겠습니까? 삭제된 데이터는 복구할 수 없습니다."
- **삭제 버튼 (위험)**: 리드를 최종 삭제합니다. 빨간색 계열로 표시됩니다.
- **취소 버튼**: 다이얼로그를 닫고 삭제를 취소합니다.

## 언제 표시되나요

- 리드 관리(SCR-070) 테이블에서 삭제(휴지통) 아이콘을 클릭할 때 표시됩니다.

## 주요 흐름

1. 담당자가 삭제할 리드의 휴지통 아이콘을 클릭합니다.
2. 이 다이얼로그가 표시됩니다.
3. "삭제"를 클릭하면 해당 리드가 영구 삭제되고 목록이 갱신됩니다.
4. "취소"를 클릭하면 다이얼로그가 닫히고 리드가 유지됩니다.
