## 목적

리퍼럴 이벤트를 삭제하기 전 최종 확인을 받는 다이얼로그입니다. 진행 중인 이벤트 삭제 시 기존 추천 코드가 무효화된다는 사실을 안내합니다.

> **현재 상태**: 미구현 기능입니다.

## 구성요소

- **제목**: "리퍼럴 이벤트 삭제"
- **안내 메시지**: "이 이벤트를 삭제하시겠습니까? 삭제 시 발급된 추천 코드가 모두 무효화됩니다."
- **삭제 버튼 (위험)**: 이벤트를 최종 삭제합니다.
- **취소 버튼**: 다이얼로그를 닫고 삭제를 취소합니다.

## 언제 표시되나요

- 리퍼럴 프로그램(SCR-077)에서 이벤트의 삭제 버튼을 클릭할 때 표시됩니다.

## 주요 흐름

1. 담당자가 삭제할 이벤트의 삭제 버튼을 클릭합니다.
2. 이 다이얼로그가 표시됩니다.
3. "삭제"를 클릭하면 이벤트가 삭제되고 발급된 추천 코드가 무효화됩니다.
4. "취소"를 클릭하면 다이얼로그가 닫힙니다.
