## 화면 목적

회원별 마일리지 잔액과 적립·차감 이력을 조회하고, 적립 규칙을 설정하는 화면입니다. 마일리지는 회원의 재방문을 유도하고 충성도를 높이는 리텐션 도구로 활용됩니다.

## 주요 구성요소

- **요약 지표 카드**: 총 발행 마일리지 / 총 사용 마일리지 / 소멸 예정 마일리지 / 현재 유통 잔액을 표시합니다.
- **회원별 마일리지 테이블**: 회원명 / 연락처 / 현재 잔액 / 총 적립 / 총 사용 / 최근 적립일 컬럼으로 구성됩니다.
- **이력 조회**: 특정 회원을 선택하면 날짜 / 구분(적립·사용·차감·소멸) / 금액 / 사유 / 처리자 컬럼의 상세 이력이 표시됩니다.
- **수동 적립·차감 버튼**: DLG-074-002를 통해 관리자가 직접 특정 회원의 마일리지를 조정합니다.
- **적립 규칙 설정**: DLG-074-001을 통해 결제 금액 대비 적립률, 특별 이벤트 적립 조건 등을 설정합니다.
- **검색 바**: 회원 이름 또는 연락처로 특정 회원을 검색합니다.
- **엑셀 내보내기**: 마일리지 이력 전체를 엑셀로 내려받습니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 회원 마일리지 현황 | 수동 적립·차감, 적립 규칙 설정, 엑셀 내보내기 |
| FC | 담당 회원 마일리지 조회 | 이력 조회만 (수동 조정 불가) |
| 회원 본인 | 본인 마일리지 잔액 및 이력 | 조회만 가능 (앱에서 확인) |
| 트레이너 / 스태프 | 접근 제한 | 마일리지 관리 불가 |

## 주요 UX 흐름

1. 사이드바 "마케팅 > 마일리지 관리"로 진입합니다.
2. 요약 카드로 전체 마일리지 현황을 파악합니다.
3. 특정 회원을 검색하거나 테이블에서 선택해 개인별 이력을 확인합니다.
4. 조정이 필요한 경우 "수동 적립·차감"(DLG-074-002)으로 마일리지를 조정하고 사유를 기록합니다.
5. "적립 규칙 설정"(DLG-074-001)에서 자동 적립 기준을 변경합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 로딩 중 | 마일리지 데이터 로딩 중 로딩 표시가 나타납니다. |
| 정상 목록 | 회원별 마일리지 테이블이 표시됩니다. |
| 회원 없음 | "마일리지 데이터가 없습니다" 안내가 표시됩니다. |
| 이력 조회 중 | 특정 회원 선택 시 이력 패널이 로딩됩니다. |

- DLG-074-002 수동 적립·차감
