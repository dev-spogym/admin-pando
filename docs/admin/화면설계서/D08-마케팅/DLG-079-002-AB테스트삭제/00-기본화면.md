## 목적

A/B 테스트를 삭제하기 전 최종 확인을 받는 다이얼로그입니다. 진행 중인 테스트를 삭제하면 수집된 데이터와 테스트가 중단된다는 사실을 안내합니다.

> **현재 상태**: 미구현 기능입니다.

## 구성요소

- **제목**: "A/B 테스트 삭제"
- **안내 메시지**: "이 A/B 테스트를 삭제하시겠습니까? 진행 중인 테스트는 즉시 중단되며, 수집된 데이터도 삭제됩니다."
- **삭제 버튼 (위험)**: 테스트를 최종 삭제합니다.
- **취소 버튼**: 다이얼로그를 닫고 삭제를 취소합니다.

## 언제 표시되나요

- A/B 테스트 화면(SCR-079)에서 테스트의 삭제 버튼을 클릭할 때 표시됩니다.

## 주요 흐름

1. 담당자가 삭제할 A/B 테스트의 삭제 버튼을 클릭합니다.
2. 이 다이얼로그가 표시됩니다.
3. "삭제"를 클릭하면 테스트가 삭제되고 목록에서 제거됩니다.
4. "취소"를 클릭하면 다이얼로그가 닫힙니다.
