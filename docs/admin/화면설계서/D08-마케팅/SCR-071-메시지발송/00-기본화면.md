## 화면 목적

회원에게 SMS, 카카오 알림톡, 앱 푸시 메시지를 직접 작성해 발송하고, 발송 이력을 관리하는 화면입니다. 수신자를 조건별로 선택하고 예약 발송도 설정할 수 있어 회원 소통의 핵심 창구 역할을 합니다.

## 주요 구성요소

- **탭 전환**: "메시지 전송" 탭과 "발송 이력" 탭으로 구성됩니다.
- **보유 포인트 표시**: 메시지 발송에 사용되는 잔여 포인트를 상단에 표시합니다.
- **발신 설정**: 발신 번호 선택 및 카카오 알림톡 활성화 여부를 설정합니다.
- **수신자 선택**: 전체 회원 / 활성 회원 / 만료 회원 등 조건별 그룹 선택 또는 개별 검색으로 수신자를 지정합니다. DLG-071-001(수신자 검색)을 통해 상세 검색이 가능합니다.
- **발송 채널 선택**: 알림톡 / SMS·LMS / 앱 푸시 중 선택합니다.
- **예상 비용 안내**: 선택한 수신자 수와 채널에 따른 예상 발송 비용이 실시간으로 표시됩니다.
- **메시지 내용 작성**: 자유 텍스트로 메시지를 작성하며, 회원명·만료일 등 변수를 삽입할 수 있습니다.
- **발송 미리보기**: DLG-071-002를 통해 실제 수신자 화면 형태로 메시지를 미리 확인합니다.
- **발송 / 예약 발송 버튼**: 즉시 발송 또는 원하는 일시에 예약 발송합니다.
- **발송 이력 테이블**: 발송 일시 / 채널 / 수신자 수 / 내용 요약 / 발송 결과를 조회합니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 발송 기능 및 발송 이력 | 메시지 작성·발송, 예약 발송, 발신 설정 변경 |
| FC | 메시지 전송 탭 | 담당 회원 대상 메시지 발송 (발신 설정 변경 불가) |
| 트레이너 / 스태프 | 접근 제한 | 마케팅 메시지 발송 불가 |

## 주요 UX 흐름

1. 사이드바 "마케팅 > 메시지 발송"으로 진입합니다.
2. 수신자를 조건 그룹 선택 또는 개별 검색(DLG-071-001)으로 지정합니다.
3. 발송 채널을 선택하면 예상 비용이 표시됩니다.
4. 메시지 내용을 작성하고 변수를 삽입합니다.
5. "미리보기"(DLG-071-002)로 최종 내용을 확인합니다.
6. "발송" 또는 "예약 발송"을 클릭해 전송합니다.
7. "발송 이력" 탭에서 과거 발송 결과를 조회합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 수신자 미선택 | 발송 버튼이 비활성화됩니다. |
| 메시지 미입력 | 발송 버튼이 비활성화됩니다. |
| 포인트 부족 | 예상 비용이 잔여 포인트를 초과하면 경고 메시지와 함께 발송 버튼이 비활성화됩니다. |
| 발송 중 | 발송 버튼이 로딩 상태로 변경됩니다. |
| 발송 완료 | 성공 알림 후 발송 이력 탭으로 자동 이동됩니다. |
| 이력 없음 | 발송 이력 탭에 "발송 이력이 없습니다" 안내가 표시됩니다. |
