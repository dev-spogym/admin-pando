## 목적

신규 잠재 고객(리드)을 등록하거나 기존 리드 정보를 수정하는 다이얼로그입니다. 문의 경로·상담 단계·담당 FC를 함께 기록해 체계적인 상담 파이프라인을 구성합니다.

## 구성요소

- **이름** (필수): 잠재 고객 이름을 입력합니다.
- **연락처**: 전화번호를 입력합니다.
- **문의 유형** (필수): 방문(WI), 전화문의(TI), 체험, 일일입장 중 선택합니다. 기본값은 "전화문의(TI)"입니다.
- **유입경로 / 상세경로**: 간판·인터넷·홍보물·회원소개·기타 중 1차 상세 경로를 선택하고, 필요한 경우 블로그·인스타·당근·지역카페·현수막·전단지·소개자명 등 2차 상세 경로를 입력합니다.
- **상담 방식**: 1차·2차·3차 상담 방식은 대면·유선·부재 중 선택합니다. 전화문의(TI) 리드에서 한 번이라도 대면이 기록되면 TI 방문 전환으로 집계됩니다.
- **상태**: 신규·연락완료·상담예정·방문완료·등록완료·미전환·보류 중 선택합니다. 기본값은 "신규"입니다.
- **담당 FC**: 담당자 이름을 직접 입력합니다.
- **문의일**: 날짜 선택기로 입력합니다. 기본값은 오늘입니다.
- **후속 예정일**: 다음 상담 또는 연락 예정일을 입력합니다.
- **메모**: 문의 내용, 특이사항 등을 자유롭게 입력합니다.
- **저장 버튼**: 입력 내용을 저장하고 다이얼로그를 닫습니다.
- **취소 버튼**: 저장하지 않고 다이얼로그를 닫습니다.

## 언제 표시되나요

- 리드 관리(SCR-070)에서 "+ 리드 등록" 버튼을 클릭할 때 (빈 폼으로 열림).
- 리드 목록 테이블의 수정(편집) 아이콘을 클릭할 때 (기존 데이터가 채워진 상태로 열림).

## 주요 흐름

1. "+ 리드 등록" 또는 수정 아이콘을 클릭하면 다이얼로그가 열립니다.
2. 이름(필수), 문의 유형(필수), 유입경로/상세경로를 포함한 정보를 입력합니다.
3. "저장"을 클릭하면 데이터가 저장되고 리드 목록이 갱신됩니다.
4. 이름이 비어있으면 "이름을 입력하세요" 오류 메시지가 표시되고 저장이 차단됩니다.
5. "취소"를 클릭하면 변경 내용 없이 다이얼로그가 닫힙니다.

## 데이터 매핑

| 입력 항목 | 외부 템플릿 | 매핑 기준 |
|---|---|---|
| 상태 / 결과 | `[신규DB]` A열 | 등록·관리·실패 |
| 문의 유형 | `[신규DB]` C열 `경로` | `WI=방문`, `TI=전화문의`, `체험`, `일일입장` |
| 상세 경로 | `[신규DB]` D/E열 | D열 `상세 경로1`, E열 `상세 경로2` |
| 상담 방식 | `[신규DB]` N/R/V열 | 1차·2차·3차 상담 방식. 값은 `대면`, `유선`, `부재` |
