## 목적

생성된 쿠폰을 특정 회원 또는 회원 그룹에게 직접 발급하는 다이얼로그입니다. 개별 발급과 그룹 일괄 발급을 모두 지원합니다.

## 구성요소

- **발급 쿠폰 정보**: 발급할 쿠폰의 이름·할인 내용·유효 기간을 상단에 요약합니다.
- **발급 대상 선택 방식**: 개별 회원 검색 또는 조건 그룹(활성 회원 전체, 만료 회원 등) 중 선택합니다.
- **회원 검색 필드**: 이름 또는 연락처로 특정 회원을 검색하고 선택합니다.
- **선택 회원 목록**: 발급 대상으로 추가된 회원 목록을 표시하며, 개별 제거가 가능합니다.
- **발급 수량 확인**: 현재 선택된 발급 대상 수와 잔여 발급 가능 수량을 표시합니다.
- **발급 버튼**: 선택된 대상에게 쿠폰을 발급합니다.
- **취소 버튼**: 발급하지 않고 다이얼로그를 닫습니다.

## 언제 표시되나요

- 쿠폰 관리(SCR-073)에서 특정 쿠폰의 "발급" 버튼을 클릭할 때 표시됩니다.

## 주요 흐름

1. 담당자가 SCR-073에서 발급할 쿠폰의 "발급" 버튼을 클릭합니다.
2. 이 다이얼로그가 열리며 해당 쿠폰 정보가 표시됩니다.
3. 발급 대상을 개별 검색하거나 그룹으로 선택합니다.
4. 발급 수량과 잔여 한도를 확인합니다.
5. "발급"을 클릭하면 선택한 회원에게 쿠폰이 지급됩니다.
6. 발급 후 해당 회원은 앱에서 쿠폰을 확인하고 사용할 수 있습니다.
