## 목적

마일리지 자동 적립 기준을 설정하는 다이얼로그입니다. 결제 금액 대비 적립률, 특별 이벤트 적립 조건, 소멸 정책 등을 관리자가 직접 설정합니다.

## 구성요소

- **기본 적립률**: 결제 금액 100원당 적립되는 마일리지 포인트를 입력합니다. (예: 100원당 1포인트)
- **최소 적립 결제 금액**: 마일리지 적립이 발생하는 최소 결제 금액을 설정합니다.
- **특별 적립 조건**: 생일·첫 방문·목표 달성 시 추가 적립 포인트를 설정합니다.
- **마일리지 소멸 정책**: 마일리지 유효 기간(개월 수)을 설정합니다. 무제한 선택도 가능합니다.
- **저장 버튼**: 설정을 저장하고 다이얼로그를 닫습니다.
- **취소 버튼**: 변경 내용을 저장하지 않고 닫습니다.

## 언제 표시되나요

- 마일리지 관리(SCR-074)에서 "적립 규칙 설정" 버튼을 클릭할 때 표시됩니다.

## 주요 흐름

1. 관리자가 SCR-074에서 "적립 규칙 설정" 버튼을 클릭합니다.
2. 이 다이얼로그가 열리며 현재 적립 규칙이 표시됩니다.
3. 기본 적립률과 특별 조건을 수정합니다.
4. "저장"을 클릭하면 변경된 규칙이 즉시 적용됩니다.
5. 변경 이후 발생하는 결제부터 새 규칙이 적용됩니다.
