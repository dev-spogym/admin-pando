## 목적

관리자가 특정 회원의 마일리지를 직접 적립하거나 차감하는 다이얼로그입니다. 시스템 오류 보정, 이벤트 지급, 환불 차감 등 예외적인 마일리지 조정 상황에 활용합니다.

## 구성요소

- **대상 회원 정보**: 조정할 회원의 이름·현재 마일리지 잔액을 상단에 표시합니다.
- **조정 유형 선택**: 적립 또는 차감 중 선택합니다.
- **조정 금액**: 적립하거나 차감할 마일리지 포인트를 숫자로 입력합니다.
- **조정 후 잔액 미리보기**: 입력한 금액에 따라 조정 후 예상 잔액이 실시간으로 표시됩니다.
- **사유 입력** (필수): 조정 사유를 입력합니다. 이력에 기록되어 향후 확인할 수 있습니다.
- **확인 버튼**: 입력한 내용으로 마일리지를 조정합니다.
- **취소 버튼**: 조정하지 않고 다이얼로그를 닫습니다.

## 언제 표시되나요

- 마일리지 관리(SCR-074)에서 특정 회원의 "수동 적립·차감" 버튼을 클릭할 때 표시됩니다.

## 주요 흐름

1. 관리자가 SCR-074에서 조정할 회원의 "수동 적립·차감" 버튼을 클릭합니다.
2. 이 다이얼로그가 열리며 해당 회원의 현재 마일리지 잔액이 표시됩니다.
3. 조정 유형(적립/차감), 금액, 사유를 입력합니다.
4. 조정 후 잔액 미리보기로 결과를 확인합니다.
5. "확인"을 클릭하면 마일리지가 조정되고 이력에 기록됩니다.
6. 차감 시 잔액보다 큰 금액을 입력하면 경고 메시지가 표시됩니다.
