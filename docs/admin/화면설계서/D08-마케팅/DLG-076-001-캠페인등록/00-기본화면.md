## 목적

새로운 마케팅 캠페인을 등록하는 다이얼로그입니다. 캠페인 목표, 대상 세그먼트, 기간, 사용할 채널과 콘텐츠를 설정합니다.

> **현재 상태**: 미구현 기능입니다.

## 구성요소

- **캠페인명** (필수): 캠페인을 식별하는 이름을 입력합니다.
- **캠페인 목표**: 신규 유치 / 재등록 유도 / 브랜드 인지도 중 목표를 선택합니다.
- **대상 세그먼트**: 발송 대상 회원 그룹(활성·만료·특정 등급 등)을 선택합니다.
- **캠페인 기간**: 시작일과 종료일을 설정합니다.
- **사용 채널**: 메시지·쿠폰·이벤트 중 활용할 채널을 선택합니다.
- **저장 버튼**: 캠페인을 등록하고 다이얼로그를 닫습니다.
- **취소 버튼**: 등록하지 않고 닫습니다.

## 언제 표시되나요

- 캠페인 관리(SCR-076)에서 "캠페인 생성" 버튼을 클릭할 때 표시됩니다.

## 주요 흐름

1. 담당자가 SCR-076에서 "캠페인 생성" 버튼을 클릭합니다.
2. 이 다이얼로그가 열립니다.
3. 캠페인명, 목표, 대상, 기간을 입력합니다.
4. "저장"을 클릭하면 캠페인이 등록됩니다.
