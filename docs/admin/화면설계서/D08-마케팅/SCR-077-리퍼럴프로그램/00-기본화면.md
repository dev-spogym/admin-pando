## 화면 목적

기존 회원이 지인을 추천해 새 회원을 유치하면 추천인과 피추천인 모두에게 혜택을 제공하는 리퍼럴(회원 추천) 프로그램을 설정하고 실적을 관리하는 화면입니다. 구전 마케팅을 체계화해 신규 회원 유치 비용을 절감합니다.

> **퍼블리싱 현황**: 퍼블리싱 완료 / 데이터 미연동. 현재 UI는 목업·임시 데이터 기준이며 실데이터 연동은 후속 단계입니다.

## 주요 구성요소

- **리퍼럴 이벤트 목록**: 이벤트명 / 추천인 혜택 / 피추천인 혜택 / 기간 / 참여 수 / 상태 컬럼으로 구성됩니다.
- **이벤트 등록 버튼**: DLG-077-001을 열어 새 리퍼럴 이벤트를 등록합니다.
- **실적 현황**: 전체 추천 건수 / 성공 전환 건수 / 지급된 혜택 합계를 요약합니다.
- **추천인/피추천인 이력**: 누가 누구를 추천했는지, 전환 여부와 혜택 지급 여부를 목록으로 표시합니다.
- **이벤트 삭제**: DLG-077-002를 통해 이벤트 삭제를 확인합니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 리퍼럴 이벤트 및 실적 | 이벤트 생성·수정·삭제, 실적 조회 |
| FC / 트레이너 / 스태프 | 접근 제한 | 리퍼럴 관리 불가 |

## 주요 UX 흐름

1. 사이드바 "마케팅 > 리퍼럴 프로그램"으로 진입합니다.
2. "이벤트 등록"(DLG-077-001)으로 추천인·피추천인 혜택과 기간을 설정합니다.
3. 이벤트가 시작되면 회원 앱에 리퍼럴 코드가 발급됩니다.
4. 추천 및 전환 실적을 이력 목록에서 모니터링합니다.
5. 혜택 지급 여부를 확인하고 미지급 건을 처리합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 데이터 미연동 | 화면은 표시되지만 추천 이력·혜택 지급 데이터는 목업 기준으로 표시됩니다. |
| 이벤트 없음 | 등록된 리퍼럴 이벤트가 없을 때 생성 안내가 표시됩니다. |
| 이벤트 진행 중 | 현재 활성화된 이벤트와 실시간 실적이 표시됩니다. |

## 관련 다이어그램

- DLG-077-001 리퍼럴 이벤트 등록
- DLG-077-002 리퍼럴 이벤트 삭제 확인
