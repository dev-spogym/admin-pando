## 화면 목적

회원과의 이용 계약서를 전자 방식으로 작성하고 서명까지 완료하는 화면입니다. 종이 계약 없이 태블릿 또는 모바일에서 전자 서명을 받을 수 있으며, 완료된 계약서는 이력으로 저장되어 언제든지 다시 조회할 수 있습니다.

## 주요 구성요소

- **계약 유형 선택**: 이용권 계약 / PT 계약 / 필라테스 계약 등 계약 유형을 선택합니다.
- **계약 대상 회원 선택**: 계약을 체결할 회원을 검색해 지정합니다.
- **계약 내용 입력**: 계약 기간 / 이용 서비스 / 금액 / 결제 조건 등 계약 핵심 항목을 입력합니다.
- **계약서 미리보기**: 입력된 정보가 반영된 계약서 전문을 화면에서 확인합니다.
- **전자 서명 패널**: 회원이 화면에서 직접 서명하거나, 링크를 전송해 원격 서명을 받습니다.
- **계약 이력 탭**: 이전에 체결된 계약서 목록을 날짜·계약 유형·회원명 기준으로 조회합니다.
- **계약서 재발송**: 완료된 계약서를 회원 이메일 또는 앱으로 다시 전송합니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 계약 이력 및 작성 화면 | 계약 작성, 서명 요청, 계약서 재발송 |
| FC | 담당 회원 계약 이력 | 계약 작성, 서명 요청 |
| 트레이너 / 스태프 | 접근 제한 | 전자 계약 화면 진입 불가 |

## 주요 UX 흐름

1. 사이드바 "마케팅 > 전자 계약"으로 진입합니다.
2. 계약 유형을 선택하고 대상 회원을 지정합니다.
3. 계약 내용을 입력하고 미리보기로 확인합니다.
4. 회원에게 서명 패드를 건네거나 서명 링크를 전송합니다.
5. 서명 완료 후 계약이 자동으로 저장됩니다.
6. "계약 이력" 탭에서 완료된 계약서를 조회하거나 재발송합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 작성 중 | 계약 내용 입력 단계입니다. |
| 서명 대기 | 회원 서명을 기다리는 상태입니다. 서명 완료 전에는 저장되지 않습니다. |
| 서명 완료 | 계약이 체결되고 이력에 저장됩니다. |
| 이력 없음 | 완료된 계약이 없을 때 안내 메시지가 표시됩니다. |
| 원격 서명 대기 | 링크를 전송한 뒤 회원이 서명 전 상태입니다. |
