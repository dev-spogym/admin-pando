## 목적

노쇼 또는 늦은 취소 발생 시 페널티를 자동으로 부과하는 규칙을 설정하는 팝업입니다. 한번 설정해 두면 이후 해당 조건이 충족될 때마다 수동 개입 없이 정책이 자동 적용됩니다.

## 구성요소

- **자동 페널티 활성화 토글**: 전체 자동 페널티 기능 켜기/끄기
- **노쇼 자동 페널티 규칙**:
  - 적용 여부 토글
  - 페널티 내용: 예약 제한 N일 / 횟수 N회 차감 / 병행
  - 1회 노쇼 / 누적 N회 노쇼 시 각각 다른 페널티 적용 가능
- **늦은 취소 자동 페널티 규칙**:
  - 적용 여부 토글
  - 늦은 취소 기준 시간 (수업 시작 N시간 전 이내)
  - 페널티 내용: 예약 제한 N일 / 횟수 N회 차감
- **자동 알림 설정**: 페널티 부과 시 회원에게 자동 알림 발송 여부 토글
- **저장 버튼** / **취소 버튼**

## 언제 표시되나요

- 페널티 관리 화면(SCR-C008) 정책 설정 탭에서 "자동 페널티 정책 설정" 버튼을 클릭했을 때

## 주요 흐름

1. 정책 설정 탭에서 "자동 페널티 정책 설정" 클릭 → 팝업 열림
2. 현재 적용 중인 정책 값 자동 표시
3. 노쇼·늦은 취소 각 규칙 설정
4. 자동 알림 발송 여부 설정
5. 저장 클릭 → 이후 발생하는 노쇼·늦은 취소에 즉시 새 정책 적용
6. 취소 클릭 → 변경 없이 팝업 닫힘
