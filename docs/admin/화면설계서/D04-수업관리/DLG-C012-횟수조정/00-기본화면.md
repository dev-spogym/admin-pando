## 목적

회원의 PT 잔여 횟수를 관리자가 수동으로 추가하거나 차감하는 팝업입니다. 수업이 자동으로 차감되지 않았거나, 환불·보상·오류 정정이 필요한 경우 사용합니다. 조정 내역은 이력에 기록되어 추후 확인이 가능합니다.

## 구성요소

- **회원 정보**: 회원명, 이용권명, 현재 잔여 횟수 (읽기 전용)
- **조정 유형 선택**: 추가 / 차감 라디오 버튼
- **조정 횟수 입력**: 추가 또는 차감할 횟수 수 입력 (1 이상 양의 정수)
- **조정 후 예상 잔여 횟수**: 입력값 반영 후 잔여 횟수 실시간 미리보기
- **조정 사유**: 사유 텍스트 입력 (필수) — 예: 수업 취소 보상, 오류 정정, 이벤트 추가 등
- **확인 버튼** / **취소 버튼**

## 언제 표시되나요

- 횟수 관리 화면(SCR-C007)에서 특정 회원 행의 "조정" 버튼을 클릭했을 때

## 주요 흐름

1. "조정" 버튼 클릭 → 팝업 열림, 회원 현재 잔여 횟수 표시
2. 추가 또는 차감 선택 후 조정 횟수 입력
3. 조정 후 예상 잔여 횟수 실시간 확인
4. 조정 사유 입력 (필수)
5. "확인" 클릭 → 횟수 즉시 반영, 조정 이력에 기록 저장
6. "취소" 클릭 → 팝업 닫힘, 변경 없음
