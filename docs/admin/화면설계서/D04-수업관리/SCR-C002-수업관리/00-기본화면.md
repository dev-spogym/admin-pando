## 화면 목적

등록된 수업 전체를 목록으로 조회하고 상태별로 필터링하는 화면입니다. 개별 수업의 출석 현황을 확인하고 수업 정보를 수정하거나 취소 처리할 수 있습니다. 캘린더 뷰의 보조 화면으로, 목록 기반의 정밀한 수업 관리가 필요할 때 사용합니다. 강습 세션은 KPI 집계를 위해 PT / 필라테스 / 스트레칭 / 테라피 세션 유형을 함께 표시합니다.

## 주요 구성요소

- **탭 내비게이션**: 수업 캘린더(SCR-C001)와 동일한 탭 구조. 현재 탭은 "수업 관리" 활성
- **필터 바**: 강사 드롭다운, 장소 드롭다운, 강습 세션 유형(PT/필라테스/스트레칭/테라피)으로 목록 좁히기
- **상태 필터 버튼**: 전체 / 예정 / 진행중 / 완료 / 취소 버튼 토글로 수업 상태별 분류
- **수업 목록 테이블**: 수업명, 유형(PT/GX/개인레슨/기타), 강습 세션 유형(PT/필라테스/스트레칭/테라피), 강사, 장소, 날짜·시간, 예약 현황(현재/정원), 상태 배지
- **행 클릭**: 수업 등록/수정 다이얼로그(DLG-C003) 열림
- **출석 현황 컬럼**: 수업별 출석자 수 / 노쇼 수 한눈에 확인
- **수업 등록 버튼**: 상단 우측, 새 수업 등록(DLG-C003) 진입

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 수업 목록, 모든 강사 수업 | 수업 등록·수정·취소, 출석 처리 |
| 트레이너 | 본인 담당 수업 목록 | 담당 수업 수정, 출석 처리 |
| FC / 스태프 | 전체 수업 목록 조회 | 조회만 가능 |

## 주요 UX 흐름

1. 수업 캘린더 탭에서 "수업 관리" 탭 클릭하거나 `/lessons` 직접 진입
2. 강사·장소 필터 및 상태 버튼으로 원하는 수업 좁히기
3. 목록에서 수업 행 클릭 → 수업 등록/수정(DLG-C003) 팝업으로 상세 정보 확인 및 수정
4. 완료된 수업 행에서 수업 기록 상세(DLG-C005) 열어 출석·서명 기록 확인
5. 수업 취소 처리 시 취소 확인 안내 후 상태 변경

## 화면 상태

| 상태 | 설명 |
|---|---|
| 전체 목록 | 필터 없이 전체 수업 표시 |
| 상태 필터 적용 | 예정·진행중·완료·취소 중 선택한 상태만 표시 |
| 검색 결과 없음 | 조건에 맞는 수업 없음 안내 |
| 로딩 중 | 목록 조회 중 스켈레톤 표시 |

## 비즈니스 규칙

- `수업 유형`은 화면 운영 분류이며, `강습 세션 유형`은 KPI 산출용 세부 분류입니다.
- 강습 세션 유형은 PT / 필라테스 / 스트레칭 / 테라피 4종으로 고정하며, 이 값이 본사 KPI 대시보드의 강습세션수 집계 원천이 됩니다.
