## 목적

그룹 수업 템플릿의 기본 정보를 입력하거나 기존 템플릿을 수정하는 팝업입니다. 템플릿을 잘 구성해 두면 수업 등록 시 반복 입력 없이 빠르게 수업 정보를 완성할 수 있습니다.

## 구성요소

- **수업명**: 텍스트 입력 (필수)
- **수업 유형**: GX / 스피닝 / 필라테스 / 기구필라테스 / 기타 선택
- **기본 정원**: 최대 예약 인원 수 입력
- **기본 장소**: 룸·장소 드롭다운 선택
- **기본 수업 시간**: 분 단위 입력 (예: 50분)
- **색상**: 캘린더에 표시될 수업 색상 팔레트 선택
- **활성 여부**: 활성 / 비활성 토글
- **설명**: 수업 설명 또는 특이사항 입력 (선택)
- **저장 버튼** / **취소 버튼**

## 언제 표시되나요

- 그룹 수업 템플릿 화면(SCR-C004)에서 "템플릿 등록" 버튼을 클릭했을 때
- 템플릿 목록에서 기존 템플릿의 편집 버튼을 클릭했을 때 (수정 모드)

## 주요 흐름

1. "템플릿 등록" 클릭 → 빈 폼으로 팝업 열림
2. 수업명·유형·정원·장소·시간·색상 입력
3. 저장 클릭 → 템플릿 목록에 즉시 반영
4. 수정 모드: 기존 템플릿 정보 자동 입력 → 변경 후 저장
5. 취소 클릭 → 팝업 닫힘, 변경 없음
