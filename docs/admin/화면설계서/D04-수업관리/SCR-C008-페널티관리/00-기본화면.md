## 화면 목적

노쇼(무단 불참) 또는 늦은 취소로 인해 회원에게 부과된 페널티 현황을 조회하고, 페널티 정책을 설정하는 화면입니다. 페널티를 통해 무단 결석을 억제하고 수업 자리가 낭비되지 않도록 운영 규율을 유지합니다.

## 주요 구성요소

- **탭 구조**: 페널티 현황 / 정책 설정 두 탭으로 구성
- **페널티 현황 탭**:
  - 회원명, 연락처, 페널티 유형(노쇼/늦은 취소), 부과일, 해당 수업, 페널티 내용(예약 제한 기간 또는 횟수 차감), 상태(적용 중/해제)
  - 회원 검색 필터
  - 페널티 해제 버튼 (관리자 수동 해제)
  - 페널티 수동 등록 버튼 → DLG-C014
- **정책 설정 탭**:
  - 노쇼 발생 시 자동 페널티 적용 여부 토글
  - 자동 페널티 규칙 설정 → DLG-C015
  - 늦은 취소 기준 시간 설정 (예: 수업 시작 2시간 전 이내 취소 시 페널티)
  - 페널티 내용 설정 (예약 제한 N일, 또는 횟수 차감 N회)

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 페널티 현황, 정책 설정 탭 | 페널티 등록·해제·정책 변경 |
| 트레이너 | 담당 회원의 페널티 현황 조회 | 조회만 가능 |
| FC / 스태프 | 전체 페널티 현황 조회 | 조회만 가능 |

## 주요 UX 흐름

1. 수업 캘린더 탭 > "페널티 관리" 탭 클릭 또는 `/penalties` 직접 진입
2. 페널티 현황 탭에서 현재 적용 중인 페널티 목록 확인
3. 수동 페널티 부과가 필요한 경우 "페널티 등록" 버튼 클릭 → DLG-C014
4. 부당한 페널티 해제가 필요한 경우 해제 버튼 클릭 → 확인 후 해제
5. 정책 설정 탭에서 자동 페널티 규칙 변경 → DLG-C015

## 화면 상태

| 상태 | 설명 |
|---|---|
| 페널티 없음 | 현재 적용 중인 페널티 없음 안내 |
| 페널티 적용 중 | 회원별 페널티 현황 목록 표시 |
| 정책 비활성 | 자동 페널티 비활성화 상태 안내 |
| 처리 중 | 해제 또는 등록 처리 중 로딩 표시 |
