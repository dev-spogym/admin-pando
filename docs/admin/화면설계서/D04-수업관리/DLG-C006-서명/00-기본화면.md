## 목적

PT 수업 완료 후 회원이 수업 진행 사실을 확인하는 서명을 입력하는 팝업입니다. 회원의 서명은 수업 이력에 저장되어 분쟁 발생 시 수업이 실제로 이루어졌음을 증명하는 근거 자료로 활용됩니다.

## 구성요소

- **안내 문구**: "수업이 완료되었습니다. 서명해 주세요."
- **수업 정보 요약**: 수업명, 일시, 담당 강사명 (서명 전 확인용)
- **서명 패드**: 터치 또는 마우스로 서명 입력하는 캔버스 영역
- **지우기 버튼**: 서명 초기화 후 재입력
- **확인 버튼**: 서명 저장 및 수업 기록 완료 처리
- **건너뛰기 버튼**: 서명 없이 수업 완료 처리 (관리자 권한, 서명 미수집 사유 기록)

## 언제 표시되나요

- 유효 수업 목록(SCR-C011)에서 PT 수업 완료 후 "서명 요청" 버튼 클릭 시
- 수업 완료 처리 흐름에서 자동으로 표시되는 경우

## 주요 흐름

1. 트레이너가 "서명 요청" 버튼 클릭 → 서명 팝업 열림
2. 회원에게 기기를 건네주거나 화면을 보여줌
3. 회원이 서명 패드에 직접 서명 입력
4. "지우기"로 재서명 가능
5. "확인" 클릭 → 서명 이미지 저장, 수업 기록 완료 처리
6. 서명 불가 시 "건너뛰기" → 미서명 사유 입력 후 완료 처리
