## 화면 목적

강사별 수업 배정 현황과 근무 시간을 요약해 파악하는 화면입니다. 특정 기간 동안 각 강사가 몇 건의 수업을 담당하고 총 몇 시간을 근무했는지 확인하고, 업무 부담이 편중된 강사가 없는지 점검하는 데 사용합니다.

## 주요 구성요소

- **기간 필터**: 주간 / 월간 / 사용자 지정 기간 선택
- **강사 목록 카드 또는 테이블**:
  - 강사명, 역할(트레이너/GX 강사 등)
  - 담당 수업 건수
  - 총 근무 시간 (수업 시간 합산)
  - 수업 유형별 분포 (PT/GX/개인레슨 등)
  - 강습 세션 유형별 분포 (PT/필라테스/스트레칭/테라피)
  - 출석 처리 완료율
  - O.T 배정 합계 / O.T 1 완료 / O.T 2 완료 / O.T 1·2 예정
- **강사 행 클릭**: 강사 상세 팝업(DLG-C010)에서 담당 수업 목록 및 기본 정보 확인
- **강사별 색상 구분**: 캘린더와 동일한 강사 색상 코드 적용
- **요약 통계**: 평균 근무 시간, 최다 수업 강사, 수업 없는 강사 등 상단 요약 지표
- **OT 운영 요약**: 트레이너별 OT 신규 배정, 1차 완료, 2차 완료, 예정, 거절(거부+보류), 이월 건을 표 또는 보조 패널로 표시

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 강사 근무 현황, 시간 합계 | 조회, 강사 상세 팝업 확인 |
| 트레이너 | 본인 근무 현황만 조회 | 조회만 가능 |
| FC / 스태프 | 전체 강사 근무 현황 | 조회만 가능 |

## 주요 UX 흐름

1. 사이드바 > 수업관리 > 강사 근무 현황 진입
2. 기간 필터로 조회 기간 설정
3. 강사별 수업 건수·근무 시간과 OT 배정·진행 현황 확인
4. OT 1차/2차 예정자가 많은 강사를 확인해 배정 재조정 또는 후속 연락 필요 여부 판단
5. 특정 강사 행 클릭 → DLG-C010에서 담당 수업 목록과 OT 상세 확인
6. 업무 편중 강사 파악 후 수업 재배정 검토 (캘린더 화면에서 수정)

## 화면 상태

| 상태 | 설명 |
|---|---|
| 기본 현황 | 이번 주 전체 강사 근무 현황 |
| 수업 없는 강사 | 해당 기간 배정 수업 0건 강사 표시 (회색 처리) |
| 데이터 없음 | 해당 기간 강사 정보 없음 안내 |
| 로딩 중 | 스켈레톤 표시 |

|---|---|---|---|
| O.T 배정 합계 | `[강습업무보고]` | 주간보고 E열 | 트레이너별 월 배정 회원 수 합계 |
| O.T 1 완료 | `[강습업무보고]`, `[OT관리-강습]` | 주간보고 F열, OT현황 Q열 | 진행 값이 `완료`인 1차 OT |
| O.T 2 완료 | `[강습업무보고]`, `[OT관리-강습]` | 주간보고 G열, OT현황 T열 | 진행 값이 `완료`인 2차 OT |
| O.T 1 예정 | `[OT관리-강습]` | OT현황 Q열 | 진행 값이 `예정`인 1차 OT |

## 비즈니스 규칙

- OT 지표는 정규 PT 수업 완료율과 분리해 영업 파이프라인 지표로 표시합니다.
- O.T 배정 합계는 트레이너별 업무 부하 균형을 보기 위한 개인 KPI이며, 본사 KPI 센터의 트레이너별 OT 배정 합계와 동일 원천을 사용합니다.
- 강습 세션 유형은 `PT`, `필라테스`, `스트레칭`, `테라피` 4종으로 고정하며, 트레이너별 수업수와 본사 KPI 대시보드의 강습세션수 산출 기준으로 사용합니다.
- OT 등록 전환율은 본사 KPI 표준인 `등록 / OT 진행`을 따릅니다.
- OT 거절 건은 `거부+보류`를 함께 집계했다고 화면 툴팁 또는 설명에 명시합니다.
- OT 이월률의 분모는 우선 `OT배정 인원`으로 기록하지만, `OT 진행` 기준과 구분이 필요한 산식으로 추후 확정합니다.
