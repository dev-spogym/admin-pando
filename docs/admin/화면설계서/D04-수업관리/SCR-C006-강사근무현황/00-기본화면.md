## 화면 목적

강사별 수업 배정 현황과 근무 시간을 요약해 파악하는 화면입니다. 특정 기간 동안 각 강사가 몇 건의 수업을 담당하고 총 몇 시간을 근무했는지 확인하고, 업무 부담이 편중된 강사가 없는지 점검하는 데 사용합니다.

## 주요 구성요소

- **기간 필터**: 주간 / 월간 / 사용자 지정 기간 선택
- **강사 목록 카드 또는 테이블**:
  - 강사명, 역할(트레이너/GX 강사 등)
  - 담당 수업 건수
  - 총 근무 시간 (수업 시간 합산)
  - 수업 유형별 분포 (PT/GX/개인레슨 등)
  - 출석 처리 완료율
- **강사 행 클릭**: 강사 상세 팝업(DLG-C010)에서 담당 수업 목록 및 기본 정보 확인
- **강사별 색상 구분**: 캘린더와 동일한 강사 색상 코드 적용
- **요약 통계**: 평균 근무 시간, 최다 수업 강사, 수업 없는 강사 등 상단 요약 지표

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 강사 근무 현황, 시간 합계 | 조회, 강사 상세 팝업 확인 |
| 트레이너 | 본인 근무 현황만 조회 | 조회만 가능 |
| FC / 스태프 | 전체 강사 근무 현황 | 조회만 가능 |

## 주요 UX 흐름

1. 사이드바 > 수업관리 > 강사 근무 현황 진입
2. 기간 필터로 조회 기간 설정
3. 강사별 수업 건수·근무 시간 확인
4. 특정 강사 행 클릭 → DLG-C010에서 담당 수업 목록 상세 확인
5. 업무 편중 강사 파악 후 수업 재배정 검토 (캘린더 화면에서 수정)

## 화면 상태

| 상태 | 설명 |
|---|---|
| 기본 현황 | 이번 주 전체 강사 근무 현황 |
| 수업 없는 강사 | 해당 기간 배정 수업 0건 강사 표시 (회색 처리) |
| 데이터 없음 | 해당 기간 강사 정보 없음 안내 |
| 로딩 중 | 스켈레톤 표시 |
