## 목적

시간표 일괄 등록 화면(SCR-C003)에서 입력한 설정대로 수업 일정을 대량 생성하기 전, 생성될 일정의 요약 정보를 최종 확인하고 승인하는 팝업입니다. 실수로 대량 일정이 생성되는 것을 방지하는 마지막 확인 단계입니다.

## 구성요소

- **생성 요약 정보**:
  - 수업명, 강사, 장소
  - 적용 기간 (시작일 ~ 종료일)
  - 적용 요일
  - 수업 시간 (시작~종료)
  - 생성될 총 일정 건수
- **충돌 일정 경고** (해당 시):
  - 기존 일정과 겹치는 날짜·시간 목록 표시
  - 충돌 일정 건수 강조
  - 충돌 무시하고 생성 / 충돌 일정 제외하고 생성 선택 옵션
- **제외 날짜 목록**: 사용자가 설정한 예외 날짜 목록
- **확인 버튼**: 일정 생성 실행
- **취소 버튼**: 팝업 닫기, 설정 화면으로 돌아가기

## 언제 표시되나요

- 시간표 일괄 등록 화면(SCR-C003)에서 "일괄 생성" 버튼을 클릭했을 때

## 주요 흐름

1. "일괄 생성" 클릭 → 팝업 열림, 생성 예정 일정 요약 표시
2. 충돌 일정이 있으면 경고 표시 → 충돌 처리 방식 선택
3. 요약 내용 확인 후 "확인" 클릭 → 일정 대량 생성 시작
4. 생성 완료 후 성공 토스트 + 캘린더 화면으로 이동
5. "취소" 클릭 → 팝업 닫힘, 시간표 일괄 등록 화면으로 복귀
