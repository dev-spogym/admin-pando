## 목적

자동 페널티 정책 외에, 관리자가 특정 회원에게 직접 페널티를 수동으로 부과하는 팝업입니다. 정책에 포함되지 않는 특수 상황에서 유연하게 페널티를 적용할 수 있습니다.

## 구성요소

- **대상 회원 선택**: 회원 검색 또는 드롭다운으로 페널티 부과 대상 선택
- **페널티 유형 선택**: 노쇼 / 늦은 취소 / 기타
- **관련 수업 선택**: 페널티 사유가 된 수업 선택 (선택)
- **페널티 내용**:
  - 예약 제한: 제한 기간(일수) 입력
  - 횟수 차감: 차감 횟수 입력
  - 두 가지 병행 적용 가능
- **적용 시작일**: 페널티 적용 시작일 (기본값: 오늘)
- **사유 입력**: 페널티 부과 사유 상세 기록 (필수)
- **확인 버튼** / **취소 버튼**

## 언제 표시되나요

- 페널티 관리 화면(SCR-C008)의 페널티 현황 탭에서 "페널티 등록" 버튼을 클릭했을 때

## 주요 흐름

1. "페널티 등록" 클릭 → 팝업 열림
2. 대상 회원 검색 및 선택
3. 페널티 유형·내용·시작일 설정
4. 사유 입력 (필수)
5. "확인" 클릭 → 페널티 즉시 적용, 회원에게 알림 발송 (설정에 따라)
6. "취소" 클릭 → 팝업 닫힘, 변경 없음
