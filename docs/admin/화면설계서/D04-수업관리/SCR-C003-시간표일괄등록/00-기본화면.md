## 화면 목적

반복되는 수업 일정을 한 번에 대량으로 등록하는 화면입니다. 매주 같은 요일·시간에 열리는 그룹 수업이나 PT 일정을 개별 등록하는 번거로움 없이, 기간·요일·시간·강사·장소를 설정하면 지정된 기간 동안의 일정이 자동으로 생성됩니다.

## 주요 구성요소

- **기간 설정**: 시작일 ~ 종료일 날짜 선택 (달력 피커)
- **요일 선택**: 월~일 중 복수 선택 가능한 토글 버튼
- **시간 설정**: 시작 시간 ~ 종료 시간 (30분 단위 드롭다운)
- **수업 선택**: 등록된 수업 템플릿 드롭다운 (수업명, 유형, 정원 자동 연동)
- **강사 배정**: 담당 강사 드롭다운 선택
- **장소 설정**: 룸·장소 드롭다운 선택
- **정원 입력**: 최대 예약 인원 수 입력 (템플릿 기본값 자동 채워짐)
- **예외 날짜 제외**: 공휴일·휴관일 등 특정 날짜를 제외 처리
- **미리보기 영역**: 생성될 일정 건수와 날짜 목록을 등록 전에 미리 확인
- **일괄 생성 버튼**: 클릭 시 최종 확인 다이얼로그(DLG-C008) 표시

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 강사, 전체 수업 옵션 | 일괄 등록 실행 |
| 트레이너 | 본인 이름만 강사로 선택 가능 | 본인 담당 수업 일괄 등록 |
| FC / 스태프 | 접근 불가 | 이 기능 사용 불가 |

## 주요 UX 흐름

1. 사이드바 > 수업관리 > 시간표 일괄 등록 진입
2. 적용 기간(시작일~종료일) 선택
3. 수업할 요일 복수 선택
4. 시작·종료 시간 설정
5. 수업 템플릿·강사·장소 선택
6. 제외할 날짜(공휴일 등) 추가
7. 미리보기에서 생성될 일정 건수·날짜 확인
8. "일괄 생성" 버튼 클릭 → 최종 확인 다이얼로그(DLG-C008) → 확인 후 일정 생성

## 화면 상태

| 상태 | 설명 |
|---|---|
| 초기 상태 | 기간·요일·시간 미설정, 미리보기 비어 있음 |
| 설정 완료 | 필수 항목 모두 입력, 미리보기에 생성 예정 건수 표시 |
| 충돌 경고 | 기존 일정과 겹치는 시간대 있을 때 충돌 날짜 목록 표시 |
| 생성 중 | 일괄 생성 처리 중 로딩 표시, 버튼 비활성 |
| 생성 완료 | 성공 토스트 후 캘린더 화면으로 이동 |
