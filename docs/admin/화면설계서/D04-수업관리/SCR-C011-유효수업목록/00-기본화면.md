## 화면 목적

이용권이 유효하고 예약이 확정된 수업 중 실제로 진행 가능한 수업 목록을 보여주는 화면입니다. 만료되거나 취소된 수업을 제외한 "진행 예정 확정 수업"만 필터링해 출석 처리, 서명 진행, 당일 운영 확인에 활용합니다.

## 주요 구성요소

- **탭 내비게이션**: 수업 캘린더와 동일한 탭 구조. 현재 탭은 "유효 수업 목록" 활성
- **날짜 필터**: 오늘 / 이번 주 / 날짜 직접 선택
- **수업 목록 테이블**:
  - 수업명, 수업 유형(PT/GX 등), 강습 세션 유형(PT/필라테스/스트레칭/테라피), 강사명
  - 일시 (날짜, 시작~종료 시간)
  - 장소
  - 예약 회원명, 연락처
  - 이용권 잔여 횟수
  - 출석 상태 (미처리 / 출석 / 결석 / 노쇼)
- **출석 처리 버튼**: 각 행에서 바로 출석·결석·노쇼 처리 가능
- **서명 요청 버튼**: PT 수업 완료 후 회원 서명 받기(DLG-C006) 진입
- **수업 기록 상세 버튼**: 완료 수업의 기록 확인(DLG-C005) 진입

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 유효 수업 목록 | 출석 처리, 서명 요청 |
| 트레이너 | 본인 담당 유효 수업 목록 | 출석 처리, 서명 요청 |
| FC / 스태프 | 전체 유효 수업 목록 | 출석 처리 (서명 요청은 트레이너 권한) |

## 주요 UX 흐름

1. 수업 캘린더 탭 > "유효 수업 목록" 탭 클릭 또는 `/valid-lessons` 직접 진입
2. 날짜 필터로 오늘 또는 특정 날짜 수업 표시
3. 출석 처리가 필요한 수업 행에서 출석 상태 선택 (출석/결석/노쇼)
4. PT 수업 완료 후 "서명 요청" 버튼 → DLG-C006에서 회원 서명 받기
5. 노쇼 처리 시 페널티 자동 부과 여부 확인 (정책에 따라)
6. 완료 수업의 기록 확인 → DLG-C005 세션 기록 조회

## 화면 상태

| 상태 | 설명 |
|---|---|
| 오늘 수업 | 오늘 날짜 기준 유효 수업 목록 기본 표시 |
| 출석 미처리 강조 | 수업 시작 시간 경과 후 미처리 항목 주황 표시 |
| 목록 없음 | 해당 날짜 유효 수업 없음 안내 |
| 전체 처리 완료 | 모든 수업 출석 처리 완료 시 완료 안내 |

## 비즈니스 규칙

- 유효 수업 목록에서 처리된 출석/완료 이력은 강습 세션 유형별 수업수 검증에 사용됩니다.
- 강습 세션 유형은 PT / 필라테스 / 스트레칭 / 테라피 4종으로 고정합니다.
