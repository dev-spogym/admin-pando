## 화면 목적

PT 및 그룹 수업 전체 일정을 캘린더 형태로 한눈에 파악하는 메인 수업 관리 화면입니다. 월·주·일·목록 뷰를 자유롭게 전환하며 수업 유형과 강사를 색상으로 구분해 일정을 직관적으로 확인하고, 직접 클릭하거나 드래그해 빠르게 수업을 등록·수정할 수 있습니다.

## 주요 구성요소

- **뷰 전환 탭**: 월·주·일·목록(list) 뷰 전환 버튼. 뷰에 따라 캘린더 표시 밀도가 달라짐
- **상단 액션 버튼**: 스케줄 일괄 변경(DLG-C004 진입), 수업 등록(DLG-C001 진입)
- **탭 내비게이션**: 일정표 / 수업 관리 / 횟수 관리 / 페널티 관리(건수 배지) / 유효 수업 목록 — 탭 클릭 시 각 전용 화면으로 이동
- **필터 바**:
  - 강사 필터: 드롭다운으로 특정 강사 일정만 표시
  - 수업명 필터: 드롭다운으로 특정 수업만 표시
  - 정원 상태 필터: 전체 / 여유 / 마감 버튼 토글
  - 분류 멀티필터: 상담·OT·체성분·방문·수업·PT·기타 복수 선택 가능
- **색상 범례**: 강사별 색상, 수업 유형별 색상, 일정 분류별 색상을 화면 하단에 표시
- **캘린더 영역**: 수업 블록 클릭 시 일정 상세(DLG-C002) 팝업, 빈 슬롯 클릭 시 수업 등록(DLG-C001) 팝업
- **우측 패널**: 선택한 날짜의 수업 목록을 카드 형태로 표시. 수업명·강사·시간·예약 현황·정원 바 포함
- **미승인 일정 알림 배너**: 미승인 일정이 있을 때 상단에 경고 배너 표시

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 강사 일정, 모든 수업 유형 | 수업 등록·수정·삭제, 일괄 변경, 미승인 일정 승인 |
| 트레이너 | 본인 담당 수업 중심 (전체 조회 가능) | 담당 수업 등록·수정, 드래그 이동 |
| FC / 스태프 | 전체 수업 일정 조회 | 조회만 가능, 등록·수정 불가 |

## 주요 UX 흐름

1. 사이드바 > 수업관리 > 수업/캘린더 진입 시 이번 달 월 뷰로 기본 표시
2. 뷰 전환 버튼(월/주/일/목록)으로 캘린더 밀도 변경
3. 강사·수업명·분류 필터로 원하는 일정만 표시
4. 수업 블록 클릭 → 일정 상세(DLG-C002)에서 정보 확인 및 수정 진입
5. 빈 시간 슬롯 클릭 또는 상단 "수업 등록" 버튼 → 수업 등록(DLG-C001) 팝업
6. 드래그&드롭으로 수업 시간 이동, 블록 하단 드래그로 시간 연장
7. 우측 패널에서 해당 날짜 수업 목록 카드 형태로 빠르게 확인

## 화면 상태

| 상태 | 설명 |
|---|---|
| 기본(월 뷰) | 이번 달 전체 수업 블록 표시, 일별 최대 3건 + 더보기 링크 |
| 필터 적용 | 선택한 강사·수업명·분류에 맞는 수업만 표시 |
| 빈 일정 | 해당 기간에 수업 없음 안내 문구 표시 |
| 미승인 일정 존재 | 상단에 주황색 경고 배너 + 건수 표시 |
| 드래그 중 | 수업 블록 반투명 이동 시각 피드백 |
| 시간 충돌 감지 | 겹치는 일정 있을 때 경고 표시 후 저장 차단 |
