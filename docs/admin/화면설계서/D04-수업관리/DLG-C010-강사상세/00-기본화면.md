## 목적

특정 강사의 기본 정보와 해당 기간 내 담당 수업 목록을 빠르게 확인하는 팝업입니다. 강사 근무 현황 화면에서 개별 강사를 클릭할 때 열리며, 별도 페이지 이동 없이 핵심 정보를 조회할 수 있습니다.

## 구성요소

- **강사 기본 정보**:
  - 이름, 역할(트레이너/GX강사 등), 프로필 사진
  - 연락처, 담당 수업 유형
- **기간 내 수업 요약**:
  - 총 수업 건수, 총 근무 시간
  - 수업 유형별 분포 (PT N건, GX N건 등)
  - 강습 세션 유형별 분포 (PT / 필라테스 / 스트레칭 / 테라피)
- **기간 내 OT 요약**:
  - O.T 배정 합계, O.T 1 완료, O.T 2 완료, O.T 1·2 예정
  - OT 거절 건은 `거부+보류`를 함께 집계해 표시
  - OT 이월 건은 별도 배지로 표시
- **담당 수업 목록**:
  - 수업명, 날짜·시간, 장소, 예약 현황, 출석 처리 상태
  - 목록은 최근순 정렬
- **담당 OT 목록**:
  - 회원명, 등록일, 구분, OT 1 일자·진행, OT 2 일자·진행, 등록여부 표시
  - 진행 값은 `완료`, `예정`으로 구분
- **닫기 버튼**

## 언제 표시되나요

- 강사 근무 현황 화면(SCR-C006)에서 강사 행을 클릭했을 때

## 주요 흐름

1. 강사 근무 현황 목록에서 강사 행 클릭 → 팝업 열림
2. 강사 기본 정보, 기간 내 수업 요약, OT 배정·진행 요약 확인
3. 담당 수업 목록과 담당 OT 목록을 스크롤하며 개별 현황 파악
4. 닫기 버튼 또는 외부 클릭으로 팝업 닫힘

## 데이터 기준

| 항목 | 소스 파일 | 탭 / 셀 | 기준 |
|---|---|---|---|
| O.T 배정 합계 / O.T 1 / O.T 2 | `[강습업무보고]` | 주간보고 E:G열 | 트레이너별 행 기준 |
| 담당 OT 진행 상태 | `[OT관리-강습]` | OT현황 J/Q/T/V열 | 담당자, 1차 진행, 2차 진행, 등록여부 |
| OT 거절 | `[OT관리-강습]` | OT현황 V열 | `거부+보류`를 함께 집계 |
| 강습 세션 유형별 수업수 | 수업/출석/일정 원장 | 강습 세션 유형 | PT / 필라테스 / 스트레칭 / 테라피별 완료 수업수 |
