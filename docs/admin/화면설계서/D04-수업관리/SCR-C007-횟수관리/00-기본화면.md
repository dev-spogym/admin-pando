## 화면 목적

PT 이용권을 보유한 회원의 잔여 수업 횟수를 조회하고, 필요 시 횟수를 수동으로 조정하는 화면입니다. 수업 진행 후 자동 차감되지 않은 경우나, 환불·보상·오류 정정이 필요한 경우 관리자가 직접 횟수를 추가하거나 차감합니다.

## 주요 구성요소

- **회원 검색**: 이름 또는 전화번호로 회원 검색
- **회원 목록 테이블**:
  - 회원명, 연락처, 이용권명, 구매 횟수, 사용 횟수, 잔여 횟수, 만료일
  - 잔여 횟수가 적을수록 강조 표시 (예: 3회 이하 주황, 0회 빨강)
- **횟수 조정 버튼**: 각 행의 조정 버튼 클릭 → DLG-C012 횟수 조정 팝업
- **차감 이력 버튼**: 각 행의 이력 버튼 클릭 → DLG-C013 차감 이력 팝업
- **필터**: 잔여 횟수 부족 회원만 보기, 만료 임박 이용권 보기

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 센터장 / 매니저 | 전체 회원 횟수 목록 | 횟수 조정, 이력 조회 |
| 트레이너 | 담당 회원 횟수 목록 | 이력 조회만 가능 (횟수 조정 불가) |
| FC / 스태프 | 전체 회원 횟수 목록 | 조회·이력 조회만 가능 |

## 주요 UX 흐름

1. 수업 캘린더 탭 > "횟수 관리" 탭 클릭 또는 `/lesson-counts` 직접 진입
2. 회원 검색 또는 목록 스크롤로 대상 회원 확인
3. 잔여 횟수 확인 후 조정이 필요한 경우 "조정" 버튼 클릭 → DLG-C012
4. 횟수 사용 이력 확인이 필요한 경우 "이력" 버튼 클릭 → DLG-C013
5. 조정 완료 후 목록 자동 갱신

## 화면 상태

| 상태 | 설명 |
|---|---|
| 기본 목록 | 전체 PT 회원 횟수 목록 |
| 잔여 부족 필터 | 잔여 3회 이하 회원만 표시 |
| 검색 결과 없음 | 조건에 맞는 회원 없음 안내 |
| 만료 임박 강조 | 만료일 7일 이내 이용권 주황 배지 |
