## 목적

노쇼 및 늦은 취소 발생 시 적용할 정책을 설정하는 팝업입니다. 페널티를 자동으로 부과할 기준과 내용을 정의해 두면, 이후 노쇼·늦은 취소 발생 시 수동 개입 없이 정책이 자동 적용됩니다.

## 구성요소

- **노쇼 정책 섹션**:
  - 노쇼 자동 페널티 적용 여부 토글
  - 페널티 내용 선택: 예약 제한 N일 / 잔여 횟수 N회 차감 / 병행 적용
  - N값 직접 입력 (일수 또는 횟수)
- **늦은 취소 정책 섹션**:
  - 늦은 취소 기준 시간 설정 (수업 시작 몇 시간 전 이내 취소 시 패널티 적용)
  - 페널티 내용 선택: 예약 제한 N일 / 잔여 횟수 N회 차감
- **누적 노쇼 기준 추가 페널티** (선택):
  - 월 N회 이상 노쇼 시 추가 페널티 적용 여부
- **저장 버튼** / **취소 버튼**

## 언제 표시되나요

- 페널티 관리 화면(SCR-C008) 정책 설정 탭에서 "노쇼/취소 정책" 편집 버튼 클릭 시
- 자동 페널티 정책(DLG-C015)에서 상세 정책 설정으로 진입 시

## 주요 흐름

1. 정책 설정 탭에서 편집 버튼 클릭 → 팝업 열림
2. 현재 설정된 정책 값이 자동 입력됨
3. 노쇼·늦은 취소 각각의 페널티 내용과 기준 시간 수정
4. 저장 클릭 → 이후 발생하는 노쇼·늦은 취소에 새 정책 적용
5. 취소 클릭 → 변경 없이 팝업 닫힘
