## 목적

수업 관리 화면(SCR-C002)에서 새 수업을 등록하거나 기존 수업 정보를 수정하는 팝업입니다. 캘린더 기반의 DLG-C001과 동일한 입력 항목을 제공하되, 목록 화면의 맥락에서 호출됩니다.

## 구성요소

- **수업 템플릿 선택**: 드롭다운으로 기존 수업 선택 (수업명·유형·정원 자동 완성)
- **수업명**: 직접 입력 가능 (템플릿 미선택 시)
- **수업 유형**: PT / GX / 개인레슨 / 기타 선택
- **강습 세션 유형**: PT / 필라테스 / 스트레칭 / 테라피 선택. 강습세션수 KPI 집계 대상 수업일 때 필수
- **일시**: 날짜 + 시작·종료 시간 선택
- **강사 배정**: 담당 강사 드롭다운
- **장소**: 룸·장소 드롭다운
- **정원**: 최대 예약 인원 입력
- **상태**: 예정 / 진행중 / 완료 / 취소 (수정 모드에서만 표시)
- **메모**: 내부 메모 입력 (선택)
- **저장 버튼** / **취소 버튼**

## 언제 표시되나요

- 수업 관리 화면(SCR-C002)에서 "수업 등록" 버튼을 클릭했을 때
- 수업 목록에서 특정 수업 행을 클릭했을 때 (수정 모드)

## 주요 흐름

1. "수업 등록" 클릭 → 빈 폼으로 팝업 열림
2. 수업 템플릿 선택 또는 직접 입력으로 수업 정보 채우기
3. 강사·장소·일시 설정
4. 저장 클릭 → 시간 충돌 감지 후 수업 목록에 즉시 반영
5. 수정 모드: 기존 정보 자동 입력 → 변경 후 저장

## 비즈니스 규칙

- `수업 유형`은 운영 분류이고, `강습 세션 유형`은 본사 KPI 산출용 세부 분류입니다.
- 강습 세션 유형은 PT / 필라테스 / 스트레칭 / 테라피 4종으로 고정합니다.
