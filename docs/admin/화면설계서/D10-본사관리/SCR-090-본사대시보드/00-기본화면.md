## 화면 목적
로그인 직후 가장 먼저 진입하는 화면으로, 현재 지점의 운영 현황을 한눈에 파악합니다.
회원 현황, 매출, 출석, 주의 대상 회원(만료 예정·생일자·미수금·홀딩), 최근 활동 이력을 종합적으로 표시합니다.

## 주요 구성요소

### 핵심 지표 요약 카드
- 전체 회원 수
- 활성 회원 수
- 만료 예정 회원 수 (30일 이내)
- 오늘 출석 수
- 이번 달 매출

### 분포 차트
- 회원 성별 분포 (원형 차트)
- 회원 연령대 분포 (막대 차트)

### 추이 차트
- 월별 매출 추이 (선 또는 막대 차트)
- 주간 출석 현황 (막대 차트)

### 주의 대상 위젯
- 오늘 생일 회원 목록
- 미수금 발생 회원 목록
- 홀딩 중인 회원 목록
- 만료 임박 회원 목록 (각 위젯별 전체보기 링크)

### 최근 활동 로그
- 최근 시스템 활동 이력 요약 (누가 무엇을 했는지)

### 상단 액션
- 수동 새로고침 버튼

## 계정별 차이
| 계정 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 | 현재 선택된 지점의 대시보드 | 새로고침, 위젯 내 회원 상세 이동 |
| 센터장 / 직원 | 자신이 속한 지점의 대시보드 | 새로고침, 위젯 내 회원 상세 이동 |

## 주요 UX 흐름
1. 로그인 완료 시 자동으로 이 화면으로 이동한다.
2. 첫 방문 시 환영 배너가 상단에 표시되며 닫기 버튼으로 제거한다.
3. 데이터 로딩 중에는 각 카드와 위젯에 스켈레톤이 표시된다.
4. 위젯 내 회원 이름 클릭 → 해당 회원 상세 화면으로 이동.
5. 위젯 하단 "전체보기" 클릭 → 해당 메뉴 화면으로 이동.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 지표 카드, 차트, 위젯 정상 표시 |
| 로딩 | 데이터 불러오는 중 (스켈레톤 표시) |
| 첫 방문 | 환영 배너 상단 표시 |
| 데이터 없음 | 각 위젯에 데이터 없음 안내 표시 |
| 오류 | 데이터 로드 실패 시 재시도 안내 표시 |
