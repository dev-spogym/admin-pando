## 화면 목적
신규 회원의 초기 정착 과정을 단계별로 추적하는 화면입니다.
리드 유입부터 등록, 초기 출석 정착까지의 퍼널과 이탈 위험 회원을 조기에 파악하여 적절한 후속 조치를 취할 수 있도록 지원합니다.

## 주요 구성요소

### 프리-온보딩 KPI
- 이번 달 신규 리드 수
- 연락 완료 건수
- 방문 완료 건수
- 등록 전환 건수
- 리드 전환율
- WI vs TI 건수

### 신규 유치 KPI
- 이번 달 신규 등록 회원 수
- 리드 누락률
- 회원소개 유입 비중
- 가입경로(등록자 기준) 분포

### 신규 안정 KPI
- 7일 이용률
- 30일 활동률
- PT 체험 참여율
- 체험 → 등록 전환율
- 일일입장 → 회원 전환율
- GX 첫 참여율
- 초기 이탈 회원 수

### 상담/가입 경로 분포
- `[주간월간보고]` I10:Q10의 등록자 기준 가입 경로 분포를 막대 차트로 표시합니다.
- `[주간월간보고]` R10:Z10은 WI 상세 경로, AA10:AI10은 TI 상세 경로로 분리해 표시합니다.
- 상세 경로는 간판·블로그·인스타·당근·지역카페·현수막·전단지·회원소개·기타 순서를 유지합니다.

### 온보딩 단계 진행률
- 가입 → 초기 상담 → 체성분 측정 → 프로그램 배정 → 첫 수업 완료 5단계를 진행률 막대로 표시합니다.
- 단계별 이탈률이 높은 구간을 강조합니다.

### 신규 회원 목록
- 이번 달 등록한 신규 회원의 등록일, 경과일, 방문 횟수, PT 체험 완료 여부, GX 참여 여부, 온보딩 상태를 테이블로 조회합니다.
- 이탈 위험 또는 이탈위험 상태의 회원은 별도 섹션으로 강조합니다.

## 계정별 차이
| 계정 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 | 현재 선택된 지점의 온보딩 현황 | 지점 전환, 데이터 조회 |
| 센터장 / 지점 관리자 | 자신의 지점 온보딩 현황 | 데이터 조회 |
| FC | 담당 리드와 신규 회원 중심 온보딩 현황 | 후속 상담 대상 확인 |
| 트레이너 | 체험·첫 수업 관련 신규 회원 현황 | 체험 후속 조치 확인 |

## 주요 UX 흐름
1. 화면 진입 시 현재 지점의 이번 달 신규 회원 온보딩 현황을 자동 조회합니다.
2. 상단 KPI 카드에서 리드 유입, 방문, 등록, 초기 정착 상태를 빠르게 확인합니다.
3. 상담/가입 경로 분포에서 등록자 기준으로 실제 전환이 높은 유입 경로를 확인합니다.
4. 체험 → 등록 전환율과 일일입장 → 회원 전환율을 통해 비용성 유입의 회수 성과를 판단합니다.
5. 이탈 위험 회원이 있으면 하단에 별도 섹션이 자동으로 나타납니다.
6. 신규 회원 테이블에서 방문 횟수와 온보딩 상태에 따라 다른 색상으로 상태를 구분합니다.

## 비즈니스 규칙

- `WI`는 방문 문의, `TI`는 전화 문의로 화면 표기를 통일합니다.
- `[신규DB]` C열 `경로`는 문의 유형이며 `WI`, `TI`, `체험`, `일일입장` 값을 사용합니다.
- `[신규DB]` D열 `상세 경로1`은 세부 유입 경로이며 회원소개 유입 비중의 기준 컬럼입니다.
- `[신규DB]` N/R/V열은 1차·2차·3차 상담 방식이며 값은 `대면`, `유선`, `부재`입니다.
- 7일 이용률은 등록 후 7일 이내에 2회 이상 출석한 신규 회원의 비율입니다.
- 30일 활동률은 등록 후 30일 이내에 4회 이상 출석한 신규 회원의 비율입니다.
- 체험 참여율과 체험 → 등록 전환율은 별도 지표입니다. 체험 참여율은 앞단 참여 비율, 체험 → 등록 전환율은 체험자 중 본상품 등록 비율입니다.
