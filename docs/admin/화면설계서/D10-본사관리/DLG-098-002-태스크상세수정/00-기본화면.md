## 화면 목적
오늘의 할 일 목록에서 기존 업무 항목의 내용을 수정하는 다이얼로그입니다.

## 주요 구성요소
- "할 일 수정" 제목
- 할 일 제목 입력란 (필수, 기존 내용 자동 로드)
- 할 일 상세 설명 입력란 (기존 내용 자동 로드)
- 우선순위 선택 (높음 / 보통 / 낮음)
- 담당자 선택 드롭다운
- 마감 시각 선택
- 완료 여부 토글 (수정 다이얼로그에서도 완료 처리 가능)
- 취소 버튼
- 저장 버튼 (변경 사항이 있을 때 활성)

## 주요 UX 흐름
1. 오늘의 할 일 목록에서 수정 버튼 클릭 → 다이얼로그 표시 (기존 내용 자동 로드)
2. 내용 수정 후 저장 버튼 클릭 → 목록에 즉시 반영, 다이얼로그 닫힘
3. 취소 → 다이얼로그 닫힘, 변경 사항 폐기

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 기존 할 일 내용이 채워진 상태 |
| 변경 전 | 저장 버튼 비활성 |
| 변경 후 | 저장 버튼 활성 |
| 저장 완료 | 다이얼로그 닫히고 목록 항목 갱신 |
