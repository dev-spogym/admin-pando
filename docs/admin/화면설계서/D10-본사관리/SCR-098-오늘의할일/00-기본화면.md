## 화면 목적
오늘 처리해야 할 업무 목록을 관리하는 화면입니다.
직원 개인별 또는 지점 공유 할 일을 등록·수정·완료 체크하여 일일 업무를 체계적으로 관리합니다.

## 주요 구성요소

### 오늘의 할 일 목록
- 할 일 항목 (제목, 우선순위, 담당자, 마감 시각, 완료 여부)
- 완료 체크박스 (클릭 시 취소선 표시 및 완료 상태로 전환)
- 우선순위 배지 (높음 / 보통 / 낮음)
- 항목별 수정 버튼 → DLG-098-002
- 항목별 삭제 버튼

### 상단 액션
- 할 일 추가 버튼 → DLG-098-001

### 진행 현황 요약
- 전체 항목 수 / 완료 항목 수 / 미완료 항목 수
- 완료율 프로그레스 바

### 필터 탭
- 전체 / 미완료 / 완료 탭 전환

## 계정별 차이
| 계정 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 | 전체 지점 할 일 목록 | 추가·수정·삭제·완료 처리 |
| 센터장 | 소속 지점 할 일 목록 | 추가·수정·삭제·완료 처리 |
| 매니저 | 자신이 담당자인 항목 | 자신 항목 완료 처리 |
| 기타 | 접근 불가 | — |

## 주요 UX 흐름
1. 화면 진입 시 오늘 날짜 기준 할 일 목록을 불러온다.
2. 할 일 추가 버튼 클릭 → DLG-098-001 표시 → 저장 후 목록에 추가.
3. 완료 체크박스 클릭 → 해당 항목 완료 처리 (취소선 + 완료 배지).
4. 수정 버튼 클릭 → DLG-098-002 표시 → 저장 후 목록 갱신.
5. 삭제 버튼 클릭 → 삭제 확인 후 목록에서 제거.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 오늘 할 일 목록 표시 |
| 전체 완료 | 모든 항목 완료 시 축하 메시지 표시 |
| 목록 없음 | 등록된 할 일이 없을 때 추가 안내 표시 |
