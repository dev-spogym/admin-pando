## 화면 목적
회원 대상 NPS(순추천지수) 설문 결과를 조회하고 분석하는 화면입니다.
회원 만족도 추이를 파악하고, 부정적 피드백을 빠르게 식별하여 서비스 개선에 활용합니다.

## 주요 구성요소

### NPS 요약 카드
- 전체 NPS 점수 (-100 ~ +100)
- 응답자 분류 비율 (추천자 / 중립자 / 비추천자)
- 전기간 대비 NPS 점수 변화

### NPS 추이 차트
- 기간별 NPS 점수 변화 선 차트
- 응답자 수 추이

### 응답 분포
- 0~10 점수별 응답 수 막대 차트
- 지점별 NPS 비교 테이블

### 응답 목록
- 설문 응답 목록 (점수, 응답일, 자유 의견, 회원 정보)
- 점수 범위 필터 (추천자 9~10 / 중립 7~8 / 비추천 0~6)
- 자유 의견 키워드 분석 (자주 등장하는 키워드 태그 클라우드)

### 설문 발송 관리
- NPS 설문 발송 주기 설정 (월 1회 / 분기 1회)
- 발송 대상 설정 (활성 회원 전체 / 특정 이용권 보유 회원)

## 계정별 차이
| 계정 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 | 전체 지점 NPS 통합 및 지점별 비교 | 설문 발송 설정, 내보내기 |
| 센터장 | 소속 지점 NPS 결과 | 설문 발송 설정 |
| 매니저 이하 | 접근 불가 | — |

## 주요 UX 흐름
1. 화면 진입 시 최근 분기 기준 NPS 결과를 불러온다.
2. 기간 필터 변경 → NPS 점수 및 차트 자동 갱신.
3. 비추천자 필터 선택 → 비추천 응답 목록만 표시하여 빠른 대응 가능.
4. 자유 의견 키워드 클릭 → 해당 키워드가 포함된 응답 목록 필터링.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | NPS 요약 및 응답 목록 표시 |
| 로딩 | 데이터 불러오는 중 (스켈레톤 표시) |
| 응답 없음 | 선택 기간에 응답 없음 안내 |
| 설문 미발송 | 아직 설문을 발송한 이력이 없을 때 설문 설정 안내 |
