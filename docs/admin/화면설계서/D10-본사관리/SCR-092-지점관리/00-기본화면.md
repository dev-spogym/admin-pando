## 화면 목적
전체 지점의 현황을 조회하고, 신규 지점을 등록하거나 기존 지점의 상태를 변경하는 화면입니다.
지점 간 회원 이동 신청 및 지점별 통합 성과 비교도 이 화면에서 처리합니다.

## 주요 구성요소

### 지점 목록
- 지점명, 지점 코드, 주소, 연락처, 회원 수, 직원 수, 운영 상태, 등록일 표시
- 행 클릭 → 지점 상세 정보 패널 또는 상세 페이지 이동

### 검색 및 필터
- 지점명 또는 주소 키워드 검색
- 운영 상태 필터 (전체 / 운영 중 / 임시휴업 / 폐점)

### 상단 액션
- 신규 지점 등록 버튼 → 신규 지점 등록 다이얼로그(DLG-092-001)

### 지점별 행 액션
- 수정 버튼 → 지점 정보 수정
- 비활성화 버튼 → 비활성화 확인 다이얼로그(DLG-092-002)

### 통합 현황 비교 탭
- 지점별 회원 수, 매출, 출석률을 기간(월간 / 주간 / 연간) 기준으로 비교

## 계정별 차이
| 계정 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 | 전체 지점 목록 | 신규 등록, 수정, 비활성화, 비교 |
| 센터장 | 소속 지점만 | 소속 지점 정보 수정 |
| 매니저 이하 | 접근 불가 | — |

## 주요 UX 흐름
1. 화면 진입 시 전체 지점 목록을 불러온다.
2. 신규 지점 등록 버튼 클릭 → DLG-092-001 표시 → 저장 후 목록에 추가.
3. 비활성화 버튼 클릭 → DLG-092-002 표시 → 확인 후 운영 상태 변경.
4. 검색어 입력 또는 필터 선택 → 조건에 맞는 지점만 목록에 표시.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 지점 목록 표시 |
| 로딩 | 데이터 불러오는 중 (스켈레톤 표시) |
| 검색 결과 없음 | 조건에 맞는 지점 없음 안내 |
| 지점 없음 | 등록된 지점 없을 때 등록 안내 |
