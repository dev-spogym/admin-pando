## 화면 목적
지점별 매출, 회원, 수업 성과를 비교 분석하는 리포트 화면입니다.
지점 간 성과 차이를 시각적으로 파악하고, 기간별 추이를 확인하여 운영 전략 수립에 활용합니다.

## 주요 구성요소

### 필터 및 기간 선택
- 조회 기간 선택 (주간 / 월간 / 분기 / 연간, 직접 날짜 입력)
- 지점 선택 (전체 또는 특정 지점 복수 선택)
- 비교 기준 선택 (전기간 대비 / 지점 간 비교)

### 매출 성과 섹션
- 지점별 총 매출 비교 차트 (막대 차트)
- 지점별 매출 순위 테이블
- 목표 달성률 표시

### 회원 성과 섹션
- 지점별 신규 회원 수, 활성 회원 수, 이탈 회원 수 비교
- 회원 유지율 차트

### 수업 성과 섹션
- 지점별 수업 운영 횟수, 예약율, 출석율 비교
- 인기 수업 유형 분석

### 내보내기
- 엑셀 다운로드 버튼
- PDF 다운로드 버튼

## 계정별 차이
| 계정 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 | 전체 지점 성과 비교 | 필터 변경, 내보내기 |
| 센터장 | 소속 지점 데이터만 | 기간 필터, 내보내기 |
| 매니저 이하 | 접근 불가 | — |

## 주요 UX 흐름
1. 화면 진입 시 현재 월 기준 전체 지점 성과 데이터를 불러온다.
2. 기간 또는 지점 필터 변경 → 차트와 테이블 자동 갱신.
3. 내보내기 버튼 클릭 → 선택한 형식(엑셀/PDF)으로 현재 필터 조건의 데이터 다운로드.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 당월 전체 지점 성과 표시 |
| 로딩 | 데이터 불러오는 중 (차트 스켈레톤 표시) |
| 데이터 없음 | 선택 기간에 데이터 없음 안내 |
| 내보내기 처리 중 | 다운로드 버튼 처리 중 표시 |
