## 화면 목적
핵심 성과 지표(KPI)의 달성 현황을 시각화하여 목표 대비 실적을 한눈에 파악하는 화면입니다.
지점별·기간별 목표 설정과 달성률을 비교하여 운영 방향 결정을 지원합니다.

## 주요 구성요소

### KPI 카드 영역
- 매출 목표 대비 달성률 (게이지 차트 + 수치)
- 신규 회원 목표 대비 달성률
- 출석률 목표 대비 달성률
- 회원 유지율 목표 대비 달성률
- 각 카드에 전월 대비 증감 표시

### 목표 달성 추이 차트
- 월별 KPI 달성률 추이 선 차트
- 목표선(100%) 기준선 표시

### 지점별 KPI 비교 테이블
- 지점명, 매출 달성률, 신규 회원 달성률, 출석률 달성률 비교
- 달성률에 따른 색상 구분 (초과 달성: 초록 / 근접: 노랑 / 미달: 빨강)

### 매출 목표 설정
- 지점별 월 매출 목표 설정 버튼 → DLG-094-001

## 계정별 차이
| 계정 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 | 전체 지점 KPI 통합 현황 | 목표 설정, 지점별 비교 |
| 센터장 | 소속 지점 KPI | 소속 지점 목표 설정 |
| 매니저 이하 | 접근 불가 | — |

## 주요 UX 흐름
1. 화면 진입 시 현재 월 기준 KPI 데이터를 불러와 표시한다.
2. 기간 필터 변경 → KPI 카드 및 추이 차트 자동 갱신.
3. 매출 목표 설정 버튼 클릭 → DLG-094-001 표시.
4. 지점별 KPI 비교 테이블에서 지점 행 클릭 → 해당 지점 상세 리포트(SCR-093)로 이동.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | KPI 카드 및 차트 정상 표시 |
| 로딩 | 데이터 불러오는 중 (스켈레톤 표시) |
| 목표 미설정 | 목표 미설정 지점에 설정 안내 배지 표시 |
| 초과 달성 | 달성률 100% 초과 시 초록 강조 표시 |
