## 화면 목적
핵심 성과 지표(KPI)의 달성 현황을 시각화하여 목표 대비 실적을 한눈에 파악하는 화면입니다.
지점별·기간별 목표 설정과 달성률을 비교하여 운영 방향 결정을 지원합니다.

## 주요 구성요소

### KPI 카드 영역
- 매출 목표 대비 달성률 (게이지 차트 + 수치)
- 신규 회원 목표 대비 달성률
- 출석률 목표 대비 달성률
- 회원 유지율 목표 대비 달성률
- GX 수업 KPI #19~#22: 이번 달 수업 건수 / 예약 수 / 출석 수 / 수업 출석률
- 강습세션수 KPI: PT / 필라테스 / 스트레칭 / 테라피별 월간·주차별 수업수
- 상담 퍼널 KPI: WI vs TI 건수 / TI 방문 전환율 / TI 등록률 / WI 등록률
- 재등록/이탈 회복 KPI: 재등록 상담 완료 건수 / 재등록 성공률
- OT / 수업운영 KPI: 강습 OT 신규 배정 / 골프 OT 신규 배정 / OT 등록 전환율 / OT 거부율 / OT 이월률 / OT 체험 → 등록률
- 각 카드에 전월 대비 증감 표시

### 상담/퍼널 세분화
- `WI`는 방문 문의, `TI`는 전화 문의로 표시합니다.
- WI vs TI 건수는 방문 문의와 전화 문의를 분리한 막대 또는 2분할 카드로 표시합니다.
- TI 방문 전환율은 전화 문의 중 실제 대면 상담으로 이어진 비율을 표시합니다.
- TI 등록률과 WI 등록률은 동일 기간 각 문의 유형 대상자 중 등록 완료 비율로 표시합니다.
- FC 보드(SCR-095)의 상담 액션 큐와 연결해 전화 문의 후 미방문, 방문 후 미등록 대상을 후속 조치 대상으로 노출합니다.

### 재등록 / 이탈 회복 KPI
- 재등록 상담 완료 건수는 `[재등록DB]` B열 `상담결과`가 입력된 건수로 표시합니다.
- 재등록 성공률은 `[재등록DB]` B열 `등록` 건수 ÷ B열 `등록+확정+예상+관리+실패` 전체 상담 건수로 표시합니다.
- 기존의 `재등록률` 미구현 KPI는 이탈 정의가 확정되기 전까지 재등록 성공률로 대체합니다.
- 만료일별 대상자 분포(D-7/D-14/D-30 버킷) 위젯은 추가하지 않고, 기존 만료 예정 30일 이내 단일 지표를 유지합니다.

### GX 세부 종목 분포
- 요가 / 필라테스 / 스피닝 / 줌바 / 에어로빅 / GX 기타 6종 분포를 차트로 표시
- 상품 마스터 `category=GX`와 2단계 `subCategory`를 우선 기준으로 집계
- 외부 시트 연동 시 `[월매출]` C열 상품/종목명과 `[주간월간보고]` 주차별 T5:U5를 상품 마스터 세부종목에 매핑
- 인기 종목은 증설 후보로, 비인기 종목은 시간표 축소 또는 폐지 후보로 표시

### OT / 수업 운영 KPI
- 강습 OT 신규 배정 건수와 골프 OT 신규 배정 건수를 별도 카드로 표시합니다.
- OT 등록 전환율은 `등록 / OT 진행` 산식으로 표시하며, PT 완료율과 합산하지 않습니다.
- OT 거부율은 `거절 / OT 진행`으로 표시하고, `거절`에는 `거부+보류`를 함께 집계했다고 툴팁에 명시합니다.
- OT 이월률은 이월된 OT 재고 규모를 표시하며, 분모가 `OT배정 인원`인지 `OT 진행`인지 확정이 필요한 항목으로 별도 주석 처리합니다.
- OT 체험 → 등록률은 `[OT관리-강습/골프]` 체험 대상 중 등록 완료 비율로 표시하며, 온보딩의 일반 체험 → 등록 전환율과 별도 카드로 분리합니다.
- 골프 OT 전체 현황은 강습 OT와 별도 섹션 또는 탭으로 표시해 골프 프로그램의 영업 파이프라인을 독립적으로 확인합니다.

### 강습 세션 유형별 운영 KPI
- PT / 필라테스 / 스트레칭 / 테라피 4종의 강습세션수를 카드 또는 누적 막대 차트로 표시합니다.
- 월간 합계와 주차별 추이를 함께 제공해 특정 종목의 운영량 쏠림과 월중 증감을 확인합니다.
- 공식 원천은 수업/출석/일정 원장에 저장된 `강습 세션 유형`입니다.
- `[주간월간보고]` 주차별 `N5(강습)` 및 달성 `G열(강습)`, `[강습업무보고]` 주간보고 `O열(수업수)`는 4종 분해 공식 원천이 아니라 합계 검산용으로만 사용합니다.
- 기존 `수업 유형(PT/GX/개인레슨/기타)` 및 GX 세부 종목 6종 분포와 별도 축으로 표시합니다.

### 목표 달성 추이 차트
- 월별 KPI 달성률 추이 선 차트
- 목표선(100%) 기준선 표시

### 지점별 KPI 비교 테이블
- 지점명, 매출 달성률, 신규 회원 달성률, 출석률 달성률 비교
- 달성률에 따른 색상 구분 (초과 달성: 초록 / 근접: 노랑 / 미달: 빨강)

### 매출 목표 설정
- 지점별 월 매출 목표 설정 버튼 → DLG-094-001

## 주요 UX 흐름
1. 화면 진입 시 현재 월 기준 KPI 데이터를 불러와 표시한다.
2. 기간 필터 변경 → KPI 카드 및 추이 차트 자동 갱신.
3. 매출 목표 설정 버튼 클릭 → DLG-094-001 표시.
4. 지점별 KPI 비교 테이블에서 지점 행 클릭 → 해당 지점 상세 리포트(SCR-093)로 이동.

## 비즈니스 규칙

- 강습 세션 유형은 `PT`, `필라테스`, `스트레칭`, `테라피` 4종으로 고정합니다. 화면에서는 `PT`를 `P.T`, `필라테스`를 `필라`로 축약 표기할 수 있습니다.
- 강습세션수는 실제 수업/출석/일정 원장의 세션 유형별 완료 수업수로 산출합니다.
- `[주간월간보고]` 주차별 시트 `N5(강습)` 및 `달성` 시트 `G열(강습)`, `[강습업무보고]` 주간보고 `O열(수업수)`는 PT·필라테스·스트레칭·테라피별 분해 원천이 아니므로, 대시보드의 4종 세부 지표와 자동으로 동일시하지 않습니다.
- GX 세부 종목은 상품 마스터 `category=GX` + `subCategory` 기준으로 요가·필라테스·스피닝·줌바·에어로빅·GX 기타 6종으로 고정 분류합니다.
- 월별 외부 원천은 `[월매출]` C열 상품/종목명이며, 주차별 외부 원천은 `[주간월간보고]` T5:U5입니다.
- GX 세부 종목 분포는 KPI #19~#22 GX 수업 지표의 보조 분석 위젯으로 표시합니다.
- 종목별 분포는 비인기 클래스 폐지와 인기 클래스 증설 의사결정에 사용합니다.
- 상담 채널 원천은 `[신규DB]` C열 `경로`입니다. `WI=방문`, `TI=전화문의`, `체험`, `일일입장`으로 해석합니다.
- `[신규DB]` D열 `상세 경로1`은 간판·인터넷·홍보물·회원소개·기타 등 세부 유입 경로로 사용합니다.
- `[신규DB]` N/R/V열은 1차·2차·3차 상담 방식이며 값은 `대면`, `유선`, `부재`로 제한합니다.
- TI 방문 전환율은 TI 문의 중 N/R/V열 중 하나 이상이 `대면`인 건을 방문 전환으로 집계합니다.
- `[주간월간보고]` 주차 시트의 R10:Z10은 WI 상세 경로별 건수, AA10:AI10은 TI 상세 경로별 건수입니다.
- `[주간월간보고]` AC14:AE14는 TI 방문 전환율, AF14:AI14는 TI 등록률, AJ14:AM14는 WI 등록률 산출 셀입니다.
- 재등록 상담 완료 건수는 `[재등록DB]` B열 `상담결과`가 `등록`, `확정`, `예상`, `관리`, `실패` 중 하나인 건수입니다.
- 재등록 성공률은 `[재등록DB]` B열 `등록` 건수 ÷ B열 `등록+확정+예상+관리+실패` 건수입니다.
- 환불 → 재결제 복귀율은 본 화면 KPI로 추가하지 않습니다.
- OT 신규 배정 건수는 `[OT관리-강습]`과 `[OT관리-골프]` `OT현황`의 `Z6(OT배정 인원)`을 기본 원천으로 사용합니다.
- `[주간월간보고]` 주차 시트의 강습 OT 현황은 `I24:R24`, 골프 OT 현황은 `I28:R28`이며, `달성` 시트 월간 취합은 강습 OT `CI:CR`, 골프 OT `CS:DB` 영역입니다.
- OT 등록 전환율은 `등록 / OT 진행`입니다. `등록`은 `OT현황` `V열 등록여부=등록` 또는 요약 `Z9`, `OT 진행`은 신규 OT 배정에서 진행중·미진행을 제외한 실제 진행 건으로 해석합니다.
- OT 거부율은 `거절 / OT 진행`이며, `거절`은 `OT현황` `V열 등록여부=거부 또는 보류`와 요약 `Z7`을 기준으로 `거부+보류`를 함께 집계합니다.
- OT 이월률은 `OT현황` `V열 등록여부=이월` 또는 요약 `Z8`을 사용합니다. 분모는 우선 `OT배정 인원`으로 기록하지만 최종 산식은 추후 확정 필요 항목입니다.
- OT 체험 → 등록률은 `OT현황` 체험 요약 `AC6(등록) / AC5(체험)`으로 산출합니다.
- 원본 제안의 `Y5/Y9/AB5/AB8` 표기는 템플릿 확인 결과 실제 값 위치와 달라, 본 화면에서는 확인된 `Z5:Z9`, `AC5:AC7` 요약 셀을 기준으로 명세합니다.
