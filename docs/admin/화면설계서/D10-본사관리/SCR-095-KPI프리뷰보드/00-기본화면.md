## 화면 목적
역할별로 맞춤화된 KPI 보드를 제공하는 종합 센터 화면입니다.
본사, 지점, FC, 트레이너, 운영 5가지 관점에서 핵심 지표, 우선 업무 큐, 성과 랭킹, 단계별 퍼널, 데이터 인사이트를 한 화면에서 확인하고 바로 실행 화면으로 이동합니다.

## 주요 구성요소

### 역할별 보드 탭
- 본사 / 지점 / FC / 트레이너 / 운영 탭을 제공합니다.
- 로그인 계정 역할에 맞는 보드를 기본 선택하되, 권한이 있으면 다른 보드도 전환할 수 있습니다.

### 핵심 지표 카드
- 각 보드별 핵심 지표 4개를 카드로 표시합니다.
- FC 보드는 신규 리드, TI 방문 전환율, WI 등록률, TI 등록률을 기본 핵심 지표로 포함합니다.
- 지점/FC 보드는 재등록 상담 완료 건수와 재등록 성공률을 보조 KPI로 포함합니다.
- 트레이너 보드는 OT 배정 합계, OT 등록 전환율, OT 체험 → 등록률, 트레이너별 OT 1/2차 완료·예정 현황을 핵심 지표 또는 보조 위젯으로 포함합니다.
- 지표 카드에는 현재값, 기준 기간, 목표 대비 상태 또는 전월 대비 증감을 함께 표시합니다.

### FC 상담 퍼널
- `WI`는 방문 문의, `TI`는 전화 문의로 표시합니다.
- TI 방문 전환율은 전화 문의 대상자 중 실제 대면 상담으로 전환된 비율입니다.
- TI 등록률은 전화 문의 대상자 중 등록 완료 비율입니다.
- WI 등록률은 방문 문의 대상자 중 등록 완료 비율입니다.
- FC 퍼널은 `리드 유입 → 첫 응답 → 대면 상담/방문 → 등록 완료` 단계로 표시합니다.

### 액션 큐
- 오늘 처리해야 할 우선순위 업무 목록을 긴급 / 우선 / 일반 / 완료가능 단계로 구분합니다.
- FC 보드에는 `전화 문의 후 미방문`, `방문 후 미등록`, `상담 결과 미기입`, `만료 예정 재등록 상담`을 우선 노출합니다.

### 랭킹 / 리스트 패널
- 본사/지점 보드는 지점 또는 팀 성과 랭킹을 표시합니다.
- FC 보드는 체험 후 미등록, 만료 예정, 장기 미방문, 상담 예약 등 오늘의 처리 리스트를 표시합니다.
- 트레이너 보드는 OT 1차 예정, OT 2차 예정, OT 완료 후 미등록, 체험 후 미등록 대상자를 리스트로 표시합니다.

### 재등록 파이프라인
- 재등록 상담 완료 건수는 `[재등록DB]` B열 `상담결과`가 입력된 대상자 수로 표시합니다.
- 재등록 성공률은 `[재등록DB]` B열 `등록` 건수 ÷ B열 `등록+확정+예상+관리+실패` 전체 상담 건수로 표시합니다.
- 만료일별 대상자 분포(D-7/D-14/D-30 버킷) 위젯은 추가하지 않고, 액션 큐에서는 `만료 예정 재등록 상담` 단일 업무로 표시합니다.

### 담당자 성과(개인 KPI) 패널
- 트레이너 개인 KPI는 OT 배정 합계, 신규 매출, 재등록 매출, OT 등록 전환율, OT 체험 → 등록률을 같은 행에서 비교합니다.
- FC 개인 KPI는 담당자별 상담 건수와 상담 → 등록 전환율을 표시해 업무량과 세일즈 전환 성과를 함께 확인합니다.
- 골프 프로 KPI는 `[매출종합]` `골프매출` 탭의 등록강사별 매출을 별도 행 또는 별도 카드로 표시합니다.
- 개인 KPI는 지점 평균과 개인 값을 함께 보여 과부하, 공백, 전환율 저하 담당자를 빠르게 식별할 수 있게 합니다.

### 트레이너 OT 보드
- OT 등록 전환율은 `등록 / OT 진행`으로 표시합니다.
- OT 체험 → 등록률은 `[OT관리-강습/골프]`의 체험 대상 중 등록 완료 비율로 표시하며, 온보딩의 일반 체험 전환율과 분리합니다.
- 트레이너별 OT 1/2차 완료·예정 위젯은 `[강습업무보고]`의 O.T 배정, O.T 1, O.T 2 컬럼과 `[OT관리-강습]`의 진행 상태를 연결해 표시합니다.
- OT 거절 건은 `거부+보류`를 함께 집계했다고 위젯 설명에 명시합니다.

### 데이터 인사이트
- 현재 지표의 원인과 다음 액션을 3개 이내의 짧은 코멘트로 제공합니다.
- 데이터가 없는 경우 샘플 문구 대신 "실데이터 연동 필요" 안내를 표시합니다.

## 계정별 차이

|---|---|---|
| 슈퍼관리자 | 전체 보드 | 모든 보드 탭 전환, 전체 지점 지표 확인 |
| 센터장 / 지점 관리자 | 소속 지점 중심 보드 | 지점/FC/운영 보드 확인 |
| FC | FC 보드 | 담당 리드와 상담 액션 큐 확인 |
| 트레이너 | 트레이너 보드 | 담당 수업/회원 KPI 확인 |
| 스태프 | 운영 보드 | 출석·미수·운영 액션 확인 |

## 주요 UX 흐름
1. 화면 진입 시 로그인 계정의 역할에 맞는 보드 탭을 기본 선택합니다.
2. 사용자가 상단 탭을 전환하면 해당 역할의 KPI 보드가 즉시 표시됩니다.
3. 핵심 지표 카드에서 현재 상태를 확인하고, 액션 큐에서 우선 처리 대상을 선택합니다.
4. FC 보드에서는 TI 방문 전환율, TI 등록률, WI 등록률을 함께 확인해 전화 응대와 방문 상담의 병목을 분리합니다.
5. 바로가기 버튼을 통해 Today Tasks, 매출, 회원, 자동알림 등 실행 화면으로 이동합니다.

## 비즈니스 규칙

- `WI`는 Walk In의 약어이며 예약 없이 방문 상담 온 사람, 화면 표기는 `방문 문의`로 통일합니다.
- `TI`는 Telephone Inquiry의 약어이며 전화 후 상담 온 사람, 화면 표기는 `전화 문의`로 통일합니다.
- 상담 방식 드롭다운 값은 `대면`, `유선`, `부재` 3개입니다.
- TI 방문 전환은 전화 문의가 실제 대면 상담으로 넘어간 병목 지표이므로 TI 등록률과 별도 카드로 분리합니다.
- 트레이너 보드의 OT 등록 전환율은 일반 PT 완료율과 합산하지 않습니다.
- `OT 진행` 산식은 대시보드 표준으로 `등록 / OT 진행`을 사용하되, 주간월간보고의 기존 등록률 셀이 `등록 / 신규` 구조로 해석될 수 있어 최종 연동 전 사용자 확인이 필요합니다.
- OT 이월률의 분모는 우선 `OT배정 인원`으로 문서화하지만, 운영 리포트에서 `OT 진행` 기준과 분리해 확정해야 합니다.
- 트레이너별 OT 배정 합계는 `[강습업무보고]` 주간보고 E열 `O.T 배정`을 기준으로 하며, OT 배정 편중과 공백을 확인하는 업무 부하 지표입니다.
- 트레이너별 신규/재등록 매출은 `[매출종합]` 거래 단위를 공식 원천으로 사용하고, `[강습업무보고]` 주간보고 J:K열은 검산 또는 운영 보조 원천으로만 사용합니다.
- FC별 상담 건수는 `[신규DB]` K열 담당자별 전체 상담 건수로 산출하며, A열 `등록`, `관리`, `실패`를 모두 포함합니다.
- FC별 상담 → 등록 전환율은 A열 `등록` 건수 ÷ A열 `등록+관리+실패` 전체 상담 건수로 산출합니다.
- 골프프로별 매출은 강습 트레이너 KPI와 별도 섹션으로 표시해 골프 담당자 KPI 공백을 보완합니다.
- 환불 → 재결제 복귀율은 KPI 센터에 추가하지 않습니다.
