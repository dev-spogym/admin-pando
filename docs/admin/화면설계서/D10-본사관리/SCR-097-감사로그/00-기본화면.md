## 화면 목적
시스템 내에서 발생한 주요 작업 이력을 기록하고 조회하는 화면입니다.
누가, 언제, 무엇을 했는지 추적하여 보안 감사, 문제 발생 시 원인 파악, 운영 투명성 확보에 활용합니다.

## 주요 구성요소

### 로그 목록
- 작업 일시, 작업자(이름·계정), 지점명, 작업 유형, 작업 대상, 작업 상세 내용 표시
- 각 행 클릭 시 상세 내용 펼침 또는 상세 패널 표시

### 검색 및 필터
- 작업자 이름 또는 계정 검색
- 지점 선택 필터
- 작업 유형 필터 (로그인 / 회원 관리 / 결제 / 설정 변경 / 권한 변경 / 데이터 삭제 등)
- 조회 기간 선택 (날짜 범위 직접 입력)

### 내보내기
- 엑셀 다운로드 버튼 (현재 필터 조건 기준)

## 계정별 차이
| 계정 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 | 전체 지점 로그 | 필터 변경, 내보내기 |
| 센터장 | 소속 지점 로그 | 필터 변경, 내보내기 |
| 매니저 이하 | 접근 불가 | — |

## 주요 UX 흐름
1. 화면 진입 시 최근 7일 기준 전체 로그를 최신 순으로 표시한다.
2. 필터 조건 변경 → 로그 목록 자동 갱신.
3. 행 클릭 → 작업 상세 내용(변경 전·후 값 포함) 펼침 표시.
4. 내보내기 버튼 클릭 → 현재 필터 조건의 로그를 엑셀로 다운로드.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 최근 7일 로그 표시 |
| 로딩 | 로그 불러오는 중 (스켈레톤 표시) |
| 검색 결과 없음 | 조건에 맞는 로그 없음 안내 |
| 내보내기 처리 중 | 다운로드 버튼 처리 중 표시 |
