## 화면 목적
원하는 기간과 항목을 선택하여 운영 보고서를 생성하고 엑셀 또는 PDF로 내보내는 화면입니다.
정기 보고, 경영진 보고, 고객 제출용 등 다양한 목적의 리포트를 맞춤 생성합니다.

## 주요 구성요소

### 리포트 설정 패널
- 리포트 유형 선택 (매출 리포트 / 회원 현황 리포트 / 출석 리포트 / 수업 리포트 / 종합 리포트)
- 조회 기간 선택 (날짜 범위 직접 입력 또는 빠른 선택: 이번 주 / 이번 달 / 이번 분기 / 이번 해)
- 대상 지점 선택 (전체 또는 특정 지점 복수 선택)
- 포함 항목 선택 (체크박스로 세부 항목 선택)
- 리포트 생성 버튼

### 리포트 미리보기
- 생성된 리포트 내용을 화면에서 미리 확인
- 페이지 네비게이션 (다중 페이지 리포트)

### 내보내기
- 엑셀 다운로드 버튼
- PDF 다운로드 버튼
- 인쇄 버튼

### 생성 이력
- 최근 생성한 리포트 목록 (날짜, 유형, 생성자, 다운로드 버튼)

## 계정별 차이
| 계정 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 | 전체 지점 리포트 | 모든 유형 생성 및 내보내기 |
| 센터장 | 소속 지점 리포트 | 소속 지점 데이터 리포트 생성 |
| 매니저 이하 | 접근 불가 | — |

## 주요 UX 흐름
1. 리포트 유형, 기간, 지점, 항목을 선택한다.
2. 리포트 생성 버튼 클릭 → 생성 중 처리 표시 → 완료 후 미리보기 표시.
3. 미리보기 확인 후 엑셀 또는 PDF 다운로드 버튼 클릭.
4. 생성 이력 목록에서 과거 리포트를 다시 다운로드할 수 있다.

## 화면 상태
| 상태 | 설명 |
|---|---|
| 초기 | 리포트 설정 패널만 표시, 미리보기 비어있음 |
| 생성 중 | 리포트 생성 처리 중 (처리 중 표시) |
| 생성 완료 | 미리보기 표시, 내보내기 버튼 활성 |
| 내보내기 처리 중 | 다운로드 버튼 처리 중 표시 |
| 데이터 없음 | 선택 조건에 해당하는 데이터 없음 안내 |
