## 목적

동일 회원에 대해 동일 상품이 단기간 내 재결제될 경우 직원에게 경고를 표시합니다. 실수로 인한 이중 결제를 방지하며, 직원이 의도적인 재구매임을 확인한 경우에만 결제를 계속 진행할 수 있습니다.

## 구성요소

| 구성요소 | 내용 |
|---|---|
| 경고 메시지 | "동일 회원에게 동일 상품이 최근 결제된 이력이 있습니다" 안내 |
| 기존 결제 정보 | 중복 감지된 이전 결제의 날짜, 상품명, 금액 |
| 계속 진행 버튼 | 중복임을 인지하고 결제를 그대로 진행 |
| 취소 버튼 | 결제를 중단하고 POS 화면으로 복귀 |

## 언제 표시되나요

- DLG-S003(결제 확인)에서 결제 확인 버튼을 눌렀을 때, 시스템이 동일 회원·동일 상품의 최근 결제 이력을 감지한 경우
- 중복 기준 기간은 시스템 설정에 따라 다를 수 있음 (예: 최근 30일 이내)

## 주요 흐름

1. DLG-S003 결제 확인 버튼 클릭 후 시스템이 중복 이력을 감지합니다.
2. 중복 결제 경고 다이얼로그가 표시됩니다.
3. 직원이 기존 결제 정보를 확인합니다.
4. 의도적인 재구매라면 계속 진행 버튼을 눌러 결제를 완료합니다.
5. 실수라면 취소 버튼을 눌러 POS 화면으로 돌아갑니다.
