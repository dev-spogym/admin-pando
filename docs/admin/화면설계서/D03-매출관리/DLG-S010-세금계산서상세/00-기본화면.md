## 목적

발행된 세금계산서의 상세 내용을 조회합니다. 공급자·공급받는자 정보, 공급 금액, 세액 등 발행된 세금계산서의 전체 내용을 확인하고 필요 시 재발송 또는 취소 처리를 진행합니다.

## 구성요소

| 구성요소 | 내용 |
|---|---|
| 세금계산서 번호 | 발행된 세금계산서의 고유 번호 |
| 발행 일시 | 세금계산서가 발행된 날짜 및 시각 |
| 발행 상태 | 발행 완료, 전송 완료, 취소 등 현재 상태 |
| 공급자 정보 | 자사 사업자번호, 상호, 대표자명, 주소 |
| 공급받는자 정보 | 거래처 사업자번호, 상호, 담당자, 이메일 |
| 공급 내역 | 상품명, 수량, 단가, 공급가액 목록 |
| 공급가액 합계 | 부가세 제외 금액 |
| 부가세액 | 공급가액의 10% |
| 합계 금액 | 공급가액 + 부가세 |
| 재발송 버튼 | 등록된 이메일로 세금계산서 재발송 |
| 닫기 버튼 | 다이얼로그를 닫고 이전 화면으로 복귀 |

## 언제 표시되나요

- SCR-S010(세금계산서 발행) 목록에서 발행 완료 건을 클릭했을 때
- 매출 상세에서 세금계산서 발행 이력을 선택했을 때

## 주요 흐름

1. 세금계산서 발행 목록에서 확인할 건을 선택합니다.
2. 세금계산서 상세 다이얼로그가 열리며 전체 내용이 표시됩니다.
3. 발행 정보를 확인합니다.
4. 재발송이 필요한 경우 재발송 버튼을 눌러 이메일을 다시 전송합니다.
5. 닫기 버튼을 눌러 이전 화면으로 돌아갑니다.

## 관련 다이어그램

```

        │ 재발송 클릭    │ 닫기
        ▼               ▼
```
