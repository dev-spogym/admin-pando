## 화면 목적

회원이 선납한 금액 중 아직 서비스가 제공되지 않은 금액(선수익금)의 현황을 조회하는 화면입니다. 총 선수익금·이미 인식된 금액·아직 인식되지 않은 잔여 금액·이번 달 인식 예정 금액을 파악하고, 계약별 진행률을 확인합니다. 회계 관점에서 매출이 실제로 얼마나 실현됐는지 관리할 때 활용합니다.

## 주요 구성요소

- **요약 지표 카드 (4개)**: 총 선수익금 / 인식 완료 금액 / 잔여 선수익금 / 이번 달 인식 예정 금액을 상단에 표시합니다.
- **기간 필터**: 시작일~종료일을 입력해 조회 범위를 설정합니다. 기본값은 이번 달 전체입니다.
- **선수익금 목록 테이블**: 회원명 / 상품명 / 총액 / 인식 완료 / 잔여 / 시작일 / 종료일 / 진행률 컬럼을 제공합니다.
  - 진행률 컬럼: 프로그레스 바와 숫자(%)를 함께 표시합니다. 진행률이 높을수록(서비스 제공 완료에 가까울수록) 주황~빨강 계열로 색이 변합니다.
  - 인식 완료 금액: 초록색으로 강조 표시합니다.
  - 잔여 금액: 주요 색상(포인트 컬러)으로 굵게 표시합니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 본사 최고관리자 | 전체 지점 매출 조회·분석 가능 | 기간 필터 설정, 전체 선수익금 조회 |
| 센터장 / 매니저 | 소속 지점 매출 전체 | 기간 필터 설정, 지점 선수익금 조회 |
| FC | 담당 회원 결제 처리 가능 | 조회만 가능 |
| 트레이너 / 스태프 | 매출 조회만 (처리 불가) | 조회만 가능 |

## 주요 UX 흐름

1. 사이드바에서 "매출관리 > 선수익금 조회"를 클릭해 진입합니다.
2. 상단 지표 카드로 전체 선수익금 현황을 파악합니다.
3. 기간 필터를 설정해 원하는 계약 시작일 범위로 좁힙니다.
4. 테이블에서 계약별 인식 현황과 진행률을 확인합니다.
5. 잔여 금액이 큰 계약이나 이번 달 종료 예정 계약에 우선적으로 주목합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 로딩 중 | 데이터를 불러오는 동안 테이블 영역에 로딩 상태를 표시합니다. |
| 정상 표시 | 선수익금 데이터가 있을 때 지표 카드와 테이블을 표시합니다. |
| 데이터 없음 | 해당 기간에 선수익금 계약이 없을 때 빈 테이블 안내를 표시합니다. |
| 오류 | 데이터 조회 실패 시 오류 메시지 알림(토스트)을 표시합니다. |

## 관련 다이어그램

- 선수익금 인식 진행률 계산 흐름 (서비스 기간 대비 경과 일수)
- 이번 달 인식 예정 금액 산출 흐름
- 기간 필터 적용 흐름
- 계정 권한별 접근 분기 흐름
