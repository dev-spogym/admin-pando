## 목적

월별 또는 특정 기간에 대한 매출 목표 금액을 설정합니다. 설정된 목표는 매출 현황 및 통계 화면에서 달성률을 비교하는 기준으로 활용됩니다.

## 구성요소

| 구성요소 | 내용 |
|---|---|
| 설정 기간 선택 | 목표를 설정할 월 또는 기간 선택 (월 단위 또는 날짜 범위) |
| 목표 금액 입력 | 해당 기간의 매출 목표 금액 입력 |
| 기준 유형 선택 | 전체 매출 목표 또는 상품 카테고리별 목표 구분 |
| 이전 기간 실적 | 동일 기간의 전년도 또는 전월 실제 매출액 (참고용) |
| 저장 버튼 | 설정한 목표를 저장 |
| 취소 버튼 | 설정을 취소하고 이전 화면으로 복귀 |

## 언제 표시되나요

- SCR-S001(매출 현황) 또는 SCR-S004(매출 통계) 화면에서 목표 설정 버튼을 클릭했을 때
- 이미 목표가 설정된 경우 기존 값이 입력창에 표시되며 수정 가능

## 주요 흐름

1. 매출 현황 또는 통계 화면에서 목표 설정 버튼을 클릭합니다.
2. 목표 매출 설정 다이얼로그가 열립니다.
3. 설정 기간과 기준 유형을 선택합니다.
4. 이전 기간 실적을 참고하여 목표 금액을 입력합니다.
5. 저장 버튼을 눌러 목표를 저장합니다.
6. 저장 완료 후 매출 현황 화면에서 목표 대비 달성률이 표시됩니다.

## 관련 다이어그램

```
[SCR-S001 매출현황] 또는 [SCR-S004 매출통계]
        │ 목표 설정 클릭
        ▼
[DLG-S012 목표 매출 설정]
        │              │
        │ 취소          │ 저장
        ▼              ▼
[이전 화면]    [이전 화면] (목표 달성률 갱신)
```
