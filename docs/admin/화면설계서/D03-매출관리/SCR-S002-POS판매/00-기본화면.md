## 화면 목적

센터 현장에서 직원이 직접 상품을 선택하고 구매자를 지정해 결제 화면으로 넘기는 현장 판매(POS) 화면입니다. 카운터에서 회원이 방문했을 때 이용권·PT·GX 등 다양한 상품을 빠르게 장바구니에 담고, 현금가·카드가를 선택한 뒤 결제 화면(SCR-S003)으로 이동합니다.

## 주요 구성요소

- **카테고리 탭**: 이용권 / PT / GX / 기타 탭을 전환하며 해당 카테고리의 상품만 표시합니다. 각 탭에는 소속 상품 수가 배지로 표시됩니다.
- **상품 검색**: 상품명으로 실시간 검색해 원하는 상품을 빠르게 찾을 수 있습니다.
- **상품 카드 그리드**: 카테고리별 상품을 카드 형태로 나열합니다. 각 카드에는 상품 유형 배지(회원권·수강권·대여권·일반), 상품명, 기간, 횟수, 현금가, 카드가가 표시됩니다. 클릭하면 장바구니에 추가됩니다.
- **장바구니 패널 (우측)**: 담긴 상품 목록, 수량 조절(-/+), 현금/카드 가격 유형 전환, 개별 상품 삭제, 전체 삭제 버튼이 있습니다. 장바구니가 비어있으면 빈 상태 안내를 표시합니다.
- **구매자 선택**: 장바구니 위에 위치하며, 회원 검색 버튼을 눌러 회원을 지정하거나 비워두고 비회원으로 진행할 수 있습니다. 선택된 회원은 이름·전화번호로 표시되며 X 버튼으로 해제할 수 있습니다.
- **구매자 검색 모달**: 이름 또는 전화번호로 회원을 검색해 선택합니다. 결과가 없으면 "검색 결과가 없습니다" 안내를 표시합니다.
- **총 합계 및 결제하기 버튼**: 장바구니 하단에 총 결제 금액을 표시하고, 장바구니에 상품이 있을 때만 "결제하기" 버튼이 활성화됩니다. 클릭하면 장바구니·구매자 정보를 가지고 결제 처리 화면(SCR-S003)으로 이동합니다.
- **상품 등록 버튼**: 상품 등록 화면으로 바로 이동할 수 있는 단축 버튼입니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 본사 최고관리자 | 전체 지점 매출 조회·분석 가능 | 상품 선택, 장바구니 구성, 구매자 지정, 결제 이동, 상품 등록 |
| 센터장 / 매니저 | 소속 지점 매출 전체 | 상품 선택, 장바구니 구성, 구매자 지정, 결제 이동, 상품 등록 |
| FC | 담당 회원 결제 처리 가능 | 상품 선택, 장바구니 구성, 구매자 지정, 결제 이동 |
| 트레이너 / 스태프 | 매출 조회만 (처리 불가) | POS 화면 접근 불가 또는 조회만 |

## 주요 UX 흐름

1. 사이드바에서 "매출관리 > POS 판매"를 클릭하거나, 매출 현황에서 "신규 결제(POS)" 버튼을 클릭해 진입합니다.
2. 카테고리 탭(이용권/PT/GX/기타)을 선택하거나 상품명을 검색해 원하는 상품을 찾습니다.
3. 상품 카드를 클릭하면 장바구니에 추가됩니다. 동일 상품을 다시 클릭하면 수량이 1 증가합니다.
4. 장바구니에서 수량 조절(-/+), 현금/카드 가격 전환, 개별 삭제를 통해 구성을 조정합니다.
5. 구매자 선택 영역에서 "회원 검색"을 눌러 회원을 지정합니다. (마일리지 결제 시 필수, 그 외 선택)
6. 총 합계를 확인하고 "결제하기" 버튼을 눌러 결제 처리 화면(SCR-S003)으로 이동합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 로딩 중 | 상품 목록을 불러오는 동안 상품 그리드 영역에 스켈레톤을 표시합니다. |
| 상품 정상 표시 | 카테고리에 상품이 있을 때 카드 그리드를 표시합니다. |
| 상품 없음 | 해당 카테고리에 상품이 없을 때 빈 상태 안내를 표시합니다. |
| 품절 상품 | 재고가 0인 상품은 흐리게(투명도 낮춤) 표시되고 클릭이 불가합니다. "품절" 배지가 표시됩니다. |
| 재고 부족 | 재고가 5개 이하인 상품에 "재고 부족 (N개)" 경고 배지를 표시합니다. |
| 장바구니 비어있음 | 담긴 상품이 없을 때 장바구니 아이콘과 안내 문구를 표시하고, 결제 버튼은 비활성화됩니다. |
| 장바구니 구성 중 | 1개 이상의 상품이 담겨있을 때 목록과 합계를 표시하고, 결제 버튼이 활성화됩니다. |
| 구매자 미선택 | 구매자가 지정되지 않은 상태. 비회원으로 결제 진행이 가능합니다. |
| 구매자 선택됨 | 회원 이름·전화번호가 표시됩니다. X 버튼으로 해제할 수 있습니다. |

## 관련 다이어그램

- 상품 카테고리 탭 전환 흐름
- 장바구니 추가/수량 조절/삭제 흐름
- 구매자 검색 모달 흐름
- POS → 결제 처리 화면 이동 흐름
- 계정 권한별 접근 분기 흐름
