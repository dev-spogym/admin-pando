> **퍼블리싱 현황**: 퍼블리싱 완료 / 데이터 미연동. 현재 UI는 목업·임시 데이터 기준이며 실데이터 연동은 후속 단계입니다.

## 화면 목적

회원이 분할 납부 방식으로 결제한 할부 계약의 전체 현황을 한눈에 파악하고, 매월 납입 현황을 추적하며 미납된 회차를 관리하는 화면입니다. 운영자는 이 화면을 통해 어느 회원이 몇 회차를 납부했고 남은 금액이 얼마인지 즉시 확인하고, 미납 회차에 대한 후속 조치를 취할 수 있습니다.

## 주요 구성요소

- **요약 지표 카드 (4개)**: 진행 중인 할부 계약 수 / 이번 달 납입 예정 건수 / 이번 달 납입 완료 건수 / 미납 총액을 상단에 표시합니다.
- **상태 탭**: 전체 / 진행중 / 완납 / 미납 탭으로 계약 목록을 구분합니다.
- **검색 및 필터**: 회원명 검색, 기간 범위 필터를 제공합니다.
- **할부 계약 목록 테이블**: 회원명 / 상품명 / 총 할부 금액 / 납입 완료 금액 / 잔여 금액 / 총 회차 / 납입 회차 / 다음 납입일 / 상태 컬럼을 제공합니다.
- **회차별 납입 현황**: 계약 행을 클릭하거나 펼치면 회차별(1회·2회·3회 등) 납입 예정일·납입 완료일·금액·상태를 상세히 확인할 수 있습니다.
- **미납 관리**: 납입 기한이 지난 미납 회차에 경고 표시를 하고, 납입 처리 버튼으로 상태를 업데이트할 수 있습니다.
- **엑셀 내보내기**: 현재 필터가 적용된 할부 계약 목록을 엑셀 파일로 내려받습니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 본사 최고관리자 | 전체 지점 매출 조회·분석 가능 | 전체 할부 계약 조회, 납입 처리, 엑셀 내보내기 |
| 센터장 / 매니저 | 소속 지점 매출 전체 | 지점 할부 계약 조회, 납입 처리, 엑셀 내보내기 |
| FC | 담당 회원 결제 처리 가능 | 담당 회원 할부 계약 조회, 납입 처리 |
| 트레이너 / 스태프 | 매출 조회만 (처리 불가) | 조회만 가능 |

## 주요 UX 흐름

1. 사이드바에서 "매출관리 > 할부결제 관리"를 클릭해 진입합니다.
2. 상단 지표 카드로 이번 달 납입 예정·완료·미납 현황을 파악합니다.
3. 상태 탭을 선택해 진행중·완납·미납 계약을 구분해 봅니다.
4. 회원명 검색으로 특정 회원의 할부 계약을 찾습니다.
5. 계약 행을 선택해 회차별 납입 현황 상세를 확인합니다.
6. 미납 회차에 납입 처리 버튼으로 상태를 업데이트합니다.
7. 엑셀 버튼으로 현재 목록을 파일로 저장합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 로딩 중 | 데이터를 불러오는 동안 테이블 영역에 로딩 상태를 표시합니다. |
| 정상 표시 | 할부 계약이 있을 때 지표 카드와 테이블을 표시합니다. |
| 검색 결과 없음 | 회원명 검색 후 결과가 없을 때 안내 문구를 표시합니다. |
| 탭 내 데이터 없음 | 해당 상태의 할부 계약이 없을 때 빈 테이블 안내를 표시합니다. |
| 회차 상세 펼침 | 계약 행 선택 시 하단에 회차별 납입 현황이 펼쳐집니다. |
| 미납 경고 | 납입 기한을 초과한 회차에 경고 색상과 아이콘을 표시합니다. |

## 관련 다이어그램

- 할부 계약 상태(진행중·완납·미납) 분기 흐름
- 회차별 납입 현황 펼침 흐름
- 미납 감지 및 경고 표시 흐름
- 납입 처리 상태 변경 흐름
- 계정 권한별 접근 분기 흐름
