## 목적

특정 매출 건의 상세 정보를 한 화면에서 확인합니다. 결제 내역, 결제 수단, 구매 회원 정보를 종합적으로 조회하여 고객 문의 응대 및 내부 정산 확인에 활용합니다.

## 구성요소

| 구성요소 | 내용 |
|---|---|
| 매출 번호 | 시스템이 자동 부여한 고유 식별 번호 |
| 결제일시 | 결제가 완료된 날짜 및 시각 |
| 결제 금액 | 최종 결제된 총 금액 |
| 할인 금액 | 적용된 할인 내역 및 금액 |
| 결제 수단 | 카드, 현금, 계좌이체 등 결제 방법 |
| 상품 정보 | 구매한 상품명, 수량, 단가 목록 |
| 구매 회원 | 구매자 이름, 연락처, 회원 등급 |
| 처리 직원 | 결제를 처리한 직원 이름 |
| 메모 | 해당 매출 건에 등록된 관리자 메모 |
| 닫기 버튼 | 다이얼로그를 닫고 이전 화면으로 복귀 |

## 언제 표시되나요

- 매출 현황 목록에서 특정 매출 행을 클릭했을 때
- 회원 상세 화면의 결제 이력에서 특정 건을 선택했을 때
- 검색 결과 목록에서 매출 건을 선택했을 때

## 주요 흐름

1. 사용자가 매출 목록에서 조회할 건을 선택합니다.
2. 다이얼로그가 열리며 해당 매출의 상세 정보가 표시됩니다.
3. 필요한 정보를 확인합니다.
4. 메모 편집이 필요한 경우 메모 영역을 선택하여 DLG-S005(메모 편집)로 이동합니다.
5. 닫기 버튼을 눌러 이전 화면으로 돌아갑니다.
