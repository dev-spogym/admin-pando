## 목적

할부 결제 건의 전체 납입 현황을 회차별로 확인합니다. 납입 완료 회차, 미납 회차, 다음 납입 예정일 등을 한눈에 파악하여 회원 상담 및 미납 관리에 활용합니다.

## 구성요소

| 구성요소 | 내용 |
|---|---|
| 할부 번호 | 시스템이 부여한 할부 계약 고유 번호 |
| 회원 정보 | 할부 계약자 이름, 연락처 |
| 상품 정보 | 할부 구매한 상품명 |
| 총 할부 금액 | 전체 할부 계약 금액 |
| 할부 기간 | 총 납입 회차 수 (예: 6개월, 12개월) |
| 월 납입액 | 회차당 납입해야 하는 금액 |
| 납입 회차 목록 | 회차별 납입 예정일, 납입 금액, 납입 상태(완료/미납/예정) |
| 납입 합계 | 현재까지 납입된 총 금액 및 잔여 금액 |
| 납입 처리 버튼 | 미납 회차 선택 후 납입 처리 실행 |
| 닫기 버튼 | 다이얼로그를 닫고 이전 화면으로 복귀 |

## 언제 표시되나요

- SCR-S009(할부 결제 관리) 목록에서 특정 할부 건을 클릭했을 때
- 회원 상세 화면의 할부 이력에서 특정 건을 선택했을 때

## 주요 흐름

1. 할부 결제 관리 목록에서 조회할 건을 선택합니다.
2. 할부 상세 다이얼로그가 열리며 회차별 납입 현황이 표시됩니다.
3. 미납 회차를 확인합니다.
4. 납입 처리가 필요한 경우 해당 회차 선택 후 납입 처리 버튼을 눌러 DLG-S008(납입 처리)로 이동합니다.
5. 닫기 버튼을 눌러 이전 화면으로 돌아갑니다.

## 관련 다이어그램

```
[SCR-S009 할부결제관리 목록]
        │ 할부 건 클릭
        ▼
[DLG-S007 할부 상세]
        │ 납입 처리 클릭
        ▼
[DLG-S008 납입 처리]
        │ 완료
        ▼
[DLG-S007 할부 상세] (납입 상태 갱신)
```
