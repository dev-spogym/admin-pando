## 목적

새로운 할부 결제 계약을 등록합니다. 할부 기간, 월 납입액, 첫 납입일을 설정하여 회원이 상품 대금을 분할 납입할 수 있는 계획을 수립합니다.

## 구성요소

| 구성요소 | 내용 |
|---|---|
| 회원 정보 | 할부 계약 대상 회원 이름, 연락처 (읽기 전용) |
| 상품 정보 | 할부 구매할 상품명, 정가 (읽기 전용) |
| 총 할부 금액 | 할부로 납입할 전체 금액 입력 |
| 할부 기간 | 납입 회차 수 선택 (예: 3, 6, 12개월) |
| 월 납입액 | 총 금액 ÷ 기간으로 자동 계산되어 표시 |
| 첫 납입일 | 첫 번째 납입 예정일 선택 |
| 납입 일정 미리보기 | 회차별 납입 예정일 및 금액 목록 |
| 등록 버튼 | 설정한 내용으로 할부 계약 등록 |
| 취소 버튼 | 등록을 취소하고 이전 화면으로 복귀 |

## 언제 표시되나요

- SCR-S002(POS 판매) 또는 SCR-S009(할부 결제 관리)에서 할부 등록 버튼을 클릭했을 때
- 결제 수단으로 할부를 선택했을 때

## 주요 흐름

1. POS 판매 화면 또는 할부 결제 관리 화면에서 할부 등록을 선택합니다.
2. 할부 등록 다이얼로그가 열립니다.
3. 총 할부 금액, 할부 기간, 첫 납입일을 입력합니다.
4. 월 납입액과 납입 일정 미리보기가 자동으로 계산되어 표시됩니다.
5. 내용을 확인하고 등록 버튼을 눌러 할부 계약을 생성합니다.
6. 등록 완료 후 SCR-S009(할부 결제 관리) 목록에서 확인할 수 있습니다.

## 관련 다이어그램

```
[SCR-S002 POS 판매] 또는 [SCR-S009 할부결제관리]
        │ 할부 등록 클릭
        ▼
[DLG-S009 할부 등록]
        │              │
        │ 취소          │ 등록
        ▼              ▼
[이전 화면]    [SCR-S009 할부결제관리] (신규 계약 추가)
```
