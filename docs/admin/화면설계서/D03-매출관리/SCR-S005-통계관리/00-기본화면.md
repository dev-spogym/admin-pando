## 화면 목적

수업·매출·예약·이용 현황을 하나의 화면에서 통합 조회하고, 원하는 기간·유형·담당자·구분·회원 상태로 필터링해 CSV 파일로 내려받는 통계 보고서 화면입니다. 매출현황의 `받을금액`은 VAT 제외 순매출로 명확히 정의해 재무 보고와 수익성 분석에 활용합니다. 레거시 업무 시스템 스타일로 설계되어, 기존 PC 기반 센터 관리 프로그램에 익숙한 운영자도 빠르게 활용할 수 있습니다.

## 주요 구성요소

- **기간 입력**: 시작일~종료일을 직접 입력해 조회 범위를 설정합니다. 기본값은 이번 달 전체입니다.
- **필터 드롭다운 (4종)**: 수업 유형 / 담당자 / 구분 / 회원 상태 필터를 선택합니다. 각 드롭다운의 선택지는 현재 조회된 데이터에서 동적으로 생성됩니다.
- **현황 탭 (4개)**: 수업현황 / 매출현황 / 예약현황 / 이용현황을 탭으로 전환합니다. 탭에 따라 동일한 데이터에서 해당 유형의 행만 걸러 보여줍니다.
- **통계 테이블**: 일자 / 담당자 / 회원명 / 성별 / 휴대폰 / 구분 / 판매금액 / 할인금액 / 받을금액(VAT 제외 순매출) / 카드 / 현금 / 상품권 / 포인트 / 미수발생 / 미수입금 / 받은금액 / 비고 컬럼을 제공합니다.
- **VAT제외 매출 매핑**: `[월매출]` I열 `ARRAYFORMULA` 산출값을 `받을금액`에 매핑하며, 부가세를 제외한 순매출 기준으로 표시합니다.
- **합계 행**: 테이블 하단에 전체 건수와 각 금액 컬럼의 합계를 노란색 배경으로 표시합니다.
- **조회 버튼**: 설정한 기간과 필터로 데이터를 불러옵니다.
- **신규 버튼**: 모든 필터와 조회 결과를 초기화합니다.
- **엑셀(CSV) 내보내기**: 현재 필터가 적용된 데이터를 CSV 파일로 내려받습니다. 파일명에 탭명과 기간이 포함됩니다.
- **종료 버튼**: 이전 화면으로 돌아갑니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 본사 최고관리자 | 전체 지점 매출 조회·분석 가능 | 기간 조회, 모든 필터, 탭 전환, CSV 내보내기 |
| 센터장 / 매니저 | 소속 지점 매출 전체 | 기간 조회, 모든 필터, 탭 전환, CSV 내보내기 |
| FC | 담당 회원 결제 처리 가능 | 기간 조회, 필터, 탭 전환 |
| 트레이너 / 스태프 | 매출 조회만 (처리 불가) | 조회만 가능 |

## 주요 UX 흐름

1. 사이드바에서 "매출관리 > 통계 관리"를 클릭해 진입합니다.
2. 시작일~종료일을 입력하거나 기본값(이번 달)을 그대로 사용합니다.
3. 수업 유형·담당자·구분·회원 상태 드롭다운으로 원하는 조건을 설정합니다.
4. "조회" 버튼을 눌러 데이터를 불러옵니다.
5. 수업현황·매출현황·예약현황·이용현황 탭을 전환하며 원하는 뷰를 확인합니다.
6. 하단 합계 행에서 건수와 VAT 제외 순매출을 포함한 금액 총계를 확인합니다.
7. "엑셀" 버튼으로 현재 결과를 CSV 파일로 내려받습니다.
8. 조건을 초기화하려면 "신규" 버튼을 누릅니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 초기 상태 | 이번 달 기간이 기본 설정되어 있으며, 조회 결과가 없는 빈 테이블을 표시합니다. |
| 로딩 중 | 조회 중에 테이블 영역에 로딩 표시를 합니다. 합계 행에 "조회중"이 표시됩니다. |
| 정상 표시 | 데이터가 있을 때 테이블 행과 합계 행을 함께 표시합니다. |
| 데이터 없음 | 조건에 맞는 데이터가 없을 때 빈 행 12개로 테이블 공간을 유지합니다. |
| CSV 내보내기 불가 | 내보낼 데이터가 없을 때 "내보낼 데이터가 없습니다" 알림을 표시합니다. |

## 비즈니스 규칙

- 받을금액은 VAT 제외 순매출 기준이며, 원천 데이터는 `[월매출]` I열 `ARRAYFORMULA` 값을 사용합니다.
- 받을금액 합계는 재무 보고·수익성 분석용 순매출 합계로 해석합니다.
- VAT 포함 결제액만 존재하는 경우에는 공급가액 기준으로 환산한 값을 받을금액에 반영합니다.

## 관련 다이어그램

- 현황 탭별 데이터 필터링 분기 흐름 (수업·매출·예약·이용)
- 기간 및 드롭다운 필터 적용 흐름
- CSV 내보내기 흐름
- 계정 권한별 접근 분기 흐름
