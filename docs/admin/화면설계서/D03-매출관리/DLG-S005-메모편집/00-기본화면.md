## 목적

특정 매출 건에 관리자 메모를 추가하거나 기존 메모를 수정합니다. 고객 요청 사항, 특이 사항, 내부 공유 정보를 매출 건에 직접 기록하여 담당자 간 정보를 공유합니다.

## 구성요소

| 구성요소 | 내용 |
|---|---|
| 대상 매출 정보 | 메모를 작성할 매출 번호, 날짜, 상품명 (읽기 전용) |
| 메모 입력창 | 자유 형식 텍스트 입력 영역 (여러 줄 입력 가능) |
| 글자 수 표시 | 현재 입력된 글자 수 / 최대 허용 글자 수 |
| 저장 버튼 | 입력한 메모를 해당 매출 건에 저장 |
| 취소 버튼 | 변경 사항을 저장하지 않고 이전 화면으로 복귀 |

## 언제 표시되나요

- DLG-S001(매출 상세)에서 메모 편집 버튼을 클릭했을 때
- 매출 목록에서 메모 편집 액션을 선택했을 때

## 주요 흐름

1. 매출 상세 또는 목록에서 메모 편집을 선택합니다.
2. 메모 편집 다이얼로그가 열리며 기존 메모가 입력창에 표시됩니다.
3. 메모를 작성하거나 기존 내용을 수정합니다.
4. 저장 버튼을 눌러 메모를 저장합니다.
5. 저장 완료 후 이전 화면(매출 상세 또는 목록)으로 자동 복귀합니다.
6. 취소 버튼을 누르면 변경 없이 이전 화면으로 돌아갑니다.

## 관련 다이어그램

```
[DLG-S001 매출 상세] 또는 [SCR-S001 매출현황 목록]
        │ 메모 편집 클릭
        ▼
[DLG-S005 메모 편집]
        │              │
        │ 취소          │ 저장
        ▼              ▼
[이전 화면]         [이전 화면] (메모 반영)
```
