## 화면 목적

다양한 관점(상품별·상품 유형별·결제수단별·종목별·개월별 판매 비율·GX 세부 종목·법인권·직군별·담당자별)으로 매출 데이터를 시각적으로 분석하는 통계 화면입니다. 센터장과 매니저는 이 화면에서 어떤 상품이 잘 팔리는지, 어떤 결제수단이 주로 사용되는지, 단기/장기 회원권 판매 믹스가 어떤지, GX 하위 종목별 수요가 어떤지, 법인권(B2B) 매출이 어느 정도인지, 담당자별 신규/재등록 매출과 골프프로별 매출이 어떻게 나뉘는지를 차트와 표로 파악하고 운영 전략 수립에 활용합니다.

## 주요 구성요소

- **최상위 탭 (3개)**: 전체 통계 / 직군별 / 담당자별을 전환합니다. 담당자별 탭은 총매출, 신규 매출, 재등록 매출, 기타 매출, 판매 건수, 평균 단가를 함께 표시합니다.
- **날짜 필터 (전체 통계 탭)**: 이번 달·지난 달·최근 3개월·최근 6개월·올해 프리셋 버튼과 시작일~종료일 직접 입력, 조회 버튼을 제공합니다.
- **하위 탭 (전체 통계 탭)**: 상품별 / 상품타입별 / 결제수단별 / 종목별 / 개월별 / GX종목별 / 법인권을 선택합니다. 각 탭 선택에 따라 차트 유형이 달라집니다.
  - 상품별·종목별: 가로 막대 차트 (상위 10개)
  - 상품타입별·결제수단별: 도넛 차트
  - 개월별: 1개월·3개월·6개월·12개월·기타 판매 비율 막대 차트
  - GX종목별: 상품 마스터 `category=GX`와 2단계 `subCategory` 기준의 요가·필라테스·스피닝·줌바·에어로빅·GX 기타 6종 분포 차트
  - 법인권: `[월매출]` E열 `구분` 값이 `법인권`인 거래를 집계하는 가로 막대 차트 또는 요약 테이블
- **전월 대비 토글**: 켜면 현재 기간 통계 옆에 전월 동기 요약과 비교(증감액·증감률)가 추가 표시됩니다. 증가는 초록, 감소는 빨강으로 강조됩니다.
- **집계 테이블**: 분석 항목명 / 판매 건수 / 매출액 / 비율(%) 컬럼으로 구성됩니다. 담당자별 탭에서는 신규 매출, 재등록 매출, 기타 매출, 평균 단가 컬럼이 추가됩니다.
- **골프프로별 매출**: `[매출종합]` `골프매출` 탭의 K열 `등록강사` 기준으로 프로별 골프 매출 합계와 비중을 표시합니다.
- **총 매출 표시**: 현재 필터 기준의 총 매출액을 차트 영역 상단에 표시합니다.

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|---|---|---|
| 슈퍼관리자 / 본사 최고관리자 | 전체 지점 매출 조회·분석 가능 | 모든 탭·차트·날짜 필터 사용, 기간 조회 |
| 센터장 / 매니저 | 소속 지점 매출 전체 | 모든 탭·차트·날짜 필터 사용, 기간 조회 |
| FC | 담당 회원 결제 처리 가능 | 조회·필터 사용 가능 |
| 트레이너 / 스태프 | 매출 조회만 (처리 불가) | 조회만 가능 |

## 주요 UX 흐름

1. 사이드바에서 "매출관리 > 매출 통계"를 클릭해 진입합니다.
2. 최상위 탭에서 "전체 통계" / "직군별" / "담당자별" 중 원하는 뷰를 선택합니다.
3. 전체 통계 탭에서 날짜 프리셋 또는 직접 입력으로 기간을 설정하고 "조회" 버튼을 누릅니다.
4. 하위 탭(상품별·상품타입별·결제수단별·종목별·개월별·GX종목별·법인권)을 선택해 원하는 분석 차트를 봅니다.
5. "전월 대비" 토글을 켜면 지난달과의 비교 데이터를 함께 확인합니다.
6. 하단 집계 테이블에서 순위와 비율을 수치로 정밀 확인합니다.
7. 담당자별 탭에서는 신규/재등록 매출 분리를 확인하고, 골프 매출은 골프프로별 성과로 별도 해석합니다.

## 화면 상태

| 상태 | 설명 |
|---|---|
| 로딩 중 | 데이터 조회 중 차트 영역에 스피너, 테이블에 로딩 상태를 표시합니다. |
| 정상 표시 | 데이터가 있을 때 차트와 테이블을 함께 표시합니다. |
| 데이터 없음 | 선택 기간에 데이터가 없을 때 차트 영역에 "조회된 데이터가 없습니다" 안내를 표시합니다. |
| 전월 대비 ON | 현재 기간 데이터 아래 전월 동기 비교 요약 바와 테이블이 추가로 표시됩니다. |
| 오류 | 데이터 조회 실패 시 오류 메시지 알림(토스트)을 표시합니다. |

## 비즈니스 규칙

- 개월별 판매 비율은 1개월·3개월·6개월·12개월·기타 5개 버킷으로 고정하며, `[주간월간보고]` 주차별 V5:Z5를 원천으로 사용합니다.
- 매출 원장에 `durationMonths` 값이 있는 경우 개월별 판매 비율과 선수익금 서비스일수 판정에 우선 적용합니다.
- 1개월 판매 비중이 높으면 단기 쏠림 및 락인 약화 신호로 표시하고, 3개월 이상 상품 프로모션 검토 인사이트를 제공합니다.
- GX 세부 종목 분포는 상품 마스터의 2단계 분류(`category=GX` + `subCategory`)를 기준으로 요가·필라테스·스피닝·줌바·에어로빅·GX 기타 6종으로 분류합니다.
- 외부 시트 연동 시 `[월매출]` C열 상품/종목명과 `[주간월간보고]` 주차별 T5:U5를 상품 마스터 `subCategory`에 매핑합니다.
- GX 세부 종목 분포는 비인기 클래스 폐지와 인기 클래스 증설 판단에 활용합니다.
- 법인권 매출은 `[월매출]` E열 `구분` 값이 `법인권`인 거래의 매출액 합계로 산출합니다.
- 법인권 매출은 B2B 계약·갱신 주기 관리 대상이므로 일반 회원권/B2C 매출과 별도 집계합니다.
- 환불 건은 법인권 매출 집계에서 제외합니다.
- 담당자별 신규/재등록 분리 매출은 `[매출종합]`의 E열 `구분/유형` 값과 F열 `결제금액`을 기준으로 산출합니다.
- 회원권·락카 매출은 K열 `담당자`, 강습·골프 매출은 K열 `등록강사`를 판매 귀속 담당자로 사용합니다.
- 트레이너별 신규/재등록 매출의 공식 원천은 `[매출종합]` 거래 단위 데이터이며, `[강습업무보고]` 주간보고 J열 `주간신규`, K열 `주간재등록`은 검산 또는 KPI 센터 운영 보조 원천으로만 사용합니다.
- 골프프로별 매출은 `[매출종합]` `골프매출` 탭 K열 `등록강사`별 F열 `결제금액` 합계로 산출합니다.

| 지표 | 외부 템플릿 | 셀 / 컬럼 | 산출 기준 |
|---|---|---|---|
| 담당자별 신규/재등록 매출 | `[매출종합]` | `[매출종합]` E/F/K열 | 판매 귀속 담당자별 신규·재등록 매출 분리. `[강습업무보고]` J:K열은 검산/운영 보조 |

## 관련 다이어그램

- 최상위 탭(전체·직군별·담당자별) 전환 흐름
- 하위 탭별 차트 유형 분기 흐름 (가로 막대·도넛·개월별 비율·GX종목별·법인권 요약)
- 전월 대비 비교 데이터 표시 흐름
- 날짜 프리셋 및 직접 입력 조회 흐름
- 계정 권한별 접근 분기 흐름
