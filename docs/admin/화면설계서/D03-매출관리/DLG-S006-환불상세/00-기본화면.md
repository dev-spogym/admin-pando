## 목적

완료된 환불 건의 상세 정보를 조회합니다. 환불 금액, 환불 사유, 처리 직원, 처리 시각 등 환불 과정에서 기록된 모든 정보를 한 화면에서 확인합니다.

## 구성요소

| 구성요소 | 내용 |
|---|---|
| 환불 번호 | 시스템이 부여한 환불 고유 식별 번호 |
| 원 매출 정보 | 환불 대상이 된 원 매출 번호, 결제일, 상품명 |
| 환불 금액 | 실제 환불 처리된 금액 |
| 부분 환불 여부 | 전체 환불 또는 부분 환불 구분 |
| 환불 사유 | 고객 또는 직원이 입력한 환불 사유 |
| 처리 직원 | 환불을 처리한 담당 직원 이름 |
| 처리 일시 | 환불 처리가 완료된 날짜 및 시각 |
| 환불 수단 | 환불이 이루어진 방법 (원결제 수단 환불, 현금 등) |
| 닫기 버튼 | 다이얼로그를 닫고 이전 화면으로 복귀 |

## 언제 표시되나요

- SCR-S007(환불 관리) 목록에서 특정 환불 건을 클릭했을 때
- DLG-S014(환불 상세 결과) 이후 이력 조회 시
- 회원 상세 화면의 환불 이력에서 특정 건을 선택했을 때

## 주요 흐름

1. 환불 관리 목록 또는 회원 이력에서 환불 건을 선택합니다.
2. 환불 상세 다이얼로그가 열리며 해당 환불의 모든 정보가 표시됩니다.
3. 필요한 내용을 확인합니다.
4. 닫기 버튼을 눌러 이전 화면으로 돌아갑니다.

## 관련 다이어그램

        ▼
[DLG-S006 환불 상세]
        │ 닫기
        ▼
