## 목적

기존에 등록된 할인 정책의 내용을 수정하는 모달입니다. 등록 시 설정한 모든 항목을 변경할 수 있습니다.

---

## 구성요소

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 할인명 | 필수 | 기존 이름 수정 가능 |
| 할인 유형 | 필수 | 정률(%) / 정액(원) |
| 할인 값 | 필수 | 변경할 할인율 또는 금액 |
| 최소 계약 기간 | 선택 | 조건 기간 수정 |
| 최대 할인금액 | 선택 | 한도 금액 수정 |
| 활성 상태 | 선택 | 활성/비활성 전환 |

하단 버튼: **취소** / **수정 저장**

---

## 언제 표시되나요

- SCR-P004(할인 설정) 목록에서 수정 아이콘(연필)을 클릭했을 때

---

## 주요 흐름

1. 기존 할인 정책 정보가 폼에 자동 입력된 상태로 열림
2. 원하는 항목 수정
3. "수정 저장" 버튼 클릭
   - 필수 항목 미입력 시: 안내 메시지
   - 정상 저장 시: 모달 닫힘 + 목록 갱신 + 변경 이력 기록
4. "취소" 클릭: 모달 닫힘
