## 목적

등록된 할인 정책을 삭제하기 전 최종 확인을 받는 다이얼로그입니다. 삭제 후에는 해당 할인 정책을 복구할 수 없습니다.

---

## 구성요소

- 삭제 대상 할인 정책명 표시
- 안내 문구: "정말로 이 할인 정책을 삭제하시겠습니까?"
- 하단 버튼: **취소** / **삭제**

---

## 언제 표시되나요

- SCR-P004(할인 설정) 목록에서 삭제 아이콘(휴지통)을 클릭했을 때

---

## 주요 흐름

1. 삭제 아이콘 클릭 → 이 다이얼로그 표시
2. "취소" 클릭: 다이얼로그 닫힘, 목록 유지
3. "삭제" 클릭: 할인 정책 삭제 처리 → 완료 메시지 → 목록 갱신 + 변경 이력 기록
