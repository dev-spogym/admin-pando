## 화면 목적

센터에서 판매 중인 상품을 고객이 보기 좋은 카탈로그 형태로 안내하는 화면입니다. 회원 상담 시 직원이 태블릿이나 모니터로 보여주거나, 인쇄·공유하여 외부 홍보에 활용할 수 있습니다.


---

## 주요 구성요소

### 카탈로그 뷰

- 상품 카드 그리드 레이아웃
- 각 카드: 상품명, 가격, 기간/횟수, 주요 혜택 요약
- 상품 대분류별 색상 구분(이용권, PT, GX 등)
- GX 상품은 요가 / 필라테스 / 스피닝 / 줌바 / 에어로빅 / GX 기타 세부종목 배지를 함께 표시
- 이미지 첨부 가능 (미구현)

### 필터 바

- 1단계 상품 대분류 탭 필터 (이용권 / PT / GX / 상품 / 서비스)
- GX 탭 선택 시 2단계 세부종목 필터 표시 (요가 / 필라테스 / 스피닝 / 줌바 / 에어로빅 / GX 기타)
- 활성 상품만 표시 (비활성 상품 제외)

### 인쇄 · 공유 기능

- 인쇄: 브라우저 인쇄 다이얼로그 연동 (카탈로그 전용 레이아웃)
- 공유: 카탈로그 페이지 링크 복사

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|-----------|--------------|--------------|
| 일반 직원 | 활성 상품 카탈로그 뷰 | 조회, 인쇄, 공유 |
| 관리자 | 전체 카탈로그 + 설정 | 조회, 인쇄, 공유, 카탈로그 옵션 설정 |
| 슈퍼관리자 | 전체 카탈로그 + 설정 | 조회, 인쇄, 공유, 카탈로그 옵션 설정 |

---

## 주요 UX 흐름

1. **카탈로그 조회**: 화면 진입 → 활성 상품이 카드 형태로 표시
2. **필터**: 상품 유형 탭 클릭 → 해당 유형 상품만 표시
3. **카탈로그 미리보기**: [DLG-P009-카탈로그미리보기]에서 고객에게 보여질 화면 확인
4. **옵션 설정**: [DLG-P010-카탈로그설정]에서 노출 옵션 설정
5. **내용 편집**: [DLG-P011-카탈로그편집]에서 카탈로그 문구·순서 편집
6. **인쇄**: 인쇄 버튼 클릭 → 카탈로그 전용 레이아웃으로 인쇄 다이얼로그 실행

## 화면 상태

| 상태 | 조건 | 표시 내용 |
|------|------|-----------|
| 빈 상태 | 활성 상품 없음 | "표시할 상품이 없습니다." 안내 |
| 정상 | 활성 상품 존재 | 카드 그리드 표시 |
| 인쇄 모드 | 인쇄 버튼 클릭 | 헤더·내비게이션 숨김, 카탈로그 전용 레이아웃 |
