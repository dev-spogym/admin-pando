## 목적

재고 실사(직접 세어보기) 결과가 시스템 재고와 다를 때 실제 수량으로 맞춰주는 모달입니다. 입출고 이력 없이 재고 수량을 직접 수정하므로 조정 사유를 반드시 기록해야 합니다.

---

## 구성요소

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 상품명 | 자동 입력 | 조정할 상품 (변경 불가) |
| 현재 시스템 재고 | 자동 표시 | 현재 시스템에 기록된 수량 |
| 실제 재고 수량 | 필수 | 실사 후 실제 확인된 수량 |
| 조정 사유 | 필수 | 분실 / 파손 / 오류 수정 / 기타 중 선택 또는 직접 입력 |
| 조정일 | 필수 | 실사 날짜 (기본값: 오늘) |
| 메모 | 선택 | 상세 경위 기록 |

하단 버튼: **취소** / **조정 적용**

---

## 언제 표시되나요

- SCR-P007(재고 관리) 목록에서 특정 상품의 "조정" 버튼을 클릭했을 때

---

## 주요 흐름

1. 대상 상품명, 현재 시스템 재고가 표시된 상태로 모달 열림
2. 실제 재고 수량과 조정 사유 입력
3. "조정 적용" 버튼 클릭
   - 실제 수량 미입력 또는 0 미만 시: 안내 메시지
   - 조정 사유 미입력 시: 안내 메시지
   - 정상 적용 시: 재고 수량 변경 + 조정 이력 기록 + 모달 닫힘 + 완료 메시지
4. "취소" 클릭: 모달 닫힘, 재고 변경 없음
