## 목적

상품을 영구 삭제하기 전 최종 확인을 받는 다이얼로그입니다. 삭제 후에는 복구할 수 없으므로 실수로 인한 삭제를 방지합니다.

---

## 구성요소

- 안내 문구: "정말 삭제하시겠습니까? 삭제 후에는 복구할 수 없습니다."
- 하단 버튼: **취소** / **삭제**

---

## 언제 표시되나요

- 상품 수정 패널에서 "삭제" 버튼 클릭 후, 해당 상품의 판매 이력이 없을 때
- 판매 이력이 있는 상품 삭제 시도 시에는 대신 [DLG-P004-비활성화안내]가 표시됩니다.

---

## 주요 흐름

1. 수정 패널에서 "삭제" 버튼 클릭
2. 판매 이력 없음 확인 → 이 다이얼로그 표시
3. "취소" 클릭: 다이얼로그 닫힘, 작업 화면 유지
4. "삭제" 클릭: 상품 삭제 처리 → 완료 메시지 → 목록 갱신
