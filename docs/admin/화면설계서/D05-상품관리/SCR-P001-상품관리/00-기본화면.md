## 화면 목적

센터에서 판매하는 모든 상품(이용권, PT 수강권, GX 수강권, 대여권, 일반 상품 등)을 한눈에 파악하고 관리하는 메인 화면입니다. 상품별 가격, 이용 기간, 요일·시간 제한, 판매 현황을 목록으로 조회하고, 우측 패널을 통해 상세 정보를 확인하거나 수정할 수 있습니다.

---

## 주요 구성요소

### 상단 통계 카드

| 카드 | 설명 |
|------|------|
| 전체 상품 수 | 센터에 등록된 상품 총 개수 |
| 활성 상품 수 | 현재 판매 중인 상품 수 |
| 비활성 상품 수 | 판매 중지된 상품 수 |
| 카테고리 수 | 등록된 상품 분류 개수 |

### 3단계 필터

| 단계 | 필터 유형 | 옵션 |
|------|-----------|------|
| 1단계 | 상품 유형 탭 | 전체 / 회원권 / 수강권 / 대여권 / 일반 |
| 2단계 | 카테고리 분류 | 전체 / 이용권 / PT / GX / 기타 (등록된 분류에 따라 동적 생성) |
| 3단계 | 종목 필터 | 전체 / 헬스 / 필라테스 / 요가 / 수영 / 복싱 / 크로스핏 / 기타 |

- 우측 상단에 상품명 검색 입력창 제공

### 상품 목록 테이블

패널이 닫혀 있을 때 전체 컬럼이 표시됩니다.

| 컬럼 | 내용 |
|------|------|
| 상품명 | 유형 배지 + 패키지 배지 + 상품명 |
| 현금가 | 현금 결제 시 가격 |
| 카드가 | 카드 결제 시 가격 |
| 기간 | 이용 가능 기간 (일 단위) |
| 횟수 | 이용 가능 횟수 |
| 요일 (월~일) | 이용 가능 요일 표시 (체크/엑스) |
| 시간 | 이용 가능 시간대 |
| 홀딩 | 홀딩(일시정지) 가능 여부 |
| 양도 | 양도 가능 여부 |
| 포인트 | 포인트 적립 여부 |
| 판매 | 해당 상품 판매 건수 |
| 상태 | 사용 / 미사용 |

> 우측 상세 패널이 열리면 요일/시간/홀딩/양도/포인트/판매 컬럼은 자동으로 숨겨지고 핵심 컬럼만 표시됩니다.

### 분류 관리 탭

상품 목록 탭과 분류 관리 탭을 전환할 수 있습니다.

- 분류 등록 폼: 분류명, 정렬 순서, 활성 여부 입력
- 분류 목록 테이블: 분류명 / 정렬 순서 / 상태 / 수정·삭제 버튼

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|-----------|--------------|--------------|
| 일반 직원 | 상품 목록 조회, 통계 카드 | 상품 조회만 가능 |
| 상품 편집 권한 보유 직원 | 전체 화면 | 상품 등록, 수정, 삭제, Excel 다운로드 |
| 슈퍼관리자 | 전체 화면 + 전 지점 배포 버튼 | 상품 등록/수정/삭제 + 전체 지점 동시 배포 |

---

## 주요 UX 흐름

1. **상품 조회**: 화면 진입 시 센터 전체 상품 목록 표시 → 필터/검색으로 원하는 상품 찾기
2. **상품 상세 보기**: 목록에서 상품 행 클릭 → 우측에 상세 패널이 슬라이드로 열림
3. **상품 등록**: 상단 "+ 상품 등록" 버튼 클릭 → 우측 패널이 등록 모드로 열림
4. **분류 관리**: "분류 관리" 탭 클릭 → 상품 분류 등록/수정/삭제
5. **전 지점 배포** (슈퍼관리자): "전 지점 배포" 버튼 → 배포 모달에서 상품·지점 선택 후 일괄 배포
6. **Excel 내보내기**: "Excel" 버튼 → 현재 필터된 상품 목록을 엑셀 파일로 다운로드

## 화면 상태

| 상태 | 조건 | 표시 내용 |
|------|------|-----------|
| 로딩 중 | 상품 데이터 불러오는 중 | 목록 영역 로딩 표시 |
| 빈 상태 | 등록된 상품 없음 또는 필터 결과 없음 | "상품이 없습니다." 안내 + 상품 등록 링크(권한 있는 경우) |
| 정상 | 상품 목록 존재 | 상품 목록 테이블 표시 |
| 패널 열림 | 상품 행 클릭 또는 등록 버튼 클릭 | 목록 55% + 우측 패널 45% 분할 표시 |
| 패널 닫힘 | 패널 X 버튼 클릭, 저장/삭제 완료 | 목록 100% 표시 |
