## 목적

기존에 등록된 상품의 정보를 복사하여 새 상품 등록의 기초 정보로 활용하는 모달입니다. 유사한 상품을 빠르게 등록할 때 유용하며, 가져온 정보는 새 상품으로 저장되어 원본 상품에 영향을 주지 않습니다.

---

## 구성요소

- 상품명 검색 입력창
- 현재 편집 중인 상품을 제외한 전체 상품 목록
  - 상품명, 가격, 유형 표시
  - 행 클릭으로 선택

하단 버튼: **닫기**

---

## 언제 표시되나요

- SCR-P002(상품 등록/수정) 패널 하단의 "상품정보 가져오기" 버튼을 클릭했을 때

---

## 주요 흐름

1. 모달 열림 → 전체 상품 목록 표시 (현재 편집 중인 상품 제외)
2. 검색어 입력으로 원하는 상품 찾기
3. 상품 행 클릭 → 해당 상품 정보가 등록 폼에 자동 입력
4. 모달 닫힘 + "상품 정보를 불러왔습니다. 등록 시 새 상품으로 저장됩니다." 안내 메시지
5. 가져온 정보를 바탕으로 수정 후 새 상품으로 등록
