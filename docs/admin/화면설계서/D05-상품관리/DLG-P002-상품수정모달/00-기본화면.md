## 목적

기존에 등록된 상품의 정보를 수정하는 모달입니다. 등록 시 설정한 모든 항목을 수정할 수 있으며, 가격 변경 시에는 변경 이력이 자동으로 기록됩니다.

---

## 구성요소

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 상품 구분 | 필수 | 레슨 / 이용 / 락커 / 판매 |
| 상품명 | 필수 | 중복 불가 (기존 이름 제외) |
| 금액 | 필수 | 현금가 기준 |
| 이용 기간 / 횟수 | 조건부 | 상품 구분에 따라 표시 |
| 이용 요일·시간 | 선택 | 요일별 체크박스 + 시간 선택 |
| 각종 옵션 | 선택 | 홀딩/양도/포인트/키오스크 등 |
| 활성 상태 | 선택 | 활성/비활성 전환 |

하단 버튼: **삭제** / **닫기** / **저장**

---

## 언제 표시되나요

- SCR-P001(상품 관리) 또는 SCR-P003(상품 상세 패널)에서 기존 상품을 선택했을 때
- 상품 편집 권한이 있는 계정에서만 수정 가능합니다.

---

## 주요 흐름

1. 기존 상품 정보가 폼에 자동 입력된 상태로 열림
2. 원하는 항목 수정
3. "저장" 버튼 클릭
   - 필수 항목 미입력 시: 안내 메시지 표시
   - 상품명 중복 시: 중복 안내 메시지 표시
   - 정상 저장 시: 모달 닫힘 + 목록 갱신 + 완료 메시지
   - 가격 변경 시: 가격 변경 이력 자동 기록
4. "삭제" 버튼 클릭 시: [DLG-P003-상품삭제확인] 또는 [DLG-P004-비활성화안내]로 이동
