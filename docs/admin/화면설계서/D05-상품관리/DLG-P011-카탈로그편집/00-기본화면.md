## 목적

상품 카탈로그에 표시되는 상품 설명 문구, 혜택 안내, 노출 순서 등을 편집하는 모달입니다. 고객에게 더 매력적으로 보일 수 있도록 카탈로그 내용을 자유롭게 커스터마이징할 수 있습니다.

---

## 구성요소

- 상품 목록 (드래그로 순서 변경 가능)
- 각 상품별 카탈로그 표시 설정
  - 카탈로그 노출 여부 토글
  - 카탈로그 전용 홍보 문구 입력 (선택)
  - 강조 태그 입력 (예: "인기", "추천", "신규")

하단 버튼: **취소** / **저장**

---

## 언제 표시되나요

- SCR-P005(상품 카탈로그) 화면에서 "편집" 버튼을 클릭했을 때

---

## 주요 흐름

1. 모달 열림 → 현재 카탈로그 구성 표시
2. 상품 순서 드래그로 변경
3. 각 상품별 노출 여부, 홍보 문구 편집
4. "저장" 클릭: 편집 내용 저장 → 카탈로그에 반영
5. "취소" 클릭: 변경 사항 폐기, 모달 닫힘
