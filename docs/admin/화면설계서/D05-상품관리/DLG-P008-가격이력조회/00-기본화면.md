## 목적

선택한 기간 내의 상품 가격 변동 이력을 상세하게 조회하는 다이얼로그입니다. 특정 기간을 지정하여 그 기간 내 모든 가격 변경 내역을 확인할 수 있습니다.

---

## 구성요소

- 조회 기간 선택 (시작일 ~ 종료일 날짜 피커)
- 가격 변동 이력 테이블

| 컬럼 | 내용 |
|------|------|
| 변경 일시 | 가격이 변경된 날짜와 시간 |
| 변경 전 가격 | 변경 이전 가격 |
| 변경 후 가격 | 변경 이후 가격 |
| 변경자 | 변경한 직원 이름 |

하단 버튼: **닫기**

---

## 언제 표시되나요

- SCR-P003(상품 상세 패널)의 "가격 변경 이력" 섹션에서 "전체 이력 보기" 또는 상세 조회 버튼을 클릭했을 때

---

## 주요 흐름

1. 모달 열림 → 기본 기간(최근 3개월 등)으로 이력 표시
2. 기간 변경 시 해당 기간 이력 갱신
3. 이력 확인 후 "닫기" 클릭으로 모달 닫힘
