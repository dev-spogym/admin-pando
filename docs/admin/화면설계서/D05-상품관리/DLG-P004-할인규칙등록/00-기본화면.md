## 목적

새로운 할인 정책을 등록하는 모달입니다. 할인명, 유형(정률/정액), 할인 값, 적용 조건을 설정합니다.

---

## 구성요소

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 할인명 | 필수 | 예: "재등록 10% 할인" |
| 할인 유형 | 필수 | 정률(%) / 정액(원) 중 선택 |
| 할인 값 | 필수 | 정률: 숫자%, 정액: 원 단위 금액 |
| 최소 계약 기간 | 선택 | N일 이상 계약 시 적용 (예: 90일) |
| 최대 할인금액 | 선택 | 할인 한도 금액 (예: 100,000원) |
| 활성 상태 | 선택 | 기본값: 활성 |

하단 버튼: **취소** / **등록**

---

## 언제 표시되나요

- SCR-P004(할인 설정) 화면에서 "+ 할인 추가" 버튼을 클릭했을 때

---

## 주요 흐름

1. 할인명, 유형, 할인 값 입력
2. 필요 시 조건·한도 추가 입력
3. "등록" 버튼 클릭
   - 할인명 미입력 시: 안내 메시지
   - 할인 값 미입력 시: 안내 메시지
   - 정상 등록 시: 모달 닫힘 + 목록 갱신 + 변경 이력 기록
4. "취소" 클릭: 모달 닫힘
