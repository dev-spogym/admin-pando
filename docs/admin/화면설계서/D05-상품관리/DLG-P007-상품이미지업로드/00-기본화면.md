## 목적

상품 카탈로그 또는 키오스크 화면에 표시될 상품 대표 이미지를 업로드하는 모달입니다. 이미지를 통해 고객에게 상품을 시각적으로 소개할 수 있습니다.

---

## 구성요소

- 이미지 업로드 영역 (드래그 앤 드롭 또는 파일 선택)
- 업로드 가능 형식 안내: JPG, PNG, WebP
- 권장 크기 안내
- 미리보기 영역: 업로드한 이미지 확인
- 하단 버튼: **취소** / **적용**

---

## 언제 표시되나요

- SCR-P002(상품 등록/수정) 패널 또는 SCR-P005(상품 카탈로그)에서 이미지 업로드 버튼을 클릭했을 때

---

## 주요 흐름

1. 이미지 업로드 버튼 클릭 → 모달 열림
2. 파일 선택 또는 드래그 앤 드롭으로 이미지 업로드
3. 미리보기에서 이미지 확인
4. "적용" 클릭: 이미지 저장 → 상품 정보에 반영
5. "취소" 클릭: 모달 닫힘, 이미지 변경 없음
