## 화면 목적

새로운 상품을 등록하거나 기존 상품을 수정하는 전용 패널입니다. 상품 구분(레슨/이용/락커/판매)에 따라 필요한 항목이 동적으로 표시되며, 이용 요일·시간 제한, 옵션 설정, 패키지 구성까지 한 화면에서 완성할 수 있습니다.

---

## 주요 구성요소

### 상품 구분 선택

| 구분 | 설명 | 추가 표시 항목 |
|------|------|----------------|
| 레슨 | PT, GX 등 수업형 상품 | 레슨 시간, 레슨 유효기간, 수업구분(개인/정규클래스) |
| 이용 | 기간제 또는 횟수제 회원권 | 이용구분(기간/횟수/포인트), 기간 또는 횟수 입력 |
| 락커 | 락커 이용권 | 기간 입력 |
| 판매 | 실물 상품 판매 | 태그 입력 |

### 기본 정보 입력

| 항목 | 필수 | 설명 |
|------|------|------|
| 상품명 | 필수 | 중복 불가 |
| 금액 | 필수 | 현금가 기준 (0 이상) |
| 설명 | 선택 | 상품 상세 설명 |
| 태그 | 선택 | 검색·분류용 자유 입력 |

### 이용 요일 · 시간 설정

- 요일별(월~일) 체크박스로 이용 가능 요일 선택
- 체크된 요일에 시작·종료 시간 지정 (10분 단위)
- 체크하지 않은 요일은 이용 불가로 처리

### 옵션 설정

| 옵션 | 설명 |
|------|------|
| 예약 가능 | 해당 상품으로 시설/수업 예약 허용 |
| 시설 이용 | 시설 입장 허용 여부 |
| 홀딩 가능 | 이용권 일시정지 허용 |
| 양도 가능 | 이용권 타인 양도 허용 |
| 포인트 적립 | 구매 시 포인트 적립 여부 |
| 회원 직접 휴회 | 회원이 앱에서 직접 휴회 신청 가능 |
| 키오스크 노출 | 키오스크 화면에 상품 노출 |
| 활성 상태 | 판매 활성/비활성 전환 |

### 상세 옵션 (조건부 표시)

| 항목 | 표시 조건 | 선택지 |
|------|-----------|--------|
| 시설 이용 시간 | 시설 이용 체크 시 | 기본 / 오전 / 오후 / 종일 |
| 예약 오픈일 | 예약 가능 체크 시 | 당일 / 1일 전 / 3일 전 / 7일 전 / 14일 전 / 30일 전 |
| 예약 시간 간격 | 예약 가능 체크 시 | 10분 / 20분 / 30분 / 40분 / 50분 / 60분 |
| 휴회 횟수 | 회원 직접 휴회 체크 시 | 1회 / 2회 / 3회 |
| 휴회 기간 | 회원 직접 휴회 체크 시 | 3일 / 7일 / 15일 / 30일 / 60일 |
| 판매 유형 | 항상 표시 | 전체 / 카운터 / 키오스크 / 온라인 |
| 일일 횟수 제한 | 횟수 이용구분 시 | 1회 / 2회 / 3회 / 5회 / 10회 |

### 패키지 구성 (선택)

- "이 상품을 패키지 상품으로 등록" 체크 시 활성화
- 구성 상품 선택 후 추가
- 구성 상품별 개별 가격 표시
- 합산 정가 자동 계산
- 패키지 할인가 별도 입력

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|-----------|--------------|--------------|
| 상품 편집 권한 없는 직원 | 진입 불가 (버튼 자체 비표시) | 없음 |
| 상품 편집 권한 보유 직원 | 전체 등록/수정 폼 | 상품 등록, 수정, 삭제 |
| 슈퍼관리자 | 전체 등록/수정 폼 | 상품 등록, 수정, 삭제 |

---

## 주요 UX 흐름

1. **등록 모드**: SCR-P001에서 "+ 상품 등록" 클릭 → 빈 폼으로 패널 열림 → 정보 입력 → "등록" 버튼 클릭 → 저장 완료 후 목록 갱신
2. **수정 모드**: SCR-P001에서 상품 행 클릭 → 기존 정보가 채워진 폼으로 패널 열림 → 수정 → "저장" 버튼 클릭
3. **상품 정보 가져오기**: 하단 "상품정보 가져오기" 버튼 → DLG-P005-상품가져오기 → 선택한 상품 정보 자동 입력 (새 상품으로 등록 처리)
4. **삭제**: 수정 모드에서 "삭제" 버튼 클릭 → 판매 이력 확인 후 삭제 확인 또는 비활성화 안내 다이얼로그 표시
5. **작업 취소**: "닫기" 버튼 또는 패널 외부 영역 클릭 → DLG-P002-작업취소확인 (변경 사항 있을 경우)

## 화면 상태

| 상태 | 조건 | 표시 내용 |
|------|------|-----------|
| 등록 모드 | 신규 등록 진입 | 헤더 "상품 등록", 저장 버튼 "등록", 삭제 버튼 없음 |
| 수정 모드 | 기존 상품 선택 | 헤더 "상품 수정" + 현재 활성 상태 배지, 저장 버튼 "저장", 삭제 버튼 표시 |
| 가져오기 모드 | 상품 정보 가져온 직후 | 수정 모드 외형이지만 "등록" 버튼으로 표시 (새 상품으로 저장) |
| 저장 중 | 저장 버튼 클릭 후 완료 전 | 저장 버튼 로딩 상태, 모든 입력 비활성화 |
| 삭제 확인 중 | 삭제 버튼 클릭 후 | 하단에 "정말 삭제하시겠습니까?" 인라인 확인 버튼 표시 |
| 비활성화 확인 중 | 판매 이력 있는 상품 삭제 시도 | "이용 중인 회원이 N명 있습니다. 미사용으로 전환하시겠습니까?" 표시 |
