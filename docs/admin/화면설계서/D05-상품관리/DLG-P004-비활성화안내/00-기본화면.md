## 목적

현재 이용 중인 회원이 있는 상품을 삭제하려 할 때, 삭제 대신 "미사용(비활성화)" 전환을 안내하는 다이얼로그입니다. 기존 회원의 이용권에 영향을 주지 않으면서 신규 판매만 중단할 수 있습니다.

---

## 구성요소

- 안내 문구: "이 상품을 이용 중인 회원이 N명 있습니다. 삭제 대신 '미사용'으로 전환하시겠습니까?"
- 현재 이용 중인 회원 수 표시
- 하단 버튼: **취소** / **미사용으로 전환**

---

## 언제 표시되나요

- 상품 수정 패널에서 "삭제" 버튼 클릭 시, 해당 상품을 현재 이용 중인 회원이 1명 이상 있을 때

---

## 주요 흐름

1. 수정 패널에서 "삭제" 버튼 클릭
2. 이용 중인 회원 수 조회 → 1명 이상이면 이 다이얼로그 표시
3. "취소" 클릭: 다이얼로그 닫힘, 상품 유지
4. "미사용으로 전환" 클릭: 상품 상태를 비활성(미사용)으로 변경 → 완료 메시지 → 패널 닫힘
