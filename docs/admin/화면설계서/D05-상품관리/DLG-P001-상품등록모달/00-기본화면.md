## 목적

상품 목록 화면을 벗어나지 않고 빠르게 새 상품을 등록할 수 있는 모달입니다. 자주 사용하는 핵심 항목만 간소화하여 입력하고, 상세 설정이 필요한 경우 등록 후 상세 패널에서 추가 수정할 수 있습니다.

---

## 구성요소

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 상품 구분 | 필수 | 레슨 / 이용 / 락커 / 판매 중 선택 |
| 상품명 | 필수 | 센터 내 중복 불가 |
| 금액 | 필수 | 현금가 기준 |
| 이용 기간 | 조건부 | 기간제 상품 선택 시 표시 |
| 이용 횟수 | 조건부 | 횟수제 상품 선택 시 표시 |
| 활성 상태 | 선택 | 기본값: 활성 |

하단 버튼: **취소** / **등록**

---

## 언제 표시되나요

- SCR-P001(상품 관리) 화면에서 "+ 상품 등록" 버튼을 클릭했을 때
- 상품 편집 권한이 있는 계정에서만 버튼이 표시됩니다.

---

## 주요 흐름

1. 상품 구분 선택 → 관련 입력 항목 표시
2. 상품명, 금액 등 필수 항목 입력
3. "등록" 버튼 클릭
   - 상품명 미입력 또는 금액 미입력 시: 안내 메시지 표시 후 중단
   - 동일 이름의 상품 존재 시: 중복 안내 메시지 표시 후 중단
   - 정상 등록 시: 모달 닫힘 + 목록 갱신 + 완료 메시지
4. "취소" 클릭 또는 모달 외부 클릭 시: 모달 닫힘
