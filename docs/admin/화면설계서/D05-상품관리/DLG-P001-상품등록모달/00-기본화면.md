## 목적

상품 목록 화면을 벗어나지 않고 빠르게 새 상품을 등록할 수 있는 모달형 패널입니다. SCR-P001에서는 우측 패널로 열리며, 직접 경로(`/products/new`)에서는 동일한 레거시 모달 레이아웃을 단독 화면으로 표시합니다. 등록 버튼을 누르면 Supabase `products` 테이블에 실제 상품 원장이 생성되고 목록이 갱신됩니다.

---

## 구성요소

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 상품 구분 | 필수 | 레슨 / 이용 / 락커 / 판매 중 선택 |
| 상품명 | 필수 | 센터 내 중복 불가 |
| 현금가 | 필수 | 현금 결제 기준 가격 |
| 카드가 | 선택 | 미입력 시 현금가와 동일 저장 |
| 상품그룹 | 선택 | 상품관리 > 분류 관리 탭에서 등록한 활성 그룹 중 선택 |
| 이용 기간 | 조건부 | 기간제 상품 선택 시 표시 |
| 이용 횟수 | 조건부 | 횟수제 상품 선택 시 표시 |
| GX 세부종목 | 조건부 | 레슨 > GX 선택 시 요가 / 필라테스 / 스피닝 / 줌바 / 에어로빅 / GX 기타 중 필수 |
| 요일·시간 | 선택 | 이용 가능 요일과 시작/종료 시간 |
| 주요 옵션 | 선택 | 시설 이용, 예약 가능, 양도, 키오스크 판매, 회원 직접 휴회 |
| 활성 상태 | 선택 | 기본값: 활성 |

하단 버튼: **취소** / **등록**

---

## 저장 규칙

| 항목 | Supabase 저장 위치 | 설명 |
|------|-------------------|------|
| 지점 | `products.branchId` | 현재 선택된 지점 기준 |
| 상품명 | `products.name` | 동일 지점 + 동일 대분류 내 중복 등록 차단 |
| 구분 | `products.category`, `products.productType` | 레슨=`LESSON`, 이용=`MEMBERSHIP`, 락커=`RENTAL`, 판매=`GENERAL` |
| 상품그룹 | `products.productGroupId` | 선택한 `product_groups.id` 저장. 자동 추론으로 고정하지 않음 |
| 가격 | `price`, `cashPrice`, `cardPrice` | `price`는 현금가 기준 |
| 기간/횟수 | `duration`, `sessions`, `totalCount` | 조건부 입력값에 따라 저장 |
| 옵션 | `usage_restrictions` JSON 및 관련 boolean 컬럼 | 요일·시간·시설·예약·휴회·양도·키오스크 상태 저장 |

---

## 언제 표시되나요

- SCR-P001(상품 관리) 화면에서 "+ 상품 등록" 버튼을 클릭했을 때
- 상품 편집 권한이 있는 계정에서만 버튼이 표시됩니다.

---

## 주요 흐름

1. 상품 구분 선택 → 관련 입력 항목 표시
2. 상품명, 금액 등 필수 항목 입력
3. "등록" 버튼 클릭
   - 상품명 미입력 또는 현금가 미입력 시: 안내 메시지 표시 후 중단
   - 동일 이름의 상품 존재 시: 중복 안내 메시지 표시 후 중단
   - 정상 등록 시: Supabase insert 성공 → 모달/패널 유지 또는 닫힘 + 목록 갱신 + 완료 메시지
   - 저장 실패 시: 입력값 유지 + 오류 메시지 표시
4. "취소" 클릭 또는 모달 외부 클릭 시: 모달 닫힘
