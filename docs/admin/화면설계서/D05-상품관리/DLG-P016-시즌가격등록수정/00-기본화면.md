## 목적

시즌 특별 가격을 등록하거나 기존 시즌 특가를 수정하는 모달입니다. 특정 기간 동안만 적용되는 할인 가격을 상품별로 설정할 수 있습니다.

---

## 구성요소

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 시즌 이름 | 필수 | 예: "2026 여름 특가", "신년 이벤트" |
| 적용 시작일 | 필수 | 특가 시작 날짜 |
| 적용 종료일 | 필수 | 특가 종료 날짜 (시작일 이후) |
| 대상 상품 선택 | 필수 | 체크박스로 특가 적용 상품 1개 이상 선택 |
| 특가 설정 방식 | 필수 | 정률(%) 할인 / 정액(원) 할인 / 직접 입력 가격 |
| 특가 값 | 필수 | 할인율, 할인 금액, 또는 직접 입력 가격 |
| 활성 상태 | 선택 | 기본값: 활성 |

하단 버튼: **취소** / **저장**

---

## 언제 표시되나요

- SCR-P008(시즌 가격 관리) 화면에서 "시즌 특가 등록" 버튼을 클릭했을 때 (등록 모드)
- SCR-P008 목록에서 수정 버튼을 클릭했을 때 (수정 모드)

---

## 주요 흐름

**등록 모드:**
1. 빈 폼으로 모달 열림
2. 시즌명, 기간, 대상 상품, 특가 설정 입력
3. "저장" 클릭
   - 필수 항목 미입력 시: 안내 메시지
   - 종료일이 시작일보다 이르면: "종료일을 확인하세요." 안내
   - 대상 상품 미선택 시: 안내 메시지
   - 정상 저장 시: 모달 닫힘 + 목록 갱신

**수정 모드:**
1. 기존 정보가 채워진 상태로 모달 열림
2. 원하는 항목 수정
3. "저장" 클릭 → 수정 저장 + 목록 갱신
4. "취소" 클릭: 변경 사항 폐기, 모달 닫힘
