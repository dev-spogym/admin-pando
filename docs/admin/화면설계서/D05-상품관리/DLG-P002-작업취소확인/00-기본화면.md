## 목적

상품 등록 또는 수정 작업 중 입력한 내용이 있을 때, 작업을 취소하면 변경 사항이 사라진다는 것을 사용자에게 알리고 최종 확인을 받는 다이얼로그입니다.

---

## 구성요소

- 안내 문구: "입력한 내용이 저장되지 않습니다. 작업을 취소하시겠습니까?"
- 하단 버튼: **계속 작업** / **취소하기**

---

## 언제 표시되나요

- 상품 등록 또는 수정 모달/패널에서 내용을 변경한 상태로 "닫기" 버튼 또는 패널 외부를 클릭했을 때
- 변경 사항이 전혀 없으면 이 다이얼로그 없이 바로 닫힙니다.

---

## 주요 흐름

1. 작업 중 닫기 시도
2. 변경 사항 감지 시 이 다이얼로그 표시
3. "계속 작업" 클릭: 다이얼로그 닫힘, 작업 화면으로 복귀
4. "취소하기" 클릭: 변경 사항 폐기, 작업 화면 닫힘
