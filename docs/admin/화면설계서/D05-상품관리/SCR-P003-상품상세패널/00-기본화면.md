## 화면 목적

상품 목록(SCR-P001)에서 상품을 클릭하면 화면 우측에서 슬라이드로 나타나는 상세 정보 패널입니다. 상품의 모든 설정값을 한눈에 확인하고, 수정 모드로 전환하거나 가격 변경 이력을 조회할 수 있습니다.

---

## 주요 구성요소

### 헤더 영역

- 상품명 및 현재 상태(활성/비활성) 배지
- 닫기(X) 버튼

### 상품 기본 정보

| 항목 | 내용 |
|------|------|
| 상품 구분 | 레슨 / 이용 / 락커 / 판매 |
| 상품명 | 등록된 상품명 |
| 현금가 | 현금 결제 시 가격 |
| 카드가 | 카드 결제 시 가격 |
| 이용 기간 | 기간제 상품의 유효 기간 |
| 이용 횟수 | 횟수제 상품의 총 이용 횟수 |
| 수업 구분 | 개인 / 정규클래스 (레슨 상품) |
| 레슨 시간 | 1회 수업 시간 (레슨 상품) |
| 이용 구분 | 기간 / 횟수 / 포인트 |
| 태그 | 분류·검색용 태그 |
| 설명 | 상품 부가 설명 |

### 이용 요일 · 시간

- 요일별 이용 가능 여부 (체크/엑스 표시)
- 이용 가능 시간대

### 옵션 현황

| 옵션 | 활성 여부 |
|------|-----------|
| 예약 가능 | O / X |
| 시설 이용 | O / X |
| 홀딩 가능 | O / X |
| 양도 가능 | O / X |
| 포인트 적립 | O / X |
| 회원 직접 휴회 | O / X |
| 키오스크 노출 | O / X |
| 판매 유형 | 전체 / 카운터 / 키오스크 / 온라인 |

### 패키지 구성 (패키지 상품만)

- 구성 상품 목록 및 개별 가격
- 합산 정가 및 패키지 할인가

### 가격 변경 이력 (수정 모드)

- 가격 변경 내역을 시간순으로 표시
- 변경 전 가격 → 변경 후 가격, 변경 일시, 변경자 이름

### 판매 현황

- 해당 상품의 누적 판매 건수

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|-----------|--------------|--------------|
| 일반 직원 | 상품 상세 정보 조회 | 조회만 가능 |
| 상품 편집 권한 보유 직원 | 전체 정보 + 수정/삭제 버튼 | 상품 수정, 삭제 |
| 슈퍼관리자 | 전체 정보 + 수정/삭제 버튼 | 상품 수정, 삭제 |

---

## 주요 UX 흐름

1. **상세 조회**: SCR-P001 목록에서 상품 클릭 → 우측에 패널 슬라이드 오픈 → 상세 정보 확인
2. **수정 진입**: 패널 하단 "저장" 버튼 영역에서 수정 가능 (패널 자체가 등록/수정 통합)
3. **가격 이력 조회**: "가격 변경 이력" 섹션 클릭하여 펼치기 → DLG-P008-가격이력조회로 전체 이력 보기
4. **삭제**: 하단 "삭제" 버튼 클릭 → 판매 이력 여부에 따라 삭제 확인 또는 비활성화 안내
5. **패널 닫기**: X 버튼 또는 목록 영역 클릭 → 패널 닫힘

## 화면 상태

| 상태 | 조건 | 표시 내용 |
|------|------|-----------|
| 정상 조회 | 상품 정보 로드 완료 | 전체 상세 정보 표시 |
| 가격 이력 로딩 중 | 이력 섹션 펼치는 중 | "불러오는 중..." 텍스트 |
| 삭제 확인 중 | 삭제 버튼 클릭 후 | 하단 인라인 확인 버튼 표시 |
| 비활성화 확인 중 | 판매 이력 있는 상품 삭제 시도 | 이용 중 회원 수 안내 + 미사용 전환 확인 버튼 |
