## 목적

슈퍼관리자가 현재 센터에 등록된 상품을 다른 지점 전체 또는 선택한 지점에 동시에 배포하는 모달입니다. 이미 같은 이름의 상품이 있는 지점은 자동으로 건너뛰어 중복 등록을 방지합니다.

---

## 구성요소

### 배포할 상품 선택 영역

- 전체 상품 목록을 체크박스로 표시
- "전체 선택" 버튼 제공
- 선택된 상품 수 표시 (예: "3개 선택됨")

### 대상 지점 선택 영역

- "전체 지점" 체크박스 (기본 선택)
- 전체 지점 체크 해제 시: 개별 지점 체크박스 목록 표시
- 현재 로그인 지점은 목록에서 제외

하단 버튼: **취소** / **배포하기**

---

## 언제 표시되나요

- SCR-P001(상품 관리) 화면에서 슈퍼관리자 계정으로 "전 지점 배포" 버튼을 클릭했을 때

---

## 주요 흐름

1. 배포할 상품 선택 (1개 이상 필수)
2. 대상 지점 선택 (1개 이상 필수)
3. "배포하기" 버튼 클릭
   - 상품 미선택 시: "배포할 상품을 선택하세요." 안내
   - 지점 미선택 시: "배포할 지점을 선택하세요." 안내
   - 정상 진행 시: 배포 최종 확인 단계로 이동
4. 최종 확인에서 승인 시: 지점별 배포 처리
   - 이미 같은 이름의 상품이 있는 지점은 건너뜀
   - 완료 후 "N건 추가, N건 건너뜀" 결과 메시지 표시
