## 화면 목적

운동복, 보충제, 용품 등 실물 상품의 재고 현황을 관리하는 화면입니다. 현재 재고 수량을 확인하고 입고·출고 처리를 기록하며, 재고 실사 후 수동 조정도 할 수 있습니다.


---

## 주요 구성요소

### 재고 현황 테이블

| 컬럼 | 내용 |
|------|------|
| 상품명 | 실물 상품 이름 |
| 카테고리 | 상품 분류 |
| 현재 재고 | 현재 보유 수량 |
| 안전 재고 | 최소 유지 기준 수량 |
| 재고 상태 | 충분 / 부족 / 품절 |
| 최근 입고일 | 마지막 입고 처리 날짜 |
| 액션 | 입고 / 출고 / 조정 버튼 |

### 재고 통계 카드

| 카드 | 내용 |
|------|------|
| 전체 품목 수 | 관리 중인 실물 상품 총 종류 |
| 부족 재고 | 안전 재고 미달 품목 수 |
| 품절 | 재고 0인 품목 수 |

### 필터 및 검색

- 상품 카테고리별 필터
- 재고 상태별 필터 (충분 / 부족 / 품절)
- 상품명 검색

---

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|-----------|--------------|--------------|
| 일반 직원 | 재고 현황 조회 | 조회만 가능 |
| 재고 관리 권한 보유 직원 | 전체 화면 | 입고·출고 처리, 재고 조정 |
| 관리자 / 슈퍼관리자 | 전체 화면 + 통계 | 입고·출고·조정 + 이력 조회 |

---

## 주요 UX 흐름

1. **재고 조회**: 화면 진입 → 전체 실물 상품 재고 현황 표시
2. **입고 처리**: 해당 상품의 "입고" 버튼 클릭 → [DLG-P012-입고등록]에서 수량·날짜 입력 후 저장
3. **출고 처리**: 해당 상품의 "출고" 버튼 클릭 → [DLG-P013-출고등록]에서 수량·사유 입력 후 저장
4. **재고 조정**: 재고 실사 후 수량 불일치 시 "조정" 버튼 → [DLG-P014-재고조정]에서 실제 수량 입력
5. **입출고 이력 조회**: 상품별 "이력" 버튼 또는 이력 탭 → [DLG-P015-입출고이력] 조회

## 화면 상태

| 상태 | 조건 | 표시 내용 |
|------|------|-----------|
| 로딩 중 | 재고 데이터 불러오는 중 | 테이블 로딩 표시 |
| 빈 상태 | 등록된 실물 상품 없음 | "관리 중인 상품이 없습니다." 안내 |
| 부족 재고 경고 | 안전 재고 미달 품목 존재 | 해당 행 강조 표시 (노란색 또는 주황색) |
| 품절 경고 | 재고 0 품목 존재 | 해당 행 강조 표시 (빨간색) |
