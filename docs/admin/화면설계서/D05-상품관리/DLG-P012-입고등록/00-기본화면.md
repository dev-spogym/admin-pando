## 목적

실물 상품의 입고 수량을 등록하는 모달입니다. 입고 처리 시 재고 수량이 자동으로 증가하며 입출고 이력에 기록됩니다.

---

## 구성요소

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 상품명 | 자동 입력 | 입고 처리할 상품 (변경 불가) |
| 입고 수량 | 필수 | 입고할 개수 (1 이상) |
| 입고일 | 필수 | 실제 입고 날짜 (기본값: 오늘) |
| 공급처 | 선택 | 납품 업체명 |
| 메모 | 선택 | 특이사항 기록 |

하단 버튼: **취소** / **입고 등록**

---

## 언제 표시되나요

- SCR-P007(재고 관리) 목록에서 특정 상품의 "입고" 버튼을 클릭했을 때

---

## 주요 흐름

1. 대상 상품명이 자동 입력된 상태로 모달 열림
2. 입고 수량, 입고일 입력
3. "입고 등록" 버튼 클릭
   - 수량 미입력 또는 0 이하 시: 안내 메시지
   - 정상 등록 시: 재고 수량 증가 + 이력 기록 + 모달 닫힘 + 완료 메시지
4. "취소" 클릭: 모달 닫힘
