## 화면 목적

최대 3개 상품을 나란히 비교하여 회원 상담 시 최적의 상품을 안내할 수 있는 화면입니다. 가격, 이용 기간, 포함 서비스, 옵션 항목을 동일한 기준으로 나란히 표시하여 상품 간 차이를 직관적으로 파악할 수 있습니다.


---

## 주요 구성요소

### 상품 선택 영역

- 비교할 상품 최대 3개 선택 (드롭다운 또는 목록에서 선택)
- 선택된 상품은 각 열에 표시
- 상품 제거(X) 버튼으로 비교 대상에서 제외

### 비교 테이블

| 비교 항목 | 설명 |
|-----------|------|
| 상품명 | 상품 이름 및 유형 배지 |
| 현금가 | 현금 결제 시 가격 |
| 카드가 | 카드 결제 시 가격 |
| 이용 기간 | 유효 기간 (일 단위) |
| 이용 횟수 | 총 이용 횟수 |
| 이용 요일 | 이용 가능 요일 |
| 이용 시간 | 이용 가능 시간대 |
| 홀딩 가능 | 일시정지 허용 여부 |
| 양도 가능 | 타인 양도 허용 여부 |
| 포인트 적립 | 포인트 적립 여부 |
| 키오스크 노출 | 키오스크 판매 여부 |

### 강조 표시

- 항목별로 가장 유리한 값에 하이라이트 표시 (예: 가장 저렴한 가격, 가장 긴 기간)

---

## 계정별 차이

| 계정 유형 | 볼 수 있는 것 | 할 수 있는 것 |
|-----------|--------------|--------------|
| 모든 직원 | 비교 화면 전체 | 상품 선택, 비교 조회 |
| 관리자 이상 | 비교 화면 + 삭제 확인 연동 | 비교 후 특정 상품 삭제 진입 가능 |

---

## 주요 UX 흐름

1. **상품 선택**: 화면 상단에서 비교할 상품 1~3개 선택
2. **비교 조회**: 선택한 상품의 항목별 정보가 나란히 표시
3. **상품 교체**: 열 상단의 X 버튼으로 상품 제거 후 다른 상품 선택
4. **상품 삭제**: 비교 화면에서 특정 상품 삭제 필요 시 → [DLG-P006-상품삭제확인]으로 이동

---

## 화면 상태

| 상태 | 조건 | 표시 내용 |
|------|------|-----------|
| 초기 상태 | 상품 미선택 | "비교할 상품을 선택하세요." 안내 |
| 상품 1개 선택 | 상품 1개만 선택됨 | 1열 정보 표시, 나머지 열에 "상품 추가" 영역 |
| 비교 중 | 상품 2~3개 선택 | 비교 테이블 전체 표시 |
