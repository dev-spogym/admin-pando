## 목적

고객에게 실제로 보여질 상품 카탈로그 화면을 미리 확인하는 모달입니다. 인쇄 또는 공유 전에 레이아웃과 내용을 점검할 수 있습니다.

---

## 구성요소

- 카탈로그 미리보기 영역 (실제 고객 화면과 동일한 레이아웃)
- 인쇄 버튼: 바로 인쇄 다이얼로그 실행
- 하단 버튼: **닫기**

---

## 언제 표시되나요

- SCR-P005(상품 카탈로그) 화면에서 "미리보기" 버튼을 클릭했을 때

---

## 주요 흐름

1. "미리보기" 버튼 클릭 → 모달 열림
2. 고객 화면 기준으로 카탈로그 레이아웃 표시
3. 인쇄 필요 시 "인쇄" 버튼 클릭 → 인쇄 다이얼로그 실행
4. "닫기" 클릭으로 모달 닫힘
