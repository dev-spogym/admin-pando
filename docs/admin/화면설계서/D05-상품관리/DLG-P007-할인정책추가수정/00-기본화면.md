## 목적

여러 조건을 동시에 적용하는 복합 할인 정책을 등록하거나 수정하는 모달입니다. 단순 정률/정액 외에 다중 조건(기간 + 금액 한도 + 대상 상품 등)을 조합한 정책을 설정할 수 있습니다.

---

## 구성요소

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 정책명 | 필수 | 복합 할인 정책 이름 |
| 기본 할인 유형 | 필수 | 정률(%) / 정액(원) |
| 기본 할인 값 | 필수 | 할인율 또는 금액 |
| 추가 조건 | 선택 | 최소 계약 기간, 최대 할인 한도 |
| 적용 기간 | 선택 | 정책 유효 시작일 ~ 종료일 |
| 활성 상태 | 선택 | 활성/비활성 |

하단 버튼: **취소** / **저장**

---

## 언제 표시되나요

- SCR-P004(할인 설정) 화면에서 복합 할인 정책 추가 또는 수정을 선택했을 때

---

## 주요 흐름

1. 정책명, 할인 유형, 할인 값 등 필수 항목 입력
2. 추가 조건 및 적용 기간 선택 (선택 사항)
3. "저장" 버튼 클릭
   - 필수 항목 미입력 시: 안내 메시지
   - 정상 저장 시: 모달 닫힘 + 목록 갱신
4. "취소" 클릭: 모달 닫힘
