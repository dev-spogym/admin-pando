## 화면 목적
로그인 직후 가장 먼저 마주하는 화면으로, 지점 운영 현황을 한눈에 파악하는 통합 현황판입니다.
계정 역할에 따라 보이는 정보 범위와 가능한 액션이 달라집니다.

## 주요 구성요소
- 오늘의 핵심 수치 요약 (오늘 출석, 당일 매출, 만료 예정 회원 수 등)
- 수업 현황 위젯 (오늘 수업 일정 및 예약 현황)
- 회원 현황 위젯 (신규 등록, 만료 예정 등)
- 매출 현황 위젯 (당월 누적 등)
- 빠른 액션 버튼 (회원 등록, 결제 처리 등)
- 엑셀 내보내기 버튼 (권한 있는 계정만)

## 계정별 차이
| 계정 유형 | 볼 수 있는 정보 | 가능한 액션 |
|---|---|---|
| 슈퍼관리자 | 전체 지점 KPI + 지점 필터 | 모든 액션 + 엑셀 내보내기 |
| 본사 최고관리자 / 센터장 / 매니저 | 소속 지점 전체 KPI | 모든 액션 + 엑셀 내보내기 |
| FC / 코치 | 오늘 일정 + 담당 출석 현황 | 읽기 전용 (엑셀 불가) |
| 스태프 | 오늘 일정 + 출석 현황 | 읽기 전용 (엑셀 불가) |

## 주요 UX 흐름
1. 로그인 완료 → 대시보드 자동 진입
2. 각 위젯을 클릭하면 해당 상세 화면으로 이동
3. 빠른 액션 버튼으로 주요 업무 즉시 시작
4. 슈퍼관리자는 상단 지점 필터로 다른 지점 현황 조회 가능

## 화면 상태
| 상태 | 설명 |
|---|---|
| 로딩 | 데이터 불러오는 중 (스켈레톤 표시) |
| 정상 | 모든 데이터 표시 |
| 빈 상태 | 오늘 데이터 없음 (신규 센터 등) |
| 오류 | 데이터 로드 실패 안내 |
| 오프라인 | 네트워크 연결 없음 안내 |
