## 화면 목적
사용자가 명시적으로 로그아웃을 요청했을 때 세션을 안전하게 종료하고 로그인 화면으로 이동하는 처리 흐름입니다.

## 주요 구성요소
- 로그아웃 확인 다이얼로그 (DLG-001)
- 로그아웃 처리 중 안내 (로딩)
- 완료 후 로그인 화면으로 자동 이동

## 주요 UX 흐름
1. 사이드바 하단 또는 프로필 메뉴에서 "로그아웃" 클릭
2. 로그아웃 확인 다이얼로그 표시 (DLG-001)
3. 확인 클릭 → 세션 종료 처리
4. 로그인 화면으로 자동 이동

## 화면 상태
| 상태 | 설명 |
|---|---|
| 확인 대기 | 로그아웃 확인 다이얼로그 표시 |
| 처리 중 | 세션 종료 처리 중 |
| 완료 | 로그인 화면으로 이동 |
