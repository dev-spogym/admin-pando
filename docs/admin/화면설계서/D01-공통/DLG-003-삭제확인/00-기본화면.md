## 화면 목적
회원, 상품, 직원, 수업 등 데이터를 삭제하기 전에 사용자의 의사를 재확인하는 공통 다이얼로그입니다.
삭제는 되돌릴 수 없기 때문에 명시적인 확인 단계를 거칩니다.

## 주요 구성요소
- 삭제 경고 아이콘
- "삭제하시겠습니까?" + 대상 항목명 표시
- "삭제 후에는 복구할 수 없습니다" 안내 문구
- 삭제 확인 버튼 (강조색)
- 취소 버튼

## 주요 UX 흐름
1. 삭제 버튼 클릭 → 이 다이얼로그 표시
2. 삭제 확인 → 삭제 처리 → 목록 화면으로 이동 + 완료 토스트
3. 취소 → 다이얼로그 닫고 현재 화면 유지

## 화면 상태
| 상태 | 설명 |
|---|---|
| 표시 | 삭제 확인 메시지 |
| 삭제 중 | 처리 중 (버튼 비활성) |
| 삭제 완료 | 완료 토스트 후 목록으로 이동 |
| 삭제 실패 | 오류 안내 메시지 |
