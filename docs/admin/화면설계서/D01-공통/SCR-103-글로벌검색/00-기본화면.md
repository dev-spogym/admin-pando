## 화면 목적
어느 화면에서든 상단 검색창을 통해 회원, 직원, 수업, 상품 등을 빠르게 찾을 수 있는 통합 검색 기능입니다.

## 주요 구성요소
- 상단 검색 입력창 (클릭 또는 단축키로 열기)
- 검색어 입력 영역
- 검색 결과 드롭다운 (카테고리별 구분)
  - 회원
  - 직원
  - 수업
  - 상품
- 최근 검색어 목록
- 결과 클릭 시 해당 상세 화면으로 이동

## 계정별 차이
| 계정 유형 | 검색 가능 범위 |
|---|---|
| 슈퍼관리자 | 전체 지점 데이터 |
| 센터장 / 매니저 / FC | 소속 지점 데이터만 |
| 트레이너 / 스태프 | 담당 회원 및 수업만 |

## 주요 UX 흐름
1. 상단 검색창 클릭 (또는 Cmd+K) → 검색 오버레이 열림
2. 검색어 입력 시 실시간으로 결과 표시
3. 결과 항목 클릭 → 해당 화면으로 이동
4. ESC 키 또는 바깥 클릭으로 닫기

## 화면 상태
| 상태 | 설명 |
|---|---|
| 닫힘 | 상단 검색 아이콘/바만 표시 |
| 열림 (초기) | 최근 검색어 표시 |
| 검색 중 | 입력하면서 실시간 결과 표시 |
| 결과 없음 | "검색 결과가 없습니다" 안내 |
