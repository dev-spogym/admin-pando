## 화면 목적
회원 만료 예정, 수업 취소, 결제 완료 등 주요 운영 이벤트를 실시간으로 알려주는 알림 모음 패널입니다.

## 주요 구성요소
- 상단 알림 아이콘 (미읽음 수 배지 표시)
- 알림 목록 (시간순 정렬)
  - 알림 유형 아이콘
  - 알림 내용 요약
  - 발생 시각
  - 읽음/미읽음 상태
- 전체 읽음 처리 버튼
- 전체 삭제 버튼 (권한 있는 계정만)
- 알림 설정 바로가기

## 계정별 차이
| 계정 유형 | 받는 알림 종류 | 관리 가능 액션 |
|---|---|---|
| 슈퍼관리자 | 전체 지점 시스템 알림 | 전체 읽음 / 전체 삭제 / 설정 |
| 본사 관리자 / 센터장 / 매니저 | 소속 지점 운영 알림 | 전체 읽음 / 전체 삭제 / 설정 |
| FC / 코치 | 담당 수업·회원 관련 알림 | 개별 읽음 / 개별 삭제만 |
| 스태프 | 출석·회원 관련 알림 | 개별 읽음 / 개별 삭제만 |

## 주요 알림 유형
| 알림 유형 | 예시 |
|---|---|
| 회원 만료 예정 | "홍길동 회원 이용권이 3일 후 만료됩니다" |
| 결제 완료 | "김철수 회원 결제가 완료되었습니다" |
| 수업 취소 | "오전 10시 필라테스 수업이 취소되었습니다" |
| 시스템 공지 | "시스템 점검이 예정되어 있습니다" |

## 주요 UX 흐름
1. 상단 알림 아이콘 클릭 → 알림 패널 열림
2. 알림 항목 클릭 → 관련 화면으로 이동 + 읽음 처리
3. "전체 읽음" 버튼으로 한 번에 처리
4. ESC 또는 바깥 클릭으로 패널 닫기

## 화면 상태
| 상태 | 설명 |
|---|---|
| 미읽음 있음 | 아이콘에 숫자 배지 표시 |
| 전부 읽음 | 배지 없음 |
| 알림 없음 | "새 알림이 없습니다" 안내 |
