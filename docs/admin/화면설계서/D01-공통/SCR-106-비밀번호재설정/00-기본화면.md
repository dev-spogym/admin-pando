## 화면 목적
비밀번호를 잊어버린 사용자가 이메일 인증을 통해 새 비밀번호를 설정할 수 있는 화면입니다.

## 주요 구성요소
**1단계 - 이메일 입력**
- 이메일 입력란
- 인증 메일 발송 버튼
- 로그인 화면으로 돌아가기 링크

**2단계 - 인증 메일 확인 안내**
- 이메일 발송 완료 안내 메시지
- 이메일 재발송 버튼

**3단계 - 새 비밀번호 설정** (메일 링크 클릭 후 진입)
- 새 비밀번호 입력란
- 새 비밀번호 확인 입력란
- 비밀번호 변경 완료 버튼

## 계정별 차이
모든 계정 동일하게 적용 (비로그인 상태에서 접근)

## 주요 UX 흐름
1. 로그인 화면에서 "비밀번호 재설정" 클릭 → 이 화면 진입
2. 가입된 이메일 입력 → 인증 메일 발송
3. 이메일 수신 → 메일의 링크 클릭
4. 새 비밀번호 입력 및 확인 → 변경 완료
5. 로그인 화면으로 자동 이동

## 화면 상태
| 상태 | 설명 |
|---|---|
| 이메일 입력 | 1단계: 이메일 입력 대기 |
| 메일 발송 완료 | 2단계: 인박스 확인 안내 |
| 새 비밀번호 입력 | 3단계: 새 비밀번호 설정 |
| 완료 | 변경 완료 후 로그인 안내 |
| 링크 만료 | 인증 링크 유효시간 초과 안내 |
