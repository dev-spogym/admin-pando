## 화면 목적
사용자가 등록/수정 화면에서 저장하지 않은 내용이 있는 상태로 다른 화면으로 이동하려 할 때, 데이터 손실을 방지하기 위해 표시하는 경고 다이얼로그입니다.

## 주요 구성요소
- "저장하지 않은 내용이 있습니다" 경고 메시지
- 나가기 버튼 (변경사항 버리고 이탈)
- 계속 편집 버튼 (다이얼로그 닫고 편집 계속)

## 언제 표시되나요
- 회원 등록/수정 중 다른 메뉴 클릭
- 상품 등록/수정 중 뒤로가기
- 직원 등록/수정 중 다른 화면 이동
- 기타 입력 폼이 있는 화면에서 미저장 이탈 시도

## 주요 UX 흐름
1. 미저장 상태에서 화면 이탈 시도
2. 이 다이얼로그 자동 표시
3. "나가기" → 변경사항 취소 후 이동
4. "계속 편집" → 다이얼로그 닫고 현재 화면 유지
