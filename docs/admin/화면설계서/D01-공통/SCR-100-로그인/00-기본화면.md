## 화면 목적
관리자가 이메일과 비밀번호로 본인을 확인하고 CRM 시스템에 접속하는 유일한 진입 화면입니다.
로그인 후에는 계정 역할에 따라 각자의 전용 화면으로 자동 이동합니다.

## 주요 구성요소
- 브랜드 로고 및 서비스명
- 이메일 입력란
- 비밀번호 입력란 (표시/숨김 전환 버튼 포함)
- 로그인 상태 유지 체크박스
- 로그인 버튼
- 비밀번호 재설정 링크

## 계정별 로그인 후 이동 화면
| 계정 유형 | 로그인 후 이동 |
|---|---|
| 슈퍼관리자 | 슈퍼 대시보드 |
| 본사 최고관리자 / 센터장 / 매니저 / FC | 지점 대시보드 |
| 트레이너 | 수업 캘린더 |
| 스태프 / 프론트 | 출석 관리 |

## 주요 UX 흐름
1. 이메일 + 비밀번호 입력 후 로그인 버튼 클릭
2. 2단계 인증(2FA) 활성 계정은 추가 인증 코드 입력 단계 진행
3. 인증 성공 → 계정 역할에 맞는 화면으로 자동 이동
4. 이미 로그인된 상태라면 로그인 화면 없이 대시보드로 바로 이동

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 이메일·비밀번호 입력 대기 |
| 로그인 중 | 버튼 클릭 후 처리 중 (버튼 비활성) |
| 오류 | 잘못된 정보 입력 시 안내 메시지 표시 |
| 2단계 인증 | 추가 인증 코드 입력 단계 |
| 서비스 점검 | 시스템 점검 중 안내 화면 |
