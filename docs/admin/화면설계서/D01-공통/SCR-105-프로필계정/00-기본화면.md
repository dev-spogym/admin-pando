## 화면 목적
현재 로그인한 계정의 개인 정보를 확인하고 수정하는 화면입니다.
이름, 연락처, 비밀번호 변경 등 기본 계정 관리를 할 수 있습니다.

## 주요 구성요소
- 프로필 사진 (업로드/변경)
- 이름
- 이메일 (변경 불가 - 읽기 전용)
- 연락처
- 소속 지점명
- 역할/직책
- 비밀번호 변경 버튼
- 저장 버튼
- 2단계 인증 설정 (활성화/비활성화)
- 로그아웃 버튼

## 계정별 차이
| 계정 유형 | 특이사항 |
|---|---|
| 슈퍼관리자 | 모든 항목 수정 가능 |
| 일반 계정 | 역할·소속 지점은 수정 불가 (관리자만 변경 가능) |

## 주요 UX 흐름
1. 상단 우측 프로필 아이콘 클릭 → 프로필 메뉴 열림
2. "내 계정" 또는 "프로필" 클릭 → 이 화면으로 이동
3. 정보 수정 후 저장 버튼 클릭 → 완료 안내
4. 비밀번호 변경 클릭 → 비밀번호 재설정 화면으로 이동

## 화면 상태
| 상태 | 설명 |
|---|---|
| 기본 | 현재 정보 표시 |
| 편집 중 | 입력 필드 활성화 |
| 저장 중 | 저장 버튼 처리 중 |
| 저장 완료 | 완료 토스트 메시지 표시 |
