## 화면 목적
사용자가 일정 시간 동안 아무 활동을 하지 않거나 세션이 만료되었을 때 자동으로 표시되는 알림 다이얼로그입니다.
현재 작업 중이던 위치를 기억해두었다가 재로그인 후 바로 돌아올 수 있습니다.

## 주요 구성요소
- "세션이 만료되었습니다" 안내 메시지
- 재로그인 버튼
- 현재 페이지 위치 안내 (재로그인 후 복귀 예정)

## 주요 UX 흐름
1. 세션 만료 감지 → 현재 작업 화면 위에 다이얼로그 자동 표시
2. 재로그인 버튼 클릭 → 로그인 화면으로 이동 (현재 URL 기억)
3. 로그인 완료 → 만료 전 화면으로 자동 복귀

## 화면 상태
| 상태 | 설명 |
|---|---|
| 표시 | 세션 만료 안내 다이얼로그 열림 |
| 재로그인 이동 | 로그인 화면으로 이동 |
