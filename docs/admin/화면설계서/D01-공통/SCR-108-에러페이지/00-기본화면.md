## 화면 목적
존재하지 않는 페이지 접근, 권한 없는 페이지 접근, 서버 오류 등이 발생했을 때 사용자에게 상황을 안내하고 정상 화면으로 돌아갈 수 있게 하는 화면입니다.

## 주요 구성요소
- 오류 유형 안내 아이콘
- 오류 제목 (예: "페이지를 찾을 수 없습니다")
- 오류 설명 메시지
- 대시보드로 돌아가기 버튼
- 이전 화면으로 돌아가기 링크

## 오류 유형별 안내
| 오류 유형 | 제목 | 설명 |
|---|---|---|
| 404 (없는 페이지) | 페이지를 찾을 수 없습니다 | 주소를 다시 확인해주세요 |
| 403 (권한 없음) | 접근 권한이 없습니다 | 관리자에게 문의하세요 |
| 500 (서버 오류) | 일시적인 오류가 발생했습니다 | 잠시 후 다시 시도해주세요 |
| 503 (서비스 점검) | 서비스 점검 중입니다 | 점검 완료 후 이용 가능합니다 |

## 주요 UX 흐름
1. 잘못된 접근 발생 → 이 화면으로 자동 이동
2. 오류 내용 확인
3. "대시보드로 돌아가기" 클릭 → 메인 화면으로 이동
