## 4. 인터랙션 정의

날짜 범위 선택, 유형 필터, 상태 필터, 통계 카드 갱신, 엑셀 다운로드

## 5. 에러 및 상태 처리

### 에러 메시지

- 데이터 없음: '해당 기간 매출이 없습니다'
- 다운로드 실패: 재시도 안내
