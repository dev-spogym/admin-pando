## 4. 인터랙션 정의

실시간 유효성 검사, 연락처 자동 포맷, 주소 검색 API 연동

## 5. 에러 및 상태 처리

### 에러 메시지

- 필수 항목 미입력: 필드 하단 에러 메시지
- 중복 연락처: '이미 등록된 연락처입니다'
- 서버 오류: 토스트 알림

## 6. 접근성 체크리스트

- 필수 항목 aria-required
