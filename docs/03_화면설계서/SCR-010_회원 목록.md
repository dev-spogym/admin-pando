## 4. 인터랙션 정의

실시간 검색, 필터 조합, 정렬, 페이지네이션, 엑셀 다운로드

## 5. 에러 및 상태 처리

### 에러 메시지

- 검색 결과 없음: '조건에 맞는 회원이 없습니다'
- 로딩 실패: 스켈레톤 + 재시도

## 6. 접근성 체크리스트

- 테이블 aria-label, 스크린리더 페이지 안내

## 7. 반응형 체크

- 테이블 반응형 스크롤

### Web 특이사항
- 테이블 가로스크롤 (좁은 화면)
