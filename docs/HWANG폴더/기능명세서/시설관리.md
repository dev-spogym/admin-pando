# 시설관리 기능명세서

## 1. 목적

- 락커, 카드, 공간, 운동복 등 실물 자산 운영 기준을 정리한다.
- 회원 운영과 시설 운영이 끊기지 않게 만든다.

## 2. 포함 범위

- 락커
- 사물함 배정
- RFID/밴드 카드
- 운동룸
- 골프 타석
- 운동복

## 3. 핵심 기능

- 자산 상태 조회
- 자산 배정/회수
- 상태 변경
- 이력 확인
- 만료 임박 또는 점검 대상 표시

## 4. 운영 규칙

- 자산은 `마스터`, `현재상태`, `배정대상`, `이력`으로 본다.
- 고장과 비활성은 같은 상태가 아니다.
- 락커/옵션 자산 만료 기준은 이용권 자동화 정책과 연결한다.
- 회원 탈퇴 또는 이관 시 연결 자산 회수 체크리스트를 제공한다.

## 5. 자산별 기준

### 5.1 락커

- 상태: 사용가능, 사용중, 만료임박, 만료, 고장, 비활성

### 5.2 카드

- 카드 상태와 회원 매핑 상태를 분리한다.

### 5.3 운동룸/타석

- 예약형 자산으로 본다.

### 5.4 운동복

- 재고형 자산으로 본다.

## 6. 권한 기준

| 기능 | 최소 권한 |
| --- | --- |
| 자산 조회 | staff |
| 자산 배정/회수 | staff |
| 고장/점검 처리 | manager |
| 일괄 배정 | manager |

## 7. 퍼블리싱 포인트

- 시설 화면은 숫자보다 상태 가시성이 중요하다.
- 그리드 화면은 셀만 봐도 현재 사용 가능 여부가 읽혀야 한다.
- 고장, 만료, 사용중은 색이 겹치면 안 된다.
