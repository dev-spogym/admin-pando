# FitGenie CRM 기능명세서 — HWANG 운영 우선본

## 목적

- 이 폴더는 기존 `docs/기능명세서`를 수정하지 않고, 실제 운영 기준으로 다시 정리한 기능명세서 세트다.
- 원본 기능명세서의 범위를 참고하되, 퍼블리싱과 화면 기획에 바로 쓸 수 있도록 핵심 기능과 규칙만 남겼다.

## 사용 원칙

- 운영 기준 판단은 이 폴더를 우선한다.
- 세부 구현 방식, API 연결, DB 스키마는 범위에서 제외한다.
- 본사와 지점의 정책 소유권은 명확히 분리한다.
- 자동화와 만료 알림은 본사 정책 세트 방식으로 본다.
