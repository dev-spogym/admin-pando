# SCR-H104 사이드바 네비게이션 — 기본화면

## 1. 화면 목적

- 관리자 웹 전역 내비게이션 컴포넌트다.

## 2. 핵심 기능

- 권한별 메뉴 노출
- 현재 메뉴 강조
- 조직 범위 표시
- 로그아웃 진입

## 3. 퍼블리싱 포인트

- 본사 메뉴와 지점 메뉴가 섞여 보이면 안 된다.
- 모바일에서는 드로어, 데스크톱에서는 고정 사이드바 기준으로 본다.
