# SCR-H203 회원 등록수정 — 기본화면

## 1. 화면 목적

- 신규 등록과 기존 정보 수정을 처리한다.

## 2. 핵심 기능

- 기본정보 입력
- 연락처 중복 확인
- 지점/담당자 지정
- 저장 검증

## 3. 퍼블리싱 포인트

- 필수 입력과 선택 입력을 분리해서 과도한 입력 피로를 줄인다.
