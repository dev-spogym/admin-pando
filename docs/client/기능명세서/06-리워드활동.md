## 0. 도입 배경

기존 리워드 도메인은 `쿠폰(MA-135) + 마일리지(MA-136) + 배지(MA-137)` 3종으로 정의되어 있다.

운영 시나리오에서는 회원의 누적 결제 / 활동량을 등급화하고, 친구 추천을 통해 양쪽 회원에게 리워드를 지급하며, 회원이 자신의 활동을 차트로 시각화해 동기를 부여받는 흐름이 추가로 필요하다.

본 문서는 기존 리워드 화면을 변경하지 않고, 다음 세 도메인을 `MA-5XX / MFN-500` 시리즈로 신규 정의한다.

- 멤버십 등급 — 누적 이용 금액에 따른 5단계 등급(BRONZE → VVIP)
- 친구 초대 — 추천 코드 + 단계별 리워드(가입 / 결제 / 후기)
- 활동 이력 — 누적 통계, 월별 방문/결제 추이, 카테고리 비중, 마일스톤 배지, 타임라인

## 1. 기본 정책

### 1.1 멤버십 등급 정책

- 등급 산정 기준: `누적 결제 금액(원)`.
- 등급 임계값: `BRONZE 0원 / SILVER 50만원 / GOLD 100만원 / VIP 250만원 / VVIP 500만원`.
- 회원은 항상 단일 등급을 가지며, 등급은 누적 결제액 변경 시 자동 재산정한다.
- 다음 등급까지 진척률(0~100)을 회원 화면에 노출한다.
- 등급별 혜택 카테고리: `할인 / 인바디 / 우선예약 / 락커 / 프라이빗 / 매니저 / 후기 / 선물`.

### 1.2 친구 초대 정책

- 회원은 고유 추천 코드를 가진다 (`MA-501`).
- 추천 단계는 3단계: `친구 가입 → 첫 결제 → 첫 후기 작성`.
- 단계별 리워드: `5,000원 / 5,000원 / 3,000원` (한 친구당 최대 13,000원).
- 추천한 친구 목록과 단계별 진행 상태를 추천인 본인 화면에서 확인한다.
- 한 ID당 추천 가능 인원은 무제한.
- 적립금은 친구의 첫 결제 후 7일 이내 자동 지급된다.

### 1.3 활동 이력 정책

- 활동 통계 4종: `누적 결제 금액 / 방문 횟수 / 수업 횟수 / 후기 수`.
- 월별 추이: 최근 12개월 방문 / 결제 추이를 라인/바 차트로 표시한다.
- 카테고리 비중: PT / 필라테스 / 요가 / 헬스 / 골프 등 카테고리별 결제 비중을 도넛으로 표시한다.
- 베스트 강사 / 자주 가는 센터: 회원 활동 기반 자동 추출.
- 마일스톤 배지: 첫 PT 완주 / 10회 연속 출석 / 누적 결제 200만원 등 8종.
- 활동 타임라인: 최근 활동 12건을 `방문 / 예약 / 결제 / 후기` 4종 액션으로 표시한다.
- 활동 0건 회원(신규)은 빈 상태 메시지 + `둘러보기` CTA를 노출한다.

## 3. 데이터 도메인 정의

### 3.1 신규 엔티티

| 엔티티 | 설명 | 핵심 필드 |
|---|---|---|
| MembershipGrade | 멤버십 등급 메타 | id, name, threshold, color, summary, benefits[] |
| GradeBenefit | 등급 혜택 | title, description, icon |
| ReferralFriend | 추천한 친구 | id, name, joinedAt, status, rewardAmount |
| ReferralStep | 추천 단계 | step, title, description, reward, status |
| ActivityStats | 누적 활동 통계 | totalSpend, visitCount, lessonCount, reviewCount |
| MonthlyVisitPoint / MonthlyPaymentPoint | 월별 추이 | month, visits / amount |
| CategoryShare | 카테고리 비중 | name, value, color |
| BestInstructor / FavoriteCenter | 베스트 항목 | name, totalSessions / visitCount, lastVisitedAt |
| MilestoneBadge | 마일스톤 배지 | id, title, description, earnedAt, earned |
| TimelineEntry | 활동 타임라인 | id, date, action, title, meta, amount |

### 3.2 기존 엔티티와의 연결

- 멤버십 등급의 `누적 결제 금액`은 `OrderItem.amount`(`MFN-420`) 합계로 계산한다.
- 추천 적립금은 기존 `MFN-114 마일리지`에 적립한다.
- 마일스톤 배지는 기존 `MA-137 배지 컬렉션`과 분리해 활동 기반 자동 부여 카테고리로 운용한다.
- 카테고리 비중은 마켓 카테고리(`MarketProduct.category`)와 동일 키를 사용한다.

## 6. 주요 기능 상세

### MFN-500. 멤버십 등급 조회

| 항목 | 내용 |
|---|---|
| 목적 | 회원이 자신의 등급, 누적 결제액, 다음 등급 진척률을 한눈에 본다 |
| 입력 | 회원 ID |
| 출력 | 현재 등급 카드(그라디언트), 누적 결제 금액, 다음 등급까지 남은 금액, 진척 바 |
| 비고 | 등급별 혜택은 카드 / 매트릭스로 함께 노출 |

### MFN-510. 추천 코드 발급 / 공유

| 항목 | 내용 |
|---|---|
| 목적 | 회원이 친구에게 본인 추천 코드를 공유한다 |
| 출력 | 추천 코드, 카카오톡 / 링크 / 복사 / 문자 공유 진입 |
| 액션 | 공유 시트(`ShareSheet`) 진입 |
| 비고 | 공유 시 OG 메타로 추천 가입 화면 딥링크 포함 |

### MFN-512. 추천한 친구 목록

| 항목 | 내용 |
|---|---|
| 목적 | 회원이 본인 추천으로 가입한 친구의 진행 상태를 본다 |
| 출력 | 친구 카드(이름 / 가입일 / 단계 칩 / 누적 리워드 금액) |
| 정책 | 단계: `가입 완료(neutral) / 결제 완료(primary) / 후기 작성(success)` |

### MFN-520. 활동 통계 조회

| 항목 | 내용 |
|---|---|
| 목적 | 회원이 자신의 누적 활동을 4가지 핵심 지표로 본다 |
| 출력 | 누적 결제액, 방문 횟수, 수업 횟수, 후기 수 |
| 비고 | 모든 지표 0건이면 신규 회원 빈 상태로 분기 |

### MFN-521. 월별 방문 / 결제 차트

| 항목 | 내용 |
|---|---|
| 목적 | 최근 12개월의 방문 / 결제 추이를 시각화한다 |
| 출력 | 라인 차트(방문) + 바 차트(결제) 또는 동일 화면 토글 |
| 비고 | 데이터 0건 월은 0으로 표시 |

### MFN-524. 마일스톤 배지 컬렉션

| 항목 | 내용 |
|---|---|
| 목적 | 활동 기반 자동 부여 배지(8종)를 회원이 본다 |
| 출력 | 배지명 / 조건 / 획득일 / 획득 여부 |
| 배지 8종 | 첫 PT 완주, 10회 연속 출석, 첫 후기, 한 달 개근, 누적 결제 200만원, 친구 초대 마스터, 얼리버드, 마라토너 |

### MFN-525. 활동 타임라인

| 항목 | 내용 |
|---|---|
| 목적 | 최근 활동 12건을 시간 역순으로 본다 |
| 출력 | 일자, 액션 종류(방문/예약/결제/후기), 제목, 메타(시간/금액/적립 등) |
