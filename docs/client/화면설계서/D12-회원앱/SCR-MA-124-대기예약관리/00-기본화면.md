## 화면 목적

만석인 수업에 대기 신청한 목록을 회원 앱에서 조회하고 관리한다. 대기 순번, 확정 전환 여부, 알림 설정 등을 확인하며 필요 시 대기 신청을 취소할 수 있다.

## 주요 구성요소

### 대기 예약 목록

| 항목 | 설명 |
|------|------|
| 수업명 | 대기 신청한 수업 이름 |
| 수업 일시 | 수업 날짜 및 시작 시각 |
| 강사명 | 담당 강사 이름 |
| 대기 순번 | 현재 대기 순서 (예: 2번째) |
| 신청일 | 대기 신청한 날짜 |
| 상태 | 대기 중 / 확정 전환 / 취소됨 배지 |

### 빈 상태

대기 신청한 수업이 없는 경우 '대기 신청한 수업이 없습니다.' 안내 및 수업 예약 화면으로 이동 링크 표시.

### 버튼 (각 항목별)

| 버튼 | 표시 조건 | 설명 |
|------|-----------|------|
| 대기 취소 | 대기 중 상태 | 해당 수업 대기 신청 취소 |
| 예약 확인 | 확정 전환 상태 | 확정된 예약 상세 화면으로 이동 |

### 알림 안내

대기 순번이 앞당겨지거나 예약 확정 시 앱 푸시 알림이 발송됨을 안내한다.

## 계정별 차이

회원 앱 화면으로 로그인한 회원 본인의 대기 예약 목록만 조회 가능하다.

## 주요 UX 흐름

1. 회원 앱 메뉴 → '대기 예약' 탭 진입
2. 현재 대기 중인 수업 목록 확인
3. 대기 취소 버튼 → 취소 확인 팝업 → 확인 → 대기 취소 처리
4. 확정 전환된 항목 → '예약 확인' 버튼으로 예약 상세 이동
5. 수업 자리 발생 시 푸시 알림 수신 → 앱에서 예약 자동 확정 또는 수락 처리

## 화면 상태

| 상태 | 설명 |
|------|------|
| 대기 목록 있음 | 대기 예약 항목 목록 표시 |
| 빈 상태 | '대기 신청한 수업이 없습니다.' 안내 |
| 확정 전환 항목 있음 | 해당 항목 강조 배지 + '예약 확인' 버튼 표시 |
| 취소 처리 중 | 해당 항목 로딩 표시 |
| 로딩 | 목록 조회 중 스켈레톤 표시 |
