## 화면 목적

회원 앱에서 골프 타석 예약의 상세 정보를 조회하고, 필요 시 예약을 취소한다. 예약 일시, 타석 번호, 이용 조건 등 예약 확정 내용을 한눈에 확인할 수 있다.

## 주요 구성요소

### 예약 정보 섹션

| 항목 | 설명 |
|------|------|
| 예약 상태 | 예약 확정 / 취소 / 대기 배지 |
| 예약 번호 | 예약 고유 식별 번호 |
| 예약 일시 | 이용 날짜 및 시작·종료 시각 |
| 타석 번호 | 배정된 골프 타석 번호 |
| 이용 시간 | 총 이용 시간 (예: 1시간) |
| 이용권 | 차감될 이용권 종류 및 잔여 횟수 |

### 이용 안내 섹션

| 항목 | 설명 |
|------|------|
| 센터 위치 | 골프 타석 위치 안내 |
| 준비물 | 이용 시 필요 용품 안내 |
| 주의사항 | 이용 규칙 및 취소 정책 안내 |

### 취소 정책 안내

| 항목 | 설명 |
|------|------|
| 무료 취소 기한 | 예약일 X시간 전까지 취소 시 페널티 없음 |
| 유료 취소 기한 | 기한 이후 취소 시 이용권 차감 안내 |

### 버튼

| 버튼 | 표시 조건 | 설명 |
|------|-----------|------|
| 예약 취소 | 예약 확정 상태이며 취소 가능 기한 내 | 예약 취소 처리 |
| 닫기 / 뒤로가기 | 항상 | 이전 화면으로 이동 |

## 계정별 차이

회원 앱 화면으로 로그인한 회원 본인의 예약 정보만 조회 가능하다. 별도 계정 구분 없이 회원 단일 계정으로 운영된다.

## 주요 UX 흐름

1. 회원 앱 내 예약 목록 → 골프 예약 항목 탭 → 상세 화면 진입
2. 예약 정보 확인
3. 취소 가능 기한 내인 경우 → '예약 취소' 버튼 표시
4. 취소 버튼 클릭 → 취소 확인 팝업 → 확인 → 취소 처리 완료
5. 취소 완료 후 예약 목록으로 자동 이동

## 화면 상태

| 상태 | 설명 |
|------|------|
| 예약 확정 (취소 가능) | 예약 정보 + 취소 버튼 표시 |
| 예약 확정 (취소 불가) | 예약 정보만 표시, 취소 버튼 비표시 또는 비활성 |
| 취소된 예약 | '취소된 예약입니다.' 안내, 취소 버튼 비표시 |
| 이용 완료 | '이용이 완료된 예약입니다.' 안내 |
| 로딩 | 예약 정보 조회 중 스켈레톤 표시 |
| 조회 실패 | 오류 안내 및 재시도 유도 |
