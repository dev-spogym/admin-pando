# FitGenie 회원앱 화면설계서 — 탐색 플랫폼 (NEW)

> 작성일: 2026-04-29
> 버전: 0.1 (신규 제안)
> 문서 성격: 마켓플레이스 발견(Discover) 레이어 화면 기획
> 기준: `APP-IA.md`, `기능명세서/04-탐색플랫폼.md`
> 연계: BODY SWITCH 류 멀티센터 탐색 플랫폼 패턴

---

## 1. 문서 사용 원칙

- 본 문서는 기존 단일 센터 회원앱(`MA-1XX`) 화면을 변경하지 않고, 신규 마켓플레이스 화면(`MA-3XX`)만 정의한다.
- 모든 화면은 기존 회원앱 디자인 시스템을 그대로 사용한다 (Pretendard, primary 청록 #0E7C7B, rounded-card, shadow-card-soft 등).
- 기획 단계 문서이며 구현 경로/파일 구조 정보는 포함하지 않는다.
- 화면 ID는 `MA-3XX` 시리즈를 사용하고 기능 ID는 `MFN-3XX`를 따른다.

---

## 2. 사이트맵 (확장)

기존 IA의 회원 모드에 `둘러보기 (Discover)` 도메인을 병렬로 추가한다.

```text
회원 모드 (기존 단일 센터, MA-1XX)
└── 홈 / 출석 / 예약 / 이용권 / MY

회원 모드 (신규 둘러보기, MA-3XX)
├── 둘러보기 홈
│   └── MA-300 둘러보기 홈
│       ├── 지역 선택 / 위치
│       ├── 프로모션 배너 슬라이더
│       ├── 내 주변 / 역세권 위치 카드
│       ├── 4 quick action (예약 / 회원권 / 커뮤니티 / 건강기록)
│       ├── 추천 센터 가로 스크롤
│       └── 추천 강사 가로 스크롤
│
├── 센터
│   ├── MA-310 내 주변 센터
│   ├── MA-311 센터 카테고리 검색
│   ├── MA-312 센터 상세
│   └── MA-313 지도 보기
│
├── 강사
│   ├── MA-320 강사 둘러보기
│   └── MA-321 강사 상세 (마켓)
│
├── 스크랩
│   └── MA-330 스크랩
│
├── 메신저
│   ├── MA-340 메신저 허브
│   └── MA-341 1:1 대화방
│
└── 리뷰
    ├── MA-350 센터 리뷰 작성
    └── MA-351 센터 리뷰 목록
```

탭바는 `홈 / 센터 / 강사 / 스크랩 / MY` 5개 마켓 패턴으로 재구성하거나, 기존 5개를 유지하고 둘러보기 진입을 홈 안에 배치하는 두 안 중 선택한다 (§7 참고).

## 4. 핵심 화면 설계

### 4.1 MA-300 둘러보기 홈

**목적**

- 회원/비회원 모두 위치와 카테고리 기준으로 새 센터·강사를 발견한다.

**필수 요소**

- 상단: 행정구역 드롭다운(예: `강남구 논현동 ▾`), 우측 QR/알림 아이콘
- 프로모션 배너 슬라이더 (1/N 인디케이터)
- 위치 카드 2종 (`내 주변 센터`, `역세권 센터`)
- 4 quick action: `예약하기`, `내 회원권`, `커뮤니티`, `내 건강기록`
- 섹션 1 — `처음은 가볍게, 체험부터!` 가로 스크롤 카드
- 섹션 2 — `고민하지마, 리뷰 좋은 강사들!` 가로 스크롤 강사 카드
- 섹션 3 — `최근 본 센터` (있을 때만)
- 하단 탭바: 홈 활성

**예외/빈 상태**

- 위치 미허용 시: 행정구역 드롭다운만 활성, 위치 카드는 안내 텍스트로 대체
- 비로그인 시: `내 회원권 / 내 건강기록`은 로그인 유도 모달

---

### 4.2 MA-310 내 주변 센터

**목적**

- 단말 좌표 기준 가까운 센터 리스트를 정렬·필터해 본다.

**필수 요소**

- 상단 헤더: `내 주변` 제목 + 알림 아이콘
- 검색바: `장소, 종목으로 검색해보세요` 좌측 필터 토글 아이콘
- 정렬 칩 행: `최신순` / `거리순` / `리뷰순` / `체험권` (단일 선택)
- 결과 카드 (수직 카드):
  - 좌측 정사각형 이미지(센터 대표 사진)
  - 우측 정보: 센터명, 행정구역 주소, 거리(`내 위치에서 N.Nkm`), 별점(★ 0.0 후기 N), 정가/할인율/판매가
- 하단 우측 floating `지도보기` 버튼 → MA-313
- 카드 클릭 → MA-312

**예외/빈 상태**

- 결과 0건: `검색 조건에 맞는 센터가 없습니다 — 필터를 조정해보세요`
- 위치 미허용 시: 행정구역 수동 선택 모달

---

### 4.3 MA-311 센터 카테고리 검색

**목적**

- 1차 카테고리(피트니스/필라테스/골프 등) + 2차 상품 분류로 좁혀 검색한다.

**필수 요소**

- 상단 검색바: `원하시는 수업을 찾아보세요` (테두리 강조 청록)
- 1차 카테고리 가로 탭: `전체` `피트니스` `필라테스` `골프` `요가` `크로스핏` ... (좌우 스크롤)
- 2차 상품 칩 행: `전체` `수강권` `이용권` `체험권` `그룹` `개인` (랩핑)
- 결과 카드 (수직 큰 이미지 카드):
  - 16:9 대표 이미지
  - 카테고리 텍스트 (예: `필라테스 / 피트니스 / 골프 / 요가`)
  - 상품명 (`바디스위치 피트니스`)
  - 위치 텍스트
  - 우측: `대표 상품` 라벨 (있을 때)
  - 가격 라인: 정가 취소선 + 할인율 빨강 + 판매가
  - 시설 태그 칩 행: `샤워` `탈의실` `주차장` `운동복` `타올` `유산소기구` `개인레슨` ...
- 하단 탭바: 센터 활성

**예외/빈 상태**

- 결과 0건: `검색 결과가 없습니다 — 다른 카테고리를 선택하거나 필터를 줄여보세요`

---

### 4.4 MA-312 센터 상세

**목적**

- 한 센터의 정보·상품·리뷰·강사를 한 화면에서 본다.

**필수 요소**

- 상단: 대형 이미지 캐러셀 (스와이프, 페이지 인디케이터)
- 헤더 행: 뒤로가기, 공유, 스크랩 토글, 알림
- 센터 기본 정보: 카테고리, 센터명, 행정구역 주소, 거리, 별점/리뷰 수
- 가격 요약: 대표 상품 정가/할인율/판매가
- 시설 태그 칩 행
- 탭: `홈` `상품` `리뷰` `강사` `위치/영업`
  - 홈: 소개, 대표 상품 1~2개, 최근 리뷰 3개, 강사 3명
  - 상품: 모든 상품 카드 + 결제 진입 (`체험권 / 수강권 / 이용권 / 그룹 / 개인`)
  - 리뷰: 별점 분포 + 리뷰 카드 (정렬 가능)
  - 강사: 이 센터 소속 강사 그리드
  - 위치/영업: 지도, 주소, 영업시간, 휴무일, 연락처
- 하단 floating action: `센터에 문의` (메신저 진입) + `결제하기` (결제 진입)

**예외/빈 상태**

- 비로그인 + 결제 시도: 로그인 모달
- 비로그인 + 스크랩 시도: 로그인 모달

---

### 4.5 MA-313 지도 보기

**목적**

- 검색 결과를 지도 위에서 시각적으로 본다.

**필수 요소**

- 풀스크린 지도 + 상단 검색바 + 하단 미니 카드 슬라이더
- 마커: 가격/별점 라벨 표시
- 클러스터링 (줌 레벨에 따라 묶기)
- 마커 클릭 → 하단 미니 카드 강조 + 슬라이더 이동
- 미니 카드 클릭 → MA-312
- 우측 상단: 현 위치 재중심 버튼, `목록으로 보기` 토글 → MA-310

**예외**

- 위치 권한 거부 시: 지도 중심을 행정구역 기본 좌표로 설정

---

### 4.6 MA-320 강사 둘러보기

**목적**

- 카테고리/지역/평점 기준으로 강사를 탐색한다.

**필수 요소**

- 상단 검색바: `강사 이름, 종목 검색`
- 1차 카테고리 가로 탭: `PT` `필라테스` `요가` `골프` `크로스핏` ...
- 2차 필터 칩: `평점 4.5+` `여성 강사` `남성 강사` `체험 가능`
- 결과 카드 (그리드 또는 리스트):
  - 강사 프로필 이미지
  - 이름 + 소속 센터
  - 별점 + 리뷰 수
  - 전문 분야 태그 (예: `다이어트` `재활` `근력`)
- 카드 클릭 → MA-321

**예외**

- 결과 0건: `조건에 맞는 강사가 없습니다`

---

### 4.7 MA-321 강사 상세 (마켓)

**목적**

- 강사 한 명의 프로필, 후기, 가능 시간을 확인하고 1:1 톡 또는 예약 요청을 시작한다.

**필수 요소**

- 상단: 강사 프로필 이미지 + 이름 + 소속 센터(클릭 시 MA-312)
- 별점, 리뷰 수, 누적 수업, 보유 자격
- 자기 소개 본문
- 전문 분야 태그
- 후기 카드 3개 + 더 보기
- 가능 시간 캘린더 (요일별 빈 시간)
- 하단 floating action: `1:1 톡` (메신저 강사톡 진입), `예약 요청` (회원이 가입한 같은 센터 강사면 MA-125, 외부 강사면 문의로 유도)
- 우측 상단: 스크랩 토글

**예외**

- 비로그인 + 1:1 톡 / 예약 요청 / 스크랩: 로그인 모달

---

### 4.8 MA-330 스크랩

**목적**

- 회원이 저장한 모든 관심 항목을 한 곳에서 본다.

**필수 요소**

- 상단 헤더: `스크랩` + 편집 버튼
- 탭: `센터` `강사` `상품` `수업`
- 각 탭 결과 카드:
  - 센터 → MA-310 카드 동일 양식
  - 강사 → MA-320 카드 동일 양식
  - 상품 → MA-134 상품 카드 + 가격
  - 수업 → MA-120 수업 카드
- 카드 우상단 X 버튼으로 스크랩 해제

**예외/빈 상태**

- 빈 탭: `스크랩한 항목이 없습니다 — 마음에 드는 센터를 저장해보세요` + `센터 둘러보기` 버튼

---

### 4.9 MA-340 메신저 허브

**목적**

- 회원이 사람과 주고받는 모든 대화를 통합 관리한다.

**필수 요소**

- 상단 헤더: 뒤로/홈 + 제목 `메신저` + 알림 아이콘
- 탭: `센터 문의` / `강사톡` / `강의노트`
- 각 탭 리스트:
  - 좌측 아바타 (센터 로고 또는 강사 사진)
  - 중앙: 이름 + 마지막 메시지(미리보기) + 시간
  - 우측: 미읽음 카운트 배지
- 탭 상단에 `검색` 입력 (대화방 검색)
- 빈 상태: 탭별로 다름
  - 센터 문의 빈 상태: `센터 없습니다` (+ `센터 둘러보기 →` 진입)
  - 강사톡 빈 상태: `대화 중인 강사가 없습니다` (+ `강사 찾아보기 →`)
  - 강의노트 빈 상태: `아직 받은 강의 노트가 없습니다`

**행동**

- 리스트 항목 클릭 → MA-341 1:1 대화방
- 신규 대화는 MA-312 / MA-321의 floating action에서 시작

---

### 4.10 MA-341 1:1 대화방

**목적**

- 단일 대화 상대와 메시지를 주고받는다.

**필수 요소**

- 상단: 뒤로 / 상대 이름·아바타 / 옵션(차단·신고)
- 메시지 영역: 내 메시지(우측 청록 말풍선), 상대 메시지(좌측 회색 말풍선), 시간/읽음 표시
- 강의노트 탭에서 진입한 경우: 메시지 사이에 `수업 노트` 카드 형태로 임베드 (수업명, 일자, 코치 코멘트)
- 입력창: 텍스트 + 첨부(이미지/파일) + 전송 버튼
- 빠른 답장 버튼(권장): `예약 가능 시간이 있나요?`, `가격 문의드립니다`, `위치 알려주세요`

**예외/제약**

- 첨부 용량 제한: 이미지 5MB, 파일 10MB
- 신고/차단 시 즉시 대화 비활성화 + 안내 메시지

---

### 4.11 MA-350 센터 리뷰 작성

**목적**

- 회원이 방문/결제 이력이 있는 센터에 대해 별점과 후기를 남긴다.

**필수 요소**

- 상단: 센터 이름 + 카테고리
- 종합 별점: 1~5 슬라이더 (탭/슬라이드)
- 시설 항목별 별점: `시설 청결도`, `강사 친절`, `운동 효과`, `편의시설`, `가격 만족도`
- 본문: 최소 30자, 최대 500자
- 사진 첨부: 최대 5장
- 동의: `허위 후기는 삭제될 수 있음` 체크
- 하단 CTA: `리뷰 등록`

**예외**

- 출석/결제 이력 없음: 작성 불가 안내
- 이미 작성됨: 작성 30일 이내라면 수정 화면, 이후엔 보기 전용

---

### 4.12 MA-351 센터 리뷰 목록

**목적**

- 한 센터의 모든 리뷰를 본다.

**필수 요소**

- 상단: 평균 별점 + 리뷰 수 + 별점 분포 막대 (5★ ~ 1★ 비율)
- 정렬 옵션: `최신순` / `별점 높은 순` / `별점 낮은 순` / `사진 있는 순`
- 리뷰 카드:
  - 작성자(닉네임 + 마스킹) + `방문 인증` 배지
  - 별점 + 작성일
  - 본문 + 사진 (있을 때)
  - 좋아요 / 신고

**예외/빈 상태**

- 0건: `아직 리뷰가 없습니다 — 첫 리뷰를 남겨보세요`

## 6. 회원 화면 흐름

### 6.1 신규 사용자가 가까운 센터 발견 → 결제

```text
MA-300 둘러보기 홈
  → MA-310 내 주변 센터 (거리순)
    → MA-312 센터 상세
      → 상품 탭에서 체험권 선택
      → MA-140 결제하기 (기존)
        → MA-142 영수증 (기존)
        → 자동 등록되어 MA-131 이용권에 추가
```

### 6.2 강사 검색 → 1:1 톡 → 예약

```text
MA-300 또는 MA-320 강사 둘러보기
  → MA-321 강사 상세
    → 1:1 톡 클릭
    → MA-341 대화방 (강사톡)
    → 강사가 가능 시간 안내
    → MA-321 가능 시간 캘린더에서 예약 요청
    → 회원의 MA-122 내 예약에 승인 대기 추가
```

### 6.3 메신저 진입 → 강의노트 확인

```text
하단 탭 또는 홈 quick action
  → MA-340 메신저 허브 → 강의노트 탭
    → 트레이너가 보낸 노트 카드
    → MA-341 대화방에서 코멘트 답변
```

### 6.4 결제 후 리뷰 작성

```text
MA-131 이용권 / MA-122 수업 이력
  → 결제/방문 이력이 있는 센터
  → MA-312 센터 상세 → 리뷰 작성 진입
  → MA-350 센터 리뷰 작성
  → MA-351 리뷰 목록에 반영
```

## 7. 탭바 옵션

### 옵션 A — 마켓 패턴 5탭 (권장 안)

| 탭 | 진입 화면 | 비고 |
|---|---|---|
| 홈 | MA-300 둘러보기 홈 | 멀티센터 진입 + 내 센터 요약 카드 |
| 센터 | MA-310 / MA-311 | 내 주변 / 카테고리 검색 토글 |
| 강사 | MA-320 강사 둘러보기 | |
| 스크랩 | MA-330 | |
| MY | MA-130 마이페이지 | 내 센터 컨텍스트(이용권, 출석, QR, 예약 이력)는 MY 안에 묶음 |

기존 `MA-100 홈`은 MY → 내 센터 요약 카드의 진입점으로 통합되거나, MA-300 안의 `내 센터 카드` 영역에 흡수된다.

### 옵션 B — 기존 5탭 유지 + 홈 안에 둘러보기 섹션

| 탭 | 진입 화면 | 비고 |
|---|---|---|
| 홈 | MA-100 (기존) | 둘러보기 섹션을 본문 안에 추가 |
| 예약 | MA-120 | 그대로 |
| QR | MA-110 | 그대로 |
| 이용권 | MA-131 | 그대로 |
| MY | MA-130 | 그대로 |

이 안에서는 둘러보기 / 센터 / 강사 / 스크랩 / 메신저 / 리뷰 화면은 모두 홈 또는 MY의 하위 스택으로 진입한다.

## 8. 권장 진입 흐름

1. 비로그인 사용자: MA-300 → MA-310 / MA-311 → MA-312 까지 열람 가능, 결제/스크랩 시 로그인 모달.
