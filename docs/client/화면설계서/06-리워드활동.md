## 4. 핵심 화면 설계

### 4.1 MA-500 멤버십 등급

**목적**

- 회원이 자신의 현재 등급, 누적 결제액, 다음 등급까지의 진척률을 한눈에 본다.

**필수 요소**

- 상단 등급 카드:
  - 등급명(BRONZE / SILVER / GOLD / VIP / VVIP)
  - 등급별 그라디언트 배경
  - 한줄 요약 (예: `100만원+ · PT 10% 할인 + 무료 인바디`)
- 누적 결제액 + 다음 등급까지 남은 금액
- 진척 바 (0~100%)
- 5단계 등급 매트릭스 (가로 카드 또는 비교 표):
  - 등급명 / 임계값 / 핵심 혜택 1줄
- 현재 등급 혜택 카드 리스트 (아이콘 + 제목 + 설명):
  - icon: `discount / inbody / priority / locker / private / manager / review / gift`
- 다음 등급 혜택 미리보기 (잠금 처리)

**예외 / 빈 상태**

- 누적 결제 0원 (BRONZE 0%): `첫 결제로 등급을 시작해 보세요` + 둘러보기 진입

---

### 4.2 MA-501 친구 초대

**목적**

- 회원이 추천 코드를 공유하고, 추천한 친구의 진행 상태와 누적 적립을 본다.

**필수 요소**

- 상단 헤더: `친구 초대` + 알림
- 일러스트 + 헤드라인: `친구도, 나도 최대 13,000원 적립`
- 추천 코드 카드:
  - 코드 (예: `PANDO-A1B2`) + 복사 버튼
  - 우측: 카카오톡 / 문자 / 시스템 공유 버튼 → ShareSheet
- 단계 카드 3종 (가로 또는 세로):
  - ① 친구 가입 — 5,000원
  - ② 첫 결제 — 5,000원
  - ③ 첫 후기 — 3,000원
- 누적 추천 리워드 합계 박스 (이번 달 / 누적)
- 추천한 친구 목록:
  - 아바타(이니셜 + 색), 이름, 가입일, 단계 칩(`가입 완료 / 결제 완료 / 후기 작성`), 누적 리워드 금액

**예외 / 빈 상태**

- 추천 친구 0명: `친구를 처음 초대해 보세요!` + 공유 버튼

---

### 4.3 MA-502 활동 이력

**목적**

- 회원이 자신의 누적 활동을 통계 / 차트 / 배지 / 타임라인으로 본다.

**필수 요소**

- 상단 헤더: `활동 이력`
- 4지표 카드 (2x2 또는 가로 4):
  - 누적 결제 / 방문 횟수 / 수업 횟수 / 후기 수
- 월별 방문 라인 차트 (12개월) + 월별 결제 바 차트 (12개월) — recharts 기반
- 카테고리 비중 도넛: PT / 필라테스 / 요가 / 헬스 / 골프 (색 매핑 고정)
- 베스트 강사 카드 리스트 (이름 / 누적 수업 / 별점)
- 자주 가는 센터 카드 리스트 (센터명 / 지역 / 방문 횟수 / 마지막 방문일)
- 마일스톤 배지 8종 그리드 (획득 / 미획득 분리, 획득 시 날짜 표시)
- 활동 타임라인 12건:
  - 일자, 액션 칩(`방문 / 예약 / 결제 / 후기`), 제목, 메타(시간/금액/적립)

**예외 / 빈 상태**

- 활동 0건 (신규 회원): 일러스트 + `아직 활동 이력이 없어요` + `둘러보기 시작` 버튼
- 차트 데이터 0건 월: 0으로 표시

## 5. 디자인 시스템 매핑

| 요소 | 사용 컴포넌트 |
|---|---|
| 등급 카드 | `Card variant="gradient"` (등급별 색) |
| 진척 바 | `ProgressBar` |
| 혜택 카드 | `Card variant="soft"` + 아이콘 |
| 단계 카드 (3단) | `Card` + `Step indicator` |
| 친구 카드 | `ListItem` + `Avatar (이니셜)` + `Badge tone` |
| 4지표 카드 | `StatCard` |
| 차트 | recharts `LineChart` / `BarChart` / `PieChart` |
| 배지 그리드 | `Card variant="soft"` (획득 = 색, 미획득 = 회색) |
| 타임라인 | `Timeline` (날짜 그룹 헤더 + 항목) |
| 빈 상태 | `EmptyState` |

## 6. 회원 화면 흐름

### 6.1 결제 → 등급 상승 알림

```text
MA-403 주문 상세 → 결제 완료
  → MA-500 멤버십 등급에 누적액 즉시 반영
  → 등급 변경 시 PushToast 알림 + MA-500 자동 진입 유도
```

### 6.2 친구 초대 → 적립

```text
MA-501 친구 초대 → 추천 코드 공유
  → 친구가 가입 (단계 ①)
  → 친구 첫 결제 (단계 ②) → MA-501 친구 목록 갱신 + 마일리지 적립
  → 친구 첫 후기 (단계 ③) → 단계 완료 표시
```

### 6.3 활동 이력 → 배지 획득 → 등급 상승

```text
MA-502 활동 이력 갱신
  → 마일스톤 조건 충족 시 배지 자동 부여
  → 누적 결제 임계값 초과 시 MA-500 등급 상승
  → 두 변화 모두 PushToast로 비차단 알림
```
