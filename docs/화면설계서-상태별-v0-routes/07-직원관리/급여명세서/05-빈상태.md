## 📝 디스크립션

### 사용 시점 (Trigger)

- 케이스 A: 급여 미확정 월

- 케이스 C: 직원 미등록

### 필수 데이터

### 인터랙션 (User Actions)
1. 이번 달로 이동 → 현재 월 명세서 조회
2. 급여 관리로 이동 → SCR-064 (급여 확정 유도)

### 비즈니스 룰
- 급여 명세서는 SCR-064 급여 확정 후 자동 생성
- 미확정 월은 항상 빈 상태
- 과거 미확정 월도 소급 확정 불가 (정책)

### 에지 케이스
- 급여 확정 후 명세서 생성 실패 → 빈 상태 표시 + 재생성 버튼 제공 (선택적)
- 권한 없는 월 조회 → 이 화면 진입 전 차단

### 접근성 (A11y)
- 빈 상태 `role="status"` + `aria-label="급여 명세서 없음"`
- CTA 버튼 명확한 `aria-label`
