## 📝 디스크립션

### 사용 시점 (Trigger)
- `isLoadingData === false && statements.length > 0`
- SCR-064 급여 확정 후 명세서 자동 생성된 상태

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 급여 명세서 목록 | `payroll_statements` (가상) | `month = selectedMonth AND branchId = ?` |
| 직원 정보 | `staff` | 명세서 기반 JOIN |

### 인터랙션 (User Actions)
1. 월 Select 변경 → 해당 월 명세서 재조회
2. 체크박스 선택 → 일괄 액션 활성화
3. [보기] → 명세서 상세 모달
4. [PDF] → 개별 PDF 생성
5. [발송] → 이메일 발송 모달
6. 전체 PDF → 일괄 PDF 생성

### 비즈니스 룰
- SCR-064 급여 확정 후에만 명세서 존재
- 지급 완료: status="paid" (실제 지급 처리 후)
- 대기: status="pending" (확정 후 지급 전)

### 에지 케이스
- 급여 미확정 월 → 명세서 없음 → 05-빈상태
- 직원 이메일 미등록 → 발송 버튼 disabled

### 접근성 (A11y)
- 테이블 `role="grid"` + 선택 체크박스 `aria-label="명세서 선택"`
