## 📝 디스크립션

### 사용 시점 (Trigger)
- 퇴근 기록 완료 후 `workMinutes > 480` 조건 충족
- 일별 뷰에서 자동 감지

### 필수 데이터
| 블록 | 소스 | 설명 |
|------|------|------|
| workMinutes | `staff_attendance` | 근무시간(분) |
| overtimeMinutes | 로컬 계산 | workMinutes - 480 |

### 인터랙션 (User Actions)
1. 퇴근 버튼 → clockOut 기록 → workMinutes 계산 → 자동 연장 판정
2. 연장근무 배지 표시 (자동)

### 비즈니스 룰
- 기준 시간: 480분 (8시간)
- 연장근무 배지: bg-[#EEF4FF] text-[#3B7CF4] (파란색 계열)
- 연장근무 수당 계산은 급여 관리(SCR-064)에서 처리

### 에지 케이스
- 당일 퇴근 미기록 → 연장 판정 불가
- 야간 근무(24시 이후) → date 기준으로 별도 처리 필요

### 접근성 (A11y)
- 연장 배지 `aria-label="연장근무 {N}시간 {M}분"`
