## 📝 디스크립션

### 사용 시점 (Trigger)
- 케이스 A: `staffList.length === 0` (직원 없음)
- 케이스 B: `attendances.length === 0` (해당 날짜 기록 없음)

### 필수 데이터
- staffList (직원 목록 여부 판단)
- attendances (해당 날짜 기록 여부 판단)

### 인터랙션 (User Actions)
1. 케이스 A → 직원 등록하기 버튼
2. 케이스 B → 날짜 변경 또는 출근 버튼으로 기록 시작

### 비즈니스 룰
- 직원은 있지만 기록 없으면 결근 상태로 행 표시 권장
- 통계 카드: 전원 0

### 에지 케이스
- 공휴일/주말 → 근태 기록 없음이 정상
- 신규 센터 (직원 없음) → 직원 등록 유도

### 접근성 (A11y)
- 빈 상태 영역 `role="status"`
