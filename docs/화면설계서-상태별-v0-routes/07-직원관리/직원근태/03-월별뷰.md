    컬럼: 직원명 | 직급 | 출근일수 | 지각횟수 | 결근횟수 | 연차일수 | 총근무시간

컴포넌트 구조:

2. fetchMonthlyAttendances():
   supabase.from("staff_attendance")
     .select("*")
     .eq("branchId", getBranchId())
     .gte("date", `${selectedMonth}-01`)
     .lte("date", `${selectedMonth}-${lastDay}`)
3. 직원별 집계:
   const summary = staffList.map(s => {
     const records = monthlyAttendances.filter(a => a.staffId === s.id);
     return {
       name: s.name,
       role: s.role,
       workDays: records.filter(r => r.clockIn).length,
       lateDays: records.filter(r => r.status === "지각").length,
       absentDays: records.filter(r => r.status === "결근").length,
       annualDays: records.filter(r => r.status === "연차").length,
       totalMinutes: records.reduce((sum, r) => sum + (r.workMinutes ?? 0), 0),
     }
   })
4. 총근무시간: `${Math.floor(totalMinutes/60)}시간 ${totalMinutes%60}분`

데이터:
- monthlyAttendances: StaffAttendanceItem[]
- selectedMonth: string (YYYY-MM)
- summary: 직원별 집계 배열

## 📝 디스크립션

### 사용 시점 (Trigger)
- 월별 요약 탭 클릭 → `setViewMode("monthly")`
- `viewMode === "monthly"` 조건

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 월별 근태 | `staff_attendance` | `date >= monthStart AND date <= monthEnd AND branchId = ?` |
| 직원 목록 | `staff` | `branchId = ?` |

### 인터랙션 (User Actions)
1. 월 input 변경 → 해당 월 데이터 재조회
2. 일별 뷰 탭 클릭 → viewMode="daily"
3. 행 클릭 → 해당 직원 일별 뷰로 이동 (선택적)

### 비즈니스 룰
- 월별 집계: 지점 내 전체 재직 직원 대상
- 결근은 출근 기록 없는 근무일 기준
- 총근무시간 = workMinutes 합계

### 에지 케이스
- 당월 데이터 없는 직원 → 전체 0으로 표시
- 월말(28~31일) 처리: `new Date(year, month, 0).getDate()`

### 접근성 (A11y)
- 월 선택 input `aria-label="조회 월 선택"`
- 집계 테이블 `summary` 역할
