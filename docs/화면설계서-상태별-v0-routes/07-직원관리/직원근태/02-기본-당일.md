## 📝 디스크립션

### 사용 시점 (Trigger)
- `isLoading === false && viewMode === "daily"`
- 기본 날짜: 오늘

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 일별 근태 | `staff_attendance` | `date = selectedDate AND branchId = ?` |
| 직원 목록 | `staff` | `branchId = ? AND isActive = true` |

### 인터랙션 (User Actions)
1. 날짜 input 변경 → fetchAttendances 재호출
2. 출근 버튼 → `clockIn(staffId)` → 현재시각 insert
3. 퇴근 버튼 → `clockOut(staffId)` → update clockOut + workMinutes
4. 월별 요약 탭 → `setViewMode("monthly")`

### 비즈니스 룰
- 출근 판정: clockIn ≤ 09:00 → 정상, > 09:00 → 지각
- 조퇴: clockOut < 18:00
- 결근: clockIn 기록 없음
- workMinutes = (clockOut - clockIn) / 60000

### 에지 케이스
- 출근 기록 없는 직원 → 결근 상태로 행 표시
- 이미 퇴근한 직원 → 완료(비활성) 버튼

### 접근성 (A11y)
- 출근/퇴근 버튼 `aria-label="출근 기록" / "퇴근 기록"`
- StatusBadge `aria-label="근태 상태: {상태명}"`
