## 📝 디스크립션

### 사용 시점 (Trigger)
- 사용자가 폼 필드를 작성하기 시작한 상태
- 역할 Select 선택 완료 또는 연락처 입력 중

### 필수 데이터
| 블록 | 소스 | 설명 |
|------|------|------|
| watchedRole | react-hook-form watch | 선택된 역할 키 |
| ROLE_PERMISSIONS | 상수 | 역할별 권한 설명 + 태그 |
| formatPhone | '@/lib/validations' | 연락처 하이픈 자동 포맷 |

### 인터랙션 (User Actions)
1. 이름 입력 → blur → 필수 검증
2. 역할 Select → 권한 미리보기 표시
3. 연락처 입력 → 실시간 formatPhone()
4. 입사일 선택 → date picker
5. 저장하기 → 전체 검증 트리거

### 비즈니스 룰
- 권한 미리보기: watchedRole이 ROLE_PERMISSIONS에 있을 때만 표시
- 연락처: maxLength=13 (포맷 포함)
- 이메일: 선택 입력, zod email 형식 검증

### 에지 케이스
- 역할 선택 없이 저장 → "역할을 선택해주세요." 에러
- 연락처에 숫자 아닌 문자 입력 → formatPhone이 필터링

### 접근성 (A11y)
- 권한 미리보기 박스 `role="region"` + `aria-label="역할 권한 안내"`
- 권한 태그 `aria-label="{권한명}"`

- 다음: 04-저장중 (유효성 통과) / 06-검증에러 (실패)
