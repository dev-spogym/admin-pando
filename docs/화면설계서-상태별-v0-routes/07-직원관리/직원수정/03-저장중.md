```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-061 직원 수정 — 상태: 저장 중 (supabase update 진행 중)

파일: src/app/staff/edit/page.tsx

컴포넌트 구조:
1. isSaving === true 조건
2. PageHeader actions 수정 저장 버튼 disabled + 스피너:
   <Button variant="primary" disabled>
     <Loader2 size={16} className="animate-spin"/> 저장 중...
   </Button>
3. onSubmit 함수 (수정 모드):
   const onSubmit = async (data: StaffFormData) => {
     if (!editId) return;
     setIsSaving(true);
     try {
       const { error } = await supabase
         .from("staff")
         .update({
           name: data.name,
           phone: data.contact,
           email: data.email || null,
           role: ROLE_DB_MAP[data.role] ?? data.role,
           hireDate: data.joinDate,
           salary: data.salary ? Number(data.salary) : null,
           memo: data.memo || null,
         })
         .eq("id", editId);
       if (error) throw error;
       toast.success("직원 정보가 수정되었습니다.");
       moveToPage(974);
     } catch (err: any) {
       toast.error(`수정 실패: ${err.message}`);
     } finally {
       setIsSaving(false);
     }
   }

데이터:
- isSaving: boolean = true
- supabase.from("staff").update({...}).eq("id", editId)

인터랙션:
- update 성공 → toast.success("직원 정보가 수정되었습니다.") → moveToPage(974)
- update 실패 → toast.error → isSaving=false → 폼 복원

사용 유틸:
- supabase from '@/lib/supabase'
- toast from 'sonner'
- moveToPage from '@/internal'
- lucide-react: Loader2, Save
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- 수정 저장 버튼 클릭 → zod 유효성 통과 → `isSaving = true`
- `supabase.from("staff").update()` 실행 중

### 필수 데이터
| 블록 | 테이블 | 동작 |
|------|--------|------|
| 직원 정보 수정 | `staff` | UPDATE WHERE id = editId |

### 인터랙션 (User Actions)
1. 저장 중 → 버튼 disabled (재클릭 방지)
2. 성공 → toast.success + moveToPage(974)
3. 실패 → toast.error + 폼 복원

### 비즈니스 룰
- 토스트: "직원 정보가 수정되었습니다."
- 실패 토스트: `"수정 실패: {error.message}"`
- finally 블록에서 isSaving=false 보장

### 에지 케이스
- 변경 사항 없이 수정 저장 → update 실행 (멱등)
- 동시 수정 충돌 → supabase 최후 write 승리

### 접근성 (A11y)
- 저장 버튼 `aria-busy="true"`
- 스피너 `aria-label="수정 저장 중"`
