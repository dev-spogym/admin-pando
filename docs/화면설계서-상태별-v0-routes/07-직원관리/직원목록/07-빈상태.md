## 📝 디스크립션

### 사용 시점 (Trigger)
- 케이스 A: 검색/필터 결과 0건 (`filtered.length === 0 && staffData.length > 0`)
- 케이스 B: DB에 직원 데이터 없음 (`staffData.length === 0`)

### 필수 데이터
| 조건 | 표시 |
|------|------|
| staffData.length === 0 | "아직 등록된 직원이 없습니다." + 등록 CTA |
| filtered.length === 0 (검색) | "검색 결과가 없습니다." + 필터 초기화 버튼 |

### 인터랙션 (User Actions)
1. 케이스 A: 필터 초기화 → 전체 직원 목록 복원
2. 케이스 B: 첫 직원 등록하기 → SCR-061 직원 등록
3. PageHeader + 직원 등록 버튼 → SCR-061

### 비즈니스 룰
- 통계 카드는 0명으로 정상 표시
- 액션 버튼 바 전체 disabled (선택 직원 없음)
- 빈 상태 CTA는 센터장 이상만 노출

### 에지 케이스
- 브랜치 ID 오류로 데이터 미조회 → 케이스 B와 동일 UI 표시
- 네트워크 에러 후 retry → 로딩 상태로 돌아감

### 접근성 (A11y)
- 빈 상태 영역 `role="status"` + `aria-label="직원 데이터 없음"`
- CTA 버튼 포커스 우선순위 설정
