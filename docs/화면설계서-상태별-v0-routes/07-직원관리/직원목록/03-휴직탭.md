## 📝 디스크립션

### 사용 시점 (Trigger)
- 재직상태 Select에서 "휴직" 선택 시
- `filterValues.status === "leave"` 조건

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 휴직 직원 목록 | `staff` | `staffStatus = "ON_LEAVE"` AND `branchId = ?` |
| 휴직 기간 | `staff` | `leaveStartAt`, `leaveEndAt` |

### 인터랙션 (User Actions)
1. 재직상태 Select → "전체" → 필터 해제
2. 재직상태 Select → "재직" → 재직 직원만 표시
3. 상태 변경 버튼 → DLG-060-002 (휴직 → 재직 전환)
4. 직원명 클릭 → 직원 수정 화면

### 비즈니스 룰
- 휴직 중 직원은 `staffStatus = "ON_LEAVE"` 상태
- 휴직 기간(leaveStartAt~leaveEndAt) 컬럼 표시
- 휴직 직원도 퇴사 처리 가능 (DLG-060-001)
- 휴직 종료일 지난 직원 → 별도 알림 없음 (수동 상태 변경 필요)

### 에지 케이스
- 휴직 직원 0명 → "휴직 중인 직원이 없습니다." 메시지 표시
- 휴직 기간 미설정 → "-" 표시

### 접근성 (A11y)
- StatusBadge `aria-label="휴직 중"`
- 휴직 기간 컬럼 `aria-label="휴직 시작일 ~ 종료일"`
