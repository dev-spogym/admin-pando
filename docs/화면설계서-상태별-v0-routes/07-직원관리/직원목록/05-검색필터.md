## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-060 직원 목록 — 상태: 검색·필터 조합 적용

파일: src/app/staff/page.tsx

레이아웃:
- <AppLayout> 데스크톱 레이아웃
- <PageHeader> (동일)
- <StatCardGrid cols={4}>: 전체 카운트 유지 (필터 무관)
- <SearchFilter>:
    searchQuery 입력 중 (예: "홍길동" 또는 "010-1234")
    filterValues.role = "트레이너" (또는 선택 역할)
    filterValues.status = "active" (또는 선택 상태)
    [필터 초기화] 버튼 표시
- <DataTable>: filtered 결과 렌더링 (결과 개수 표시)

컴포넌트 구조:
1. SearchFilter onChange → setSearchQuery / setFilterValues / setCurrentPage(1)
2. filtered = staffData.filter(s => {
     const matchSearch = !searchQuery || s.name.includes(searchQuery) || s.contact.includes(searchQuery);
     const matchRole = !filterValues.role || s.role === filterValues.role;
     const matchStatus = !filterValues.status || s.status === filterValues.status;
     return matchSearch && matchRole && matchStatus;
   })
3. 결과 카운트 표시: `총 ${filtered.length}명`
4. [필터 초기화] 버튼: setSearchQuery(""), setFilterValues({role:"",status:""})

데이터:
- searchQuery: string (이름 또는 연락처)
- filterValues: { role: string, status: string }
- filtered: StaffRow[] (클라이언트사이드 필터링)

인터랙션:
- 검색어 입력 → 실시간 필터링 (onChange)
- 역할 Select 변경 → matchRole 조건 업데이트
- 재직상태 Select 변경 → matchStatus 조건 업데이트
- 필터 초기화 → 모든 필터 리셋, 전체 목록 복원
- 결과 0건 → 07-빈상태로 전환

사용 유틸:
- toast from 'sonner'
- formatNumber from '@/lib/format'
- lucide-react: Search, X (초기화 버튼)
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- `searchQuery !== ""` 또는 `filterValues.role !== ""` 또는 `filterValues.status !== ""`
- 검색창 입력 또는 Select 변경 시

### 필수 데이터
| 블록 | 소스 | 설명 |
|------|------|------|
| staffData | 로컬 상태 | 전체 직원 목록 (이미 fetch 완료) |
| filtered | 클라이언트 계산 | AND 교집합 필터링 결과 |

### 인터랙션 (User Actions)
1. 검색창 입력 → 실시간 이름/연락처 필터
2. 역할 Select 변경 → matchRole 갱신
3. 재직상태 Select 변경 → matchStatus 갱신
4. 필터 초기화 버튼 → 전체 상태 리셋
5. 결과 클릭 → 직원 상세/수정

### 비즈니스 룰
- 검색은 이름과 연락처 모두 포함 검색 (OR)
- 역할/상태 필터는 AND 교집합
- 필터 적용 시 currentPage = 1로 리셋
- 검색 결과 건수 표시 (`총 N명`)

### 에지 케이스
- 검색어 특수문자 입력 → 오류 없이 처리 (includes 기반)
- 모든 필터 조합 시 0건 → 07-빈상태

### 접근성 (A11y)
- 검색창 `role="searchbox"` + `aria-label="직원 이름, 연락처 검색"`
- 필터 초기화 버튼 `aria-label="필터 초기화"`
