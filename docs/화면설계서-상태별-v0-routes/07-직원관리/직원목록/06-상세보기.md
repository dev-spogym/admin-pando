## 📝 디스크립션

### 사용 시점 (Trigger)
- `selectedRows.size > 0` (체크박스 1개 이상 선택)
- 개별 행 체크박스 또는 전체 선택 체크박스 클릭

### 필수 데이터
| 블록 | 소스 | 설명 |
|------|------|------|
| selectedRows | 로컬 Set<number> | 선택된 직원 ID 집합 |
| DLG-060-001 | supabase update | isActive=false, staffStatus="RESIGNED" |
| DLG-060-002 | supabase update | staffStatus, leaveStartAt, leaveEndAt, leaveReason |

### 인터랙션 (User Actions)
1. 체크박스 클릭 → selectedRows add/delete
2. 전체 선택 → 현재 페이지 전원 선택
3. 메시지 전송 → `moveToPage(980)` (선택 직원 정보 전달)
4. 상태 변경 → DLG-060-002 모달
5. 퇴사 처리 → DLG-060-001 ConfirmDialog → 확인 시 update
6. 명단 다운로드 → `exportToExcel`

### 비즈니스 룰
- 퇴사 처리 성공: `toast.success("{N}명의 퇴사 처리가 완료되었습니다.")`
- 퇴사 처리 실패: `toast.error("퇴사 처리에 실패했습니다.")`
- 처리 완료 후 `fetchStaff()` 재호출 + `selectedRows.clear()`
- DLG-060-002: 휴직 선택 시 leaveStartAt/leaveEndAt 필수

### 에지 케이스
- 이미 퇴사한 직원 선택 후 퇴사 처리 → 멱등 처리 (isActive 이미 false)
- 전체 선택 후 필터 변경 → selectedRows 초기화

### 접근성 (A11y)
- 선택 카운트 `aria-live="polite"` 영역에 표시
- ConfirmDialog 포커스 트랩 + `role="alertdialog"`
