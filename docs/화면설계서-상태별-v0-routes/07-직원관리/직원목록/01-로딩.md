## 📝 디스크립션

### 사용 시점 (Trigger)
- `/staff` 라우트 최초 진입 시
- `fetchStaff()` 비동기 함수 실행 중 (`isLoadingData === true`)
- 퇴사 처리 후 목록 재조회 중

### 필수 데이터
- `isLoadingData: boolean` — true이면 로딩 상태 유지
- supabase `staff` 테이블 쿼리 진행 중

### 인터랙션 (User Actions)
1. 자동 전환 — 데이터 fetch 완료 시 02-기본-재직으로 이동
2. fetch 실패 시 → `toast.error` 표시 후 빈 테이블 유지

### 비즈니스 룰
- 센터장/최고관리자/슈퍼관리자만 `/staff` 접근 가능
- `getBranchId()`로 현재 지점 필터링
- 로딩 중 액션 버튼(메시지 전송, 상태 변경, 퇴사 처리) 모두 disabled

### 에지 케이스
- 네트워크 지연 2초 이상 시 스켈레톤 유지
- fetch 도중 페이지 이탈 시 메모리 누수 없이 정리 (cleanup)

### 접근성 (A11y)
- `aria-busy="true"` DataTable 영역에 적용
- 스켈레톤 요소 `aria-label="데이터 로딩 중"`
