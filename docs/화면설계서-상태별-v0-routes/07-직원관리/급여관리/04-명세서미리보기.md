## 📝 디스크립션

### 사용 시점 (Trigger)
- DataTable 행의 [상세] 버튼 클릭
- `isDetailModalOpen === true`

### 필수 데이터
| 블록 | 소스 | 설명 |
|------|------|------|
| 급여 명세 | PayrollRow | 지급/공제 항목 상세 |
| 상태 | status | confirmed/pending |

### 인터랙션 (User Actions)
1. 읽기 전용 명세서 확인
2. 편집하기 → DLG-064-001 편집 모드로 전환
3. 닫기 → 모달 닫기

### 비즈니스 룰
- 확정 상태: StatusBadge variant="mint"
- 미확정 상태: StatusBadge variant="warning"
- 지급 합계 = 기본급 + 수당 합산

### 에지 케이스
- 수당/공제 항목 없는 경우 → 0원으로 표시
- 확정 상태에서 편집 시 → 경고 메시지 (재편집 가능)

### 접근성 (A11y)
- Modal `role="dialog"` + `aria-label="급여 명세서 상세"`
- 금액 `aria-label="{항목명} {금액}원"`
