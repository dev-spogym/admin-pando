## 📝 디스크립션

### 사용 시점 (Trigger)
- `payrollData.length === 0` (재직 직원 없거나 급여 미등록)
- 과거 월 조회 시 데이터 없는 경우

### 필수 데이터
- payrollData.length === 0

### 인터랙션 (User Actions)
1. 이번 달로 이동 → selectedMonth 현재 월로 변경
2. 직원 등록하기 → SCR-061

### 비즈니스 룰
- 직원 있어도 기본급 0원이면 빈 상태와 다름 (급여 0원 행 표시)
- 재직 직원 없는 경우에만 진정한 빈 상태

### 에지 케이스
- 과거 특정 월 선택 시 데이터 없음 → 빈 상태 표시
- 권한 없는 접근 → 이 화면 진입 불가 (01-로딩에서 차단)

### 접근성 (A11y)
- 빈 상태 `role="status"` + `aria-label="급여 데이터 없음"`
