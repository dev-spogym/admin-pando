## 📝 디스크립션

### 사용 시점 (Trigger)
- DataTable 행의 [편집] 버튼 클릭
- `isEditModalOpen === true`

### 필수 데이터
| 블록 | 소스 | 설명 |
|------|------|------|
| 직원 급여 기본 | PayrollRow | baseSalary, name, role |
| 수당/공제 항목 | payroll 가상 테이블 | PT수당, 초과수당, 인센티브 등 |

### 인터랙션 (User Actions)
1. 수당/공제 input 변경 → 실시간 실수령액 재계산
2. 저장 → supabase upsert → toast.success → 목록 갱신
3. 취소 → 모달 닫기

### 비즈니스 룰
- 실수령액 = 기본급 + 총수당 - 총공제 (자동 계산)
- 저장 시 payroll 테이블에 upsert
- 편집 후 status="pending"으로 유지

### 에지 케이스
- 음수 실수령액 → 경고 표시 (저장은 허용)
- 기본급 0원 → 수당만으로 급여 구성 가능

### 접근성 (A11y)
- Modal `role="dialog"` + 포커스 트랩
- 실수령액 `aria-live="polite"` (실시간 업데이트)
