## 📝 디스크립션

### 사용 시점 (Trigger)
- `isLoadingData === false && payrollData.length > 0`
- 기본 월: 현재 월 (`MONTHS[0].value`)

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 직원 급여 기본 | `staff` | `branchId = ? AND isActive = true` |
| 급여 상세 | `payroll`(가상) | 수당/공제 항목 |

### 인터랙션 (User Actions)
1. 월 Select 변경 → 해당 월 급여 재조회
2. 검색/상태 필터 → 클라이언트 필터링
3. [상세] → 읽기 전용 모달
4. [편집] → 편집 가능 모달
5. 급여 확정 → DLG-064-002 → 전체 확정 처리
6. 엑셀 다운로드 → 필터된 목록 내보내기

### 비즈니스 룰
- 실수령액 = 기본급 + 수당합계 - 공제합계
- 확정: status="confirmed", 미확정: status="pending"
- 급여 확정 후 명세서 자동 생성 (SCR-065)

### 에지 케이스
- 직원 없는 월 → 06-빈상태
- 이미 전원 확정 → 급여 확정 버튼 비활성

### 접근성 (A11y)
- 실수령액 컬럼 `aria-label="실수령액 {금액}원"`
