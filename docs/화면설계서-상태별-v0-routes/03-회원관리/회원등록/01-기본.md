## 📝 디스크립션

### 사용 시점 (Trigger)
- 회원 목록 [회원 추가] 버튼 클릭
- URL 직접 접근 `/members/new`

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 담당FC 목록 | `staff` | branchId, 활성 직원 |
| 브랜치 정보 | `useAuthStore` | user.branchId |

### 인터랙션 (User Actions)
1. 필드 입력 시작 → `02-입력중`
2. [SMS 인증] 클릭 → `03-SMS인증`
3. [취소] → 이전 페이지

### 비즈니스 룰
- fc는 접근 불가 (hasPermission 체크) → `/forbidden`
- [등록] 버튼 초기 disabled (필수 필드 미입력)
- branchId는 로그인 사용자 기준 자동 설정

### 에지 케이스
- 권한 없는 역할 직접 URL 접근 → `/forbidden`

### 접근성 (A11y)
- 폼 `role="form"` + `aria-label="회원 등록 폼"`
- 필수 필드 `aria-required="true"`
