## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-M001 회원 목록 — 상태: 검색 필터 적용 중

파일: src/app/members/page.tsx (searchQuery 또는 filters 활성 상태)

레이아웃:
- <SearchFilter> 영역이 확장된 상태 (필터 칩 노출)
- 적용된 필터: 파란색 chip 태그 형태로 표시 (x 버튼으로 개별 제거 가능)
- DataTable: 필터 결과 표시

컴포넌트 구조:
1. [검색 입력창] — 회원명/연락처 검색 (debounce 300ms)
   - 입력 시 우측에 X 버튼 노출
   - 검색 아이콘 + placeholder "회원명 또는 연락처 검색"
2. [필터 드롭다운 행]
   - 계약상품: Select (전체/이용권/PT/GX/기타)
   - 성별: Select (전체/남/여)
   - 최종만료일: DateRangePicker
   - 최근방문일: DateRangePicker
3. [추가 필터 행] (toggle 버튼으로 펼침)
   - ☑ 관심회원만 | 미방문 기간 Select | 회원구분 Select | 유입경로 Select | ☐ 만료 숨기기
4. [활성 필터 Chip 행] — 적용된 필터만 chip으로 노출
   - 예: "활성" × | "남성" × | "검색: 김민" ×
   - "필터 초기화" 링크 버튼
5. [DataTable] 필터 결과 (결과 건수 "총 N건" 표시)

데이터:
- searchQuery: string (debounce 300ms)
- filters: { membershipType, gender, expiryRange, visitRange, isFavoriteOnly, noVisitDays, memberType, referralSource, hideExpired }
- useMembers({ branchId, search: searchQuery, ...filters, page })
- URL searchParams 동기화 (useSearchParams + router.replace)

인터랙션:
- 검색어 입력 → 300ms 후 자동 검색
- 필터 변경 → 즉시 재조회
- 필터 chip X 클릭 → 해당 필터 제거
- "필터 초기화" 클릭 → 모든 필터 초기화

사용 유틸:
- useDebounce from '@/hooks/useDebounce'
- useSearchParams, useRouter from 'next/navigation'
- useMembers from '@/api/hooks/useMembers'
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- 검색창에 텍스트 입력 시
- 필터 드롭다운 값 변경 시
- 추가 필터 체크박스 변경 시
- URL에 쿼리 파라미터 포함된 채로 직접 진입

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 검색 결과 | `members` | name ILIKE '%search%' OR phone LIKE '%search%' |
| 필터 결과 | `members` | 복합 AND 조건 |

### 인터랙션 (User Actions)
1. 검색어 입력 → debounce 300ms → 자동 API 호출
2. 드롭다운 선택 → 즉시 필터 적용
3. 필터 chip X → 해당 필터 제거 + 재조회
4. "필터 초기화" → 모든 파라미터 초기화
5. 결과 0건 → `08-빈상태` 인라인 표시

### 비즈니스 룰
- 검색은 name과 phone 컬럼 대상 (대소문자 무시)
- 필터는 AND 조합 (OR 아님)
- URL searchParams에 필터 상태 동기화 (북마크/공유 가능)
- 페이지 전환 시 필터 유지

### 에지 케이스
- 특수문자 포함 검색어 → SQL injection 방어 (Supabase 파라미터 바인딩)
- 필터 조합 결과 0건 → `08-빈상태` 화면으로 전환
- 검색 중 빠른 연속 입력 → debounce로 마지막 입력만 반영

### 접근성 (A11y)
- 검색창 `aria-label="회원 검색"` + `role="searchbox"`
- 필터 chip `aria-label="{필터명} 필터 제거"` + `role="button"`
- 결과 건수 `aria-live="polite"` 로 스크린리더 알림

### 연결 화면
- 이전: `02-회원전체탭`
- 다음: `08-빈상태` (0건) / `02-회원전체탭` (초기화)
