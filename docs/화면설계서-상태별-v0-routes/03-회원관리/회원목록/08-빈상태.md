## 📝 디스크립션

### 사용 시점 (Trigger)
- 케이스 A: 센터에 등록된 회원이 한 명도 없을 때 (신규 센터 등)
- 케이스 B: 검색어/필터 조합 결과 회원이 0건일 때

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 회원 수 확인 | `members` | branchId, 현재 필터 조건 |
| 통계 카드 | `members` | 모두 0 |

### 인터랙션 (User Actions)
1. [회원 추가] 클릭 → `/members/new` (케이스 A, owner/manager/staff 권한)
2. [필터 초기화] 클릭 → 필터 제거 후 전체 목록 재노출 (케이스 B)

### 비즈니스 룰
- fc는 [회원 추가] 버튼 비노출 (권한 없음)
- 케이스 A/B 구분: `hasActiveFilters` 플래그로 판별

### 에지 케이스
- 상태 서브탭 변경으로 해당 상태 회원 없는 경우도 케이스 B로 처리
- 보유상품별/이용권목록 탭에서도 동일한 EmptyState 사용

### 접근성 (A11y)
- EmptyState `role="status"` + `aria-label="검색 결과 없음"`
- 버튼에 명확한 텍스트 레이블 (아이콘 단독 사용 금지)

### 연결 화면
- 이전: `01-로딩` / `05-검색필터`
- 다음: `02-회원전체탭` (회원 추가 또는 필터 초기화 후)
