## 📝 디스크립션

### 사용 시점 (Trigger)
- 메인 탭에서 '보유상품별' 탭 클릭
- URL 파라미터 `?tab=product` 로 직접 진입

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 상품별 카운트 | `members` | GROUP BY membershipType, branchId 필터 |
| 회원 목록 | `members` | membershipType 필터 |

### 인터랙션 (User Actions)
1. 상품 요약 카드 클릭 → 해당 상품으로 테이블 필터링
2. 드롭다운 변경 → 테이블 필터링
3. 회원명 클릭 → 회원 상세 페이지

### 비즈니스 룰
- membershipType이 null이거나 빈 문자열인 회원은 '기타'로 분류
- PT 계약의 잔여횟수는 `contract` 테이블에서 조인하여 가져옴

### 에지 케이스
- 특정 상품 유형의 회원이 0명이면 해당 카드에 0 표시 (숨기지 않음)
- 전체 선택 시 모든 회원 노출

### 접근성 (A11y)
- 상품 카드 `role="button"` + `aria-pressed` 상태 표시
- 드롭다운 `aria-label="상품 유형 필터"`

### 연결 화면
- 이전: `02-회원전체탭`
- 다음: `04-이용권목록탭` / 회원 상세
