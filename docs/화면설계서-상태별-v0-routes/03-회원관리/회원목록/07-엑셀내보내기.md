## 📝 디스크립션

### 사용 시점 (Trigger)
- PageHeader의 "엑셀 다운로드" 버튼 클릭
- fc/staff는 버튼 자체가 비노출 (hasPermission 체크)

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 내보낼 데이터 | `members` | 현재 필터 조건 또는 전체 |
| 담당FC | `staff` | members.staffId = staff.id (컬럼 선택 시) |

### 인터랙션 (User Actions)
1. "엑셀 다운로드" 버튼 → 옵션 모달 오픈
2. 범위/컬럼 선택 → [다운로드] 클릭 → 파일 다운로드
3. [취소] → 모달 닫힘

### 비즈니스 룰
- primary/owner/manager만 엑셀 다운로드 가능
- 전체 내보내기 시 최대 10,000건 제한 (초과 시 경고)
- 파일명: `회원목록_YYYYMMDD_HHmmss.xlsx`
- 개인정보 포함 파일이므로 다운로드 이력 로깅 권장

### 에지 케이스
- 데이터 0건인 상태에서 내보내기 → "내보낼 데이터가 없습니다" 경고
- 10,000건 초과 → "10,000건까지만 내보낼 수 있습니다" 경고 모달
- 네트워크 오류 → 에러 토스트 + isExporting=false 복구

### 접근성 (A11y)
- 모달 `role="dialog"` + `aria-modal="true"` + 포커스 트랩
- 진행 중 버튼 `aria-disabled="true"` + 스피너 `aria-label="다운로드 중"`
- ESC 키로 모달 닫기

### 연결 화면
- 이전: `02-회원전체탭`
- 다음: `02-회원전체탭` (완료/취소 후 복귀)
