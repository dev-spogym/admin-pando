## 📝 디스크립션

### 사용 시점 (Trigger)
- 수정 폼에서 하나 이상의 필드 값 변경 시 (`isDirty === true`)

### 필수 데이터
- react-hook-form `formState.isDirty`, `formState.dirtyFields`
- 실시간 유효성 검사 결과

### 인터랙션 (User Actions)
1. 필드 수정 → isDirty=true, 변경 필드 하이라이트
2. [저장] 클릭 → 폼 검증 실행
3. [취소] 또는 뒤로가기 → 이탈 방지 confirm

### 비즈니스 룰
- 이탈 방지: 브라우저 beforeunload + Next.js 라우터 가드
- 만료일 < 시작일 조합은 실시간 검증 (submit 전)

### 에지 케이스
- 연락처 포맷 오류 → 실시간 인라인 에러
- 이탈 방지 confirm에서 "취소" → 페이지 유지

### 접근성 (A11y)
- 변경 감지 배지 `aria-live="polite"`
- 인라인 에러 `role="alert"` + `aria-describedby` 연결

### 연결 화면
- 이전: `01-기본`
- 다음: `03-저장중` / `05-검증에러` / 이탈 confirm
