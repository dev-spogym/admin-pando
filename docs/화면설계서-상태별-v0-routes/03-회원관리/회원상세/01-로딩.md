## 📝 디스크립션

### 사용 시점 (Trigger)
- `/members/detail?id={memberId}` 진입 직후
- `isLoading === true` 구간

### 필수 데이터
- URL searchParams에서 `id` 추출
- 해당 id로 `members` 테이블 단건 조회 시작

### 인터랙션 (User Actions)
1. 자동 전환 — 데이터 수신 시 `02-기본정보`로 전환
2. memberId 없음 → `/members` 리다이렉트

### 비즈니스 룰
- 다른 센터 회원 id 직접 접근 시 → `/forbidden` 리다이렉트 (branchId 검증)

### 에지 케이스
- 존재하지 않는 memberId → 404 처리 + 목록으로 돌아가기
- 네트워크 오류 → 에러 토스트 + 재시도 버튼

### 접근성 (A11y)
- `aria-busy="true"` + `aria-label="회원 정보 로딩 중"`

### 연결 화면
- 이전: 회원 목록 (`/members`)
- 다음: `02-기본정보`
