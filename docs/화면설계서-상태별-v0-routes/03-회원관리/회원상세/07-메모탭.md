```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-M004 회원 상세 — 상태: 메모 탭

파일: src/app/members/detail/page.tsx (activeTab === 'memo' 조건부)

레이아웃:
- Sticky 프로필 헤더 + TabNav (동일)
- 메모 작성 폼 + 메모 이력

컴포넌트 구조:
1. [메모 작성 폼]
   - Textarea (rows=3, placeholder="회원에 대한 메모를 입력하세요")
   - 글자수 카운터 (N/300자)
   - [저장] 버튼 (primary, 내용 없으면 disabled)
2. [메모 이력 리스트] — 카드 스타일
   각 항목 (bg-white border rounded-lg p-3):
   - 작성자 이름 + 역할 배지 (text-xs)
   - 작성일시 (formatDate 'yyyy.MM.dd HH:mm')
   - 메모 내용
   - 우측: [수정] [삭제] — 본인 작성 또는 owner/manager
3. [고정 메모 섹션] — members.memo 필드 (최상단 고정 표시)
   - "특이사항" 라벨 + 내용 표시
   - [편집] 버튼 → 인라인 편집

데이터:
- interface MemberMemo { id, memberId, content, createdAt, staffId, staff: { name, role } }
- supabase.from('member_memos').select('*, staff(name, role)').eq('memberId', memberId).order('createdAt', { ascending: false })
- members.memo: 회원 고정 특이사항 (members 테이블)

인터랙션:
- [저장] → INSERT member_memos + 리스트 최상단 추가 + 폼 초기화
- [수정] → 해당 항목 인라인 편집 모드
- [삭제] → 확인 없이 즉시 삭제 (단, 토스트로 실행취소 제공, 5초)
- 고정메모 [편집] → members.memo UPDATE

사용 유틸:
- supabase from '@/lib/supabase'
- useAuthStore from '@/stores/authStore'
- formatDate from '@/lib/format'
- toast from 'sonner' (실행취소 포함)
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- '메모' 탭 클릭
- 회원 관련 비정형 메모 기록 시

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 메모 이력 | `member_memos` | memberId, ORDER BY createdAt DESC |
| 고정 메모 | `members` | members.memo 필드 |
| 작성자 | `staff` | staffId = staff.id |

### 인터랙션 (User Actions)
1. 메모 입력 후 [저장] → INSERT
2. [수정] → 인라인 편집
3. [삭제] → 즉시 삭제 + 5초 실행취소 토스트

### 비즈니스 룰
- 모든 역할 메모 작성 가능 (fc 포함)
- 타인 메모 삭제: owner/manager만
- 메모 최대 300자 제한

### 에지 케이스
- 메모 없음 → EmptyState (고정메모 섹션은 항상 표시)
- 실행취소 5초 내 클릭 → DELETE 취소

### 접근성 (A11y)
- Textarea `aria-label="메모 내용 입력"` + `aria-describedby="char-count"`
- 글자수 카운터 `id="char-count"` + `aria-live="polite"`
