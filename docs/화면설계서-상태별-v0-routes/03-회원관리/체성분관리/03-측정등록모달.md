## 📝 디스크립션

### 사용 시점 (Trigger)
- [측정 등록] 버튼 클릭
- 회원 상세 체성분 탭에서 진입 (memberId pre-fill)

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 회원 키 | `members` | height (BMI 계산용) |
| INSERT | `bodyComposition` | 새 측정 기록 |

### 인터랙션 (User Actions)
1. 회원 선택 → 해당 회원 키 자동 조회 (BMI 계산)
2. 체중 입력 → BMI 실시간 계산
3. [등록] → INSERT → 목록 갱신

### 비즈니스 룰
- 필수: 회원, 측정일, 체중
- BMI = 체중(kg) / (키(m))²
- 동일 날짜 중복 측정 가능 (재측정 허용)

### 에지 케이스
- 회원 키 없음 → BMI 계산 불가 표시 ("키 정보 없음")
- 체중 범위 초과 → 검증 에러

### 접근성 (A11y)
- 모달 포커스 트랩 + ESC 닫기
- BMI 계산 결과 `aria-live="polite"`
