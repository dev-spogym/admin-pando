## 📝 디스크립션

### 사용 시점 (Trigger)
- DataTable [상세] 버튼 클릭 → 우측 슬라이드 패널 오픈

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 상세 수치 | `bodyComposition` | 선택된 record |
| 변화 추이 | `bodyComposition` | memberId, 최근 5건, weight+measuredAt |

### 인터랙션 (User Actions)
1. [수정] → 측정등록모달에 기존값 pre-fill
2. [삭제] → 확인 후 DELETE
3. X / 외부 클릭 → 패널 닫기

### 비즈니스 룰
- 삭제: owner/manager만 가능
- 변화 추이: 동일 회원 최근 5회 체중 데이터

### 에지 케이스
- 추이 데이터 1건만 있으면 그래프 대신 "이전 측정 없음" 표시

### 접근성 (A11y)
- 패널 `role="complementary"` + `aria-label="체성분 상세"`
- 닫기 버튼 `aria-label="상세 패널 닫기"`
