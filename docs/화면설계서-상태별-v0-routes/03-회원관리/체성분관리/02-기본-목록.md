## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-BODY 체성분 관리 — 상태: 기본 목록

파일: src/app/body-composition/page.tsx

레이아웃:
- <AppLayout> 데스크톱
- <PageHeader title="체성분 관리" subtitle="회원별 체성분 측정 데이터를 관리합니다."
    actions={[<Button onClick={openMeasureModal} leftIcon={<Plus />}>측정 등록</Button>]} />
- StatCardGrid 3열 + 검색 필터 + DataTable

컴포넌트 구조:
1. [StatCardGrid 3열]
   - 총 측정 건수: bodyCompositions.length
   - 이번달 측정: 이번달 createdAt 건수
   - 측정 대상 회원: 고유 memberId 수

2. [필터 행]
   - 회원 검색: Input (회원명/연락처 debounce)
   - 기간 필터: Select (전체 / 이번달 / 지난달 / 최근 3개월)
   - 담당 FC: Select (staff 목록)

3. [DataTable]
   컬럼: 측정일 | 회원명 | 체중(kg) | 체지방률(%) | 골격근량(kg) | BMI | 내장지방 | 담당FC | 상세
   - 측정일: YYYY-MM-DD
   - 각 수치: 소수점 1자리 표시
   - BMI: 18.5 미만 → 저체중(blue), 25 이상 → 과체중(red), 그 외 정상(green) 배지
   - [상세] 버튼 → `04-상세보기` 슬라이드 패널

4. [Pagination]

데이터:
- interface BodyComposition {
    id, memberId, measuredAt, weight, bodyFatPercent, muscleMass,
    bmi, visceralFat, boneMineral, basalMetabolicRate,
    members: { name, phone }
  }
- supabase.from('bodyComposition').select('*, members(name, phone, staffId), staff(name)')
    .eq('branchId', branchId).order('measuredAt', { ascending: false })

인터랙션:
- [측정 등록] → `03-측정등록모달`
- [상세] → `04-상세보기` 슬라이드 패널
- 필터 변경 → 클라이언트 필터링

사용 유틸:
- supabase from '@/lib/supabase'
- useAuthStore from '@/stores/authStore'
- formatDate from '@/lib/format'
- lucide-react: Plus, ChevronRight
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- `01-로딩` 완료 후 데이터 1건 이상 존재

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 체성분 목록 | `bodyComposition` | branchId, JOIN members |
| 담당 FC | `staff` | members.staffId |

### 인터랙션 (User Actions)
1. [측정 등록] → 모달 오픈
2. [상세] → 슬라이드 패널
3. 필터 변경 → 클라이언트 필터링

### 비즈니스 룰
- BMI 색상 기준: <18.5 저체중, 18.5-24.9 정상, ≥25 과체중
- fc는 본인 담당 회원만 조회 가능 (hasFeature 기준)

### 에지 케이스
- 데이터 0건 → `05-빈상태`
- 특정 수치 null → "-" 표시

### 접근성 (A11y)
- BMI 배지 `aria-label="BMI {값}: {저체중/정상/과체중}"`
- DataTable `role="grid"`

### 연결 화면
- 이전: `01-로딩`
- 다음: `03-측정등록모달` / `04-상세보기`
