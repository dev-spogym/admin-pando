## 📝 디스크립션

### 사용 시점 (Trigger)
- 대상 지점 Select에서 특정 브랜치 선택 시

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 선택된 지점 상세 | `branches` | id = selectedBranchId |
| 해당 지점 회원 수 | `members` | branchId = selectedBranchId COUNT |

### 인터랙션 (User Actions)
1. 다른 지점으로 재선택 → 지점 정보 카드 업데이트
2. 사유 + 이용권처리 선택 완료 → [이관 신청] 활성화

### 비즈니스 룰
- 선택된 지점의 현재 회원 수 표시 (정원 제한 있는 경우 경고)

### 에지 케이스
- 선택된 지점 fetch 실패 → "지점 정보를 불러올 수 없습니다" 인라인 에러

### 접근성 (A11y)
- 지점 정보 카드 `aria-live="polite"` (동적 업데이트)

### 연결 화면

- 다음: `04-이관확인`
