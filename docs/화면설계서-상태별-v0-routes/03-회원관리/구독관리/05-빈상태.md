- 필터 조합 결과 0건일 때

### 필수 데이터
- subscriptions 빈 배열
- hasActiveFilters 플래그

### 인터랙션 (User Actions)
1. [상품 관리로 이동] → `/products` (케이스 A)
2. [필터 초기화] → 전체 구독 목록 복귀 (케이스 B)
