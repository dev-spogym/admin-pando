화면: SCR-ATT 출석 관리 — 상태: 빈 상태

인터랙션:
- [수동 출석 추가] → `04-수동출석모달` (케이스 A)

## 📝 디스크립션

### 사용 시점 (Trigger)
- 케이스 A: 선택한 날짜 출석 기록 0건
- 케이스 B: 검색어 입력 후 결과 0건

### 필수 데이터
- `attendance` 조회 결과 빈 배열
- `hasActiveSearch` 플래그로 케이스 구분

### 인터랙션 (User Actions)
1. [수동 출석 추가] → 수동 출석 모달 (케이스 A)
2. [검색 초기화] → 검색 해제 (케이스 B)

### 비즈니스 룰
- fc도 수동 출석 추가 가능 (오늘 날짜만)
- 과거 날짜 빈 상태: [수동 출석 추가] 버튼 — owner/manager만 표시

### 에지 케이스
- 날짜 변경으로 이동한 날짜가 빈 상태인 경우 케이스 A

### 접근성 (A11y)
- EmptyState `role="status"` + `aria-label="출석 기록 없음"`

### 연결 화면
- 이전: `01-로딩` / `03-날짜변경` / `05-검색결과`
- 다음: `04-수동출석모달` / `02-기본-오늘`
