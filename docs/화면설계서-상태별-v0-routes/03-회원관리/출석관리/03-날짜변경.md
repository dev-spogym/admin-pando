# SCR-ATT 출석 관리 — 상태: 날짜 변경

## 메타

| 항목 | 값 |
|------|----|
| 화면 ID | SCR-ATT |
| 상태 코드 | `attendance-date-changed` |
| 경로 | `/attendance?date=YYYY-MM-DD` |
| 역할 | primary / owner / manager / fc / staff |
| 우선순위 | P1 |
| 이전 상태 | `02-기본-오늘` (날짜 이동 버튼 또는 DatePicker) |
| 다음 상태 | `02-기본-오늘` (오늘로 복귀) / `06-빈상태` (해당일 출석 없음) |

## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-ATT 출석 관리 — 상태: 날짜 변경 (과거 날짜 조회)

파일: src/app/attendance/page.tsx (selectedDate !== today)

레이아웃:
- 02-기본-오늘과 동일한 구조
- 날짜 바에서 선택된 날짜 표시 + "오늘로 돌아가기" 링크

컴포넌트 구조:
1. [날짜 선택 바] — 선택된 날짜로 업데이트
   - "YYYY년 MM월 DD일 (요일)" — 오늘이 아닌 날짜는 gray 텍스트
   - "오늘로 돌아가기" 텍스트 링크 (text-blue-600 underline) — 오늘 아닐 때만 표시
   - [◀] [▶] 날짜 이동 버튼 (미래 날짜: ▶ disabled)

2. [날짜별 출석 통계]
   - "{YYYY년 MM월 DD일} 출석 현황" 소제목
   - 해당일 출석 수 / 전체 대비

3. [출석 DataTable] — 선택된 날짜의 출석 기록
   - 과거 날짜: [취소] 버튼 → owner/manager만 활성 (staff/fc는 disabled)

데이터:
- selectedDate: DatePicker 또는 ◀▶ 버튼으로 변경된 날짜
- supabase.from('attendance').select('*').eq('date', selectedDate).eq('branchId', branchId)
- URL searchParams 동기화: router.replace('/attendance?date='+selectedDate)

인터랙션:
- [◀] → selectedDate - 1일 → 재조회
- [▶] → selectedDate + 1일 (오늘 이후 불가)
- DatePicker → 선택한 날짜로 이동
- "오늘로 돌아가기" → selectedDate = today

사용 유틸:
- format, addDays, subDays, isToday, isFuture from 'date-fns'
- useRouter, useSearchParams from 'next/navigation'
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- [◀][▶] 날짜 이동 버튼 클릭
- DatePicker에서 특정 날짜 선택
- URL `?date=YYYY-MM-DD` 파라미터로 직접 진입

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 선택일 출석 | `attendance` | date = selectedDate, branchId |
| 해당일 통계 | `attendance` | COUNT, branchId, date |

### 인터랙션 (User Actions)
1. [◀][▶] → 하루씩 이동
2. DatePicker → 원하는 날짜 선택
3. "오늘로 돌아가기" → 오늘 날짜로 복귀
4. 과거 출석 [취소] → owner/manager만 가능

### 비즈니스 룰
- 미래 날짜 선택 불가 (today + 1일 이후 DatePicker disable)
- 과거 출석 삭제: owner/manager 권한만
- URL searchParams에 날짜 동기화 (공유 가능)

### 에지 케이스
- 선택한 날짜 출석 0건 → `06-빈상태` 인라인 표시
- 30일 이상 과거 날짜 → 경고 없이 조회 허용 (이력 확인 목적)

### 접근성 (A11y)
- 날짜 헤딩 `aria-live="polite"` (날짜 변경 시 스크린리더 알림)
- 미래 날짜 [▶] `aria-disabled="true"`

### 연결 화면
- 이전: `02-기본-오늘`
- 다음: `02-기본-오늘` (오늘 복귀) / `06-빈상태`
