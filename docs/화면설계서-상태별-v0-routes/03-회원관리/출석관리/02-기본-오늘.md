## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-ATT 출석 관리 — 상태: 기본 (오늘 출석 현황)

파일: src/app/attendance/page.tsx

레이아웃:
- <AppLayout> 데스크톱
- <PageHeader title="출석 관리" subtitle="회원 출석 현황을 관리합니다."
    actions={[<Button onClick={openManualModal} leftIcon={<Plus />}>수동 출석</Button>]} />
- StatCardGrid 3열
- 날짜 선택 바
- 출석 DataTable

컴포넌트 구조:
1. [StatCardGrid 3열]
   - 오늘 출석: todayCount (명)
   - 출석률: (todayCount / activeMembers × 100).toFixed(1) + "%"
   - 미출석 활성 회원: (activeMembers - todayCount) 명

2. [날짜 선택 바]
   - [◀ 이전날] [오늘: YYYY년 MM월 DD일 (요일)] [다음날 ▶]
   - [날짜 직접 선택] DatePicker 아이콘 버튼
   - 오늘이면 "오늘" 배지 표시

3. [검색 입력창]
   - 회원명/연락처 검색 (debounce 300ms)

4. [출석 DataTable]
   컬럼: # | 프로필 | 회원명 | 출석 시각 | 출석 방법 | 이용권 | 만료일 | 액션
   - 출석 시각: HH:mm (오늘 출석 기준)
   - 출석 방법: QR / RFID / 수동 / 키오스크 배지
   - [취소] 버튼 → 해당 출석 삭제

5. [페이지네이션]

데이터:
- selectedDate: today (yyyy-MM-dd)
- supabase.from('attendance')
    .select('*, members(name, phone, membershipType, membershipExpiry, profileImage)')
    .eq('branchId', branchId)
    .eq('date', selectedDate)
    .order('createdAt', { ascending: false })
- todayCount: attendanceData.length
- activeMembers: memberStats.active

인터랙션:
- [◀][▶] → 날짜 변경 → `03-날짜변경`
- 검색 입력 → `05-검색결과`
- [수동 출석] → `04-수동출석모달`
- [취소] → 해당 출석 DELETE + 낙관적 업데이트

사용 유틸:
- supabase from '@/lib/supabase'
- useAuthStore from '@/stores/authStore'
- format, addDays, subDays, isToday from 'date-fns'
- toast from 'sonner'
- lucide-react: Plus, ChevronLeft, ChevronRight, Calendar
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- `01-로딩` 완료 후 오늘 날짜 출석 데이터 수신 (1건 이상)
- 날짜 선택 후 해당일 출석 있을 때

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 오늘 출석 | `attendance` | branchId, date=today, JOIN members |
| 활성 회원 수 | `members` | branchId, status=ACTIVE COUNT |

### 인터랙션 (User Actions)
1. [◀][▶] → 날짜 하루 이동
2. DatePicker → 특정 날짜 선택
3. 검색 → 해당일 출석 회원 검색
4. [수동 출석] → 출석 추가 모달
5. [취소] → 해당 출석 삭제

### 비즈니스 룰
- 미래 날짜 선택 불가 (today 이후 DatePicker 비활성)
- 출석 취소: 오늘 날짜 출석만 가능 (과거 날짜는 owner/manager만)
- 중복 출석 방지 (동일 회원 동일 날짜 1건만)

### 에지 케이스
- 오늘 출석 0건 → `06-빈상태`
- 출석 취소 실패 → 롤백 + 에러 토스트

### 접근성 (A11y)
- 날짜 이동 버튼 `aria-label="이전날" / "다음날"`
- 출석 방법 배지 `aria-label="출석 방법: QR코드"`

### 연결 화면
- 이전: `01-로딩`
- 다음: `03-날짜변경` / `04-수동출석모달` / `05-검색결과`
