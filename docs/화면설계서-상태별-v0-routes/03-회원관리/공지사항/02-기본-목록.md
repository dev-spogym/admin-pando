```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-NOTICE 공지사항 — 상태: 기본 목록

파일: src/app/notices/page.tsx

레이아웃:
- <AppLayout> 데스크톱
- <PageHeader title="공지사항" subtitle="회원 앱에 표시될 공지사항을 관리합니다."
    actions={[
      hasPermission(user, 'notices.create') &&
      <Button onClick={openCreateModal} leftIcon={<Plus />}>공지 작성</Button>
    ]} />
- 검색 필터 + 공지 카드 목록 또는 DataTable

컴포넌트 구조:
1. [필터 행]
   - 검색: Input (제목 키워드 검색)
   - 게시 상태: Select (전체 / 게시 중 / 미게시)
   - 고정 여부: Checkbox (고정 공지만)

2. [공지 목록] — 카드 리스트 (divide-y border)
   각 카드 (flex, p-4):
   - 좌측: 📌 고정 아이콘 (isPinned=true 시) + 상태 배지 (게시/미게시)
   - 중앙:
     · 제목 (font-medium, 클릭 시 확장)
     · 작성자 + 작성일 (text-sm text-gray-500)
     · 미리보기 (content 앞 100자 + "..." truncate)
   - 우측: [수정] [삭제] 버튼 (owner/manager만)
   - 고정 공지: 목록 최상단 고정, bg-yellow-50 배경

3. [페이지네이션]

4. [아코디언 확장] — 카드 클릭 시 전체 내용 표시
   - content 전체 (허용: 기본 텍스트, 이미지 URL)

데이터:
- interface Notice { id, branchId, title, content, isPublished, isPinned, createdAt, updatedAt, staffId, staff: { name } }
- supabase.from('notices').select('*, staff(name)').eq('branchId', branchId)
    .order('isPinned', { ascending: false }).order('createdAt', { ascending: false })

인터랙션:
- 카드 클릭 → 아코디언 확장 (전체 내용 표시)
- [수정] → `04-수정모달`
- [삭제] → `05-삭제확인`
- [공지 작성] → `03-작성모달`

사용 유틸:
- supabase from '@/lib/supabase'
- hasPermission from '@/lib/permissions'
- formatDate from '@/lib/format'
- lucide-react: Plus, Pin, Edit, Trash2
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- `01-로딩` 완료 후 공지 1건 이상 존재

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 공지 목록 | `notices` | branchId, ORDER BY isPinned DESC, createdAt DESC |
| 작성자 | `staff` | staffId = staff.id |

### 인터랙션 (User Actions)
1. 카드 클릭 → 아코디언으로 전체 내용 표시
2. [공지 작성] → 작성 모달
3. [수정] → 수정 모달
4. [삭제] → 삭제 확인 모달

### 비즈니스 룰
- fc/staff는 공지 작성/수정/삭제 불가 (조회만)
- 고정 공지: 최상단 고정 + bg-yellow-50 강조
- 미게시 공지: 회원 앱에 미표시 (CRM에서만 보임)

### 에지 케이스
- 공지 0건 → `06-빈상태`
- 내용 이미지 URL 포함 시 img 태그로 렌더

### 접근성 (A11y)
- 아코디언 `aria-expanded` + `aria-controls`
- 고정 공지 `aria-label="고정 공지"` 표시

### 연결 화면
- 이전: `01-로딩`
- 다음: `03-작성모달` / `04-수정모달` / `05-삭제확인`
