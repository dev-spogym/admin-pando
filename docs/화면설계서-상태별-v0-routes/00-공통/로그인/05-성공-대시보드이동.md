## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-LOGIN 로그인 — 상태: 성공 처리 플로우

파일: src/app/(auth)/login/page.tsx

성공 처리 순서 (onSuccess 내부):
1. res.success === true 확인
2. clearFailCount() — localStorage 실패 횟수 초기화
3. rememberMe ? localStorage.setItem(REMEMBER_KEY, id) : localStorage.removeItem(REMEMBER_KEY)
4. branchId 결정:
   - isSuperAdmin=true → selectedBranchId || ''
   - 일반 사용자 → selectedBranchId || String(user.branchId ?? 1)
5. localStorage.setItem('branchId', branchIdToUse)
6. authStore.login({
     id: String(user.id),
     name: user.name,
     email: '',
     role: user.role,
     branchId: branchIdToUse,
     branchName: isSuperAdmin ? (selectedBranch?.name ?? '전체 지점') : (selectedBranch?.name ?? ''),
     tenantId: String(user.tenantId ?? 1),
     isSuperAdmin,
     currentBranchId: user.currentBranchId ? String(user.currentBranchId) : null,
   }, accessToken)
7. moveToPage(966) — 대시보드 이동

UI 포인트:
- 성공 직후 화면은 03-제출중과 동일 (버튼 스피너 상태)
- moveToPage(966) 호출 → 즉시 화면 전환 (시각적 성공 메시지 없음)
- 필요시 toast.success('로그인되었습니다.') 추가 가능

사용 유틸:
- useAuthStore().login from '@/stores/authStore'
- moveToPage(966) from '@/internal'
- localStorage (REMEMBER_KEY, branchId)
- clearFailCount() — 로컬 함수
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- loginMutation onSuccess에서 `res.success === true` 수신
- `res.data.user` 및 `res.data.accessToken` 존재

### 필수 데이터
| 항목 | 소스 | 설명 |
|------|------|------|
| user | res.data.user | id, name, role, branchId, isSuperAdmin, tenantId |
| accessToken | res.data.accessToken | JWT 토큰 |
| selectedBranchId | state | 로그인 시 선택한 지점 |
| branches | state | 지점명 조회용 |

### 인터랙션 (User Actions)
- 이 상태는 자동 전환 — 사용자 액션 없음
- moveToPage(966) 즉시 실행

### 비즈니스 룰
- 슈퍼관리자(isSuperAdmin=true): branchName = '전체 지점' (선택 지점 없을 경우)
- 일반 사용자: branchName = selectedBranch?.name (Select에서 선택한 지점명)
- authStore.login() 이후 모든 보호된 라우트에서 user 접근 가능
- 감사 로그 자동 기록: LOGIN 액션 (서버 측에서 처리)

### 에지 케이스
- moveToPage(966) 실패 → fallback: router.push('/') 권장
- accessToken 없음 → 로그인 실패로 처리
- isSuperAdmin=true & selectedBranchId='' → branchId를 '' 또는 '0'으로 저장 (전체 지점 모드)

### 접근성 (A11y)
- 성공 후 즉시 페이지 이동 — 별도 A11y 처리 불필요
- 포커스는 새 페이지 (대시보드)의 첫 포커스 가능 요소로 이동

### 연결 화면

- 다음: SCR-090 대시보드 (`/`) — moveToPage(966)
