## 📝 디스크립션

### 사용 시점 (Trigger)
- `reserved` 상태 타석의 [예약 확인] 버튼 클릭

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 예약 정보 | `golf_bays` | `id = ?` |

### 인터랙션 (User Actions)
1. [사용 시작] → `in_use` 전환 + 60분 타이머 시작
2. [예약 취소] → 확인 다이얼로그 → `available` 전환
3. [닫기] → 모달만 닫힘

### 비즈니스 룰
- 예약 시간 초과 시 "예약 시간이 지났습니다." 경고 배지
