## 📝 디스크립션

### 사용 시점 (Trigger)
- [▶ 시작] 또는 [사용 시작] 클릭 후 `in_use` 전환
- 카운트다운 타이머 활성화 상태

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 사용중 타석 | `golf_bays` | `status = 'in_use'`, `branch_id = ?` |
| 이동 가능 타석 | `golf_bays` | `status = 'available'`, `branch_id = ?` |

### 인터랙션 (User Actions)
1. [📺 on/off] → 프로젝터 토글 + toast
2. [↔ 이동] → 이동 모달 → 대상 선택 → 데이터 교환
3. [⬜ 종료] → 타석 초기화 → 대기열 자동 처리

### 비즈니스 룰
- 타이머 만료(0분) 시 자동 toast.warning + 대기열 체크
- 이동 시 남은 시간/시작 시각 그대로 유지
- 이동 대상 타석 없으면 "이동 가능한 타석이 없습니다." 안내

### 에지 케이스
- 이동 대상 available 타석 0 → 이동 버튼 disabled
- 종료 후 대기열 자동 배정 실패 → toast.error + available 유지

### 접근성 (A11y)
- 카운트다운 `aria-live="polite"` (변경 시 스크린리더 읽기)
- 이동 모달 포커스 트랩
