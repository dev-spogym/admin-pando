## 📝 디스크립션

### 사용 시점 (Trigger)
- RoomCard 슬롯바 클릭
- 목록 뷰에서 룸 행의 [스케줄] 버튼 클릭

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 수업 스케줄 | `class_schedule` | `branch_id = ?`, `date = selectedDate` |
| 룸 정보 | `rooms` JOIN | - |
| 강사 정보 | `instructors` JOIN | - |

### 인터랙션 (User Actions)
1. 날짜 이전/다음 → 스케줄 재조회
2. [오늘] → 오늘 날짜
3. 슬롯 클릭 → 수업 상세 툴팁
4. 모달 닫기 → `02-기본-룸목록`

### 비즈니스 룰
- 타임라인: 오전 6시~오후 10시 (960분 기준)
- 현재 시각 선: 오늘 날짜 선택 시만 표시
- 예약률 80% 이상: 빨간색 / 50%~: 주황색 / ~50%: 초록색

### 에지 케이스
- 선택일 스케줄 없음 → "해당 날짜 수업 없음" 안내
- 룸 0건 → 타임라인 행 없음

### 접근성 (A11y)
- 타임라인 슬롯 `role="button"`, `aria-label="{수업명} {시간}"`
- 현재 시각 선 `aria-hidden="true"`

- 다음: `02-기본-룸목록` (모달 닫기)
