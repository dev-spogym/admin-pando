```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-053 운동룸 관리 — 상태: 빈상태 (룸 미등록)

파일: src/app/(facilities)/rooms/page.tsx

레이아웃:
- <AppLayout> 데스크톱 레이아웃
- <PageHeader title="운동룸 관리" subtitle="센터 내 운동룸 현황과 예약 슬롯을 관리합니다."
    actions={[<Button variant="primary" onClick={openAddModal}><Plus /> 새 운동룸 등록</Button>]} />
- StatCardGrid (4컬럼) — 모두 0
- 빈 상태 영역 (min-h-[400px] flex flex-col items-center justify-center):
    DoorOpen 아이콘 (w-16 h-16 text-gray-300)
    "등록된 운동룸이 없습니다." (text-lg font-medium text-gray-500 mt-4)
    "운동룸을 등록하여 시설 관리를 시작하세요." (text-sm text-gray-400 mt-1)
    <Button variant="primary" onClick={openAddModal} className="mt-4">
      <Plus size={16} className="mr-1" /> 운동룸 등록하기
    </Button>

룸 등록 모달 (DLG-053-001):
  <Modal title="새 운동룸 등록" isOpen={addModalOpen} onClose={closeAddModal}>
    <form onSubmit={handleAddRoom} className="space-y-4">
      {/* 룸명 */}
      <div>
        <label className="text-sm font-medium">룸명 *</label>
        <Input value={name} onChange={e => setName(e.target.value)} placeholder="예: GX룸 A" required />
      </div>
      {/* 유형 선택 */}
      <div>
        <label className="text-sm font-medium">유형 *</label>
        <div className="flex gap-2 mt-1 flex-wrap">
          {['GX', 'PT', '스피닝', '필라테스', '기타'].map(t => (
            <button
              key={t}
              type="button"
              onClick={() => setType(t)}
              className={`px-3 py-1.5 rounded-full text-sm border transition-colors ${
                type === t ? 'bg-primary text-white border-primary' : 'border-gray-300 text-gray-600 hover:border-primary'
              }`}
            >
              {t}
            </button>
          ))}
        </div>
      </div>
      {/* 수용인원 */}
      <div>
        <label className="text-sm font-medium">수용인원</label>
        <div className="flex gap-2 mt-1">
          <Input type="number" min={1} value={capacity} onChange={e => setCapacity(Number(e.target.value))} className="w-24" />
          <div className="flex gap-1 flex-wrap">
            {[1,2,6,10,14,16,20,22].map(n => (
              <button key={n} type="button" onClick={() => setCapacity(n)}
                className="px-2 py-1 text-xs border rounded hover:bg-gray-50">{n}명</button>
            ))}
          </div>
        </div>
      </div>
      {/* 게이트 */}
      <div>
        <label className="text-sm font-medium">게이트</label>
        <Input value={gate} onChange={e => setGate(e.target.value)} placeholder="예: 1층 A게이트" />
      </div>
      {/* 설명 */}
      <div>
        <label className="text-sm font-medium">설명</label>
        <Textarea value={description} onChange={e => setDescription(e.target.value)} rows={2} placeholder="룸 설명 (선택)" />
      </div>
    </form>
    <div slot="footer">
      <Button variant="ghost" onClick={closeAddModal}>취소</Button>
      <Button variant="primary" onClick={handleAddRoom} disabled={!name || !type}>등록</Button>
    </div>
  </Modal>

handleAddRoom():
  supabase.from('rooms').insert({
    name, type, capacity: capacity || null, gate: gate || null,
    description: description || null, status: 'active', branchId
  })
  toast.success(`${name}이 등록되었습니다.`)
  closeAddModal(); fetchRooms()

데이터:
  - rooms.length === 0
  - name, type, capacity, gate, description: form state

인터랙션:
  - [운동룸 등록하기] → 등록 모달 오픈
  - 유형 버튼 클릭 → type 선택
  - 수용인원 퀵선택 → capacity 설정
  - [등록] → handleAddRoom() → toast → 02-기본-룸목록 전환
  - [취소] → 모달 닫기

사용 유틸:
  - useAuthStore, supabase, toast
  - lucide-react: DoorOpen, Plus
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- `rooms` 테이블 데이터 0건
- 로딩 완료 후 `rooms.length === 0`

### 필수 데이터
- 없음

### 인터랙션 (User Actions)
1. [운동룸 등록하기] → 등록 모달 오픈
2. 폼 입력 → [등록] → insert → `02-기본-룸목록`

### 비즈니스 룰
- 센터장/매니저만 등록 가능 (스태프는 버튼 hidden)
- 룸명 + 유형 필수, 수용인원/게이트/설명 선택
- 등록 즉시 `status: 'active'` 기본값

### 에지 케이스
- 유형 필터 결과 0건과 전체 0건 구분: 필터 결과 0건은 별도 인라인 메시지 사용

### 접근성 (A11y)
- `role="status"` on empty state container
- 유형 버튼 그룹 `role="group"`, `aria-label="운동룸 유형 선택"`
- Modal 포커스 트랩

### 연결 화면
- 이전: `01-로딩`
- 다음: `02-기본-룸목록` (등록 완료)
