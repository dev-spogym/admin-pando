## 📝 디스크립션

### 사용 시점 (Trigger)
- 락커 셀 호버 → 오버레이 [기록] 버튼 클릭
- DLG-050-001 기록 모달 내 비밀번호/메모 편집 영역

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 이력 | `locker_history` | `locker_id = ?` ORDER BY created_at DESC LIMIT 20 |
| 현재 비밀번호/메모 | `lockers` | `id = ?` |

### 인터랙션 (User Actions)
1. 비밀번호 수정 → Input 편집
2. 메모 수정 → Textarea 편집
3. [저장] → DB 업데이트 + toast
4. [닫기] → 모달만 닫힘

### 비즈니스 룰
- 비밀번호는 숫자 4~10자리 권장 (소프트 제한, 경고만)
- 저장 시 이력 테이블에 'password_changed' 이벤트 기록 (선택)

### 에지 케이스
- 이력 0건 → DataTable 빈 상태 메시지
- 비밀번호 미입력 저장 → null 허용 (비밀번호 없음 처리)

### 접근성 (A11y)
- Modal 포커스 트랩
- 비밀번호 Input `type="text"` (락커 번호 확인용, 보안 불필요)
