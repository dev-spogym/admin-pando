## 📝 디스크립션

### 사용 시점 (Trigger)
- overtime 상태 셀(만료일 초과) 호버 → [해제] 클릭
- [만료 락커 일괄 해제] 버튼 클릭 (overtimeLockers.length > 0 시 표시)

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 만료 락커 | `lockers` | `status = 'IN_USE'`, `expiresAt < now()` |

### 인터랙션 (User Actions)
1. 개별 [해제] → 확인 다이얼로그 → 해제 처리
2. [일괄 해제] → 확인 다이얼로그 → 일괄 처리
3. [취소] → 다이얼로그만 닫힘

### 비즈니스 룰
- 해제 시 memberId/memberName/assignedAt/expiresAt/password/memo 모두 null 처리
- 해제 이력 `locker_history` 기록 권장
- 일괄 해제는 현재 탭(일일/개인/골프) 내 만료 락커만 대상

### 에지 케이스
- 일괄 해제 대상 0건 → 버튼 hidden 또는 disabled
- 해제 중 네트워크 오류 → toast.error + 재시도 안내

### 접근성 (A11y)
- 확인 다이얼로그 `role="alertdialog"`
- [일괄 해제] 버튼 `aria-label="만료된 {N}개 사물함 일괄 해제"`
