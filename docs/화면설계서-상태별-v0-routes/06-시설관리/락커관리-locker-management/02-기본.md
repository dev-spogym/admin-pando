## 📝 디스크립션

### 사용 시점 (Trigger)
- `01-로딩` 완료, 데이터 정상 로드
- 기본 진입 상태 (일일 사물함 탭)

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 사물함 목록 | `lockers` | `branch_id = ?`, `type = 'daily'` ORDER BY number |
| 회원 검색 | `members` | `branch_id = ?` |

### 인터랙션 (User Actions)
1. 회원 검색 → 추천 락커 자동 갱신
2. 그리드 셀 클릭 → 락커 선택
3. [배정하기] → 배정 처리
4. [만료 락커 일괄 해제] → 일괄 처리
5. 탭 전환 → 사물함 유형 변경

### 비즈니스 룰
- overtime: `IN_USE` + 만료일 초과
- abnormal: 상태 이상 (시스템 감지)
- 배정 시 회원 + 락커 모두 선택 필수

### 에지 케이스
- available 락커 0건 → [배정하기] disabled + "사용 가능한 사물함이 없습니다."
- 회원 미선택 → [배정하기] disabled

### 접근성 (A11y)
- 그리드 셀 `role="button"`, `aria-selected`, `aria-label`
- 배정 폼 `role="form"`, `aria-label="사물함 배정"`
