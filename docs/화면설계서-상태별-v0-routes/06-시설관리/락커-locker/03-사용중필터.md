파일: src/app/(facilities)/locker/page.tsx

상태 필터 UI:
  <div className="flex items-center gap-2 mb-3">
    <Select
      value={statusFilter}
      onChange={setStatusFilter}
      options={[
        { value: 'all', label: '전체' },
        { value: 'IN_USE', label: '사용중' },
        { value: 'AVAILABLE', label: '공실' },
        { value: 'MAINTENANCE', label: '점검' },
        { value: 'expiring', label: '만료임박' },
      ]}
    />
    {statusFilter !== 'all' && (
      <Button size="sm" variant="ghost" onClick={() => setStatusFilter('all')}>

      </Button>
    )}
    <span className="text-sm text-gray-500">{filteredLockers.length}개 락커</span>
  </div>

사용중 필터 적용 시:
  filteredLockers = lockers.filter(l => {
    const statusMatch = statusFilter === 'expiring'
      ? (l.status === 'IN_USE' && dDay(l) !== null && dDay(l)! <= 7)
      : statusFilter === 'all' ? true : l.status === statusFilter;
    const searchMatch = !searchQuery || l.number.toString().includes(searchQuery) || l.memberName?.includes(searchQuery);
    return statusMatch && searchMatch;
  });

사용중 락커 카드 스타일:

  - 회원명 + D-Day 표시
  - 만료임박(D≤7): bg-amber-50 border-amber-400

데이터:
  - statusFilter = 'IN_USE'
  - filteredLockers: Locker[] (IN_USE만)

인터랙션:
  - 필터 Select '사용중' 선택 → 사용중 락커만 표시

## 📝 디스크립션

### 사용 시점 (Trigger)
- StatCard "사용중" 클릭 → `statusFilter = 'IN_USE'` 자동 설정
- 상태 필터 Select에서 '사용중' 선택

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 락커 목록 | `lockers` | `status = 'IN_USE'`, `branch_id = ?` |

### 인터랙션 (User Actions)
1. 필터 Select → 다른 상태 선택 시 해당 필터로 전환
2. [필터 해제] → `02-기본-그리드`로 복귀
3. 셀 호버 → 회수/고장 액션

### 비즈니스 룰
- 만료임박(D≤7)은 IN_USE의 서브셋으로 별도 처리
- 필터 결과 카운트 표시: "{N}개 락커"

### 에지 케이스
- 사용중 락커 0건 → "사용중인 락커가 없습니다." 인라인 메시지

### 접근성 (A11y)
- 필터 Select `aria-label="락커 상태 필터"`
- 결과 수 `aria-live="polite"`
