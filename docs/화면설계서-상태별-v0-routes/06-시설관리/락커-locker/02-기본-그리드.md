## 📝 디스크립션

### 사용 시점 (Trigger)
- `01-로딩` 완료 후 `lockers.length > 0`
- 기본 진입 상태 (A구역 탭, 검색 없음)

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 락커 목록 | `lockers` | `branch_id = ?`, `zone = 'A'` ORDER BY number |
| 통계 집계 | `lockers` | SUM per status |

### 인터랙션 (User Actions)
1. 구역 탭 전환 → 해당 구역 락커 표시
2. 검색어 입력 → 번호/회원명 필터
3. 셀 호버 → 액션 오버레이 표시
4. [기록] → 이력+비밀번호 모달
5. [이동] → 번호 변경 모달
6. [회수] → 회수 확인 다이얼로그
7. [배정] → 회원 배정 모달 (→ `06-배정모달`)
8. [고장/복구] → 상태 토글 확인
9. [일괄 배정] → bulkMode 진입
10. [만료임박 일괄 해제] → 일괄 해제 확인

### 비즈니스 룰
- 만료임박: `IN_USE` + `expiresAt` 7일 이내 자동 판정
- D-Day: `Math.floor((expiresAt - today) / 86400000)`
- D≤0: `text-state-error`, D 1~7: `text-amber-600`, D>7: `opacity-60`
- `MAINTENANCE` → UI: `broken` (사선 표시)
- 스태프는 삭제/락커추가 불가

### 에지 케이스
- 검색 결과 0건 → "검색 결과가 없습니다." 인라인 메시지
- 구역 내 락커 0건 → `06-빈상태`에 준하는 안내

### 접근성 (A11y)
- 셀 `role="button"`, `tabIndex={0}`, `aria-label="{zone}{number}번 락커 {상태} {memberName}"`
- `onKeyDown`: Enter/Space → 클릭 동작
