## 📝 디스크립션

### 사용 시점 (Trigger)
- 상태 필터 Select에서 '점검' 선택
- StatCard "고장" 클릭 시 자동 적용

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 고장 락커 | `lockers` | `status = 'MAINTENANCE'`, `branch_id = ?` |

### 인터랙션 (User Actions)
1. 셀 호버 → [복구] 버튼 클릭 → 확인 → `AVAILABLE` 전환
2. 다중 선택 → [일괄 복구] → 일괄 처리
3. [필터 해제] → `02-기본-그리드`

### 비즈니스 룰
- 고장 락커는 배정 불가 (오버레이에 [배정] 버튼 미노출)
- 복구는 매니저/센터장만 가능

### 에지 케이스
- 고장 락커 0건 → "고장 락커가 없습니다." 안내

### 접근성 (A11y)
- 고장 셀 `aria-label="{N}번 락커 고장 처리 중"`
- 사선 표시 `aria-hidden="true"` (장식용)
