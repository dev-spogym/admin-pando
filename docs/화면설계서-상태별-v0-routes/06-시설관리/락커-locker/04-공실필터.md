## 📝 디스크립션

### 사용 시점 (Trigger)
- 상태 필터 Select에서 '공실' 선택
- [일괄 배정] 버튼 클릭 시 자동으로 공실 필터 적용

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 공실 락커 | `lockers` | `status = 'AVAILABLE'`, `branch_id = ?`, `zone = ?` |

### 인터랙션 (User Actions)
1. 셀 호버 → [배정] 버튼 클릭 → `06-배정모달`
2. bulkMode → 다중 선택 → [N개 락커 배정]
3. [필터 해제] → `02-기본-그리드`

### 비즈니스 룰
- 공실 락커만 배정 가능
- bulkMode 선택된 락커 수에 따라 배정 버튼 레이블 동적 변경

### 에지 케이스
- 공실 락커 0건 → "모든 락커가 사용 중입니다." 안내

### 접근성 (A11y)
- 선택된 셀 `aria-selected="true"`
- bulkMode 배너 `role="status"`

### 연결 화면
- 이전: `02-기본-그리드`
- 다음: `06-배정모달` / `02-기본-그리드`
