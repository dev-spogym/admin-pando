## 📝 디스크립션

### 사용 시점 (Trigger)
- 대여 이력 탭 진입 후 상태 필터를 '세탁중'으로 설정
- 반납 처리 시 '세탁 필요' 선택 후 자동 이동

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 세탁 목록 | `clothing_rentals` | `status = 'washing'`, `branch_id = ?` |
| 회원명 | `members` JOIN | - |
| 품목/사이즈 | `clothing` JOIN | - |

### 인터랙션 (User Actions)
1. [세탁 완료] → 단건 세탁 완료 처리
2. 체크박스 선택 → [선택 항목 세탁 완료] 일괄 처리
3. 상태/품목 필터 변경 → 목록 갱신

### 비즈니스 룰
- 세탁 경과일 자동 계산 (반납일 기준)
- 7일 이상 세탁 중인 항목 → 빨간색 강조 + 알림 뱃지
- 세탁 완료 시 `washingCount` 감소, `availableCount` 자동 증가

### 에지 케이스
- 세탁 중인 항목 0건 → "세탁 중인 운동복이 없습니다." 빈 상태 메시지

### 접근성 (A11y)
- 경과일 셀 `aria-label="세탁 경과 {N}일"`
- [세탁 완료] 버튼 `aria-label="{품목} {사이즈} 세탁 완료 처리"`

- 다음: `02-기본-재고목록` (세탁 완료 후 재고 반영)
