## 📝 디스크립션

### 사용 시점 (Trigger)
- 대기 탭 또는 처리완료 탭에서 행 클릭

### 필수 데이터
- `selectedRefund: RefundItem` — 클릭한 환불 건

### 인터랙션 (User Actions)
1. [승인] (대기 상태) → handleApprove + 모달 닫기
2. [거절] (대기 상태) → `04-거절처리` 모달
3. [회원 상세] → 회원 상세 페이지
4. [닫기] / ESC → 모달 닫기

### 비즈니스 룰
- 처리완료 건: 승인/거절 버튼 없음 (읽기 전용)
- 거절 건: 거절사유 표시 (rejectReason 필드)

### 에지 케이스
- selectedRefund null → 모달 미렌더링

### 접근성 (A11y)
- `role="dialog"` + `aria-labelledby="modal-title"` + 포커스 트랩

### 연결 화면
- 이전: `02-대기탭` / `03-승인탭`
- 다음: 동일 탭 (닫기) / 회원 상세
