```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-S008 미수금 — 상태: 기본 (미수금 목록)

파일: src/app/unpaid/page.tsx

레이아웃:
- <AppLayout> 데스크톱 레이아웃
- <PageHeader title="미수금 관리" subtitle="미납된 금액을 조회하고 수금 처리합니다."
    actions={[<Button variant="outline" onClick={handleExcel}>엑셀</Button>]} />
- <StatCardGrid cols={3}>
    <StatCard label="총 미수금" value={formatKRW(totalUnpaid)} variant="error" icon={AlertCircle} />
    <StatCard label="미수금 건수" value={`${unpaidItems.length}건`} icon={Users} />
    <StatCard label="최장 연체" value={`${maxOverdueDays}일`} icon={TrendingDown} />
  </StatCardGrid>
- 뷰 전환 버튼: [건별 목록(active)] [회원별 그룹]
- <SearchFilter> 검색어 + 날짜범위 + 연체기간(Select)
- <DataTable columns={columns} data={filteredUnpaid} onRowClick={handleRowClick} />

DataTable 컬럼:
| No | 결제일 | 회원명 | 상품명 | 결제금액 | 미수금액 | 연체일수 | 액션 |
- 회원명: text-primary (memberId 있으면 클릭 → 회원 상세)
- 결제금액: formatKRW(salePrice) right
- 미수금액: formatKRW(unpaidAmount) right font-bold text-state-error
- 연체일수: daysOverdue>30 ? text-state-error : daysOverdue>7 ? text-warning : text-content-secondary
  표시: `${daysOverdue}일`
- 액션: [수금처리] Button variant="primary" size="sm"

컴포넌트 구조:
1. unpaidItems: UnpaidItem[] — sales.unpaid>0 필터
2. daysOverdue: Math.floor((today - saleDate) / (1000*60*60*24))
3. filteredUnpaid: searchValue + filterValues 클라이언트 필터
4. totalUnpaid: sum(unpaidItems.map(u=>u.unpaidAmount))
5. maxOverdueDays: max(unpaidItems.map(u=>u.daysOverdue))
6. handleRowClick(row): setSelectedUnpaid(row) + showCollectModal → 04-수금처리

데이터:
- interface UnpaidItem { id, memberId, memberName, productName, salePrice, unpaidAmount, saleDate, daysOverdue }
- 소스: supabase.from('sales').select('id,memberId,memberName,productName,salePrice,unpaid,saleDate,branchId').eq('branchId',getBranchId()).gt('unpaid',0).order('saleDate',{ascending:false})
- unpaidAmount = row.unpaid
- daysOverdue = Math.floor((Date.now() - new Date(row.saleDate)) / 86400000)

인터랙션:
- [수금처리] 클릭 → 04-수금처리 모달
- [회원별 그룹] → 03-회원별그룹
- 행 클릭 → 04-수금처리 모달

사용 유틸:
- supabase from '@/lib/supabase'
- formatKRW from '@/lib/format'
- lucide-react: AlertCircle, Users, TrendingDown
```
