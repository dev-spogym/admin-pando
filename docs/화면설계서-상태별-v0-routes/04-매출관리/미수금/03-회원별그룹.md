```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-S008 미수금 — 상태: 회원별 그룹 뷰

파일: src/app/unpaid/page.tsx

레이아웃:
- 기본 레이아웃 동일
- 뷰 전환 버튼: [건별 목록] [회원별 그룹(active)]
- 아코디언 형태의 회원별 그룹 목록:
  {memberGroups.map(group => (
    <div key={group.memberId} className="border rounded-xl overflow-hidden mb-3">
      <div className="flex items-center justify-between p-4 bg-gray-50 cursor-pointer"
        onClick={()=>toggleGroup(group.memberId)}>
        <div className="flex items-center gap-3">
          <User className="w-5 h-5 text-gray-400" />
          <div>
            <p className="font-semibold text-[14px]">{group.memberName}</p>
            <p className="text-xs text-content-secondary">{group.items.length}건</p>
          </div>
        </div>
        <div className="flex items-center gap-3">
          <span className="font-bold text-state-error">{formatKRW(group.totalUnpaid)}</span>
          <ChevronDown className={`w-4 h-4 transition-transform ${expandedGroups.includes(group.memberId)?'rotate-180':''}`} />
        </div>
      </div>
      {expandedGroups.includes(group.memberId) && (
        <div className="divide-y">
          {group.items.map(item => (
            <div key={item.id} className="flex items-center justify-between px-4 py-3">
              <div>
                <p className="text-sm font-medium">{item.productName}</p>
                <p className="text-xs text-content-tertiary">{item.saleDate} · 연체 {item.daysOverdue}일</p>
              </div>
              <div className="flex items-center gap-2">
                <span className="text-sm font-semibold text-state-error">{formatKRW(item.unpaidAmount)}</span>
                <Button size="sm" variant="primary" onClick={()=>handleCollect(item)}>수금</Button>
              </div>
            </div>
          ))}
          <div className="px-4 py-3 bg-gray-50 flex justify-between">
            <span className="text-sm font-medium">합계</span>
            <div className="flex gap-2">
              <span className="font-bold text-state-error">{formatKRW(group.totalUnpaid)}</span>
              <Button size="sm" variant="outline" onClick={()=>handleCollectAll(group)}>전체 수금</Button>
            </div>
          </div>
        </div>
      )}
    </div>
  ))}

컴포넌트 구조:
1. viewMode: 'list' | 'group' — useState('list')
2. memberGroups: groupBy(unpaidItems, 'memberId').map(...)
   - 각 그룹: { memberId, memberName, items: UnpaidItem[], totalUnpaid }
   - 정렬: totalUnpaid 내림차순
3. expandedGroups: string[] — 펼쳐진 그룹 memberId 배열
4. toggleGroup(memberId): expandedGroups에 추가/제거
5. handleCollect(item): 단건 수금 → 04-수금처리
6. handleCollectAll(group): 그룹 전체 수금 → 04-수금처리 (전체)

데이터:
- memberGroups: unpaidItems를 memberId로 group by
- totalUnpaid: sum(group.items.map(i=>i.unpaidAmount))

인터랙션:
- 그룹 헤더 클릭 → 아코디언 토글
- [수금] 단건 → 04-수금처리 모달
- [전체 수금] → 04-수금처리 모달 (그룹 전체)
- [건별 목록] → 02-기본

사용 유틸:
- formatKRW from '@/lib/format'
- lucide-react: User, ChevronDown
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- [회원별 그룹] 뷰 전환 버튼 클릭

### 필수 데이터
- `unpaidItems: UnpaidItem[]` — 전체 미수금
- `memberGroups` — memberId 기준 group by

### 인터랙션 (User Actions)
1. 그룹 헤더 클릭 → 아코디언 토글
2. [수금] 단건 → `04-수금처리`
3. [전체 수금] → `04-수금처리` (그룹 전체)
4. [건별 목록] → `02-기본`

### 비즈니스 룰
- 그룹 정렬: totalUnpaid 내림차순 (가장 많은 미수금 상단)
- 전체 수금: 그룹 내 모든 미수금을 한 번에 처리

### 에지 케이스
- 비회원(memberId=null) → "비회원" 그룹으로 합산

### 연결 화면
- 이전: `02-기본`
- 다음: `04-수금처리`
