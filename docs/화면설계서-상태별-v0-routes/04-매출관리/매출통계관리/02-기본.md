## 📝 디스크립션

### 사용 시점 (Trigger)
- `01-로딩` 완료 후 `statItems.length > 0`

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 통계 항목 | `sales` (집계) | `branchId = getBranchId()`, group by type |
| 활성 상태 | 통계 설정 | isActive 토글 |

### 인터랙션 (User Actions)
1. Toggle 클릭 → isActive 즉시 업데이트 (optimistic update)
2. [수정] 클릭 → `03-수정저장` 모달 오픈
3. 체크박스 선택 → selectedIds 누적
4. [일괄 처리] → `04-일괄처리` 모달
5. [+ 항목 추가] → 빈 `03-수정저장` 모달

### 비즈니스 룰
- isActive=false 항목: 매출 통계 화면(SCR-S004)에서 미표시
- 삭제 시 단건 확인 toast + supabase delete
- 집계값은 현재 기간(이번달) 기준

### 에지 케이스
- Toggle 실패 → optimistic rollback + toast.error
- 선택 항목 0개 → [일괄 처리] disabled

### 접근성 (A11y)
- Toggle `role="switch"` + `aria-checked={isActive}`
- [삭제] 버튼 `aria-label="{항목명} 삭제"`
