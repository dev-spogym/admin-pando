레이아웃:
- <AppLayout> 데스크톱 레이아웃
- <PageHeader title="이연매출 (선수익금)" subtitle="미래에 인식될 매출을 조회합니다."
    actions={[<Button variant="outline" onClick={handleExcel}>엑셀</Button>]} />
- <StatCardGrid cols={3}>
    <StatCard label="총 이연매출" value={formatKRW(totalDeferred)} variant="peach" icon={TrendingUp} />
    <StatCard label="이번달 인식예정" value={formatKRW(thisMonthRecognition)} icon={Calendar} />
    <StatCard label="계약 건수" value={`${deferredItems.length}건`} icon={FileText} />
  </StatCardGrid>
- 뷰 전환: [계약별 목록(active)] [월별 인식] [차트]
- <SearchFilter> 검색어 + 날짜범위(계약기간)
- <DataTable columns={columns} data={filteredItems} />

DataTable 컬럼:
| No | 회원명 | 상품명 | 계약시작 | 계약종료 | 총금액 | 인식금액 | 이연금액 | 월 인식액 | 잔여개월 |
- 회원명: text-primary
- 총금액: formatKRW(totalAmount) right
- 인식금액: formatKRW(recognizedAmount) right text-state-success
- 이연금액: formatKRW(deferredAmount) right font-bold text-primary
- 월 인식액: formatKRW(monthlyRecognition) right text-content-secondary
- 잔여개월: `${remainingMonths}개월` center
- 진행바: 인식률 (recognizedAmount/totalAmount × 100) h-1.5 bg-gray-200 + bg-primary

컴포넌트 구조:
1. deferredItems: DeferredItem[] — 전체 로드
2. 이연매출 계산:
   - deferredAmount = totalAmount - recognizedAmount
   - recognizedAmount = totalAmount × (경과일수 / 전체일수)
   - monthlyRecognition = totalAmount / 전체개월수
   - remainingMonths = Math.ceil((contractEndDate - today) / 30)
3. totalDeferred = sum(deferredItems.map(i=>i.deferredAmount))
4. thisMonthRecognition = sum(deferredItems.map(i=>i.monthlyRecognition))
5. 뷰 전환: viewMode state

데이터:
- supabase.from('deferred_revenue').select('*').eq('branchId', getBranchId())
  또는 supabase.from('sales').select('*').eq('branchId',getBranchId()).not('contractEndDate','is',null)
  → 클라이언트에서 이연매출 계산

인터랙션:
- [월별 인식] → 03-월별뷰
- [차트] → 04-차트뷰
- 행 클릭 → 계약 상세 (회원 상세 이동 또는 인라인 확장)

사용 유틸:
- formatKRW from '@/lib/format'
- lucide-react: TrendingUp, Calendar, FileText
- differenceInDays, differenceInMonths from 'date-fns'

4. 엑셀 → CSV 다운로드

### 비즈니스 룰
- 이연매출 산식: `deferredAmount = totalAmount × (잔여일수 / 전체일수)`
- 계약 만료 건: deferredAmount=0 (목록에서 제외 또는 별도 표시)
- 월별 인식액: `totalAmount / 전체 계약월수`

### 에지 케이스
- contractEndDate 없는 건 → 이연매출 계산 불가, 제외
- 이미 만료된 계약 → deferredAmount=0
