```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.

파일: src/app/deferred-revenue/page.tsx

레이아웃:
- 뷰 전환: [계약별 목록] [월별 인식(active)] [차트]
- 월별 인식 스케줄 테이블:
  DataTable 컬럼: | 월 | 인식 예정액 | 누적 인식액 | 잔여 이연액 | 계약 건수 |
  - 월: YYYY-MM 형식
  - 이번달: 행 하이라이트 (bg-primary/5 border-l-2 border-primary)
  - 인식 예정액: formatKRW(monthlyTotal) right font-semibold
  - 누적 인식액: formatKRW(cumulativeAmount) right
  - 잔여 이연액: formatKRW(remainingDeferred) right text-primary
  - 계약 건수: count center

컴포넌트 구조:
1. viewMode = 'monthly'
2. monthlySchedule: 월별 집계 계산
   - 각 DeferredItem에서 contractStartDate ~ contractEndDate 기간의 월별 인식액 생성
   - { month, monthlyTotal, cumulativeAmount, remainingDeferred, count }
3. 계산 로직:
   deferredItems.forEach(item => {
     const months = eachMonthOfInterval({ start: item.contractStartDate, end: item.contractEndDate })
     const perMonth = item.totalAmount / months.length
     months.forEach(m => {
       const key = format(m, 'yyyy-MM')
       schedule[key] = (schedule[key] || 0) + perMonth
     })
   })
4. 이번달 행: format(today, 'yyyy-MM') === month → 하이라이트

데이터:
- monthlySchedule: { month, monthlyTotal, cumulativeAmount, remainingDeferred, count }[]
- 정렬: month 오름차순 (미래 포함)
- 범위: 가장 이른 contractStartDate ~ 가장 늦은 contractEndDate

인터랙션:
- 탭 전환: 02-기본 / 04-차트뷰
- 이번달 행 자동 스크롤 (useEffect)

사용 유틸:
- eachMonthOfInterval, format from 'date-fns'
- formatKRW from '@/lib/format'
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- [월별 인식] 탭 클릭

### 필수 데이터
- `deferredItems: DeferredItem[]` — 전체 이연매출 계약
- `monthlySchedule` — 월별 집계 (클라이언트 계산)

### 인터랙션 (User Actions)
1. 탭 전환 → `02-기본` / `04-차트뷰`
2. 행 클릭 → 해당 월의 계약 목록 드릴다운 (optional)

### 비즈니스 룰
- 월별 인식액 = totalAmount / 계약 전체 월수 (균등 안분)
- 이미 지난 달: 인식 완료 (회색 처리)
- 미래 달: 인식 예정 (기본 스타일)
- 이번달: 강조 하이라이트

### 에지 케이스
- 계약 기간 내 특정 월에 집중 (비균등 안분): 현재 균등 안분 적용
