  <div className="space-y-4 mt-4 border-t pt-4">
    <h4 className="font-medium text-sm text-content-secondary">현금 결제</h4>
    <div>
      <label className="text-sm font-medium">받은 금액</label>
      <input type="number" placeholder="0" value={receivedAmount}

      <div className="flex gap-2 mt-2">
        {[1000,5000,10000,50000].map(v=>(

            onClick={()=>setReceivedAmount(prev=>prev+v)}>+{formatNumber(v)}</button>

          onClick={()=>setReceivedAmount(totalAmount)}>정액</button>
      </div>
    </div>
    <div className="bg-gray-50 rounded-lg p-3 space-y-2">
      <div className="flex justify-between text-sm">
        <span>총 결제금액</span><span className="font-medium">{formatKRW(totalAmount)}</span>
      </div>
      <div className="flex justify-between text-sm">
        <span>받은 금액</span><span className="font-medium">{formatKRW(receivedAmount)}</span>
      </div>
      <div className="flex justify-between font-bold text-[15px]">
        <span>거스름돈</span>
        <span className={changeAmount>=0?'text-state-success':'text-state-error'}>
          {changeAmount>=0?formatKRW(changeAmount):`부족: ${formatKRW(-changeAmount)}`}
        </span>
      </div>
    </div>
    {unpaidAmount > 0 && (
      <div className="border border-state-warning/30 rounded-lg p-3 bg-yellow-50">
        <p className="text-sm text-yellow-700">미수금 {formatKRW(unpaidAmount)} 발생</p>

      </div>
    )}
    <Button variant="primary" className="w-full" onClick={handleCashPayment}

      {formatKRW(totalAmount)} 현금 결제

컴포넌트 구조:
1. receivedAmount: number — useState(0)
2. changeAmount: receivedAmount - totalAmount
3. unpaidAmount: receivedAmount < totalAmount ? totalAmount - receivedAmount : 0
4. 빠른 금액 버튼: +1000, +5000, +10000, +50000, 정액(totalAmount)
5. handleCashPayment():
   await supabase.from('sales').insert({
     ...salesBase,
     paymentMethod: 'CASH',
     cash: Math.min(receivedAmount, totalAmount),
     card: 0,
     unpaid: unpaidAmount,
     status: unpaidAmount>0 ? 'UNPAID' : 'COMPLETED'

   if (changeAmount > 0) toast.info(`거스름돈: ${formatKRW(changeAmount)}`)
   router.push('/pos?success=true')

데이터:
- receivedAmount: number (받은 현금)
- changeAmount: receivedAmount - totalAmount
- unpaidAmount: max(0, totalAmount - receivedAmount)
- DB: sales.paymentMethod='CASH', cash=min(received,total), unpaid=unpaidAmount

인터랙션:
- 금액 직접 입력 또는 빠른 버튼으로 누적
- [정액] → receivedAmount=totalAmount (거스름돈 0)
- [현금 결제] → handleCashPayment

사용 유틸:

## 📝 디스크립션

### 사용 시점 (Trigger)
- [현금] 결제 수단 버튼 클릭

### 필수 데이터
- `totalAmount` — 결제금액
- `receivedAmount` — 받은 현금액 (사용자 입력)

### 인터랙션 (User Actions)
1. 금액 입력 (직접 또는 빠른 버튼)
2. [정액] → receivedAmount=totalAmount
3. [현금 결제] → DB insert → 성공/실패
4. 거스름돈 자동 계산 표시

### 비즈니스 룰
- 거스름돈 = receivedAmount - totalAmount (음수면 부족금 표시)
- 미수금 = max(0, totalAmount - receivedAmount) → `sales.unpaid` 저장
- 미수금 > 0 이면 `status='UNPAID'`, 0이면 `status='COMPLETED'`
- receivedAmount=0 이면 [현금 결제] disabled

### 에지 케이스
- receivedAmount < totalAmount → 미수금 경고 배너 + 진행 가능
- receivedAmount > totalAmount → 거스름돈 표시 (초록색)

### 접근성 (A11y)
- 거스름돈 `aria-live="polite"` (실시간 업데이트)
- 금액 input `inputMode="numeric"` + `aria-label="받은 현금 금액"`
