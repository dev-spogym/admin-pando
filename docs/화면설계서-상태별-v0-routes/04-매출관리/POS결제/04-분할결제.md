레이아웃:
- paymentMethod==='분할결제' 선택 시 분할 결제 폼:
  <div className="space-y-4 mt-4 border-t pt-4">
    <h4 className="font-medium text-sm text-content-secondary">분할 결제</h4>
    <div className="grid grid-cols-2 gap-3">
      <div>

          className="w-full border rounded-lg px-3 py-2 text-right tabular-nums" />
      </div>
      <div>
        <label className="text-sm font-medium">현금 금액</label>

      </div>

        <input type="number" value={splitForm.mileage} min={0} max={buyer.mileage}
          onChange={e=>handleSplitChange('mileage', Number(e.target.value))}
          className="w-full border rounded-lg px-3 py-2 text-right tabular-nums" />
        <p className="text-xs text-content-tertiary mt-1">보유 마일리지: {formatNumber(buyer.mileage)}P</p>
      </div>
    )}
    <div className="bg-gray-50 rounded-lg p-3 space-y-1">
      <div className="flex justify-between text-sm">
        <span>총 결제금액</span><span className="font-semibold">{formatKRW(totalAmount)}</span>
      </div>
      <div className="flex justify-between text-sm">
        <span>입력 합계</span>
        <span className={splitTotal===totalAmount?'text-state-success font-medium':'text-state-error font-medium'}>

        </span>
      </div>
      {splitTotal !== totalAmount && (
        <p className="text-xs text-state-error">
          {splitTotal < totalAmount ? `${formatKRW(totalAmount-splitTotal)} 부족` : `${formatKRW(splitTotal-totalAmount)} 초과`}
        </p>
      )}
    </div>
    <Button variant="primary" className="w-full" onClick={handleSplitPayment}

      {formatKRW(totalAmount)} 분할 결제

   })

데이터:
- splitForm: { card, cash, mileage }

## 📝 디스크립션

### 사용 시점 (Trigger)
- [분할결제] 결제 수단 버튼 클릭

### 필수 데이터
- `totalAmount` — 총 결제금액
- `splitForm: { card, cash, mileage }`
- `buyer.mileage` — 보유 마일리지 (마일리지 사용 시)

### 인터랙션 (User Actions)
1. 카드 금액 입력 → 현금 자동 계산 (나머지)
2. 현금 금액 입력 → 카드 자동 계산
3. 마일리지 추가 (회원일 때) → 세 방향 분할
4. splitTotal === totalAmount → 버튼 활성화
5. [분할 결제] → DB insert

### 비즈니스 룰
- splitTotal === totalAmount 이어야 결제 버튼 활성
- 마일리지: buyer가 있는 경우만 활성 (비회원 불가)
- 마일리지 사용량 ≤ buyer.mileage (보유량 초과 불가)
- DB: card, cash, mileageUsed 각각 저장

### 에지 케이스
- splitTotal > totalAmount → "초과" 경고 (빨간색)
- splitTotal < totalAmount → "부족" 경고 (빨간색)
- 마일리지 > 보유량 → 경고 + 버튼 disabled

### 접근성 (A11y)
- 합계 차이 `aria-live="polite"` (실시간 업데이트)
- 각 금액 input `aria-label="{수단} 결제 금액"`
