## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-S002 POS — 상태: 회원 선택 모달 (구매자 설정)

파일: src/app/pos/page.tsx

레이아웃:
- 기본 페이지 위에 <Modal> 오버레이
- Modal title="회원 검색" size="md"
- 모달 본문:
  <div className="space-y-4">
    <input type="text" placeholder="이름 또는 전화번호 검색..."
      value={memberSearch} onChange={e=>setMemberSearch(e.target.value)}
      className="w-full border rounded-lg px-3 py-2" autoFocus />
    <div className="max-h-64 overflow-y-auto space-y-1">
      {filteredMembers.map(m => (
        <div key={m.id}
          className="flex items-center justify-between p-3 rounded-lg hover:bg-gray-50 cursor-pointer"
          onClick={() => handleSelectMember(m)}>
          <div>
            <p className="font-medium text-[14px]">{m.name}</p>
            <p className="text-xs text-content-secondary">{m.phone} | {m.membershipType}</p>
          </div>
          <ChevronRight className="w-4 h-4 text-gray-400" />
        </div>
      ))}
      {filteredMembers.length===0 && (
        <p className="text-center py-8 text-content-tertiary text-sm">검색 결과가 없습니다.</p>
      )}
    </div>
  </div>
- 모달 푸터:
  [비회원으로 진행] variant="outline" → setBuyer(null) + setShowMemberModal(false)
  [닫기] variant="ghost" → setShowMemberModal(false)

컴포넌트 구조:
1. showMemberModal: boolean
2. memberSearch: string — debounce 300ms
3. filteredMembers: members.filter(m => m.name.includes(q) || m.phone.includes(q))
4. members: Member[] — supabase.from('members').select('id,name,phone,membershipType').eq('branchId',getBranchId()).eq('status','active')
5. handleSelectMember(m): setBuyer(m) + setShowMemberModal(false) + toast.success(`${m.name}님 선택`)

데이터:
- interface Member { id, name, phone, membershipType }
- buyer: Member | null — 선택된 구매자
- 검색: 이름 + 전화번호 포함 검색

인터랙션:
- 검색 input 실시간 필터 (debounce 300ms)
- 회원 행 클릭 → handleSelectMember → 03-장바구니
- [비회원으로 진행] → buyer=null + 모달 닫기
- ESC / [닫기] → 모달 닫기

사용 유틸:
- Modal from '@/components/ui/Modal'
- supabase from '@/lib/supabase'
- toast from 'sonner'
- lucide-react: ChevronRight, User
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- 장바구니 우측 [회원 검색] 버튼 클릭

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 회원 목록 | `members` | `branchId=?`, `status='active'` |

### 인터랙션 (User Actions)
1. 검색 입력 → 실시간 이름/전화번호 필터
2. 회원 행 클릭 → buyer 설정 + 모달 닫기
3. [비회원으로 진행] → buyer=null + 진행
4. ESC / [닫기] → 취소

### 비즈니스 룰
- 비회원 결제 허용: buyer=null로 진행 가능
- 탈퇴/휴면 회원 목록에서 제외 (status='active')
- 선택 후 장바구니에 구매자 이름 표시

### 에지 케이스
- 검색 결과 0건 → "검색 결과가 없습니다."
- 회원 선택 후 변경 → X 버튼으로 buyer=null + 재검색

### 접근성 (A11y)
- 모달 오픈 시 검색 input `autoFocus`
- 회원 목록 `role="listbox"`, 행 `role="option"`

- 다음: `03-장바구니` (회원 선택)
