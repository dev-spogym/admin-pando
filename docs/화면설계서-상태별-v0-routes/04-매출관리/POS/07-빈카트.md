## 📝 디스크립션

### 사용 시점 (Trigger)
- 페이지 최초 진입 시 (`cart=[]`)
- [전체삭제] 클릭 후
- `06-완료` (결제 완료 후 초기화)

### 필수 데이터
- `cart: []`

### 인터랙션 (User Actions)
1. 상품 카드 클릭 → `03-장바구니` 전환
2. [회원 검색] → `02-회원선택` (카트 상태와 독립)
3. [결제하기] disabled — 인터랙션 없음

### 비즈니스 룰
- 결제하기 버튼은 cart 비어있으면 항상 disabled
- buyer 선택은 cart 비어있어도 가능

### 에지 케이스
- 결제 완료 후 돌아온 경우: 성공 배너 표시 + 빈 카트

### 접근성 (A11y)
- disabled 버튼 `aria-disabled="true"` + `title="상품을 먼저 선택하세요"`
- 빈 상태 `role="status"`

### 연결 화면
- 이전: `03-장바구니` (전체삭제) / `06-완료`
- 다음: `03-장바구니` (상품 추가)
