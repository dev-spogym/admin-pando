## 📝 디스크립션

### 사용 시점 (Trigger)
- `/pos` 페이지 진입 시 기본 상태
- 장바구니가 비어있는 초기 상태

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 상품 목록 | `products` | `branchId=?`, `isActive=true` |
| 카테고리 | `products` | group by type |

### 인터랙션 (User Actions)
1. 상품 카드 클릭 → 장바구니 추가 → `03-장바구니`
2. 탭 클릭 → 카테고리 필터
3. 검색 입력 → 실시간 상품명 필터
4. [회원 검색] → `02-회원선택`
5. [+ 상품 등록] → 상품 등록 페이지 (권한자만)

### 비즈니스 룰
- 품절(stock===0) 카드: 클릭 불가, 하단 정렬
- 트레이너: [+ 상품 등록] 버튼 숨김
- 동일 상품 재클릭: 수량 +1

### 에지 케이스
- 상품 0개 → `07-빈카트` 상태에서 상품 없음 안내
- 카테고리 없는 상품 → "기타" 탭

### 접근성 (A11y)
- 상품 카드 `role="button"` + `tabIndex={0}` + Enter/Space로 추가
- 품절 카드 `aria-disabled="true"`
