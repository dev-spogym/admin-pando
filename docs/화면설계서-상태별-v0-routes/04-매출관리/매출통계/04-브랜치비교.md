     .in('branchId', selectedBranches)

       <YAxis tickFormatter={v=>formatKRW(v)} />
       {selectedBranches.map((bid,i) => (
         <Line key={bid} dataKey={bid} name={branchName[bid]} stroke={COLORS[i]} />
       ))}
       <Tooltip formatter={(v)=>formatKRW(v)} />
       <Legend />
     </LineChart>

## 📝 디스크립션

### 사용 시점 (Trigger)
- PageHeader [브랜치 비교] 버튼 클릭 (본사/슈퍼 권한)

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 브랜치 목록 | `branches` | 전체 |
| 브랜치별 매출 | `sales` | `branchId IN (selectedBranches)` |

### 인터랙션 (User Actions)
1. [브랜치 비교] 토글 → 비교 모드 활성
2. 브랜치 multiSelect → 최대 5개 선택
3. 차트 레전드 클릭 → 해당 브랜치 라인 숨김/표시
4. [비교 해제] → 단일 브랜치 모드 복귀

### 비즈니스 룰
- 권한: 본사(슈퍼관리자)만 브랜치 비교 가능
- 최소 2개 브랜치 선택 필요
- 최대 5개 브랜치 비교 (가독성)
- 집계: `sum(salePrice) group by branchId, month`

### 에지 케이스
- 선택 브랜치 1개 → "2개 이상 선택하세요" toast.warning
- 브랜치 없음 → 선택 불가

### 접근성 (A11y)
- 차트 `aria-label="브랜치별 매출 비교 차트"`
- 색상 외 패턴/점선으로도 구분 (색맹 고려)

### 연결 화면
- 이전: `02-기본-차트`
- 다음: `02-기본-차트` (비교 해제)
