```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-S004 매출 통계 — 상태: 기본 (월별 매출 차트)

파일: src/app/sales/stats/page.tsx

레이아웃:
- <AppLayout> 데스크톱 레이아웃
- <PageHeader title="매출 통계" subtitle="기간별·브랜치별 매출 추이를 분석합니다."
    actions={[기간선택 Select, 브랜치비교 Button]} />
- <StatCardGrid cols={4}>
    <StatCard label="총 매출" value={formatKRW(summary.totalRevenue)} icon={TrendingUp} />
    <StatCard label="거래 건수" value={`${summary.totalCount}건`} icon={BarChart2} />
    <StatCard label="평균 매출" value={formatKRW(summary.avgRevenue)} icon={DollarSign} />
    <StatCard label="환불율" value={`${summary.refundRate.toFixed(1)}%`} icon={RefreshCw} variant={summary.refundRate>5?'error':'default'} />
  </StatCardGrid>
- 기간 탭: [최근 3개월] [최근 6개월] [올해] [직접입력]
- 차트 영역 (bg-white rounded-xl border p-6):
  · 탭: [막대차트] [선형차트]
  · Recharts BarChart or LineChart
  · X축: month (YYYY-MM), Y축: revenue (formatKRW)
  · Tooltip: month + revenue + count
  · Legend: 매출액 / 환불액
- DataTable: 월별 집계 테이블 (월 / 매출 / 건수 / 환불 / 순매출)

컴포넌트 구조:
1. StatCardGrid — 전체 기간 집계
2. 기간 탭 + 차트 타입 토글
3. Recharts BarChart (기본) or LineChart
4. DataTable 월별 집계

데이터:
- interface MonthlyData { month: string, revenue: number, count: number, refund: number, net: number }
- 쿼리: supabase.from('sales').select('saleDate,salePrice,status,branchId').eq('branchId',getBranchId())
- 집계: salesData.reduce → month별 group by
  · month: saleDate.slice(0,7)
  · revenue: status!=='환불' ? salePrice : 0
  · refund: status==='환불' ? salePrice : 0
  · count: status!=='환불' 건수
  · net: revenue - refund

인터랙션:
- 기간 탭 변경 → statsData 필터 업데이트 (클라이언트)
- [막대차트]/[선형차트] 토글 → chartType 변경
- 테이블 행 클릭 → 해당 월 상세 (moveToPage(970) + 날짜필터)

사용 유틸:
- recharts: BarChart, LineChart, XAxis, YAxis, Tooltip, Legend, ResponsiveContainer
- formatKRW / formatNumber from '@/lib/format'
- lucide-react: TrendingUp, BarChart2, DollarSign, RefreshCw
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- `01-로딩` 완료 후 데이터 존재
- 기간 탭 변경 시 클라이언트 필터 재적용

### 필수 데이터
| 블록 | 테이블 | 집계 |
|------|--------|------|
| 월별 매출 | `sales` | `sum(salePrice) group by month` |
| 환불율 | `sales` | `count(status='환불') / count(*)` |
| 브랜치 | `branches` | `branchId, name` |

### 인터랙션 (User Actions)
1. 기간 탭(3개월/6개월/올해) 클릭 → statsData 범위 변경
2. 차트 타입 토글 → BarChart ↔ LineChart
3. 테이블 행 클릭 → 해당 월 매출 목록으로 이동
4. [브랜치 비교] → `04-브랜치비교` 상태

### 비즈니스 룰
- 환불율 = 환불 건수 / 전체 건수 × 100
- 환불율 > 5% → StatCard variant="error" (경고색)
- 집계 탭(TAB-002~005) 데이터와 동일 로직 재사용

### 에지 케이스
- 데이터 없는 월 → 차트 0값으로 표시 (공백 없음)
- 단일 데이터 → 라인차트 점 표시

### 접근성 (A11y)
- 차트 컨테이너 `role="img"` + `aria-label="월별 매출 차트"`
- DataTable 대체 텍스트로 동일 데이터 제공

### 연결 화면
- 이전: `01-로딩`
- 다음: `03-기간변경` / `04-브랜치비교` / `05-상품별`
