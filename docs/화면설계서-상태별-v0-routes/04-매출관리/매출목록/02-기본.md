## 📝 디스크립션

### 사용 시점 (Trigger)
- `01-로딩` 완료 후 `sales.length > 0`인 경우
- 기본 필터: 이번달 1일 ~ 오늘, 유형 전체, 상태 전체

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 매출 목록 | `sales` | `branchId = getBranchId()`, `saleDate` 내림차순 |
| 요약 집계 | `sales` (클라이언트 계산) | filteredData 기반 |

### 인터랙션 (User Actions)
1. 기간 프리셋 [오늘/이번주/이번달] 클릭 → 날짜범위 변경
2. SearchFilter 검색어 입력 → 300ms debounce 후 클라이언트 필터
3. 유형/상태 multiSelect → 클라이언트 필터
4. 탭 클릭 → activeTab 변경 + 데이터 전환
5. 행 클릭 → `05-상세모달`로 전환
6. 상품명 클릭 → `/products/[id]` (moveToPage 971)
7. 구매자 클릭 → `/members/[id]` (moveToPage 985)
8. 엑셀 버튼 → CSV 다운로드 + `07-엑셀내보내기`

### 비즈니스 룰
- 트레이너: 본인 담당 매출만 조회 (staffId 자동 필터) + 안내 배너 표시
- 환불 건: `status === '환불'` → TAB-006에만 포함, 집계 탭에서 제외
- 미납 건: `unpaid > 0` → TAB-007에도 포함
- netTotal = summary.total - summary.refund

### 에지 케이스
- 검색결과 0건 → `06-빈상태`
- 미수금 > 0인 StatCard → `border-state-error/20` 하이라이트

### 접근성 (A11y)
- DataTable 행 `role="row"` + `tabIndex={0}` + Enter/Space로 모달 열기
- StatCard `aria-label="{label} {value}"`
