   - 서비스 일수: `${serviceDays}일`

7. [메모] 섹션 (memo 값 있을 때): whitespace-pre-wrap text-content-tertiary

## 📝 디스크립션

### 사용 시점 (Trigger)
- TAB-001(매출내역), TAB-006(환불내역), TAB-007(미납내역)에서 테이블 행 클릭
- 집계 탭(TAB-002~005)에서는 행 클릭 이벤트 없음

### 필수 데이터
- `selectedSale: SaleItem` — 클릭한 행의 데이터
- `showDetailModal: boolean` — 모달 표시 여부

### 인터랙션 (User Actions)
1. 행 클릭 → selectedSale 설정 + showDetailModal=true
2. [회원 상세 보기] → 회원 상세 페이지 이동
3. [닫기] / ESC / 배경 클릭 → showDetailModal=false
4. 영수증 재발행(optional) → window.print()

### 비즈니스 룰
- 카드 승인 정보 섹션: cardCompany, cardNumber, approvalNo 중 하나라도 있으면 표시
- 서비스 섹션: serviceDays>0 OR serviceCount>0일 때만 표시
- 메모 섹션: memo 값이 존재할 때만 표시
- 미수금 > 0 → 빨간 텍스트(text-state-error)

### 에지 케이스
- selectedSale이 null인 경우 → 모달 렌더링 안 함 (`selectedSale && <Modal>`)
- 구매자 없는 거래(비회원 현금 결제) → buyer 필드 '-' 표시
- 할인 없는 거래 → 할인 행 '-' 표시

### 접근성 (A11y)
- Modal `role="dialog"` + `aria-modal="true"` + `aria-labelledby="modal-title"`
- 모달 오픈 시 포커스 트랩 (첫 번째 버튼으로 포커스 이동)
- ESC 키로 닫기

- 다음: `02-기본` (닫기) / 회원 상세 페이지(moveToPage 985)
