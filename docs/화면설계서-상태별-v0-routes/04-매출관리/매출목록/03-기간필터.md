```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-S001 매출 목록 — 상태: 기간 필터 적용 (프리셋 or 날짜범위 직접 입력)

파일: src/app/sales/page.tsx

레이아웃:
- 기본 레이아웃 동일
- 기간 프리셋 버튼: 선택된 버튼은 bg-primary text-white, 나머지는 outline
- SearchFilter 날짜범위 필드: 시작일~종료일 하이라이트
- StatCardGrid: 선택 기간 기반 집계값으로 업데이트
- DataTable: 클라이언트 필터 적용된 filteredData 표시

컴포넌트 구조:
1. PRESETS 상수:
   const PRESETS = {
     오늘: { start: fmtLocal(today), end: fmtLocal(today) },
     이번주: { start: fmtLocal(startOfWeek), end: fmtLocal(endOfWeek) },
     이번달: { start: fmtLocal(startOfMonth), end: fmtLocal(endOfMonth) }
   }
2. activePreset state: '오늘' | '이번주' | '이번달' | null
3. handlePreset(preset): setActivePreset(preset) + setFilterValues({dateRangeStart, dateRangeEnd, ...})
4. 날짜 직접 입력 시 setActivePreset(null) — 프리셋 해제
5. filteredData: filterValues.dateRangeStart ~ End 범위로 purchaseDate 클라이언트 필터
6. StatCardGrid 집계 재계산

데이터:
- 기존 salesData(전체) + filterValues.dateRangeStart/End 필터 적용
- 클라이언트 필터링 (DB 재호출 없음)
- 필터링 로직: item.purchaseDate >= dateRangeStart && item.purchaseDate <= dateRangeEnd

인터랙션:
- [오늘] 클릭 → activePreset='오늘', 날짜범위 오늘~오늘
- [이번주] 클릭 → activePreset='이번주', 월요일~일요일
- [이번달] 클릭 → activePreset='이번달', 1일~말일
- 날짜 input 직접 변경 → activePreset=null, 날짜범위 직접 적용
- 결과 0건 → 06-빈상태

사용 유틸:
- useAuthStore from '@/stores/authStore'
- formatKRW / formatNumber from '@/lib/format'
- startOfWeek, endOfWeek, startOfMonth, endOfMonth (date-fns)
- lucide-react: Calendar
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- 기간 프리셋 버튼(오늘/이번주/이번달) 클릭
- SearchFilter 날짜범위 직접 입력

### 필수 데이터
- `salesData: SaleItem[]` — 전체 데이터 (초기 1회 로드)
- `filterValues.dateRangeStart / dateRangeEnd` — 날짜 필터값
- `activePreset: string | null` — 활성 프리셋

### 인터랙션 (User Actions)
1. [오늘] → 오늘~오늘 필터
2. [이번주] → 이번 주 월~일 필터
3. [이번달] → 이번 달 1일~말일 필터 (기본)
4. 날짜 직접 입력 → 커스텀 범위 적용
5. 결과 없음 → `06-빈상태`

### 비즈니스 룰
- 클라이언트 필터링 (DB 재호출 없음 — 전체 데이터 한 번에 로드)
- `purchaseDate`는 `saleDate.slice(0,10)` (YYYY-MM-DD)
- 필터 초기화(onReset): activePreset='이번달'로 복귀

### 에지 케이스
- 선택 기간에 데이터 없음 → `06-빈상태`
- 시작일 > 종료일 → SearchFilter 자체 validation으로 방지

### 접근성 (A11y)
- 활성 프리셋 버튼 `aria-pressed="true"`
- 날짜 input `type="date"` + `aria-label="시작일"/"종료일"`

### 연결 화면
- 이전: `02-기본`
- 다음: `02-기본` (필터 결과) / `06-빈상태`
