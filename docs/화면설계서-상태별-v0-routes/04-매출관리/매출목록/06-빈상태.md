## 📝 디스크립션

### 사용 시점 (Trigger)
- 데이터 로드 후 `filteredData.length === 0`
- 필터/검색 결과가 0건인 경우
- 아직 매출이 전혀 없는 신규 센터

### 필수 데이터
- `filteredData: []` — 빈 배열
- `hasActiveFilter: boolean` — 필터 활성 여부 (빈상태 메시지 분기)

### 인터랙션 (User Actions)
1. [필터 초기화] 클릭 → handleReset() → `02-기본` 복귀
2. [신규 결제(POS)] 클릭 → moveToPage(982) → POS 페이지 이동
3. 기간 프리셋/필터 직접 변경 → 다른 기간 조회

### 비즈니스 룰
- 필터 있음 + 결과 0건: "필터 조건에 해당하는 매출이 없습니다." + 초기화 버튼
- 필터 없음 + 결과 0건: "매출 데이터가 없습니다." + POS 이동 버튼
- StatCardGrid는 ₩0으로 표시 (숨기지 않음)

### 에지 케이스
- 트레이너 역할 + 담당 매출 0건 → "담당 매출이 없습니다." 문구 변형 가능

### 접근성 (A11y)
- 빈 상태 컨테이너 `role="status"` + `aria-live="polite"`
- 아이콘은 `aria-hidden="true"` (장식용)

### 연결 화면

- 다음: `02-기본` (필터 초기화)
