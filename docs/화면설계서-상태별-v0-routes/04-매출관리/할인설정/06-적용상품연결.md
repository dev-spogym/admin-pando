## 📝 디스크립션

### 사용 시점 (Trigger)
- DataTable [상품 연결] 버튼 클릭

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 상품 목록 | `products` | `branchId=?`, `isActive=true` |
| 기존 연결 | `discount_policies.applicableProducts` | 현재 정책 |

### 인터랙션 (User Actions)
1. 상품 검색 → 실시간 필터
2. 체크박스 → 개별 선택/해제
3. [전체 선택/해제] → 일괄 처리
4. [연결 저장] → DB UPDATE applicableProducts
5. [취소] → 취소

### 비즈니스 룰
- applicableProducts=[] → 모든 상품에 적용 가능 (전체 허용)
- applicableProducts에 id 목록 → 해당 상품만 적용 가능
- POS 할인 Select에서 선택한 상품과 정책 연결 여부 검증

### 에지 케이스
- 상품 없는 브랜치 → "등록된 상품이 없습니다." 빈 상태
- 저장 실패 → toast.error + 모달 유지

### 접근성 (A11y)
- `role="dialog"` + 포커스 트랩
- 체크박스 그룹 `role="group"` + `aria-label="적용 상품 선택"`
- 선택 수 `aria-live="polite"`
