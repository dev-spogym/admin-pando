## 📝 디스크립션

### 사용 시점 (Trigger)
- [+ 할인 정책 추가] 버튼 클릭

### 필수 데이터
- `form: DiscountPolicyForm` — 빈 폼
- DB INSERT: `discount_policies`

### 인터랙션 (User Actions)
1. 정책명 입력 (필수)
2. 유형 선택 (율/정액)
3. 할인값 입력 (필수)
4. 최소금액/최대할인 (선택)
5. 코드 (선택, 대문자 자동 변환)
6. 기간 (선택, 없으면 상시)
7. 즉시 활성화 토글
8. [저장] → INSERT + 목록 갱신

### 비즈니스 룰
- 정책명: 필수, 최대 50자
- 할인율: 1~100%
- 코드: 대문자, 최대 20자, 중복 불가
- 기간 없음 = 상시 적용

### 에지 케이스
- 코드 중복 → DB unique constraint → toast.error('이미 사용 중인 코드입니다.')
- 종료일 < 시작일 → form validation

### 접근성 (A11y)
- `role="dialog"` + 포커스 트랩
- 필수 필드 `aria-required="true"`

### 연결 화면
- 이전: `02-기본`
- 다음: `02-기본` (저장 완료)
