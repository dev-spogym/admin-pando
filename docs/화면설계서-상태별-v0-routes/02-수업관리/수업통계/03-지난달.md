```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-C005 수업 통계 — 상태: 지난달

파일: src/app/(classes)/class-stats/page.tsx

레이아웃:
- 이번달 상태와 동일 구조
- [지난달] 탭 활성 상태

기간 계산:
- lastMonthStart = startOfMonth(subMonths(new Date(), 1))
- lastMonthEnd = endOfMonth(subMonths(new Date(), 1))

StatCard 추가 정보:
- 각 카드에 이번달 대비 증감률 표시
- trend prop: { value: +12.5, direction: 'up' | 'down' | 'same' }
- 증가: text-green-600 ▲ N%
- 감소: text-red-500 ▼ N%
- 동일: text-gray-400 - 0%

DataTable 컬럼 (지난달):
- 수업명 | 유형 | 담당강사 | 총 횟수 | 총 예약 | 평균 수강률 | 이번달 대비

데이터:
- supabase.from('lesson_schedules').select('*, lessons(*), staff(*), lesson_bookings(count)')
    .gte('start_at', lastMonthStart.toISOString())
    .lte('start_at', lastMonthEnd.toISOString())
- 이번달 데이터도 함께 fetch → 증감률 계산

인터랙션:
- [이번달] 탭 클릭 → 02-이번달 전환
- 수업 행 클릭 → 04-차트렌더

사용 유틸:
- useAuthStore from '@/stores/authStore'
- supabase from '@/lib/supabase'
- formatNumber from '@/lib/format'
- date-fns: startOfMonth, endOfMonth, subMonths
- lucide-react: TrendingUp, TrendingDown, Minus
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- [지난달] 탭 클릭

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 지난달 수업 | `lesson_schedules` | `branch_id = ?`, 지난달 범위 |
| 이번달 수업 | `lesson_schedules` | `branch_id = ?`, 이번달 범위 (증감률 계산용) |

### 인터랙션 (User Actions)
1. [이번달] 탭 → `02-이번달` 전환
2. 수업 행 클릭 → `04-차트렌더`

### 비즈니스 룰
- 증감률 = (이번달 - 지난달) / 지난달 × 100
- 지난달 값 0인 경우 증감률 표시 불가 → "-" 표시

### 에지 케이스
- 지난달 수업 0건: 모든 지표 0 + "지난달 데이터가 없습니다." 안내
