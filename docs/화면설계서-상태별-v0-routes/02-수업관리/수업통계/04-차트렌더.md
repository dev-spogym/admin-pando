## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-C005 수업 통계 — 상태: 특정 수업 차트 렌더링

파일: src/app/(classes)/class-stats/page.tsx

레이아웃:
- 상단 StatCard + 테이블 유지
- 테이블 선택 행 하이라이트 (bg-primary/5 border-l-2 border-primary)
- 하단에 차트 패널 슬라이드다운 (animate-slideDown)

차트 패널 구조:
1. [패널 헤더]
   - 수업명 (font-bold) + 유형배지 + [닫기 X]
   - 기간 셀렉터: 최근 3개월 / 최근 6개월 / 올해

2. [수업 횟수 추이 차트] (recharts LineChart 또는 BarChart)
   - X축: 월 (YYYY-MM)
   - Y축: 수업 횟수
   - 라인: 수업 횟수(파랑), 예약 인원(초록)

3. [수강률 추이 차트] (recharts LineChart)
   - X축: 월
   - Y축: 수강률 %
   - 목표선: 80% 점선 (stroke-dashed text-gray-400)

4. [요일별 분포] (recharts BarChart)
   - X축: 월~일
   - Y축: 수업 횟수
   - 가장 많은 요일 강조 (bg-primary)

데이터:
- selectedLesson: { id, name, type, instructorName }
- chartPeriod: '3M' | '6M' | '1Y'
- supabase.from('lesson_schedules')
    .select('start_at, current_count, capacity, status')
    .eq('lesson_id', selectedLesson.id)
    .gte('start_at', periodStart)
- 클라이언트에서 월별/요일별 집계

인터랙션:
- 기간 셀렉터 변경 → 차트 데이터 재집계
- [닫기] 클릭 → selectedLesson = null → 차트 패널 숨김
- 다른 수업 행 클릭 → selectedLesson 교체 → 차트 갱신

사용 유틸:
- supabase from '@/lib/supabase'
- formatNumber from '@/lib/format'
- recharts: LineChart, BarChart, XAxis, YAxis, Tooltip, Legend, ResponsiveContainer
- lucide-react: X, TrendingUp
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- 수업별 통계 테이블에서 특정 수업 행 클릭

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 수업 이력 | `lesson_schedules` | `lesson_id = ?`, 기간 범위 |

### 인터랙션 (User Actions)
1. 기간 탭 변경 (3M/6M/1Y) → 차트 데이터 재집계
2. 다른 수업 행 클릭 → 차트 수업 교체
3. [닫기] → 차트 패널 숨김

### 비즈니스 룰
- recharts 미설치 시 대체: CSS 바 차트 직접 구현
- 데이터 3개월 미만 시: 있는 데이터만 표시

### 에지 케이스
- 선택 수업 데이터 없음: "해당 기간에 수업 이력이 없습니다."

### 접근성 (A11y)
- 차트 `role="img"` + `aria-label="{수업명} 수강률 추이 차트"`

### 연결 화면
- 이전: `02-이번달` / `03-지난달`
- 다음: 닫기 → 이전 탭 상태
