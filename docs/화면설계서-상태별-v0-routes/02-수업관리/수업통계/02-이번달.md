## 📝 디스크립션

### 사용 시점 (Trigger)
- 로딩 완료 후 기본 진입 상태
- [이번달] 탭 클릭

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 수업 일정 | `lesson_schedules` | `branch_id = ?`, 이번달 범위 |
| 예약 인원 | `lesson_bookings` | `schedule_id IN (이번달 schedules)` |
| 강사 정보 | `staff` | JOIN lesson_schedules |

### 인터랙션 (User Actions)
1. [지난달] 탭 → `03-지난달` 전환
2. 수업 유형 Select 변경 → 클라이언트 필터링
3. 수업 행 클릭 → `04-차트렌더` (월별 추이 차트)
4. 강사 행 클릭 → 강사 상세 드로어 (향후)

### 비즈니스 룰
- 수강률 = 총 출석 / 총 예약 × 100 (소수점 1자리)
- 완료율 = 완료 수업 / 총 수업 × 100
- 이번달 = 1일~말일 (한국 시간대 기준)

### 에지 케이스
- 이번달 수업 0건: StatCard 모두 0 표시 + 테이블 빈상태
- 수업 유형 필터 후 결과 없음: "해당 유형의 수업이 없습니다."

### 접근성 (A11y)
- 탭 버튼 `role="tab"` + `aria-selected`
- 테이블 `caption="이번달 수업 통계"`
