## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-C001-B 일정 캘린더 — 상태: 주뷰 (timeGridWeek)

파일: src/app/calendar/page.tsx

FullCalendar 설정:
- initialView="timeGridWeek"
- slotMinTime: "06:00:00", slotMaxTime: "22:00:00"
- slotDuration: "00:30:00"
- nowIndicator: true
- editable: hasPermission(user, 'class:edit') && !isPastEvent
- eventResizableFromStart: false

이벤트 블록 (eventContent 커스텀):
- 컨테이너: 좌측 3px border (강사/카테고리 색상), rounded-sm, p-1
- 수업명 (text-[12px] font-semibold, truncate)
- 분류 태그 (text-[10px] bg-opacity-20 rounded px-1)
- "{currentCount}/{capacity}명" (text-[11px])
- 강사명 (text-[11px] text-content-secondary)
- 장소 (text-[11px])
- 미승인: opacity-70 + "미승인" amber 배지 (text-[9px])

드래그앤드롭 (eventDrop):
- 권한 체크 → 과거/시작임박 체크 → 강사 충돌 체크 → `06-충돌경고` or DB 업데이트
- supabase.from('lesson_schedules').update({start_at, end_at}).eq('id', eventId)
- 성공: toast.success('일정이 이동되었습니다.')
- 실패: info.revert() + toast.error

리사이즈 (eventResize):
- supabase.from('lesson_schedules').update({end_at}).eq('id', eventId)
- 성공: toast.success('시간이 변경되었습니다.')

빈 시간 셀 클릭 (dateClick):
- hasPermission → `05-추가모달` (date+time 사전 입력)

사용 유틸:
- hasPermission from '@/lib/permissions'
- supabase from '@/lib/supabase'
- toast from 'sonner'
- @fullcalendar/react, @fullcalendar/timegrid, @fullcalendar/interaction
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- 툴바 "주" 버튼 클릭

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 주간 이벤트 | `lesson_schedules` JOIN `lessons`, `staff` | 주 범위 |
| 드래그 업데이트 | `lesson_schedules` | id → start_at, end_at |

### 인터랙션 (User Actions)
1. 이벤트 드래그앤드롭 → `04-드래그이동` → DB 업데이트
2. 강사 충돌 시 → `06-충돌경고`
3. 이벤트 리사이즈 → DB 업데이트
4. 빈 시간 셀 클릭 → `05-추가모달`
5. 이벤트 클릭 → EventDetailModal

### 비즈니스 룰
- 드래그 제한: 과거 날짜, 이미 시작, 시작 2시간 이내 → revert
- 트레이너: 본인 수업만 드래그 가능 (editable per event)

### 에지 케이스
- 드래그 중 네트워크 에러 → revert() + toast.error
- 동시간대 이벤트 다수 → FullCalendar 자동 컬럼 분할

### 접근성 (A11y)
- 드래그 이벤트 `aria-grabbed` + `aria-dropeffect`

### 연결 화면
- 이전: `02-월뷰`
- 다음: `04-드래그이동` / `05-추가모달` / `06-충돌경고`
