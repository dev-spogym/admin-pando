## 📝 디스크립션

### 사용 시점 (Trigger)
- 사이드바 > 수업관리 > 수업/캘린더 클릭
- 월/주/일 뷰 전환 시 데이터 재fetch 중
- 수업 등록/수정/삭제 후 캘린더 리프레시 중

### 필수 데이터
| 테이블 | 쿼리 목적 |
|--------|-----------|
| `lesson_schedules` | 캘린더 이벤트 |
| `staff` | 강사 필터 목록 |
| `lessons` | 수업명 필터 목록 |
| `schedule_requests` | 미승인 건수 배지 |

### 인터랙션 (User Actions)
1. 로드 완료 → `02-월뷰` 자동 전환
2. 로드 실패 → toast.error('수업 일정을 불러오지 못했습니다.')

### 비즈니스 룰
- branchId: useAuthStore().user.branchId 기준
- 트레이너: instructor_id = staffId 조건 추가

### 에지 케이스
- 로딩 중 TabNav 클릭: 현재 fetch 취소 후 해당 탭 이동

### 접근성 (A11y)
- `role="status"` + `aria-label="캘린더 로딩 중"`

### 연결 화면
- 이전: 사이드바
- 다음: `02-월뷰` / 에러 토스트
