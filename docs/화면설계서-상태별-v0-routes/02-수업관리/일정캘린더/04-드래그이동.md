## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-C001-B 일정 캘린더 — 상태: 드래그 이동 중 및 완료

파일: src/app/calendar/page.tsx

드래그 이동 UX:
- 드래그 중: 이벤트 블록 opacity-50, 드롭 대상 슬롯 bg-primary/10 하이라이트
- 드롭 후 낙관적 UI: 이벤트 즉시 새 위치로 이동 표시

eventDrop 콜백 전체 로직:
1. 권한 체크: !hasPermission(user, 'class:edit') → info.revert() + toast.warning('편집 권한이 없습니다.')
2. 과거 날짜 체크: newStart < now → info.revert() + toast.warning('과거 일정은 수정할 수 없습니다.')
3. 시작 임박 체크: newStart - now < 2시간 → info.revert() + toast.warning('수업 시작 2시간 전까지만 수정할 수 있습니다.')
4. 강사 충돌 체크: 동일 강사 동일 시간대 다른 수업 있는지 확인 → conflictCount > 0 → `06-충돌경고` (confirm 후 진행 또는 revert)
5. DB 업데이트:
   - supabase.from('lesson_schedules').update({ start_at: newStart.toISOString(), end_at: newEnd.toISOString() }).eq('id', eventId)
   - 성공: toast.success('일정이 이동되었습니다.')
   - 실패: info.revert() + toast.error('일정 이동에 실패했습니다.')

충돌 감지 쿼리:
- supabase.from('lesson_schedules').select('id, lessons(name)').eq('instructor_id', instructorId).neq('id', eventId).lt('start_at', newEnd.toISOString()).gt('end_at', newStart.toISOString()).eq('branch_id', branchId)

리사이즈 완료 (eventResize):
- 동일 체크 순서 (과거/임박/충돌)
- supabase.from('lesson_schedules').update({ end_at: newEnd.toISOString() }).eq('id', eventId)
- 성공: toast.success('시간이 변경되었습니다.')

사용 유틸:
- hasPermission from '@/lib/permissions'
- supabase from '@/lib/supabase'
- toast from 'sonner'
- lucide-react: AlertTriangle
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- 주뷰/일뷰에서 이벤트 블록 드래그 시작 ~ 드롭 완료

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 충돌 체크 | `lesson_schedules` | `instructor_id = ?`, 시간 범위 겹침 |
| 업데이트 | `lesson_schedules` | `id = eventId` |

### 인터랙션 (User Actions)
1. 드래그 드롭 → 권한/과거/임박 체크 → (충돌 없음) DB 업데이트
2. 충돌 감지 → `06-충돌경고` confirm
3. 권한/과거/임박 실패 → revert + toast.warning

### 비즈니스 룰
- 트레이너: 본인 수업(instructorId = staffId)만 드래그 가능
- 낙관적 UI: 드롭 즉시 UI 반영, DB 실패 시 revert

### 에지 케이스
- DB timeout: revert + toast.error
- 드래그 중 다른 사용자가 같은 슬롯 선점: 충돌 체크에서 감지

### 접근성 (A11y)
- 드래그 완료 후 `aria-live="polite"` 영역에 "일정이 이동되었습니다." 알림

### 연결 화면
- 이전: `03-주뷰`
- 다음: 성공 → `03-주뷰` / 충돌 → `06-충돌경고`
