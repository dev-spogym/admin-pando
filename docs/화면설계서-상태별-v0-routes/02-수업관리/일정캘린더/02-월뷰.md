## 📝 디스크립션

### 사용 시점 (Trigger)
- 로딩 완료 후 기본 진입 상태 (dayGridMonth)
- 툴바 "월" 버튼 클릭

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 월간 이벤트 | `lesson_schedules` JOIN `lessons`, `staff` | `branch_id = ?`, 월 범위 |
| 미승인 건수 | `schedule_requests` | `status = 'pending'` |
| 강사 목록 | `staff` | `branch_id = ?` |
| 수업 목록 | `lessons` | `branch_id = ?`, `is_active = true` |

### 인터랙션 (User Actions)
1. "주" → `03-주뷰` / "일" → 일뷰
2. 날짜 셀 클릭 → 일뷰 전환
3. 이벤트 클릭 → EventDetailModal
4. [수업 등록] / 빈 셀 클릭 → `05-추가모달`
5. 강사 필터 → 필터 상태
6. [처리하기] → moveToPage('/schedule-requests')

### 비즈니스 룰
- 카테고리 색상: 방문=#3B82F6, OT=#10B981, 상담=#F59E0B, 체성분=#8B5CF6, 수업=#EF4444, PT=#F97316, 기타=#6B7280
- 강사별 색상: TRAINER_COLORS (ID 기반 5색 + 기본)
- 수업 편집 불가 조건: 과거 / 이미 시작 / 시작 2시간 이내

### 에지 케이스
- 당월 수업 0건: 빈 캘린더 (에러 아님)

### 접근성 (A11y)
- 캘린더 이벤트 `aria-label="{수업명} {시간} {강사명}"`

### 연결 화면
