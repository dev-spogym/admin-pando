## 📝 디스크립션

### 사용 시점 (Trigger)
- 로딩 완료 후 staff(trainer) 1명 이상

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 강사 목록 | `staff` | `branch_id = ?`, `role = 'trainer'`, `is_active = true` |
| 수업 횟수 | `lesson_schedules` | `instructor_id IN (...)`, 기간 범위 |
| 예약 인원 | `lesson_bookings` | `schedule_id IN (...)` |

### 인터랙션 (User Actions)
1. 기간 Select 변경 → `04-필터적용`
2. 강사명 검색 → 즉시 클라이언트 필터
3. 강사 카드/행 클릭 → `03-세부보기`
4. 그리드/테이블 뷰 전환 버튼 → 레이아웃 전환

### 비즈니스 룰
- 수강률 = 출석 / 예약 × 100
- 완료율 = 완료 수업 / 전체 수업 × 100

### 에지 케이스
- 강사 검색 결과 없음: "검색 결과가 없습니다." 인라인

### 접근성 (A11y)
- 강사 카드 `role="button"` + `aria-label="{강사명} 상세 보기"`

### 연결 화면
- 이전: `01-로딩`
- 다음: `03-세부보기` / `04-필터적용`
