## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-C006 강사 현황 — 상태: 기간 필터 적용

파일: src/app/(classes)/instructor-status/page.tsx

기간 필터 UI:
- 기간 Select 강조: border-primary bg-primary/5
- 선택 기간 라벨: "지난달" / "최근 3개월" 표시
- 우측 "초기화" 링크 버튼 (text-sm text-primary)

필터 기간 옵션:
- "이번달": startOfMonth(now) ~ endOfMonth(now)
- "지난달": startOfMonth(subMonths(now,1)) ~ endOfMonth(subMonths(now,1))
- "최근 3개월": startOfMonth(subMonths(now,2)) ~ endOfMonth(now)

재fetch 로직:
- 기간 Select onChange → setPeriod(value) → useEffect([period]) → fetchInstructorStats()
- fetchInstructorStats: supabase.from('lesson_schedules').select('instructor_id, count').gte('start_at', periodStart).lte('start_at', periodEnd)

StatCard 업데이트:
- 총 수업, 총 예약, 평균 수업 횟수 → 선택 기간 기준으로 재계산
- 기간 표시 라벨: "이번달" → "지난달" / "최근 3개월" 로 변경

강사 카드:
- 기간 내 수업 횟수 표시
- 비교 기준 변경: "이번달 대비" → 선택 기간에 따라 문구 변경

사용 유틸:
- useAuthStore from '@/stores/authStore'
- supabase from '@/lib/supabase'
- formatNumber from '@/lib/format'
- date-fns: startOfMonth, endOfMonth, subMonths
- lucide-react: Filter, X
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- 기간 Select에서 "지난달" 또는 "최근 3개월" 선택

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 필터 기간 수업 | `lesson_schedules` | `branch_id = ?`, 선택 기간 범위 |

### 인터랙션 (User Actions)
1. 기간 Select 변경 → 서버 재fetch → StatCard + 카드 업데이트
2. "초기화" 클릭 → 이번달로 복귀 → `02-기본-목록`
3. 강사 카드 클릭 → `03-세부보기`

### 비즈니스 룰
- 필터 기간이 길수록 로딩 시간 증가 → 미니 스피너 표시
- "최근 3개월": StatCard 수치는 3개월 합산

### 에지 케이스
- 선택 기간에 수업 없는 강사: 카드에 0회 표시 (제외 안 함)

### 접근성 (A11y)
- 기간 Select `aria-label="조회 기간 선택"`

### 연결 화면
- 이전: `02-기본-목록`
- 다음: `02-기본-목록` (초기화) / `03-세부보기`
