```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-C006 강사 현황 — 상태: 특정 강사 세부 보기 (우측 드로어 또는 하단 패널)

파일: src/app/(classes)/instructor-status/page.tsx

세부보기 UI 방식: 우측 슬라이드인 드로어 (w-96, h-full, fixed right-0 top-0)
- 배경 오버레이: bg-black/30

드로어 구조:
1. [헤더] p-6 border-b
   - 아바타(이니셜 원형, w-14 h-14, bg-primary/10 text-primary text-xl) + 강사명(text-xl font-bold) + 역할배지
   - X 버튼 (우측 상단)
   - 연락처 + 입사일 (text-sm text-gray-500)

2. [이번달 요약] p-6 border-b grid grid-cols-3 gap-4
   - 수업 횟수: N회 (font-bold text-lg)
   - 총 예약: N명
   - 수강률: N%

3. [수업 목록] p-6 border-b
   - 소제목: "담당 수업"
   - 각 수업 칩: 수업명 배지 (bg-primary/10 text-primary rounded-full px-3 py-1 text-sm)

4. [일별 수업 스케줄] p-6 border-b
   - 이번달 날짜별 수업 수 (미니 캘린더 또는 리스트)
   - 각 행: 날짜 + 수업명 + 시간 + 예약인원/정원

5. [월별 추이 차트] p-6
   - 최근 6개월 수업 횟수 바 차트 (recharts BarChart, 소형)
   - X: 월, Y: 수업 횟수

데이터:
- selectedInstructor: InstructorStat (이미 메모리에 있음)
- 상세 스케줄: supabase.from('lesson_schedules').select('*, lessons(name), lesson_bookings(count)').eq('instructor_id', selectedInstructor.id).gte('start_at', monthStart).order('start_at')
- 월별 추이: 최근 6개월 lesson_schedules GROUP BY month

인터랙션:
- X 버튼 / 오버레이 클릭 → 드로어 닫기 → `02-기본-목록`
- 스케줄 행 클릭 → /calendar 로 이동 (해당 날짜 일뷰)

사용 유틸:
- supabase from '@/lib/supabase'
- formatDate from '@/lib/format'
- lucide-react: X, Phone, Calendar, Clock
- recharts: BarChart, Bar, XAxis, YAxis, Tooltip, ResponsiveContainer
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- `02-기본-목록`에서 강사 카드 또는 테이블 행 클릭

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 강사 스케줄 | `lesson_schedules` | `instructor_id = ?`, 이번달 |
| 월별 추이 | `lesson_schedules` | `instructor_id = ?`, 최근 6개월 |

### 인터랙션 (User Actions)
1. X / 오버레이 클릭 → 드로어 닫기
2. 스케줄 행 클릭 → `/calendar` 해당 날짜 이동

### 비즈니스 룰
- 드로어 열린 상태에서 다른 강사 카드 클릭 → 드로어 내용 교체
