## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-C009 일정 요청 — 상태: 승인 처리

파일: src/app/schedule-requests/page.tsx

승인 처리 방식: 인라인 즉시 처리 (별도 모달 없음) + 낙관적 UI

승인 버튼 클릭 시 동작:
1. [승인] 버튼 → isApproving[requestId] = true → 버튼 Loader2 스피너
2. requestType별 처리:
   - '일정추가': supabase.from('lesson_schedules').insert(requestData)
   - '일정수정': supabase.from('lesson_schedules').update(requestData).eq('id', originalData.scheduleId)
   - '일정삭제': supabase.from('lesson_schedules').delete().eq('id', originalData.scheduleId)
   - '일정취소': supabase.from('lesson_schedules').update({status:'취소'}).eq('id', originalData.scheduleId)
3. supabase.from('schedule_requests').update({ status: 'approved', processed_by: staffId, processed_at: now }).eq('id', requestId)
4. 성공:
   - 해당 행 배경 bg-green-50 + CheckCircle 아이콘 플래시 (500ms)
   - toast.success('일정이 승인되었습니다.')
   - 목록에서 해당 행 제거 (낙관적 UI)
   - pendingCount 배지 감소
5. 실패: toast.error('승인 처리에 실패했습니다.') + 버튼 복원

일괄 승인:
- [일괄 승인] 클릭 → Promise.all(selectedRequests.map(approve))
- 완료: toast.success(`${successCount}건이 승인되었습니다.`) + 재fetch

데이터:
- isApproving: Record<string, boolean>
- requestData: JSON (lesson_schedules 업데이트 데이터)

사용 유틸:
- useAuthStore from '@/stores/authStore'
- supabase from '@/lib/supabase'
- toast from 'sonner'
- lucide-react: Check, Loader2, CheckCircle
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- `02-대기탭`에서 개별 [승인] 버튼 클릭
- `02-대기탭`에서 [일괄 승인] 클릭

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 요청 업데이트 | `schedule_requests` | `id = requestId` → `status = 'approved'` |
| 일정 처리 | `lesson_schedules` | requestType에 따라 INSERT/UPDATE/DELETE |

### 인터랙션 (User Actions)
1. [승인] → 즉시 처리 → 행 제거 + toast.success
2. [일괄 승인] → Promise.all → 완료 toast + 재fetch

### 비즈니스 룰
- 승인 시 lesson_schedules 자동 처리 (requestType별)
- processed_by: 현재 로그인 스태프 ID 기록
- 처리 후 요청자(트레이너)에게 알림 발송 (향후)

### 에지 케이스
- lesson_schedule 처리 실패 시 schedule_requests 업데이트도 롤백
- 이미 처리된 요청 재승인: "이미 처리된 요청입니다." toast.warning

### 접근성 (A11y)
- 처리 중 버튼 `aria-busy="true"` + `aria-disabled="true"`
- 완료 행 `aria-label="승인 완료"` flash

### 연결 화면
- 이전: `02-대기탭`
- 다음: `02-대기탭` (재fetch)
