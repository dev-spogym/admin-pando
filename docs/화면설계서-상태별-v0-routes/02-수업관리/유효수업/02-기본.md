## 📝 디스크립션

### 사용 시점 (Trigger)
- 로딩 완료 후 is_active=true 수업 1건 이상

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 활성 수업 | `lessons` | `branch_id = ?`, `is_active = true` |
| 예정 일정 수 | `lesson_schedules` | `lesson_id IN (...)`, `start_at >= now()`, count |
| 강사명 | `staff` | JOIN lessons.instructor_id |

### 인터랙션 (User Actions)
1. 카드 클릭 → `/calendar?lesson={id}` (캘린더 수업 필터 적용)
2. [일정 보기] → 동일
3. 유형 필터 → 즉시 클라이언트 필터
4. 검색 → 즉시 클라이언트 필터

### 비즈니스 룰
- is_active=true 수업만 표시 (비활성 수업 제외)
- 예정 일정 0회: "예정 일정 없음" 표시 (카드 노출 유지)

### 에지 케이스
- 검색 결과 없음: "검색 결과가 없습니다." 인라인 메시지
- 강사 미배정 수업: "-" 표시

### 접근성 (A11y)
- 카드 `role="button"` + `aria-label="{수업명} 일정 보기"`
- 색상 dot `aria-hidden="true"`

### 연결 화면
- 이전: `01-로딩`
- 다음: `/calendar?lesson={id}` (캘린더로 이동)
