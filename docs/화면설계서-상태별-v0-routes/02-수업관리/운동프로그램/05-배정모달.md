## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-C010 운동 프로그램 — 상태: 회원 배정 모달

파일: src/app/(classes)/exercise-programs/page.tsx

모달 크기: max-w-md (448px)

모달 구조:
1. [헤더]
   - "회원 배정 — {programName}" (font-bold text-lg) + X 버튼

2. [프로그램 요약] bg-gray-50 rounded-lg p-3 mb-4
   - 담당 강사 / 난이도 / 기간 인라인 표시

3. [회원 검색] mb-4
   - input placeholder="회원명 또는 연락처 검색"
   - 검색어 입력 시 debounce(300ms) → supabase.from('members').select('id, name, phone').ilike('name', `%${query}%`).eq('branch_id', branchId)
   - 검색 결과 드롭다운 (max-h-48 overflow-y-auto)
     · 각 행: 회원명 + 연락처 + [선택] 버튼
     · 이미 배정된 회원: [이미 배정됨] 비활성 배지

4. [배정 예정 목록] mt-4
   - 선택된 회원 칩 (이름 + x 제거)
   - 빈 상태: "배정할 회원을 검색하여 선택하세요."

5. [배정 시작일] date input (기본: 오늘)

6. [하단 버튼]
   - [취소] variant=ghost
   - [배정 완료] variant=primary (선택된 회원 없으면 disabled)

배정 로직:
- selectedMembers.forEach(member =>
    supabase.from('member_exercise_programs').insert({ program_id, member_id: member.id, branch_id, start_date, assigned_by: staffId })
  )
- Promise.all 로 병렬 INSERT
- 성공: toast.success(`${selectedMembers.length}명에게 프로그램을 배정했습니다.`) + 모달 닫기 + 재fetch
- 실패: toast.error('배정에 실패했습니다.')

사용 유틸:
- useAuthStore from '@/stores/authStore'
- supabase from '@/lib/supabase'
- toast from 'sonner'
- lucide-react: X, Search, UserPlus, Check
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- `02-기본` 테이블 [배정] 버튼 클릭
- `04-편집모달`에서 [+ 추가 배정] 클릭

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 회원 검색 | `members` | `branch_id = ?`, `name ILIKE %query%` |
| 기존 배정 목록 | `member_exercise_programs` | `program_id = ?` (중복 방지용) |

### 인터랙션 (User Actions)
1. 회원명 검색 → debounce 후 드롭다운 표시
2. 드롭다운 [선택] → 배정 예정 목록에 추가
3. 배정 예정 칩 X → 목록에서 제거
4. [배정 완료] → Promise.all INSERT → toast.success
5. [취소] / ESC → 모달 닫기

### 비즈니스 룰
- 같은 회원에게 동일 프로그램 중복 배정 불가 (unique constraint)
- 배정 시작일 기본값: 오늘 날짜
- 한 번에 최대 20명까지 배정 가능

### 에지 케이스
- 검색 결과 없음: "검색 결과가 없습니다." 드롭다운
- 이미 배정된 회원 선택 시: "[이미 배정됨]" 배지로 선택 불가

### 접근성 (A11y)
- `role="dialog"` + `aria-labelledby`
- 검색 input `aria-autocomplete="list"` + `aria-controls="search-results"`
- 드롭다운 `role="listbox"`

### 연결 화면
- 이전: `02-기본` / `04-편집모달`
- 다음: 배정 완료/취소 → 이전 상태
