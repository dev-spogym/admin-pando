## 📝 디스크립션

### 사용 시점 (Trigger)
- 잔여 횟수 필터 Select에서 "5회 이하" 선택
- 목록 정렬 상 잔여 위험 항목이 상단에 있는 상태

### 필수 데이터
- `lesson_counts` 이미 fetch된 데이터에서 클라이언트 필터링

### 인터랙션 (User Actions)
1. [+ 횟수 추가] 인라인 버튼 → `05-추가모달` (해당 회원 사전 입력)
2. [전체보기] → 필터 해제 → `02-기본-목록`
3. [일괄 안내 메시지 발송] → 메시지 발송 confirm 모달

### 비즈니스 룰
- 0회 회원: 차감 버튼 disabled + "소진" 배지
- 만료일 지난 + 잔여 있음: "만료" 배지 (횟수 사용 불가)

### 에지 케이스
- 잔여 위험 0명: 경고 배너 숨김, 일반 목록 표시

### 접근성 (A11y)
- 경고 배너 `role="alert"`
- 0회 행 `aria-label="{회원명} 잔여 횟수 소진"`

### 연결 화면
- 이전: `02-기본-목록`
- 다음: `05-추가모달` / `02-기본-목록` (필터 해제)
