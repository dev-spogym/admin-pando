## 📝 디스크립션

### 사용 시점 (Trigger)
- 로딩 완료 후 lesson_counts 1건 이상

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 횟수 목록 | `lesson_counts` | `branch_id = ?` |
| 회원 정보 | `members` | JOIN lesson_counts.member_id |
| 수업 정보 | `lessons` | JOIN lesson_counts.lesson_id |
| 강사 정보 | `staff` | JOIN lesson_counts.instructor_id |

### 인터랙션 (User Actions)
1. [이력] 아이콘 → `04-차감이력모달`
2. [추가] 아이콘 / PageHeader 버튼 → `05-추가모달`
3. [차감] 아이콘 → window.confirm("1회 차감하시겠습니까?") → UPDATE remaining_count -= 1
4. 검색/필터 → 즉시 클라이언트 필터
5. 잔여 ≤5회 행: 배경 강조 (자동, 인터랙션 없음)

### 비즈니스 룰
- 잔여 횟수 0회 미만 차감 불가: "잔여 횟수가 없습니다." toast.warning
- 만료일 지난 수강권: expiryDate 빨간 표시 + 차감 불가

### 에지 케이스
- 차감 후 0회: 해당 행 잔여 횟수 빨간 bold + "0" 표시
- 검색 결과 없음: "검색 결과가 없습니다." 인라인

### 접근성 (A11y)
- 잔여 위험 행 `aria-label="잔여 횟수 {N}회 — 주의 필요"`
- [차감] 버튼 `aria-label="{회원명} 횟수 1회 차감"`
