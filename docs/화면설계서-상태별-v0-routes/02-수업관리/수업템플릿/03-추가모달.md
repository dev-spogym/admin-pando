## 📝 디스크립션

### 사용 시점 (Trigger)
- `02-기본-목록`에서 [+ 템플릿 추가] 버튼 클릭

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 강사 목록 | `staff` | `branch_id = ?`, `role = 'trainer'`, `is_active = true` |

### 인터랙션 (User Actions)
1. 필드 입력 후 [저장] → INSERT → toast.success → 목록 재fetch
2. [취소] / X / ESC → 폼 초기화 + 모달 닫기
3. 색상 팔레트 클릭 → 색상 선택

### 비즈니스 룰
- 같은 브랜치 내 동일 템플릿명 중복 허용 (DB 제약 없음)
- 정원 최솟값: 1명

### 에지 케이스
- 필수 필드 미입력 시 [저장] 비활성 또는 인라인 에러
- 강사 목록 fetch 실패 시 Select 비활성 + "강사를 불러오지 못했습니다."

### 접근성 (A11y)
- `role="dialog"` + `aria-modal="true"` + `aria-labelledby="add-template-title"`
- 포커스 트랩, ESC 닫기
- 색상 팔레트 각 버튼 `aria-label="색상 선택: {hex}"`
