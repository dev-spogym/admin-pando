```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-C002 수업 관리 — 상태: 수업 일정 완료 처리 모달

파일: src/app/(classes)/lessons/page.tsx

완료처리 모달:
- max-w-md (448px)

레이아웃:
1. [헤더] "수업 완료 처리" + X 버튼

2. [수업 정보 요약] bg-gray-50 rounded-lg p-3 mb-4
   - 수업명 / 날짜 / 시간 / 강사명

3. [출석 처리] space-y-2
   - "예약자 목록 ({count}명)" 소제목
   - 각 예약자 행:
     · 회원명 + 예약 시각
     · 출석 상태 토글: [출석] / [결석] / [지각]
     · 기본: "출석" 선택
   - 전체 출석 체크박스: "전원 출석 처리"

4. [메모] textarea placeholder="수업 메모 (선택)"

5. [하단 버튼]
   - [취소] variant=ghost
   - [완료 처리] variant=primary → UPDATE + INSERT

완료 처리 로직:
- supabase.from('lesson_schedules').update({ status: '완료' }).eq('id', scheduleId)
- 각 예약자별: supabase.from('lesson_bookings').update({ attendance_status: attendanceMap[memberId] }).eq('id', bookingId)
- 출석자 수강권 차감: supabase.from('lesson_counts').update({ remaining_count: remaining_count - 1, used_count: used_count + 1 }).eq('member_id', memberId).eq('lesson_id', lessonId)
- Promise.all 병렬 처리
- 성공: toast.success('수업이 완료 처리되었습니다.') + 닫기 + 재fetch

출석 상태:
- '출석': bg-green-100 text-green-700 (수강권 차감)
- '지각': bg-amber-100 text-amber-700 (수강권 차감)
- '결석': bg-red-100 text-red-700 (수강권 차감 없음 → 페널티 대상)

사용 유틸:
- supabase from '@/lib/supabase'
- toast from 'sonner'
- lucide-react: X, CheckCircle, XCircle, Clock
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- `02-기본` 테이블에서 완료 가능한 수업(과거 일정)의 [완료처리] 버튼 클릭
- lesson_schedules status='예약가능' AND start_at < now()

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 예약자 목록 | `lesson_bookings` JOIN `members` | `schedule_id = ?`, `status != '취소'` |
| 수강권 | `lesson_counts` | `member_id = ?`, `lesson_id = ?` |

### 인터랙션 (User Actions)
1. "전원 출석" 체크박스 → 모든 예약자 출석으로 설정
2. 개별 출석 상태 토글 → 해당 예약자만 변경
3. [완료 처리] → UPDATE + 차감 → toast.success
4. [취소] / ESC → 모달 닫기

### 비즈니스 룰
- 출석/지각: 수강권 1회 차감 + lesson_count_histories INSERT
- 결석: 차감 없음 + 페널티 자동 적용 여부 (센터 설정에 따름)
- 이미 완료된 수업: [완료처리] 버튼 숨김

### 에지 케이스
- 예약자 0명: "예약자가 없습니다." + [완료 처리] 버튼 활성 유지 (일정만 완료)
- 수강권 잔여 0회인 회원 출석: "잔여 횟수가 없지만 출석 처리됩니다." 경고

### 접근성 (A11y)
- `role="dialog"` + `aria-labelledby`
- 출석 토글 `aria-label="{회원명} 출석 상태 선택"`

### 연결 화면

- 다음: 완료/취소 → `02-기본`
