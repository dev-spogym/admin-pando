## 📝 디스크립션

### 사용 시점 (Trigger)
- 로딩 완료 후 lessons 1건 이상

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 수업 목록 | `lessons` | `branch_id = ?` |
| 일정 수 | `lesson_schedules` | `lesson_id IN (...)`, count |
| 강사 | `staff` | JOIN lessons.instructor_id |

### 인터랙션 (User Actions)
1. [+ 수업 추가] → `03-수업추가모달`
2. 행 클릭 / [편집] → `04-수업편집`
3. [일정 보기] → `/calendar` (수업 필터 적용)
4. [비활성화] → `05-취소확인`
5. [완료처리] (완료 가능 수업) → `06-완료처리`
6. 검색/필터 → 클라이언트 필터링

### 비즈니스 룰
- 트레이너: 본인 담당 수업만 편집 가능
- 비활성 수업: 새 일정 등록 불가, 기존 일정 유지
