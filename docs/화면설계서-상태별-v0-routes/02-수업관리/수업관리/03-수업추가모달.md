- supabase.from('lessons').insert({ branch_id, name, type, instructor_id, capacity, duration, color, is_active, memo })
- 성공: toast.success('수업이 추가되었습니다.') + 닫기 + 재fetch

## 📝 디스크립션

### 사용 시점 (Trigger)
- `02-기본`에서 [+ 수업 추가] 버튼 클릭

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 강사 목록 | `staff` | `branch_id = ?`, `is_active = true` |

### 인터랙션 (User Actions)
1. 수업명 입력 → 중복 체크 (blur 시)
2. [저장] → INSERT → toast.success → 목록 재fetch
3. [취소] / ESC → 모달 닫기

### 비즈니스 룰
- 수업명 중복: 같은 지점 내 동일명 불가
- 트레이너 역할: 수업 추가 권한 없음 (PageHeader 버튼 숨김)

### 에지 케이스
- 강사 목록 fetch 실패: Select 비활성 + "강사 목록을 불러오지 못했습니다."

### 접근성 (A11y)
- `role="dialog"` + `aria-labelledby`
- 수업명 중복 에러 `aria-describedby`
