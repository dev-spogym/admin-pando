## 📝 디스크립션

### 사용 시점 (Trigger)
- 로딩 완료 후 penalties 1건 이상

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 페널티 목록 | `penalties` | `branch_id = ?` |
| 회원 | `members` | JOIN |
| 수업 | `lessons` | JOIN |
| 처리자 | `staff` | JOIN |

### 인터랙션 (User Actions)
1. [면제] 버튼 → `03-면제확인`
2. [이력] 아이콘 → `04-적용이력`
3. 검색/유형/상태/기간 필터 → 즉시 클라이언트 필터

### 비즈니스 룰
- 페널티 자동 발생: 수업 완료처리 시 결석 회원에게 자동 INSERT
- 면제 가능: 상태='적용중'인 페널티만
- 페널티 점수 누적: 회원별 총 점수 → 예약 제한 트리거 (센터 설정)

### 에지 케이스
- 면제 이유 필수 입력
- 검색 결과 없음: "검색 결과가 없습니다." 인라인
