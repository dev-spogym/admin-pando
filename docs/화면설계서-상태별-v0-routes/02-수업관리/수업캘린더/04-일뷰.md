```

화면: SCR-C001 수업 캘린더 — 상태: 일뷰 (timeGridDay)

파일: src/app/(classes)/class-schedule/page.tsx

- 각 수업 카드: 시간 + 수업명 + 강사 + 인원/정원 + 상태배지

인터랙션:

    .gte('start_at', dayStart).lte('start_at', dayEnd)
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- 캘린더 툴바 "일" 버튼 클릭
- 월뷰에서 날짜 셀 클릭
- 우측 패널 날짜 네비게이션 이동

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 당일 이벤트 | `lesson_schedules` JOIN `lessons`, `staff` | `branch_id = ?`, 당일 00:00~23:59 |

### 인터랙션 (User Actions)
1. 빈 시간 셀 클릭 → 수업 등록 모달 오픈 (날짜+시간 사전 입력)
2. 이벤트 클릭 → `06-예약확인모달`
3. prev/next 버튼 → 전날/다음날 이동
4. nowIndicator 자동 표시 (당일만)

### 비즈니스 룰
- nowIndicator는 오늘 날짜 일뷰에서만 활성
- 과거 날짜 일뷰: editable=false, 빈 셀 클릭 비활성

### 에지 케이스
- 당일 수업 0건: 빈 시간 그리드 표시 (에러 아님)
- 공휴일/센터 휴무: 별도 표시 없음 (v1)

### 접근성 (A11y)
- `aria-label="YYYY년 MM월 DD일 수업 일정"` on 캘린더 컨테이너

### 연결 화면
- 이전: `02-월뷰` / `03-주뷰`
- 다음: `06-예약확인모달` / 수업 등록 모달
