## 📝 디스크립션

### 사용 시점 (Trigger)
- FullCalendar 이벤트 블록 클릭
- 우측 패널 수업 카드 클릭

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 수업 상세 | `lesson_schedules` JOIN `lessons`, `staff` | `id = scheduleId` |
| 예약자 목록 | `lesson_bookings` JOIN `members` | `schedule_id = scheduleId` |
| 좌석 정보 | `lesson_schedules` | `seat_rows`, `seat_cols`, `reserved_seats` |

### 인터랙션 (User Actions)
1. [수정] 클릭 → 수업 등록/수정 모달 오픈 (기존 데이터 사전 입력)
2. [삭제] 클릭 → window.confirm → DELETE → toast.success('수업이 삭제되었습니다.') + 모달 닫기
3. [좌석 배치도] 접기/펼치기 토글
4. 빈 좌석 클릭 → 예약 처리 (권한 있을 시)
5. ESC / X / [닫기] → 모달 닫기

### 비즈니스 룰
- 삭제 권한: `hasPermission(user, 'class:delete')` — 트레이너는 본인 수업만
- 수정 불가 조건: 과거 날짜 / 이미 시작됨 / 시작 2시간 이내 → [수정] 버튼 disabled
- 완료 상태 수업: [삭제] 버튼 숨김

### 에지 케이스
- 예약자 0명: "예약자 없음" 텍스트
- 좌석 미사용(seatRows=0): 좌석 배치도 섹션 숨김
- 미승인 이벤트: amber 배너 "아직 승인되지 않은 일정입니다." + [승인]/[거절] 버튼

### 접근성 (A11y)
- `role="dialog"` + `aria-modal="true"` + `aria-labelledby="modal-title"`
- 포커스 트랩: 모달 내 Tab 순환
- ESC 닫기

### 연결 화면
- 이전: `02-월뷰` / `03-주뷰` / `04-일뷰`
- 다음: 닫기 → 이전 뷰 / [수정] → 수업 등록/수정 모달
