## 📝 디스크립션

### 사용 시점 (Trigger)
- 로딩 완료 후 기본 진입 상태
- 캘린더 툴바에서 "월" 버튼 클릭

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 캘린더 이벤트 | `lesson_schedules` JOIN `lessons`, `staff` | `branch_id = ?`, 월 범위 |
| 강사 목록 | `staff` | `branch_id = ?`, `role = 'trainer'` |
| 수업 목록 | `lessons` | `branch_id = ?`, `is_active = true` |
| 미승인 건수 | `schedule_requests` | `status = 'pending'` |

### 인터랙션 (User Actions)
1. "월" 버튼 → dayGridMonth 뷰 (현재 상태)
2. "주" 버튼 → `03-주뷰` 전환
3. "일" 버튼 → `04-일뷰` 전환
4. 날짜 셀 클릭 → `04-일뷰` 해당 날짜로 전환
5. 이벤트 클릭 → `06-예약확인모달` 오픈
6. 강사 Select 변경 → `05-강사필터` 상태
7. [수업 등록] 버튼 → 수업 등록 모달 오픈
8. [처리하기] 클릭 → `/schedule-requests`로 이동

### 비즈니스 룰
- 트레이너 역할: 본인 수업만 수정 가능, 타 강사 수업은 조회만
- dayMaxEvents: 3 초과 시 "+N more" 팝오버
- 미승인 이벤트: opacity-70 + "미승인" 8px amber 텍스트
- 카테고리 색상 매핑: 방문=#3B82F6, OT=#10B981, 상담=#F59E0B, 체성분=#8B5CF6, 수업=#EF4444, PT=#F97316, 기타=#6B7280

### 에지 케이스
- 당월 수업 0건 → 빈 캘린더 표시 (에러 아님)
- 이벤트 겹침 → FullCalendar 자동 처리

### 접근성 (A11y)
- 캘린더 이벤트 `aria-label="{수업명} {시간} {강사명}"`
- 필터 Select `aria-label="강사 필터"`, `aria-label="수업명 필터"`
