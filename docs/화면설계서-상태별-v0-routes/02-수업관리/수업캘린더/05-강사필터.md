```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-C001 수업 캘린더 — 상태: 강사필터 적용됨

파일: src/app/(classes)/class-schedule/page.tsx

레이아웃:
- 동일 AppLayout + PageHeader
- 현재 활성 뷰(월/주/일) 위에 필터가 적용된 상태

필터 바 강조:
- 강사 Select: 특정 강사 선택됨 → border-primary bg-primary/5 강조
- 선택된 강사명이 Select에 표시됨
- 우측 "필터 초기화" 링크버튼 노출 (text-primary text-sm underline)

캘린더 필터링 로직:
- selectedInstructorId 상태 변수
- filteredEvents = events.filter(e => e.instructorId === selectedInstructorId)
- FullCalendar events={filteredEvents} 로 교체

필터 유형별 동작:
1. 강사 필터: selectedInstructorId로 이벤트 클라이언트 사이드 필터링
2. 수업명 필터: selectedLessonName으로 이벤트 클라이언트 사이드 필터링
3. 정원 상태 필터:
   - "여유": currentCount < capacity
   - "마감": currentCount >= capacity
4. 카테고리 멀티필터: selectedCategories 배열에 scheduleCategory 포함 여부

복합 필터:
- 모든 필터 AND 조건 적용
- filteredEvents = events.filter(e =>
    (!selectedInstructorId || e.instructorId === selectedInstructorId) &&
    (!selectedLessonName || e.title === selectedLessonName) &&
    (capacityFilter === '전체' || ...) &&
    (selectedCategories.length === 0 || selectedCategories.includes(e.scheduleCategory))
  )

필터 초기화:
- "필터 초기화" 클릭 → 모든 필터 상태 리셋
- selectedInstructorId = '', selectedLessonName = '', capacityFilter = '전체', selectedCategories = []

우측 패널:
- 필터된 수업만 표시
- "필터 적용됨" 배지 표시 (bg-primary/10 text-primary text-xs px-2 py-0.5 rounded-full)

사용 유틸:
- useAuthStore from '@/stores/authStore'
- supabase from '@/lib/supabase'
- lucide-react: Filter, X
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- 강사 Select에서 특정 강사 선택
- 수업명 Select에서 특정 수업 선택
- 정원 상태 ButtonGroup에서 "여유" 또는 "마감" 선택
- 카테고리 Pill 버튼 클릭 (멀티 선택)

### 필수 데이터
| 블록 | 설명 |
|------|------|
| `events` | 전체 이벤트 (이미 fetch됨, 클라이언트 필터링) |
| `staff` | 강사 목록 (Select 옵션 생성용) |
| `lessons` | 수업명 목록 (Select 옵션 생성용) |

### 인터랙션 (User Actions)
1. 강사 Select 변경 → filteredEvents 즉시 업데이트 (서버 재요청 없음)
2. 수업명 Select 변경 → filteredEvents 즉시 업데이트
3. 정원 상태 버튼 토글 → filteredEvents 즉시 업데이트
4. 카테고리 Pill 토글 → filteredEvents 즉시 업데이트
5. "필터 초기화" 클릭 → 전체 필터 리셋

### 비즈니스 룰
- 필터 적용은 클라이언트 사이드 (추가 API 호출 없음)
- 복합 필터: 모든 조건 AND 적용
- 트레이너 역할: 강사 필터가 본인으로 고정, 변경 불가

### 에지 케이스
- 필터 결과 0건: 빈 캘린더 표시 + 우측 패널 "해당 조건에 수업이 없습니다."
- 강사 탈퇴 시 필터 목록에서 제거 (is_active = false 제외)

### 접근성 (A11y)
- Select `aria-label="강사 필터"` + 현재 선택값 aria-selected
- 카테고리 Pill `aria-pressed="true/false"`

### 연결 화면
- 이전: `02-월뷰` / `03-주뷰` / `04-일뷰`
- 다음: 동일 뷰 (필터 유지) / 필터 초기화 시 전체 복귀
