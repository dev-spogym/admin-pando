```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-097 감사로그 — 상태: 필터 적용 (특정 조건으로 필터링된 결과)

파일: src/app/(admin)/audit-log/page.tsx

필터 적용 시각적 표시:
1. 활성 필터 칩 표시 (필터 바 하단):
   const activeFilters = [
     actionFilter && { key: 'action', label: `액션: ${getActionBadge(actionFilter).label}` },
     targetTypeFilter && { key: 'target', label: `대상: ${getTargetTypeLabel(targetTypeFilter)}` },
     branchFilter && { key: 'branch', label: `지점: ${branches.find(b => String(b.id) === branchFilter)?.name}` },
     userSearch && { key: 'user', label: `사용자: ${userSearch}` },
     (fromDate !== thirtyDaysAgoStr() || toDate !== todayStr()) && { key: 'date', label: `기간: ${fromDate} ~ ${toDate}` },
   ].filter(Boolean);

   // 칩 렌더링
   {activeFilters.length > 0 && (
     <div className="flex flex-wrap gap-2 mt-3 pt-3 border-t border-line">
       {activeFilters.map(filter => (
         <span key={filter.key} className="inline-flex items-center gap-1 px-2 py-1 rounded-full bg-primary/10 text-primary text-xs font-medium">
           {filter.label}
           <button onClick={() => clearFilter(filter.key)} className="hover:text-primary-dark">
             <X size={12} />
           </button>
         </span>
       ))}
       <button
         onClick={clearAllFilters}
         className="text-xs text-content-secondary hover:text-content underline"
       >
         전체 초기화
       </button>
     </div>
   )}

2. 결과 카운트 강조:
   <span className="text-sm text-content-secondary">
     <span className="font-semibold text-primary">{formatNumber(totalCount)}</span>건 검색됨
   </span>

필터 초기화 함수:
function clearFilter(key: string) {
  if (key === 'action') setActionFilter('');
  if (key === 'target') setTargetTypeFilter('');
  if (key === 'branch') setBranchFilter('');
  if (key === 'user') setUserSearch('');
  if (key === 'date') { setFromDate(thirtyDaysAgoStr()); setToDate(todayStr()); }
}

function clearAllFilters() {
  setActionFilter('');
  setTargetTypeFilter('');
  setBranchFilter('');
  setUserSearch('');
  setFromDate(thirtyDaysAgoStr());
  setToDate(todayStr());
}

필터별 결과 예시:
- actionFilter='DELETE' → 삭제 액션만 표시 (빨간 배지)
- targetTypeFilter='member' → 회원 대상 로그만
- userSearch='김관리' → userName 포함 필터링 (클라이언트)
- branchFilter='2' → 특정 지점 로그만 (슈퍼관리자)

사용 유틸:
- X from 'lucide-react'
- formatNumber from '@/lib/format'
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- actionFilter, targetTypeFilter, branchFilter, userSearch 중 하나 이상 설정
- 날짜 범위가 기본값(30일)과 다를 때

### 필수 데이터
- activeFilters: 현재 적용된 필터 목록
- totalCount: 필터 적용 후 건수
- logs: 필터링된 결과

### 인터랙션 (User Actions)
1. 칩 X 클릭 → 해당 필터 초기화 → 재조회
2. "전체 초기화" → 모든 필터 기본값으로 → 재조회
3. 추가 필터 변경 → 누적 적용 → 재조회

### 비즈니스 룰
- 사용자 검색은 클라이언트 필터 (API 파라미터 아님)
- 나머지 필터는 API 파라미터로 서버 필터링
- 필터 변경 시 currentPage 자동 1로 리셋
- 날짜 기간: toDate는 `${toDate}T23:59:59` 적용

### 에지 케이스
- 필터 적용 결과 0건 → `04-빈상태`
- 사용자 검색 후 다른 필터 변경 → userSearch 유지
- 슈퍼관리자 아닌 사용자: branchFilter 칩 표시 안 됨

### 접근성 (A11y)
- 필터 칩 X 버튼: `aria-label="{필터명} 필터 제거"`
- "전체 초기화": `aria-label="모든 필터 초기화"`

### 연결 화면
- 이전: `02-기본-목록`
- 다음: `02-기본-목록` (초기화) / `04-빈상태` (결과 없음)
