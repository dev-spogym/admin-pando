## 📝 디스크립션

### 사용 시점 (Trigger)
- 감사 로그 테이블 행 클릭
- selectedLog !== null && isDetailOpen === true

### 필수 데이터
| 항목 | 타입 | 설명 |
|------|------|------|
| selectedLog | AuditLogEntry | 클릭한 로그 전체 데이터 |
| beforeValue | object \| null | 변경 전 값 (UPDATE 시) |
| afterValue | object \| null | 변경 후 값 (UPDATE 시) |
| detail | object \| null | 부가 정보 |

### 인터랙션 (User Actions)
1. 배경 클릭 → 모달 닫기 (setIsDetailOpen(false))
2. X 버튼 → 모달 닫기
3. ESC 키 → 모달 닫기 (keydown 이벤트 추가 권장)

### 비즈니스 룰
- beforeValue/afterValue: UPDATE 액션에서만 존재
- detail JSON: LOGIN, BRANCH_SWITCH 등 부가 정보
- fromBranchId/toBranchId: BRANCH_SWITCH 액션에서 표시
- 모달 배경 클릭 시 닫힘 (stopPropagation 처리 필수)

### 에지 케이스
- beforeValue/afterValue 없음 → 해당 섹션 숨김
- JSON 데이터가 매우 큰 경우 → max-h-32 + overflow-auto 처리
- ipAddress 없음 → '-' 표시

### 접근성 (A11y)
- 모달: `role="dialog"` + `aria-modal="true"` + `aria-labelledby`
- ESC 키 닫기: `useEffect(() => { ... keydown 'Escape' ... })`
- 포커스 트랩: 모달 내부 Tab 순환

- 다음: `02-기본-목록` (닫기)
