## 📝 디스크립션

### 사용 시점 (Trigger)
- `/today-tasks` 첫 진입 (Next.js App Router 라우팅)
- localStorage useEffect 실행 전 첫 렌더 순간

### 필수 데이터
| 항목 | 초기값 | 설명 |
|------|--------|------|
| randomTasks | useMemo 즉시 계산 | 동기 함수 |
| customTasks | [] | localStorage 로드 전 |
| filter | 'all' | 전체 탭 기본값 |

### 인터랙션 (User Actions)
- 없음 (순간적 상태)

### 비즈니스 룰
- getTodayTasks()는 동기 함수 → 별도 로딩 스피너 없음
- localStorage는 useEffect로 비동기 로드 → 첫 렌더 시 customTasks=[]
- randomTasks는 dateKey+branchName+role+isSuperAdmin 기반 useMemo → 재렌더 시 안정적

### 에지 케이스
- localStorage 파싱 실패 → catch에서 removeItem → customTasks=[] 유지
- authUser 아직 로드 중 → role='MANAGER' 기본값, isSuperAdmin=false

### 접근성 (A11y)
- StatCard: `aria-label` 포함
