## 📝 디스크립션

### 사용 시점 (Trigger)
- filter 탭 전환 후 해당 상태 태스크 없음
- 모든 태스크 완료 후 filter='all' 또는 filter='대기'/'진행중'

### 필수 데이터
| 항목 | 조건 | 설명 |
|------|------|------|
| tasks | length === 0 | 현재 필터 기준 빈 결과 |
| filter | 현재 선택 탭 | 빈 상태 메시지 컨텍스트 |
| allTasks | length > 0 | StatCard 표시용 |

### 인터랙션 (User Actions)
1. TabNav "전체" 탭 → filter='all' → allTasks 전체 표시
2. TabNav "완료" 탭 → 완료된 태스크 목록 표시
3. isSuperAdmin: "업무 추가" → 새 태스크 등록

### 비즈니스 룰
- 빈 상태 텍스트: "해당 상태의 업무가 없습니다."
- StatCard / TabNav / 배정 로직 패널: 빈 상태에서도 항상 표시
- isSuperAdmin 입력 폼: 빈 상태에서도 표시 (새 업무 추가 가능)
- 모두 완료 특별 메시지: 현재 미구현 (향후 추가 권장)

### 에지 케이스
- randomTasks가 0건인 경우 (getTodayTasks 결과 없음): allTasks=customTasks 만 존재
- 다음날 접속: dateKey 변경 → randomTasks 새로 계산 → 빈 상태 해소
- localStorage의 완료 상태 커스텀 태스크: 다음날에도 완료 상태 유지

### 접근성 (A11y)
- 빈 상태 메시지: `role="status"` `aria-live="polite"`
- 완료 축하 메시지 (향후): `role="status"` `aria-live="assertive"`
