## 📝 디스크립션

### 사용 시점 (Trigger)
- monthlyTarget 설정 후 monthlyRevenue < monthlyTarget × 0.5 (50% 미달)
- revenueMoM < 0 (전월 대비 매출 하락)
- ptNoShowRate > 5% 또는 classAttendRate < 80%

### 필수 데이터
| 항목 | 조건 | 설명 |
|------|------|------|
| pctValue | < 50 | 매출 달성률 50% 미만 |
| revenueMoM | < 0 | 전월 대비 매출 하락 |
| monthlyTarget | !== null | 목표 설정 필요 |

### 인터랙션 (User Actions)
1. 경고 배너 확인 (액션 없음)
2. "목표 수정" → TargetModal → 목표 하향 조정 가능
3. "새로고침" → fetchMetrics 재실행

### 비즈니스 룰
- pctValue < 50 → 빨간 경고 배너 + bg-red-500 게이지
- pctValue 50~69 → 주황 주의 배너 + bg-yellow-400 게이지
- pctValue 70~99 → 배너 없음 + bg-blue-500 게이지
- pctValue >= 100 → `05-목표달성` 상태
- StatCard variant: 수치 기준 자동 선택 (peach=경고, mint=양호)

### 에지 케이스
- monthlyTarget null → 배너 미표시 (목표 없음)
- 월 초반 → pctValue 낮아도 정상 범위일 수 있음 (잔여 일수 고려 미구현)

### 접근성 (A11y)
- 경고 배너: `role="alert"` `aria-live="assertive"`
- 주의 배너: `role="status"` `aria-live="polite"`
