- PageHeader "새로고침" 버튼 클릭

### 필수 데이터
| 항목 | 초기값 | 설명 |
|------|--------|------|

| metrics | EMPTY_METRICS | 모든 값 0 |

- CohortAnalysis / FunnelAnalysis 별도 useEffect로 독립 로딩
