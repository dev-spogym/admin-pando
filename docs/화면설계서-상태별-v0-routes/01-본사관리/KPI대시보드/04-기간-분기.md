현재 구현 상태:

- 코호트 분석(CohortAnalysis)이 분기 수준의 장기 트렌드를 대리함
- CohortAnalysis: 최근 6개월 코호트 × 1~6개월 유지율 → 분기 단위 패턴 파악 가능

CohortAnalysis 구현:
function CohortAnalysis({ branchId }: { branchId: number }) {
  const [rows, setRows] = useState<CohortRow[]>([]);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    async function fetchCohort() {
      setLoading(true);
      const today = new Date();
      const cohorts: CohortRow[] = [];

      for (let i = 5; i >= 0; i--) {
        const cohortDate = new Date(today.getFullYear(), today.getMonth() - i, 1);
        const cohortYear = cohortDate.getFullYear();
        const cohortMonth = cohortDate.getMonth(); // 0-based
        const cohortLabel = `${cohortYear}-${String(cohortMonth+1).padStart(2,'0')}`;

        const cohortStart = new Date(cohortYear, cohortMonth, 1).toISOString();
        const cohortEnd = new Date(cohortYear, cohortMonth+1, 0, 23, 59, 59).toISOString();

        const { data: cohortMembers } = await supabase
          .from('members')
          .select('id, status, membershipExpiry, registeredAt')
          .eq('branchId', branchId)
          .is('deletedAt', null)
          .gte('registeredAt', cohortStart)
          .lte('registeredAt', cohortEnd);

        const total = cohortMembers?.length ?? 0;
        const retention: (number | null)[] = [];

        for (let m = 1; m <= 6; m++) {
          const targetMonthEnd = new Date(cohortYear, cohortMonth + m + 1, 0, 23, 59, 59);
          if (targetMonthEnd > today) { retention.push(null); continue; }
          if (total === 0) { retention.push(null); continue; }

          const targetMonthEndStr = targetMonthEnd.toISOString().slice(0, 10);
          const retained = (cohortMembers ?? []).filter((member: any) => {
            if (member.status === 'ACTIVE') return true;
            if (member.membershipExpiry && member.membershipExpiry >= targetMonthEndStr) return true;
            return false;
          }).length;
          retention.push(Math.round((retained / total) * 100));
        }
        cohorts.push({ cohortLabel, cohortStart: cohortDate, total, retention });
      }
      setRows(cohorts);
      setLoading(false);
    }
    fetchCohort();
  }, [branchId]);

  // SimpleTable 렌더: 등록코호트 / 등록수 / 1개월~6개월
  // cohortCellBg: >=90 bg-green-200, >=70 bg-green-100, >=50 bg-yellow-100, >=30 bg-orange-100, <30 bg-red-100
  // null → '-'
}

## 📝 디스크립션

### 사용 시점 (Trigger)
- `02-기본-종합` 하단 코호트 분석 섹션 렌더 시
- 6개월 코호트 × 6개월 유지율 → 분기 단위 장기 트렌드 파악

### 필수 데이터
| 항목 | 소스 | 설명 |
|------|------|------|
| rows | CohortAnalysis 내부 fetchCohort | 최근 6개월 코호트 배열 |
| branchId | props (KpiDashboard에서 전달) | 현재 지점 |

### 인터랙션 (User Actions)
1. 코호트 테이블 스크롤 (overflow-x-auto) — 가로 스크롤
2. 색상 범례 확인 (하단 색상 기준 뱃지)

### 비즈니스 룰
- 코호트 등록 월 + m개월 이후 월말이 오늘 이후 → null(미도달)
- 유지 조건: status='ACTIVE' OR membershipExpiry >= targetMonthEndStr
- 유지율 = retained / total × 100 (반올림)
- 분기별 기간 선택 UI: 미구현, 향후 Select 컴포넌트로 확장 예정

### 에지 케이스
- total = 0인 코호트 → 모든 유지율 null('-')
- supabase 쿼리 실패 → catch → total:0, retention:[null×6]로 fallback
- 이번 달 코호트 → 1개월 유지율도 null (미도달)

### 접근성 (A11y)
- 코호트 테이블: `<table>` 시맨틱 (SimpleTable 내부)
- 색상 범례: 색상 외 텍스트로도 구분 (90%+ 등 레이블)
