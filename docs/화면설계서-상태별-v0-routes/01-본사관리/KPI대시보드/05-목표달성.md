RevenueGauge 달성 UI:
function RevenueGauge({ revenue, target }: { revenue: number; target: number | null }) {
  if (target === null) return <p className="text-[12px] text-content-secondary mt-sm">목표를 설정하세요</p>;

  const pctValue = target > 0 ? Math.min(Math.round((revenue / target) * 100), 999) : 0;
  const barWidth = Math.min(pctValue, 100);

  // 달성 시: bg-green-500
  const barColor = pctValue >= 100 ? 'bg-green-500'
    : pctValue >= 70 ? 'bg-blue-500'
    : pctValue >= 50 ? 'bg-yellow-400'
    : 'bg-red-500';

  const revManwon = Math.round(revenue / 10_000);
  const targetManwon = Math.round(target / 10_000);

  return (
    <div className="mt-sm">
      <div className="flex items-center justify-between mb-[4px]">
        <span className="text-[11px] text-content-secondary">
          ₩{formatNumber(revManwon)}만 / ₩{formatNumber(targetManwon)}만 ({pctValue}%)
        </span>
        <span className={cn('text-[11px] font-semibold',
          pctValue >= 100 ? 'text-green-600' : 'text-content-secondary'
        )}>
          {pctValue >= 100 ? '목표 달성!' : `${pctValue}% 달성`}
        </span>
      </div>
      <div className="w-full h-[6px] bg-surface-secondary rounded-full overflow-hidden">
        <div className={cn('h-full rounded-full transition-all duration-500', barColor)}
          style={{ width: `${barWidth}%` }} />
      </div>
    </div>
  );
}

## 📝 디스크립션

### 사용 시점 (Trigger)
- monthlyRevenue >= monthlyTarget (목표 달성)
- RevenueGauge pctValue >= 100

### 필수 데이터
| 항목 | 조건 | 설명 |
|------|------|------|
| monthlyRevenue | >= monthlyTarget | 이번 달 실제 매출 |
| monthlyTarget | !== null | localStorage에서 로드된 목표 |
| pctValue | >= 100 | 달성률 |

### 인터랙션 (User Actions)
1. 달성 배너 확인 → 추가 액션 없음
2. "목표 수정" → TargetModal → 새 목표로 변경

### 비즈니스 룰
- pctValue = Math.min(Math.round((revenue/target)*100), 999) — 최대 999% cap
- barWidth = Math.min(pctValue, 100) — 바는 100%로 고정
- 달성 표시: "목표 달성!" 텍스트 (green-600)
- 달성 배너: 선택적 컴포넌트 (구현 여부 팀 판단)

### 에지 케이스
- pctValue 999%: 초과 달성 표시 ("999% 달성") — 바는 100% 고정
- monthlyTarget 0: pctValue = 0 → 목표 설정 유도

### 접근성 (A11y)
- 달성 배너: `role="status"` `aria-live="polite"`
- 게이지 바: `aria-valuenow={pctValue}` `aria-valuemax="100"`
