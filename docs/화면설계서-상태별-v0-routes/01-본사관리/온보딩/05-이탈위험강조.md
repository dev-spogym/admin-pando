  const daysSince = Math.floor((today.getTime() - new Date(m.registeredAt).getTime()) / (1000*60*60*24));
  const visits = attendanceMap[m.id] ?? 0;
  let onboardStatus = '정상';
  if (daysSince <= 7 && visits === 0) onboardStatus = '미방문';
  else if (daysSince <= 7 && visits < 2) onboardStatus = '주의';
  else if (daysSince > 7 && visits < 2) onboardStatus = '위험';
  else if (daysSince > 14 && visits < 4) onboardStatus = '이탈위험';
  return { id: m.id, name: m.name, registeredAt: m.registeredAt,
    daysSinceReg: daysSince, visitCount: visits,
    hasPtTrial: ptTrialSet.has(m.id), hasGxParticipation: gxSet.has(m.id),
    status: onboardStatus };
});

## 📝 디스크립션

### 사용 시점 (Trigger)
- `04-Step3-신규안정` DataTable 렌더 직전
- atRisk.length > 0 (status='위험' 또는 '이탈위험' 회원 존재 시)

### 필수 데이터
| 항목 | 소스 | 설명 |
|------|------|------|
| atRisk | newMembers 필터 | 위험/이탈위험 회원 배열 |
| newMembers | fetchData 결과 | 최근 30일 신규 회원 전체 |

### 인터랙션 (User Actions)
1. 이탈 위험 DataTable 확인 → 별도 액션 없음 (표시 전용)
2. atRisk.length === 0 → 섹션 미표시

### 비즈니스 룰
- '위험': daysSince > 7 AND visits < 2
- '이탈위험': daysSince > 14 AND visits < 4
- 이탈 위험 섹션: 전체 DataTable 위에 별도 강조 렌더 (bg-red-50 border-red-200)
- 헤더: AlertCircle 아이콘 + "이탈 위험 회원" + 인원 수 뱃지

### 에지 케이스
- atRisk.length === 0 → return null (섹션 완전 숨김)
- 전체 신규 회원이 이탈위험 → 전체 DataTable + 이탈위험 DataTable 중복 표시 (의도적)

### 접근성 (A11y)
- 섹션 헤더: `role="alert"` (위험 상태 강조)
- AlertCircle 아이콘: `aria-hidden="true"`
- 이탈위험 DataTable: `aria-label="이탈 위험 회원 목록"`
