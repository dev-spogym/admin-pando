## 📝 디스크립션

### 사용 시점 (Trigger)
- `01-로딩` 완료 후 reports.length === 0 (DB 데이터 없음)
- typeFilter 변경 후 해당 유형 리포트 없음

### 필수 데이터
| 항목 | 조건 | 설명 |
|------|------|------|
| filteredReports | length === 0 | 빈 상태 트리거 |
| typeFilter | 현재 선택 필터 | 빈 상태 메시지 분기 |

### 인터랙션 (User Actions)
1. "리포트 생성" CTA → setShowGenerateModal(true) → `04-PDF생성중`
2. 필터 탭 전환 → 다른 유형 리포트 있으면 `02-기본-리포트목록` 전환

### 비즈니스 룰
- typeFilter='ALL' + reports=[] → "생성된 리포트가 없습니다" + 생성 유도
- typeFilter=특정유형 + 해당 리포트 없음 → "N 리포트가 없습니다" + 다른 탭 유도
- StatCardGrid: latestMonthly=undefined → 미표시
- PageHeader 버튼 ("리포트 생성", "새로고침") 항상 활성

### 에지 케이스
- generateReportList 실패 → reports=[] → 빈 상태 (에러 상태와 동일 UI)
- 모든 리포트 FAILED → filteredReports.length > 0 → DataTable 표시 (빈 상태 아님)

### 접근성 (A11y)
- 빈 상태 아이콘: `aria-hidden="true"`
- CTA 버튼: 명확한 레이블 "리포트 생성"

### 연결 화면
- 이전: `01-로딩`
- 다음: `04-PDF생성중` (생성 클릭) / `02-기본-리포트목록` (필터 전환)
