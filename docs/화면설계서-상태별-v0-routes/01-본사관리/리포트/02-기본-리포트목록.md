function formatAmount(n: number): string {
  if (n >= 100_000_000) return `${(n/100_000_000).toFixed(1)}억`;
  if (n >= 10_000) return `${Math.round(n/10_000).toLocaleString('ko-KR')}만`;
  return n.toLocaleString('ko-KR');
}
function formatDateRange(start: string, end: string): string {
  const fmt = (d: Date) => d.toLocaleDateString('ko-KR', { month: '2-digit', day: '2-digit' });
  const s = new Date(start); const e = new Date(end);
  return `${s.getFullYear()}.${fmt(s)} ~ ${fmt(e)}`;
}
function formatGeneratedAt(iso: string): string {
  return new Date(iso).toLocaleString('ko-KR', { month: '2-digit', day: '2-digit', hour: '2-digit', minute: '2-digit', hour12: false });
}

## 📝 디스크립션

### 사용 시점 (Trigger)
- `01-로딩` 완료 후 reports.length > 0

### 필수 데이터
| 항목 | 소스 | 설명 |
|------|------|------|
| reports | generateReportList | 14개 리포트 배열 |
| latestMonthly | reports.find(MONTHLY+READY) | 최신 월간 요약 카드용 |
| typeFilter | useState('ALL') | 현재 필터 |
| filteredReports | reports 필터 결과 | DataTable data |

### 인터랙션 (User Actions)
1. 필터 탭 클릭 → typeFilter 변경 → filteredReports 즉시 업데이트
2. DataTable 행 클릭 (READY) → setSelectedReport → `03-상세조회`
3. actions "보기" 버튼 → setSelectedReport → `03-상세조회`
4. actions "발송" 버튼 → toast.info('이메일 발송 기능은 준비 중입니다.')
5. "리포트 생성" → GenerateModal → `04-PDF생성중`
6. ExportButton → exportData CSV 다운로드

### 비즈니스 룰
- 리포트 순서: 일간(7개) → 주간(4개) → 월간(3개)
- GENERATING 상태: 총 매출/신규/출석 '-' 표시, 버튼 disabled
- FAILED 상태: 총 매출 '실패' 표시, 버튼 disabled
- 행 클릭: READY만 selectedReport 설정 (GENERATING/FAILED 무시)
- ExportButton: READY 상태 리포트만 내보내기

### 에지 케이스
- latestMonthly=undefined → StatCardGrid 미표시
- 모든 리포트 FAILED → 빈 테이블처럼 보임 (emptyMessage 미표시)
- typeFilter 변경 → 필터 결과 0건 → DataTable emptyMessage 표시

### 접근성 (A11y)
- 필터 탭: `role="tablist"` `aria-selected`
- actions "보기" disabled: `aria-disabled="true"`
- DataTable 행: `role="button"` (READY 행만 클릭 가능)
