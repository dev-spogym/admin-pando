## 📝 디스크립션

### 사용 시점 (Trigger)
- `01-로딩` 완료 후 branches.length > 0

### 필수 데이터
| 항목 | 소스 | 설명 |
|------|------|------|
| branches | supabase branches | 전체 지점 배열 |
| integratedStats | Promise.all 집계 | 지점별 회원/매출 |
| activeTab | useState('list') | 현재 탭 |

### 인터랙션 (User Actions)
1. "신규 지점 등록" 클릭 → `04-추가모달`
2. "지점 이동 신청" 클릭 → 지점이동 모달 열기
3. 지점명 클릭 → switchBranch() + toast.success
4. "수정" 클릭 → `05-편집모달`
5. "비활성" 클릭 → `06-비활성화확인`
6. "상태변경" Select → 임시휴업/폐점 처리 다이얼로그
7. "관리" 클릭 → moveToPage(966) 해당 지점 관리 모드
8. 탭 전환 → 통합현황 / 지점간이동 탭

### 비즈니스 룰
- 지점명 클릭 시 switchBranch(String(row.id), name) → 해당 지점 컨텍스트로 전환
- StatusBadge: active → 운영중(success), inactive → 임시휴업(warning), closed → 폐점(default)
- 폐점 지점 포함 시 amber 경고 배너 표시
- 엑셀 파일명: `지점목록.xlsx`

### 에지 케이스
- 폐점 지점: 조회 전용, 경고 배너 표시
- 회원 수 / 직원 수 null → formatNumber(0) 처리
- 탭 전환 시 animate-in fade-in duration-300

### 접근성 (A11y)
- DataTable 정렬 가능 컬럼(name): `aria-sort` 속성
- StatusBadge dot: `aria-label` 포함

- 이전: `01-로딩`
- 다음: `04-추가모달` / `05-편집모달` / `06-비활성화확인`
