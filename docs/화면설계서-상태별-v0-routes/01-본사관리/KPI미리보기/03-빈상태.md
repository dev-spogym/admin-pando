## 📝 디스크립션

### 사용 시점 (Trigger)
- 특정 보드의 queue / ranking / funnel / insights 배열이 비어있는 경우
- 현재 정적 데이터에서는 발생하지 않음 — Supabase 연동 후 대비

### 필수 데이터
| 항목 | 설명 |
|------|------|
| board.queue | length === 0 시 빈 상태 |
| board.ranking | length === 0 시 빈 상태 |
| board.funnel | length === 0 시 빈 상태 |
| board.insights | length === 0 시 빈 상태 |

### 인터랙션 (User Actions)
1. 다른 탭 선택 → 해당 보드 데이터로 전환
2. Hero 영역 / 실행 바로가기는 항상 표시

### 비즈니스 룰
- 빈 상태는 섹션별로 독립적으로 처리 (전체 페이지 빈 상태 아님)
- PageHeader actions 버튼은 빈 상태와 무관하게 항상 활성

### 에지 케이스
- boardContents 정적 데이터는 모든 보드에 데이터 존재 → 현재 빈 상태 미발생
- 향후 Supabase 연동 시 빈 배열 방어 코드 적용

### 접근성 (A11y)
- 빈 상태 아이콘: `aria-hidden="true"`
- 빈 상태 메시지: 충분한 대비 색상
