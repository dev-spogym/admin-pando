## 📝 디스크립션

### 사용 시점 (Trigger)
- `01-로딩` 완료 후 boardContents 렌더
- 탭 전환 (boardTabs 중 하나 선택) 시 동일 상태에서 board 내용만 변경

### 필수 데이터
| 항목 | 소스 | 설명 |
|------|------|------|
| activeBoard | useState(initialBoard) | 현재 선택된 보드 키 |
| board | boardContents[activeBoard] | 정적 콘텐츠 객체 |
| authUser | useAuthStore | 역할/isSuperAdmin/branchName |

### 인터랙션 (User Actions)
1. TabNav 탭 클릭 → setActiveBoard → board 콘텐츠 즉시 교체
2. PageHeader "자동알림" → moveToPage(992)
3. PageHeader "Today Tasks" → window.location.href='/today-tasks'
4. PageHeader "매출 보기" → moveToPage(970)
5. PageHeader "회원 보기" → moveToPage(967)
6. RankingPanel "상세 보기" → moveToPage(970)
7. 실행 바로가기 버튼 클릭 → 각 페이지 이동

### 비즈니스 룰
- boardContents: 정적 Mock 데이터 (5개 보드: hq/branch/fc/trainer/operations)
- 역할별 초기 탭: superAdmin→hq, fc→fc, manager/owner/primary→branch, staff→trainer
- userLabel: isSuperAdmin ? '본사 운영' : `${branchName} 운영`
- ActionQueue StatusBadge: error→'긴급', info→'우선', success→'완료가능', default→'일반'

### 에지 케이스
- 탭 전환 시 애니메이션 없음 (즉시 교체)
- branchName null → '센터' fallback

### 접근성 (A11y)
- TabNav: `role="tablist"` `aria-selected` 각 탭에 설정
- 실행 바로가기 버튼: `aria-label` 포함
