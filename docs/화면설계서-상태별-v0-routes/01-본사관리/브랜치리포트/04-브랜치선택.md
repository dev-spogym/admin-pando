파일: src/app/(admin)/branch-report/page.tsx

정렬 상태:
- sortKey: keyof BranchStat (초기: 'totalSales')
- sortAsc: boolean (초기: false — 내림차순)

정렬 토글:
const handleSort = (key: keyof BranchStat) => {
  if (sortKey === key) {
    setSortAsc(a => !a);
  } else {
    setSortKey(key);
    setSortAsc(false); // 새 컬럼은 항상 내림차순부터
  }
};

sortedStats:
const sortedStats = useMemo(() => {
  return [...branchStats].sort((a, b) => {
    const av = a[sortKey];
    const bv = b[sortKey];
    if (typeof av === 'number' && typeof bv === 'number') {
      return sortAsc ? av - bv : bv - av;
    }
    return sortAsc
      ? String(av).localeCompare(String(bv))
      : String(bv).localeCompare(String(av));
  });
}, [branchStats, sortKey, sortAsc]);

SortHeader 컴포넌트:
const SortHeader = ({ label, colKey }: { label: string; colKey: keyof BranchStat }) => (
  <button
    className="flex items-center gap-xs hover:text-primary transition-colors"
    onClick={() => handleSort(colKey)}
  >
    {label}
    <ArrowUpDown
      size={12}
      className={sortKey === colKey ? 'text-primary' : 'text-content-secondary/50'}
    />
  </button>
);

정렬 가능 컬럼: totalMembers / newMembers / activeMembers / expiredMembers / totalSales / avgSales

현재 정렬 컬럼 시각 강조:
- ArrowUpDown: sortKey 일치 시 text-primary, 불일치 시 text-content-secondary/50
- (선택) 정렬 컬럼 셀 배경: bg-primary/5

차트 데이터도 sortedStats 기준:
const salesChartData = useMemo(() =>

const memberChartData = useMemo(() =>

## 📝 디스크립션

### 사용 시점 (Trigger)
- DataTable 컬럼 헤더 정렬 버튼 클릭
- sortKey 또는 sortAsc 변경

### 필수 데이터
- sortKey: 현재 정렬 컬럼 키
- sortAsc: 오름차순/내림차순
- sortedStats: 정렬 적용된 BranchStat[]

### 인터랙션 (User Actions)
1. 컬럼 헤더 클릭 → 해당 컬럼으로 sortKey 변경 (내림차순)
2. 같은 컬럼 재클릭 → sortAsc 토글
3. 차트: sortedStats 순서로 자동 업데이트

### 비즈니스 룰
- 초기 정렬: totalSales 내림차순 (매출 높은 순)
- 매달 순위(메달)는 항상 totalSales 기준 고정 (sortKey 무관)
- 엑셀 다운로드는 sortedStats 기준

### 에지 케이스
- 동점 지점 있는 경우 → localeCompare로 이름 순 fallback
- 숫자/문자열 혼합 컬럼(attendanceRate) → String 비교

### 접근성 (A11y)
- 정렬 버튼: `aria-sort="ascending"/"descending"/"none"`
- 정렬 상태 스크린리더 공지
