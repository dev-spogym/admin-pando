## 📝 디스크립션

### 사용 시점 (Trigger)
- `01-로딩` 완료 후 branchStats.length > 0

### 필수 데이터
| 항목 | 소스 | 설명 |
|------|------|------|
| branchStats | Supabase 병렬 집계 | 지점별 통계 배열 |
| kpi | branchStats 합산 | 전체 KPI 카드용 |
| salesChartData | sortedStats 매핑 | 매출 차트 |
| memberChartData | sortedStats 매핑 | 회원수 차트 |

### 인터랙션 (User Actions)
1. 월 Select 변경 → `03-기간변경`
2. 컬럼 헤더 정렬 → sortKey/sortAsc 변경 → sortedStats 재계산
3. 엑셀 다운로드 → exportToExcel() + toast.success
4. DataTable onDownloadExcel → handleDownloadExcel()

### 비즈니스 룰
- 매달 순위: `[...branchStats].sort((a,b) => b.totalSales - a.totalSales)`
- 메달: 1위🥇 2위🥈 3위🥉 4위 이하: N위 텍스트
- 출석률: `attendanceCount / (activeMembers × daysInMonth) × 100`
- 객단가: `totalSales / totalMembers` (0명이면 0)
- 엑셀 파일명: `지점비교리포트_{selectedLabel}`

### 에지 케이스
- 지점 1개 → 바 차트 단일 바 표시 (너비 100%)
- totalSales=0인 지점 → 바 최소 너비 2%
- 출석률 계산 불가 (activeMembers=0) → '-'

### 접근성 (A11y)
- 정렬 버튼: `aria-sort` 속성 (ascending/descending/none)
- 차트: 대체 텍스트 또는 DataTable이 접근성 대안
