## 📝 디스크립션

### 사용 시점 (Trigger)
- `/branch-report` 첫 진입
- Select에서 다른 월 선택 시 (selectedMonth 변경)

### 필수 데이터
| 항목 | 초기값 | 설명 |
|------|--------|------|
| selectedMonth | 이번 달 YYYY-MM | 집계 기준 월 |
| monthOptions | 최근 12개월 | Select 옵션 목록 |

### 인터랙션 (User Actions)
1. 월 Select 변경 → selectedMonth 업데이트 → useEffect 재실행 → 로딩
2. 엑셀 다운로드 버튼: disabled (데이터 로드 중)

### 비즈니스 룰
- 지점별 순차 집계 (Promise.all로 병렬): branches 수에 따라 로딩 시간 증가
- 각 지점: 총회원/신규/활성/만료/매출/출석률 6종 쿼리

### 에지 케이스
- branches 쿼리 실패 → toast.error + isLoading=false
- 지점 0개 → 빈 상태 (branchStats=[])

### 접근성 (A11y)
- 스피너: `aria-label="데이터 불러오는 중"` `role="status"`
- 엑셀 버튼 disabled: `aria-disabled="true"`
