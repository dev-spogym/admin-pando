월 옵션 생성 (최근 12개월):
function getMonthOptions() {
  const options: { value: string; label: string }[] = [];
  const now = new Date();
  for (let i = 0; i < 12; i++) {
    const d = new Date(now.getFullYear(), now.getMonth() - i, 1);
    const value = `${d.getFullYear()}-${String(d.getMonth() + 1).padStart(2, '0')}`;
    const label = `${d.getFullYear()}년 ${d.getMonth() + 1}월`;
    options.push({ value, label });
  }
  return options;
}
// useMemo로 메모이제이션

useEffect(() => {

const [year, month] = selectedMonth.split('-').map(Number);

const endDate = new Date(year, month, 0); // 해당 월 마지막 날

## 📝 디스크립션

### 사용 시점 (Trigger)
- PageHeader actions의 월 Select에서 다른 월 선택

### 필수 데이터
| 항목 | 소스 |
|------|------|
| selectedMonth | YYYY-MM 형식 |
| monthOptions | 최근 12개월 배열 |
| startDate / endDateStr | selectedMonth 기반 계산 |

### 인터랙션 (User Actions)
1. 월 Select 변경 → setSelectedMonth → useEffect 재실행 → `01-로딩`
2. 로딩 중 추가 월 변경 → 이전 요청 결과 무시 (최신 selectedMonth 기준)

### 비즈니스 룰
- 최근 12개월만 선택 가능 (미래 월 없음)
- 월 변경 시 정렬 상태(sortKey/sortAsc) 유지
- 엑셀 파일명에 선택 월 반영: `지점비교리포트_{selectedLabel}`

### 에지 케이스
- 이번 달 → 오늘까지의 데이터만 집계 (endDateStr = 이번 달 마지막 날)
- 이전 달 → 해당 월 전체 데이터
- 지점 데이터 없는 월 → 모든 값 0 (빈 배열 아님)

### 접근성 (A11y)
- Select: `aria-label="집계 기준 월 선택"`
- 로딩 중 변경 상태: `aria-live="polite"` 로 기간 표시 업데이트 공지

### 연결 화면
- 이전: `02-기본-차트`
- 다음: `01-로딩` → `02-기본-차트` (새 월 데이터)
