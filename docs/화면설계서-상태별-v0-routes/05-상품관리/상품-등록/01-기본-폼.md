## 📝 디스크립션

### 사용 시점 (Trigger)
- 상품 목록에서 "+ 상품 등록" 버튼 클릭 또는 `/products/new` 직접 접근
- canEdit=true 권한 필수

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 분류 목록 | `product_groups` | `branchId = ?`, `isActive = true` |

### 인터랙션 (User Actions)
1. 필드 입력 시작 → `02-입력중` 상태
2. 저장 클릭 (미입력) → `05-검증에러`
3. 취소 → `/products` 이동 (변경사항 경고 없음, 미입력이므로)

### 비즈니스 룰
- 상품명, 상품타입, 카테고리, 기본가 필수
- branchId는 useAuthStore().user.branchId 자동 적용
- 현금가/카드가 미입력 시 기본가와 동일하게 저장

### 에지 케이스
- product_groups 0건 → 카테고리 Select에 "분류 없음" 표시
- 권한 없는 접근 → `/products` redirect

### 접근성 (A11y)
- 필수 필드 `aria-required="true"`
- 폼 `role="form"`, `aria-label="상품 등록 폼"`

### 연결 화면
- 이전: `/products` (목록)
- 다음: `02-입력중` / `05-검증에러` / `03-저장중` (저장 성공 시)
