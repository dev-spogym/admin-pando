## 📝 디스크립션

### 사용 시점 (Trigger)
- 폼 내 어느 필드든 값 변경 발생 시 (isDirty=true)

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 분류 목록 | `product_groups` | `branchId = ?` (이미 로드됨) |

### 인터랙션 (User Actions)
1. 필드 입력 → isDirty=true, 해당 필드 실시간 업데이트
2. 저장 버튼 → `03-저장중` (필수값 충족) / `05-검증에러` (누락)
3. 취소 버튼 → 미저장 경고 모달 → 확인 시 `/products`
4. 브라우저 이탈 → beforeunload 경고

### 비즈니스 룰
- 상품 타입에 따라 관련 필드만 표시 (동적 폼)
- 현금가 미입력 시 기본가와 동일하게 저장 처리
- 이용 가능 요일: 빈 배열이면 전체 허용으로 해석

### 에지 케이스
- 가격 필드 0 또는 음수 입력 → 검증 오류 즉시 표시
- 기간과 횟수 모두 0 → 허용 (무제한으로 해석)

### 접근성 (A11y)
- 실시간 오류: `aria-invalid="true"` + `aria-describedby="error-msg"`
- 미저장 경고 모달: focus trap, ESC 닫기

### 연결 화면
- 이전: `01-기본-폼`
- 다음: `03-저장중` / `05-검증에러` / 취소 → `/products`
