## 📝 디스크립션

### 사용 시점 (Trigger)
- 알림설정 탭 클릭 (isDirty=false 시 즉시 전환)
- 알림 항목 토글 조작 시

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 알림 설정 저장 | `settings` | `UPDATE smsEnabled, pushEnabled, autoExpireNotify WHERE branchId = ?` |

### 인터랙션 (User Actions)
1. 탭 클릭 → 알림설정 탭 표시 (isDirty=false 시 즉시)
2. 토글 조작 → notificationSettings 업데이트, isDirty=true
3. 저장하기 → `05-저장중`

### 비즈니스 룰
- 8개 알림 항목 각각 독립 ON/OFF
- SMS 발송은 SMS 포인트 잔액 필요 (포인트 0 시 토글 ON해도 미발송, 경고 없음)
- 저장 전 토글 변경은 메모리 상태에만 반영 (DB 즉시 반영 아님)

### 에지 케이스
- 알림 설정 저장 전 탭 이동 → DLG-080-001 미저장 경고
- SMS 포인트 부족 → 토글 ON 허용, 실제 발송 시 자동 스킵

### 접근성 (A11y)
- 각 Toggle `aria-label="{item.label} {ON/OFF}"`
- 카테고리 `role="group"` + `aria-labelledby`

### 연결 화면
- 이전: `01-기본-탭메뉴`
- 다음: `05-저장중`
