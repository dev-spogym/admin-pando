화면: SCR-080 설정 홈 — 상태: 기본정보 탭 편집 중 (isDirty=true)

  <div className="flex items-center gap-2 px-4 py-2 bg-amber-50 border-b border-amber-200">

  </div>

>

- 센터명: value={centerInfo.name} onChange → handleCenterChange('name', v)

- 센터 소개: Textarea value={centerInfo.description}
- 대표 연락처: value={centerInfo.phone}
- 주소: value={centerInfo.address} + [주소검색] 버튼

- 평일 운영시간: openTime / closeTime Input[time]
- 주말 운영시간: weekendOpenTime / weekendCloseTime

변경 감지:
const handleCenterChange = (key: keyof CenterInfo, value) => {
  setCenterInfo(prev => ({ ...prev, [key]: value }));
  setIsDirty(true);
};

beforeunload:
useEffect(() => {
  const handler = (e: BeforeUnloadEvent) => {
    if (isDirty) e.preventDefault();
  };
  window.addEventListener('beforeunload', handler);
  return () => window.removeEventListener('beforeunload', handler);
}, [isDirty]);

사용 유틸:
- supabase, toast
- lucide-react: AlertTriangle, CheckCircle2

## 📝 디스크립션

### 사용 시점 (Trigger)
- 기본정보 탭에서 어느 필드든 값 변경 시 (isDirty=true)

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 설정 저장 | `settings` | `UPDATE WHERE branchId = ?` |
| 지점 저장 | `branches` | `UPDATE WHERE id = branchId` |

### 인터랙션 (User Actions)
1. 필드 수정 → isDirty=true, amber 배너 표시
2. 다른 탭 클릭 → DLG-080-001 미저장 경고
3. 저장하기 → `05-저장중`
4. 브라우저 이탈 → beforeunload 네이티브 경고

### 비즈니스 룰
- 업종 체크박스: 최소 1개 필수
- 센터명 필수 (빈 값 저장 불가)
- 운영시간: openTime < closeTime 검증

### 에지 케이스
- 주소 검색 → 외부 API 호출 (카카오 또는 Daum 주소 API)
- 업종 전체 해제 → 저장 시 검증 오류

### 접근성 (A11y)
- 업종 체크박스 그룹: `role="group"` + `aria-labelledby="sector-label"`
- 미저장 경고 모달: focus trap + ESC 닫기

### 연결 화면
- 이전: `01-기본-탭메뉴`
- 다음: `05-저장중` / `01-기본-탭메뉴` (취소)
