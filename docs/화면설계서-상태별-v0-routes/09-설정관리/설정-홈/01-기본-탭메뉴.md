## 📝 디스크립션

### 사용 시점 (Trigger)
- `/settings` 진입, 데이터 로드 완료
- 기본정보 탭이 기본 활성

### 필수 데이터
| 블록 | 테이블 | 조건 |
|------|--------|------|
| 설정 | `settings` | `branchId = ?` |
| 지점 | `branches` | `id = branchId` |

### 인터랙션 (User Actions)
1. 탭 클릭 (isDirty=false) → 즉시 탭 전환
2. 탭 클릭 (isDirty=true) → 미저장 경고 다이얼로그
3. 필드 수정 → isDirty=true, amber 배지 표시
4. 저장하기 → `05-저장중`

### 비즈니스 룰
- 센터장/슈퍼관리자만 접근 (매니저 이하 → 403)
- isDirty=false → 저장 버튼 disabled (opacity-50 cursor-not-allowed)
- saveSuccess 배지 3초 후 자동 해제

### 에지 케이스
- settings 레코드 없음 → 기본값으로 초기화 후 표시
- branchId 없음 → `/` redirect

### 접근성 (A11y)
- TabNav `role="tablist"`, 각 탭 `role="tab"`, 패널 `role="tabpanel"`
- 저장 버튼 disabled 시 `aria-disabled="true"`

### 연결 화면
- 이전: 사이드바 네비게이션
- 다음: 각 탭 콘텐츠 편집 상태
