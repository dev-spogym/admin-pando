## 📝 디스크립션

### 사용 시점 (Trigger)
- "화면 설정" 탭 클릭 시

### 필수 데이터
- 없음 (localStorage 기반)

### 인터랙션 (User Actions)
1. 로고 업로드 → FileReader로 base64 → kioskSettings.logoUrl
2. 배경 색상 → color picker + hex input 동기화
3. 환영 메시지 → 50자 제한, 실시간 미리보기
4. 저장 → localStorage 저장

### 비즈니스 룰
- 로고 이미지: 최대 2MB, image/* 허용
- 배경색: hex 형식만 허용
- 환영 메시지: 최대 50자

### 에지 케이스
- 이미지 2MB 초과 → toast.error("이미지 크기는 2MB 이하여야 합니다.")
- 유효하지 않은 hex 색상 → 기본값 #FFFFFF 복원

### 접근성 (A11y)
- 색상 picker `aria-label="키오스크 배경 색상"`
- 이미지 업로드 영역 `role="button"`, `aria-label="로고 이미지 업로드"`

- 이전: `02-기본`
- 다음: `02-기본` (저장 후)
