파일: src/app/settings/kiosk/page.tsx

락커 후처리 탭 (activeTab='locker'):

[출석 후 처리 방식] — 라디오 그룹:
<div className="space-y-3">
  {[
    { value: 'checkin_only', label: '출석만', desc: '락커 배정 없이 출석만 처리합니다.' },
    { value: 'manual', label: '직원 수동 배정', desc: '출석 완료 후 인포데스크에서 락커를 배정합니다. (기본값)' },
    { value: 'auto', label: '자동 배정', desc: '출석 완료 시 빈 락커를 자동으로 배정합니다.' },
  ].map(opt => (
    <label key={opt.value} className={cn(
      "flex items-start gap-3 p-4 rounded-xl border cursor-pointer transition-colors",
      kioskSettings.afterCheckinMode === opt.value
        ? "border-primary bg-primary/5"
        : "border-gray-200 hover:border-gray-300"
    )}>
      <input type="radio" name="afterCheckinMode" value={opt.value}
        checked={kioskSettings.afterCheckinMode === opt.value}
        onChange={() => handleChange('afterCheckinMode', opt.value)}
        className="mt-0.5" />
      <div>
        <p className="font-medium text-sm">{opt.label}</p>
        <p className="text-xs text-gray-500 mt-0.5">{opt.desc}</p>
      </div>
    </label>
  ))}
</div>

[자동 배정 대상 존] — afterCheckinMode === 'auto'일 때만 표시:
<div className="mt-4 p-4 bg-gray-50 rounded-xl">
  <p className="text-sm font-medium mb-3">자동 배정 대상 존</p>
  <div className="flex flex-wrap gap-2">
    {availableZones.map(zone => (
      <button key={zone} onClick={() => toggleZone(zone)}
        className={cn("px-3 py-1.5 rounded-full text-sm border transition-colors",
          kioskSettings.autoAssignZones.includes(zone)
            ? "bg-primary text-white border-primary"
            : "bg-white text-gray-600 border-gray-300"
        )}>
        {zone} 구역
      </button>
    ))}
  </div>
</div>

[미배정 안내 문구] — manual 모드일 때:
<Textarea value={kioskSettings.lockerUnassignedMsg}
  onChange={e => handleChange('lockerUnassignedMsg', e.target.value)}
  placeholder="인포데스크에서 락커를 배정받아 주세요."
  rows={2} />

[출석 완료 메시지 미리보기]:
<div className="mt-6 p-4 bg-blue-50 border border-blue-200 rounded-xl">
  <p className="text-xs text-blue-600 font-medium mb-2">출석 완료 메시지 미리보기</p>
  <p className="text-sm text-blue-800">
    {kioskSettings.afterCheckinMode === 'auto'
      ? "{이름}님, 오늘의 락커는 A-12번입니다."
      : kioskSettings.afterCheckinMode === 'manual'
      ? `출석이 완료되었습니다. ${kioskSettings.lockerUnassignedMsg}`
      : "출석이 완료되었습니다."}
  </p>
</div>

사용 유틴:
- lucide-react: Lock, CheckCircle

## 📝 디스크립션

### 사용 시점 (Trigger)
- "락커 후처리" 탭 클릭 시

### 필수 데이터
- 없음 (localStorage 기반)

### 인터랙션 (User Actions)
1. 라디오 선택 → afterCheckinMode 변경, 메시지 미리보기 즉시 업데이트
2. 자동 배정 선택 → 존 선택 UI 표시
3. 안내 문구 입력 → 미리보기 즉시 반영
4. 저장 → localStorage 저장

### 비즈니스 룰
- 기본값: 'manual' (직원 수동 배정)
- 자동 배정: IoT 락커 컨트롤러 연동 필요
- 출석만 모드: 락커 관련 모든 처리 스킵

### 에지 케이스
- 자동 배정 선택 + IoT 기기 없음 → 경고 배너: "락커 컨트롤러가 등록되지 않았습니다."
- 자동 배정 존 미선택 → 저장 시 toast.warning

### 접근성 (A11y)
- 라디오 그룹 `role="radiogroup"` + `aria-labelledby`
- 존 선택 버튼 `aria-pressed={selected}`
