## 📝 디스크립션

### 사용 시점 (Trigger)
- `01-로딩` 완료 후 기본 설정 탭 활성 상태

### 필수 데이터
| 블록 | 소스 | 조건 |
|------|------|------|
| 키오스크 설정 | `localStorage: kiosk_settings_{branchId}` | 이미 로드됨 |

### 인터랙션 (User Actions)
1. Toggle 조작 → isDirty=true, 우측 미리보기 실시간 반영
2. 탭 전환 → `03-디스플레이설정` / `04-결제연동`
3. 저장 → localStorage 저장, toast.success
4. 연결 테스트 → 키오스크 단말 ping (IOT 연동 시)
5. 미리보기 → 전체화면 미리보기 모달

### 비즈니스 룰
- rfidEnabled + qrEnabled + phoneEnabled 중 최소 1개 ON 필수
- blockExpired=true → blockMessage 필드 표시
- 저장은 localStorage 기반 (향후 DB 마이그레이션 예정)

### 에지 케이스
- 모든 체크인 방식 OFF → 저장 시 검증 오류
- faceEnabled=true → "베타 기능 안내" toast.info

### 접근성 (A11y)
- Toggle `aria-label="{항목명} {ON/OFF}"`
- 미리보기 `aria-label="키오스크 화면 미리보기"`

### 연결 화면
- 이전: `01-로딩`
- 다음: `03-디스플레이설정` / `04-결제연동` / `05-에러`
