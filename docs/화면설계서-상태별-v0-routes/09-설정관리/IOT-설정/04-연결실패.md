## 📝 디스크립션

### 사용 시점 (Trigger)
- `03-연결중` 상태에서 ping 타임아웃(3초) 또는 오류 응답 시

### 필수 데이터
- 없음 (로컬 상태 + localStorage 장애 로그)

### 인터랙션 (User Actions)
1. 오류 행 인라인 "재시도" → `03-연결중`
2. toast "수동 재시도" 액션 → `03-연결중`
3. 무시 → `02-기본-기기목록` (오류 상태 유지)

### 비즈니스 룰
- 오류 기기: 자동 락커 배정 대상 제외
- 장애 로그 localStorage 보관 (최근 100건)
- 오류 상태 기기는 출입 이력 탭에서 배지로도 노출

### 에지 케이스
- 모든 기기 오류 → 전체 IOT 장애 배너 표시 (PageHeader 하단)
- 반복 실패 (3회 이상) → "기기 점검이 필요합니다" 추가 안내

### 접근성 (A11y)
- 오류 행 `aria-label="연결 실패"`, `role="alert"`

### 연결 화면
- 이전: `03-연결중`
- 다음: `03-연결중` (재시도) / `02-기본-기기목록`
