등록 로직:
const handleAddDevice = () => {
  const device: IotDevice = {
    id: crypto.randomUUID(),
    ...newDevice,
    status: 'disconnected',
    lastPing: new Date().toISOString(),
  };
  const updated = [...devices, device];
  setDevices(updated);
  localStorage.setItem(`iot_devices_${branchId}`, JSON.stringify(updated));
  toast.success(`'${device.name}' 기기가 추가되었습니다.`);
  setShowAddModal(false);
  setNewDevice({ name:'', type:'gate', ip:'', port:undefined, location:'' });
};

사용 유틸:
- lucide-react: Plus, Wifi
- toast from 'sonner'

## 📝 디스크립션

### 사용 시점 (Trigger)
- "기기 추가" 버튼 클릭 시

### 필수 데이터
- 없음 (localStorage 저장)

### 인터랙션 (User Actions)
1. 필드 입력 → 등록 버튼 활성화 (name + type + ip 필수)
2. 연결 테스트 → 모달 내에서 ping 시도
3. 등록 → localStorage 저장, 목록 업데이트, 모달 닫힘
4. 취소 → 모달 닫힘, 입력값 초기화

### 비즈니스 룰
- 기기명, 유형, IP 필수
- 등록 후 status='disconnected' (연결 테스트 별도 필요)
- IP 형식 검증: 간단한 regex `/^\d{1,3}(\.\d{1,3}){3}$/`

### 에지 케이스
- 중복 IP → 허용 (다른 포트 사용 가능)
- IP 형식 오류 → 인라인 오류 메시지

### 접근성 (A11y)
- 모달 `role="dialog"`, `aria-modal="true"`, focus trap
- 필수 필드 `aria-required="true"`
