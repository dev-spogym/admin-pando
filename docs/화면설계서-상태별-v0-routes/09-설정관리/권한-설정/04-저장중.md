```
저장 로직:
const handleSave = async () => {
  // 충돌 검증
  if (conflicts.length > 0) {
    setShowConflictModal(true); // → 06-충돌경고
    return;
  }

  setIsSaving(true);
  try {
    // localStorage에 현재 역할 권한 저장
    localStorage.setItem(
      `permissions_${selectedRole.code}`,
      JSON.stringify(permissions[selectedRole.code])
    );
    setSavedPermissions(prev => ({
      ...prev,
      [selectedRole.code]: { ...permissions[selectedRole.code] }
    }));
    setIsDirty(false);
    setChangedCells(new Set());
    // → 05-완료 상태
    setSaveSuccess(true);
    setTimeout(() => setSaveSuccess(false), 3000);
    toast.success(`'${selectedRole.name}' 권한이 저장되었습니다.`);
  } catch {
    toast.error('권한 저장에 실패했습니다.');
  } finally {
    setIsSaving(false);
  }
};

UI (isSaving=true):
- 저장 버튼: <Loader2 className="animate-spin w-4 h-4 mr-1" /> "저장 중..." disabled
- 초기화 버튼: disabled
- 권한 매트릭스 체크박스: pointer-events-none opacity-60
- 역할 목록: pointer-events-none

사용 유틸:
- toast from 'sonner'
- lucide-react: Loader2
```
