## 🧩 바이브코딩 프롬프트

```
Next.js 15 App Router + TypeScript + Tailwind + Supabase 기반 'use client' 컴포넌트를 작성하라.
화면: SCR-081 권한 설정 — 상태: 역할 목록 + 권한 매트릭스 로딩 중

파일: src/app/settings/permissions/page.tsx

레이아웃:
- <AppLayout> 데스크톱 레이아웃
- <PageHeader title="권한 설정" subtitle="역할별 메뉴 접근 권한을 관리합니다." />
- 좌우 2패널 스켈레톤

스켈레톤:
- 좌측 역할 목록 (w-48):
  6개 역할 카드 skeleton (h-14 bg-gray-100 animate-pulse rounded-lg, space-y-2)
- 우측 권한 매트릭스 (flex-1):
  - 상단 버튼 2개 skeleton (w-24 h-8 bg-gray-200 animate-pulse)
  - 테이블 헤더 skeleton (h-10 bg-gray-200)
  - 22행 × 5열 체크박스 skeleton (h-10 bg-gray-50 animate-pulse)

데이터 로드:
useEffect(() => {
  setIsLoading(true);
  // 6개 시스템 역할 초기화
  const systemRoles: Role[] = [
    { id:'1', code:'primary', name:'최고관리자', isSystem:true, userCount:1 },
    { id:'2', code:'owner', name:'센터장', isSystem:true, userCount:3 },
    { id:'3', code:'manager', name:'매니저', isSystem:true, userCount:5 },
    { id:'4', code:'fc', name:'피트니스 코치', isSystem:true, userCount:8 },
    { id:'5', code:'staff', name:'스태프', isSystem:true, userCount:4 },
    { id:'6', code:'readonly', name:'조회전용', isSystem:true, userCount:2 },
  ];
  // 커스텀 역할 + 권한 로드 (localStorage)
  const customRoles = JSON.parse(localStorage.getItem(`custom_roles_${branchId}`) || '[]');
  const allRoles = [...systemRoles, ...customRoles];
  setRoles(allRoles);
  setSelectedRole(allRoles[1]); // 센터장 기본 선택

  // 각 역할 권한 로드
  const perms: Record<string, PermissionMatrix> = {};
  allRoles.forEach(role => {
    const stored = localStorage.getItem(`permissions_${role.code}`);
    perms[role.code] = stored ? JSON.parse(stored) : getDefaultPermissions(role.code);
  });
  setPermissions(perms);
  setIsLoading(false);
}, [branchId]);

사용 유틸:
- useAuthStore, getBranchId
- lucide-react: Shield, Users
```

## 📝 디스크립션

### 사용 시점 (Trigger)
- `/settings/permissions` 진입 직후
- 6개 시스템 역할 + 커스텀 역할 + 각 역할 권한 매트릭스 로드 중

### 필수 데이터
| 블록 | 소스 | 조건 |
|------|------|------|
| 역할 목록 | 하드코딩 + `localStorage: custom_roles_{branchId}` | 가상 |
| 권한 매트릭스 | `localStorage: permissions_{roleCode}` | 가상 |

### 인터랙션 (User Actions)
1. 로딩 중 — 모든 인터랙션 차단
2. 로드 완료 → `02-역할별탭` (센터장 기본 선택)

### 비즈니스 룰
- 최고관리자(primary)는 모든 권한 true 고정
- 권한 저장소: localStorage (향후 Supabase role_permissions 마이그레이션 예정)

### 에지 케이스
- localStorage 없음 → getDefaultPermissions() 함수로 역할별 기본값 생성

### 접근성 (A11y)
- `aria-busy="true"`, 스켈레톤 `aria-hidden="true"`
