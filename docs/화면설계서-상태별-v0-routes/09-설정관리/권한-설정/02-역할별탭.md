## 📝 디스크립션

### 사용 시점 (Trigger)
- `01-로딩` 완료 후 센터장 역할이 기본 선택된 상태

### 필수 데이터
| 블록 | 소스 | 조건 |
|------|------|------|
| 역할 목록 | 하드코딩 + localStorage | `custom_roles_{branchId}` |
| 권한 매트릭스 | localStorage | `permissions_{roleCode}` |

### 인터랙션 (User Actions)
1. 역할 카드 클릭 → selectedRole 변경, 해당 역할 권한 매트릭스 표시
2. 체크박스 클릭 → `03-권한토글변경`
3. 모두 허용/거부 → 전체 권한 일괄 변경, isDirty=true
4. 변경 사항 저장 → `04-저장중` (충돌 검증 후)
5. 초기화 → savedPermissions로 복원
6. + 역할 추가 → DLG-081-003 모달

### 비즈니스 룰
- 최고관리자: 체크박스 클릭 불가 (cursor-default)
- isDirty=false → 저장 버튼 disabled
- 역할 전환 시 isDirty=true → 미저장 경고 모달

### 에지 케이스
- 커스텀 역할 0건 → 시스템 6역할만 표시
- 역할 전환 중 변경사항 있음 → 저장 여부 확인

### 접근성 (A11y)
- 역할 목록 `role="listbox"`, 각 역할 `role="option"`, `aria-selected`
- 권한 테이블 `role="grid"`, 체크박스 셀 `role="gridcell"`
