## 📝 디스크립션

### 사용 시점 (Trigger)
- 케이스 A: 신규 센터 / 리드 데이터 한 건도 없을 때
- 케이스 B: 필터/검색어 조합으로 결과 0건일 때
- 칸반: 특정 파이프라인 단계에 리드 없을 때

### 필수 데이터
| 조건 | 표시 |
|------|------|
| leads.length === 0 | 케이스 A 빈상태 |
| filtered.length === 0, leads.length > 0 | 케이스 B 필터 빈상태 |

### 인터랙션 (User Actions)
1. [+ 리드 등록] → `openAdd()` → `05-리드추가모달`
2. 필터 초기화 → 전체 리드 목록 복귀

### 비즈니스 룰
- 케이스 A에서는 통계 카드 모두 0 표시 (숨기지 않음)
- 상태별 분포 섹션만 숨김 (stats.total === 0 조건)
- 빈상태에서도 [+ 리드 등록] 버튼은 항상 노출

### 에지 케이스
- 모든 필터 조합 시도 후 여전히 0건 → 필터 초기화 안내 메시지 추가

### 접근성 (A11y)
- 빈상태 영역 `role="status"` + `aria-label="리드 데이터 없음"`

### 연결 화면
- 이전: `01-로딩` / `02-기본-신규탭`
- 다음: `05-리드추가모달`
