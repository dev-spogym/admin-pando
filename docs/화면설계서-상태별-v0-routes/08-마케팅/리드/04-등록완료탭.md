## 📝 디스크립션

### 사용 시점 (Trigger)
- 상태 필터 Select에서 "등록완료" 선택 시
- `filterStatus` state가 "등록완료"로 변경될 때

### 필수 데이터
| 블록 | 조건 |
|------|------|
| `filtered` | `leads.filter(l => l.status === '등록완료')` |
| 통계 | 전체 leads 기준 (필터 무관) |

### 인터랙션 (User Actions)
1. 상태 필터 "전체 상태" 재선택 → `02-기본-신규탭`
2. 수정 아이콘 클릭 → `06-담당FC배정` 수정 모달
3. 검색어 추가 입력 → 등록완료 필터 + 검색어 교집합

### 비즈니스 룰
- 등록완료 리드는 `convertedMemberId`가 연결된 경우 회원 상세로 이동 가능
- 센터장/매니저만 등록완료 리드 삭제 가능 (FC는 삭제 불가)
- 전환율 통계는 이 필터 뷰에서 쉽게 확인 가능

### 에지 케이스
- 등록완료 건 0건 → 빈 DataTable + emptyMessage
- 검색어 + 등록완료 필터 조합 결과 0건 → 빈상태

### 접근성 (A11y)
- Select `aria-label="리드 상태 필터"` aria-selected 반영
