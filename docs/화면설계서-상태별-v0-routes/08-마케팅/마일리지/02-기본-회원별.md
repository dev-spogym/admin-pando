## 📝 디스크립션

### 사용 시점 (Trigger)
- `01-로딩` 완료 후 (isLoading=false)
- 기본 탭 (activeTab="status")

### 필수 데이터
| 항목 | 소스 |
|------|------|
| members | `members` 테이블 mileage 필드 기반 가공 |
| summary | members 집계 (reduce) |

### 인터랙션 (User Actions)
1. 회원명 클릭 → 회원 상세 (moveToPage 985)
2. [수동 처리] → ManualAdjustmentModal
3. [이력] → 마일리지 이력 탭 전환
4. [엑셀 다운로드] → exportToExcel(members)

### 비즈니스 룰
- balance = earned - used (현재 used=0이므로 mileage와 동일)
- expiryDate: '-' (만료일 DB 미연동 → ExpiryBadge 안전 처리)
- lastEarnedAt: '-' (DB 미연동)

### 에지 케이스
- members 빈 배열 → 빈 상태 테이블 표시 (06-빈상태)
- expiryDate='-' → ExpiryBadge 날짜 파싱 실패 방어 필요

### 접근성 (A11y)
- 회원명 버튼: `role="link"` + `aria-label="{name} 회원 상세"`
- DataTable: `role="grid"`

### 연결 화면
- 이전: `01-로딩`
- 다음: `03-수동지급` / `05-이력보기` / `마일리지 정책 탭`
