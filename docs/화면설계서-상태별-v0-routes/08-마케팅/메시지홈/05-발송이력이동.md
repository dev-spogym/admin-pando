## 📝 디스크립션

### 사용 시점 (Trigger)
- TabNav에서 [발송 이력] 탭 클릭 시
- `setActiveTab("history")` 호출

### 필수 데이터
| 항목 | 소스 |
|------|------|
| history | `messages` 테이블 전체 (최신순) |
| fetchMessageHistory | 컴포넌트 마운트 + confirmSend 후 갱신 |

### 인터랙션 (User Actions)
1. 탭 진입 → `history` 데이터 렌더 (pageSize=10)
2. selectable=true → 행 체크박스 선택 (향후 일괄 삭제 확장)
3. [엑셀 다운로드] → `exportToExcel(history, exportColumns, {filename:'발송이력'})`
4. [메시지 전송] 탭 → `activeTab="send"` 전환

### 비즈니스 룰
- 전체 발송 이력 pageSize=10 (메시지 전송 탭의 최근 발송 이력 pageSize=5와 구분)
- status: SENT(완료) / SCHEDULED(예약) / FAILED(실패)
- recipients 배열 길이 = 수신자 수 (전체 발송 시 빈 배열 → 0명 표시)

### 에지 케이스
- history 배열 빈 경우: DataTable emptyMessage 표시
- 전체 발송(useAllMembers=true) 시 recipients=[] → 수신자 수 0명 표시 (실제 발송 수와 다름)

### 접근성 (A11y)
- DataTable selectable: 체크박스 `role="checkbox"` + `aria-label="행 선택"`

### 연결 화면
- 이전: `01-기본` (탭 전환)
- 다음: `01-기본` (메시지 전송 탭 클릭)
