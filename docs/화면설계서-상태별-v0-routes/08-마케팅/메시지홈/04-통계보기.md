## 📝 디스크립션

### 사용 시점 (Trigger)
- 메시지 발송 탭 진입 시 자동 알림 섹션 항상 표시
- 토글 클릭 시 이 상태로 전환 (UI 인터랙션)

### 필수 데이터
| 항목 | 소스 |
|------|------|
| autoAlarms | 로컬 state (기본값 하드코딩) |
| 실제 저장 | `/message/auto-alarm` 페이지 localStorage |

### 인터랙션 (User Actions)
1. 토글 클릭 → `autoAlarms` 즉시 업데이트 (미리보기용)
2. [상세 설정 →] → `/message/auto-alarm` 페이지 이동
3. 상세 설정 페이지에서 실제 localStorage 저장 처리

### 비즈니스 룰
- 이 패널의 토글은 미리보기 용도 (실제 DB/localStorage 저장은 auto-alarm 페이지에서)
- 활성화된 토글 = 해당 이벤트 발생 시 자동 발송 예약됨
- 13종 중 9종만 이 패널에 표시 (나머지는 auto-alarm 페이지에서 확인)

### 에지 케이스
- 이 패널 토글 변경 후 페이지 이동 시 변경 내용 소실 (저장 안 됨)
- auto-alarm 페이지에서 설정된 값이 실제 적용값

### 접근성 (A11y)
- Toggle: `role="switch"` + `aria-checked`
- 각 항목: `aria-label="{레이블} 자동 알림 {활성/비활성}"`
