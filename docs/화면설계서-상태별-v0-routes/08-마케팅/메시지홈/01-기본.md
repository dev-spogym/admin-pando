## 📝 디스크립션

### 사용 시점 (Trigger)
- `/message` 진입 시 (activeTab="send" 초기)
- 채널 기본값: "kakao" (알림톡)

### 필수 데이터
| 블록 | 소스 |
|------|------|
| 발송 이력 | `messages` 테이블 (최근 5건) |
| 회원 수 | `members` count (전체 발송용) |
| 자동 알림 상태 | `autoAlarms` state (localStorage) |

### 인터랙션 (User Actions)
1. 그룹 버튼 [전체] → `useAllMembers` 토글
2. [대상 검색] → RecipientModal 오픈
3. 채널 변경 → `02-SMS카드` / `03-카카오톡카드`
4. 변수 삽입 버튼 → content에 태그 추가
5. [메시지 발송] → 내용 검증 → `isPreviewModalOpen=true`
6. [상세 설정 →] → `/message/auto-alarm`

### 비즈니스 룰
- SMS 90자 이하=SMS(70원), 90자 초과=LMS(30원) 자동 전환
- 예약 발송: isReserved=true + reserveDate 필수
- 발송 이력 실시간 갱신 (confirmSend 후 fetchMessageHistory)

### 에지 케이스
- content 비어있으면 `toast.warning("메시지 내용을 입력하세요.")`
- isSending=true 중 재클릭 → 무시 (중복 방지)

### 접근성 (A11y)
- 채널 라디오: `role="radiogroup"` + `role="radio"` + `aria-checked`
