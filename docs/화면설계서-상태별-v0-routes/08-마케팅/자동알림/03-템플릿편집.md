## 📝 디스크립션

### 사용 시점 (Trigger)
- RuleCard MoreHorizontal 클릭 → `handleEdit(rule)`
- [설정 추가] 버튼 → 빈 규칙 생성 후 `handleEdit(newRule)`

### 필수 데이터
| 항목 | 소스 |
|------|------|
| editingRule | 선택된 AlarmRule |
| modalData | editingRule.template 기반 초기화 |

### 인터랙션 (User Actions)
1. 채널 버튼 → `modalData.channel` 업데이트
2. Select(시점/대상) → `modalData.timing / target` 업데이트
3. 변수 삽입 버튼 → `modalData.content` 에 변수 문자열 append
4. Textarea 입력 → 실시간 미리보기 업데이트
5. [테스트 발송] → 미구현 (향후 실제 발송 연동)
6. [저장하기] → handleSave() → localStorage 저장 → 모달 닫기

### 비즈니스 룰
- 편집 후 [저장하기]만 실제 저장 (취소 시 변경 내용 소실)
- persistAlarm = localStorage 저장 (DB 미연동)
- 글자 수 제한: 알림톡 1,000자 (화면에 표시만, 실제 제한 없음)

### 에지 케이스
- 새 규칙(설정 추가) content="" → 미리보기 "(내용을 입력하세요)"
- [취소] 후 재진입 → 이전 저장 상태로 복원

### 접근성 (A11y)
- 모달: `role="dialog"` + `aria-modal="true"` + `aria-labelledby`
- 닫기 버튼: `aria-label="모달 닫기"`
