## 📝 디스크립션

### 사용 시점 (Trigger)
- [신규 쿠폰 생성]: `setEditingCoupon(null)` + `setIsFormOpen(true)`
- [수정]: `setEditingCoupon(coupon)` + `setIsFormOpen(true)`

### 필수 데이터
| 항목 | 소스 |
|------|------|
| editingCoupon | null(신규) 또는 Coupon 객체(수정) |
| formData | editingCoupon || 기본값 |

### 인터랙션 (User Actions)
1. 쿠폰명 입력
2. [재생성] → 쿠폰 코드 갱신
3. 유형 선택 → 할인 설정 섹션 토글
4. 유효기간 방식 선택
5. [저장하기] → supabase insert/update → fetchCoupons

### 비즈니스 룰
- 쿠폰 코드: 자동 생성 `COUP-XXXX-XXXX` 형식
- 할인값 저장: `coupons.value` 컬럼 (Number 변환)
- maxUsage → `totalIssued` 한도 (초과 시 computedStatus='exhausted')

### 에지 케이스
- 쿠폰명 비어있음 → 서버 에러 (현재 클라이언트 검증 없음)
- 수정 시 code 필드는 변경 불가 (DB update에서 code 제외)

### 접근성 (A11y)
- 모달: `role="dialog"` + `aria-modal="true"`
- 라디오: `role="radiogroup"` + `role="radio"` + `aria-checked`
