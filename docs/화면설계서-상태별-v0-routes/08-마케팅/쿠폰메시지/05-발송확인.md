## 📝 디스크립션

### 사용 시점 (Trigger)
- TabNav [발급 이력] 탭 클릭
- `handleTabChange("history")` 실행

### 필수 데이터
| 항목 | 소스 |
|------|------|
| issuanceLogs | 현재 useState([]) — coupon_issues 테이블 미연동 |
| coupons | coupons 테이블 (쿠폰 선택 필터 옵션) |

### 인터랙션 (User Actions)
1. 발급 이력 탭 클릭 → 이력 목록 표시 (현재 빈 상태)
2. 검색/필터 → filteredLogs 실시간 업데이트
3. 회원명 클릭 → 회원 상세 페이지 이동
4. [엑셀 다운로드] → exportToExcel

### 비즈니스 룰
- coupon_issues 테이블 미연동 → issuanceLogs=[] (향후 연동 예정)
- status 배지: used=success / unused=warning / expired=error
- 탭 전환 시 필터 초기화 (handleTabChange 내 setFilterValues({}))

### 에지 케이스
- issuanceLogs 빈 배열 → emptyMessage="발급 이력이 없습니다."
- coupon_issues 테이블 연동 전: 항상 빈 상태

### 접근성 (A11y)
- 회원명 링크 버튼: `role="link"` + `aria-label="{memberName} 회원 상세보기"`
