  mermaid.initialize({
    startOnLoad: false,
    theme: 'default',
    securityLevel: 'loose',
  });
  const { svg } = await mermaid.render(`mermaid-${id}`, code);
  if (mermaidRef.current) mermaidRef.current.innerHTML = svg;
}, []);

## 📝 디스크립션

### 사용 시점 (Trigger)
- `01-로딩` 완료 후 첫 번째 다이어그램 자동 선택 및 렌더링

### 필수 데이터
| 블록 | 소스 | 조건 |
|------|------|------|
| 다이어그램 목록 | 하드코딩 DIAGRAMS 배열 | 카테고리/제목/Mermaid 코드 |

### 인터랙션 (User Actions)
1. 사이드바 항목 클릭 → 해당 다이어그램 렌더링
2. 검색 → 제목/카테고리 포함 검색
3. 확대 보기 → `03-확대뷰` (전체화면 모달)
4. PDF 내보내기 → `04-PDF내보내기`
5. 새로고침 → 현재 다이어그램 재렌더링

### 비즈니스 룰
- Mermaid 렌더링: 클라이언트 사이드 only (SSR 비활성)
- 선택된 다이어그램 좌측 border-l-2 border-primary 강조

### 에지 케이스
- Mermaid 코드 오류 → `05-에러` 상태 (뷰어 내 오류 메시지)
- 다이어그램 목록 0건 → "다이어그램이 없습니다" 빈 상태

### 접근성 (A11y)
- 사이드바 `role="navigation"`, 각 항목 `role="link"`
- 뷰어 SVG `aria-label="{다이어그램 제목}"`

### 연결 화면
- 이전: `01-로딩`
- 다음: `03-확대뷰` / `04-PDF내보내기`
