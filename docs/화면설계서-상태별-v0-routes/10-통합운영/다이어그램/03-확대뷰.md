줌 조작:
const [zoom, setZoom] = useState(1);
// 마우스 휠 줌
const handleWheel = (e: WheelEvent) => {
  if (e.ctrlKey) {
    e.preventDefault();
    setZoom(z => Math.max(0.3, Math.min(4, z - e.deltaY * 0.001)));
  }
};

단축키:
useEffect(() => {
  const handler = (e: KeyboardEvent) => {
    if (e.key === 'Escape') setIsZoomed(false);
    if (e.key === '+' && e.ctrlKey) setZoom(z => Math.min(4, z + 0.1));
    if (e.key === '-' && e.ctrlKey) setZoom(z => Math.max(0.3, z - 0.1));
  };
  document.addEventListener('keydown', handler);
  return () => document.removeEventListener('keydown', handler);
}, []);

사용 유틸:

## 📝 디스크립션

### 사용 시점 (Trigger)
- "확대 보기" 버튼 클릭 시
- isZoomed=true

### 필수 데이터
- 없음 (현재 선택된 다이어그램 SVG 재사용)

### 인터랙션 (User Actions)
1. 확대/축소 버튼 → zoom 조절 (0.5x ~ 3x)
2. Ctrl + 마우스 휠 → 줌 조절
3. 초기화 → zoom = 1
4. PDF → `04-PDF내보내기`
5. 닫기 / ESC → `02-기본`

### 비즈니스 룰
- 배경 dim (bg-black/80), 클릭으로 닫기 불가 (오작동 방지)
- 줌 범위: 0.3x ~ 4x
- SVG는 재렌더링 없이 기존 DOM 재사용

### 에지 케이스
- 복잡한 다이어그램 → 스크롤로 탐색
- 모바일 접근 → pinch-to-zoom 미지원 안내

### 접근성 (A11y)
- `role="dialog"`, `aria-modal="true"`, `aria-label="{제목} 확대 보기"`
- ESC로 닫기, focus trap

### 연결 화면
- 이전: `02-기본`
- 다음: `02-기본` (닫기)
