## 📝 디스크립션

### 사용 시점 (Trigger)
- Mermaid.js 동적 임포트 실패 (네트워크 오류)
- 특정 다이어그램 코드 문법 오류로 렌더링 실패
- SVG 생성 중 예외 발생

### 필수 데이터
- 없음 (오류 상태 표시)

### 인터랙션 (User Actions)
1. "재시도" → 동일 다이어그램 재렌더링
2. "다른 다이어그램 보기" → 첫 번째 항목으로 초기화
3. "페이지 새로고침" → window.location.reload() (전체 실패 시)

### 비즈니스 룰
- 개발 환경: Mermaid 오류 메시지 raw 표시
- 운영 환경: 일반적인 오류 메시지만 표시
- 특정 다이어그램 오류 ≠ 전체 기능 실패 (다른 다이어그램 선택 가능)

### 에지 케이스
- Mermaid 코드 문법 오류 → 해당 다이어그램만 오류, 다른 항목 정상
- 전체 로드 실패 → 좌측 목록 + 오류 메시지만 표시

### 접근성 (A11y)
- 오류 컨테이너 `role="alert"`
- 재시도 버튼이 초기 포커스 수신

### 연결 화면
- 이전: `01-로딩` / `02-기본`
- 다음: `02-기본` (재시도 성공)
