# SCR-H801 리드 관리 — 기본화면

## 1. 화면 목적

- 신규문의부터 등록완료까지의 리드 파이프라인을 운영하는 핵심 화면이다.

## 2. 레이아웃

```text
Header
KpiCards
ViewToggle
FilterBar
LeadList or Kanban
```

## 3. 상태 기준

- 신규문의
- 연락완료
- 상담예정
- 방문완료
- 등록완료
- 보류
- 미전환

## 4. 핵심 기능

- 목록/칸반 전환
- 상태 이동
- 담당자 지정
- 신규 리드 등록
- 회원 전환

## 5. 권한 차이

- `fc`는 기본 운영 권한
- `manager` 이상은 팀 전체 조정 가능

## 6. 퍼블리싱 포인트

- FC가 하루 종일 쓰는 화면이라는 전제로 속도와 상태 인지가 중요하다.
- 칸반 컬럼 수는 운영 상태와 정확히 일치해야 한다.
