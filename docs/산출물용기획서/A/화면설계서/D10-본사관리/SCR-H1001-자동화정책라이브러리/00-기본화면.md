# SCR-H1001 자동화 정책 라이브러리 — 기본화면

## 1. 화면 목적

- 본사가 전사 자동화 정책 세트를 관리하는 화면이다.

## 2. 레이아웃

```text
Header
PolicySetCards
PolicyTable
StepPreviewPanel
UsageStatusPanel
```

## 3. 핵심 기능

- 정책 세트 생성/수정/중지
- 적용 범위 설정
- 스텝 구성
- 템플릿 연결
- 지점 수정 허용 범위 설정

## 4. 운영 규칙

- 고정 D-day 대신 스텝 리스트 구조를 사용한다.
- 사용 중지된 정책은 신규 적용만 막고 기존 사용 지점은 영향 범위를 고지한다.

## 5. 퍼블리싱 포인트

- 본사 화면은 정책 설계 도구처럼 보여야 한다.
- 카드에서 정책 성격, 범위, 사용중 지점 수가 한 번에 읽혀야 한다.
