# SCR-H401 수업 캘린더 — 기본화면

## 1. 화면 목적

- 수업 일정과 예약 현황을 한 화면에서 관리한다.

## 2. 레이아웃

```text
PageHeader
FilterBar
Calendar
RightDetailPanel
```

## 3. 핵심 기능

- 일정 조회
- 수업 등록/수정
- 예약 수와 정원 확인
- 승인대기 일정 처리

## 4. 상태 규칙

- 수업 상태와 예약 상태를 분리 배지로 표시한다.
- 마감, 승인대기, 취소는 강한 대비가 필요하다.

## 5. 퍼블리싱 포인트

- 시간표 가독성이 우선이다.
- 우측 상세 패널은 빠른 운영 판단용으로 카드형 정보가 적합하다.
