# SCR-H101 로그인 — 기본화면

## 1. 화면 목적

- 서비스의 공통 진입 화면이다.

## 2. 핵심 기능

- 이메일/비밀번호 로그인
- 세션 만료 후 재로그인
- 역할별 진입 분기

## 3. 퍼블리싱 포인트

- 로그인 실패와 잠금 상태 문구가 명확해야 한다.
- 관리자 웹과 앱 공통 기준을 유지하되, 폼 밀도는 플랫폼에 맞춘다.
