# 회원앱 기능명세서

## 1. 목적

- 회원앱을 관리자 웹과 연결된 운영 채널로 정리한다.
- 역할별 메뉴와 동작 범위를 실제 사용 기준으로 정의한다.

## 2. 포함 범위

- 회원 홈
- 예약/내 예약
- QR 체크인
- 이용권 조회
- 결제 이력
- 알림함
- FC/트레이너/강사용 운영 메뉴
- 스태프 체크인 지원

## 3. 역할별 핵심 기능

### 3.1 member

- 예약
- 체크인
- 이용권 확인
- 결제/혜택 확인
- 공지/문의

### 3.2 trainer / golf_trainer

- 오늘 수업
- 회원 확인
- 수업 완료
- 서명 처리

### 3.3 fc

- 만료예정 회원 큐
- 재등록 후속관리
- 상담 기록 확인

### 3.4 staff

- 현장 체크인 지원
- 회원 방문 상태 확인
- 공지/운영 안내 확인

## 4. 운영 규칙

- 앱은 관리자 웹 전체 기능을 복제하지 않는다.
- 회원은 자신의 정보만 본다.
- FC/트레이너/강사는 현장 운영에 필요한 최소 화면만 본다.
- 스태프는 현장 체크인과 운영 보조에 필요한 최소 화면만 본다.
- 앱의 상태 라벨은 운영 친화적 한글을 사용한다.
- 자동화 알림과 앱 알림함의 이력은 일관되어야 한다.

## 5. 퍼블리싱 포인트

- 회원 홈은 예약, 이용권, 알림 우선이다.
- 현장 인력 홈은 오늘 할 일과 수업 처리 우선이다.
- 앱은 관리자 웹보다 한 단계 더 단순해야 한다.
